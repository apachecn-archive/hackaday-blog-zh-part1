# 在没有微控制器的情况下改进您的焊机

> 原文:[https://hack aday . com/2011/09/10/改善你的无微控制器焊机/](https://hackaday.com/2011/09/10/improving-your-welder-without-a-microcontroller/)

![](../Images/55bc1786f4dec094e869a4970f824a82.png "tig")

当一个硬件被拆开、重新组装并最终超出原始设备的能力时，我们总是印象深刻。[戴夫]和[威尔]的自制 TIG 焊机也不例外。

当[Dave]和[Will]开始致力于将一台简单的交流电焊机改造成一台拥有一切可以想象的功能的焊机时，他们决定保持简单。在查看了一些高价格的商业焊接机之后，他们提出了一系列他们想要的功能，并决定在 TTL 和 CMOS 逻辑中实现这些功能。这些人不想采用微控制器解决方案，因为不是每个人都会编码，而且只要使用最少的工具，分立芯片很容易排除故障。

对于高压部分的建设，原来的反激式变压器取代了霓虹灯变压器和自制火花隙和电容器。自制火花隙和上限的计划并不十分奏效，所以他们被商业单位所取代。这些家伙包括他们建造的原理图和 [PCB 布局](http://www3.telus.net/public/a5a26316/PCB_Layout_PDFs/RoutedPCBbottom.pdf) (PDF 警告)。看到一个令人惊叹的逻辑芯片总是很棒，改进一个现有的工具永远不会有坏处。
