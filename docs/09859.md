# 制造磁性旋转编码器

> 原文:[https://hack aday . com/2012/03/08/building-a-magnetic-rotary-encoder/](https://hackaday.com/2012/03/08/building-a-magnetic-rotary-encoder/)

![](../Images/8f2a0d1824a747a878f6e633af6ec751.png "magnetic-rotary-encoder")

[长发黑客]承担了一个高分辨率的 3D 打印机建设。他得到了一些电机来驱动构建平台，但它没有内置编码器。他知道，由于灰尘、光线以及环境光线，光学编码器轮可能会出现问题，因此他开始寻找更好的方法向控制器提供反馈。他最终[建造了自己的磁性旋转编码器](http://dangerousprototypes.com/forum/viewtopic.php?f=56&t=3669)，如上图所示。

该系统的核心是一个 AS5043 磁性旋转传感器。这种芯片的价格从 6.50 美元到 11 美元不等，可以非常精确地检测和报告磁场的旋转。旋转数据可以使用 SPI 以度数读取，但如果更符合您的需要，一些引脚上似乎也有格雷码输出。芯片测量的磁铁安装在一个套中，该套铣成座在轴承环内。

长毛黑客选择的 3D 打印方法使用投影仪和光固化树脂来实现另一台高分辨率打印机的效果。
