# 遥控玩具 Hexababy 保证会让你做噩梦

> 原文：<https://hackaday.com/2011/07/30/rc-hexababy-is-guaranteed-to-give-you-nightmares/>

![hexapod_baby](img/a59d8da48528d55cb1c5b7ee9d9ac0f4.png "hexapod_baby")

在本周的黑客活动中，[Dino]正在开发一个机械猫玩具，但由于某种原因，该项目在接近尾声时失败了。随着时间的流逝，他不得不在很短的时间内想出一些东西，利用他手头所有的东西。幸运的是，他在当地的旧货店买了一些非常奇怪的东西，并为这个最后一刻的项目准备了一个无形的洋娃娃的头。

受《玩具总动员》中的锡德·菲利普斯(Sid Phillips)的启发，[迪诺]玩偶的头部六足动物既简单又令人毛骨悚然。他有一个来自 RadioShack 的遥控六足机器人，觉得把它和洋娃娃的头结合起来会很有趣。他用一些发光二极管取代了娃娃的眼睛，当六足动物后退时，这些发光二极管是绿色的，但当它向你前进时，会发出明亮的红色。唯一可能更恐怖的是，如果[迪诺]增加了一个声音盒，当娃娃四处爬行时，它会哀怨地叫“妈妈”！

这是一个相对愚蠢的项目，但它给了我们一个很好的笑声。最令人不安的亮点是[迪诺]在 6 点钟左右用木钻头挖掉娃娃的眼睛。

如果你想消磨几分钟，一定要去看看——[迪诺的]作品一如既往地有趣。
