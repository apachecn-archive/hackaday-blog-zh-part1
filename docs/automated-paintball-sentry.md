# 自动彩弹哨兵

> 原文：<https://hackaday.com/2009/06/26/automated-paintball-sentry/>

[https://www.youtube.com/embed/7vYKwih8Izk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/7vYKwih8Izk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

读者[Ben Godding]发来了他的高级设计团队的自动彩弹哨兵的视频。框架由等离子切割铝制成。彩弹枪使用远离枪体安装的定制料斗。它有两个网络摄像头，提供 160 度的视野，图像处理是由双核奔腾 CPU 从闪存卡启动 windows xp 完成的。计算机使用 PIC24 与 1/4 比例 RC 伺服系统连接。彩弹哨兵既可以通过计算机图形用户界面(当显示器可用时)进行配置，也可以通过键盘和 4×20 字符显示器进行配置。
