# 带手套控制的电动登山板

> 原文：<https://hackaday.com/2011/10/10/electric-mountain-board-with-glove-control/>

![](img/6c838a640746dc2c4eb9b12dd5b5b8b8.png "board")

去年夏天，我们看到[安德烈斯·古兹曼]的电动登山板在伊利诺伊大学校园里四处撕裂。[他又回来了](http://www.youtube.com/watch?v=686oJ7bNFIU)，只不过这次板子不是用 PlayStation 控制器控制的。[安德烈斯]建立了一个无线手套来控制他的登山板。

Arduino 和电源安装在手套上。2.4GHz 收发器充当手套和电路板之间的通信链路。速度控制由 Sparkfun 的这个柔性传感器来控制。将 flex 传感器夹在中指和无名指之间,[Andres]需要做的就是稍微弯曲他的手指。

主板还内置了许多安全功能。为了给电路板的马达供电，在拇指下面的手套上有一个安全开关。如果(安德烈斯)要冒一次险，他会松开开关，棋盘就会停下来。[Andres]还确保如果无线连接中断，电路板会关闭。这个建筑看起来很安全，即使在下面的视频中他在校园里四处奔跑。
