# 基于 Arduino 的蓝牙手表

> 原文:[https://hack aday . com/2011/08/23/bluetooth-watch-based-on-an-arduino/](https://hackaday.com/2011/08/23/bluetooth-wristwatch-based-on-an-arduino/)

![](../Images/6b65884f1124d9f7054cc48751de5deb.png "watch")

我们不得不承认，上周我们错过了 TI Chronos 手表的交易。虽然我们仍然对这些手表销售如此之快感到有点恼火，但[Ahmet]送来了他的[开源蓝牙手表](http://mclightning.com/innowatch-opensource-bluetooth-watch-initiative/announcements)，我们认为这最终可能是一个不错的替代品。

这款手表是围绕 Arduino Pro Mini、回收的诺基亚 LCD 和  打造的。一个小小的蓝牙客户端应用程序[Ahmet]写道，蓝牙可以连接到诺基亚 N900。他还为手表的 LCD 显示屏编写了一个小 GUI。后来，他能够显示未接来电和新邮件，现在正在支持 N900 媒体播放器上的歌曲更换。

不可否认，在外壳上需要做一些工作。尽管如此，这款手表的潜力比我们去年看到的作为手表的 iPod 项目要大得多。现在，我们正在考虑可以给[Ahmet]的手表添加什么。加速度计可能是我们列表的第一位，但是如果你有任何想法，请在评论中留下。
