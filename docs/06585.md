这是可爱的星球大战机器人没有保护皮肤的样子吗？这个受 R2D2 启发的机器人是另一个奥林工程学院(那个[的计算机控制蛋糕装饰师](http://hackaday.com/2011/01/05/cnc-cake-decorator/)就是来自那里)由【Nathaniel Ting】和他的同学开发的。唉，它缺乏自主性，依赖于操作员的指导。但是我们喜欢它的建造质量。两个摩托车电池为三轮车两条后腿的 DC 马达供电。它可以通过无线 Xbox 控制器或 Python 界面驱动，该界面还可以随机播放电影中的 droid 音频片段。顶部是一个倾斜的投影仪，用来显示莱娅公主请求帮助的画面。也就是说，在操作员插上延长线给它通电之后。哦，看起来还是很有趣的。休息之后自己看吧。
