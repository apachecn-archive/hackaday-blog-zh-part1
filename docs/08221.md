# 给你的家庭影院添加流光溢彩克隆系统可以大幅降低价格

> 原文:[https://hackaday . com/2011/08/01/add-ambi light-clone-system-to-your-home-theater-just-get-a-big-price-cut/](https://hackaday.com/2011/08/01/adding-ambilight-clone-system-to-your-home-theater-just-got-a-big-price-cut/)

![](../Images/f043a84c0c47d53790adb5a9fdf8b9e2.png "ambilight-from-inexpensive-parts")

每当我们得到一个声称一个项目既便宜又容易的提示时，我们都会怀疑地扬起眉毛。但是[Yonsje]并没有告诉我们一个故事，他的阿米立夫克隆体确实降低了复杂性并削减了价格。

对于外行人来说，这是飞利浦 Amilight 系统的克隆产品，多年来该系统一直是他们一些电视的选择。它将 RGB LEDs 放在框架的背面，指向墙壁。它们被调整到显示器的边缘颜色，将室内环境光的颜色与屏幕上的颜色联系起来。这些年来，我们已经看了[一吨](http://hackaday.com/2011/03/03/arduino-based-ambient-lighting-improvements/)的[克隆](http://hackaday.com/2011/06/05/livelight-is-an-expertly-crafted-ambilight-clone/)，只需在我们的博客中搜索“流光溢彩”。

像其他人一样，这个迭代依赖于你从计算机上回放视频。[Yonsje]正在用一个 Arduino 和他自己的盾牌连接到 HTPC。屏蔽中的 NPN 晶体管驱动 RGB LEDs。真正节约成本的是他的照明光源。Deal Extreme RGB LED 条的价格仅为 11.30 美元(含运费)，可以分成六个不同的部分，均匀分布在电视机周围。休息后在视频里看看。
