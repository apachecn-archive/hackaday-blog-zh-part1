# 热水瓶温度计

> 原文：<https://hackaday.com/2010/03/15/thermos-temperature-meter/>

 <https://www.youtube.com/embed/UXsxtEmKgbw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>我们总是对真空热水瓶的工作效率感到有点惊讶，但最终内容物会冷却下来(或升温，取决于里面有什么)。[Gamesh_]使用 Arduino 和温度传感器在他的热水瓶上添加了一个温度计。最初的帖子是葡萄牙语的，但是【布鲁诺】<a href="http://interblogando.blogspot.com/2010/03/smart-vacuum-bottle-with-arduino.html" target="_blank">用英语重新发布了它</a>。</p> <p>温度传感器是由测量体温的数字温度计改造而来的。组成指示条的发光二极管的孔被熔化在塑料外壳的侧面。当热液体在大约 0:45 被倒入视频时，你可以看到 Arduino 挂在锅的另一边，一根电源线在笔记本电脑后面。很高兴看到这个移植到一个不太强大的芯片上，并从一个小硬币电池上运行，但我们喜欢这个概念。</p> </body> </html>
