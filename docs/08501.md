# 捷克人从 31 公里外看风景

> 原文:[https://hack aday . com/2011/09/05/捷克-ing-out-of-view-from-31 公里/](https://hackaday.com/2011/09/05/czech-ing-out-the-view-from-31-kilometers/)

![](../Images/8ed99bfc1a464f1738c1a4e33c99a519.png "balloon")

捷克业余近地天体小组 cz anzo T1 的团队上周末发射了一个我们见过的最好的高空气球，由此产生的视频非常出色。

该团队的[构建博客](http://blog.czanso.com/) ( [谷歌翻译链接](http://translate.google.com/translate?hl=en&sl=cs&tl=en&u=http%3A%2F%2Fblog.czanso.com%2F)面向所有没有 Chrome 的人)介绍了他们有效载荷的设计和构造。像我们见过的所有其他气球一样，[一个泡沫聚苯乙烯冷却器](http://blog.czanso.com/2011/06/kompletace-nakladu-czanso-payload-box/)被用于封闭，但是有很多非常整洁的附加物使这个建筑很特别。

该团队包括一个[切割装置](http://blog.czanso.com/2011/05/odpojeni-balonu-servem/)，以防气球被喷气流卡住。如果没有这个减速装置，气球可能会在离发射点数百英里远的地方降落。这就是他们给出削减装置的原因，尽管我们怀疑这是一个玩弄[烟火绳索削减](http://blog.czanso.com/2010/05/bezpecnostni-cut-down-zarizeni/)的借口。T4 的视觉和听觉警报 T5 是我们在许多高空气球发射中没有见过的，这很奇怪，因为它使地面回收更加容易。

该团队有许多来自飞行的视频，[Pavel Richter] [转储到 Vimeo](http://vimeo.com/czanso) 上。我们真的很喜欢展示整个布拉格的镜头，你可以在休息后看看。
