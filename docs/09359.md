# 手持捷豹使 32 或 64 位成为可移植的

> 原文:[https://hack aday . com/2011/12/22/hand held-jaguar-makes-32-或-64-bits-portable/](https://hackaday.com/2011/12/22/handheld-jaguar-makes-32-or-64-bits-portable/)

[![](../Images/f3ff586ceee1f8a7ba86cb4ad404d1c2.png "jaguar")T2】](http://hackaday.com/wp-content/uploads/2011/12/jaguar.png)

便携 NESs，超级任天堂，N64，甚至一张便携世嘉 CD 都没有问题。然而，备受争议的便携式版本雅达利捷豹 T1 怎么样呢？[邪恶的点头]成功了，看起来很棒。

这种构造对于便携式控制台来说是相当标准的。PS1 屏幕用于显示，切割和重新布线的控制器提供输入。从我们在[构建日志](http://www.made-by-bacteria.com/viewtopic.php?f=18&t=316)上看到的，将 104 针盒式插槽移动到带状电缆上是一种耐心的锻炼。这种情况是绝对惊人的纹理完成，我们希望看到一个 90 年代初的主机。当然，[点头]保留了数字键盘；反正还有剩余的空间。

我们不能嘲笑捷豹或者 Nod 的体型。这是一个伟大的执行，有一个令人印象深刻的游戏库，包括*蠕虫*、*雷曼、毁灭战士、*和*神秘侠*。尽管如此，我们还是想知道安装了[捷豹光盘](http://en.wikipedia.org/wiki/Atari_Jaguar_CD)后的版本会是什么样子。
