# 无人机医疗快递员

> 原文：<https://hackaday.com/2008/09/14/uav-medical-couriers/>

![](img/08fd0c1b21914553fbaef6c151636e40.png "had_meduav")

我们对大多数旨在帮助偏远村庄的技术持怀疑态度([是的，甚至是那个](http://laptop.org/))，但这些新的[无人机医疗快递](http://technology.newscientist.com/article/dn14718-robot-spyplanes-get-new-role-as-medical-couriers.html)看起来是个好主意。在偏远的南非村庄，医学样本分析的周转时间可能会非常痛苦。一个工程师团队试图改造两种不同的无人驾驶飞行器来运输医学样本。这些可能是需要检测的血液或唾液。测试结果将像现在一样通过电话传递，但最初的传输时间会快得多。两架无人机中较大的一架可携带 500g 这足够运两个单位的血来输血了。无人机可以用手发射，并能在风速高达 45 公里/小时的情况下生存。它们自主飞行预编程的路线，不需要任何操作员干预。该团队已经成功进行了两次试飞，正在等待南非民航局的批准。出于安全考虑，他们只运送可以在飞行前消毒的样本。《新科学家》在休息之后有一个短片。
