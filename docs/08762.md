# 从电器中回收废热

> 原文:[https://hack aday . com/2011/10/06/从电器中回收废热/](https://hackaday.com/2011/10/06/reclaiming-waste-heat-from-appliances/)

![](../Images/96e07e3ae318dd09398ec414e1ed694c.png "laundry-dryer-heat-exchanger")

在这里，我们看到[Christopher Suprock]在他的地下室洗衣区闲逛，炫耀他的智能热交换器。使用该装置的原因很简单，当你使用干衣机、热水器和其他产生热量的设备时，能量经常以热废气的形式被浪费掉。为什么不通过获取热空气来温暖你的房子，从而最大限度地为你的衣服烘干省钱呢？

你在烘干机顶部看到的是一个热交换器。来自干燥器的废气在排出室外之前通过散热器组件。一些控制硬件监控输入侧的温度，并在检测到高于环境空气的温度时开启风扇。然后空气流经散热器，从废气中吸收热能。在广告之后的视频中，可以看到克里斯托弗的解释，以及烘干机运行时的一些热量读数。

这让我们想知道，如果热交换器在排气之前将废气降低 50 度，这是否会导致任何冷凝问题？
