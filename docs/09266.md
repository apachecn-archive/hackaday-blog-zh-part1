# 照明风扇拉节省你在黑暗中四处扑腾

> 原文:[https://hackaday . com/2011/12/09/lighted-fan-pull-saves-you-from-flapping-around-around-in-the-dark/](https://hackaday.com/2011/12/09/lighted-fan-pull-saves-you-from-flailing-around-in-the-dark/)

![lighted-fan-pull](../Images/60a130cf58dd69e4b40c6a55249e7c9d.png "lighted-fan-pull")

像许多人一样，[yardleydobon]在晚上他的房间完全黑暗的时候，很难找到他的吊扇的拉链。与其继续瞎抓链子，他决定想办法照亮它。

他从拆卸太阳能庭院灯开始，保留太阳能电池、光敏电阻和电池。在将电子产品削减到最基本的部分后，他将它们安装在一个塑料电池存储盒中，并将其连接到风扇灯的外部。[yardleydobon]然后将一对电线从电子箱向下连接到链条的末端，在那里他添加了一个 LED 和一个半透明的拉手来漫射光线。

他承认，这不是周围最好看的修改，但它在紧要关头做了工作。他有一些想法，如果他有时间修改设计，他可能会付诸实施，我们打赌你们中的许多人也一样。如果有，一定要在评论里分享出来。
