# 万圣节黑客:适合[骷髅杰克]的灯光音乐秀

> 原文：<https://hackaday.com/2011/10/28/halloween-hacks-a-light-and-music-show-fit-for-jack-skellington/>

![Halloween Hacks Banner](img/b509209447a3a514553925a79033946f.png "Halloween Hacks Banner")

![singing_pumpkin_halloween_show](img/5b1dbdcefc488bba0e00b66622ccaf21.png "singing_pumpkin_halloween_show")

[Greg] [正在用他最近购买的 Arduino 寻找可以制作的东西](http://www.instructables.com/id/Singing-Pumpkins-Arduino/?ALLSTEPS)，随着万圣节的临近，他认为一个很酷的灯光展示会是一个很好的项目。他在网上浏览了一下，[找到了这个教程](http://www.instructables.com/id/Singing-PumpkinsParallel-Port-Relay-Box/?ALLSTEPS)，它展示了如何通过计算机的并口来控制一个唱歌的南瓜合唱团。因为他再也没有带并行端口的电脑了，他决定尝试用 Arduino 重新创建它。

[Greg]收集了八个点亮的南瓜灯，以及一些继电器和其他各种部件。他连接了继电器，当 Arduino 切换时，就会触发每个南瓜的内置灯。他坐下来，仔细听完了《圣诞节前的噩梦》中的“这是万圣节”，编排了每一个南瓜来扮演电影中一个角色的声音。

当表演开始时，展示从一群不起眼的闪烁着蜡烛的南瓜变成了一群赞美万圣节优点的食尸鬼。

看起来真的很有趣，所以一定要看看下面的视频。如果你想在大日子到来之前快速展示一下，[Greg 的]源代码和图表应该会让你朝着正确的方向前进。
