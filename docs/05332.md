# Midi 手套

> 原文:[https://hackaday.com/2010/05/12/midi-gloves/](https://hackaday.com/2010/05/12/midi-gloves/)

![](../Images/73582356703a79305bb0446cdb331628.png "midi-gloves")

我们被基于手套的外围黑客淹没了。这是康奈尔大学的另一个最终项目，通过在手套上增加 8 个压电传感器，从而避开 T2 的小手指。我们喜欢这个，因为它很容易构建，并且如果您想构建自己的 midi 接口，它的实现有很好的文档记录。

正如你在休息后所看到的，这很容易与 Garage Band 这样的音乐软件一起使用，因为它是一个标准的 MIDI 设备。此外，如果您想改变每个手指的功能，MATLAB 界面允许自定义映射。

我们还记得多年前托德·马乔夫的《Bug Mudra》,这是我们第一次用手套表演。我们希望[我们看到的音乐输入黑客](http://hackaday.com/2010/01/20/hey-man-ssh-to-my-guitar-and-setup-the-multitouch/)将会催生出全新一代的音乐创新者。
