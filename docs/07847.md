# 回放 Blox

> 原文：<https://hackaday.com/2011/06/14/blox-that-play-back/>

![](img/6222e205670e8ef7b2fa19513e54ddfd.png "blox2")

我相信我们大多数人都记得小时候玩积木，现在这个古老的儿童玩具已经被来自德克萨斯大学的一组电气工程师填充了足够的电气优点。Blox 的大小约为标准的魔方，每个包含 16 个红外传感器，4 个触摸面板，3 轴加速度计，ZigBee 无线模块，2 英寸有机发光二极管显示器和电池，所有这些都由 ARM Cortex 处理器控制。

Blox 在作为交互式分布式计算系统一起使用时显示了它们的全部潜力。Blox 是一个开源项目，所以所有的构建细节、原理图和代码都可以在网站上找到。关于 Blox 的概述和演示，请在休息后观看视频。
