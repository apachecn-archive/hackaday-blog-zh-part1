# 多点触摸背投电视

> 原文:[https://hack aday . com/2008/05/20/multi touch-back-projection-TV/](https://hackaday.com/2008/05/20/multitouch-rear-projection-tv/)

克里斯托夫·杰特做了一件惊人的工作[将一台 56 英寸的背投电视改造成了多点触控显示屏](http://www.instructables.com/id/Back-projection-56-inch-multitouch-television./)。他最初的灵感来源于这个[制图桌项目](http://www.instructables.com/id/Interactive-Multitouch-Display/)。屏幕是一块 1/2 英寸的大亚克力板，背面贴有屏幕材料。屏幕边缘由 168 个红外 led 包围。当指尖接触表面时，它会散射 led 的红外线。网络摄像头看到这些散射光，并确定手指的位置。盒子里面是一个标准的视频投影仪。这是对旧设备的再利用，我们喜欢看到业余爱好者在制造商不在的地方开拓市场。关于多点触摸项目的更多信息，我们建议使用[自然用户界面组](http://www.multitouch.nl/)。下面是[Christopher]的展示视频:
