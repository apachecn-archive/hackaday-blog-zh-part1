# 增强木琴

> 原文：<https://hackaday.com/2009/11/02/augmented-xylophone/>

在一个名为多木琴综合项目中，Ania 一直致力于扩展木琴。她在每个木琴键的底部安装了一个压电传感器，与 Arduino 接口。当木槌敲击一个键时，相应的盒子会以几种方式之一增强声音。看起来她做了一个盒子的原型，这个盒子会拨动橡皮筋，一个用螺线管把小钹合在一起，一个会拨弄装满珠子的玻璃杯，一个会振动装满水的玻璃杯，还有一个会拨弄链条。

很高兴看到木琴对于[乐器黑客](http://hackaday.com/2009/09/23/berlin-hack-day/)是多么的多才多艺。她的 Flickr 套装在上面有链接，但我们也在休息后嵌入了一些原型视频。
