# 电子呜呜祖拉

> 原文：<https://hackaday.com/2010/07/13/electronic-vuvuzela/>

[https://www.youtube.com/embed/AHsgRuAKJdQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/AHsgRuAKJdQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

想惹恼其他粉丝，但没有足够的肺活量来完成这项工作？电子呜呜祖拉正是你需要的东西。塑料制噪器在今年的世界杯上如此普遍，以至于一些[人想出了滤音器](http://hackaday.com/2010/06/15/vuvuzela-removal/)来消除声音。电子演奏要小得多，使用一个 555 定时器来模拟小扬声器上的乐器。[亨特的]建设有五个不同的音符按钮，可以改变一些电位计。没有示意图，但是对于令人讨厌的东西，你不想让它太容易复制。

亨特在他的网站上添加了这张示意图，它为那些享受和平与宁静的人带来了厄运。
