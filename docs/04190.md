# 金属竞赛结合了操作和计算器游戏

> 原文：<https://hackaday.com/2009/10/09/metal-race-combines-operation-and-calculator-game/>

![metal_race](img/f5a243b1698d2e9b5f38fc031c3053dc.png "metal_race")

[Greg]对 TI-83 图形计算器的比赛感到怀旧。在游戏中，你的车是静止的，通过控制从一边到另一边移动滚动迷宫来避免撞车。他着手建立一个游戏的实体版本[,这个游戏有一个“不要碰边”的概念，让我们想起了](http://www.urbanhonking.com/ideasfordozens/2009/10/metal_race_game_fabrication_an.html)[行动](http://en.wikipedia.org/wiki/Operation_(game))。

游戏板是由回形针构成的金属丝框架，然后连接到电动框架上。该车也是金属的，从一个报废的光盘驱动器连接到镜头滑板。迷宫从左向右滚动，由两个拱廊按钮控制车辆上下运动。Arduino 控制电机并监控按钮输入。他计划添加一个蜂鸣器，当金属车“撞”进迷宫的铁丝墙时会发出声音。休息之后我们嵌入了工作游戏的视频。更多的建筑照片[请看他的 flickr 套装](http://www.flickr.com/photos/unavoidablegrain/sets/72157622356805495/)。T3
