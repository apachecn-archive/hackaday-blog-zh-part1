# 用磁铁控制的机器人眼科手术

> 原文:[https://hack aday . com/2010/09/14/用磁铁控制的机器人眼睛手术/](https://hackaday.com/2010/09/14/robotic-eye-surgery-controlled-with-magnets/)

![](../Images/4553ce6c430cf4d76c465ab5f85bf55c.png "robotic-surgery-controlled-by-magnets")

如果你需要做眼部手术，你可能会发现自己被绑在这个装置上。这是一个[磁场发生器，用来操纵一个微小的、无束缚的探针](http://www.botjunkie.com/2010/09/13/giant-magnets-control-tiny-robots-in-your-eyeballs/)。它被称为 OctoMag，其想法是将一个不到半毫米大小的机器人注射到你的血管系统中，通过使用这些线圈，它可以粉碎眼睛内部小通道中的血凝块。

一旦你看了广告之后的剪辑，我们相信你会同意这是一项了不起的技术。尽管如此，一想到在活的有机体上进行的手术，我们就感到害怕，但是我们确信，随着时间的推移，这种恐惧会消退。目前，这看起来更像是《发条橙》中的一种疗法。
