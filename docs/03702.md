# 万向摄像机稳定器

> 原文：<https://hackaday.com/2009/07/15/gimbal-camera-stabilizer/>

![steadycam](img/ea4370fed57afbb8faa5f1c3690281c1.png "steadycam")

专业摄影师使用稳定相机使他们的照片看起来平滑干净。然而，它们的价格对于独立电影公司的预算来说通常太高了。[之前的尝试](http://steadycam.org/)尝试过增加配重，将相机从手中移开。【yb2 normal】[采取了一种不同的方法](http://www.yb2normal.com/DIYsteadicam.html)，使用一个摆锤和万向架来保持相机直立。万向架可以沿 3 个轴自由旋转，因此摄像机可以保持在原位。整个花费不到 15 美元。他和何坐骑一起做的第一个视频是在休息之后。
