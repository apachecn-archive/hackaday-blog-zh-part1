# 32 LED 视点地球仪

> 原文：<https://hackaday.com/2010/05/19/32-led-pov-globe/>

[https://www.youtube.com/embed/bt1UEGD3BDk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/bt1UEGD3BDk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这里有一个[光滑版的 POV 地球仪](http://easyelectronics.ru/3d-led-globus.html) ( [谷歌翻译](http://translate.google.com/translate?js=y&prev=_t&hl=en&ie=UTF-8&layout=1&eotf=1&u=http://easyelectronics.ru/3d-led-globus.html&sl=ru&tl=en))。由[席惟伦]创造的这个地球仪有 32 个发光二极管，由一个旋转线圈供电。该布局在操作中看起来相当稳固，POV 效果在相机上显示得非常好。您可以从项目页面获得原理图和源代码。

我们找到了这个项目，就像之前[HackedGadgets.com](http://hackaday.com/2010/05/04/pov-led-sphere/)上[神奇的 LED 地球仪](http://hackedgadgets.com/2010/05/18/diy-spinning-led-globe-build-log/)。显然有人在评论中链接到了这个项目。这太棒了，这是评论应该有的。即使你只是链接到一个你认为更好的项目，也请保持我们的评论礼貌和有帮助。
