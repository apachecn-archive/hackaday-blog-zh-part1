# 修理坏掉的笔记本电脑电源插座

> 原文：<https://hackaday.com/2011/11/23/fixing-that-broken-laptop-power-jack/>

似乎有一个东芝卫星笔记本电脑的电源插孔设计很容易损坏的整个系列。我们从中看到一些好的和一些坏的。插孔没有安装在电路板上，所以如果它像上面一样卡在身体里，它不会影响电子设备。但这里发生的是负责保持插孔位置的外壳内的塑料支架失效了。除非你想出解决问题的方法，否则你无法插上电源适配器。

我们打赌这次修复最困难的部分是打开箱子。一旦进入，只需切掉所有损坏的支撑片，为替换的千斤顶腾出空间。这里使用的一个有一个螺纹袖口，使它可以很容易地将新部件安装到外壳上。夹掉旧的插孔，焊接电线(注意极性！)然后你就入行了。

有人知道为什么我们没有看到苹果笔记本电脑有更多的磁性连接器(MagSafe)吗？是专利问题吗？
