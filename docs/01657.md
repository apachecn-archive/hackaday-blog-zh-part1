# 便宜的 DIY 木吉他拾音器

> 原文：<https://hackaday.com/2008/02/22/cheap-diy-acoustic-guitar-pickup/>

(我只找到一本关于水听器的旧的、绝版的书，所以如果你有好的参考资料，[发送到](http://hackaday.com/tips)。)
