# LED 壁灯为宿舍聚会增添趣味

> 原文:[https://hack aday . com/2011/11/28/led-sconces-spice-up-dorm-room-parties/](https://hackaday.com/2011/11/28/led-sconces-spice-up-dorm-room-parties/)

![](../Images/879a975849a432f94e36e9c87458ab99.png "led-sconces-spice-up-the-dorm-parties")

这间宿舍已经准备好招待客人了，这在一定程度上要归功于[约瑟夫]组装的 LED 壁灯。每个灯具内都有三个 3 瓦的 LED 模块。为了更好的散热，他把它们安装在他切割出来的金属片上，包括一些手指以增加表面积。散热器的形状被选择为安装在 sconce 的扩散器后面，sconce 是一种移除了插座的白炽灯灯具。

[Joseph]为基站和 LED 模块设计了自己的控制板。它们通过 RS485 相互通信，这使他可以通过 CAT-5 电缆相互通信，但这些灯也需要外部电源。控制器本身是一个 USB 加密狗，它从计算机获取串行命令，并通过 RS485 协议将它们发送出去。在休息后的视频中，您可以很好地了解硬件和系统的整体性能。
