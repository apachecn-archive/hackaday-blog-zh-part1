# 蚀刻你自己的 PCB

> 原文:[https://hackaday.com/2011/11/25/etching-your-own-pcbs/](https://hackaday.com/2011/11/25/etching-your-own-pcbs/)

![](../Images/85bb9fada3cd99efe055695abf78c459.png "circuit-boards-at-home")

当[Adr1an]来信分享他的 PCB 蚀刻教程的链接时，他提到他知道我们已经讨论了一大堆指南。他说的完全正确，我们不仅精选了大量的他们，而且[我们不久前也写了我们自己的](http://hackaday.com/2008/07/28/how-to-etch-a-single-sided-pcb/)。但这并不意味着当他们收到小费时，我们会忽略他们。事实上，我们阅读了所有能提供一些东西的文章，并且很高兴地突出了那些呈现得很好的文章…就像这一篇！

[Adr1an]全力以赴写他的文章。他不仅涵盖了这方面的所有要素，还讨论了在哪里购买它们以及他对如何做出选择的想法。他使用的是墨粉转移法，更喜欢 Brother 品牌的墨粉，因为它的覆盖范围和抗过度蚀刻性。他在惠普 Everday 相纸上打印，然后使用层压机转移到覆铜板上。对于本指南，他使用 2 盎司铜，但更喜欢 1 盎司铜，因为它蚀刻速度更快。他选择的腐蚀剂是氯化铁，可以作为干粉订购。他使用直接蚀刻法，将蚀刻剂装入海绵中，然后涂在板上。

他在指南里做的板子看起来很棒，而且只花了他 28 分钟！
