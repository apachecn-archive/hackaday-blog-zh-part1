# Zippo 打火机的防盗报警器

> 原文:[https://hack aday . com/2010/08/26/zippo 打火机防盗报警器/](https://hackaday.com/2010/08/26/burglar-alarm-in-a-zippo-lighter/)

![surprisingly awesome](../Images/9922a69fffb53caae9caf56408f0b480.png "Pocket-Burglar-Alarm")

这个装在 zippo 里的[防盗报警器真的做得很棒。他将一个 picaxe 微控制器、一些红外发光二极管、一个红外传感器、一块电池和各种开关很好地塞进了里面。它几乎完美地关闭，我们认为如果它真的打扰他那么多，他可以补救。](http://www.instructables.com/id/Pocket-Burglar-Alarm/)

它有几种模式，都是由不同顺序的按钮启动的。有一种接近报警器，当有东西移动得非常近时就会发出声音。当你把某物从它的附近移开时，反向接近警报响起。门铃模式和无声警报模式。休息之后，请观看视频，了解实际情况。
