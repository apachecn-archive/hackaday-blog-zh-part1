# 激光灯光秀的特点是通过自制的检流计完全控制 XY

> 原文：<https://hackaday.com/2011/03/16/laser-light-show-features-full-xy-control-via-homemade-galvanometers/>

NothingLabs 的[Rich]组织了一场非常酷的激光表演,你一定要看看，以赢得 Instructables 的激光切割机。

他的漫游讨论了激光表演的机制——特别是检流计如何被典型地用来精确地瞄准镜子，以便绘制图像和书写文本。商用检流计往往相当昂贵，所以他选择自己制造，使用相对便宜且容易找到的零件。

检流计是用一对旧的低音扬声器、几块乐高积木和一些丙烯酸方形镜子制成的。镜子安装在扬声器上，然后连接到 Arduino。他从一个便宜的红色激光指示器上取下电池，并将其永久连接到 Arduino 上，该指示器现在用作脉冲电源。一旦他做好了一切，他就用一台雾机来定位激光。

正如你在下面的视频中看到的，激光表演令人印象深刻。他自制的检流计为最终的投影图像提供了某种粗糙的质量，我们非常喜欢这一点。看起来好像所有的文字和图像都是手绘的，这是一个非常酷的效果。

正如[Rich]提到的，我们希望在未来看到一些基于他的工作的很酷的黑客。
