# 将铣床改造成数控机床

> 原文：<https://hackaday.com/2012/01/24/converting-a-mill-to-cnc/>

在过去一年的大部分时间里，[Joel]一直致力于将一台手动磨机改造成一台数控磨机，并增加了电脑、支架和步进电机。他为他的项目付出了惊人的努力，结果非常棒，而且比购买和运输一个旧的布里奇波特工厂要便宜得多。

该项目始于灰熊的这家工厂。这比小型的“爱好磨坊”高了一步，但仍然非常实惠，1200 美元就能送到[乔尔]的车道上。这项工作始于[从一个配电盘盒中为电脑和电机驱动器制造一个外壳](http://jmillerid.com/wordpress/2011/06/cnc-mill-phase-1-planning/)。控制器盒包括触摸屏、键盘和运行 [Mach3](http://www.machsupport.com/) 数控软件的计算机。计算机连接到一个[分线板](http://www.kelinginc.net/c10.pdf)，三个[电机驱动器](http://www.kelinginc.net/KLDriverDigital.html)为每个轴上的步进电机提供动力。

几个月后(好事需要时间)，[Joel]准备好将步进电机安装到磨机的轴上。他刚刚上传了几个为 PCB 加工覆铜板和 ABS 表面加工的视频，广告之后可以观看。总投资少于寻找、购买和修复一个旧工厂，我们称[Joel]的项目是成功的。
