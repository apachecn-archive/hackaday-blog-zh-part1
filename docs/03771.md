# 粘性光

> 原文：<https://hackaday.com/2009/08/04/sticky-light/>

![light](img/6808391c5cddba52ba871813cff636c2.png "light")

随着网络摄像头和投影仪的出现，[多点触摸和互动演示](http://hackaday.com/2009/06/26/idisplay-webcam-multitouch/)已经成为[越来越受欢迎的](http://hackaday.com/2009/05/08/touchkit-spark-released/)，因为它们非常简单。东京大学的学生采用了一种新方法，即[使用激光](http://www.k2.t.u-tokyo.ac.jp/perception/SmartLaserTracking/)代替。他们创造了使用镜子、激光和单个光电探测器的粘性光。与摄像机跟踪系统不同，该系统不需要视觉处理。激光四处移动，碰到深色物体，粘在上面。它可以跟随桌子上的图画或空间中的物体，例如衬衫设计。他们还创造了几个基本的游戏和一个基于斑点移动发出声音的演示。休息后的项目视频。
