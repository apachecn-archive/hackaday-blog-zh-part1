# 吃豆人女士 IPod 充电盖

> 原文：<https://hackaday.com/2009/08/28/ms-pac-man-ipod-charging-cover/>

![FSPKD86FYTCGWZL.MEDIUM](img/234829de48ab8e65658e8d7e7e543e72.png "FSPKD86FYTCGWZL.MEDIUM")

iPod 充电器绝对是任何人桌子上的一个独特的实体。允许你把你的 iPod 直立起来，充电，也许还可以同时播放视频，但是它们并不漂亮。[Russ 313]想出了一个绝妙的主意，将他的 iPod(和充电器)伪装成一个迷你的吃豆人街机柜。不像我们见过的其他一些[小柜子](http://hackaday.com/2009/08/09/neo-geo-mini-arcade/ "playable mini cabinets")，它只是个幌子，并不能真正发挥作用。但是这个想法仍然是原创的，而且过程如此简单，我们很惊讶以前没有人做过；只需要硬纸板、胶水和其他一些小物件，你就有了一件艺术品。剩下的就是想办法让它适应 iPhone。休息之后，看看他的 iPod 播放的游戏视频，看起来更真实。
