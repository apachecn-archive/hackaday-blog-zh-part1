# USB 弹弓控制器是鸟

> 原文：<https://hackaday.com/2012/02/28/usb-slingshot-controller-is-for-the-birds/>

这个 USB 弹弓控制器真的给我们带来了笑容。一部分是令人愉快的愚蠢的宣传视频，你会发现休息后。[西蒙·福特]将自然和技术结合起来，创造了这个 USB 弹弓。

画框本身来自他在伦敦埃平森林找到的一根树枝。他削掉树皮，并在“Y”的底部挖空一个开口，以容纳加速板。它有一对母引脚接头，与上图中的 mbed 接口。但真正的问题是他编写的代码，将加速度计数据转换成适当的鼠标运动。他在这一领域的成功让他将《愤怒的小鸟》的虚拟世界转化为用弹弓杀人的本能体验。

我们对这类项目很感兴趣。想到的两个例子是摇滚乐队 2 的这些[乐器黑客。](http://hackaday.com/2009/11/13/instruments-as-games-games-as-instruments/)
