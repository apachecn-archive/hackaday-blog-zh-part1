# DS 上的 NES 控制器

> 原文：<https://hackaday.com/2009/02/22/nes-controller-on-a-ds/>

![nesds](img/28e65ca2e94bc21ea442ad3df4a9416a.png "nesds")

你可以在 DS 上买到大部分的老 NES 游戏，但是感觉不一样了。当然，你的手还是会抽筋，但这已经不是我们记忆中的抽筋了。解决方法是什么？[在你的 DS 上放一个旧的任天堂娱乐系统控制器](http://parkerbowab.com/nes-controller-for-ds-lite/)。[帕克]正是这样做的。他使用的方法相当简单。他打开 DS，用电线把他想用的按钮连接到控制器上。然而，他没有将它永久化，而是选择通过 GBA 插槽进行布线。这使得他可以在任何想使用控制器的时候插上电源。这让我们想起了我们在十月份报道过的带有 SNES 控制器的 [PSP。](http://hackaday.com/2008/10/29/psp-snes-controller-mod/)

休息之后你可以看到一段视频。你会在视频中注意到，他还修改了他的按钮，在它们后面有发光二极管。

在评论中，[tri-edge]指出你可以在 acidmods 论坛中[关注他的](http://www.acidmods.com/forum/index.php?topic=22502.0)版本。似乎还有一些其他的变化，包括一个 SNES 控制器。
