# IPhone 倾斜运动控制器

> 原文：<https://hackaday.com/2011/01/07/iphone-tilt-motion-controller/>

这里有一个安装系统，为 iPhone 增加了[机械倾斜控制(链接失效，试试](http://mechanicallyinclined.net/blog/?cat=47)[互联网档案版本](https://web.archive.org/web/20110906061519/http://mechanicallyinclined.net/blog/?cat=47))。它使用两个伺服电机沿 X 轴和 Y 轴旋转。模拟操纵杆与 Arduino 一起使用来控制设备的移动。正如你在休息后的视频中看到的，这在玩基于加速度计的游戏时效果相当好。但是增加一个操纵杆并不是这个项目的最终目标。[Shane]计划将摄像头对准 iPhone，使用图像识别来自动玩游戏。这听起来有点难，但是[我们已经看过吉他英雄](http://hackaday.com/2009/10/30/cheating-rockband-again/)的作品，所以我们很乐观。
