# 噪声轴微型同步

> 原文：<https://hackaday.com/2009/01/09/noiseaxe-minisynth/>

![3172157928_b226c6b2e0_o](img/bec42808896f157e3fedd04cf0f90d24.png "3172157928_b226c6b2e0_o")

[NoiseAXE](http://www.gadgetgangster.com/find-a-project/35-project-by-designer/56-Project-Detail.html?projectnum=79) 是一款基于 Picaxe 08M 微控制器的微型合成器。它的工作原理相当简单:一个导电的唱针接触八个电阻之一的腿来演奏八个音符中的一个，而光敏电阻控制调制的量，产生可变的颤音效果。虽然合成器的输出相当有限(NoiseAXE 不完全是雅马哈 DX7)，但它仍然是一个非常酷的小项目；你可以用它独特的声音为你的下一首电音作品添加那种坚韧不拔的模拟触感。看看下面的视频剪辑，看看和听听它的行动。
