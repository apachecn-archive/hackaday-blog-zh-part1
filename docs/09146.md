# 厨房黑客:自制熏肉机

> 原文：<https://hackaday.com/2011/11/23/kitchen-hacks-home-made-meat-smoker/>

[Matt]注意到在过去的一周里有太多的 sous vide builds，所以他决定用他的自制熏肉机参加比赛。没有什么比一片美味的熏肉更美味的了，事实上，它与我们见过的 sous vide hacks 非常相似，这是一个额外的奖励。

【马特】决定造一个像[大绿蛋](http://www.biggreenegg.com/eggs/)的陶瓷烟枪。他受[【阿尔顿·布朗】](http://www.youtube.com/watch?v=_Ka2kpzTAL8)的启发，用两个陶土花盆和一个热盘子招待吸烟者。为了控制热板，使用了我们在一些视频制作中看到的便宜的 35 美元的 PID 控制器。PID 无法控制热板所需的 7 安培交流电，所以[Matt]使用了他身边的一个固态继电器。

将不锈钢搅拌碗放在加热板上用于木屑。到目前为止，[Matt]已经连续运行他的吸烟器超过 12 个小时，结果真的很有希望——在早晨的寒冷和下午的炎热之间，温度没有太大的变化。[Matt]的体型非常棒，非常适合鹿肉干，因为猎鹿季节即将到来。
