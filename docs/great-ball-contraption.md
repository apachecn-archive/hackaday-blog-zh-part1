# 伟大的球装置

> 原文：<https://hackaday.com/2010/06/15/great-ball-contraption/>

[https://www.youtube.com/embed/B1A7VpQ5paU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/B1A7VpQ5paU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

伟大的球形装置的想法是从许多建造者那里获得模块，并把它们组合成一个大机器。这些模块需要找到某种方式将乐高足球和篮球从一个输入点移动到一个出口，然后将它们传递到下一个模块。他们中的一些人对球进行分类，但最后，上面八分半钟的视频显示了球一圈又一圈地旋转。这对我们来说很好，众所周知，我们喜欢过于复杂的[机器](http://hackaday.com/2010/02/20/rube-goldberg-ftw/)，可能是完全无用的。
