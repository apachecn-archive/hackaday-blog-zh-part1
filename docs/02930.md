# SwiitBoard:高冲击 Wii 平衡板

> 原文：<https://hackaday.com/2008/12/09/swiitboard-high-impact-wii-balance-board/>

![swiitboard](img/25fd404113df9ef553e9252247e60eba.png "swiitboard")

[Frits]给我们发来了 [SwiitBoard](http://www.rincker.nl/Swiitboard.html) ，一个临时版本的 [Wii 平衡板](http://en.wikipedia.org/wiki/Wii_Balance_Board)。他希望能够做一些比他在 Wii 平衡板上能做的更有冲击力的事情，并且也需要更多的空间来做这件事。他用两种不同的泡沫和一块胶合板组装了这个 SwiitBoard。我们还不完全清楚他是如何控制方向的。他声明是从 gforce.x 派生的，有人能澄清一下吗？休息之后请继续关注他的演示软件视频。
