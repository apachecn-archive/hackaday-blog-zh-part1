# 木制的 CD 换碟机

> 原文:[https://hackaday.com/2011/10/18/a-cd-changer-made-of-wood/](https://hackaday.com/2011/10/18/a-cd-changer-made-of-wood/)

[![](../Images/d2ee62ab5db7f931021faecebfff7b62.png "changer_overview")T2】](http://hackaday.com/2011/10/18/a-cd-changer-made-of-wood/changer_overview-2/)

虽然这绝不是一个新的黑客——它大约是在 2002 年制造的——[这个木制 CD 换碟机](http://www.sentex.net/~mwandel/tech/changer.html)是一个有趣的机器。整个事情是一个简单的取放设备。这个夹子非常简单，只用了一根橡皮筋、木头和一个螺线管。它通过中间的孔抓住 CD，将其拾起，然后组件移动到 CD 托盘或堆叠。一切都与 DC 电机和字符串，几个微型开关，以确保一切都是拿起和放下在正确的位置。

虽然我们可能认为这是一个非常酷的设备，但[【马提亚斯】](http://www.sentex.net/~mwandel/tech/changer.html)对他的机器的能力非常谦虚。据他估计，虽然一个人可以相对容易地复制 12 张 CD，但如果你真的想复制大量的 CD，最好还是买一张。也许他在制造业工作。
