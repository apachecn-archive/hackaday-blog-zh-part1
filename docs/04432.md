# 啊，那是一个大轮子

> 原文：<https://hackaday.com/2009/11/18/argh-thar-be-a-big-wheel/>

如果你被困在一个荒岛上，你会希望有一个能用椰子制造有用物品的教授。[LostMachine]就是其中之一，他目前正在建造一艘热爱陆地的海盗船。这辆古怪的交通工具将使用上面的巨型轮子来推动船只，而船长则舒适地坐在高高的乌鸦窝里。可卡因概念？不完全是，他计划把它带到火人节，在那里它将是一个相对于[来说相当有用的版本，那些家伙真的已经偏离了底线](http://hackaday.com/2009/08/22/the-serpent-mother/)。

这里的故事是构建质量。休息之后，请花些时间观看我们嵌入的他的视频。在第一篇文章中，他详细介绍了他在不平的车道上创造精确水平建筑表面的方法。这是通过在与中心点水平的圆上焊接支架来实现的。他继续分享他的液体冷却系统，用一个定制的夹具和一个 RV 上的旧挡风玻璃洗涤器水系统泵来切割管道支架(第二个视频)。最终的视频显示了车轮的结构，其中有 2000 个焊缝和大约 250-300 小时的施工时间。

如果你没猜到的话，[LostMachine]是一名结构工程师。不幸的是，今年春天他被解雇了，这使他的建筑计划受挫。我们希望在他的投资组合中有这样一个高质量的项目，他的新工作指日可待。
