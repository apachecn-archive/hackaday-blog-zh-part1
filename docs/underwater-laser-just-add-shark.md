# 水下激光，加鲨鱼就行了

> 原文：<https://hackaday.com/2009/10/12/underwater-laser-just-add-shark/>

![underwater-laser-beam](img/6126418083d619b2f6f00bceaca5d374.png "underwater-laser-beam")

组装一台蓝光激光器，它是防水的，威力相当大。在 182 兆瓦时，这里有足够的电力来戳破深色气球(见休息后的视频)。为了做到这一点，他找到了一个 LED 手电筒，有橡胶 o 型圈防水。虽然很小，但外壳中有足够的空间来容纳电池、驱动器和激光二极管。在激光二极管周围放置一个厚铝环作为散热器。

我们承认，我们对激光了解不多。这可能是个不该问的问题，但现在他有了这个，你会怎么处理它？给我们留下你对这个恶魔创造的计划的评论。
