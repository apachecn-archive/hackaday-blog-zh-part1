# 冒险复活一个无名的 IP 网络摄像头

> 原文:[https://hacka day . com/2011/11/23/adventures-in-reasing-a-no-name-IP-web cam/](https://hackaday.com/2011/11/23/adventures-in-resurrecting-a-no-name-ip-webcam/)

![webcam-repair](../Images/3d54bb2f125949de6bda794e3b706937.png "webcam-repair")

和我们许多人一样，[牛排]是他家人的一种随叫随到的服务台。他的岳父最近联系了他，因为他的云台摄像头坏了，他想看看是否能修好。从来没有拒绝过挑战，[牛排]决定给它一个机会。

他最后不得不拆开它，因为相机完全没有反应，他在里面发现的东西激起了他的兴趣。这款无名相机的核心是 ARM 微处理器，似乎它的一些引脚因设计不当而损坏。他认为重新焊接图钉可以达到目的，但这并不是他冒险的结束。

随着他对该设备的深入研究，他发现由于处理器上损坏的引脚，相机实际上是在自我毁灭，将数据读写到错误的位置。经过大量的搜索，他找到了一个有点兼容的固件映像，尽管并不是所有的东西都能正常工作。

他的岳父对他的作品印象深刻，以至于他要求把相机拿回来，尽管[斯牛排]还没有完全修好。虽然他告别了相机，但我们非常肯定，如果我们的读者有一些额外的见解或资源可以帮助他完成这项工作，他会非常乐意收回相机几天。
