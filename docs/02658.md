# LED POV 硬盘时钟

> 原文:[https://hackaday.com/2008/10/17/led-pov-hard-drive-clock/](https://hackaday.com/2008/10/17/led-pov-hard-drive-clock/)

![](../Images/fa59c60746be56125f598540f27668bc.png "pov_hdd")

[伊恩·史密斯]给我们发来了这个用旧硬盘制作的[POV 显示器。它的工作原理是让盘子下面的一些发光二极管闪烁，盘子上有一条线。由于盘片以大约每秒 90 转的速度旋转，我们看到的是立体图像。他受到[艾伦·帕瑞克]的](http://www.ian.org/HD-Clock/)[项目](http://alan-parekh.com/projects/hard-drive-clock/)的启发，但做了几件不同的事情。在这个项目中，安装 led 要简单得多，因为他有一条 RGB LEDs，背面有胶带。他还用一个小磁铁和一个霍尔效应传感器代替红外线传感器来测量整圈旋转。休息后查看视频。

[https://www.youtube.com/embed/K1asNB0te0o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/K1asNB0te0o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

正如你所看到的，声音很大。他说他想用有机玻璃把它围起来以降低噪音。你可以从他的网站上得到原理图和许多很好的说明。他还推荐了一些其他项目，例如[另一款带大触摸屏的 POV LED 硬盘时钟](http://instruct1.cit.cornell.edu/courses/ee476/FinalProjects/s2006/ja94/Amsel%20-%20Klitinek%20Final%20Project/index.htm)。
