# 新年前夜在你的厨房举行舞会

> 原文:[https://hack aday . com/2011/01/18/新年前夜球丢进你的厨房/](https://hackaday.com/2011/01/18/new-years-eve-ball-drop-in-your-kitchen/)

![](../Images/f480be4e1c278aea201b812875621856.png "new-years-eve-ball-drop")

[Tech B.]用他到处乱放的东西拼凑了一个新年前夜的落球游戏。这个球本身是一个古老的圣诞装饰品，他将其切成两半，并填充了 14 个 led 和 9V 电池。他把两半部分粘在一起，并为导杆加了一个孔，从而完成了工程的这一部分。底座由一些纸板盒制成，隐藏了一个 Arduino、一个伺服电机、一个 LCD 屏幕和垂直杆的底座。当一年的最后十秒倒计时时，伺服系统通过解开绕在杆顶上的纱线来降低球的高度。随着纱线被慢慢抛出，重力将球拉向目标。休息之后，我们嵌入了[技术 b]的演示视频。
