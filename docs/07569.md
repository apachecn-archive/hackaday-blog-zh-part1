# 双管空气炮

> 原文:[https://hackaday.com/2011/05/09/double-barreled-air-cannon/](https://hackaday.com/2011/05/09/double-barreled-air-cannon/)

![](../Images/6c3a7f51c866eb51aac29438ba3f25a5.png "double_shot_on_ground-600x178")

[杰里米]写信来炫耀他的最新创作。他造了一门[双管气动空气炮](http://www.jcopro.net/2011/05/08/the-double-barrel-pneumatic-air-cannon/)。避开传统的单管和压力室的方式，杰里米想要一个双管的版本。因为他是气动操作，所以他必须找到一种方法来保持每个桶中的压力彼此独立，并独立触发它们。虽然我们都同意单向阀和喷水阀不是开创性的，但很高兴看到它们都被设计出来并经过测试。如果我们如此倾向于创造我们自己的版本，我们现在有蓝图。你可以看到他在测试，把房间装满水，休息后把水溅得到处都是。
