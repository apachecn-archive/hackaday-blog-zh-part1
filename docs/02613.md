# 将 PSP 用作扩展显示器

> 原文：<https://hackaday.com/2008/10/06/use-a-psp-as-an-extended-display/>

继续这个项目，你可以[用你的 PSP 作为你电脑的显示器](http://jjs.at/software/pspdisp.html)。该软件可在网站上下载，显示分辨率为 960×544。数据被压缩，发送到 [PSP](http://www.mahalo.com/PSP_Hacks "PSP Hacks - Mahalo") ，然后解压缩显示。虽然这听起来很笨拙，但休息后的视频显示，通过一些 [Winamp](http://www.mahalo.com/Winamp "Winamp - Mahalo") 可视化，它工作得相当顺利。目前仅支持 32 位或更早版本的 Windows XP。
