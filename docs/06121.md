# 窗帘移动以遮挡行人

> 原文:[https://hack aday . com/2010/10/15/window-curtain-moves-to-screen-行人/](https://hackaday.com/2010/10/15/window-curtain-moves-to-screen-pedestrians/)

![](../Images/eec7cf473c60e25f93902cbe9d82f93b.png "curtain-screens-pedestrians")

尼克拉斯·罗伊(Niklas Roy)建造了一个电动窗帘来遮挡他窗前的行人。当你听到“电动窗帘”时，你可能会认为它会上下移动，但事实并非如此。取而代之的是，当行人在人行道上行走时，小窗帘会水平移动来遮盖他们。这是通过使用摄像头和一些运动感应软件来实现的。在休息后的视频中，你可以看到软件也预测了运动，并最终很好地保持了目标的覆盖。这要归功于硬件设置上的加工草图与旋转编码器的配合。以上链接的页面提供了两者的详细信息。

这听起来像是我们见过的其他社区参与黑客，如[地铁楼梯钢琴](http://hackaday.com/2009/10/07/the-subway-never-sounded-so-good/)，或[瓶子回收视频游戏](http://hackaday.com/2009/10/17/the-glass-recycling-game/)。这很有趣，也很古怪，这并不令人惊讶，因为当我们看[尼克拉斯的]公共喷泉[水力发电机](http://hackaday.com/2010/10/11/bicycle-hub-hydropower/)时，我们看到了这一点。
