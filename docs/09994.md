# 胶棒+伺服=线性致动器

> 原文:[https://hack aday . com/2012/03/29/glue-stick-servo-linear-actuator/](https://hackaday.com/2012/03/29/glue-stick-servo-linear-actuator/)

当你把一个胶棒和一个爱好伺服电机交叉时，你会得到什么？[当然是线性执行器](http://letsmakerobots.com/node/31838 "linear actuator glue stick")！虽然这可以用其他家用工具来完成，但这种胶棒的形状似乎非常适合放在伺服喇叭上。

正如你可能已经猜到的，伺服系统必须完全旋转，而不是典型的 180 度左右。这个技巧，真正让我们眼前一亮的是，在正常的连续旋转模式中，不是连接两个电阻，而是在胶棒上使用电位计，允许位置反馈。

这种齿轮减速致动器产生的力令人印象深刻，在用于测试的天平上给出了“err”(超过 3 千克)。[Gareth]或[Chiprobot]在休息后给出了如何制作其中一个的很好的教程，但是如果你想看它的实际操作，请跳到 8:20 左右！
