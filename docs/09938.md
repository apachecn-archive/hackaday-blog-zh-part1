# 人形机器人将最终接管世界

> 原文:[https://hack aday . com/2012/03/20/humanoid-robot-will-over-the-world/](https://hackaday.com/2012/03/20/humanoid-robot-will-eventually-take-over-the-world/)

![](../Images/d9ff06e0bd040355b53efe6f731d9c3d.png "robot")

在过去的两年里，哈佛(和巴尔的摩)黑客空间的[Mark]一直在建造 [401k，一个人形机器人](http://baltimorehackerspace.com/2012/03/meet-401k/)，它很快就可以用两条腿走路，探测物体，并在导致人类灭绝的机器人起义中与它的同志们一起战斗。

为了了解人形机器人有多复杂，要知道本田阿西莫 T1 已经是一个持续了十多年的项目，很容易被楼梯 T3 击败 T2。[马克]没有数百万美元的资金或几十名实验室助理的好处——在[的一个视频](http://www.youtube.com/watch?v=8Ba4qhyFxqA&context=C4a32c04ADvjVQa1PpcFPWx6LERKe804Q74nP3Zf1__uEO0Id6NtQ=)中，[马克]向我们展示了用[乔治·福尔曼]烤架盖制成的脚垫和用 DVD 播放机制成的髋关节。尽管他使用了“非常规”的部件，401k 仍然有一双非常先进的腿，可以很好地模拟他们的人类模拟。

尽管它仍然是一项正在进行中的工作，但是这个版本已经投入了大量的工作和专业知识。[Mark]想带着 401k 参加下个月举行的今年的 robo games 比赛。我们希望他能及时恢复健康，哪怕只是一小步。

你可以在[【Mark】的 YouTube 频道](http://www.youtube.com/user/moomooleo?feature=watch)上查看更多 401k build vlog。
