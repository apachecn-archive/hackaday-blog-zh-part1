# 每当你的朋友伸手拿零食时，你都可以毫不费力地抓到他们

> 原文:[https://hack aday . com/2011/10/04/毫不费力地-troll-your-friends-each-times-that-reach-to-a-drugs/](https://hackaday.com/2011/10/04/effortlessly-troll-your-friends-each-time-they-reach-for-a-snack/)

![trolling_your_friends_abusive_refrigerator](../Images/c8cddaf7731253fde41751f1d2092ab6.png "trolling_your_friends_abusive_refrigerator")

如果你想减肥，[Grissini] [有你需要的东西！](http://www.instructables.com/id/The-Abusive-Refrigerator/?ALLSTEPS)

他最近改装了他的冰箱，每次打开冰箱就扔出侮辱的话，尽管不是为了他自己的身体健康。虽然我们想象一个滥用的冰箱会有助于抑制你对夜宵的胃口，但[Grissini]毫不掩饰这样一个事实，他只是想毫不费力地自动骚扰他的室友。

该设备非常简单，由一个 Arduino 和一个 Adafruit 波盾组成，塞在一个聚苯乙烯泡沫咖啡杯内。一个光电池用于检测冰箱门何时被打开，触发 Arduino 播放板载 SD 卡上的一段声音。[Grissini]甚至花了大量时间使用文本到语音引擎，以便创建一个定制的侮辱列表，指出他朋友的特质——多好的家伙！

继续阅读，看看他的冰箱是如何使用的，一定要看看他的说明书，学习如何制作自己的冰箱。
