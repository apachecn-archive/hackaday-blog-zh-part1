# 免税极客团体遭遇巨额税单

> 原文：<https://hackaday.com/2010/03/16/tax-exempt-geek-group-hit-with-huge-tax-bill/>

![](img/1603fa2bb9f540ba6d1bd12bfdcc4bd2.png "MI-vs-the-geek-group")

非营利黑客空间[尽管极客群体享有免税地位，但他们却遭到了高额税单的打击。我们展示了该组织早在 11 月建造的](http://www.youtube.com/watch?v=aGOBR5Z4fSM)[吊杆相机](http://hackaday.com/2009/11/21/build-a-camera-boom-at-less-than-a-grand/)。极客小组的目标是通过提供设施、同伴团体和友谊来满足探索和创造的渴望，使知识和学习不仅是可接受的，而且是可取的。在休息后的视频中，您可以听到一些关于该组织在维修捐赠的计算机并将其投放到社区中的作用，以及通过美国童子军等团体在教育中的作用。

这一切都是在没有利润目标的情况下完成的，因此他们在联邦政府获得了 [501(c)(3)地位](http://en.wikipedia.org/wiki/501c3#501.28c.29.283.29)(我们已经看到[他们的 990 表格](http://www2.guidestar.org/ReportNonProfit.aspx?ein=38-3600686&name=geek-group#)陈述了这一点)。据我们所知，这并不意味着他们不需要支付财产税，但它确实使财产税低得离谱(我们以前听说过非营利土地持有每英亩 1 美分)。这就是为什么当镇上的人在门上贴上通知扣押并要求支付 47，652.78 美元的补税或资产将被拍卖时，会感到非常惊讶。从极客群体的角度来看，整个故事从上周六的季度董事会会议的[视频中展开。](http://www.youtube.com/watch?v=ZPnxjCxBbKY)

我们希望这只是一个错误，可以补救。也就是说，运行这种类型的操作并不容易。不幸的是，董事会除了完成组织的使命之外，还需要处理一场税务战。祝他们在前进的道路上一帆风顺。
