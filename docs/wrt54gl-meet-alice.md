# WRT54GL，认识一下爱丽丝

> 原文：<https://hackaday.com/2010/02/27/wrt54gl-meet-alice/>

说到路由器，有一个是黑客的最爱，WRT54GL。但是一个稍微不太出名的公司，Pirelli 和他们的“Alice Gate2 plus Wi-Fi ”,似乎是我们的意大利朋友们的热门选择。

从安装[串口和并口](http://translate.googleusercontent.com/translate_c?hl=en&ie=UTF-8&sl=auto&tl=en&u=http://sanniolug.org/blogs/esteban-sannin/ultimi-hack-e-aggiornamenti-sull039-quotalice-gate-2-plus-wifiquot-alias-aga&prev=_t&rurl=translate.google.com&twu=1&usg=ALkJrhhee5xXFu83pcByZ3cH1tMaC73VsA)，到安装 [Debian](http://translate.googleusercontent.com/translate_c?hl=en&ie=UTF-8&sl=auto&tl=en&u=http://sanniolug.org/wiki/installare-debianmips-sul-router-quotalice-gate-2-plus-wifiquot&prev=_t&rurl=translate.google.com&twu=1&usg=ALkJrhgKS2Atx4ujtaf6xPq4MuJlVmz7eA) 时[解锁固件](http://translate.googleusercontent.com/translate_c?hl=en&ie=UTF-8&sl=auto&tl=en&u=http://sanniolug.org/wiki/sblocco-modem-router-quotalice-gate-2-plus-wifiquot&prev=_t&rurl=translate.google.com&twu=1&usg=ALkJrhgX9dBH5-2Ox_XHT8ptNN-Y4CfqoA)，Esteban 什么都做了。我们个人最喜欢的是一个[附加 USB 端口](http://translate.google.com/translate?js=y&prev=_t&hl=en&ie=UTF-8&layout=1&eotf=1&u=http%3A%2F%2Fsanniolug.org%2Fwiki%2Faggiungere-una-seconda-usb-20-al-router-quotalice-gate2-plus-wifiquot%3Ft%3D1267216397%23comment-250&sl=auto&tl=en)的创造性布线，他必须定制一个电源电路来运行他的网络摄像头和外部驱动器。
