# Arduino + Wii 双截棍+伺服

> 原文：<https://hackaday.com/2009/03/30/arduino-wii-nunchuck-servos/>

[https://www.youtube.com/embed/phcsDNOglLI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/phcsDNOglLI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Yezzer]发布了一个他正在做的很酷的小项目的视频。他已经将 Arduino 和 Wii 双截棍连接起来控制一些伺服系统。他还在上面安装了一个标准的 [USB](http://www.mahalo.com/IPod "USB - Mahalo") [网络摄像头](http://www.mahalo.com/Webcam "Webcam - Mahalo")。没有太多的信息，但是他确实包含了一些他为这个项目修改的代码的链接。这个动作看起来很自然，似乎是一种廉价的方式来启动一些好的电子动画控制。这实际上可能是控制机器人参加克拉布挑战赛的一个很好的方法，如果他们有另一个机器人的话。

更新:正如[dokument]在下面的评论中指出的，看起来我们已经看到了一个几乎与过去的一模一样的[设置。](http://hackaday.com/2008/04/06/wii-nunchuck-arduino-pan-and-tilt-camera/)
