# 蒸汽循环感觉你的裤子着火了

> 原文:[https://hackaday . com/2011/01/08/steam-cycle-feeling-like-your ' s pants-is-fired/](https://hackaday.com/2011/01/08/steam-cycle-feels-like-your-pants-are-on-fire/)

![](../Images/a88b91e6bd06113ae4eefbb771facd80.png "steam-powered-bicycle")

通常我们不喜欢展示没有构建细节的项目，而是把它们保存起来，作为[一个链接帖子](http://hackaday.com/category/hackaday-links/)。但是这辆[蒸汽动力自行车](http://www.youtube.com/watch?v=CiJeHHGqSSw)太……奇特了，让人无法错过。骑手两腿之间是装有木头燃料火的[燃烧室](http://en.wikipedia.org/wiki/Firebox_(steam_engine))。休息过后，看看这段视频，你会发现这个装置有多吵。除了中音范围的“突突-突突-a”之外，我们还将持续的哨声归因于压力调节器。令人惊讶的是，整辆自行车没有升温，但它一定不是那么糟糕，因为试飞员没有穿石棉裤。玩笑归玩笑，看起来这只野兽跑起来没有任何问题(基于摄像机的移动)，这要感谢可再生能源。

我们会更乐意看到这种机械化串联的外形，因为我们就是无法克服两腿之间有一团火。
