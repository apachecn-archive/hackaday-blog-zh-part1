# 手持板用 NES 控制器播放音乐

> 原文:[https://hack aday . com/2011/06/28/handy-board-plays-music-with-an-nes-controller/](https://hackaday.com/2011/06/28/handy-board-plays-music-with-an-nes-controller/)

![](../Images/6bcf8d8c9134ab8b5f9e87ca10d7d59d.png "handyboard")

[Alex Busman]一直在使用一种叫做 Handy Board 的老式微控制器板。最近，他想出了如何连接一个 NES 控制器来播放音乐。通过 NES 控制器上的 8 个按钮，[Alex]可以控制整个音阶，所以他在他的视频中通过覆盖[马里奥医生主题](http://www.youtube.com/watch?v=-vjT6DjUN24)来演示这个[。](http://www.youtube.com/watch?v=thWm4hbnw0A#t=173s)

[Handy Board](http://handyboard.com/hb/) 是一种微控制器板，最初于 1995 年为乐高机器人设计。凭借 68HC11 μC 的 2MHz 运行频率和 32KB 的 RAM，Handy Board 已经被乐高 Mindstorms ~~NTX~~ NXT 取代，幸好 Handy Board[仍然受到支持](http://sourceforge.net/projects/hbloader-uml/)，它仍然是学习嵌入式设计的一个很好的平台。

很高兴在相对过时的硬件上看到构建，尤其是考虑到这将是一个使用 Arduino 的琐碎构建。我们认为[Alex]了解“困难”硬件的来龙去脉很棒，这是学习的好方法。休息之后看看[Alex]的构造。
