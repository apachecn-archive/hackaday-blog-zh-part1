# 担心今年的 Haloween？为什么不是巨大的霸王龙服装？

> 原文:[https://hacka day . com/2012/02/23/journey-about-halowen-this-year-why-not-a-giant-t-rex-costume/](https://hackaday.com/2012/02/23/worried-about-haloween-this-year-why-not-a-giant-t-rex-costume/)

[![](../Images/9515c331efc3cd09a78e4547ce21cba4.png "The-Ultimate-T-Rex-Costume")T2】](http://hackaday.com/2012/02/23/worried-about-haloween-this-year-why-not-a-giant-t-rex-costume/the-ultimate-t-rex-costume/)

虽然有些人可能会注意到[杰米]的作品可能会被误认为是迅猛龙甚至异特龙，但他的巨型[霸王龙服装/模型](http://www.instructables.com/id/The-Ultimate-T-Rex-Costume/)却是一项艺术壮举。它有 14 英尺高，10 英尺长。作为比较，你在上图中看到的房间尺寸为 25 x 25 英尺。如果你碰巧住在亚特兰大地区，或者愿意旅行，这套服装预计将在 2012 年的 Dragon*Con 上亮相，所以一定要在那里寻找。

整个东西是由数控切割机切割的泡沫塑料制成的。它还具有金属支撑结构。正如文章中提到的，理论上，你可以用手将所有这些部分切掉。但是，由于有超过 140 个部分，所以需要坚持不懈！

[Jamie]的制造能力在这一点上显然相当先进，但他正试图通过在[Instructables Shopbot 竞赛](http://www.instructables.com/contest/shopbot/)中赢得一个路由器来扩展它们。如果你喜欢他的创作，一定要投他一票！休息过后，看看这套服装的视频。T3T5

如果你看不到自己很快赢得一个商店机器人，为什么不从一个工具包中建立你自己的(较小的) [CNC 路由器？](http://hackaday.com/2011/09/01/building-a-cnc-router-to-call-my-own/)
