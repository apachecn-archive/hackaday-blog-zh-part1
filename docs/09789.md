# 用元件包装材料制造柔性传感器

> 原文:[https://hack aday . com/2012/02/28/building-a-flex-sensor-from-component-packing-materials/](https://hackaday.com/2012/02/28/building-a-flex-sensor-from-component-packing-materials/)

像这样的黑客不会帮助我们从极端的 pack-rat 行为中恢复过来。让我们明白，任何东西都不应该扔掉【Peter】[用元件包装材料](http://dev.ivanov.eu/index.php?page=flex-sensor)制作了一个弯曲传感器。它使用黑色导电泡沫，集成电路有时嵌入其中以便运输。你可以看到上面的灰色矩形是传感器本身。在图像的背景中，除了胶带之外，构件中使用的每个组件都有标签。

该项目从将泡沫切割成合适的尺寸和厚度开始。他用一些铝箔做了同样的事情，然后撕开胶带条作为外壳。来自某些电缆屏蔽层的细导线充当传感器的两根导线。他将每根电线连接到一条翻过来的胶带上，然后是箔片，最后是泡沫。休息之后，当视频中的两部分组装在一起时，[Peter]拿起万用表显示传感器弯曲时电阻的变化。

我们认为这将需要一个聪明的校准算法来可靠地工作，但它不会比我们在这个链接帖子中看到的[光学弯曲传感器更麻烦。](http://hackaday.com/2011/12/11/hackaday-links-december-11-2011/)
