# [掷弹兵]因为他的便携式 x 光项目赢得了激光切割机

> 原文:[https://hack aday . com/2012/01/13/grenadier-wins-the-laser-cutter-for-his-portable-x-ray-project/](https://hackaday.com/2012/01/13/grenadier-wins-the-laser-cutter-for-his-portable-x-ray-project/)

![](../Images/ada81c60661f36e28cc91c553967b7d3.png "xray")

在 Build Lounge 举行的全光谱激光竞赛的结果已经公布。[的 40 瓦豪华激光切割机](http://fslaser.com/40w-deluxe-hobby-laser-engraver-and-cutter)的头奖颁给了【掷弹兵】的[我们月初看到的便携式 x 光机](http://hackaday.com/2012/01/01/see-through-everything-with-a-home-made-x-ray/)。我们认为这是头奖的最佳选择，因为，来吧，这是非常硬核的。

排在第二位的是根据音乐编排的圣诞灯光秀。明尼苏达大学的开放空间主办了这次安装。休息之后，我们嵌入了一段表演视频。

三等奖由 QC 联合实验室的黑客空间获得，他们的光墙。它使用曾经流行的通用电气彩色效果灯，每个灯泡装在一个真空形成的金字塔里，作为一个扩散器。

还有几项荣誉奖。在我们心中有一个特殊的位置给[杰克·巴芬顿的] [太阳钟](http://hackaday.com/2011/12/19/jacks-solar-powered-clock/)，它也包括在这个组里。我们认为使用光纤将太阳光导入一个机械加工的指示环是很巧妙的。你不得不称赞他开发了一个不使用电力和几乎没有移动部件的项目(有一个滑块可以根据夏令时进行调整)。
