[Pedram]发给我们他的 [USB 导弹发射器接口项目](http://dvlabs.tippingpoint.com/blog/2009/02/12/python-interfacing-a-usb-missile-launcher)。他碰巧有一些 USB 导弹发射器。在输掉了最初的抽签后，他想做点什么来增加气氛。他用 python 写了一个界面，这样他就可以通过他的 iPhone 控制启动器。我们看不出这和用电脑控制它们有什么不同，但他似乎在这方面做了相当多的工作。
