# 让 IWallet 起死回生

> 原文:[https://hack aday . com/2011/07/18/起死回生/](https://hackaday.com/2011/07/18/bringing-an-iwallet-back-to-life/)

![iwallet_bluetooth_hack](../Images/a0b658215759d8685199e29d283a205c.png "iwallet_bluetooth_hack")

如果你口袋里有一大笔钱，那么 iWallet 就是一个精巧的小设备。600 美元的价格对[cmw]来说有点难以承受，所以他在易贝买了一个被水损坏的钱包，希望能修好它。一旦仔细看了看，他知道修复它是一个失败的事业，所以他决定改为黑客。

他取出了钱包里除了启动装置的马达之外的大部分电子设备，并用自己的零件替换了损坏的零件。他安装了一个 Arduino pro 和一个蓝牙模块，用一个小型可充电的 LiPo 电池为这对设备供电。iWallet 的指纹识别器随后被一系列显示设备蓝牙连接状态的 led 取代。

[cmw]现在可以把钱包和手机连接起来，通过蓝牙发出解锁命令。如果你不想付现金，他的版本几乎和真的一样好。
