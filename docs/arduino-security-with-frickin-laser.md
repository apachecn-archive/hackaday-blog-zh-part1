# Arduino 安全系统配有该死的激光

> 原文：<https://hackaday.com/2010/01/03/arduino-security-with-frickin-laser/>

![](img/282043dea434015819319d5bd3b4de2c.png "arduino-laser-tripwire")

[over9k]用他的 Arduino 设置了[一个激光绊线](http://www.instructables.com/id/Twittering-Laser-Tripwire-with-Webcam-Capture/)。激光器安装在 Arduino 旁边，从镜子反射，照射到通过分压器连接的光敏电阻上。当激光束中断时，来自分压器的信号的变化被监控。[over9k]进行设置，以便网络摄像头拍摄入侵者的照片，并在推特上发布事件以便于通知。休息后的视频将介绍每一个步骤。

这种构建在边缘处有点粗糙，但与其他激光绊线不同的是，它将所有的电子设备都放在一个地方。激光接口可能会更有说服力，我们想知道它从 Arduino 引脚上抽取了多少电流。但是如果你觉得无聊，手头有这个东西，玩玩会很有趣。T3[https://www.youtube.com/embed/weQQ363RmKQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/weQQ363RmKQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
