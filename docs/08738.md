# 可持续发展的秘诀:自动供给木材燃烧发电机

> 原文:[https://hack aday . com/2011/10/05/sustain ability-hacks-auto-feeding-wood-fired-generator/](https://hackaday.com/2011/10/05/sustainability-hacks-auto-feeding-wood-fired-generator/)

![](../Images/82388d4ad68861f7bfd6f406784ad310.png "automatic-wood-fed-generator")

这是一个很难归类的项目。它通过燃烧木头发电。右边用菱形板包裹的柱子是一个储存木材的仓库，当下面的木材被消耗时，木材通过重力输送。热量被用来驱动负责发电的动力涡轮。

这就引出了一个问题，这是一个可持续发展的黑客吗？从一个角度来看，它在燃烧可再生生物质。现在这是木材，但也可能是压缩的草块或木材制造的副产品。所以从这个意义上说，它是可持续的。不幸的是，它仍然没有解决碳排放的问题。

该项目的构建日志中包含大量的图像和视频。你可以看到最初的原型不是自动供给的，而是燃烧得如此之热，以至于整个组件都发出漂亮的粉红色光芒。但是，当他们得到最终的原型时，它的运行效率更高，可以输出超过 100 安培的峰值电流！
