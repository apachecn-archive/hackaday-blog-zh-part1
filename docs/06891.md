# DIY 挑选和放置似乎很容易建立

> 原文:[https://hackaday . com/2011/02/21/DIY-pick-and-place-is-easy-to-build/](https://hackaday.com/2011/02/21/diy-pick-and-place-seems-easy-to-build/)

![](../Images/874a1806a0f8a59a532a00cde85bd0e4.png "diy-pick-and-place")

我们并不是说这是一个简单的项目，但是[Alan Sawula]用于这个 DIY 拾取和放置的构建方法可能是我们见过的最简单的。由于这只是一台数控机床，他所使用的方法也适用于工厂或其他机器。他没有使用 X 和 Y 轴的精密杆，而是使用方管。管道定向更像钻石，九十度角提供了行进表面。两个轴承之间有一个垫片，提供了一个沿角落骑槽，因为这是方形的，而不只是“L”支架，雪橇固定在管的上方和下方。步进电机提供沿 X 和 Y 的移动，一个伺服电机用于 Z 轴，另一个用于旋转用作真空吸头的医用针。视频开始四分钟后，你会发现这不仅有效，而且快如闪电！
