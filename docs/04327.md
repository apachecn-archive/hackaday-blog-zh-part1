# 这些 Midi 控制器真臭！

> 原文:[https://hack aday . com/2009/10/27/these-midi-controllers-stink/](https://hackaday.com/2009/10/27/these-midi-controllers-stink/)

![BeatShoes-midi-controller](../Images/1278dad925d7260f6a15a2f515df2036.png "BeatShoes-midi-controller")

好吧，也许说它很臭太刺耳了。但是如果你用一双旧滑冰鞋制作一个 midi 控制器，你可以确定它们有气味。[Thobson]将气味问题放在一边，在他的鞋子上添加了四个力敏电阻器(每个鞋跟一个，每个脚掌下一个),创造了他所谓的 BeatSneaks。随着对电阻器施加力，它们的电阻变小。这种电阻变化由 Arduino 上的 ADC 输入测量，并用于通过 USB 触发 midi 事件。休息之后有视频，而且[Thobson 的]已经提供了[示意图](http://www.instructables.com/id/S89BE7DFWKQKOXW/)和[代码](http://www.instructables.com/id/S2UKLMDFWKQKOYR/)，他用它来增加一个不断增长的[不寻常的音乐接口](http://hackaday.com/2009/09/23/arduino-human-synthesizer/)。

这又让踢踏舞变酷了吗？
