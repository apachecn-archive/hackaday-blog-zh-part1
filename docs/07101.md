# 动画纸

> 原文:[https://hackaday.com/2011/03/15/animated-paper/](https://hackaday.com/2011/03/15/animated-paper/)

![](../Images/cf595cf79c2c1ab5352c368629cbad2d.png "Picture 46")

如果你能让纸对物理输入做出反应会怎么样。也许你想让它在有人伸手太快时躲开并靠近，或者你想要一些逼真的纸花？继续这个伟大的第一步是[动画纸](http://fkeel.blogspot.com/2011/02/more-experiments-with-nitinol-paper.html)，它是简单的镍钛诺记忆线通过我们最喜欢的工具胶带粘在纸上。

记忆金属丝首先被弯曲成它想要的形状，为了保持这种形状，它需要被加热到大约 540 摄氏度，这对于丙烷火炬来说是一项容易的任务。一旦它有了记忆形状，金属丝可以弯曲成任何想要的形状，当加热到大约 70 摄氏度时，它会恢复到原来的形状。

用胶带固定在一张纸上，让电池中的电流流过，电线迅速升温，使纸张充满活力，这可能正是一个更艺术的机器人或电子显示器所需要的。休息之后，请加入我们，观看一小段视频。
