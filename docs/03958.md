# 迷你 MAME 橱柜

> 原文：<https://hackaday.com/2009/09/04/mini-mame-cabinet/>

【路易斯】加入了 [MAME 项目](http://hackaday.com/2008/09/16/arcade-cabinet-that-plays-every-game-in-the-universe/)的[系列，我们已经用他的迷你 MAME 橱柜建造进行了报道。这是他的第五个迷你机柜，具有 5 英寸的 PlayStation LCD 屏幕，一个操纵杆，四个控制按钮，播放器 1 和 2 开始按钮，扬声器和一个发光的字幕。控制器通过一个](http://hackaday.com/2008/12/24/multitouch-mame-cabinet/) [I-PAC](http://www.ultimarc.com/ipac1.html) 连接，运行整个系统的是一个迷你 ITX 板，带有一个凌动处理器和两个内存。[建造日志是西班牙语的](http://www.elotrolado.net/hilo_ave-fenix-mini-la-mini-maka-nosolomame-de-lucho_1286290)，带我们从纸板原型到装满高质量艺术品的成品碎料板橱柜。
