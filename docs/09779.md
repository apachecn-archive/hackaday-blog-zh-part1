# 免费 3G 共享 Kindle

> 原文:[https://hack aday . com/2012/02/27/tethering-a-kindle-for-free-3g/](https://hackaday.com/2012/02/27/tethering-a-kindle-for-free-3g/)

[Excelangue]刚刚发布了一个使用亚马逊 Kindle 中的免费 3G 连接在你的电脑上浏览互联网的指南。

黑客需要一个 Kindle 键盘 3G 和免费的全球互联网接入。越狱 Kindle 并应用 USB 网络黑客后，[Excelangue]设法通过他的电脑将他的笔记本电脑连接到互联网。捆绑 Kindle 3G 的过程非常简单，但我们预计一键解决方案将于本周在网上出现。

当然，我们必须注意，共享 Kindle 违反了亚马逊的条款和条件，通过 Kindle 的数据与一个唯一的 ID 相关联。如果你这么做了，亚马逊知道你是谁，并且很有可能[愿意把你的设备做成砖块](http://www.nytimes.com/2009/07/18/technology/companies/18amazon.html)。[Excelangue]正在研究通过 WiFi 与 Kindle 共享，以便 Android 和 iOS 设备可以参与进来，但他仍在试验他的构建。
