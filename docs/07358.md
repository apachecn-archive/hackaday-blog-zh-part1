# 巨型打印机

> 原文:[https://hackaday.com/2011/04/11/giant-scale-printer/](https://hackaday.com/2011/04/11/giant-scale-printer/)

![Giant Printer Print Head](../Images/d501d966bc0f53ade8e848c752a226d2.png "Giant Printer (Custom)")

这台[巨型打印机](http://geekphysical.blogspot.com/2011/02/giant-printer.html)最初是由【Komponent/LAB】在 2006 年建造的，用于为一个节日打印一些大幅横幅，但最近被调出仓库，并为[创业杯比赛](http://en.wikipedia.org/wiki/Venture_Cup)进行了更新。该系统接受了一些机械和软件更新，并被安装在三脚架上，以使其完全便携。

不同于在典型打印机中使用步进电机和编码器直接控制打印头，整个打印轴垂直转动，并且两个带的相对长度是变化的(随着重力的恒定向下拉力),以精确控制平面上的运动。该软件使用 HPGL 绘图仪文件，并能够缩放它们以适应可用的打印区域。

尽管打印头因快速加速而摆动存在一些问题，但任何打印缺陷在几英尺之外似乎都难以察觉。通过调整软件来补偿这种不必要的移动，可以进一步提高精确度。

虽然我们可以想象这种打印机的许多不同应用，如建筑或街头艺术，但如果在户外进行，在非常大的范围内肯定需要一些微调，以补偿风等。
