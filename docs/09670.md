# POV 时钟旋转滤光器而不是 led

> 原文:[https://hack aday . com/2012/02/10/POV-clock-spins-light-filter-inst-of-LEDs/](https://hackaday.com/2012/02/10/pov-clock-spins-light-filter-instead-of-leds/)

![](../Images/0624e39ea280d8a592aa9375f04f872c.png "pov-hard-drive-clock")

这款[基于硬盘的视点时钟](http://hotchk155.blogspot.com/2012/02/digital-pov-clock-working-mostly.html)是绝佳设计选择的宝库。现在，我们已经看到了一堆旋转的时钟构建。几个硬盘版本[使用在盘片上切割的狭缝](http://hackaday.com/2011/07/10/slick-16-segment-pov-hard-drive-clock/)通过在适当的时候点亮狭缝后面的 LED 来创建显示。这是一个类似的想法，但[杰森霍奇基斯]抛弃了所有的盘子，并用一个滤光器取代它们。过滤盘有数字 0-9 以及一个冒号(上面看不到，因为冒号每秒闪烁一次)。当光盘旋转时，Arduino 兼容控制器会点亮八个数字位置的 led，以显示正确的数字。

过滤器由蚀刻的覆铜盘制成。这是一个很好的选择，因为玻璃纤维基板坚固，重量轻，半透明，并且可用。过滤器的想法也意味着你不需要获得电力或数据的旋转平台。[Jason]还设计了一个非常令人印象深刻的控制器板，其大小与他正在使用的笔记本电脑硬盘的大小相同。休息之后，请观看视频，了解他对硬件选择的描述。[https://www.youtube.com/embed/kPdYpST_yoE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kPdYpST_yoE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T3】
