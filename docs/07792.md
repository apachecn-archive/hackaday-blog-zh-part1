# 太阳能手机是真正的黑客

> 原文:[https://hackaday . com/2011/06/07/solar-powered-phone-a-true-hack/](https://hackaday.com/2011/06/07/solar-powered-cellphone-a-true-hack/)

![](../Images/a04cdf9559df5bef2231e87adbf297cb.png "solar-power-cellphone-hack")

这个黑客的抛光质量不太好，但我们喜欢展示的独创性和探索。[Paulie1982]向我们展示了如何利用太阳光让旧手机工作。

你可以在上面看到，他在看起来像旧智能手机的后壳上添加了光伏太阳能电池。他从两个廉价的太阳能景观灯中取出电池，通过在外壳上打孔并使用黑色硅胶密封剂将它们粘合到位来插入电池。每个可以泵出约 3V，它们合在一起超过 5V 阈值，他需要做一些充电。休息后，请查看视频中的构建过程。

从我们所看到的来看，在这个黑客中没有考虑电流，这让我们感到怀疑。尽管如此，我们喜欢涓流充电的想法，我们希望在评论中看到一些关于如何改进这一点的猜测。当然，适当调节所需的额外硬件等。可以安装在一个定制的外壳中，就像用于[这个感应充电器黑客](http://hackaday.com/2010/08/01/update-custom-cellphone-induction-charging/)的外壳一样。
