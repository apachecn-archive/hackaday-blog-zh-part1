# LED 画家

> 原文：<https://hackaday.com/2008/07/01/led-painter/>

Brilldea 的人已经开发出了 [LED Painter](http://www.brilldea.com/product_LEDPainter.html) ，这是一个 16 通道 RGB LED 控制器，能够控制多达 48 个独立的 LED。它使用德州仪器 TLC5940 来控制 LED，并可以连接到更多的 LED 画板，创建一个大型的 RGB 灯阵列。TLC5940 本身已经过改进，可以更容易地连接独立的 led。

该团队将九个这样的灯串在一起，加上一个名为 Prop Blade 的基于[推进器](http://www.parallax.com/Default.aspx?tabid=407)的控制器，并将这些灯安装在三个装有半透明玻璃的窗户上，以创建一个跳舞的随机 RGB 灯显示。如果所有的舞蹈灯都激发了你的灵感，TLC5940s 相当便宜，但你需要通孔器件和一些 SMT 元件来实现。
