# 无线电控制的 led 灯光秀

> 原文:[https://hack aday . com/2011/02/14/radio-controlled-light-show/](https://hackaday.com/2011/02/14/radio-controlled-led-light-show/)

![HL1606_radio_controlled_light_show](../Images/9e9104ebd90d6f5870bf1653aa44cbae.png "HL1606_radio_controlled_light_show")

[Alan]被委托为[流动灯光马戏团](http://travellinglightcircus.com/)制作一些可穿戴的[无线电控制 led 灯条](http://bleaklow.com/2011/01/15/radio_controlled_hl1606_strips.html)。这花了一些时间，但他最近完成了一些原型，并认为这是一个很好的时间来完成这个项目。该系统由单个控制器单元管理，该单元与任意数量的 LED 驱动器单元通信，每个单元控制 4 个 HL1606 LED 灯条。灯光显示通过 2.4 GHz 无线电与所有 LED 驱动器单元同步，每个驱动器在通电后几乎立即同步。虽然有些人可能会因为他使用 Arduino Pro Minis 来控制 led 而关闭，但这远不是一个简单的项目。

[Alan 的]博客包含了几篇关于这个项目的文章，每件事都有详细的记录。他花了相当多的时间谈论项目的软件，以及他在这个过程中遇到的硬件问题。

他的博客是必读的，但更重要的是，它是必看的。灯光效果令人着迷，因为这些装置可以产生大量不同的光模式，所以一定要看看下面的灯光视频。
