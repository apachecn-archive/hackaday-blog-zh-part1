# 海盗湾的目标是云…字面意思

> 原文:[https://hack aday . com/2012/03/21/the-pirate-bay-aims-for-the-clouds-literally/](https://hackaday.com/2012/03/21/the-pirate-bay-aims-for-the-clouds-literally/)

![tpb](../Images/23379b5454aa830f08cc0a8dffe7d387.png "tpb")

不乏政府和娱乐相关机构迫不及待地想要永远关闭海盗湾。虽然该组织没有遭受像[金·多特康姆]和 Megaupload 工作人员那样的永久服务终止袭击，但他们总是在想出新的方法来确保网站能够承受执法部门扔给他们的任何东西。

[在最近的一篇博客文章](http://thepiratebay.se/blog/210)中，该组织的代表公布了将他们的前线服务器放在云端的计划，这要归功于定制的自主无人机，称为“低轨道服务器站”该项目还处于起步阶段，但总的想法是在 GPS 控制的无人机上安装像 Raspberry Pi 这样的小型计算机，使用太阳能保持 24×7(大概)在高空。这些无人机将通过无线电发射机与地面上的客户进行通信，他们表示，无线电发射机可以提供“50 公里以外的 100Mbps 节点”。

称这种说法是夸大其词是一种轻描淡写，但海盗湾已经被证明是一个很难以任何实质性的方式取缔的组织，所以只有时间能证明一切。
