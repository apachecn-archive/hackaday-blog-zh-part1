# 向后的马里奥

> 原文:[https://hackaday.com/2011/01/23/backwards-mario/](https://hackaday.com/2011/01/23/backwards-mario/)

![](../Images/4ea65b5e968bc1cae920607ec34027db.png "backwards-mario")

所以你早就掌握了超级马里奥兄弟，它不再挑战你了？你试过[从右往左打](http://www.dashfest.com/?p=193)吗？这就是《向后的马里奥》的全部。黑客的第一部分是让图像向后显示。他正在用一台旧的阴极射线管电视机工作，这台电视机用一个磁环将电子枪对准屏幕。通过交换该环的左右线，您可以水平翻转图像。现在马里奥将从右向左移动，但控制器按钮会在屏幕上把马里奥带到错误的方向。这很容易解决，只需打开控制器，交换左右按钮的信号。现在是时候重新爱上这个经典游戏了，就像[JJ]在休息后的视频中所做的那样。
