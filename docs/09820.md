# 系列教程向你展示了从微波炉中抢救变压器所需的一切

> 原文：<https://hackaday.com/2012/03/03/tutorial-series-shows-you-everything-you-need-to-salvage-transformers-from-microwaves/>

变压器肯定会增加任何项目的成本，尤其是如果你需要一个大型变压器来完成工作。微波炉是免费变形金刚的巨大来源，尽管它们并不总是你下一个建筑所需要的形状。

[【Matt】整理了三个很棒的教程视频](http://www.youtube.com/playlist?list=PL2FD9A01DD3B7A5D7)，涵盖了回收微波炉变压器(MOTs)的基础知识，任何新手在尝试之前都应该观看。第一个视频介绍了 MOT 拆卸和拆卸，这是一个耗时但简单的过程，只要你遵循[Matt]的指示。

第二个视频深入研究变压器理论，并讨论如何在重建 MOT 或手动缠绕线圈以适应您的项目规格时实现最佳性能。该系列的第三个视频跟随[Matt]重建一个回收的变形金刚，记录他一路上的陷阱和成功。

如果你还没有花太多时间来挽救 MOTs，我们绝对建议你花点时间来完整地观看视频系列——这绝对是值得的。
