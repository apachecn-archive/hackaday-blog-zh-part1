# 迷你街机柜看起来和真的一样好

> 原文:[https://hackaday . com/2011/03/23/mini-arcade-cabinet-looks-as-good-as-the-real-thing/](https://hackaday.com/2011/03/23/mini-arcade-cabinet-looks-as-good-as-the-real-thing/)

![mame_cab](../Images/07d81996d62a84e1eaac0c7eb1114dd8.png "mame_cab")

[Ed]有一台他不再需要的上网本，他决定为他的一些家庭成员把它做成一个迷你 MAME 橱柜。MAME 橱柜非常丰富，但这个做得非常好，我们想分享它。

他从 EeePC 901 上拆下显示器，以便获得一些精确的测量数据，然后着手用中密度纤维板制作一个迷你橱柜。整个东西被裹在粘着的标签纸里，上面装饰着老派的 Galaga 图案，然后覆盖上有机玻璃，看起来很光滑，也保护了艺术品。

他使用 iPac 2 控制器板将所有按钮和操纵杆连接到上网本，选择将控制器的电线直接焊接到 eeePC 主板上的 USB 接头上。为了便于操作，顶部增加了一个电源开关，机柜是密封的，尽管在需要维护的情况下，后面确实很容易打开。该系统是使用 MAME 的 Maximus Arcade 前端来管理的，[Ed]声称这非常简单。

如果你对制作自己的 MAME 橱柜感兴趣，可以看看我们过去介绍过的一些[其他](http://hackaday.com/2011/02/02/shiny-acrylic-mame-box/)[MAME](http://hackaday.com/2010/07/22/all-thats-needed-is-a-retro-paintjob-minimame/)[项目](http://hackaday.com/2009/09/09/in-depth-mame-cocktail-cabinet-build/)，不要错过下面的[Ed's]橱柜的视频。
