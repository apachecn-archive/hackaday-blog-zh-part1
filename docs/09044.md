# 贾维斯在 P 空间打开了门

> 原文:[https://hack aday . com/2011/11/10/Jarvis-opens-the-door-at-p-space/](https://hackaday.com/2011/11/10/jarvis-opens-the-door-at-p-space/)

![](../Images/2ee1ea77107056ba6f5c2b1251af8c40.png "hackerspace-access-control")

似乎每个黑客空间的入口都应该有某种类型的组合访问系统。[Vasilis]来信分享了一个名为 Jarvis 的系统，该系统控制对 P-Space 的访问，这是一个位于希腊帕特雷的黑客空间。这是一个基于 RFID 的系统，提供了一些不错的功能。

他们已经有了一个运行网页的服务器，所以基于计算机的入口系统是一个简单的选择。你可以从两种方式中选择一种:通过在前门向读取器出示您的有效 RFID 标签，或者通过按铃并让里面的成员按下无线控制器的大毁灭按钮(BBoD)。

BBoD 内部有一个 Xbee 模块，允许它向计算机发送解锁命令。遥控器由两节 AA 电池供电，因为除非按下按钮，否则它永远不会打开，该团队估计这些电池应该可以使用大约一年。

甚至还有一个反馈系统。电脑将最后几个事件发布到网页上。所以你可以上网看看 BBoD 是什么时候用的，或者谁的标签最近解锁了入口，告诉你的朋友是否在那里。
