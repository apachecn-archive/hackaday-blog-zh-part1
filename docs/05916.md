# 猎兔犬集群

> 原文:[https://hackaday.com/2010/09/07/beagleboard-cluster/](https://hackaday.com/2010/09/07/beagleboard-cluster/)

![](../Images/1b2d08196862d9d500a632f8b0673a67.png "BeagleBoard Cluster")

你做了一个 [BeagleBoard 绘图计算器](http://hackaday.com/2010/08/06/open-graphing-calculator-beagleboard-r/)之后会做什么？[马特]在 Liquidware Antipasto 做了一个适合放在公文包里的 [BeagleBoard 弹性 R Cluster](http://antipastohw.blogspot.com/2010/09/how-to-make-beagleboard-elastic-r.html) 。十个[比格板](http://beagleboard.org/)，通过 USB 转以太网适配器和一对连接到无线路由器的以太网交换机相互连接。这种集群的成本约为 2000 美元，功耗不到 40 瓦，超过了 4500 美元的笔记本电脑。您会如何使用这个集群？你会做哪些改进？ [https://www.youtube.com/embed/dTpgcycBsQU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/dTpgcycBsQU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
