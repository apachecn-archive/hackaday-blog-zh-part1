# 简单的多氯联苯旨在

> 原文：<https://hackaday.com/2011/12/12/simple-pcb-vise/>

这一个几乎被归入一个链接帖子，但是[Ken 的]简单 PCB vise (PDF)实在是太有用了，我们不得不给它一个独立的功能。它工作得如此之好是因为他把每一个设计特点都考虑进去了。

例如，固定 PCB 的凹槽(在这里几乎看不到，但看看上面链接的 PDF 中的图表)是用燕尾形钻头切割的，而不仅仅是方形兔子。夹紧力由蓝色橡皮筋提供，该橡皮筋简单地钩在夹板每侧的金属架钉上。这些板由聚乙烯加工而成，沿着两根尼龙棒很好地滑动，使它们保持对齐。除了橡皮筋很容易更换之外，这里真的没有什么东西会坏掉或磨损。橡胶支脚可防止它在您工作时滑过长凳。

这对于焊接来说很棒，并且可以和你做的那些 [diy smd 零件夹具](http://hackaday.com/2010/12/13/diy-clamp-helps-with-surface-mount-soldering/)一起使用。这也是一个很好的方法来保持你的原型板，当你工作的固件。
