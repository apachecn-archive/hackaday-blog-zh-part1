# 用于安卓游戏的蓝牙超级任天堂控制器

> 原文：<https://hackaday.com/2011/05/17/bluetooth-super-nintendo-controller-for-android-gaming/>

![bluetooth_snes_controller](img/2187ad911801391c1e366a6070618e51.png "bluetooth_snes_controller")

[Rich]需要提出一个高级设计项目，并决定将他热爱的两样东西结合起来:[他的 Android 手机和超级任天堂](http://moddedbybacteria.freeforums.org/bluetooth-snes-controller-for-android-t1458.html)。

虽然触摸屏手机很棒，但他觉得在游戏方面，没有什么能打败物理控制器的触觉反馈。他弄清楚了控制器的信号是如何工作的，然后将其连接到 Arduino Pro Mini 328 上。Arduino 解释 SNES 控制器的信号，通过 BlueSMiRF 蓝牙模块将信号发送到他的 Android 手机。

他最初把所有的组件都塞进了一个纸箱里，但是[就像我们昨天指出的](http://hackaday.com/2011/05/16/making-the-case-for-cool-project-enclosures/)一样，他意识到当一个项目被放在一个合适的盒子里时，它才真正走到了一起。除了用来给遥控器供电的电池组之外，他设法把所有组件都塞进了 SNES 控制器的外壳。在涂了一点 Bondo 和几层油漆之后，控制器看起来非常锋利。

留下来看看他的控制器运行的快速演示视频，并查看他整理的教程，解释他用来构建它的一些原则。
