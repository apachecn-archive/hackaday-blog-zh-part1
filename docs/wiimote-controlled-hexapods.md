# Wiimote 控制的六足机器人

> 原文：<https://hackaday.com/2009/04/05/wiimote-controlled-hexapods/>

[https://www.youtube.com/embed/QsBj7koZ9lk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QsBj7koZ9lk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

凤凰号是一个令人印象深刻的六足机器人平台。它有 18 个伺服系统，使每条腿有 3 个自由度，大脑有一个 BasicAtom Pro 28。有趣的是，这个设计最初是 Lynxmotion 论坛上一个论坛成员的个人项目。结果如此之好，它已经成为一个实际的产品。我们以前看过这些视频，它们总是有一些非常流畅和有机的运动。在这种配置下，它们看起来几乎是活的。唯一可能让他们更恐怖的是加上[卢·维加的无头头颅](http://hackaday.com/2008/10/31/horrifyingly-wonderful-hexapod-dancing/)，也许再加上一些[非常漂亮的面部追踪](http://www.youtube.com/watch?v=4oXuSXCKJeY)。在上面的视频中，你可以看到有人[将其中一个与 Wiimote](http://basicmicro.com/) 配对，以获得一个非常直观的控制方案。是的，我们意识到视频是近一年前的，我们怎么会错过这个？你可以看到它在休息后四处走动的视频，以及另一个由 ps2 控制器控制的视频。
