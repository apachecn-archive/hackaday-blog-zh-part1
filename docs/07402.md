# 水族馆溢出传感器拯救你的鱼和你的地板

> 原文:[https://hack aday . com/2011/04/15/aquarium-overflow-sensor-saves-your-fish-and-your-floors/](https://hackaday.com/2011/04/15/aquarium-overflow-sensor-saves-your-fish-and-your-floors/)

如果你的家里曾经漏水，你就会知道当你走进门听到流水的声音时，那种下沉的感觉。[Greg]非常了解这种感觉，他不止一次回到家里，看到湿漉漉的地板和满溢的礁石水族箱。

他经历的两次溢流都是由于他的显示器中的排水管堵塞造成的，但他几乎无法阻止排水管堵塞。由于所有这些脆弱的生物都生活在鱼缸里，我想到的唯一可能的解决办法就是[监控鱼缸的水位](http://www.monsterminibow.com/ofp.html)。

不幸的是，除了使用探针(在盐水中会生锈)或昂贵的现成系统，他不知道如何做到这一点。来自 All About Circuits 论坛的[erich_7719]帮助[Greg]解决了这个问题，并为他设计了一个使用红外传感器监控水位的电路。如果水位变得非常高，电路会简单地关闭水泵。正如你在下面的视频中看到的，它工作得很好，作为一种安全措施，一旦高水位传感器被触发，需要手动重启泵。

如果你需要同样的设置，可以去他的网站找一份详细的原理图和材料清单。
