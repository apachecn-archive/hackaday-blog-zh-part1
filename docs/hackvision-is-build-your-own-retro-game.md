# Hackvision 是打造自己的复古游戏

> 原文：<https://hackaday.com/2010/10/24/hackvision-is-build-your-own-retro-game/>

如果你想尝试编程一些复古游戏，Hackvision 可以快速启动这个过程。这是一款基于 Arduino 的游戏控制台，采用控制器格式。你有四个方向按钮和一个功能按钮。它有两个 RCA 插孔，用于单声道音频和黑白视频。

我们很高兴地发现，有关于游戏开发的[信息](http://nootropicdesign.com/hackvision/gamedev.html)将帮助您了解太空入侵者和 Pong 示例。该系统使用 Arduino TVout 视频库，这是一个健壮且相当容易交互的库。但是一旦你在视频中看到这个游戏，你会很难抗拒建造一个这样的东西。别忘了，这是基于 Arduino 的。如果你已经有一个使用 ATmega328 的 Arduino，你只需要构建音频、视频和按钮电路。
