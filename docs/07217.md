# Kinect 二合一:MoCap 电影和机器人控制

> 原文:[https://hack aday . com/2011/03/26/Kinect-two-fer-mocap-movie-and-robot-control/](https://hackaday.com/2011/03/26/kinect-two-fer-mocap-movie-and-robot-control/)

![kinect_twofer](../Images/abc740bd5fe483cf351a65d0923a198c.png "kinect_twofer")

众所周知，我们喜欢 Kinect，这也是我们今天为您带来 Kinect two-fer 的原因。

我们以前见过使用 Kinect 的视频黑客，这一个在酷炫程度上排名靠前。在[to rben]的短片中，一名动画学生差点错过了他的作业期限，Kinect 被用来编写简笔画模型的动画。动画是在 Maya 中捕获和构建的，然后覆盖在单独的视频剪辑上以完成电影。整体质量很好，尽管你可以注意到 Kinect 的一些典型的“抖动”，并且有几个地方模型有点沉入地板。

如果你想尝试使用 Kinect 制作动画，所有用于制作电影的脚本都可以在创作者的网站上免费获得。[通过 [Kinect-Hacks](http://www.kinect-hacks.com/kinect-news/2011/03/24/all-nighter-independent-short-film-using-kinect-and-maya)

我们的第二个 Kinect 产品是一个由手势驱动的乐高思维风暴机器人。使用 OpenNI 和 Primesense 进行身体跟踪，[rasomuro]能够使用简单的动作在房子周围驾驶他的 NXT 机器人。他的动作被 Kinect 传感器跟踪，并通过笔记本电脑的蓝牙连接转换成命令传递给机器人。由于机器人有两个马达，他绘制了几个简单的手臂动作来驱动机器人。老实说，当我们说这些动作让我们想起了威尔·法雷尔在《老派》中的“坦克弗兰克”场景时，我们会说，但[罗索姆罗]说他正试图模拟使用杠杆来驱动机器人。不管怎样，都很酷。

这两次攻击的视频都嵌入在下面供您阅读。
