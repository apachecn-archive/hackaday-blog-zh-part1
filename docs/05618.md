# 开始使用语音识别

> 原文:[https://hack aday . com/2010/07/09/语音识别入门/](https://hackaday.com/2010/07/09/get-started-with-speech-recognition/)

语音识别让我们更容易对我们的设备偷懒——或者建立最酷的语音控制项目。在[语音控制家庭自动化](http://hackaday.com/2009/02/14/voice-controlled-home-automation/)的帖子之后，我们收到了许多电子邮件，询问“我如何才能让它识别我的声音？”。无论你的项目是涉及 PC 还是 Android 手机，高预算还是没有预算，都有一个解决方案。休息之后，请加入我们，了解关于设置语音识别的完整说明，以及一些满足您需求的最佳软件选项。

有麦克风吗？

使用麦克风是将你的语音命令传到计算机进行翻译的唯一方法。如果你有一台笔记本电脑，你可能已经准备好了，因为大多数笔记本电脑都已经内置了麦克风。不确定？寻找屏幕或键盘周围的小孔。它可能会被贴上标签，但并不总是如此。你也可以试着查看电脑手册中的功能列表，或者打开控制面板，选择“声音”。在这个弹出窗口中，您会发现一个名为“录制”的选项卡。如果你已经安装了麦克风，它会列在这里。

[![A built in mic](../Images/7aea457032db4f52e2fe05841d6843e7.png "Built in Microphone")T2】](http://hackaday.com/wp-content/uploads/2010/07/capture.jpg)

如果你用的是台式机，你可能需要购买一个外置麦克风。许多网络摄像头都有内置麦克风，请检查包装以确保无误。一些较新的媒体键盘还包括麦克风。如果你是这种情况，你可能需要将键盘移出有限的空间以减少回声。如果你是一个 PC 或 Xbox 360 游戏玩家，你可能会有一个耳机用于与其他玩家进行在线交流。这可以兼作语音识别的麦克风。这些都没有？去离你最近的卖电脑配件的商店——试试百思买，未来商店，RadioShack，或者你最喜欢的本地零售商。一个基本的可用话筒价格从几美元到数百美元不等。虽然除非你打算用电脑录制录音室专辑，否则 600 美元的麦克风是不必要的，但远离最便宜的麦克风可能是一个好主意——这些麦克风通常会有你的电脑无法解释的断断续续和不均匀的声音。一般来说，耳机麦克风(或游戏耳机)是最好的选择，因为它靠近你的嘴，干扰最小。确保您选择的麦克风与您的电脑操作系统兼容，并且有您的电脑使用的输入，然后购买！

**设置您的语音识别软件**

***Windows 7 或 Windows Vista***

这两种操作系统上的语音识别只需点击几下鼠标。有了这些操作系统，语音命令是彻底和简单的，允许你控制一切，从表单导航，菜单导航，办公程序，等等。几乎任何你需要做的事情，都有一个语音命令。首先，进入控制面板，选择“语音识别”。

![Control Panel in Windows 7](../Images/f3b9574f098e6f407e2fda3fd37714a8.png "Control Panel W7")

从这里，您可以测试您的麦克风，训练您的计算机理解您的个人讲话风格，或者查看和打印包含您的计算机可以理解的命令的参考卡。

![Speech Recognition Dashboard in Windows 7](../Images/1cd799c283e8acadee1db89bf64671e5.png "Speech Recognition Dashboard")

你也可以参加一个教程，用一节简单的课教你语音识别的来龙去脉。准备好开始时，请选择“开始语音识别”选项。这将引导您通过定位提示和语音测试来优化计算机的声音输入，并以非常用户友好的方式指导您完成其余的配置。完成向导后，您就可以开始了！

![Speech Recognition Wizard in Windows 7](../Images/8f83c8b89e394521b099e7b40b5fd8e8.png "Speech Recognition Wizard")

您可以根据需要随时查阅语音识别参考卡，查看计算机可以理解的命令。

![Speech Recognition Reference Card in Windows 7](../Images/7cc1346d43dc92eda84e1643234044ce.png "Reference Card")

***Windows XP***

XP 中的语音识别与较新的 Windows 操作系统一样容易设置，但是它缺少 Vista 和 7 提供的大量功能。所有 Microsoft Office 程序都支持语音识别，但是，仅支持 2002 和 2003 版本。对于 2002 年以前的版本，或者 XP 中的 2007 或 2010 版本，您就不太走运了，因为不支持内置语音识别。否则，并不总是支持基本命令，并且语音识别不能用于所有程序。一般来说，您必须为希望使用语音识别的每个程序专门启用语音识别，而且它并不是在所有程序中都可用。

Windows XP 使用 Office XP 附带的语音识别引擎，尽管默认情况下并不总是安装该引擎。打开控制面板，从经典显示中选择“语音”选项。如果你在 XP 中使用更新的分类菜单，你必须首先选择“声音、语音和音频设备”选项。

![Speech Recognition Icon in Windows XP](../Images/0664e2fa843c886a550783a6c2bc6b0b.png "Speech Icon in XP")

*Youtube:[mickmoose 429992][【链接】](http://www.youtube.com/watch?v=VhEGW8x5Wps)*

如果你在“语音属性”中看到一个“语音识别”标签，你就可以开始了，因为引擎已经安装好了。

![Speech Properties in Windows XP](../Images/3c129ede4d6059c0bd21b631d4b8380c.png "Speech Properties")

*Youtube:[mickmoose 429992][【链接】](http://www.youtube.com/watch?v=VhEGW8x5Wps)*

如果缺少此选项，您需要安装它。从控制面板中，选择“添加或删除程序”选项。

![Add or Remove Programs in Windows XP](../Images/5ccee3c3e1596b0af3f13f63cf30791f.png "Add or Remove Programs")

*Youtube:[mickmoose 429992][【链接】](http://www.youtube.com/watch?v=VhEGW8x5Wps)*

找到微软办公 XP，选择“更改”选项。注意不要卸载！

![Change Microsoft Office XP](../Images/a2d96f292827c7b4b1050bb09f7dbd50.png "Check Microsoft Office XP")

*Youtube:[mickmoose 429992][【链接】](http://www.youtube.com/watch?v=VhEGW8x5Wps)*

找到“要安装的功能”，选择“替代用户输入”选项，然后是“语音”选项。选择“从我的电脑运行”,然后单击“更新”。这将自动在所有 Office 程序中包括语音识别，并使该功能可用于其他程序。

![Add Speech to Microsoft Office XP](../Images/4baf49b3600c21f4570072e455cbd116.png "Add Speech to Microsoft Office XP")*Youtube:[mickmoose 429992][【链接】](http://www.youtube.com/watch?v=VhEGW8x5Wps)*

***麦克 OS X**T3】*

苹果是最早推出语音识别的公司之一，这在当时是一个疯狂的想法。这是在 1993 年。从那时起，我们已经走过了漫长的道路，从更加流畅、用户友好的控制到无需触摸键盘就能执行几乎任何操作的能力。在 OS X 设置语音识别是轻而易举的事。准备好麦克风后，从苹果下拉菜单中选择“系统偏好设置”。从该菜单中，选择“语音”选项。

![Mac OS X Speech Feature](../Images/829414d16b622d1706dab88a168febf7.png "Mac OS X Speech Feature")

*Youtube: [fifedjdomo] [【链接】](http://www.youtube.com/watch?v=vi9bDfmRDt8)*

启用“可口述项目”将打开默认命令，允许您执行大多数基本任务。

![Mac Speakable Items](../Images/6af4e35612098f73677bb754d5c02add.png "Mac Speakable Items")

*Youtube: [fifedjdomo] [【链接】](http://www.youtube.com/watch?v=vi9bDfmRDt8)*

通过可用的选项，您可以设置您的麦克风，并进一步定制程序的使用。用来控制计算机的命令集是完全可定制的。将它与 VoiceOver(一个为盲人设计的程序)结合起来，你几乎不需要触摸电脑就能使用它。

***Linux Ubuntu**T3】*

Linux 目前没有完整的语音识别解决方案。虽然几个项目已经开始，但没有一个完成。有几个软件可以完成 Windows 或 Mac 可以完成的一些语音识别任务，但远没有那么彻底或容易。在 Linux 上也没有用于语音识别的专有软件，但是，有一些用于 Ubuntu 的部分完成的开源解决方案。Julius 语音识别引擎就是这些工具中的一个——一个用于解释和执行一组预先确定的语音命令的程序。安装的详细说明可以在[【此处】](http://julius.sourceforge.jp/en_index.php)找到。

![Julius Main Page](../Images/c2764570091c94ce75c56fa3ff73c6ab.png "Julius")*Youtube:[jgraves 1141][【链接】](http://www.youtube.com/watch?v=s1srNOk2ISI)*

关于安装和使用 Julius 的文档是非常有限的，因为这个程序还没有完全完成，所以你可能不想尝试安装，除非你完全熟悉 Linux 的使用。可以下载的 Julius 包包含两部分——安装程序和程序。首先运行安装程序，它将带你完成 Julius 的安装。

另一个很好的解决方案是使用一个基于 Windows 的程序，例如在 combination WineHQ 中的 Dragon NatuallySpeaking，但是，在流畅性方面存在一些问题，通常需要解决。例如，在某些情况下，必须向 Dragon 的文本编辑器叙述一个基本段落，然后复制粘贴到适当的位置，而不是直接转录到适当的程序。

![WineHQ](../Images/fac3e1b82d317ea97b6f25f032ae5a04.png "WineHQ") * WineHQ: [【链接】](http://www.winehq.org/)*

**顶级第三方专有软件**

如果您的操作系统较旧，或者只是不喜欢操作系统中的语音识别软件，那么第三方程序可能是您所需要的。有几十种免费和付费的语音识别，可定制的，不可定制的，开源的，商业的，个人使用的等等。有这么多选择，你一定能以你能承受的价格找到你想要的东西。一些最受欢迎的:

***龙族天生会说话***

Dragon 是搜索语音识别软件时反复弹出的名字。它专为 PC 设计，因其速度、准确性、易用性和大量命令而备受推崇。家庭使用的基本版 Dragon 大约 100 美元，尽管许多版本都有更具体的功能，如医疗或法律办公室的软件包。这些软件包的价格可能超过 1000 美元，尽管对基本用户来说是不必要的。Dragon NaturallySpeaking 软件包还包括一个麦克风，因此您不必自己去找。除了 Windows 之外，许多用户声称 Dragon 与 WineHQ for Ubuntu 的结合取得了巨大成功。

![Dragon NaturallySpeaking](../Images/61d77799e0b2e6bedc52d015122353bc.png "Dragon") *龙自然地说:[【链接】](http://www.nuance.com/naturallyspeaking/products/editions/default.asp)*

***演讲口述***

MacSpeech 与 Dragon NaturallySpeaking 由同一家制造商生产。它是从头开始构建的，而不是移植的，所以它没有改编软件通常会出现的错误。与 Dragon 类似，MacSpeech 不仅提供听写识别，还提供可定制的语音命令，并在包中包含一个麦克风。也遵循龙的主题，医疗和法律版本可用，以及国际版，支持意大利语，法语和德语除了英语。这些 Mac 语音识别工具的价格从 150 美元到 600 美元不等。

![MacSpeech Dictate](../Images/f362d698a76083cc569f0d1398dc53d1.png "MacSpeech") * MacSpeech 口述:[【链接】](http://www.nuance.com/naturallyspeaking/products/macintosh/for-the-mac.asp)*

***IBM ViaVoice***

IBM 的 ViaVoice 识别软件主要是为小型移动设备和车辆自动化系统设计的，尽管它在计算机用户中也受到很高的评价。除了语音识别之外，ViaVoice 还提供文本到语音转换。命令库是直观的，用户不需要坚持使用标准的命令集来使用所有的功能-程序可以解释给定的大多数命令。语音库包含超过 20 万个单词；远远超过一般人的词汇量。除了许多移动操作系统之外，IBM ViaVoice 还支持标准的 Windows 和 Mac 操作系统。

**第三方开源和自由软件**

运行良好的开源或免费语音识别软件极难找到——在免费语音软件的开源竞赛中，真的没有赢家。事实上，几乎没有一个种族。许多开源 Linux 项目已经开始，但是由于规模太大，没有一个已经完成。下面是一个你可以参与的项目，目的是开发一些优秀的开源语音识别软件，以及一个供你自己使用的工具包。

***VoxForge***

VoxForge 是一个致力于编译转录语音集合的项目，用于开源和免费的语音识别引擎。在这个项目完成后，免费的开源语音识别程序应该会得到大幅增加。如果你想帮助这个项目，你可以访问 VoxForge 网站[【这里】](http://www.voxforge.org/)。

**如何安装 CMUSphinx**

设置 CMUSphinx 并不是最容易的任务，但它很可能会因为一个伟大的产品而获得回报。此安装需要手动完成。

在开始之前，您需要一些东西——Perl，以便运行脚本，以及源代码的 C 编译器。Perl 是免费的，包含在大多数 Linux 发行版中。GCC (GNU 编译器集合)是一个很好的工具，用于 C 部分的源代码。单词对齐程序也是必要的——CMU 建议使用“Sclite ”,一种专门用于语音识别程序的工具。

您将需要的数据库在[【此处】](http://www.speech.cs.cmu.edu/databases)可用。你需要一个 AN4 或者 RM1。接下来，您需要设置训练器。训练器帮助您的计算机解释您的命令。设置教程–这将包括将脚本复制到适当的区域。解码器是下一个。尽管你可以选择任何你喜欢的解码器，CMU 描述了 Sphinx 3 的安装，并鼓励你用 Sphinx 3 进行测试。一旦所有合适的文件都在正确的目录中，就该编译和设置教程了。执行训练运行，最后执行解码。这种设置极其复杂，可能最好留给专业人士来做——对于大多数普通用户来说肯定不是这样。

完整的说明可以在卡内基梅隆大学的斯芬克斯网站[【这里】](http://cmusphinx.sourceforge.net/html/tutorial.html)找到。
