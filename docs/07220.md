# WiFi 收音机以时尚的方式播放您的音乐

> 原文:[https://hack aday . com/2011/03/26/wifi-radio-plays-your-tunes-in-style/](https://hackaday.com/2011/03/26/wifi-radio-plays-your-tunes-in-style/)

![wifi_radio](../Images/96916b9162fa480b002327588fc5ef10.png "wifi_radio")

Instructables 用户[Jan]喜欢在他的工作室里一边干一边听音乐，但是听同样的 CD 过一会儿就会感到厌倦和无聊。他考虑通过互联网收听流媒体音频，但讨厌需要一台电脑一直在身边的想法。看了一会儿之后，他找到了一些关于建造 WiFi 无线电的信息，然后[开始建造他自己的](http://www.instructables.com/id/Build-your-own-Wifi-radio)。

利用他在 MightyOhm 找到的指南，他入侵了一个华硕路由器来使用 OpenWRT，添加了一个音乐播放器守护程序来调谐各个电台。他增加了一个小型液晶显示器和一个 ATmega32 来驱动它，以及一个旋转编码器来允许他在不同的电台之间切换。

这个箱子是用几层中密度纤维板制成的，这些中密度纤维板是用数控铣床切割的，然后用胶水和木榫钉连接在一起。前面板和后面板是由 alucobond 板材铣成的，箱子的其余部分覆盖着白色的木质贴面。这个建筑的细节非常棒，我们特别喜欢“WiFi 符号”扬声器格栅。

他所有的代码和原理图都可以下载，如果你想做一个自己的无线电台。留下来看看他制作完成的收音机的视频。
