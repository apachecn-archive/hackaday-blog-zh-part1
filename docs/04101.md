# 有了 Zipit，谁还需要上网本？

> 原文:[https://hackaday . com/2009/09/25/with-zipit-who-needs-a-netbook/](https://hackaday.com/2009/09/25/with-zipit-who-needs-a-netbook/)

[https://www.youtube.com/embed/x_LrI2g2VT8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/x_LrI2g2VT8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[亨特·戴维斯]继续为 Zipit 推出黑客。过去，他向我们展示了如何运行 DOSbox T1，以及如何让 to get 仿真在这个小设备上运行 T3。现在他有了运行 Fluxbox 的 Linux 内核 2.6.29，带有鼠标(新添加的)、音频和 WiFi 功能。按照他的[一步一步的闪烁指令](http://hunterdavis.com/archives/201#more-201)将内核加载到 Zipit。一旦闪存，一个分区的 microSD 卡作为文件系统和[交换](http://en.wikipedia.org/wiki/Swap_space#Linux)服务。

当你可以以大约 40 美元的价格通过 [XScale 处理器](http://en.wikipedia.org/wiki/XScale)获得这种 2.8 英寸 QVGA 的美感时，谁还需要 10.1 英寸的屏幕或 Atom 处理器呢？
