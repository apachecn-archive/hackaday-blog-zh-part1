# 无中生有的三维建模

> 原文:[https://hack aday . com/2011/03/14/3d-modeling-out-of-thin-air/](https://hackaday.com/2011/03/14/3d-modeling-out-of-thin-air/)

![kinect_3d_modeling](../Images/61bd73ebb60387a112d7d20b23518824.png "kinect_3d_modeling")

似乎随着每一天的过去，我们看到的 Kinect 黑客变得更加令人印象深刻。以今天发给我们的这个小数字为例。

[sonsofsol]结合了几个开源软件包和一点电子技术，创造了一个我们最近看到的更有用的 Kinect hacks。他的项目使他能够通过在 Kinect 传感器前移动双手来操纵 GEM 中的 3D 模型。使用 OpenNI 和 Ubuntu，他的所有动作都被计算机跟踪，并被转换成 GEM 3D 引擎中的动作。

为了让自己轻松一些，他还制作了一副电子手套来与系统交互。通过使用 Arduino，手套可以向 3D 建模软件发送不同的复杂命令，只需将不同的手指接触在一起。

你真的需要看看下面嵌入的视频，感受一下[sonsofsol 的]“简单”的网格建模器到底有多复杂。
