# 无论你去哪里，都有人盯着你

> 原文：<https://hackaday.com/2011/10/27/vigos-stare-follows-you-wherever-you-go/>

为了在万圣节装饰办公室，埃里克决定让喀尔巴阡山脉的维果盯着路人看。我们希望读者能认出这张照片，但对于那些不认识的年轻黑客来说，这幅[维戈]的画在经典电影《捉鬼敢死队 2》的[中扮演了重要角色。](http://www.youtube.com/watch?v=UnzH75FlwvU)

在电影中，他的眼睛似乎在跟随任何看这幅画的人。[Eric]拿起一个 Kinect，用处理来重现现实生活中的效果。图像显示在液晶显示屏上。用 Photoshop 做了一点工作，让他从图像中切出眼睛，然后创建由处理草图移动的精灵。它正在从 Kinect 读取数据(所以它知道“看”哪里)，你可以看到它停在隔间墙的顶部。这种幻觉是令人愉快的，请在休息后的片段中亲自观看。我们已经看了六遍了，看起来它在开放日上受到了客人们的欢迎。

你能相信他们只用一天就把这些组装起来了吗？
