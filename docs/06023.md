# 建造一个水晶烤箱

> 原文:[https://hackaday.com/2010/09/28/building-a-crystal-oven/](https://hackaday.com/2010/09/28/building-a-crystal-oven/)

![](../Images/d312d2ba6dd0c00c4e44a2c761c875c4.png "crystal-oven")

无线电通信依赖于稳定的振荡器频率，考虑到这一点，[Scott Harden]建造了一个模块来调节晶体振荡器的温度。休息后的视频中概述了这一过程，但它是这样的:一个小正方形的双面覆铜板被用作基底。晶体振荡器的主体安装在基座的一侧。另一边是一个 mosfet 和一个热敏电阻。热敏电阻的电阻开启和关闭 mosfet，以保持稳定的温度。

这是[斯科特的]水晶烤箱的第一次迭代。它是为户外使用而设计的，因为他的室内设置使用聚苯乙烯泡沫盒来隔离振荡器与环境温度。他已经在开发第二个版本，并提到了加入了惠斯通电桥，但是我们还需要等待更多的细节。
