# Twittertape Machine 会跟踪你的社交媒体股票。

> 原文:[https://hack aday . com/2012/03/17/Twitter tape-machine-keep-track-of-your-social-media-stock/](https://hackaday.com/2012/03/17/twittertape-machine-keeps-track-of-your-social-media-stock/)

![](../Images/706ae0a9bd15a16f8c27202145e40b1f.png "ticker")

在镀金时代，石油大亨、企业家和强盗大亨的书房里会有一台自动收报机。这台机器将打印股票和商品价格，并绘制这些人的资产越来越高的图表。100 年来发生了很多变化，现在[亚当]可以通过他的推特录音机得知@金卡戴珊、@巴拉克·奥巴马和@斯蒂芬里早餐吃了什么。

有趣的是，[亚当]的建造并不是从一块有 100 年历史的锈迹斑斑的黄铜开始的；他用在易贝捡到的旧钟表机芯制作了他漂亮的自动收报机。即使构建的闪亮部分仅容纳纸卷，它仍然是一个很棒的构建。现在这台机器连接到以太网，但他计划添加 WiFi 和一些电池，以实现完全无线的构建。

与我们本周看到的另一台[自动收报机](http://hackaday.com/2012/03/15/bringing-a-19th-century-stock-ticker-into-the-21st-century/)不同的是，【亚当】消除了 100 年前自动收报机中齿轮和螺线管发出的巨大碰撞声。这种自动收报机打印在收银机收据纸上，底部有一个非常小的热敏打印机。虽然[亚当]的建造听起来不像两个机器人在打架，但不需要墨水，也没有字母轮错位和拼写错误的危险。
