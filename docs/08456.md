# 有火吗？

> 原文:[https://hackaday.com/2011/08/29/got-an-ilight/](https://hackaday.com/2011/08/29/got-an-ilight/)

![](../Images/b641de3b1bd2f1642f1b23501dc46bea.png "iPhone4_Display_LEDs_Iluminating_your_Path")

[卡门]从他的[博客中给我们发了一篇关于如何处理坏掉的 iPhone](http://karman.homelinux.net/blog/archives/690) (谷歌[翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://karman.homelinux.net/blog/archives/690))的文章。正如许多 iPhone 用户发现的那样，不需要摔那么多跤，就能让你闪亮的手持指挥中心变成一堆碎玻璃。替换零件和保险是可以得到的，所以生活还在继续，但你还能做什么？

如果你的屏幕像[卡门]的一样裂开，你可以做的一件事是从设备上取下白色 LED 背光条，并重复使用。一旦他移除了 LED，他必须修复一些痕迹，但这没什么大不了的。然后探测该条带，以查看各个 LED 的位置，在这种情况下是 6 个串联。该条被连接到一个 12v 的电源，现在你有一个小，但明亮的灯来进行一些午夜黑客活动。
