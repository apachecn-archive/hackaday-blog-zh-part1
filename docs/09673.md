# 用光绘图

> 原文:[https://hack aday . com/2012/02/11/plotting-pictures-with-light/](https://hackaday.com/2012/02/11/plotting-pictures-with-light/)

![](../Images/a7c859e3688fbfc3ff53e17b0cf62055.png "light")

自行车车轮、闹钟，甚至是在空中画图像的光画棒上都有闪烁的发光二极管，用于视觉显示的持续性。如果你想用一个 LED 在空中绘制一幅图像会怎么样？这就是[acorv]从[极坐标绘图仪](http://hackaday.com/2011/11/17/polar-pen-plotter-draws-huge-images-very-slowly/)得到提示后所做的。

和极坐标绘图仪[和绘图机器人](http://hackaday.com/2010/06/06/drawbot-produces-portraits-very-slowly/)一样，【acorv】的建造始于一对[步进电机和钓鱼线](http://mesalioasi.blogspot.com/2012/01/drawbot-robot-dibujante.html) ( [翻译](http://translate.google.com/translate?hl=en&sl=es&tl=en&u=http%3A%2F%2Fmesalioasi.blogspot.com%2F2012%2F01%2Fdrawbot-robot-dibujante.html))。[acorv]的兄弟提高了赌注，建议用 LED 代替标记，并拍摄长时间曝光的照片。带着 DSLR 和极大的耐心，拍摄了几张实验照片。为了绘制图像，Lightbot 在穿过绘图区域时会闪烁其 LED。一个像素一个像素地构建图像的过程需要一段时间——这张图像需要八分钟——但是兄弟俩受到足够的鼓励，把他们的装备拿到外面。

在两个三脚架之间安装了极坐标绘图仪后，[acorv]和他的兄弟在夜深人静的时候制作了这张图片。这是对我们之前看到的 POV 主导的构建的一个有趣的旋转。看看[acorv]的 Lightbot 在休息后慢慢地画着什么。
