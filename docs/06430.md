# 使用 EL 线的 Jacob 梯子

> 原文:[https://hack aday . com/2010/12/12/Jacobs-ladder-using-El-wire/](https://hackaday.com/2010/12/12/jacobs-ladder-using-el-wire/)

![](../Images/e8957acf7079dc37cd4aa870fcca5f40.png "el-jacobs-ladder")

为了万圣节，[保罗]想做一个没有高压工作可能带来的危险的雅各布梯子。他受到电致发光电线的序列板的启发，决定用发光材料制作一个雅各布的梯子模拟器。他最后的结果很有说服力。八段 EL 线被安装在两个分叉的塔之间。当 PIR 传感器检测到房间中的运动时，Arduino 会打开模拟，播放经典的咝咝电压声音，同时使用音序板从下到上闪烁电线。休息后在视频里自己看吧。我们给[保罗]加分，因为他用乐高建造了这个基地。

但是，如果你不是一个小心谨慎的人，总会有[这种真正的雅各布之梯构建](http://hackaday.com/2008/12/30/jacobs-ladder/)。或者也许你只是想[用 EL 线](http://hackaday.com/2010/12/07/how-to-work-with-el-wire/)做些发光的东西。
