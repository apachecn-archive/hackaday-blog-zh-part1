# 无线漫游者有两把枪…你们每人一把

> 原文：<https://hackaday.com/2011/05/25/wireless-rover-has-two-guns-one-for-each-of-ya/>

![texas_ranger_airsoft_rover](img/5277948829c4b1292db9f121d874263a.png "texas_ranger_airsoft_rover")

我们从来没有对这里的遥控漫游车感到厌倦，尤其是当它们涉及到重复使用一些旧硬件和激光的时候。[大亨]来信分享他的创作，他称之为“德州骑警”。

Texas Ranger 是围绕一个旧的 Linksys WRT54GL 路由器建造的，该路由器为漫游者提供 WiFi 连接以及串行接口，通过该接口可以控制其他一切。漫游者具有一对 PIC 微控制器，它们处理所有的伺服控制以及遥测计算。

一个车载摄像头让操作者可以从驾驶座的角度观察行动，从而对车辆进行精确控制。激光三角测量被用来帮助测量物距，一对气枪弹丸枪跨在相机上，每当[大亨]想让人们知道他的存在。我们特别喜欢的一个功能是一对 Wii 双截棍，漫游者用它来监控自己的位置。它总是知道它的操作角度，自动调整相机以补偿不平坦的表面，保证[大亨]不必倾斜他的头看直。
