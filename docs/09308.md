# 改进糟糕的电脑扬声器

> 原文:[https://hack aday . com/2011/12/15/improving-terrible-computer-speakers/](https://hackaday.com/2011/12/15/improving-terrible-computer-speakers/)

![](../Images/cfd987b07ac55af4dfe4b2461e6c2842.png "amp")

[Victor]喜欢在他的笔记本电脑上看电影，但他机器上的小扬声器不能很好地表现[约翰·威廉姆斯]和其他完美的原声音乐。为了增加一点果酱，[Victor]为 Sinterklaas 买了一对 Trust Mila 2.0 扬声器。不幸的是，这些扬声器很糟糕——到处都是噪音，微弱的输出和比太阳还要亮的蓝色 LED。一旦[Victor] [更换了两个扬声器](http://www.hackvandedam.nl/blog/?p=351)中的放大器，这些问题就得到解决。

在为每个扬声器购买新的功率放大器后，[Vic]偶然发现了 [MAX9575 3.2 瓦放大器](http://www.maxim-ic.com/datasheet/index.mvp/id/4738)。这个小家伙符合[维克多]的所有要求。唯一的问题是 MAX9575 仅提供 TQFN 封装。

深吸一口气，汗流浃背之后，两个放大器都在各自的扬声器中找到了新家，死 bug 风格。蚀刻印刷电路板可能会更容易，但无论如何，我们还是要向[Victor]的精细运动技能致敬。

由于标题图片中展示的疯狂焊接技术，[维克]现在有了一对非常好的扬声器。休息之后，请查看改进后的扬声器演示。
