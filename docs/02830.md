# 简单的伺服机器人计划

> 原文:[https://hackaday.com/2008/11/22/simple-servo-bot-plans/](https://hackaday.com/2008/11/22/simple-servo-bot-plans/)

![had_arduino_servo](../Images/39584e2d83669e1256be45c90cabf317.png "had_arduino_servo")

oomlout 已经发布了一些有趣的简单机器人的[计划。它基于 Arduino，是一个类似于 Parallax](http://www.instructables.com/id/How_to_Make_an_Arduino_Controlled_Servo_Robot_SER/ "How to Make an Arduino Controlled Servo Robot (SERB)") [Boe-Bot](http://www.parallax.com/tabid/411/Default.aspx "Boe-Bot Robot Information") 的平台。Arduino 向驱动两个主轮的连续旋转伺服系统发送 PWM 信号。所有的结构部件都是用激光从丙烯酸树脂上切割下来的，上面有槽可以固定标准的六角螺母。这是一个有趣的技术，但这个设计还有很大的改进潜力。目前，它使用两种不同的电源和一个试验板进行简单的连接。从下面的视频中，你可以看到平衡也可以得到改善。
