# 骑着火箭和喷气式飞机在瑞典冰冻的荒原上兜风

> 原文：<https://hackaday.com/2012/03/27/riding-rockets-and-jets-around-the-frozen-wastes-of-sweden/>

![](img/8ce9306855e9a46e12cdfe52ecef2724.png "pulsejet")

一位细心的读者向我们透露了在 [Mobacken Racing](http://mobackenracing.se/) ( [翻译](http://translate.google.com/translate?sl=sv&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fmobackenracing.se%2F&act=url))的家伙，一群致力于在卡丁车和雪地车上安装喷气发动机和火箭发动机的瑞典人。

一种更简单的构造是[脉冲喷气雪橇](http://mobackenracing.se/category/pulsjetsparken/)。脉冲喷射器是极其简单的装置——仅仅是几根不锈钢管焊接在一起，用一个叶片鼓风机启动。脉冲射流的简单性使其能够非常热且非常大声地运行；完美的引擎让对手心中充满对北欧神的恐惧。

脉冲喷气式飞机对约翰逊来说有点太简单了，所以他致力于建造一个喷气涡轮发动机。现在它只是在测试台上，但仍然有一个可怕的推力从那东西出来，正如休息后的视频所示。

在我们看来，最有趣的建造是 1000 牛顿液体燃料火箭发动机。液冷发动机消耗氮氧化物和甲醇，与我们之前见过的液体燃料发动机有着惊人的相似。可悲的是，没有这个引擎被点燃的视频(只有它的照片[绑在卡丁车](http://mobackenracing.se/fordonet-till-raketmotorn-framletat/)上)，但休息后坐下来看其他几个有趣的马力过大的引擎扰乱宁静的森林冬季。

*编辑:* [Linus Nilsson]来信告诉我们，虽然 Mobacken Racing 的伙计们是好朋友，[Linus]，他的兄弟和第三个人(他的话)负责脉冲喷气雪橇。脉冲射流实际上是“有阀的”，不像几根不锈钢管那么简单。脉冲喷射也不是由吹叶机启动的，而是由一个*四千瓦风扇启动的。*【莱纳斯】剧组自称 [Svarthalet racing](http://svarthaletracing.se/) ，你可以在这里查看谷歌翻译。
