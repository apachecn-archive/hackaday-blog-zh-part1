# 迪斯科二十面体新年响起

> 原文：<https://hackaday.com/2011/12/31/disco-icosahedron-rings-in-the-new-year/>

![](img/832dfe2790081bc89af74360348cdf78.png "2012")

今年，当城市工程师在时代广场搭建五彩缤纷的灯球时，adafruit 的[Phil]正忙着设计 X2 时间球，这是一个迪斯科二十面体，非常适合庆祝新年。

这个球是由 20 个丙烯酸三角形用拉链绑成一个二十面体。在每张脸上，六个 [RGB 像素](https://www.adafruit.com/products/322)通过 Arduino 的命令点亮。由于一对 XBees 的帮助，整个项目能够被远程控制。[菲尔]迅速制作了一个[处理草图](https://github.com/adafruit/Adafruit-X2-Time-Ball)以任何我们可以想象的方式控制 LED 球。

虽然为今年的新年晚会制作一个 LED 迪斯科球可能有点晚，但这并不意味着我们不能在一年的其他 364 天里使用它。它非常适合聚会、婚礼和我们每周的灯光开关狂欢。休息后看看时间球的动作视频。
