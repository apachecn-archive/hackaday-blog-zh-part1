# 使用绘图计算器的对等网络

> 原文：<https://hackaday.com/2010/12/16/peer-network-using-graphing-calculators/>

![](img/125179866ac72853181505890c2dafcc.png "calculator-peer-network")

这些计算器联网在一起，能够在多屏游戏场上传递信息和玩游戏。所有这些都要归功于克里斯托弗·米切尔在一个叫做 CalcNet 的软件包上的工作。这个网络软件利用了 TI 计算器的[外壳和 GUI，称为](http://www.cemetech.net/projects/item.php?id=32) [门 CS](http://www.cemetech.net/projects/item.php?id=32) 。为了展示他的网络的高可靠性和高吞吐量，他写了 [NetPong](http://www.ticalc.org/archives/files/fileinfo/430/43098.html) ，这是一个流行游戏的多计算器版本，你可以在休息后观看一个剪辑。

这绝对是一个问‘为什么’的例子。是错误的问题。我们对如何做更感兴趣，这个问题你可以通过阅读他在上面两个链接中提供的白皮书来回答。[克里斯托弗]知道他在做什么，他用他的[面部识别增强现实](http://hackaday.com/2010/08/02/head-up-uses-facial-recognition-and-augmented-reality/)证明了这一点。
