# 旧烟雾报警器的漏水探测器

> 原文:[https://hack aday . com/2011/12/27/漏水探测器来自一个古老的烟雾报警器/](https://hackaday.com/2011/12/27/leaking-water-detector-from-an-old-smoke-alarm/)

![](../Images/0f9025dc7ebe4a156b267a3ae06e7b41.png "leaking-water-detector")

去年飓风横扫新英格兰时，他的地下室被淹了。洪水或漏水的问题是，你可能会发现，直到为时已晚。他决定通过建造自己的漏水检测器来防止将来发生这种情况。这是一个简单的装置，放在他地下室的地板上，如果水开始覆盖地板，就会触发声音警报。

他用一个旧的烟雾探测器来建造；这是一个不错的选择，因为它的声音很大，并且专为长期电池工作而设计。它还有一个测试探测器是否工作的按钮。[Thomas]从烟雾探测器盒中取出 PCB 并将电线焊接到测试按钮触点上。他切开一块海绵，将它挤压在一个 PVC 管接头外壳内。它靠着地板，测试按钮触点的电线穿过海绵。如果海绵吸水，就会接通电路并触发警报。

一些其他的设计特点确实使这成为一个很好的设置。他在聚氯乙烯连接器的底部开了一个缺口，这样水就可以自由流动，并在一根探测线上加了一个开关，这样他就可以在检查损坏情况时切断警报。
