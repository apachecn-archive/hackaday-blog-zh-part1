# 小心赛格威，蒂尔托来了

> 原文:[https://hack aday . com/2011/06/16/watch-out-Segway-here-comes-tilto/](https://hackaday.com/2011/06/16/watch-out-segway-here-comes-tilto/)

![tilto_demo](../Images/d7bb2a26f50f33818bd3dc4c645a32c5.png "tilto_demo")

虽然赛格威在销声匿迹前享受了几年的名声，但[马塞洛·福纳索]希望他的创作能有更多的持久力。受到 Segway 自我平衡能力的启发，他开始思考如何改进这个概念。他认为 Segway 的一个缺点是基础平台太硬，需要用户在设备框架外前后倾斜才能转动。他认为这使得骑行体验不舒服，也有导致骑手摔倒的风险。

他的发明 Tilto 旨在提高 Segway 的转弯能力，同时消除对车把的需求。基于他已经折腾了一段时间的倾斜山地板设计，Tilto 使用加速度计和陀螺仪来保持平衡，很像 Segway。他的目标是在向前和向后行驶时保持车辆平衡，同时允许设备从一侧倾斜到另一侧而不翻倒。这种设计使骑车人保持直立，允许用户像骑自行车一样靠着驾驶车辆。

正如你在下面的视频中看到的，Tilto 工作得非常好，即使是在它的原型形式下。

最后，一个人移动，让我们得到我们的黑帮依靠！
