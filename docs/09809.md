# mine station–为您的《我的世界》世界提供外部天气显示

> 原文:[https://hackaday . com/2012/03/01/mine station-an-external-weather-display-for-your-minecraft-world/](https://hackaday.com/2012/03/01/minestation-an-external-weather-display-for-your-minecraft-world/)

![minestation](../Images/9d6680c1af55eb48816e4caef25394d3.png "minestation")

如果你是《我的世界》的超级粉丝，[radikaldesign] [的人有一些你可能会感兴趣的东西。受他们一些热爱《我的世界》的朋友的启发，他们开发了 mine station——你的《我的世界》游戏的气象站。](http://www.radikaldesig.com/2011/11/14/minestation-a-weather-station-for-minecraft/)

概念很简单。在现实世界中，我们有能力向窗外看，看看外面是什么样子，但我们中的许多人转向数字气象站、气象频道或当地新闻，以获得真正的独家新闻。他们认为《我的世界》的世界应该没有什么不同，所以他们建造了一个 Arduino 盾牌，允许玩家在游戏时看到天气情况。

盾牌包含一个诺基亚 6100 液晶显示屏，显示各种有用的信息。它有一个反映游戏时间的时钟和日历，很容易知道夜晚何时降临。它还不断显示玩家的坐标以及该地区的天气情况。当你在矿井中辛苦工作(忘记时间和天气)时，手头有这些信息有时会非常有用。

你可以在 [Minestation.me](http://minestation.me/) 购买其中一个设备，但设计是完全开放的，所以你可以轻松地构建一个自己的设备，没有太多的麻烦。
