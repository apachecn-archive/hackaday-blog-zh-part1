# 建造一个 1300 流明的自行车灯

> 原文:[https://hack aday . com/2011/11/28/building-a-1300-lumen-bike-light/](https://hackaday.com/2011/11/28/building-a-1300-lumen-bike-light/)

![](../Images/c005d6d369ca1153da023eceae7a601a.png "building-a-1300-lumen-bike-light")

[Brainiac27]不会让孙的缺席阻止他骑自行车。他用自己制作的 1300 流明自行车灯照亮道路毫无困难。

光源是 Cree 的 3-up star。它释放了大量的光，但也产生了相当多的热量，这就是大型散热器的原因。它本来是要和 CPU 一起使用的，但由于使用了粘性导热膏来粘合这两个部分，因此效果很好。

安装支架是定制的，由 1 英寸×1/8 英寸的铝条弯曲而成。[Brainiac27]第一次尝试制作时遇到了长度问题。在第二次尝试中，他从一个过长的片段开始，从中心向外弯曲，只有在弯曲完成后才进行切割。这个支架很容易安装到他的自行车上，电池储存在自行车瓶里，遥控开关(连接到上面项目箱上的插孔)藏在一个刹车罩下面。

这种光的强度几乎是我们另一个最爱的两倍。
