# 乐高书籍扫描仪

> 原文:[https://hackaday.com/2009/12/17/lego-book-scanner/](https://hackaday.com/2009/12/17/lego-book-scanner/)

![](../Images/0bd005ab038e113acb6e51bb10d98625.png "lego-book-scanner")

这是几年前的一个好例子。【Muranushi】造了一个[扫描仪，可以自动扫描一整本书](http://www.geocities.jp/takascience/lego/fabs_en.html)。乐高被用作主要的建筑材料。一本书被放在一个乐高天平上(插图),配有一个平衡物，可以减轻升高和降低书的工作。这本书被放下，一辆乐高车在书上移动来翻页，这本书被举到一个倒置的扫描仪的玻璃上，扫描进一台笔记本电脑。

乐高和成像设备似乎是很好的搭配。这里使用的大部分部件来自乐高技术套装 [8485](http://peeron.com/inv/sets/8485-1) ，一套带有马达和马达控制器的套装，如上图所示，位于电脑后面的地板上。我们在扫描过程中书籍破损后嵌入了一些视频。 <https://www.youtube.com/embed/XRPZm3pyC4o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
