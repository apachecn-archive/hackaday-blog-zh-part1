# 让你的孩子遵守时间表

> 原文:[https://hack aday . com/2010/12/19/keep-your-kids-in-line-a-time-clock/](https://hackaday.com/2010/12/19/keep-your-kids-in-line-with-a-time-clock/)

![](../Images/9ace43dc78a4c17ea26765cc871aff25.png "kid-time-clock")

猫不在的时候，老鼠会玩耍，但如果你用这个时钟，至少你会知道它们什么时候回家。它被称为 Kid-e-log，约翰·鲍克夏尔开发它是为了帮助一个朋友，当他们十几岁的孩子还在工作的时候，他想知道他们放学后的活动。他认为让他们打卡至少会让你知道他们是否按计划直接回家了。给每个人发了一个 RFID 标签(不，他们没有[植入标签](http://hackaday.com/2008/10/15/diy-rfid-implant/))用来记录时间。为了最大限度地减少欺诈，硬件为其实时时钟配备了备用电池，标签读取事件存储在 EEPROM 中，以便在电源周期之间保留。这并不妨碍一些常见的技巧，比如随身携带阅读器，或者和兄弟姐妹一起发送你的标签，但这是一个开始。休息之后看看它的实际效果。
