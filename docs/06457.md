# 里奇分贝编码器

> 原文:[https://hackaday.com/2010/12/17/rich-decibels-kequencer/](https://hackaday.com/2010/12/17/rich-decibels-kequencer/)

![](../Images/f24170891cd8fdee40f36b8042e325d8.png "Seek Whence Her Aussie Later")

当一个简单的概念导致一些鼓舞人心和有趣的事情时，这是非常好的。[Rich db]的 [Kequencer](http://richdecibels.com/blog/?p=51#more-51) 就是一个很好的例子，像许多项目一样开始:“前几天我有了一个想法，我不知道它是好是坏，所以我就做了它来找出答案。”仍是我们可征服的心！

[Rich]从零开始构建了这个圆滑的小音序器，虽然设计一开始可能看起来并不新颖——八个带有开/关开关和音高旋钮的方波振荡器，按顺序播放——但它的妙处在于交互的细微差别和进一步黑客攻击的可能性。通过观看视频，您可以看到如何以非常有趣的方式使用控件来创建和改变可爱的活泼音调模式。交叉渐变后再来看看。

[https://www.youtube.com/embed/ucdcsxbLXME?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ucdcsxbLXME?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T3】

基于尼古拉斯·柯林斯(Nicolas Collins)的工作(就像迄今为止[Rich]博客上的所有项目一样),这种仪器已经成熟，可以进行黑客攻击:电路简单，易于修改，因此添加更复杂(或更多)的模块是轻而易举的事情。总有 [Korg Monotron 原理图](http://hackaday.com/2010/11/10/monotron-openly-monophonic/)给你一些思路。也许在混音中加入一些[数字合成](http://hackaday.com/2008/11/17/auduino-software-synth/)。或者疯狂地在设计中加入一些噪声发生器和包络，以获得一些打击乐合成器的可能性。靠边站，[罗兰](http://en.wikipedia.org/wiki/Roland_TR-808)！

严肃地说，鉴于[Rich 分贝]的工作质量，我们很有兴趣看看他未来会有什么样的成果。如果你还没有，那就去看看【柯林斯】' [](http://www.nicolascollins.com/read.htm) *[手工制作的电子音乐](http://www.nicolascollins.com/read.htm)。*其中很多是对经验丰富的黑客的评论，但充满了优秀的专业知识和音频电子产品的基本构建模块。我们的内部指南[构建合成器](http://hackaday.com/2008/05/01/how-to-make-a-digital-synthesizer/)也是一个很好的起点
