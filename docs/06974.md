# 像超人一样使用原力飞行

> 原文：<https://hackaday.com/2011/03/01/fly-like-superman-using-the-force/>

![infinity_simulator](img/ac3a0e712fd5b223157d3573b706f97b.png "infinity_simulator")

在这一点或那一点上，谁没有一个梦想，仅仅通过思考，你就可以飞翔？[Yehuda Duenyas，又名 XXXY]目前正在伦斯勒理工学院从事一个项目，可以让你做到这一点。

作为被称为“[无限模拟器](http://www.youtube.com/watch?feature=player_embedded&v=yGvsDD50cb8)的论文项目的一部分，他在 RPI 的实验媒体和表演艺术中心构建了一个系统，允许人们使用精心制作的索具系统四处飞行。他的项目允许用户在空中滑行，在墙上行走，或者仅仅通过思想来实现他们的飞行幻想。

EEG 头戴式耳机与其他可穿戴传感器一起放置在用户身上，这些传感器增强了飞行中的人的听觉和视觉体验。只要足够专注，操纵系统就能让人们神魂颠倒，随心所欲地翱翔。对我们来说，这听起来有点像在使用原力时假装超人，然而在 EMPAC 网站上，这个装置被描述为“与视频游戏交叉的真人特技表演”不管怎样，给我们报名吧！

希望我们能在不久的将来看到一些完成项目的视频，但同时继续阅读，看看飞行钻机在行动中的幕后预览。
