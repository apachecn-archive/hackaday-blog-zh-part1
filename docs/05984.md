# 报纸上的音频广告？

> 原文:[https://hackaday.com/2010/09/22/audio-ads-in-newspapers/](https://hackaday.com/2010/09/22/audio-ads-in-newspapers/)

![](../Images/9868f49af5b6bb82fc62523a30de22cd.png "newspaper")

哦，不，让我们希望这个小花招不会流行起来。大众汽车在印度报纸上发布了一则广告，当你打开报纸时会播放一段音频文件。这似乎很像那些当你打开贺卡时会播放一首歌的贺卡。有一个传感器可以检测报纸的开口，可能只是一片塑料或纸张从两个触点之间滑出。这允许电路通电，并播放音频文件。你能想象这有多讨厌吗？尤其是如果你的报纸充斥着这些和[那些电子纸屏幕](http://hackaday.com/2008/10/14/how-to-make-an-e-paper-clock-and-hack-esquire-magazine/)。话又说回来，对于这种东西来说，那个扬声器看起来太大了，可能是一个不错的电子产品，可以放在重复使用的垃圾桶里。休息之后你可以看到报纸的视频。
