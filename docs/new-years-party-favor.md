# 新年聚会的礼物？

> 原文：<https://hackaday.com/2009/12/28/new-years-party-favor/>

![](img/8c90b9330acb13b32f4b5a0c0016c71e.png "pov-pcb")

[Infernoz] [构建了一个 POV 显示](http://www.electoday.com/bbs/viewthread.php?tid=3543)来帮助迎接新年。组件数量少；ATtiny26、DIP 开关、电源开关、CR2032 电池和支架、引脚接头、8 个 led 和一个上拉电阻。该板是单面的，我们看不到任何跳线。他通过在绳子上摆动来移动显示器，但是 PCB 的形状非常适合[安装到风扇上](http://hackaday.com/2009/07/22/ceiling-fan-pov/)。我们喜欢这些[闪闪发光的显示器](http://hackaday.com/2009/06/15/rgbike-pov-2/)，如果你有一些零件，这将是新年前夕的一个很好的派对礼物。休息之后请看视频。
