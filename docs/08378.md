# 气枪无聊了就造个线圈枪！

> 原文:[https://hack aday . com/2011/08/24/when-airsoft-gets-boring-build-a-coil-gun/](https://hackaday.com/2011/08/24/when-airsoft-gets-boring-build-a-coil-gun/)

![airsoft_coilgun](../Images/fed136099a54bf33e168f4dfa14f3830.png "airsoft_coilgun")

在 Hackaday，我们致力于重新利用你不再使用的旧物品。读者[Liquider]来信分享了他的最新创作，[一把由旧气枪改造而成的线圈枪。](http://diytechgadgets.blogspot.com/2011/08/coil-gun.html)

他从手枪上取下一些部件，并在手柄内安装了一个 800 uF/300V 的电容器。枪管下增加了一个小储物格，里面放着他用来驱动电路的 AA 电池。一种改进的重新装填机制使得在发射前将金属射弹扔在线圈的正前方变得很容易。

一旦手枪充好电，安装在扳机后面的开关就会释放雷管，产生一个磁脉冲来加速金属射弹前进。[Liquider]估计线圈产生的动能为 0.1 焦耳，以合理的速度发射子弹。
