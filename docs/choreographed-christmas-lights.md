# 精心设计的圣诞灯

> 原文：<https://hackaday.com/2009/01/23/choreographed-christmas-lights/>

[https://www.youtube.com/embed/2Cj-morKHPQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/2Cj-morKHPQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们已经介绍了几种不同的圣诞彩灯设计方法。最基本的是由扬声器制成的[声控扬声器，然后是](http://hackaday.com/2008/12/02/sound-activated-christmas-lights/)[视差控制扬声器](http://hackaday.com/2008/12/19/parallax-christmas-light-show/)。这一款也使用了视差微控制器，但[prabbit22m]似乎在[结构和外壳](http://www.instructables.com/id/DIY_Christmas_Lights_Set_To_Music_Choreographed_/)上投入了更多努力。这一次所有的灯都是 LED 的，有 18 个插座，可以在一个漂亮、安全的外壳中单独控制。编舞也做得很好。能力视频令人印象深刻，看起来他已经设法做了一些淡入淡出。休息过后你就能看到了。
