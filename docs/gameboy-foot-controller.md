# Gameboy 脚踏控制器

> 原文：<https://hackaday.com/2009/08/31/gameboy-foot-controller/>

[https://www.youtube.com/embed/lQdqudTzyBs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/lQdqudTzyBs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Joey]给我们发了一个他最新版本的 [Gameboy 脚控器](http://filefreakout.com/animalstyle/?page_id=350)的链接。在上面的视频中，你可以看到他在通过 8 位滤波器弹奏吉他时如何使用它来控制背景中的循环。那是老视频，用的是以前的版本。他告诉我们在建造中使用了几台游戏机。有一次，他不得不更换内脏，因为音乐太大声了，它撞倒了他的设备，毁坏了它。当我们听到这些重复片段时，我们不禁感到一丝兴奋，因为周末租用 NES 墨盒的记忆充斥着我们的大脑。他的音乐也不差。越来越多的人支持“芯片音乐”。你可以看到看起来像是一个体面规模的聚会，在休息后欣赏一场带有一点历史课的表演。

[ [这个视频和控制器的原始版本](http://hackaday.com/2008/06/15/game-boy-foot-controller-demo/)是大约一年前发布的，很好的评论者]
