# 发光的鞋褶边——他再也不会踩你的脚趾了

> 原文:[https://hack aday . com/2011/08/09/lighted-shoe-ruffles-hell-never-step-you-toe-again/](https://hackaday.com/2011/08/09/lighted-shoe-ruffles-hell-never-step-you-your-toes-again/)

![](../Images/496a8ee00bf1248aa44476356845653b.png "led-shoe-ruffles")

无论是为了时尚、应急照明，还是仅仅为了彰显个性，[这些发光鞋夹](http://makeprojects.com/Project/LED-Shoe-Clips/1178/1)都是独一无二的鞋类配饰。我们之前见过的[黑掉自动针织机](http://hackaday.com/2010/11/08/make-a-knitting-machine-print-pixel-art/)的【贝基·斯特恩】，处理这个快速点燃的项目。

电子设备很简单，两个发光二极管通过一些导线并联到一个纽扣电池上。电路和 LED 投掷器一样，但是她用的是可缝合的电池盒。褶边是由几圈布料裁剪折叠而成的。我们不太习惯使用这种建筑材料，很高兴听到她关于用打火机融合切割边缘的建议。她还得到一个很好的提示，将一根 LED 引线弯曲成方形，另一根弯曲成圆形，以跟踪极性。在将所有东西缝合在一起并用导线完成电路后，[Becky]添加了一个纸夹，使其易于与任何鞋子搭配使用。事实上，这家伙不必错过乐趣，因为这可以很容易地作为一个布特尼尔加倍。

不要错过[Becky]在休息后嵌入的完整视频。
