# 史上最邪恶的礼物

> 原文:[https://hackaday.com/2011/12/20/the-most-evil-gift-ever/](https://hackaday.com/2011/12/20/the-most-evil-gift-ever/)

![](../Images/39d0bdee33fc3566da5d888a7465f2a6.png "evil")

Hack a Day 论坛的新用户[form]在想，“我可以送朋友什么圣诞礼物，这会很烦人吗？”我们认为他真的在这部电影中一炮而红。这是一个经过修改的电脑扬声器，它将播放“清晰”的音频，直到电源按钮被按下 200 次，光传感器被覆盖。当打开礼物时，房间里会充满不适合儿童、老人或心脏病患者的声音。

该建筑是基于一个旧的电动电脑扬声器。一台旧笔记本电脑上的六节锂离子电池提供电源，一个非常简单的电路在 ATtiny45 的帮助下从 SD 卡上提取声音。

构建的[示意图](http://hackaday.com/wp-content/uploads/2011/12/schematc.png)看起来很简单，就像一个好的构建器一样，[form]包含了源文件和十六进制文件。可悲的是(或者值得庆幸的是)，没有关于插科打诨礼物的视频；这可能是一件好事，因为这似乎是一个失去朋友的好方法。
