# 百万伏吉他震撼全场…为了科学！

> 原文:[https://hack aday . com/2011/06/06/百万伏特吉他摇滚科学之家/](https://hackaday.com/2011/06/06/million-volt-guitar-rocks-the-house-for-science/)

![science_brothers_tesla_hero](../Images/79e6974294f42565b4c5244488bb3e48.png "science_brothers_tesla_hero")

[比尔·波特]和他的朋友[丹·弗利塞克]合作上演了一场名为“科学兄弟”的与科学相关的教育舞台秀，两人试图让学生相信他们的专业领域是更酷的科学。虽然这两者在舞台上是竞争的，但这个项目的主要目标是让孩子们对科学感兴趣，不管是什么专业。

这两人目前自掏腰包资助这个项目，所以他们总是在寻找方法让事情变得有趣，同时还能控制成本。考虑到这一点，[比尔]想出了一个很棒的方法来展示他不久前制作的特斯拉线圈。他最近的教育创作是[一个他称为“特斯拉英雄”的小东西](http://www.billporter.info/tesla-hero-an-awesome-million-volt-guitar/)。

因为他已经有一个固态特斯拉线圈挂在周围，他挖了一个 PS2 吉他英雄控制器，并忙于让两个熟悉。吉他通过光纤隔离板连接到线圈，在他弹奏时弹奏五个音符中的一个。一系列 Arduino 驱动的 LED 灯装饰着吉他，在他演奏时闪烁着各种颜色，正如你在下面的视频中看到的。

这是一个相当酷的项目，我们相信他的观众会留下深刻印象！
