# (遥控机器人)汽车 Vs(太阳能充电)坦克

> 原文：<https://hackaday.com/2009/12/05/remote-controlled-robot-car-vs-solar-charged-tank/>

[https://www.youtube.com/embed/WKFECpQ8asI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/WKFECpQ8asI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Matthias]提交了他的团队的 [iPhone 机器人(汽车)](http://bagira.ringwald.ch/?n=Main.HomePage)项目，以回应昨天的 [iPhone 控制太阳能充电罐](http://hackaday.com/2009/12/03/phone-controlled-solar-charged-tank/)。与坦克不同，iPhone 汽车通过蓝牙 [BTnode](http://btnode.ethz.ch/) 和 [BTstack](http://code.google.com/p/btstack/) 控制，也不是太阳能驱动的。然而，这两个项目都使用 iPhones 和现成的组件。我们特别喜欢它不仅使用了多点触摸输入，还使用了 iPhone 中的加速度计。似乎有些竞争正在加剧，让我们看看谁会胜出。
