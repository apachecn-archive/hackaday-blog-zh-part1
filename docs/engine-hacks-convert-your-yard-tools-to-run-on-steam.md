# 引擎黑客:转换你的庭院工具运行蒸汽！

> 原文：<https://hackaday.com/2011/09/03/engine-hacks-convert-your-yard-tools-to-run-on-steam/>

![Engine Hacks Theme banner](img/3756c09b2fc3afca29ca18ae1f97b9f6.png "Engine Hacks Theme")

当使用你的草坪设备时，你是否厌倦了以高效的方式做事？不要再看了。这里有一些引擎黑客，其中常规内燃机已被修改为工作在蒸汽或压缩空气。令人惊讶的是，要做到这一点，只需拆下汽化器，换上蒸汽管，并调整气门正时。

首先是一台靠蒸汽运转的割草机。这张很棒，因为它看起来很不对劲。用[d quad]自己的话说:“我感到惊讶的是，在这次测试中，除了一个轮子着火之外，没有任何东西失败！”
