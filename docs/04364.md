# 改进的臂式火焰喷射器

> 原文:[https://hack aday . com/2009/11/03/改进型臂装火焰喷射器/](https://hackaday.com/2009/11/03/improved-arm-mounted-flame-thrower/)

![prometheus-flame-thrower](../Images/70b11cccc8f35d07ab62103522b71817.png "prometheus-flame-thrower")【埃弗雷特】再次与普罗米修斯，一个[臂装火焰喷射器](http://www.youtube.com/watch?v=z2oEP3RWppA)战斗。这是该项目的第三代，比我们去年看到的第二代有了一些巨大的进步。我们可以说他的设计已经达到了电影/舞台表演的质量。

该独立系统完全安装在臂上，在肘部后面安装有一个油箱。新版本增加了一个由伺服电机驱动的可调流量阀来调节火焰大小。电弧发生器已被纳入取代打火机从最后一个版本。微控制器测量手腕角度，并负责产生电弧和调节燃料供应。

普罗米修斯是小的，可控的，可怕的。休息之后，观看完整的演示以及一些原型制作过程的视频。这让你想知道这些年来需要多少烧伤膏吗？
