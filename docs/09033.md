# 厨房黑客:如果你有这么大的一个苏-维德，你会做什么？

> 原文:[https://hackaday . com/2011/11/09/kitchen-hacks-what-you-cook-if-you-had-a-sous-vide-this-large/](https://hackaday.com/2011/11/09/kitchen-hacks-what-would-you-cook-if-you-had-a-sous-vide-this-large/)

![](../Images/46f155737059d9745f27b1d1fc1b5ad5.png "very-large-sous-vide")

我们在这一带已经看到不少温控浸入式烹饪装置，叫做“真空烹饪”。但是这个可能有他们所有人加起来的容量！Flickr 用户[RogueGormet]并没有写这个版本，但是他的大型水炉版本照片集说明了一切。

我们打赌捐赠的容器是一个 16 加仑的胸腔冷却器。他把盖子切成两部分，用高密度聚乙烯(那些白色切菜板就是用这种材料制成的)密封绝缘腔。这给他一个安装加热元件的地方，顶部有一个 PID 控制器的盒子。潜水泵保持液体流动，以帮助调节整个烹饪温度。

你在这个里面放了什么？我们马上想到他计划了一个类似烤猪的活动。但也可能是火鸡或其他大块肉。你会用它做什么？

如果你不需要那么大的容量，你可以对你的慢炖锅做一些改变，用于你自己的浸入式烹饪。
