# 声音鼠标控制

> 原文:[https://hackaday.com/2008/10/10/vocal-mouse-control/](https://hackaday.com/2008/10/10/vocal-mouse-control/)

[https://www.youtube.com/embed/PXmccs4GIqI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/PXmccs4GIqI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

绝对迷人。华盛顿大学正在开发一种声控鼠标界面。我们以前见过计算机的语音控制，但它通常是对特定的命令和词语做出反应来执行任务，如打开文件。这个系统使用不同的元音和声音来创建光标运动。你可以在上面的视频中看到同样的系统用来控制机器人手臂。
