# 来吧，让孩子们玩你的数码相机…在一些补充

> 原文:[https://hackaday . com/2011/08/26/go-ahead-let-the-kids-play-with-your-digital-camera-after-some-additions/](https://hackaday.com/2011/08/26/go-ahead-let-the-kids-play-with-your-digital-camera-after-some-additions/)

![](../Images/26b86828e9d8ba8c0b0efa34c349c1d2.png "child-proofing-a-digital-camera")

你会把这个相机扔在人行道上，相信它不会摔坏吗？我们也很难相信，但这正是休息后视频中发生的事情。五颜六色的附件是 Sugru 创造性地定位来帮助保护相机的部件。从我们所看到的，这增加了相当多的减震，让通常微妙的硬件反弹和滚动。毕竟，[这玩意儿是硅胶做的](http://hackaday.com/2009/12/04/sugru-moldable-silicone-adhesive/)。

这看起来不像是要从相机上移除保护，尽管我们已经看到 [Sugru 在过去用于此目的](http://hackaday.com/2010/05/06/building-removable-epoxy-mounting-brackets/)所以这种方法可能是适用的。在这个项目中出现了一个错误，阻止了电池盒被打开，但后来发现你可以剥离相机的材料，所以这不是一个完全永久的转变。

我们敢打赌，最受关注的摄像头组件是液晶屏。我们有一个在没有任何合理原因的情况下破裂了。但是，如果你只是加固这个设备，让它递给你的孩子，谁会在乎液晶显示器坏了呢？如果你必须拍下所有的照片，然后等着用电脑“冲洗”它们，这有点像电影摄影机。

别忘了，如果你手头没有 Sugru，你可以试着自己混合。
