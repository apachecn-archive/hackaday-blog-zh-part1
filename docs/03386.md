# 谷歌服务器透露

> 原文:[https://hackaday.com/2009/04/11/googles-servers-revealed/](https://hackaday.com/2009/04/11/googles-servers-revealed/)

![googleservermedium](../Images/76b15b882fbd922d7d29003ef2b67fd4.png "googleservermedium")

我们经常想知道互联网巨头谷歌用什么样的硬件来处理它的数据。他们最近透露了他们的主要工作是什么。这是一个由 Gigabyte 制造的定制主板，带有两个处理器和八个 RAM 插槽。关于这些的主要兴趣点是，每个服务器和网络设备都有自己的备用电池。这可能会在设备的初始成本中增加一点钱，但显然这是一种更有效的处理功率的方式。请务必点击网站，查看他们使用的集装箱设置。每个集装箱有 1，160 台服务器。他们不是唯一使用这种方法的人。[微软已经在他们的新设备中采用了它](http://news.cnet.com/8301-13860_3-10020902-56.html)，Sun 已经对这些[便携式设备如何应对地震](http://gizmodo.com/gadgets/earthquake-destruction!/sun-blackbox-dont-handle-with-care-268009.php)做了一些广泛的测试。休息之后你可以看到地震测试。
