# 130 百万像素扫描相机

> 原文:[https://hackaday . com/2009/06/09/130 百万像素扫描仪-照相机/](https://hackaday.com/2009/06/09/130-megapixel-scanner-camera/)

[https://www.youtube.com/embed/Pn9ZoaPIHQc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Pn9ZoaPIHQc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这款相机由一台 1200 dpi 的爱普生扫描仪和一个手动对焦的佳能镜头制成，[它可以拍摄 1.3 亿像素的图像](http://www.bouncingredball.com/2009/05/30/japanese-camera-buff-builds-130-megapixel-scanner-camera-for-next-to-nothing/)。这些图片的分辨率为 13，068 x 10，173，非常清晰。你可以在他的 [flickr set](http://www.flickr.com/photos/82772083@N00/) 中看到一些例子。虽然看起来不像是全尺寸的原件。[如果你想打造自己的车，这里有一个好的开始](http://hackaday.com/2004/09/21/building-a-megapixel-digital-camera-from-a-flatbed-scanner/)。请记住，这是 2004 年的。
