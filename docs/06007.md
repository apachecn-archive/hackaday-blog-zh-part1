# PS3 六轴控制器现在可以利用漏洞了

> 原文:[https://hack aday . com/2010/09/26/PS3-six axis-controller-now-houses-exploit/](https://hackaday.com/2010/09/26/ps3-sixaxis-controller-now-houses-exploit/)

![](../Images/3a01be72080b4cd6da79bef4cf28aded.png "sixaxis-psgroove")

[Hasuky]贴出了[将 PlayStation 3 六轴控制器变成 PSGroove 漏洞利用设备的指南](http://www.elotrolado.net/hilo_guia-mod-picgroove-sixaxis-ps3_1492833) ( [译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=es&tl=en&u=http://www.elotrolado.net/hilo_guia-mod-picgroove-sixaxis-ps3_1492833))。不幸的是，你必须打开控制器，添加一些部件才能让它工作。黑客需要一个 PIC 18F2550(我们已经看到过[用作独立](http://hackaday.com/2010/09/15/psgroove-on-a-pic-microcontroller/) [PSGroove](http://hackaday.com/2010/09/15/psgroove-on-a-pic-microcontroller/) [设备](http://hackaday.com/2010/09/15/psgroove-on-a-pic-microcontroller/)的芯片)，一个晶体，以及各种电阻和电容来连接控制器的 PCB。从那里，您连接控制器和游戏控制台之间的 USB 电缆，并使用漏洞启动。

[通过 [DCEmu](http://www.dcemu.co.uk/-guide-pic18f2550-psgroopic-ps3-sixaxis-controller-mod-329787.html) 感谢克雷格]
