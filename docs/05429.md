# 踏脚转盘合成器

> 原文:[https://hackaday.com/2010/06/01/stomp-box-synthesizer/](https://hackaday.com/2010/06/01/stomp-box-synthesizer/)

![](../Images/87285532827791e64122c5ca5d23fa44.png "stomp-box-synthesizer")

[布莱恩·麦克纳玛拉]将他的吉他踏板的输出反馈到它的输入，创造了一个循环合成器。他从自己制作的效果踏板开始，但现在我们认为他已经完成了电子[踏脚转盘](http://en.wikipedia.org/wiki/Stomp_box)。休息之后请看视频中的结果。现在，他需要把旋钮做得对脚友好，这样他就可以在弹吉他的时候摆弄它了。
