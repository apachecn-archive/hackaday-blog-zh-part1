# 红外玻璃液位检测

> 原文：<https://hackaday.com/2008/11/07/ir-glass-level-detection/>

[https://www.youtube.com/embed/DG3_2lvoL8A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/DG3_2lvoL8A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[李炯楷]的同事[保罗·迪茨]利用交互式表格做了一些有趣的工作。他专门研究了如何确定酒杯有多满。在上面的视频中，他使用的是[微软的 Surface](http://www.mahalo.com/Surface) ，但这种技术应该适用于任何基于红外相机的多点触摸桌子。确定饮料的水平需要定制的玻璃器皿，里面有一个小棱镜。当液面高于棱镜时，光通过，但当液面低于顶部时，它会将更多的红外光反射回桌面。利用这些信息，餐馆员工可以更有效地提供饮料。

[Paul]在另一个项目中使用了 [RFID 和电容感应](http://www.merl.com/projects/iGlassware/ "MERL – iGlassware")来达到类似的效果。
