# Hack 从 Kindle“特价”硬件上移除广告

> 原文：<https://hackaday.com/2011/12/20/hack-removes-ads-from-kindle-special-offers-hardware/>

我们认为用不了多久就会有人想出如何从亚马逊 Kindle 硬件的“特价”版本中移除广告。有两件事让我们明白了这一点，让代码很容易以 root 身份运行的巨大缺陷，以及形成[的 MP3 标签使得解锁设备](http://hackaday.com/2011/12/14/how-the-kindle-touch-jailbreak-was-discovered/)成为可能。

[Pat Hartl]对*nix shell 了如指掌，所以一旦他获得了设备的 SSH 访问权限，他就开始搜索构成特价功能的广告图片。他在几个不同的地方找到了它们，在另一个位置备份文件，然后用一些简单的命令删除它们。他甚至将这一过程整合到一个类似越狱包的一键式安装程序中。这让我们想知道亚马逊是否有办法判断你的设备是否不再下载这些优惠的内容？

冒着听起来像说教的风险，亚马逊确实一次性收费提供了这种不带广告的硬件。规避不显眼的广告可能会导致未来更高的硬件价格，[Pat]提到了这一点。他通过这次黑客攻击展示了亚马逊的安全漏洞，而攻击他们是一种强有力的方式。

Hackaday 本身是广告支持的。我们的广告不使用声音、弹出窗口或闪烁的视频效果。为了表示你的支持，请记得关掉我们网站的广告栏。谢谢！
