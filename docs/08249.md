# 用 555 个芯片建造一台计算机

> 原文：<https://hackaday.com/2011/08/05/building-a-computer-out-of-555-chips/>

[![](img/d6cb3bb6cee21b1814b66bf259a31a6b.png "memory")](http://hackaday.com/wp-content/uploads/2011/08/memory.png)

今年早些时候，[M. Eric Carr]为 555 竞赛提出了一个有趣的构建，我们非常肯定，如果它完成的话，它会把复杂类别的[获胜者踢下宝座。虽然晚了几个月，但我们很高兴在 Hack A Day 上至少展示了他基于 555 的计算机的一部分。](http://www.vk2zay.net/article/258)

[M. Eric Carr]开始用 555 实现[布尔逻辑。在建造了一个](http://www.paleotechnologist.net/?p=530)[通用门](http://en.wikipedia.org/wiki/NAND_logic)后，他开始在一个 555 中放入一位内存。这个设计使用 555 作为[闩锁](http://www.paleotechnologist.net/wp-content/uploads/2011/02/555_SR_latch.png)，这是 555 最疯狂的不合规格的用法之一。虽然在 5 个大面包板上安装 24 位内存看起来像是一种浪费，但这与我们见过的其他计算机构建的[平方英尺](http://web.cecs.pdx.edu/~harry/Relay/)或[英亩](http://www.youtube.com/watch?v=7sNge0Ywz-M)相差甚远。

在基于 555 的存储器上的帖子中，[M. Eric Carr]说他的计算机的完整设计可能在今年夏天发布并运行。已经是八月了，所以我们希望他还在进行他的设计。看看下面这段记忆的视频。
