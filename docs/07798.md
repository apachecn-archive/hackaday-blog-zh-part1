# 磁力又让橡皮泥变得有趣了

> 原文:[https://hackaday . com/2011/06/10/magnetic-makes-silly-putty-fun-again/](https://hackaday.com/2011/06/10/magnetism-makes-silly-putty-fun-again/)

![](../Images/f97e4e105bef817e735645405f75e1b2.png "magnetic-silly-putty")

上面的图片是一个视频剪辑的截屏，其中黑色软泥吞噬了稀土磁体。它实际上是一团[橡皮泥，被轻微改变以增加磁性](http://www.instructables.com/id/magnetic-silly-putty)。[Mikeasaurus]从一家艺术用品店拿了一些氧化铁粉末，戴上手套和防尘口罩，同时将它按摩到[硅树脂聚合物](http://en.wikipedia.org/wiki/Silly_putty)中。如果你将这两种材料正确混合，你最终会得到一种流动的物质，当它暴露在磁场中时会神秘地表现出来。

休息后看看视频，看看[Mikeasaurus]能做的一些技巧。油灰看起来真的有自己的生命。它将拉伸一段显著的距离以接近磁体(无定形拉伸)。如果让它与其中一个接触，它会完全吞没它，然后形成一个球体。

现在，有什么方法可以利用电磁场来建造一个变形机器人吗？
