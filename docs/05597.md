# 机器人随处可见

> 原文：<https://hackaday.com/2010/07/04/robotseverywhere/>

[https://www.youtube.com/embed/WWh6MFUCZ-U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/WWh6MFUCZ-U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Spiritplumber]一直在向我们发送他正在进行的一个名为 RobotsEverywhere 的有前途的项目的更新。我们不完全确定这个名字是否涵盖了项目本身，或者仅仅是他的销售机器人团队的名字(T2 给了我们原理图和源代码)。不管怎样，他们有一些很酷的东西。许多机器人真正酷的部分是，它们使用[安卓手机作为大脑](http://www.robots-everywhere.com/re_site/index.php?page=technologies)，它们在陆地、空中和水上使用相同的系统。有大量的信息散布在网站和维基上，花几分钟时间四处看看。
