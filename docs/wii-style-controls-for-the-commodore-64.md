# Wii 风格的 Commodore 64 控制器

> 原文：<https://hackaday.com/2008/07/05/wii-style-controls-for-the-commodore-64/>

[https://www.youtube.com/embed/C1WOoPV_7HU?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/C1WOoPV_7HU?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
【杰夫】一直在摆弄[视差的推进器芯片](http://www.parallax.com/propeller/)。他用它来将一个 NES 控制器改装到 Commodore 64 上。然而，在最新的迭代中，他在一个杆球棒的末端添加了一个 Memsic 2125 双轴加速度计，并使用它来提供 [Wii 风格的控制器输入](http://learn1thing.blogspot.com/2008/07/cbat.html)。上面的视频显示他的儿子用它玩街头运动棒球。
