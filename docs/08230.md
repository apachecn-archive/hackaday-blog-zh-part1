# 只有失败者才会用手机发短信——这家伙随身带着他自己的电传打字机

> 原文:[https://hackaday . com/2011/08/02/only-losers-text-message-on-cells-this-guy-carry-your-own-teletype-for the/](https://hackaday.com/2011/08/02/only-losers-text-message-on-cellphones-this-guy-carries-his-own-teletype-for-that/)

是的，那是一个 SMS 短信设备。[Mdziewie]认为在普通手机上发短信太无聊，决定为自己建一个老派的短信网关。这里有[一个翻译的链接](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http%3A%2F%2Fwww.elektroda.pl%2Frtvforum%2Fviewtopic.php%3Ft%3D2037103%26sid%3D7c2fd44f621d5264dbf3bcc3a250aebe)，但是论坛帖子的格式被机器翻译搞砸了。

他正在使用的设备是 1963 年推向市场的 ASR-33 电传打字机。它通过 ARM 微控制器 STM32F103 连接到 GSM 调制解调器。这种芯片，加上一些电子元件，让[Mdziewie]设计出一种不需要对古老硬件进行改动的界面。上面链接的论坛帖子包含了这段收发文字的视频。当它敲打纸张的时候，声音大得惊人，而且似乎像预期的那样工作。

如果你渴望有一台自己的，但没有半个世纪的旧设备，还有希望。给自己找一台打字机，把它变成一台电传打字机。
