# 垃圾堆投影仪

> 原文:[https://hackaday.com/2010/03/29/trash-heap-projector/](https://hackaday.com/2010/03/29/trash-heap-projector/)

[https://www.youtube.com/embed/S5jymxi62RI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/S5jymxi62RI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

作为黑客，有时我们只想一起黑东西，而不是设计它。这台[投影仪就是一个很好的例子](http://www.instructables.com/id/homemade-projector-cornflake-box/)。主要由硬纸板和胶带制成([或者鸭子胶带，如果你喜欢的话](http://en.wikipedia.org/wiki/Duct_tape))。他拿起一个 12v 的 LED 阵列，一个便宜的菲涅耳透镜，一个来自“备用监视器”的 LCD 和一个来自放大镜的聚焦透镜。当然，[我们见过更好的](http://hackaday.com/2010/03/04/diy-projector-collection/)，好得多。但是看一晚上狂热的电线缠绕和缠绕总是令人愉快的。它在视频中可能看起来很暗，也可能很暗，但请记住，它们亲自出现或用数码相机在夜间拍摄时显得更亮是很常见的。
