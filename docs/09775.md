# 使用 Gimp 制作任何 3D 照片

> 原文:[https://hack aday . com/2012/02/27/make-any-photo-3d-using-the-gimp/](https://hackaday.com/2012/02/27/make-any-photo-3d-using-the-gimp/)

![](../Images/d8a0704540da3ebcaa51f9ae6671a892.png "cross-eyed-3D-from-the-gimp")

把你的脸靠近屏幕，眼睛交叉，直到上面的两个图像变成一个。你可能需要调整下巴的倾斜度来实现这一点，但当它们合在一起时，你会看到(约翰·列侬的)3D 效果。这是使用 Gimp 的 3D 渲染脚本制作的。

这个过程不是完全自动的，但它不会花太长时间来掩盖不同深度层的轮廓。该脚本从图像中创建了三个不同的图层。其中一个是颜色编码的深度图，它使用自定义的色板来选择每个项目的距离。如果你把背景涂成深蓝色，它将在离观看者的对眼视角最远的地方被处理，黄色是最近的。

[Don]在他的文章中提到了并行输出和对眼输出。我们理解对眼版本，但只是猜测平行版本将用于立体观看器，在两个图像之间放置一个分区，以便每只眼睛看到不同的帧。你知道，[就像一个视图大师](http://hackaday.com/2010/08/24/digital-view-master/)。
