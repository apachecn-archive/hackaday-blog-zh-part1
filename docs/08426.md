# 无线光纤情绪照明

> 原文:[https://hack aday . com/2011/08/26/wireless-fiber-optic-mood-lighting/](https://hackaday.com/2011/08/26/wireless-fiber-optic-mood-lighting/)

![ceiling_mounted_starfield](../Images/02396a85f8011ac2c452ea1789f6fd0d.png "ceiling_mounted_starfield")

仰望夜空，看到远处一排闪亮的星星，有种令人平静的感觉。[Marou]是一个观星迷，但有时条件不是最佳的，[所以他决定把星星带进去](http://www.open-electronics.org/radio-controlled-fiber-optic-ceiling-light-with-arduino/)。

他的想法是建造一盏不会让房间充满阳光的吸顶灯，而是一盏再现夜空宁静闪烁的灯。他用深色布料覆盖了一张木桌，并在表面钻了许多小洞。他在桌子上的孔里装了两束光纤，因为一束不能完全覆盖整个东西。

为了点亮电缆，他建造了一对 4-LED 照明器，其中包含红色、蓝色、绿色和白色 LED。每个光源都通过 Arduino 控制，Arduino 通过[Marou 的]红外遥控器控制其方向。

虽然这个想法并不新鲜，但实现起来却很酷。起初，我们期待一盏小灯，但将整张桌子固定在天花板上作为照明板，这绝对是我们从未见过的。

如果你想在你自己的客厅里建造类似的东西，[Marou 的] Arduino 代码是免费的。
