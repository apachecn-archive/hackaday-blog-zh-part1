# 彩色烟火

> 原文:[https://hackaday.com/2009/08/16/colored-pyrotechnics/](https://hackaday.com/2009/08/16/colored-pyrotechnics/)

[https://www.youtube.com/embed/xCDVypW_zB0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/xCDVypW_zB0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

来自 Inventgeek.com 的定期提交者[Jared Bouck]给我们发来了这个很酷的项目。他想做一个火球大炮，但不想满足于普通的老式火球。他没有使用普通的丙烷系统，而是建造了一个基于酒精的系统，这样他就有了一个“白板”来开始。然后他用一些氯化铜来获得想要的绿色和蓝色。我们看到了这么多的火焰，为什么我们没有看到更多的彩色火焰呢？休息之后，请观看概述视频。
