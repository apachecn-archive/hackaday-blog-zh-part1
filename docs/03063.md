# ui Remote:IPhone 的通用远程应用程序

> 原文：<https://hackaday.com/2009/01/07/uiremote-universal-remote-application-for-iphone/>

[https://www.youtube.com/embed/E-SlcjmqeYw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/E-SlcjmqeYw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当多伦多大学的几名学生厌倦了到处放着多个遥控器时，他们决定做点什么。他们解决这个问题的方式是 [UIRemote](http://uiremote.wordpress.com/) ，这是一款适用于 [iPhone](http://hackaday.com/tag/iphone/) 的通用远程应用。由于使用了插入手机耳机端口的定制红外适配器，该应用程序允许 iPhone 控制通常由红外遥控器控制的任何东西。这是一种类似于我们 2004 年的 [iPod 遥控器](http://hackaday.com/2004/09/13/make-an-ipod-a-universal-remote-control/)的技术。虽然 UIRemote 应用程序和适配器仍处于测试阶段，但学生们希望在接下来的两个月内同时发布这两个应用程序。
