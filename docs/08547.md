# 更新:安卓版蓝牙超级任天堂控制器

> 原文:[https://hackaday . com/2011/09/11/update-bluetooth-super-Nintendo-controller-for-Android/](https://hackaday.com/2011/09/11/update-bluetooth-super-nintendo-controller-for-android/)

![](../Images/99e17d603bd7a77c30c82bb3b329ab50.png "update-super-nintendo-bluetooth-android")

[Rich 的]最新版本的[一个蓝牙连接的超级任天堂控制器](http://www.youtube.com/watch?v=bp8Oon93Ckc)将已经很好的东西变得更好。我们在五月份看到了[的原始版本](http://hackaday.com/2011/05/17/bluetooth-super-nintendo-controller-for-android-gaming/)，它内置在一个山寨控制器中——【里奇】提到他因为损坏复古设备而受到很多抨击，他想澄清事实。这两种控制器最显著的区别是电池现在装在盒子里。他改用 850 毫安的锂离子电池，比四节 AA 电池占用的空间小得多。有一个集成充电器，通过迷你 USB 插孔连接，如上图所示。烦扰版本的控制器可以用来玩模拟器，而不需要你的 Android 设备。

不幸的是，你不能看到箱子里面。我们可以告诉你的是，该版本不再使用 BlueSMiRF 板，而是选择了更便宜的蓝牙解决方案。我们还知道该版本仍然兼容 Arduino，运行引导程序的是 ATmega328 芯片。细节的缺乏是因为[Rich]正在努力以限量的预购来销售控制器。休息之后，看看他对新版本的视频介绍。
