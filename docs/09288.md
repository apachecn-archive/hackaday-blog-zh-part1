# 一半是键盘，一半是吉他，完全是激进的鼓手！

> 原文:[https://hackaday . com/2011/12/13/半键盘半吉他-完全彻底-the-tabstrummer/](https://hackaday.com/2011/12/13/half-keyboard-half-guitar-totally-radical-the-tabstrummer/)

[![](../Images/b84995e565345c1ed21aa6937012ccdf.png "tabstrummer")T2】](http://hackaday.com/2011/12/13/half-keyboard-half-guitar-totally-radical-the-tabstrummer/tabstrummer/)

虽然基于符号谱的音乐可能会让受过“适当”训练的音乐家无休止地烦恼，但它给了许多音乐家和音乐爱好者第一次接触吉他世界的机会。[【Tabstrummer】](http://www.tabstrummer.com/Tabstrummer.php "Tabstrummer")将此提升到一个全新的水平，允许和弦被编程到该乐器中并回放。一旦预编好的和弦被设定，指挥弦就会被拨动，让和弦进行。

该设备基于 Atmel 微控制器，具有 MIDI 输出和音频输出插孔。除了有趣的电子硬件之外，外壳似乎建造得很好，似乎是丙烯酸或聚碳酸酯机身。虽然不是完全一样的东西，但可能从[【key tar】](http://en.wikipedia.org/wiki/Keytar)中获得了一些影响。它的全盛时期可能已经过去，但不会被遗忘。

查看下面的视频，在[Tabstrummer]上播放圣诞主题的果酱，或者查看他们的[视频页面](http://www.tabstrummer.com/Tips.html "Tabstrummer videos")以获得更多歌曲。这个“黑客”被认为是一个商业产品，所以发明者很乐意听到你的反馈！
