# 1480W 滑板车电机保证头部不受伤

> 原文：<https://hackaday.com/2009/10/18/1480w-scooter-motor-guarantees-head-trauma/>

[杰罗姆]一直在对他从一个朋友那里得到的电动踏板车进行一些改进。他订购了一台大功率无刷电机和一些锂电池。他的系统使用皮带驱动，在 33 伏电压下，它可以达到每小时 25 英里。

当马达第一次启动时，他遇到了扭矩过大的问题。这导致了无意识的车轮打滑，如果你不是试图抓住滑板车的人，这听起来真的很酷。[Jerome]使用 Arduino 来控制系统，因此他内置了逐渐提高电机速度的功能，还增加了通过远程控制来控制速度的功能。你应该注意到，在休息后的视频中，[杰罗姆]正在测试他的无头盔构建。

所以，我们花了一生的时间和无数的金钱来用知识填充我们的头脑。这是一项不稳定的投资，因为一个很小的碰撞就可能破坏所有的数据，并终结曾经辉煌的认知能力。如果你足够聪明，能造出时速 25 英里的踏板车，那就聪明点，骑着它的时候戴上头盔吧！
