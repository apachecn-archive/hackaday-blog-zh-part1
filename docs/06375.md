# 蒸汽动力滚动

> 原文：<https://hackaday.com/2010/12/02/steam-powered-rickrolling/>

这是一台蒸汽动力的电唱机。太棒了。但是等等，这还不是全部。休息后看大约两分钟半的视频，你会意识到这也是[一个 Rickroll](http://knowyourmeme.com/memes/rickroll) 。不，你不会被引诱去点击里克·阿斯特利的音乐视频，转盘上播放的唱片是“永远不会放弃你”的拷贝；各种牛逼。

这一切都始于一台由不锈钢螺栓和黄铜圆筒加工而成的蒸汽机。在[建造锅炉](http://www.asciimation.co.nz/bb/2010/10/10/a-boiler-and-throttle-for-nigel)之前，使用压缩空气对其进行了测试。但是没有目的的蒸汽机是什么？使用蒸汽机作为转盘电机的问题是速度控制。这就是我们转向现代技术的地方，使用 Arduino 来测量转速，并使用伺服电机调整蒸汽机。

建筑商评论说这听起来很可怕，但考虑到它是蒸汽动力的，我们认为这听起来还不错。
