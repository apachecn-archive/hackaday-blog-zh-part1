# Piecax 的恶作剧鬼重新发明了敲块

> 原文：<https://hackaday.com/2009/10/27/piecax-the-poltergeist-reinvents-the-knock-block/>

![piecax-the-poltergeist](img/a6e7b2cb38985f9549e4172c9f8df8dd.png "piecax-the-poltergeist")

作为万圣节准备的一部分，他建造了一个闹鬼的盒子。这跟随着我们在本月早些时候看到的的[敲块的脚步，但是做了一些硬件上的改变。他用一个 DC 马达取代了螺线管，这个马达转动一只手臂来敲门。他通过使用工艺品商店的软木盒子作为外壳，避免了任何 CNC 工作。对于控制电路，他使用了一个 8 引脚 PICAXE 微控制器，通过压电蜂鸣器“倾听”敲击盒子的声音。它会模仿敲门声，如果听到正确的组合](http://hackaday.com/2009/10/02/clock-knock-block-full-of-puns/)[，就会播放亚当斯家族的主题曲](http://www.youtube.com/watch?v=vVDJvrBFDDc)。这台[没用的机器](http://hackaday.com/2009/09/19/show-us-your-most-useless-machine/)将会成为一个很棒的办公室谈资，这种简化的设计比敲块更容易制造。休息之后看它表演。
