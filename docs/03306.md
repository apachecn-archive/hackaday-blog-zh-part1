# 超鼠标修改

> 原文：<https://hackaday.com/2009/03/16/ultra-mouse-modification/>

![mouse](img/6ef3af0603ed4d465d573a66f6c5adcf.png "mouse")

当带着笔记本电脑旅行时，我们经常会发现我们必须连接的外围设备列表会变得很长。尤其是如果是没有内置无线功能的旧笔记本电脑。【曙光】已经采取措施整合他的一些外设(需要注册)，将一个 USB 集线器和一个无线网卡塞进他的鼠标。

他从拆卸 USB 集线器开始。在将板子放入鼠标看看是否合适后，他意识到他必须缩小尺寸。为此，他拆除了 USB 插槽本身。这迫使他将东西直接连接到电路板上，但节省了大量空间。接下来，他从 USB 无线适配器中取出内脏并连接起来。然后，鼠标中的装饰性 LED 被移到无线卡上。通过这种方式，他可以看到网络流量的闪烁。在这一点上，这就是他添加的所有内容，尽管还有足够的空间来放置其他项目。他指出，他可能会增加一些存储空间或 CF 阅读器。他遇到的唯一问题是，他的鼠标在使用过程中容易发热。你会在里面放什么外设？

更新:[曙光]让我们知道他实际上并没有把 LED 从原来的位置移走。他从无线适配器连接了一根电线，当交通通过时，它会导致 LED 短路。另外，休息之后你可以看到一段视频。
