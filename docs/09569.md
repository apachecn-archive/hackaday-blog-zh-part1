# 转变为控制办公室行人交通的停车灯

> 原文:[https://hack aday . com/2012/01/25/stop-light-converted-to-control-office-foot-traffic/](https://hackaday.com/2012/01/25/stop-light-converted-to-control-office-foot-traffic/)

![traffic-light](../Images/0534d9fa312576131dde175f6dbfb055.png "traffic-light")

当[保罗·雷亚]开始为他现在的雇主工作时，他被他驻扎的“工程阁楼”入口附近一个闲置的交通灯所吸引。他向自己承诺，总有一天他会让它工作起来，但几年过去了，他才有机会近距离观察它。

去年感恩节周末，他把灯带回家，并开始在里面挖来挖去，看看东西是如何连接的。事实证明，这盏灯是一个非常简单的装置，尽管他发现它靠电源电压运行，这是[保罗]不太想摆弄的东西。他把交通灯的灯泡换成了一些低压型号，他可以很容易地用 12v 壁式电源供电。

[Paul]然后在灯具上添加了一个 Arduino 和 PIR 传感器，以便在有人离开工程阁楼时进行检测，并警告那些正在进来的人。他说人们并不真的很注意光线，尽管他对结果很满意。
