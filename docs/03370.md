# 幼儿电脑遥控器

> 原文:[https://hackaday.com/2009/04/06/toddler-computer-remote/](https://hackaday.com/2009/04/06/toddler-computer-remote/)

[https://www.youtube.com/embed/ACbLjz-c4LE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ACbLjz-c4LE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Killerdark]为他蹒跚学步的孩子制作了一个简单的遥控器来控制电脑上的视频。他拆除了一个 USB 数字键盘，建造了一个新的外壳，上面清楚地标注了必要的按钮，并在软件中映射这些按钮。他本来可以用更大的彩色编码按钮做得更好，但实际上，它看起来工作得很好。这让我们想起了 2006 年的[巨型 iPod 遥控器](http://hackaday.com/2006/05/12/giant-ipod-remote-for-the-visually-impaired/)。干得好[杀手标记]
