# 在真实乐器上演奏 Dubstep

> 原文:[https://hack aday . com/2011/10/07/playing-dubstep-on-real-instruments/](https://hackaday.com/2011/10/07/playing-dubstep-on-real-instruments/)

![](../Images/4c10c6d2d0735521d491b23dade040d2.png "header")

当我们第一次看到 Pinn Panelle 乐队的史奇雷克斯*恐怖怪兽和可爱精灵*T3 的[现场封面时，我们不得不知道一个乐队如何能够在真实的乐器上演奏现场 dubstep。我们给乐队发了邮件，他们透露了如何在现场环境中处理乐器的细节。](http://www.youtube.com/watch?v=ZuunY8BTqNs)

首先是贝斯手内森·纳瓦罗。他右手拇指上戴着一只源音频热手。这个小盒子是一个双轴加速度计，它使用射频频率与他的踏板进行通信。通过热手，他可以控制[热手踏板](http://www.sourceaudio.net/products/hothand/)上的两个参数。信封效果是[牛逼](http://www.youtube.com/watch?v=GVGmZCAuq6g)，但值得注意的是【内森】是由源音频赞助的。我们认为将一个 [Wii MotionPlus](http://hackaday.com/2009/06/23/wii-motionplus-arduino/) 和微控制器塞进一个腕带会相对容易。与计算机和 MIDI 接口相结合，家酿解决方案将做同样的事情。

[Derek Song]是吉他手，他在音乐生涯的大部分时间里都在使用多重效果他有一个小的[蓝牙键盘和触摸板](http://hackaday.com/wp-content/uploads/2011/10/bluetooth.jpg)安装在他的吉他前面，可以控制踏板上的几乎所有东西。蓝牙控制器向[德里克]的电脑发送命令，电脑向他的踏板输出 [MIDI CC 消息](http://www.midi.org/techspecs/midimessages.php#3)。

在工作室里，Pinn Panelle 的封面会令人印象深刻。现场演奏的事实打开了一个乐队可以实时演奏的大门。如果你有踏板或电子音乐制作，[将它发送到提示行](http://hackaday.com/contact-hack-a-day/)。此外，休息后请观看视频，更好地演示[Derek]的蓝牙设置。
