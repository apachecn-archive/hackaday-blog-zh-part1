# 双核 128 精英型 4 位 CPU

> 原文:[https://hackaday.com/2010/06/08/duo-128-elite-4-bit-cpu/](https://hackaday.com/2010/06/08/duo-128-elite-4-bit-cpu/)

![](../Images/4a9c58adc038a97cc98fd2ea73f7cc0a.png "What a weird duck game. ")

我们不知道第一次被举报时，我们是如何错过[杰克·艾森曼的] [4 位 TTL CPU](http://web.mac.com/teisenmann/iWeb/elitepage/menu.html) 的，但我们很高兴它再次被发送给我们来展示它。

41 个不同的 IC(主要是 TTL)组成了 DUO 128 Elite。虽然[的架构](http://web.mac.com/teisenmann/iWeb/elitepage/architecture.html)与我们之前看到的略有不同，但使用“nyckles”，DUO 128 Elite 仍然可以完美工作。捕捉一些例子程序的视频，包括 pong，在 divide 之后。
