# 皮卡丘电路弯曲护目镜

> 原文：<https://hackaday.com/2009/06/02/pikachu-circuit-bent-goggles/>

![pikachu_goggle1 (Custom)](img/c457cdf8df6f7fe7770a2c269116f4ac.png "pikachu_goggle1 (Custom)")

这些漂亮的护目镜实际上是一种仪器。皮卡丘玩偶的内脏被展开并安装在护目镜上。控制器是一个外盒，允许您对音高和样本部分进行各种更改。休息之后你可以看到它的视频。我们真的不觉得这是伟大的音乐，但觉得看这个家伙相当有趣。
