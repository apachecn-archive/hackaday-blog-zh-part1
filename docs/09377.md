# 弹药罐]

> 原文:[https://hackaday.com/2011/12/26/ammo-tin/](https://hackaday.com/2011/12/26/ammo-tin/)

[![](../Images/37fed5017c24f51aa155406d14dff26d.png "ammo")T2】](http://hackaday.com/wp-content/uploads/2011/12/ammo.jpg)

弹药板条箱电脑从虚幻竞技场登陆派对开始就有了，但是这个比那个更早；[西蒙]把一个苹果 II 放进一个弹药箱。这是一个合适的时代错误的建造，来自建造[塔迪斯 MAME 内阁](http://hackaday.com/2011/07/01/a-mame-cabinet-fit-for-a-doctor-who/)的同一个人。

谢天谢地，[西蒙]没有为这个版本拆苹果 IIc。这个项目需要一个基本的赛扬主板来运行 AppleWin 模拟器。西蒙没有把所有的部件都塞进盒子里就此收工，而是做对了，为电脑制作了一个[非常漂亮的框架](http://www.asciimation.co.nz/bb/wpg2?g2_itemId=2031&g2_GALLERYSID=TMP_SESSION_ID_DI_NOISSES_PMT)。有了一个软弱的中央处理器，没有扩展卡，弹药罐可以运行非常冷静，而不需要消耗大量的电力。

我们真的不得不称赞[西蒙]这个建筑。这个建筑的金属工作看起来很棒(对于正在重建奥斯汀 7 号的人来说应该如此)，我们必须尊重对苹果 II 的热爱。现在所需要的就是 USB 接口上的真正的[苹果 II 键盘](http://hackaday.com/2011/06/06/apple-converted-into-usb-keyboard/)。
