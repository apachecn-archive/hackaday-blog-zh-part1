# 在冬天的几个月里，热泥土让你的植物保持健康

> 原文:[https://hack aday . com/2012/03/19/hot-dirt-keep-your-plants-happy-in-the-winter-months/](https://hackaday.com/2012/03/19/hot-dirt-keeps-your-plants-happy-in-the-winter-months/)

[克雷格]去年冬天试图加热他的温室，但实际上只是设法推动他的公用事业账单的极限。这一次，他采取了不同的方法，建立了一个系统来加热种植植物的土壤。

这个系统的核心是这个装有植物的盒子。它的底部衬有加热带，可以加热一层污垢。植物被放在花盆里，但是因为它们被泥土包围，这并不能真正抑制土壤的变暖特性。控制器会考虑盒子内部的温度以及温室中的环境温度。当天气有点太冷时，控制器会关闭盖子，盖子上覆盖着半透明的塑料。这确保了植物周围的温度不会低于大约 41.5 华氏度。

这确实减轻了你在冬天照顾植物的工作量。每天多次的访问可以限制为每天一次或两次。现在他只需要扩展这个来调节温室本身的温度和湿度，就像另一个黑客一样。
