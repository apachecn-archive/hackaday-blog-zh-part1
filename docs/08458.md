# 一个疯狂的弹弓手拿着 220 磅重的钢珠炮再次尝试

老好人。

那个家伙就是不停止制造疯狂的弹弓。如果他不是在[磨练他的弯刀投掷技能](http://hackaday.com/2011/03/30/stupid-expert-builds-a-machete-slingshot-for-the-impending-zombie-apocalypse/)为僵尸启示录做准备，他会用巨大的钢珠炸开车窗。

你在上面看到的巨大加农炮是模仿他不久前制作的一个小弹弓，发射 8 毫米钢轴承。在它的更大的形式中，据说弹弓的大小是它的小兄弟的十倍，发射 80 毫米的钢珠具有令人难以置信的力量。在下面的视频中，[jrg]和他的朋友将弹弓推到一个巨大的空地上，在那里他们在几个不同的物体上测试它的速度。他们的第一枪飞了大约 220 码，击中了一座高压电塔，之后，男孩们将目光瞄准了一辆旧车。

弹弓发射的力量绝对令人印象深刻。拍了几个合适的镜头，这辆车就差不多报废了。

现在我们已经看到了[jrg]发射锯片、弯刀和巨型滚珠轴承，我们迫不及待地想看看接下来会发生什么！
