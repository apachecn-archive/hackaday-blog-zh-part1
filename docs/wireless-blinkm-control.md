# 无线 BlinkM 控制

> 原文：<https://hackaday.com/2009/01/10/wireless-blinkm-control/>

![blinkm](img/1b55e93ba2e755381c73d841abef56a5.png "blinkm")

[John]一直在使用几种 RGB 设备。他创造了一个[控制器，与每个 BlinkMs 无线对话](http://jarv.org/wireless_blinkm.shtml "jarv.org")，改变它们的行为。核心是一个旧的继电器测试盒，用于测试电话电路。它的四个旋钮都连接到 Arduino 上的模拟输入。使用射频链路设备传输信号。每个 BlinkM 都配有一个 ATmega168 和接收器。控制箱还有一个开关，可以同时向所有设备发送相同的信号。发送和接收代码可在他的网站上获得。你可以在下面找到它的视频。
