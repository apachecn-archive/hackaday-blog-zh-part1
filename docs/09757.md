# 所有 PCB 四轴飞行器充分利用每个组件

> 原文:[https://hack aday . com/2012/02/24/all-PCB-quad copter-makes-the-most-the-each-component/](https://hackaday.com/2012/02/24/all-pcb-quadcopter-makes-the-most-of-each-component/)

![](../Images/3af468ca6175ebf6e7e3f43860929ced.png "quadcopter-and-transmitter")

很难抑制我们对这个被称为微型飞行器的微型四轴飞行器项目的兴奋。[Frank]在开发项目时，设法将令人印象深刻的功能集合在一起。首先，四轴飞行器本身使用全 PCB 结构。甚至马达的支撑都是 PCB，带有键槽，垂直于主控制板，然后用焊点固定。我们认为这将是一个比[其他全 PCB 版本](http://hackaday.com/2011/04/29/mini-quadrocopter-is-crazy-awesome/)更有弹性的选择。

前景中看到的控制板有一个边缘连接器，它与 Wii classic 控制器连接器相匹配。这是你用来控制飞行的。但是还有更多。控制器左侧可见的插头与直升机甲板上的插座相匹配。这使你可以同步两者，这样就没有无线电频率干扰，并通过 USB 连接给电池充电。说到无线通信，[Frank]选择使用 ATmega128RFA1。这是 Atmel 公司的一款新型微控制器，内置无线电。添加一个陀螺仪传感器和一些电机控制，您就可以开始工作了。
