# 学生透雨器，无线水枪

[https://www.youtube.com/embed/0aaQIkLkSIo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/0aaQIkLkSIo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

至于恶作剧，[奥斯汀·沙夫的]无线隐藏水枪是一个真正的治疗。上面的视频简单解释了一下，并展示了运行中的设置。恶作剧者拿着一个无线 AVR 遥控器，当不知情的受害者走过时，他激活第二个 AVR 控制一个泵；到处喷水。

虽然我们大多数人现在已经离开了学校，但这个项目仍然是一个有趣的办公室或者街头恶作剧。如果你是一个注册者，原理图和源代码可以在[AVR breaks](http://www.avrfreaks.net/index.php?module=Freaks%20Academy&func=viewItem&item_type=project&item_id=2627)找到。或者，在跳转后检查两者的副本。
