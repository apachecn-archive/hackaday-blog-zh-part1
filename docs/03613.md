# EVIC 发动机

> 原文:[https://hackaday.com/2009/06/24/evic-engine/](https://hackaday.com/2009/06/24/evic-engine/)

![evic](../Images/4c5ba24db2cbe5132888679fa760491b.png "evic")

EVIC 是一台[计算机控制的内燃机](http://home.cogeco.ca/~davebowesevic/index.html)，采用无凸轮电磁驱动阀门系统。除了进气门和排气门控制，处理器还处理点火正时。有了动态气门正时，就有可能使发动机更有效率。传统的内燃机会浪费燃料，而 EVIC 可以跳过不需要的动力循环。通过增加阀门持续时间，中央处理器能够轻松启动。最新的[是 EVIC Mk3](http://home.cogeco.ca/~davebowesevic/2008%20Dev.html) ，它增加了一个排气门传感器，和 3:1 螺线管杠杆。有一个照相馆，里面有几个 [EVIC 发动机](http://home.cogeco.ca/~davebowesevic/Photo%20Gal.html)。Mk2 Twin 在下面嵌入的视频中演示。
