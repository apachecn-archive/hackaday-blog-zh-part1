# AVR Mega8 RSS 阅读器

> 原文:[https://hackaday.com/2009/02/26/avr-mega8-rss-reader/](https://hackaday.com/2009/02/26/avr-mega8-rss-reader/)

![avr_rss](../Images/61447ec6f4adf083d7e16ab53d6c4fec.png "avr_rss")

[Barney_1]建造了这个[系列 RSS 阅读器](http://jumptuck.wordpress.com/2009/02/25/dragon_rss/)。他使用的是[龙骑士 500](http://www.ecrostech.com/AtmelAvr/DragonRider/) 开发板，这是一个套件，具有扩展功能，包括 LCD、串行接口和电源。虽然你不需要这个套件，但是你可以用相似的规格来建造你自己的套件。他用 python 编写了一个程序来抓取 RSS 提要并将其发送到 LCD 上。如果你有龙骑士的话，他有一些具体的解决方法。你可以在他的网站上下载固件和源代码。休息之后你可以看到它的视频。
