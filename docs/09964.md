# 制作你自己的原子钟

> 原文:[https://hackaday.com/2012/03/24/make-your-own-atomic-clock/](https://hackaday.com/2012/03/24/make-your-own-atomic-clock/)

![](../Images/5d0e204b29ff3d7388372f210dd0cf4a.png "DIY-Atomic-Clock-Synced-LED-Clock_2-500x375")

我们看到大量的时钟项目通过，但通常是他们的视觉或机械设计脱颖而出。DCF-77 LED PIC 时钟很有趣，因为它与 T2 布伦瑞克的原子钟同步。该时钟接收 77.5 KHz 的无线电信号，称为 DCF77，这就是它得名的原因。

这个电路看起来出奇的简单，而且通常造价不到 30 美元，这取决于你如何组装。你可以从网站上下载原理图和代码，但你可能需要做一些研究，了解如何捕捉来自你所在位置的信号。写这篇文章的人住在欧洲。
