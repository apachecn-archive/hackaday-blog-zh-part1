# 两代远洋 ROV

> 原文:[https://hack aday . com/2011/01/15/两代远洋 rov/](https://hackaday.com/2011/01/15/two-generations-of-ocean-going-rov/)

![](../Images/c013d0d5ee558182539612104966351a.png "completed_rov")

[Eirik]希望帮助激励其他人接受大项目，他发了一个链接到他的 ROV 项目。他大约在一年前开始了这个项目，上面的图片显示的是第一代。休息过后，你可以看到 ROV 在最初几次航行中捕捉到的视频。它们非常清晰，你会立刻看到这个小家伙在跟踪一只水母。像许多[自制 ROV 的](http://hackaday.com/2010/02/23/building-the-yellow-submarine/)【Eirik】仍然在寻找正确的方法让电线穿过外壳而不泄漏。到目前为止，他做得很好，并为顶部系绳设计了一个很好的电缆卷轴，但一些水确实进入了。他几乎完成了第二代相机，重新设计了相机底座，向下瞄准，这样画面中的东西更有趣。
