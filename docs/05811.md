# 家庭抓爪游戏让小家伙们开心

> 原文:[https://hacka day . com/2010/08/16/home-claw-game-pleases-the-little-ones/](https://hackaday.com/2010/08/16/home-claw-game-delights-the-little-ones/)

![](../Images/1f2bdd4ea8a676b040e96e6f19d7a077.png "the-claw")

看完《玩具总动员》后，威尔·戈尔曼的儿子想玩起重机游戏。与其在比萨饼店的大厅里闲逛，不如[Will][在家里用乐高](http://www.battlebricks.com/lego-skill-crane/)建造一个。他称之为技能起重机，它有一个巨大的起重机架在宝箱的顶部。爪子可以在门架上左右移动，一旦它在正确的位置，就有机会掉落一些战利品。休息后在视频中看到一次成功的运行，如果你不能帮助自己，有[构建指令](http://www.battlebricks.com/models/lego-skill-crane/)会让你立即启动并运行。但是你不必用乐高来建造它，有时候[你只需要一些垃圾来组装这些东西](http://hackaday.com/2010/02/24/bring-the-crane-game-home/)。 [https://www.youtube.com/embed/V_DFsHLyQno?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/V_DFsHLyQno?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
