![](../Images/15e9b6820d45238d8d4382371a3757e4.png "desktop")

Pyxis 2 是最近发布的新[嵌入式操作系统](http://www.skewworks.com/)。第一个 Pyxis 操作系统可以在一个带触摸屏的 arduino 上运行，并提供了一个漂亮的 GUI。另一方面，Pyxis 2 已经脱离了它的大部分预测器，现在是在。NET 微框架，并且原生支持 [FEZ Cobra 和 ChipworkX 系统](http://www.ghielectronics.com/)。它支持 320×240，480×272，甚至 800×480(如果你有内存)，液晶显示器。

该操作系统的一些功能包括阿尔法混合桌面图标，文件对话框，提示，输入，颜色选择，文件盒，按钮，所有标准的东西来构建您的应用程序，支持网络，USB，SD 和 Mp3。既然是。你也可以使用 C# visual studio 来构建一切，而不需要在准备好之前致力于硬件。
