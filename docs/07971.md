# CNC-PBDU(煎饼面糊分配器单元)

> 原文:[https://hack aday . com/2011/07/01/CNC-pdbu-pancake-batter-dispenser-unit/](https://hackaday.com/2011/07/01/cnc-pdbu-pancake-batter-dispenser-unit/)

![](../Images/ffa05061f3dc506cfdd1d1bf0c3bbefc.png "lego-pancake-maker")

机器制作的烙饼味道好得多。嗯，对[墨西哥维京人]来说是真的，他[建造了一个自动煎饼制作机](http://makermig.blogspot.com/2011/06/how-pancake-bot-works.html)让他的家人高兴。

显然，建筑材料的选择是乐高。该机器由一个底座和两侧的两个线性齿轮组成。一个龙门架高高地悬挂在基座上，在齿轮传动塔上向两边移动。书写喷嘴由番茄酱瓶子制成，可以沿着这个门架前后移动，在 X 轴和 Y 轴上做全方位的运动。乐高气动泵提供压缩空气，迫使面糊从瓶子里出来。这个配药系统非常聪明，值得多读一点。但是如果你只是想让我们做好吃的，你可以在休息后的视频中看到它烤薄饼。

唯一缺少的是[自动翻转](http://hackaday.com/2010/07/28/flipping-pancakes/)。
