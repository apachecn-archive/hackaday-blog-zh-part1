# 无人机实验室将 Hypnotoad 带来的兴奋带回家

> 原文：<https://hackaday.com/2009/10/22/drone-lab-brings-the-excitement-of-hypnotoad-home/>

![Drone-lab-hypnotoad](img/4b334e80160293005f75ec95be948878.png "Drone-lab-hypnotoad")

[Peter]向我们透露了他的新合成器套件，无人机实验室。它有我们寻找合成器的东西:旋钮、输入、开关、古怪的声音。您可以使用套件版本进行焊接，也可以购买这些预制组件。Peter 称这是一个开源工具包，但我们没有看到电路板插图，只有原理图。

我们没想到的是它能够模仿[催眠状态](http://futurama.wikia.com/wiki/Hypnotoad)。正如在休息后的视频中看到的，你最喜欢的电视节目的美妙声音现在可以在你自己的家里创造出来了。如果你从未看过《催眠狂》(喘息！)休息之后我们也会谈到这一点。

所有荣耀归于催眠者！
