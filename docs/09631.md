# 用 55 加仑的桶做一个风力涡轮机

> 原文：<https://hackaday.com/2012/02/09/make-a-wind-turbine-from-55-gallon-drums/>

[![vertical axis wind turbine](img/56e5898e77a9719ebb29b466ef8b5a30.png "vawt")](http://hackaday.com/2012/02/09/make-a-wind-turbine-from-55-gallon-drums/vawt/)

虽然在 greenterrafirma 的页面上列出了几个垂直轴风力涡轮机，但我们对装有 55 加仑滚筒的那个特别感兴趣。尽管这些设计中的配偶认可因素是有争议的，但 100 美元，55 加仑的桶设计可以提供非常好的投资回报。制造其中一个所需的工具相对简单，所以这可以让那些没有大量设备的人也能进行这项实验。

如果你不喜欢蓝色的大桶，这篇文章还展示了其他几种涡轮机设计，包括一种由木头和铝箔制成的，以及一种由 PVC 管制成的。休息后的视频很好地解释了“蓝桶”的建设过程，但如果你宁愿只看这个[VAWT]的行动，快进到 5:25。
