# 翻书式数字化

> 原文：<https://hackaday.com/2010/03/19/flip-book-style-digitization/>

[https://www.youtube.com/embed/tCOXC5PTJj8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/tCOXC5PTJj8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这种图书数字化的方法允许你通过翻页来扫描整本书。它使用每秒捕捉 500 帧的高速摄像机，以便清楚地看到每一页。处理软件隔离每一页，分析纸张中由于翻转而产生的任何曲线，并平滑图像以获得更好的光学字符识别结果。这大大减少了[数字化一本书](http://hackaday.com/2009/04/20/high-speed-book-scanner-from-trash/)的时间，甚至比自动翻页的[设置](http://hackaday.com/2009/12/17/lego-book-scanner/)还要短。
