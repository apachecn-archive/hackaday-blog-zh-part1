# 燃烧的人 2011:彼得·哈德森的卡戎频闪雕塑

> 原文：<https://hackaday.com/2011/09/16/burning-man-2011-peter-hudsons-charon-strobe-sculpture/>

![](img/8e03874e221c43031d39acd731e7846d.png "Charon")

在 Hackaday，我们唯一喜欢的比基于巨型旋转艺术沙漠的装置更好的东西是*交互式*基于巨型旋转艺术沙漠的装置。彼得·哈斯顿的《T2》中的卡戎也不例外。因他的[频闪雕塑](http://www.hudzo.com/?p=186)【彼得】而闻名，在中断两年后，他带着可能是深海海滩上发现的最疯狂、最恐怖的雕塑之一回归。这件作品的特点是一个巨大的纺车，在它的内缘安装了人体骨骼。旁观者可以拉动一系列 6 对绳索，使轮子快速旋转。当绳索牵引器足够协调以使轮子以正确的速度旋转时，一个闪光灯被激活，显示骨骼的动画。

一天晚上，在听到当地对这件作品的议论后，我来到了这个地方。高耸的轮子正在旋转，当时的拉绳人拼命想让闸门启动，每隔几分钟左右就会有人试图协调拉动，但只是为了混淆视听。从我的角度来看，获得正确的速度似乎非常困难，而且拉力器必须将绳子几乎拉向地面。在我看这件作品的短暂时间里(下巴贴着地板)，闪光灯启动了一两次，老实说，这完全值得。想看看这个怪物在行动中是什么样子，请查看跳跃后的视频。
