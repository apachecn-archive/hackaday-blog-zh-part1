# 太空中的机器人

[https://www.youtube.com/embed/nl6lZbyLkzs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/nl6lZbyLkzs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

球体(同步位置保持、接合、重新定向实验卫星)是麻省理工学院制造的实验机器人，用于测试机器人在太空中的运动。正如你在上面的视频中看到的，它们能够很好地操纵。他们似乎保持队形相当紧密。他们使用压缩二氧化碳，通过 12 个不同的推进器进行定位。它们应该能够在彼此周围自主导航，也能够彼此对接。
