# 一个友好的名叫筷子的蜘蛛机器人

> 原文:[https://hack aday . com/2011/05/09/a-friendly-spider bot-named-quicks/](https://hackaday.com/2011/05/09/a-friendly-spiderbot-named-chopsticks/)

![chopsticks_the_spiderbot](../Images/084c33dc3e18c8f2d2e28c173400b38d.png "chopsticks_the_spiderbot")

在互联网上看到他的六足机器人后， [[Russell]决定建造一个自己的](http://makeprojects.com/Project/-Chopsticks-the-spider-robot/748/1)。制造这些机器人的一个不利因素是成本。他经常看到它们由激光切割零件和非常昂贵的伺服系统构成。与其在这个机器人上花费成百上千的美元，[拉塞尔]决定用筷子和变形塑料制造一个便宜的轻量级机器人。

他的八足机器人被恰当地称为“筷子”，利用 28 种不同的伺服系统来控制它的运动。24 个伺服系统用于它的腿，另外 3 个用于头部运动，而一个额外的伺服系统操纵机器人的下颚。机器人的腿和主要结构由筷子组成，而变形机器人用于脚、伺服支架和几乎任何筷子做不到的地方。

[Russell]甚至添加了一组眼柄来完成蜘蛛主题，用红外复眼武装它们以进行物体跟踪。正如你在下面的视频中看到的，这个机器人非常具有互动性。

继续阅读，看看筷子的视频，[或者如果你有机会的话，访问他的“让我们制造机器人”网站](http://letsmakerobots.com/node/26107)——他有一份非常详细的建筑日志，以及大量展示他的蜘蛛机器人运行的视频。
