# 无线电控制 Nerf 坦克

> 原文：<https://hackaday.com/2009/04/23/radio-controlled-nerf-tank/>

[https://www.youtube.com/embed/5FjT9nUkj-I?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/5FjT9nUkj-I?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Travis7s]建造了这个巨大的 [Nerf 坦克](http://www.instructables.com/id/RC-Nerf-Tank/)。它有无线电控制、网络摄像头、激光瞄准器和播放音乐的功能，非常棒。他用的是 Nerf Vulcan 步枪，临时改装了远程射击的伺服系统。这个东西非常大，正如你从视频中看到的，它大约和他家椅子的坐垫一样高。音响系统是一个放大器和一些连接到 iPod 的扬声器。这东西可以涂上一层漂亮的油漆，让它看起来更有威胁性，也不那么烦人。然而，它真正需要的是从声卡播放声音的能力。想象当帝国进军进入一个房间时，或者可能是一个带有适当的侮辱和攻击短语的声音板。
