# 《我的世界》的 16 位 ALU

> 原文:[https://hackaday.com/2010/09/29/16-bit-alu-in-minecraft/](https://hackaday.com/2010/09/29/16-bit-alu-in-minecraft/)

我们喜欢这里的游戏，但我们并不经常有理由去写它们。然而，这是值得一提的。独立热门电影《《我的世界》》正在吞噬整个互联网。游戏本身在我们的员工中时好时坏，但这个项目被一致认为是令人敬畏的。[【the internetftw】创建了一个 16 位处理器](http://www.youtube.com/watch?v=LGkkyKZVzug&feature=player_embedded)的 ALU 部分的模拟。他可以让它运转起来，然后边跑边观察状态的变化。这是一个更大的项目的一部分，该项目旨在创造出《计算机系统的元素[一书中所示的整个处理器。](http://mitpress.mit.edu/catalog/item/default.asp?ttype=2&tid=10218)

对于那些想知道他花了多长时间放置所有这些片段的人，他实际上从另一个程序中导入了大部分。你可以在这个[论坛的帖子](http://www.minecraftforum.net/viewtopic.php?f=25&t=17924)中获得更多关于他们是如何做到这一点的细节。休息之后一定要看视频。
