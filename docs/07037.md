# 用你的意识(和拇指)控制火

> 原文:[https://hack aday . com/2011/03/08/用你的头脑和拇指控制火势/](https://hackaday.com/2011/03/08/controlling-fire-with-your-mind-and-your-thumb/)

![](../Images/077cc626a387aa2c4e05e02b46a783ae.png "mind-controlled-fire")

【TechB】在[用意念控火](http://kbcarte.wordpress.com/2011/03/08/more-fun-with-mindflex/)。他真正做的是用意念控制打火机。

许多人会记得来自[的 Mindflex 这个令人震惊的可怕黑客](http://hackaday.com/2010/03/04/using-mindflex-to-shock-the-heck-out-of-people/)。但是[TechB]的方法是跟随[为设备](http://hackaday.com/2010/04/08/hacking-the-mindflex-more/)开发 Arduino 接口的脚步。他正在使用 Arduino 大脑库读取头戴式脑电图的数据，并向自己的火控系统发送命令。该系统由点烟器和伺服电机组成。马达连接到打火机上的气体调节器，当你集中注意力时打开它，当你走神时关闭它。结果是更高的火焰显示更有组织的大脑功能。原型的唯一问题是，当你试图理清思路时，按下打火机的阀门会烫伤你的拇指。

休息之后，请观看视频，了解实际情况。[TechB]看起来有点像剪辑开始时的博格人，这让我们想知道他还做过哪些 Mindflex 黑客。
