# 大规模涂鸦武器

> 原文:[https://hackaday.com/2012/01/31/weapon-of-mass-graffiti/](https://hackaday.com/2012/01/31/weapon-of-mass-graffiti/)

什么东西用灭火器，自行车打气筒，提供几个小时的缓刑，社区服务，甚至可能坐牢？如果你说是自动涂鸦作家，那你就对了！奥利维尔·范·赫普特称这份小工作为时间作家。我们称之为破坏财产……但是西红柿，西红柿。

细节有点缺乏，但休息后，您可以从视频中大致了解该系统。[Olivier]将帖子标记为 Arduino 它显然是在运行由金属杆上的七个电磁阀组成的点阵打印机。这些墨水是通过一根与灭火器相连的管子注入的，而灭火器则是储墨器。自行车打气筒是用来给围栏加压的，这样外出时就不需要打气筒了。

显然你不应该在家里尝试这个，但是让我们作为一个学术练习来谈谈可能的改进。首先，油墨/油漆的混合需要控制，以消除滴落。我们也希望看到包含一些适当的喷雾罐喷嘴来整理结果。与 IMU 板配合使用，应该能够消除印刷设计。

这可能会成为彩虹涂鸦作家的一个有趣的附件。
