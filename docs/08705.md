# 可持续发展秘诀:定日镜

> 原文:[https://hack aday . com/2011/09/30/sustainability-hacks-定日镜/](https://hackaday.com/2011/09/30/sustainability-hacks-heliostats/)

![](../Images/a79fc3f1a15633ed9189af9ae7408e9e.png "heliostat")

太阳能电池板在那些试图少花钱多办事的人中很受欢迎，但至少在过去，安装太阳能电池板相当昂贵。对于某些用途，你可以忘记使用太阳能电池板，直接使用太阳光，效率损失很小。一种可以让你做到这一点的装置叫做定日镜，它实际上只是一面别致的镜子，你可以把阳光反射到你想要的任何地方。你可以通过窗户瞄准它，使它击中你的天花板并在整个房间内扩散，或者你可以将它指向一个可以收集太阳能直接加热某物的位置。

对于定日镜如何工作以及如何建造一个真正好的纲要，请查看本页，在这里你可以找到各种信息。见鬼，他们甚至有一个 Arduino 控制他们中的一些人！
