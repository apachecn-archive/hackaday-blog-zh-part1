# 望远镜的两项改进

> 原文：<https://hackaday.com/2011/12/06/two-telescope-improvements/>

尽管作为一个大学班级，每个人都认为是容易的 A，天文学是非常困难的工作。不仅拍摄一张图像要花费数小时，设备本身也必须在寒冷的夜晚接受不间断的监控。[Jerry]提交了一些简洁的项目，让他的夜晚更加舒适。

首先是一个[焦点控制器](http://www.cloudynights.com/ubbthreads/showflat.php?Cat=0&Number=4921421&page=0&view=collapsed&sb=5&o=all&fpart=1&vc=&PHPSESSID=)的模型。望远镜的焦距随着温度、大气条件，尤其是所用的滤光器而不断变化。Jerry 使用的 USB-nSTEP focuser 需要很难找到的单极步进器，所以他修改了他的 USB-nSTEM，使其能够接受带有 Pololu A4988 驱动器的双极器。

接下来是[Jerry]自己加工的令人印象深刻的 DIY 离轴导向器。一个离轴导向器允许天文学家引导望远镜，而不需要处理一个非常小的，非常灵活的引导望远镜。我们对[杰里]的机器技能印象深刻，但那是当你有一台像他这样的[牛逼磨坊](http://www.coultersmithing.com/forums/viewtopic.php?f=16&t=399)时才会得到的。
