# 力反馈 PSP 模式

> 原文:[https://hackaday.com/2007/07/14/force-feedback-psp-mod/](https://hackaday.com/2007/07/14/force-feedback-psp-mod/)

![](../Images/0bcb199bc50f15f7a76f7d30a667865f.png)
【F00 F00】发来了他最新的 PSP mod。通过结合一个电压放大器，一个 PIC 控制器和一个手机马达，他完成了力反馈功能的 PSP。他将 PIC 设置为当电机通过放大器检测到来自扬声器的特定声音频率时启动电机。
