# 球形变形六角机器人是一种机械水母

> 原文：<https://hackaday.com/2011/12/12/sphere-morphing-hexabot-is-a-mechanical-jellyfish/>

![](img/d13ae4c40443b328fec4e7e035f57d64.png "hex")

我们再一次对[品红]的机器人技术赞叹不已。几个月前，[品红]发布了他的 [MorpHex hexapod spherebot](http://hackaday.com/2011/08/18/morphing-hexapod-has-us-drooling/) 的视频，让我们肃然起敬。几个月后，更多的 MorpHex 芯片出现在机箱上。[品红]说他的项目还没有完成，但仍然足以让我们大吃一惊。

翻阅[品红]档案，这次会有更多的发现。MorpHex 将由两个半球组成，但只有底部的一个能够行走。这真的没有那么糟糕，因为[品红]给了上面的面板 1 个自由度。我们确信只够吓走掠食者。

[底盘](http://robot-kits.org/2011/04/07/a-little-update-on-morphex/)和[腿](http://robot-kits.org/2010/11/30/morphex-taking-form/)都是令人惊叹的小工程。尽管[品红]在他的 MorpHex 上投入了所有的工作，但仍有工作要做。他还没有得到球体滚动的命令。我们一定会把机器人跳舞的视频发布到一些高保真音响上。跳完之后看看这个。
