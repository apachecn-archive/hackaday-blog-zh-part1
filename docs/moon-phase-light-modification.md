# 月相光修改

> 原文：<https://hackaday.com/2009/08/19/moon-phase-light-modification/>

[https://www.youtube.com/embed/F0yKRRh9LF0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/F0yKRRh9LF0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在这个项目中，[一个月相灯被修改](http://blogs.msdn.com/coding4fun/archive/2009/08/18/9727393.aspx)来显示基于你电脑内部时钟的实际月相。在工厂里，这些月亮灯通过红外线遥控进行更新。他把大脑从时钟里取出来，换上了一个 Arduino。然后他运行一个程序，根据系统时钟更新 Arduino。他还添加了一个内部时钟来维护设置，因此不需要不断更新。如果能通过遥控器保留手动功能就好了，这样你就可以在满月时按下按钮，而不必改变系统时间。
