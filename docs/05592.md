# Robostool 跟着你来到你的椅子前

> 原文：<https://hackaday.com/2010/07/03/robostool-follows-you-to-your-chair/>

[https://www.youtube.com/embed/yF6ZL-WerYo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/yF6ZL-WerYo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Steve]与我们分享了另一个项目，这次是一个机器人脚凳。一个视差推进器微控制器，一些线性致动器，伺服系统和一系列传感器被封装在这个乙烯覆盖的立方体中。它有 3 种模式，远程控制，跟我来(在上面的视频中看到)，来到信标。这一点，再加上[【史蒂夫】的其他创造](http://hackaday.com/2010/06/09/baxter-the-butlerbot-and-robofridge/)，让我们认为他的最终目标是拥有一屋子的仆人机器人，他们渴望完成每一项让他从椅子上站起来的任务。对我们来说，这似乎是一个崇高的目标。你可以从他的网站下载源代码和原理图。休息后加入我们，查看更多 Robostool 的图片和视频。

[https://www.youtube.com/embed/lPnwycXY6lk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/lPnwycXY6lk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
