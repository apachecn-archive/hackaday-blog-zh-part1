# 单伺服机器人

> 原文:[https://hackaday.com/2009/10/30/single-servo-robot/](https://hackaday.com/2009/10/30/single-servo-robot/)

[https://www.youtube.com/embed/EFRPZRovn74?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/EFRPZRovn74?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[古伊列梅·马丁斯]接受挑战，制造了一个只有一个伺服系统的机器人。他的机器人是一个傀儡控制器，叫做步话机。实时地，它会随着你说话的声音移动它的嘴。对于那些真正好奇的人，他说葡萄牙语，他粗略地说“嗨，你好”。他使用的 Arduino 带有一个定制的声音传感器、一个单一的伺服系统、一个盒子和一个文件夹纸嘴。

建造一个单一的伺服机器人不应该是一个很大的挑战，我们甚至已经看到了只有一个电机的步行机器人。有这种两条腿的[履带](http://www.solarbotics.net/library/circuits/bot_walker_uCrawler.html)，我们记得看到过一个四条腿的 B.E.A.M .步行机，只有一个伺服系统，但现在找不到了。
