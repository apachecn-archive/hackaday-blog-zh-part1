# 普渡电气和电子工程师协会遥控潜水器

> 原文:[https://hackaday.com/2011/07/26/purdue-ieee-rov/](https://hackaday.com/2011/07/26/purdue-ieee-rov/)

[![](../Images/6e560bb632d33acea2e627c8ab379fcd.png "purdue-rov-hybris")](http://hackaday.com/wp-content/uploads/2011/07/purdue-rov-hybris.jpg) 
普渡大学的 IEEE 分部参加了今年的海洋先进技术教育中心竞赛，为上面看到的[Hybris ROV 获得了第二名。这项比赛包括几项强制性功能，包括封堵水下油井、收集生物样本和采集深水水样的能力。](https://sites.google.com/site/purduerovhybris/home)

他们想出的是一种快速灵活的船只，可以轻松克服许多通常困扰 ROV 制造的硬件障碍。有八个推进器，四个用于垂直运动，另外四个负责水平运动。在飞船的前面可以清楚地看到抓手装置，两个圆柱形容器容纳着电子元件。

不要错过[项目定义页面](https://sites.google.com/site/purduerovhybris/home)。每个挑战都有详细的讨论，以及团队的解决方案。他们发布的信息量给我们留下了深刻的印象，包括每个电子元件的概述以及设计文件和源代码。如果你想看看第一轮比赛进行得如何，请点击休息时间查找嵌入的视频。
