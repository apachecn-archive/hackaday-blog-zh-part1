# 多点触控桌面游戏

> 原文:[https://hackaday.com/2009/01/18/multitouch-tabletop-gaming/](https://hackaday.com/2009/01/18/multitouch-tabletop-gaming/)

[https://www.youtube.com/embed/QflrIK-m4Ts?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QflrIK-m4Ts?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

读者 Ramon Viladomat 发来了他去年在巴塞罗那 Pompeu Fabra 大学所做的工作。厌倦了只看到缩放地图演示，他创造了一个角色扮演游戏，利用了[可触摸](http://mtg.upf.es/reactable/ "reactable")的多点触摸界面。除了[多点触摸](http://hackaday.com/tag/multitouch "multitouch  - Hack a Day")，reacTable 还使用有形的[基准标记](http://en.wikipedia.org/wiki/Fiduciary_marker "Fiduciary marker - Wikipedia, the free encyclopedia")来代表物体，并作为一种替代的输入方法。上面嵌入的是演示界面和游戏的视频(从 3:43 开始)。这个游戏让你在虚拟世界中移动你的模型。当你的动作点再生时，表面显示你可以动态地移动多远，当你发现它时，慢慢地显示更多的地牢。您可以暂停动作，并使用手势来设置攻击模式。我们真的很喜欢这个演示，并希望看到有人使用像[战锤 40K](http://en.wikipedia.org/wiki/Warhammer_40,000 "Warhammer 40,000 - Wikipedia, the free encyclopedia") 这样的流行桌面游戏来制作一个。下面嵌入的是相关地图编辑器的演示。
