# 你的音响里有死虫子吗？

> 原文:[https://hack aday . com/2012/01/09/get-dead-bugs-in-your-stereo/](https://hackaday.com/2012/01/09/got-dead-bugs-in-your-stereo/)

![](../Images/04ec7a3d3830faf0b17b2361ea14ef7b.png "IMG109")

这种情况会发生在最优秀的设计师身上，他们花费大量的时间设计一个复杂的器件，却发现自己遗漏了一条轨迹，或者无法将某个东西路由到另一个东西。随着时间的推移，打开一个商业生产的器件，看到一根或两根小跳线飞过 pcb，或者一个电阻跨在 IC 上，这种情况越来越少见了。

但是当[Ilektron] [打开一个雅马哈杜比 Pro Logic 接收器寻找零件](http://ilektron-x.blogspot.com/2012/01/best-commercial-white-wire-hack-ive.html)时，他看到了一个非常大的“哎呀”和一个更疯狂的修复。制造者拿了一对继电器，把它们翻过来，然后用热胶把它们粘在一对集成电路上。然后，使用绝缘和未绝缘的总线将这些乱七八糟的东西“死虫”连接到电路上，然后使用更多的热胶加固/绝缘触点。

这是我们在商业产品中见过的最糟糕的修复之一，但是我们很想听听你有趣的恐怖故事“WTH 他们做了什么？”请在休息后加入我们的评论。
