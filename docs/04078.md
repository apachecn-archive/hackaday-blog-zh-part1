# 虚拟池，真实界面

> 原文：<https://hackaday.com/2009/09/21/virtual-pool-real-world-interface/>

[https://www.youtube.com/embed/2Wc_DXGe2fQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/2Wc_DXGe2fQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

周日，我们看到[机器人在玩台球和增强现实台球游戏。](http://hackaday.com/2009/09/20/pool-playing-robot-arpool/)今天我们将完成泳池三连胜:*虚拟*泳池使用*真实*球杆和另一个[老式视频](http://hackaday.com/2009/09/18/vintage-video-computing-across-america/)中的球，来自黑客一天的秘密地下金库。该视频值得注意的原因有两个:

首先是它的生产年份:1990 年。最近有很多关于现实世界游戏界面的讨论，比如任天堂 Wii [运动控制器](http://hackaday.com/2009/06/23/wii-motionplus-arduino/)或者微软的 Project Natal。这里我们看到的是近二十年前的一个更简单但非常有效的物理接口。

第二:视频的中间部分揭示了这一切背后的诀窍，事实证明这是惊人的简单。没有复杂的传感器或计算机视觉算法；球的速度和方向由一个 8 位处理器和四个红外发射器/探测器对的巧妙排列来计算。

视觉效果可能已经过时了，但是即使在今天，界面本身也是巧妙而令人印象深刻的，这种方法对于普通的车库修理工来说很容易实现。你能理解这个吗？我们在这里看到读者的*迷你高尔夫英雄 III* 游戏只是时间问题吗？
