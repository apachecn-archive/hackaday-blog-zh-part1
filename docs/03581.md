# 特雷门控制了马里奥

> 原文:[https://hackaday.com/2009/06/15/theremin-controlled-mario/](https://hackaday.com/2009/06/15/theremin-controlled-mario/)

[https://www.youtube.com/embed/YnZeI8uLJnw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/YnZeI8uLJnw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在上面的视频中，[征服者]正在使用特雷门琴控制超级马里奥兄弟。将他的手移向和移离垂直天线可以提高特雷门琴的音调。电脑实时监控这一点，并左右移动马里奥。环形天线控制特雷门琴的音量，并作为跳转按钮。控制器似乎工作得很好，比一个人在特雷门琴上挥舞的声音好得多。
