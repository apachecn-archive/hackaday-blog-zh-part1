# Cythbot，气动吉他英雄

> 原文：<https://hackaday.com/2008/11/22/cythbot-pneumatic-guitar-hero/>

![cythbot](img/af69748df5f21e1f59cd2092748a15f7.png "cythbot")

又有一个机器人希望通过岩石的力量统治人类。 [Cythbot](http://blog.wired.com/gadgets/2008/11/guitar-hero-rob.html "Guitar Hero Robot Plays Videogame With Electronic Precision | Gadget Lab from Wired.com") 是为了展示 Cyth Systems 机器视觉系统而建造的。该设备使用摄像头来观看吉他英雄监视器，并识别按钮按压的音符。一段时间后，拨弦杆被触发。这些音符只能通过像素强度来识别，因为星光可以改变它们的形状和颜色。所有按钮的按压都是用气动技术完成的。整个系统是独立的，不需要单独的计算机进行处理。我们最喜欢的部分是，控制器保持完全不变，工业光树用于指示音符。该团队表示，气动技术还不够快，无法达到 100%，[不像某些人](http://www.youtube.com/watch?v=S5GpRJItqjw "YouTube - Through the Fire and Flames TTFAF FC 100% 987,786")。休息后机器人的视频。
