# N64 控制器中的蓝牙媒体遥控器

> 原文:[https://hack aday . com/2011/05/14/bluetooth-media-remote-in-an-n64-controller/](https://hackaday.com/2011/05/14/bluetooth-media-remote-in-an-n64-controller/)

![](../Images/1be386d12c7f950c0c2b1b986f01176d.png "S")

[Phik]来信与我们分享了他的第一个微控制器项目。他在一个旧的任天堂 64 控制器中内置了一个[蓝牙遥控器来控制他电脑上的一个音频应用程序。一段时间以来，他一直在脑海中构建控制器的各个模块，但最初不知道应该将它放在什么样的外壳中。在一次将它塞进 XBox 控制器的失败尝试后(当然这里有足够的空间](http://waitwhatquestionmark.wordpress.com/2011/05/14/bluetooth-media-remote/))，他意识到他有一个坏了的 N64 控制器可以用。我们认为他在安装方面做得非常出色，它看起来几乎像一个商业产品。他记录了每个独立模块的构建和测试。通过查阅档案，你可以在他的网站上找到每个人的帖子。干得好，尤其是你的第一个项目！
