# Arduino 的大脑控制

> 原文：<https://hackaday.com/2009/10/20/brain-control-for-the-arduino/>

![brain-controlled-arduino](img/a85efeb725b9449a675747855158ee6b.png "brain-controlled-arduino")

当我们听到大脑控制的 Arduino 项目时，我们会立即想到一场编码噩梦。一如既往，简单的黑客是最好的黑客。[乔尔]和[阿克谢] [使用儿童游戏中的硬件作为 Arduino 的大脑接口](http://www.youtube.com/watch?v=28DnXLtrVqU)。

我们偶然看到了他们工作的视频(休息后嵌入的),并询问了更多关于我们认为难以置信的困难的信息。原来他们购买了[米尔顿叔叔的力量训练器](http://www.unclemilton.com/starwarsscience/)，它使用耳机来测量脑电波，并有一个对这些测量做出反应的基本单元。入侵这个设备不需要任何逆向工程。他们选择了简单的路线，接入了基本单元上的五个发光二极管。随着游戏测量更高的浓度水平，它会点亮更多的发光二极管。

到目前为止，进入游戏只是一个概念的证明。实现一个大脑控制的[啤酒机器人](http://hackaday.com/2009/08/19/kegbot/)取决于你。
