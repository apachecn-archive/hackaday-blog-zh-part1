# X-mas Hack: 8 频道音乐剧

> 原文：<https://hackaday.com/2009/12/02/x-mas-hack-8-channel-musical-show/>

![](img/9f7609d298aba5d261d2c4a03e8c17f2.png "8-channel-xmas-controller")

在过去的几周里，随着圣诞彩灯一年一度的亮相，我们看到它们一次出现在一所房子里。有些人只是在灌木丛中扔了一套网型灯，并称之为好的，但这并不适合(诺埃尔)。他给自己的房子配备了一个展示装置，包括音乐、8 个可控灯光频道和一个互联网接口。

他用一个塑料工具箱作为外壳来装所有的东西。固定在外壳底部的是八个固态继电器，用于灯串。Arduino 用于控制 SSR 开关，通过 FM 发射器播放音乐，以及与无线网桥连接。

[追踪者](http://en.wikipedia.org/wiki/Rubbernecking)可以将他们的收音机调到广播频率，并登录网络界面请求他们喜欢的音乐或追踪圣诞老人的当前位置。该设备甚至实现了 VU 监控，使灯光表演与音乐同步。如果你想了解更多，可以在休息后观看视频或查看他的[分步指导](http://www.instructables.com/id/xmas-box-ArduinoioBridge-internet-controlled-Chr)。Arduino 库确实使得将所有这些放在一起的代码变得非常简单！
