# 简单的晶体管测试仪使分类变得容易

> 原文:[https://hack aday . com/2011/09/02/simple-transistor-tester-makes-sorting-easy/](https://hackaday.com/2011/09/02/simple-transistor-tester-makes-sorting-easy/)

![simple_transistor_sorter](../Images/1814c0a5dbc398d0a20b793ed720cfc9.png "simple_transistor_sorter")

黑客[迪诺·塞戈维斯]带着他的黑客一周系列的另一部[回来了，看起来他在蹲下面对飓风艾琳后并没有太糟糕。](http://hackaweek.com/hacks/?p=358)

本周，似乎[Dino]在将他的 PNP 晶体管与他的 npn 分离时遇到了一些问题。在阿尔伯特·爱因斯坦被证明在分类电子元件方面用处不大后，[迪诺]决定建造一个简单的晶体管测试仪来帮助他区分 PNP 和 npn，而不必求助于查找产品数据手册。

测试器本身的构建相对简单。正如你在下面的视频中看到的，它由一个电源，一个 LED，几个电阻，一对已知的晶体管组成，除此之外别无其他。当一切都连接在一起时，NPN/PNP 对会使 LED 亮起，但只要移除一个晶体管，电路就会断开。将一个新的晶体管插入试验板上的空白处，可以立即让你知道你插入的是哪种晶体管。

当然，你可以用万用表区分晶体管，但如果你有一整个抽屉的松散元件，这是一个更有效的选择。
