# 个人激光蚀刻

> 原文：<https://hackaday.com/2008/07/12/epilog-zing-personal-laser-etching/>

<object width="450" height="364"><param name="movie" value="http://www.youtube.com/v/hxptHvpp6eY&amp;hl=en&amp;fs=1&amp;rel=0&amp;color1=0x3a3a3a&amp;color2=0x999999"> <param name="allowFullScreen" value="true"> </object> [新款 Epilog Zing](http://www.epiloglaser.com/zing_home.htm) 旨在将激光雕刻带到家中供个人使用。它的功率为 25 瓦，体积足够小，可以整齐地放在你的桌子上，上面的视频显示它的分辨率相当高。但是[与 mini18](http://www.epiloglaser.com/mini18_overview.htm) 相比，Zing 的 Z 移动更少，瓦数更低，雕刻面积更小，速度大约是 mini18 的一半。此外，由于价格设定在 8，000 美元左右，我们认为还没有多少人会购买它们供个人使用。然而，如果你有一个需要这些服务的小型家庭企业，Zing 可能是完美的。如果您想了解激光雕刻的更多信息，请参见【ladyada 的】[激光信息页面](http://www.ladyada.net/library/laser/)。
