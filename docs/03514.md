# 数字骰子

> 原文：<https://hackaday.com/2009/05/20/digital-dice/>

![dnd](img/4cce5bb8ada87250fb1fe42d22f3b2ab.png "dnd")

已经有几次尝试用现代科技来更新《龙与地下城》。大多数尝试都是以电脑游戏的形式进行的，不知何故未能捕捉到本质的体验。然而，这一尝试似乎在保持游戏不变的同时增加了一些技术含量。【Itay】已经[造了一些数字骰子](http://itay.bazoo.org/projects/ledmatrix)。简单地选择你希望你的骰子有多少面，然后摇一摇。好吧，一个随机数发生器并不具有开创性，但他确实做了一些非常激烈的焊接。LED 矩阵很酷，但我们喜欢看背面。休息后的视频里可以看到。
