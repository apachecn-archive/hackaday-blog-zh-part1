# 电锯/手电筒过度杀伤

> 原文:[https://hackaday . com/2010/08/08/电锯手电筒-过度杀伤/](https://hackaday.com/2010/08/08/chainsawflashlight-overkill/)

这个疯狂的电锯聚光灯赢得了我们的赞赏。它看起来非常危险，效率极低，而且可悲的是设计不足。我们绝对喜欢它。

这可能是从视频游戏或电影中提取的，似乎是一个紫外线灯泡远离终极后世界末日僵尸吸血鬼大决战武器。他拆开了一把链锯，用几个自行车链轮和一根链条把它装到 DC 跑步机的马达上。DC 发动机直接给汽车前灯供电。正如你在视频中看到的，当他转动油门时，它发出非常明亮的光。这项未完成的工作是由于仪器竞赛的截止日期，所以他计划添加一个保护罩，一些花哨的油漆工作，以及一个电压调节器。
