# 由于雾的投影，鬼魅般的图像出现了

> 原文:[https://hack aday . com/2011/11/12/ghost-images-appear-thanks-to-projections-on-fog/](https://hackaday.com/2011/11/12/ghostly-images-appear-thanks-to-projections-on-fog/)

![](../Images/ebfcc3183d16352b1e511021a837000c.png "SAMSUNG")

这个线框立方体看起来像是漂浮在半空中，因为它实际上是。这是一个[汤姆]称之为层流雾屏的项目。他制造了一种装置，可以发出微弱的雾气，投影仪发出的强光可以照亮这种雾气。这里真正的技巧是得到一个均匀的雾墙，这是层流部分的地方。[层流](http://en.wikipedia.org/wiki/Laminar_flow)是一种流体以完全平行的方式流动的现象，不允许错误的部分引入湍流。这是[和水](http://hackaday.com/2010/08/22/laminar-water-jet-explained/)最喜欢的把戏。

[Tom 的]雾幕从一个电脑风扇开始移动空气。这种气流由海绵和多根吸管的组合来平滑和引导。这种装置负责建立层流，因为空气在途中从超声波雾器中获得雾。

这里唯一真正的问题是你想让投影仪射向无限远。否则，投影会穿过雾气并显示在墙上，破坏效果。户外应用非常适合这一点，只要没有空气流动来扰乱你精心建立的雾幕。
