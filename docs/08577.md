# 激光切割机的挑战

> 原文:[https://hackaday.com/2011/09/14/a-laser-cutter-challenge/](https://hackaday.com/2011/09/14/a-laser-cutter-challenge/)

![](../Images/042ae7ce7bf063563b11ce3f8a2f7ffc.png "microlaser")

当路易斯维尔黑客空间 LVL1 讨论购买一台新的激光切割机时，一名成员说，“我可以在你买之前造一台。”战书已经发出，挑战已经设定，在黑客空间交付他们的新激光切割机之前，建造一台[微型激光切割机](http://wiki.lvl1.org/Micro-Laser_Cutter_Project)的竞赛已经开始。

建造的机械方面相当简单。X 轴只是一个安装在托架上的步进电机、螺纹杆和激光模块。该托架在两侧的两个步进电机的帮助下沿 Y 轴移动。所有的东西都装在比想象中更好的木板上。

对于该项目的电子设备，三个电机驱动器由几个逻辑芯片制成，激光点火继电器是从为 LVL1 的[跨大西洋气球建造](http://whitestarballoon.com/)开发的测试设备中偷来的。马达和激光控制由 Arduino 处理，以保持建造简单，因为比赛在第一台激光切割完成后就结束了。

LVL1 现在正在制作激光切割机挑战赛冠军的第二个版本。他们计划在触摸屏界面上切割出信用卡大小的塑料空白。我们迫不及待地想看到那个构建的结果。
