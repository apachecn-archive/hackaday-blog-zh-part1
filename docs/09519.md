# 忘记 Siri——让 Wolfram Alpha 成为你的私人助理

> 原文：<https://hackaday.com/2012/01/17/forget-siri-make-wolfram-alpha-your-personal-assistant/>

![](img/1f54bb80b789366cbc4b52654b7afdcc.png "20120115-qcyawspq6c4aay1tpxm8tnchpr")

所以你可以花一大笔钱买一部新手机，它配有声控数字助理。但是说实话，如果您自己编写这个特性，会更令人满意。这里有一个关于如何通过将 Asterisk 服务器和 Wolfram Alpha API 结合起来的指南。

Asterisk 是一个我们已经很熟悉的包。这是一个开源的专用小交换机套件，可以让你[建立自己的电话网络](http://hackaday.com/2011/03/18/free-home-phone-method-uses-parts-were-familiar-with/)。很有可能，你不会只为这个项目建造一个，但是如果你确实要记录这个过程并且[让我们知道它](http://hackaday.com/contact-hack-a-day/)。有了星号服务器，您只需要给助手脚本一个扩展名(在本例中是 4747)。

但是接下来的问题是将你的演讲翻译成可以作为 Wolfram 查询提交的文本。也有一个使用谷歌翻译的 API。从那里你可以调整缩写和其他参数，但总的来说，你的新助手已经准备好了。打电话问问当你的轮胎瘪了的时候该怎么办(是的，那个广告也让我们抓狂)。
