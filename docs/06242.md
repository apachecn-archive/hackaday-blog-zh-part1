# 机器人的眼睛看你做什么

> 原文:[https://hack aday . com/2010/11/05/robot-eyes-look-where-you-do/](https://hackaday.com/2010/11/05/robot-eyes-look-where-you-do/)

![](../Images/2775ee150c7a8cf58a9a410570774e80.png "robot-eye")

这只机器眼的移动速度比人眼快五倍。它可以用来跟踪人的目光，正如你看到的这枚硬币，它足够小，可以成对使用。当用来跟踪你的目光时，它需要一个定制的眼球跟踪器。这里的想法是，通过监测一个人的注意力，可以了解他的心理。但是我们想知道像这样的设置的[增强现实](http://hackaday.com/2009/08/28/arduino-augmented-reality/)属性。

想象一副眼镜作为平视显示器。如果这台相机知道你在看哪里，它就能处理你凝视的东西并覆盖数字信息。与所有新技术一样，这显然有军事用途，但我们更感兴趣的是 Flickr pool 类型的人们现实世界经验的集合。比如在纽约订阅那个[拇指驱动网络](http://hackaday.com/2010/10/30/this-is-a-peer-to-peer-filesharing-network/)的位置，让相机/眼镜指引你去最近的安装点。
