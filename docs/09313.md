# 电子装饰品肯定会把[戴夫]列入圣诞老人的“淘气”名单

> 原文:[https://hacka day . com/2011/12/16/electronic-corrupt-will-but-put-Dave-on-Santas-naughty-list/](https://hackaday.com/2011/12/16/electronic-ornament-will-definitely-put-dave-on-santas-naughty-list/)

![mischievous-xmas-ornament](../Images/6e223f8a8eb033faf22f1ae3a93c4a48.png "mischievous-xmas-ornament")

[Dave Vandenbout]说他的姐姐很重视圣诞传统，决定全家人每年都要开始为圣诞树做装饰品。没有人会放过摆弄电子产品的机会，[戴夫]开始头脑风暴，为他们的圣诞树设计完美的电子装饰品。

他选定了你在上面看到的圣诞树设计，它最终将容纳 15 个 RGB LEDs。在电路板的背面，他计划安装一个 PIC 18F27J53 微控制器，该控制器将负责 LED 显示屏和其他更淘气的组件。

你看，他姐姐的节日精神并没有吓住他，[戴夫]想要用一张肮脏的嘴武装装饰品，并让它试图从树上摇下其他装饰品。为了做到这一点，他在 PCB 的背面安装了一个振动马达，以及一个扬声器和 MicroSD 卡，以提供装饰品的声音片段。

老实说，我们认为他的想法很有趣，我们只能想象当可爱的、发光的圣诞树装饰品脱口而出:“吃了我吧，圣诞老人！”时奶奶会是什么表情

我们只是希望他完成项目后会给我们发些视频。
