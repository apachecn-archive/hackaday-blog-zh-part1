# 蜘蛛世界

> 原文:[https://hackaday.com/2010/03/15/mondo-spider/](https://hackaday.com/2010/03/15/mondo-spider/)

![](../Images/1f88045536cef125ac500d4dfd17fcf2.png "mondo-spider")

这辆车被恰当地命名为[蒙多蜘蛛](http://www.mondospider.com)。这不是从一些启示录电影，但在这里看到的燃烧的人。像许多[燃烧的人](http://hackaday.com/2009/11/18/argh-thar-be-a-big-wheel/) [展览](http://hackaday.com/2009/09/08/hand-of-man-mechanical-claw/)一样，它是为了建造的乐趣而建造的，预算相当高:15000 美元。休息之后，我们嵌入了许多视频中的一个，以及一些硬件细节。

这个八条腿的怪物重约 1600 磅，比我们不久前看到的可骑乘的六足动物要大得多。一个本田 24 马力 V 型双引擎提供了运行液压腿所需的动力。它的设计目的是转弯，但正如你在视频中看到的，牵引力可能有点问题。对于夜间行动来说，腿的轮廓已经用发光的文字勾勒出来，好像这还不足以吓到内心脆弱的人。
