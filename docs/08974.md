# LED 色情短信皮带扣

> 原文:[https://hackaday.com/2011/11/01/led-sexting-belt-buckle/](https://hackaday.com/2011/11/01/led-sexting-belt-buckle/)

![](../Images/b9648e16e55c1a11ee6238caad42a078.png "buckel")

出于某种邪恶的原因，[斯科特]有一个朋友想要一套“色情短信主题”的万圣节服装。我们不会试图对[斯科特]的朋友的创造力或精神稳定性做出任何假设，但他提出的[短信滚动 LED 皮带扣](http://store.curiousinventor.com/sms-to-scrolling-led)非常酷。

这款腰带是基于一个在网上找到的价值 13 美元的滚动 LED 皮带扣[Scott]。不过，皮带扣有个问题。13 美元意味着[斯科特]没有得到他的皮带扣的全部功能，所以整个设备上只有 3 个按钮:字母向上、字母向下和输入。不是按下一个按钮 80 次来得到一个小写的“z”，而是一个 Arduino 被扔进混合物中来处理所有的按钮按下。

Arduino 草图现在可以在几秒钟内将任何信息输入皮带扣。剩下要做的就是处理构建中的短信发送部分。为此，[Scott]使用了 [Sparkfun USB 主机保护罩](http://www.sparkfun.com/products/9947)和一个定制的 Android 应用程序。每当手机收到短信时，信息就会通过 USB 保护盾发送到 Arduino，并在皮带扣上输出。

我们不会对万圣节派对期间的信息内容做任何假设，但至少这个构建的视频演示是家庭友好的。休息之后来看看。
