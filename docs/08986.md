# USB 和 PS/2 键盘记录器，扰乱你的语法

> 原文:[https://hack aday . com/2011/11/02/USB-and-PS2-key-loggers-and-mess-with-your-grammar/](https://hackaday.com/2011/11/02/usb-and-ps2-key-loggers-and-mess-with-your-grammar/)

![](../Images/85bc2a9d6e62368eea6242712b5d4c94.png "usb-passthrough-keylogger")

[Irongeek]用这个新的键盘记录器原型再次达到他的老把戏。正如试验电路板搭建的电路所证明的，这还处于早期阶段，但[Adrian]仍然在休息后给我们一个演示视频，展示他现在的情况。它有两种风格，上面看到的 USB pass through，或者另一种仍然通过 USB 连接到计算机，但使用 PS/2 键盘。

除了键盘记录器窃取你输入的所有内容这一明显问题之外，这个设备也有一些恶作剧的价值。Teensy 有足够的处理能力来观察你输入的内容，并在输入过程中进行修改。他炫耀明目张胆的改写，比如把“has”改成“haz”，或者把“you”改成“U”。我们认为最好把“他们的”改成“他们的”，或者“它的”改成“它的”。这些将很难看到发生，如果你给替换发生的频率增加随机性，你的受害者会更快地得出结论，他们要疯了，而不是他们是一个小恶作剧的目标。事实上，这可能是多年来我们自己语法错误的原因；爆炸！
