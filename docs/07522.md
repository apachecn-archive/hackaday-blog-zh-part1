# 隔离变压器 101

> 原文:[https://hackaday.com/2011/05/02/isolation-transformer-101/](https://hackaday.com/2011/05/02/isolation-transformer-101/)

![](../Images/b8e73c16b6720491efc06e74a662872b.png "Untitled")

[托德·哈里森]发布了一个大约 50 分钟的视频，涵盖了你想知道的关于隔离变压器的一切，以保护你自己和你昂贵的设备。诚然，我没有给这个问题太多的想法，但如果你需要测量高电压，你可能应该看看这个视频。

[Todd]解释了不同类型的变压器之间的差异，包括成本、为什么你会想要一个、安全性和大量的理论。手拿白板，所有细节都解释清楚了。如果这还不足以说服你，还有一些有趣的“不要在家里尝试这个”的实验，击中了要害。

他有一个相当结实的 8 安培 40 磅的怪物，价格相当高，但为了安全还是值得的。除非你认为你会需要那种类型的安培数，你可能会很好地与一个较小的模型。视频中显示的产品是医院级的，需要一个 mod 来确保它对您的工作台安全。[Todd]也解释了那个 mod。即使你不打算这么做，看到一个比你的手还大的变形金刚也很酷。另外，它能发出很酷的声音。

视频休息后加入我们。
