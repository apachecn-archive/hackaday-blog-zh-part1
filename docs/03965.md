# 可编程弹簧致动器腿

> 原文：<https://hackaday.com/2009/09/05/programmable-spring-actuator-legs/>

 <https://www.youtube.com/embed/_g79mOSvSsE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>这个概念验证展示了一种制造机器人腿的有趣方式。他们使用<a href="http://www.informatics.sussex.ac.uk/users/wb23/PGS/PGS.html" target="_blank">可编程弹簧致动器</a>让这些腿像有“反射”一样工作。建造者用皮带牵引着机器人。当腿达到向后运动的极限时，一个信号从肩膀向下发送到膝盖，将腿抬起，使其回到向前的位置。一个信号也被发送到另一侧的肩膀，以抑制提升反射，防止他们同时迈步。由此产生的运动是非常有机的感觉。我们很好奇这是如何在依靠自身动力移动的机器人身上实现的。</p> <p>[via <a href="http://blog.makezine.com/archive/2009/09/walking_on_programmable_springs.html?CMP=OTC-0D6B48984890" target="_blank"> makezine </a></p> </body> </html>
