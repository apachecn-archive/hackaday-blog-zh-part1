</span> <p>[Rick]一直在以各种方式建造特斯拉涡轮机，并将他的项目发布到 youtube 上。对于那些不熟悉的人来说，<a href="http://en.wikipedia.org/wiki/Tesla_turbine" target="_blank">特斯拉涡轮机</a>是一种无风扇涡轮机，它使用一个光滑的中心圆盘，通过气体或液体的摩擦旋转。由于涡轮本身没有突起，所以非常稳定。还有很多其他的好处，可以在维基百科页面上探索。[瑞克]，正如你从上面的视频中看到的，已经为它们找到了几个用途，从万圣节道具，到<a href="http://ca.youtube.com/watch?v=9DOzgSOFh_4" target="_blank">灯的发电机</a>，到<a href="http://ca.youtube.com/watch?v=rczWjuwZVFA" target="_blank">自动摇动手电筒的摇动器</a>，你甚至可以看着他<a href="http://www.youtube.com/watch?v=KBLm1lpSBMk" target="_blank">启动一个直到毁灭</a>。尽管其中大部分速度相对较低，但他已经展示了他可以用稳定到每分钟 10，000 转的 CD 轴来制作一个。[Rick]承认他们真正做的只是快速旋转和发出很酷的声音，但他们做得很好。</p> <p>更新:这篇文章发表后不久，我们发现了一篇由[Rick]撰写的关于如何用特斯拉涡轮制造搅拌机的指导。</p> </body> </html>
