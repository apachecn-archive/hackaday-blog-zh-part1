# 使用最少的工具进行细间距 SMD 焊接

> 原文：<https://hackaday.com/2012/03/13/fine-pitch-smd-soldering-with-minimal-tools/>

我们试图坚持 0805 部分，因为他们仍然足够大，以手工焊接。但是[Scott]向我们展示了在家里回流精细间距元件并不需要太多特殊工具。在这种情况下，他使用 0402 电阻，我们认为这种尺寸在功能上不可能用烙铁焊接。

他花了一些钱的等式的两个部分是专业生产的 PCB 和焊料模板。模板由 Kapton 激光切割而成，这种材料耐热，因此在切割过程中不会翘曲。一个丙烯酸框架将 PCB 固定在适当的位置，他只需用胶带将模板粘在上面，并用一块丙烯酸树脂作为橡胶滚轴来均匀地涂抹焊锡膏。在 PCB 和模板上挥霍意味着你将获得成功的容忍度。

下一个问题是放置组件。[Scott]展示了一些他用水族箱泵做的真空镊子。休息后观看视频，看看当他从胶带包装中取出一个电阻时，这些 0402 器件有多小。手动填充电路板(检查所有内容两次！)他将电路板移到一个完全没有改变的烤箱中进行回流。我们已经看到了很多为这些烤箱增加控制器的项目，但是他真的证明了你不需要控制器。相反，他使用一个由万用表读取的热电偶，只是为了让他知道温度是怎么回事。他用一部智能手机作为计时器，根据焊料的温度来开关烤箱。T3[https://www.youtube.com/embed/xqv9HF_2GlY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/xqv9HF_2GlY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
