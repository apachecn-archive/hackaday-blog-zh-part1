# 机器人来了

> 原文：<https://hackaday.com/2009/08/12/ecce-robot/>

[https://www.youtube.com/embed/cI9H4FoA0b4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/cI9H4FoA0b4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

根据视频，[幼儿保育和教育机器人是机器人领域的新典范](http://eccerobot.org/)。我们不知道这是真是假，但我们真的很喜欢这个驱动系统。他们使用弹性电缆和廉价的钻机马达作为肌肉，模仿人类的生物结构。观察一个简单的举臂动作所需要的复杂性是很有趣的。这确实表明这种类型的设置是低效的，但我们仍然认为它很酷。我们不明白为什么要使用廉价的电钻马达。除了成本，他们提到的控制问题似乎可以通过稍微好一点的电机设置来解决。话说回来，我们肯定他们也想到了这一点。这似乎是提出一个常见问题的绝佳时机。尝试用业余机器人来模仿我们的自然生物结构的低效率值得吗？需要什么样的技术才能让它的复杂性物有所值？
