# 使用乳胶漆和喷砂珠的投影屏幕

> 原文:[https://hack aday . com/2012/02/22/投影幕使用乳胶漆和喷砂珠/](https://hackaday.com/2012/02/22/projection-screen-using-latex-paint-and-sand-blasting-beads/)

![](../Images/367ccf92592cdbad0ddbb54b69302c7c.png "projection-screen")

这种建立自己的投影屏幕的方法对我们来说是新的。[西恩·迈克尔·拉冈]在 Harbor Freight 偶然发现了一些由微小玻璃珠组成的喷砂材料，并产生了灵感。他买了一个 50 磅重的袋子，开始着手[研究是否可以用它和普通乳胶漆一起制作一个投影屏幕](http://makeprojects.com/Project/Glass-Bead-Projection-Screen/685/1)。答案绝对是肯定的，但结果取决于你如何应用它。

现在你可以买到一种能把你的墙变成投影屏幕的涂料，但是它很贵。[Sean 的] hack 并不是直接的替代，因为他发现仅仅将珠子与颜料混合并将其应用于垂直表面的结果并不符合他所寻求的标准。但是如果你做一个屏幕挂在墙上，你可以让重力为你工作。他把屏幕放平，在表面涂了一层厚厚的油漆。然后，他在未干的油漆上洒了一层厚厚的玻璃珠，让它干透。最后，他把没有粘住的材料擦掉，挂在墙上。

这个黑客没有投影仪吗？没问题，自己[造](http://hackaday.com/2010/03/04/diy-projector-collection/)就行了。
