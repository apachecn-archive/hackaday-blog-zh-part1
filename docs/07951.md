# 插接板

> 原文:[https://hackaday.com/2011/06/28/the-pixboard/](https://hackaday.com/2011/06/28/the-pixboard/)

[![](../Images/df46c8c21caf06901800c18d5b0d17e7.png "pixboard")T2】](http://hackaday.com/2011/06/28/the-pixboard/pixboard/)

当然，许多黑客本身就很有趣。然而，像这样的黑客通过帮助人们克服残疾实现了一个更大的目标。 [PixBoard](http://cristiancasorran.com/pixboard/ "the PixBoard") 是“一种适用于残疾人，特别是脑瘫患者的照明板，让他们能够使用 shift 技术玩游戏。”

这个设备是为一个患有脑瘫并且不能有效控制胳膊或腿的女孩开发的。为了更有效地与世界互动以玩棋盘游戏之类的游戏，开发了这个系统。PixBoard 扫描一个 8×8 的 led 矩阵，允许她选择要移动的棋子和移动的位置。这仅使用简单的按钮和处理器的定时器来完成。

这个系统是由 Arduino 控制的，所以任何人都可以根据具体情况为这个系统创建自己的游戏。下面的视频是西班牙语，但显示了这个设备在 1:20 左右的行动。 [https://www.youtube.com/embed/5_AF6t9jtK8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/5_AF6t9jtK8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T4】
