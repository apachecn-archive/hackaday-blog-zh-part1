# 燃烧的热砖玩游戏，促进肌腱炎

> 原文:[https://hack aday . com/2010/04/21/flaming-hot-brick-plays-games-promotes-tendonitis/](https://hackaday.com/2010/04/21/flaming-hot-brick-plays-games-promotes-tendonitis/)

![](../Images/c49f061f701cf4933ebed3532a8da28f.png "tiny-n64-portable")

这场持续的争夺最小的便携式控制台的战斗继续进行着。与我们看到的[最后一个](http://hackaday.com/2010/01/25/gorgeous-portable-n64-built-to-order/) [两个](http://hackaday.com/2008/12/27/nintendo-sixtyfree-lite-r-portable-n64/)不同，这个版本选择了上下而不是并排的控制方案。这导致了一个小的整体尺寸，但因为这是一个厚砖，我们不知道玩几个小时会不会让你从肘部以下严重受伤和残废。[Bentomo]可能也有同样的想法，因为他也建立了一个突破连接器来使用原来的控制器(并在那个小屏幕上与多个玩家一起玩)。如果你像我们一样是建造细节的奴隶，看看他的建造日志和休息后的视频。
