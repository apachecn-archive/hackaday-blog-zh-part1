# IPod 和 IPhone 的外置 GPS

> 原文：<https://hackaday.com/2009/10/09/external-gps-for-ipod-and-iphone/>

![ipod_touch_gps](img/ca0e5f4048c0b4095c1b078ee480a31a.png "ipod_touch_gps")

曾经希望你的 iPod touch 或老一代 iPhone 有 GPS 功能吗？现在，它可以通过使用一个蓝牙 GPS 模块和 roqyGPS 应用程序来实现。

4 月份，我们看到了一个非常有创意的方法，通过使用自制配件将 GPS 添加到 iPod 中。新应用是一个更好的解决方案，因为它利用了 iPod touch 更大的屏幕和更多功能的用户界面。我们很高兴看到这种情况的出现，因为我们[不愿意仅仅为了获得 GPS 支持而升级到 iPhone 3G。roqyGPS 有一个相当长的](http://hackaday.com/2008/06/09/hackit-what-to-do-with-a-1st-gen-iphone/)[支持的硬件列表](http://www.roqy-bluetooth.net/hardware.html)，这应该使它相对便宜地获得一个出售或二手的 GPS 模块。
