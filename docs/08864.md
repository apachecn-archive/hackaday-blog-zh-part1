# 可以用咖啡和维生素 C 冲洗胶卷吗？

> 原文：<https://hackaday.com/2011/10/18/can-you-develop-film-with-coffee-and-vitamin-c/>

你是不是有点想买个电影摄影机？也许你想尝试用咖啡和维生素 C 代替传统的冲洗溶液来冲洗照片。[丹麦人 Puthan Valiyandi]很好地向我们介绍了他所采取的步骤，包括精确的测量、温度和实现良好结果的时间安排。这可能不是第一次，因为他确实使用与传统开发方法相关的特殊设备。

这个过程使用了一些容易获得的材料:速溶咖啡、维生素 C 粉和洗涤苏打(碳酸钠)。一旦胶卷曝光，它就被放在夹具上进行冲洗(丹麦人在开灯的情况下进行冲洗，以使休息后的视频值得观看，但你需要在黑暗中进行)。一旦进入开发容器，他就混合一批他的 diy 开发者，并根据时间表进行搅拌。当显影完成后，化学定影剂——这里没有使用 diy 替代物——被加入以固定胶片。擦干纸条，用扫描仪将你的作品数字化。我们对成品的质量感到惊讶，但我们不应该…他当然知道自己在做什么。
