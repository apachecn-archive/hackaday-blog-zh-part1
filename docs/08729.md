# DIY 信号发生器探头

> 原文:[https://hackaday.com/2011/10/03/diy-signal-generator-probe/](https://hackaday.com/2011/10/03/diy-signal-generator-probe/)

![](../Images/07d2472c2baf5a60b01fe01475bc1bd4.png "probe")

信号发生器是一个很方便的工具，只要有合适的组件，就很容易制作一个。制造合适的信号发生器探头完全是另一回事。[Frank]的 [DIY 信号发生器探针](http://blog.frankvh.com/2011/10/02/do-it-yourself-signal-generator-probe/)确实如其声称的那样，而且启动起来非常便宜。

在[弗兰克]用身边的几个零件做了一个简单的信号发生器后，他需要一个探针。他不想处理电路周围松动的电线，决定改装一个示波器探头。六美元和两个星期后，[弗兰克]在他的门挡上安装了一个合适的示波器探头，从地球的另一端运来。

探针上的应变消除被移除，并且 PCB 上的电阻器和调整帽被脱焊。剩下要做的就是将一段电线从 BNC 插孔焊接到探针引线上。将应变消除器放回原处，并清楚地标记为用作信号发生器探针。工作 10 分钟就不错了。
