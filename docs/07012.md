# 自动驾驶汽车已经行驶在我们的道路上

> 原文：<https://hackaday.com/2011/03/05/autonomous-cars-already-drive-the-roads-among-us/>

谷歌现在正在 TED 大会上展示这款自动驾驶汽车，但是它的硬件已经可以从旧金山自动行驶到洛杉矶。根据休息后视频中的解说，上面的场景显示汽车在封闭的球场上“拖着普锐斯的屁股”。这辆车在由人驾驶时学习了这条路线，现在这辆车将带着乘客在锥形装饰的停车坡道上通过一个积极驱动的环路。但是在公路上，你不需要教它任何东西。从一个地点旅行到另一个地点时，选择 GPS 路线并遵循道路规则是没有问题的。

上面的链接不包括硬件信息，但他们确实指向了《纽约时报》的一篇文章，其中包括一张信息图。汽车顶部的旋转盒是半径 200 英尺的 3D 测绘激光雷达。其中一个车轮上有一个旋转编码器，用于精确的运动数据，前后保险杠上有雷达传感器，后视镜上有一个摄像头用于图像处理。这让我们想知道当汽车被道路垃圾覆盖时，该系统的表现如何？也许你只是为每个传感器增加一个专用的雨刷。
