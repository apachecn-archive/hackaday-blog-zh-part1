# 纳米级 3D 打印

> 原文:[https://hackaday.com/2012/03/14/nanoscale-3d-printing/](https://hackaday.com/2012/03/14/nanoscale-3d-printing/)

![](../Images/55575f756a611aa637c03e6afe8da485.png "bridge")

这个伦敦塔桥的 3D 打印模型只有 200 微米长。形象地说，两座塔之间的距离只有一根头发的宽度。该模型是维也纳技术大学增材制造系研究的[产品](http://www.tuwien.ac.at/en/news/news_detail/article/7444/)

模型的制作很像普通的立体平版印刷术——一束激光照射到一桶光敏树脂上。当暴露在光线下时，树脂会硬化，模型会一层一层地建立起来。这些纳米尺度的模型是使用一种叫做“双光子光刻”的工艺制作的，我们不会假装完全理解这种工艺，但是[这里有](http://rogers.matse.illinois.edu/files/2006/optexpr2photon.pdf)一篇很好的论文提供了一个很好的概述。不用说，这些印刷品展示的精度几乎是可笑的。研究人员声称精度为 1 米，对于非常高科技的加工应用来说，这是相当高的精度。
