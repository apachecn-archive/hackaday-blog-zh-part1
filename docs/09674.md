# 使用这款超棒的喷雾蚀刻机器，在几分钟内制作出 PCB

> 原文:[https://hack aday . com/2012/02/11/create-PCBs-in-just-minutes-with this-awesome-spray-etching-machine/](https://hackaday.com/2012/02/11/create-pcbs-in-just-minutes-with-this-awesome-spray-etching-machine/)

![pcb-sprayer](../Images/21be4eaca71f94716d03300c67b940fa.png "pcb-sprayer")

如果你曾经在家里制作过自己的 PCB，你就会知道这是一个相当耗时的过程。花 20 分钟左右的时间手动搅动一块板是一种拖累，虽然水族馆的起泡器设置改善了这一过程，但它们远非理想。[Christian Reed]知道如果他真的想提高 PCB 生产的效率，他必须效仿那些大公司，建造自己的 PCB 喷雾器。

他的喷雾蚀刻机装在一个定制的丙烯酸外壳中，大部分是以前项目的废料。它包含两个隔间-一个用于在 PCB 上喷洒蚀刻剂，另一个用于冲洗完成的作品。至少可以说，该系统令人印象深刻，其特点是错综复杂的管道，使他能够轻松地蚀刻电路板和管理他的化学品。

[Christian]说虽然零件清单一开始看起来令人望而生畏，但是组装设备真的很容易。鉴于他可以在两分钟内蚀刻和清洗一块电路板，我们认为任何努力都是值得的。

[Christian]指出，他无法在网上任何地方找到制造这种 PCB 喷雾器的指南，所以他煞费苦心地详细记录了这一过程，以便尽可能容易地复制他的工作。一定要看看下面的视频，看看他的蚀刻坦克在行动-我们非常肯定，它会让你渴望在这个周末建立一个。
