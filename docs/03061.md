# 制作等离子切割机

> 原文:[https://hackaday.com/2009/01/06/make-a-plasma-cutter/](https://hackaday.com/2009/01/06/make-a-plasma-cutter/)

![plasma](../Images/9050a7b5d51058759de157b83b6d3526.png "plasma")

对于那些不幸的没有等离子切割机的人来说，[jandgse812]是来帮助你的。他带你从头开始[建造自己的等离子切割机](http://www.instructables.com/id/HOMEMADE_PLASMA_CUTTER/)的过程。将零件清单加起来，整个项目大约需要 300 美元。他包括了所有线路的可视图表，以及具体的零件号和获取它们的位置。小心，这可能非常危险，但也非常酷。你可能需要得到一个碟中谍主题的副本，以便在你使用它的时候播放。至少，[jandgse812]是这么建议的。休息之后你可以看到一段视频。
