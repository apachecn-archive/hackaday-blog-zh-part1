# Wii 适合魔兽世界

> 原文:[https://hackaday.com/2008/06/13/wii-fit-world-of-warcraft/](https://hackaday.com/2008/06/13/wii-fit-world-of-warcraft/)

[https://www.youtube.com/embed/ucEV7N0N9Pw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ucEV7N0N9Pw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
现在你可能已经看过来自德国人工智能研究中心(DFKI) [的两名研究人员通过使用 Wii Fit 平衡板](http://www.youtube.com/watch?v=lKUDU9lE--E)在谷歌地球上导航地图的视频。他们现在已经走得更远了，用这块板来[导航魔兽世界](http://www.youtube.com/watch?v=ucEV7N0N9Pw)。很明显，该板可用于任何 3D 环境。黑客完全是基于软件的，因为电路板是完全未经修改的。它通过蓝牙将数据传输到笔记本电脑，压力数据通过用 C#编写的自定义应用程序转换成方向指令。DFKI 的网站上没有关于该项目的说明，但我们打赌一旦所有的问题都解决了，他们会向公众发布该软件。
