# 自画像机

> 原文：<https://hackaday.com/2009/07/02/self-portrait-machine/>

![selfportraitmachine](img/49e515a072ad6b89813e6584bf167e69.png "selfportraitmachine")

[Jen Hui Liao]发明了一种引导用户为自己画像的装置。这部名为[自拍机器](http://www.we-make-money-not-art.com/archives/2009/07/selfportrait-machine.php)的电影评论了社会中有多少东西是由机器创造的，以及我们对它们的依赖。与[之前的绘图机器人](http://hackaday.com/2008/06/08/portrait-robot/)不同，用户是草图绘制过程的一部分。该机器握住用户的手，并使用步进电机和伺服系统来移动它们，就像一只标志海龟一样。廖承诺将很快提供更多的细节。机器跳跃后的视频。
