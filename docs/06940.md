# DIY 闪电特效

> 原文:[https://hack aday . com/2011/02/26/DIY-lightning-special-effects/](https://hackaday.com/2011/02/26/diy-lightning-special-effects/)

![](../Images/eacc204007060f95f1502a73fbc3e8fc.png "Lightning-effect-using-camera-flash-units")

万圣节可能来了又去，但没有理由不看看这个整洁的小特效设置。基本上，它使用模拟电路来监控音频信号，并使用 5V 继电器触发一些相机闪光灯。这个想法是，你可以播放雷击和其他怪异的声音，系统将触发相机闪光灯，以配合雷击。除了相机闪光灯之外，增加一些颜色器官会使你的灯光变暗，以帮助实现一个类似打雷的效果。不幸的是，没有任何关于颜色器官(技术上可能只是光器官)的示意图，但这并没有减损看似设计良好的模拟信号处理。休息之后，请查看实际情况。
