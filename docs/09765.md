# 2012 年 2 月 25 日每周综述

> 原文：<https://hackaday.com/2012/02/25/weekly-roundup-22512/>

![](img/75ecefa9b2814c00b010823d9ba9bee8.png "weeklyRoundup")


如果你第一次错过了它们，这里是过去一周最受欢迎的帖子:

排在第一位的是一个关于[Darrin]建造的 DIY 滑雪缆车的帖子。当你可以从山上滑下来，然后再滑回来的时候，你为什么要让一次费力的爬山毁了你的一天呢？好样的[达林]！你是我们那种懒！

我们第二受欢迎的帖子是对数码管的现代改造。这个装置是由许多层透明的丙烯酸树脂制成的，上面刻有数字。丙烯酸是边缘照明，以实现谢妮管外观相似。

接下来是一个关于[的帖子，这是一个检查手表](http://hackaday.com/2012/02/23/checking-the-accuracy-of-fake-watches/)准确性的聪明方法。【th3badwolf】是这样做的，以每分钟一帧的速度做一个视频，看看秒针是怎么做的。在理想情况下，秒针会在整个视频中保持在同一个位置。

看看这个！你认为数控机床需要有一个非常刚性的结构来控制切割钻头吗？这个没有。你用手拿着它！

最后，我们留给你一个上周的重复。我们这周没有发布，但是你们会不断回来看这个帖子，在这个帖子里，[超光速]和一些朋友把他们的结婚礼物扔进一个大的粘性方糖里，送给一对不幸的新娘和新郎。
