# 用 WiiMote 模拟 IPhone N64

> 原文：<https://hackaday.com/2009/12/23/iphone-n64-emulation-with-wiimote/>

[https://www.youtube.com/embed/ql0V4SNt-c4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ql0V4SNt-c4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[ZodTTD]发布了一款用于 iPhone 的[任天堂 64 模拟器。它可以在 Cydia 商店买到(需要付费),并且可以安装在越狱的 iPhones 上。视频显示 Wii Remote support 作为一个控制界面，使用按钮和加速度计，这是自从我们上次](http://gizmodo.com/5432501/n64-emulator-hits-jailbroken-iphones-with-bluetooth-wiimote-support)[看他的作品](http://hackaday.com/2009/12/10/wiimote-iphone-update/)以来的一个新增功能。没有关于双截棍功能的词，如果你要尝试 100% Mario64，这是必须的。
