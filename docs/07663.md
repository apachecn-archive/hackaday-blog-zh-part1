# 自动国际象棋从下面开始

> 原文：<https://hackaday.com/2011/05/23/automated-chess-set-does-it-from-below/>

这年头最酷的东西都能拿到班级学分。举个例子，[这个自动棋盘](http://acboard.blogspot.com/)是【布莱恩】和【詹姆斯】为他们今年春天的一个班级的期末项目做的。我们刚刚在周一看了一个机器人象棋设置，T2 使用安装在龙门架上的夹子来移动棋子。这款更加人性化，近乎神奇。这是因为活动部件都位于电路板下方，如果在边缘周围放置一个合适的盒子，就可以隐藏起来。

这个版本有两个主要组件。第一个是一个由簧片开关组成的网格，用来检测人的动作。这是可行的，因为玩家使用的每个棋子底部都粘有一个弱磁铁，其强度足以启动簧片开关，并让计算机感知刚才的移动。在机器人方面，这就像一个绘图仪。电脑的每个部件都有一个粘在底座上的金属圆盘。基本上相当于棋盘下的绘图仪使用稀土磁铁来抓取电脑的棋子，并将其拖到下一个游戏位置。

使用两个独立的磁系统带来了一些有趣的设计挑战。休息后，您可以在视频中看到该设备的运行情况，完整的文章和源代码包可在该功能顶部链接的博客中获得。但是为了您的方便，我们也在休息后镜像了 PDF 白皮书，展示了所有有趣的细节。
