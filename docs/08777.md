# 升级旧的超级示波器

> 原文:[https://hack aday . com/2011/10/09/upgrading-an-old-super-scope/](https://hackaday.com/2011/10/09/upgrading-an-old-super-scope/)

![rechargeable_super_scope](../Images/b799aec71b5d5c15c0a50bac66d0eade.png "rechargeable_super_scope")

[Brian Knoll]仍然相对频繁地使用他的超级任天堂，他只是无法获得足够的超级范围动作。如果你从未拥有过，超级瞄准镜会是一大乐趣，但它也是一个不可思议的耗电大户。它一箱一箱地吃 AA 电池，所以[布莱恩]想改用充电电池。由于镍氢 AA 电池不能在超级范围内使用，他自己设计了一个充电解决方案。

他首先计算玩 8 小时游戏需要什么样的电池，然后开始设计电路。他建造的电路板包含一个 DC/DC 转换器，以提供超级示波器所需的 9V 电压，以及内置的脂肪充电器。他让 BatchPCB 制作了他的电路板，在解决了一个小的生产错误后，他把所有的东西放在一起，给了他的改良示波器一个机会。

事情进展顺利，虽然他说他真的应该在电路中内置一个低压关断，但他对结果非常满意。
