# 自动鼓手

> 原文:[https://hackaday.com/2009/11/13/auto-drummer/](https://hackaday.com/2009/11/13/auto-drummer/)

[https://www.youtube.com/embed/COl1xmWAjUw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/COl1xmWAjUw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

将一杯机械工程与少量的架子鼓混合在一起，你最终会得到机器人鼓手史蒂夫。我们知道它使用了一个 [MSA-T Midi 解码器](http://highlyliquid.com/kits/msa-t/)，但这大约是已经被分享的程度。仅从观察视频来看，我们认为与我们在 2005 年看到的机器人鼓手 T4·皮尔特相比，史蒂夫有一些优势。史蒂夫的特点是每个鼓和符号都有两根棍子，似乎反应很快。

史蒂夫是伟大的，但我们仍然认为里克·艾伦的 T1 是最好的。虽然这是一个高质量的构建，但没有人可以替代一个可以以数百万种微妙不同的方式敲鼓的人。这并不是说我们看不到黑客的潜力。也许是时候更新一个经典的想法了，机器人管弦乐队。不知道我们的意思？休息后，请看 3 点 58 分的视频。
