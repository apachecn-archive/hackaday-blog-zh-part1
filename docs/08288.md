# 高压等离子灯也是有品位的蒸汽朋克

> 原文：<https://hackaday.com/2011/08/18/high-voltage-plasma-lamp-is-also-tasteful-steampunk/>

Instructables 用户 Aaron Ravensdale 上将]刚刚完成了一个高压等离子灯泡的制造，创造性地使用了现成的零件。作为一个自称蒸汽朋克的人，[拉文斯代尔上将]也因为在他的设计中加入了工作装置而赢得了一些信任。

建筑的核心是一个“闪烁的火焰”[蜡烛灯泡](http://www.youtube.com/watch?v=xPdzbOygJZs)。这些灯泡内部有两个火焰形状的板作为电极。蜡烛灯泡中充满了氖，而不是通常填充在白炽灯泡中的氩。当氩气被激发时，它会发出一种相当不自然的[紫色光芒](http://theodoregray.com/periodictable/NobleRack.html)——对于一根模拟蜡烛来说，这不是很有说服力，当然也不是蒸汽朋克。闪烁的蜡烛灯泡中的霓虹灯发出明亮的橙色，非常适合模拟火焰，肯定会在下午茶时给公爵夫人留下深刻印象。

在找到正确的等离子灯泡后，[上将]从一次性相机中清理出剩余的高压电子设备。将三个电极连接到一个黄铜齿轮上，整个装置利用一个旧时钟和一个 CD 驱动马达的零件旋转起来。我们总是对蒸汽船的清扫能力印象深刻——如果我们尝试这样做，我们仍然会等待我们的装备到达。休息之后，看看这个非常酷而且非常便宜的等离子灯泡的视频。
