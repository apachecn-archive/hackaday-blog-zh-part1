# OmniZero.9 多格式两足动物

> 原文：<https://hackaday.com/2009/09/28/omnizero-9-multi-format-biped/>

[https://www.youtube.com/embed/uqnxt9vGAmE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/uqnxt9vGAmE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

网络上流传着如此多的双足机器人，以至于我们往往会忽略它们。这个今天早上引起了我们的注意，因为它有改变布局的有趣能力。这种两足动物被命名为 [OmniZero.9](http://www.1mm.jp/m/robo.html) ，可以像汽车一样用 4 个轮子驱动，像两足动物一样行走，甚至可以载人。虽然它看起来肯定不像是最舒适的交通方式，但它看起来比一些来自大牌的[最新“创新”](http://www.youtube.com/watch?v=8T3pZhtAWvU)不那么尴尬。
