# 在 Eagle CAD 中使用 3D 鼠标

> 原文:[https://hack aday . com/2010/10/12/use-a-3d-mouse-with-eagle-CAD/](https://hackaday.com/2010/10/12/use-a-3d-mouse-with-eagle-cad/)

![](../Images/06ac3c48a70289687bc3fb79311408cf.png "3d-mouse-with-eagle-cad")

[Sebastian]用 Eagle CAD 制作了一个 3D 鼠标。他是在使用 3DConnexion 设备操纵 Solid Edge 3D CAD 建模软件副本中的视图后受到启发的。该系统一只手使用 3D 鼠标，另一只手使用普通鼠标。

事实证明，3DConnecxion 提供了许多工具来让他们的设备与您的软件一起工作。[塞巴斯蒂安]下载了。NET 示例，并对其进行修改以从设备中读入数据。通过热键的使用，他成功地实现了缩放、平移和居中(休息后自己看吧)。他实现的唯一缺点是在拖动部件时不能平移。但那只是一个等待你找到解决办法的小问题。
