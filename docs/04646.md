# 无人驾驶飞机向你复仇

> 原文：<https://hackaday.com/2009/12/30/uav-reigns-down-vengeance-upon-thee/>

![](img/95fb40b53498272aa143cd3c295935ea.png "uav-up-to-no-good")

失业的电气工程师可能是一件非常危险的事情。通过[建造一架武装无人机](http://www.youtube.com/watch?v=sh10v52DBLM)，Cybrown 已经将他的技能转向更黑暗、更可怕的应用。这是一架遥控飞机，驾驶舱里装有一架可移动的摄像机。视频和 GPS 数据被发送回飞行员，飞行员通过[可穿戴显示器](http://hackaday.com/2009/10/26/head-mounted-computer/)观看图像。我们打赌这没有[100 公里无人机](http://hackaday.com/2009/09/27/homemade-uav/)的射程，但这很好，因为这一架带来了来自天空的厄运。检查上图中的翅膀，这个遥控玩具可以放烟花。休息后我们嵌入了飞行片段和攻击视频。
