# 视频:ATmega328p 上的中断

> 原文：<https://hackaday.com/2011/11/17/video-interrupts-on-the-atmega328p/>

![](img/f79139cbfffb494f4f32540e96a15974.png "3pi")

本周，我们将为您带来我们系列的最后一个视频，其中[Jack]使用 3pi 机器人作为 ATmega328p 处理器的奇特开发板。今天的视频处理中断。如果你一直想让你的程序同时做多件事，中断就是解决办法。[Jack]讨论了在程序中使用中断的各种方法，然后展示了他如何创建驱动 3pi 蜂鸣器的中断例程。他还展示了启用、禁用和控制中断的例程。

由于这是这一系列视频的最后一篇，我们发布了之前所有视频的代码。点击[这里](http://www.buffingtonfx.com/temp/test.c)抢一份。

在我们的下一个系列视频中，我们将尝试一些更具挑战性的东西，所以在这里展示之前，我们很可能会花几周时间做一些开发。别走开，我们会回来的。
