# 笔画解锁

> 原文:[https://hackaday.com/2010/01/12/stroke-to-unlock/](https://hackaday.com/2010/01/12/stroke-to-unlock/)

![](../Images/790de77b0708589be3710bbb41da31d0.png "Stroke-Sensor")

如果你喜欢嵌入式衣服，这个[行程传感器](http://www.instructables.com/id/Stroke-Sensor/)就是为你准备的。正如在休息后的视频中所展示的那样，在特定方向上抚摸螺纹将产生一个电路，该电路可以感应并在这种情况下打开 LED。这个概念在一块氯丁橡胶的背面使用了两条导电总线。然后添加导电和不导电的线，以获得毛茸茸或硬毛的效果。当垂直于电源总线划动时，导电线聚集在一起并形成电路。

出于某种原因，这似乎有点令人毛骨悚然，但也许这只是因为我们还没有想出正确的应用技术。我们非常确定，一件带有 [LED 字幕](http://hackaday.com/2009/12/14/fabric-display/)和“多毛”背部的运动衫是错误的应用。
