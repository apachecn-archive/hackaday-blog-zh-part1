# Arduino 笔记本外壳便于在任何地方进行修补

> 原文:[https://hack aday . com/2011/12/21/arduino-notebook-cover-make-it-easy-to-tinker-anywhere/](https://hackaday.com/2011/12/21/arduino-notebook-cover-makes-it-easy-to-tinker-anywhere/)

![arduino-notebook-cover](../Images/fa4a08a3f60046c3529b1a08c89a27c2.png "arduino-notebook-cover")

[Erv]正在整理他的假日购物清单，决定不去商店给他的朋友买东西，而是给他们一些更有用的东西。Erv 以前是一名电子工程师，他通常更喜欢 PIC 微控制器，但他说 Arduinos 用于原型制作非常方便，所以他喜欢手头总是有一个。

他认为他的朋友可能也喜欢轻松使用 Arduino，所以他为他们制作了一些光滑的 ZapBook 封面，使他们能够随时随地拥有一个原型平台。盖子由 PCB 制成，包括一个用于 Arduino Pro Mini 的插座，以及一些内置 led。他还从 Arduino 扩展了一些其他 I/O 引脚，但他说连接 led 的小型焊接桥可以在必要时拆除，轻松释放 8 个额外的引脚。我们非常热衷于便携式原型装置的想法，尽管[Erv]在他的设计中加入了一个带亮眼的日黑骷髅也没什么坏处！

我们不确定他是否打算发布电路板的原理图，但是这些笔记本对于任何主持 Arduino 编程初学者课程的黑客空间都非常有用。
