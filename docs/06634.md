# 建立一个令人毛骨悚然的娃娃军队

> 原文:[https://hack aday . com/2011/01/24/building-a-creepy-doll-army/](https://hackaday.com/2011/01/24/building-a-creepy-doll-army/)

![](../Images/10cb301844652e03bb5e2a60728935ce.png "creepy-doll-army")

想要一个自己的玩具娃娃来吓唬室友吗？[Gzip]向你展示如何通过给旧玩偶添加基于伺服的电子动画来实现这一点。在休息后的视频中，你可以看到娃娃举起她的手臂，并转动她的头，这要归功于每个肩膀和她的甜瓜上的一个电机。你在视频中看不到它，但腿也是电动的，这意味着一些创造性的编码可能会让这个老女人笨拙地爬过房间(手里拿着刀)。话又说回来，也许这正是你需要的灵感，让你振作起来，完成[圣诞老人挑战赛](http://hackaday.com/2010/12/27/santa-pede-challenge-take-two/)。提交截止日期是从今天起的一周内！
