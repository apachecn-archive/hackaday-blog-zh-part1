# 使用手指通过网络摄像头控制世爵

> 原文：<https://hackaday.com/2009/04/14/controlling-spykee-via-web-cam-using-your-fingers/>

![spykee](img/6a00c2cae793272d2e7a721a9bb8a09f.png "spykee")

[epokh]提交了这个很酷的项目，他写了一些自定义代码来使用手势控制 Spykee 机器人。他通过网络摄像头过滤掉除绿色以外的所有东西，然后用绿色胶带裹住手指。然后他运行一系列的过滤器来稍微清理一下。由此产生的“斑点”被跟踪并转换成运动指令。休息之后，您可以在视频中看到实际设置。这个家伙可能看起来很熟悉，因为我们最近发布了一个他用乐高做的超级快速[头部跟踪装置。你们中的一些人在评论中提到，乐高是一种浪费，你会发现他也是这样认为的，](http://hackaday.com/2009/04/03/lego-head-tracking/)[最终制造了一个简单的钻机来代替乐高](http://www.epokh.org/blog/?p=194)。
