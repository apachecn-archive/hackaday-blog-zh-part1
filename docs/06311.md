# 纸质触摸板

> 原文:[https://hackaday.com/2010/11/16/paper-touchpad/](https://hackaday.com/2010/11/16/paper-touchpad/)

![](../Images/247dec5bd7ccf36af1e450ef5f61ba9b.png "paper-touchpad")

如果你不介意弄脏你的手指，你可以用一张纸代替你的鼠标。West 博士]自己制作了这个触摸板，它使用四个分压器测量纸张角落的信号。这种纸已经完全被铅笔上的石墨覆盖(这种石墨[我们不时在 hacks](http://hackaday.com/2009/12/20/better-resistors-from-a-pencil/) 中看到)，使其具有导电性。用户戴上一个防静电腕带，使他们的手接地，当手指完成电路时，Arduino 可以计算两个轴上的接触点。休息之后，看这个控制视频中的光标。
