# 颜色传感器给出任何东西的 RGB 值

> 原文:[https://hack aday . com/2012/02/07/color-sensor-gives-the-RGB-values-of-any/](https://hackaday.com/2012/02/07/color-sensor-gives-the-rgb-values-of-anything/)

![](../Images/eabfdf93b01a6884b20aea698f85906a.png "color")

[Rick Osgood]想要建立一个颜色传感器，可以举起任何物体来获取 RGB 颜色值。他最初用光敏电阻和几个发光二极管开始，但无法让它可靠地工作。[Rick]在 Adafruit 上找到一个数字亮度传感器后，终于完成了他的颜色传感器,最终得到了一个非常准确的判断事物颜色的硬件。

颜色传感器背后的想法是点亮红色、绿色和蓝色 led，并用[亮度传感器](http://www.adafruit.com/products/439)观察有多少光从物体反射回来。[Rick]选择了一个 Arduino 来完成所有繁重的光线传感器和激活 led 的工作。

经过几次测试后,[Rick]让他的颜色传感器工作了，但是它没有达到他预期的标准。这并不是一个真正的问题:led 可能没有相同的亮度，亮度传感器在整个彩虹上的响应也不均匀。不过，这些事情总是可以通过软件解决的。这是一个很好的项目，可以作为这款拾色器笔原型的一部分。
