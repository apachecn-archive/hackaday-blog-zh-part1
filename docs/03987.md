# 单体主流:莱特曼表演

> 原文:[https://hack aday . com/2009/09/08/monel-mainstream-performance-on-letterman/](https://hackaday.com/2009/09/08/monome-mainstream-performance-on-letterman/)

![monome_played_on_letterman](../Images/d2b30ba564b3202906a24799453f4eb7.png "monome_played_on_letterman")

这种单体被发现用于伊莫金·希普和大卫·莱特曼在《深夜秀》中的表演。伊莫金使用[单体 256 型号](http://monome.org/series/)连接到放在钢琴上的笔记本电脑。在她的表演中，她结合了现场样品和预先录制的循环，证明了这一产品在一位有成就的艺术家手中是多么伟大。

虽然没有指明名字(或功能)，莱特曼确实注意到了表演结束时的单体。看到一个创新的开源产品有这样的曝光率真是太好了。看看莱特曼剪辑以及休息后的单体奖金。

[https://www.youtube.com/embed/54X7t8bDq_4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/54X7t8bDq_4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)[https://player.vimeo.com/video/6255134](https://player.vimeo.com/video/6255134)

这个额外的视频以一个单体为特色，它不仅可以播放音频片段，也可以播放视频。通过这种方式，您可以同时观看现场视频表演。几周前，艾略特偶然发现了它，并提到它让他想起了[业余爱好者——拉塞·杰特森](http://www.youtube.com/watch?v=JzqumbhfxRo)的视频。我们同意并认为单体是制造这种产品的最佳选择。
