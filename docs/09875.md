# 一个人体模型头+ Arduino +网络摄像头=许多令人毛骨悚然的乐趣！

> 原文:[https://hack aday . com/2012/03/14/a-mannequin-head-arduino-web cam-lot-creepy-fun/](https://hackaday.com/2012/03/14/a-mannequin-head-arduino-webcam-lots-of-creepy-fun/)

[![styrofoam head robot](../Images/1f070ed0ec4fe1d3021969e65e74e5b4.png "styrofoam-head")T2】](http://hackaday.com/2012/03/14/a-mannequin-head-arduino-webcam-lots-of-creepy-fun/styrofoam-head/)

这个[人体模型头部](http://www.spaceduino.com/2012/03/head-part-1.html "Styrofoam Mannequin Head")是几年前以不到 3 美元的价格购买的。就像购物时看到的许多东西一样，它当时没有目的，但似乎以后会有用。添加一个 Arduino，一些伺服系统，以及以类似方式获得的电子部件，你就拥有了一个酷黑客所需的所有成分。

休息后的视频中很好地记录了这一构建，我们尤其喜欢在 2:24，当我们认为是母亲的人说“看看这一团糟！”显然，下一次迭代将是一个机器人来清理一切！

这个迭代给人留下了深刻的印象，因为它使用一个摄像头来跟踪使用伺服电机的对象，并列出所使用的代码。对于 it 跟踪内容的视图以及 PC 视图，快进到 8:45 左右。除了使用伺服系统跟踪零件之外，非网络摄像头的眼睛还会根据它是否在跟踪而从绿色变为黄色。它还配有一条闪烁的项链，这在我们看来也是一个优点。
