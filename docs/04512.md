# 用于自动鸡尾酒的简单液体分配器

> 原文:[https://hack aday . com/2009/12/07/simple-liquid-dispenser-for-auto-cocktails/](https://hackaday.com/2009/12/07/simple-liquid-dispenser-for-auto-cocktails/)

![](../Images/3e49c1cce58db49adbdeaa169d32d2e8.png "diy-liquid-doser")

[Qdot]想出了一个简单的方法来分配液体，用于他的巴特里斯项目。正如你在上面看到的，挠性管连接到一些装液体的倒置瓶子上。一根筷子的一端系在木板上，另一端通过绳子连接到伺服系统上。当伺服机构转动时，它将筷子紧紧地拉在板上，切断了液体通过管道的流动。这没有 [Bar2D2](http://hackaday.com/2009/09/15/bar2d2-on-the-discovery-channel/) 使用的系统优雅，但也便宜得多。
