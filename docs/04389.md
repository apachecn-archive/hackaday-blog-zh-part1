# Gamecube To Go

> 原文:[https://hackaday.com/2009/11/08/gamecube-to-go/](https://hackaday.com/2009/11/08/gamecube-to-go/)

![gamecube-to-go](../Images/8b9f53e049a9f649ddf5c2d85d9a3f42.png "gamecube-to-go")

建造了一个手持游戏立方体，这样他就可以随身携带他的游戏。最终产品相当不错，提供了一个大显示屏和大约 3 小时的锂聚合物电池播放时间。

从 Kidz Delight Datamax 游戏中的案例开始，他使用了 Bondo ABS 水泥和塑料保险杠填充物来改变案例，但仍然提供了专业的外观。显示器是一个五英寸的 PlayStation One 液晶显示屏，他还在上面安装了扬声器。至少有四个控制器被拆卸下来用作按钮、操纵杆、触发器和方向键。我们最喜欢的功能是安装在背面的完全暴露的光学头。

休息之后，我们嵌入了视频和光驱的图片。这远远不止是让 [GameCube 成为一个一体化系统](http://hackaday.com/2008/12/14/gamecube-with-built-in-screen/)。如果你喜欢这个版本，可以在[【hail Razer 的】YouTube 频道](http://www.youtube.com/user/jonjandran)上查看任天堂 64 和 Dreamcast 的 mods。
