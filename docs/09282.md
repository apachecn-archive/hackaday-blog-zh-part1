# Muff-Fones 听起来很脏；嗯，不是字面意思

> 原文：<https://hackaday.com/2011/12/12/muff-fones-sound-dirty-well-not-literally/>

![](img/adb8c6823cc00e684f63f8954b9eda68.png "muff-phones")

宝贝，外面很冷。但这并不能阻止格里西尼在不在家的时候听一些音乐。他[给一对暖耳器](http://www.instructables.com/id/Muff-Fones)加了耳机。我们猜想你可以称它们为耳罩，这就是这个名字的由来。但这是包在你后脑勺上的新型号。

[Grissini]拿起一套耳机，同样环绕在你的后脑勺。将扬声器从塑料外壳中取出后，他需要一种方法来软化压在耳朵上的尖锐边缘。Sugru 再次展示了它的多功能性，提供了一个柔软、自粘合、可塑的表面。最后一步非常简单，因为暖耳器的每只耳朵旁边都有一个织物口袋，非常适合容纳扬声器。

现在我们需要更进一步，让它们无线化。我们认为[侵入蓝牙耳机板](http://hackaday.com/2009/12/23/bluetooth-handset-hacks/)可以让它与你的手机兼容。或者你可以卷起你自己最小的 MP3 板，把它放在脖子上。
