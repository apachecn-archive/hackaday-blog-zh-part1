# 遥控 PPM 被黑

> 原文:[https://hackaday.com/2010/10/02/rc-ppm-hacked/](https://hackaday.com/2010/10/02/rc-ppm-hacked/)

![](../Images/59da2c438c7effe6701d9841308cd544.png "If model aircrafts are designed to be a simulator for real aircraft. Then what does that make model aircraft simulators running on a PC? ")

[托马斯·普费菲]利用模型飞机无线控制器产生的 PPM 信号，通过 ATMega8，[将信号转换成 USB 操纵杆](http://thomaspfeifer.net/)。这意味着你现在可以用一个标准的遥控装置在你的电脑上驾驶模型飞机。当然，现在 PPM 解码后，你也可以用这个信号来控制任何电子设备。像你的[割草机](http://hackaday.com/2010/05/03/remote-control-reel-mower/)、 [iPod](http://hackaday.com/2009/09/08/arduino-as-ipod-remote-control/) ，我们甚至见过[遥控弹丸枪](http://hackaday.com/2009/12/10/remote-control-pellet-gun-with-scope/)。捕捉[托马斯]在跳跃后驾驶模拟的[四旋翼直升机](http://hackaday.com/2010/09/08/easy-quadrotor-helicopter-instructions/)的视频。
