# RFID 控制概念

> 原文：<https://hackaday.com/2009/12/26/rfid-control-concept/>

![](img/1b4d0fb525611908440e4ad2649ef54e.png "rfid-control-concept")

那不是烟雾探测器，是一个 [RFID 探测圆盘](http://www.exothermia.net/monkeys_and_robots/2009/12/14/puck/)。[埃里克]和[布伦特]一直致力于这个概念，以生产一个虚拟旋钮。当该设备检测到 RFID 标签时，它会振动，然后圆盘可以像虚拟旋钮一样转动，加速度计拾取运动数据。该建筑采用 XBee 模块进行连接，并封装在一个印刷盒中。

休息后的视频显示了控制彩色灯光的设备。每种颜色都有不同的标签，当阅读器在其中一种颜色上时，可以旋转圆盘来调高或调低亮度。我们预见到它的许多重要用途。通过在组合中添加字符显示，将它变成公共场所的恒温器。一个标签可以贴在墙上，当你想改变恒温器设置时，只需将游标放在标签上，然后转动，直到屏幕显示所需的温度。
