</span> <p>看看动态自主伸展的六足机器人(DASH)。上图中的<a href="http://www.youtube.com/watch?v=LsTKAtBBkfU" target="_blank">视频是在 2009 年智能机器人和系统国际会议上展示的。在其中，我们看到了加州大学伯克利分校仿生纳米系统实验室团队的辛勤劳动。他们开发了一种机器人推进系统，模仿蟑螂和其他昆虫身体的一些最佳方面:速度，运动的经济性，从大瀑布中幸存而不受损的能力，以及穿越障碍的能力。休息之后让我们来看看他们是如何组合的。<span id="more-17222"/></a></p> <p>我们看到一个用纸板和聚合物薄片制造的六足机器人。首先，每个零件的一般特征是从一张纸板(非瓦楞纸)上激光切割成镜像图案。然后将聚合物片粘合到图案的一侧，另一侧折叠并粘合到聚合物的顶部。整个过程需要通过层压机，然后回到激光切割机完成切割。如果你有这些工具，这将使构建过程非常快。他们声称机器人可以在一小时内组装好。</p> <p>通过这种特殊的设计，机器人的整个身体提供推进力。有两个相对旋转的不同框架，每个框架以划动的方式移动三条腿。通过使用肌肉线对角弯曲整个框架来提供定向推进。由于这种建筑材料的弹性，框架弯曲和恢复形状的能力也为跌落提供了保护。我们看到该装置从建筑物上被扔出，并继续运行，没有任何明显的损坏。</p> <p>一定要看一看伯克利团队之前在他们网站上发布的<a href="http://robotics.eecs.berkeley.edu/~ronf/Prototype/index.html" target="_blank">折叠机器人原型信息</a>。他们更深入地探讨了为车身生产层压纸板复合材料的细节。视频中的工作是他们原型的一大飞跃，让我们想知道接下来会发生什么？</p> <p>[via <a href="http://spectrum.ieee.org/blog/robotics/robotics-software/automaton/dash-hexapedal-cockroach-inspired-robot" target="_blank"> IEEE 频谱</a></p> </body> </html>
