# 制作轨道炮(再次！)

> 原文：<https://hackaday.com/2010/01/01/making-a-rail-gun-again/>

[Rp181]又开始了他的第二版[轨道炮项目](http://rp181.wordpress.com/category/railgun-build-log-v2/)(链接失效，试试[互联网档案](http://web.archive.org/web/20100527233706/http://rp181.wordpress.com/category/railgun/railgun-build-log-v2/))。[最初的](http://hackaday.com/2009/03/20/making-a-rail-gun/)确实有一些 18 个 400V 3900uf 电容器的电力，但他现在用 40 个电容器来增加电力！达到两倍以上的能量，12kJ 对 5.6kJ！其他一些变化包括一个新的注射器螺线管设置和他的[断线计时器](http://rp181.wordpress.com/category/data-acquistion/)的修订版 3。遗憾的是，他没有提到这是否和他的第一把轨道炮一样环保。查看注射器点火的视频和解释跳跃后一些新更新的动画。
