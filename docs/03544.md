# WiNet，无线 Arduino 触摸屏

> 原文：<https://hackaday.com/2009/05/30/winet-wireless-arduino-touchscreen/>

![winet](img/768ba36e7ddb162f29dead97c3eecaed.png "winet")

Liquidware 已经组装了一个有趣的工具包，叫做 [WiNet](http://antipastohw.blogspot.com/2009/05/winet.html "Liquidware Antipasto: WiNet") 。它有一个由 Arduino 控制的电池供电的触摸屏。使用 XBee shield，它可以向连接到计算机的配对 XBee 和 Arduino 发送命令。计算机也可以更新触摸屏显示。下面的视频演示了这一点。这是一个独特的界面，但 TouchShield 本身就要 174 美元，所以我们猜测在你走这条路之前，你会对你的项目非常有信心。
