# 任天堂将你所有的任天堂控制器连接到一台电脑上

> 原文:[https://hackaday . com/2011/11/30/funtendo-connects-all-your ' s Nintendo-controllers-to-a-PC/](https://hackaday.com/2011/11/30/funtendo-connects-all-your-nintendo-controllers-to-a-pc/)

![](../Images/b837649bdd1605bb03dad1f96cb3d2d0.png "funtendo")

对于那些每当我们想玩时间的陶笛或 NES 时就懒得去找 N64 的人来说，模拟器是天赐之物。不过，有一个问题。键盘无法提供经典的 NES“砖块”或 N64 tritopus 控制器那样的体验。进入 [Funtendo](http://www.gadgetgangster.com/news/54/518) ，这是一个将你所有喜爱的控制器转换成 USB 的分线盒。

Funtendo 使用带有端子板模块的小工具 Gangster [螺旋桨平台](http://www.gadgetgangster.com/find-a-project/56.html?projectnum=257)。组装电子设备相当容易；只需剥去控制器的末端，将它们拧紧到接线板上。任天堂支持 N64、NES 和 Wii Classic 控制器。选择经典控制器而不是超级任天堂控制器降低了构建的复杂性。经典控制器可以玩 SNES 游戏，并使用 I2C 总线，使其更容易布线。

为了将控制器连接到计算机，[视差推进器工具](http://www.parallax.com/ProductInfo/Microcontrollers/PropellerGeneralInformation/PropellerMediaPage/tabid/832/Default.aspx)、视差串行终端和 [PPJoy](http://ppjoy.bossstation.dnsalias.org/) 将按钮混合转换成仿真器可读的按钮。构建可能比从阁楼上拉出一个 NES 花更多的时间，但即使有一个大的项目箱，它占用的空间也少得多。
