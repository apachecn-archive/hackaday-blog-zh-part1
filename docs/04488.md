# 破坏光学介质

> 原文：<https://hackaday.com/2009/12/01/destroying-optical-media/>

![](img/b28414e77ae1f342e14ca9b397c235e6.png "destroying-optical-media")

我们得到了一条关于 USB 光盘销毁器的线索。我们发现它的方法很有趣，因为它只是刮擦 CD，如左图所示。如果你真的有数据安全问题，也许应该使用比刮擦塑料更多的东西。有很多碎纸机也可以粉碎 CD，那么把那个电机烧坏的碎纸机改造成手摇碎纸机怎么样？

有大量需要移除的光学介质吗？这些人开发了自动进给的 CD 粉碎链枪。这包括一个 CD 碎纸机和一系列用包装带连接的光盘。正如在休息后的视频中看到的那样，碎纸机向前推进，下一张光盘被拉入它的嘴里。

微波已经成为流行的光盘媒体的死亡使者。[灯光秀和由此产生的混乱艺术](http://www.youtube.com/watch?v=0JkCIfLE_-M)(上图右侧和休息后)使这变得有趣，但这对于备受喜爱的厨房电器来说相当困难。我们真正寻找的是一种强制 CD/DVD 刻录机覆盖数据的方法。事实上，通常需要处理的是刻录光盘，而不是工厂压制的版本，这使得这种黑客行为迟早会发生。为什么这不是所有驱动器的标准硬件功能，它可以在软件中实现吗？

总有一些低技术含量的抓拍、抓拍或撕扯方法。我们通常只需刮去光盘顶部的箔片。
