# 电动摩托车摇动一个轮子

> 原文:[https://hack aday . com/2010/08/20/电动-摩托车-岩石-一轮/](https://hackaday.com/2010/08/20/electric-motorcycle-rocks-one-wheel/)

![](../Images/8de590e2af237f17abc33b935a4a75e3.png "electric-unicycle")

我们不得不在标题中称它为电动摩托车，因为电动独轮车听起来很蹩脚。但是休息之后的视频向你展示了这个原型一点也不蹩脚。当你谈论电动交通工具时，它将极简主义发挥到了极致。自动平衡器让人想起了赛格威，但驾驶者骑在一个轮子上，而不是像【沃兹】[喜欢的游戏](http://hackaday.com/2006/04/24/the-woz-playing-segway-polo/)那样站着。看来 [Ryno Motors](http://rynomotors.wordpress.com/) 正试图筹集资金将这些投入生产。我们不会屏住呼吸，直到我们在野外看到它们，但如果它们在不久的将来不会出现在大屏幕上，我们会感到惊讶。
