# Roomba 带簸箕式抓钳

> 原文：<https://hackaday.com/2009/05/09/roomba-with-dustpan-style-grasper/>

[https://www.youtube.com/embed/DCa8NeG3ypc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/DCa8NeG3ypc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

还记得 El-E 吗，这个服务机器人可以取回你用激光发现的东西？El-E 的创造者正在研究制造辅助机器人的其他方法。[他们的最新发明是一个 iRobot Create，基本上是一个 Roomba，有一只定制的抓取手](http://www.hizook.com/blog/2009/05/08/irobot-create-roomba-dustpan-end-effector-grasps-objects-floor)。代替复杂的多自由度组件，他们制造了与簸箕原理相同的东西。它有一个薄楔和一个清扫臂，可以将物品装载到上面。视频中可以看到，还是挺有效果的。
