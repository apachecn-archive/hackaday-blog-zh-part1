# 自行车刹车灯感应到你在减速

> 原文：<https://hackaday.com/2009/11/09/bike-brake-light-senses-you-slowing-down/>

![bike-brake-light](img/9a8f56a5be0e8b4fb15a582596d1a14c.png "bike-brake-light")

如果速度很快，而骑手没有注意到前面的速度变慢，那么集体骑行可能会有点危险。[WyoJustin] [为骑自行车的人设计了一个刹车灯系统](http://www.instructables.com/id/Bar-End-Brake-Light-BEBL/)，试图解决这个问题。led 安装在公路自行车车把的端盖中。当加速计感应到自行车减速时，发光二极管就会亮起，警告你后面的人你正在减速。

该系统是便携式的，因为许多认真的骑手有多辆自行车。为了做到这一点，所有的电子设备都装在车把管中，便于转移。这包括一个内置稳压器的加速度计，一个控制一切的 Arduino 和一个电池。休息之后，看看刹车灯的工作情况。

我们看到的大多数[自行车灯是用于机器的前面](http://hackaday.com/2009/10/22/unreasonably-bright-bike-light-apparently-hunts-deer/)，但这种面向后的包装是一种干净、简单的解决方案，我们可以(安全地)放在后面。
