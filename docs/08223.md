# 机器人在建造漫游四轴飞行器着陆平台时只听领导者的

> 原文:[https://hack aday . com/2011/08/02/robots-listen-only-to-the-leader-when-build-a-roving-quad copter-landing-pad/](https://hackaday.com/2011/08/02/robots-listen-only-to-the-leader-when-building-a-roving-quadcopter-landing-pad/)

![](../Images/e20a52bfe5f7376fc0f7dac1138cf6f6.png "mobile-quadcopter-platform")

群体机器人真的开始产生一些有趣的结果。这张图片来自休息后嵌入的视频，显示了一组[五个机器人为四旋翼直升机](http://www.tedmacdonald.com/projects/multi-robot-formation-control)创建着陆平台。实际上构成平台的四个人并不相互联系，而是听从领导者的命令。我们对直升机瞄准并降落在移动平台上的能力印象深刻。起飞似乎是另一个问题，因为平台机器人停止移动，直到四轴飞行器再次升空。

这些机器人是佐治亚理工学院研究生项目的一部分。[Ted Macdonald]一直在和其他人一起工作，实现一个引导蜂群的组织算法。这种方法要求机器人对群体中所有其他机器人的位置有一个总体的了解。这是用高速摄像机完成的，就像我们在其他机器人控制项目中看到的一样。但这并没有让我们气馁。如果你已经有一个飞行机器人作为蜂群的一部分，你不妨再增加几个作为天空中的眼睛。
