# 让针织机打印像素艺术

> 原文:[https://hack aday . com/2010/11/08/make-a-knitting-machine-print-pixel-art/](https://hackaday.com/2010/11/08/make-a-knitting-machine-print-pixel-art/)

[Becky Stern]展示了如何将一台旧的电子针织机与电脑连接。在看到视频中的兄弟 KH-930E 针织机后，休息后，它看起来像控制功能很像一个数控铣床。图案可以编程并存储在软盘上。既然我们不想再使用它们(除非它们被[当作 SD 卡托架](http://hackaday.com/2010/11/04/mac-se-reborn-as-a-server-and-mac-emulator/)攻击)，很高兴看到这就是机器连接到计算机的方式。使用修改过的 FTDI 电缆和用 Python 编写的软盘驱动器仿真器，可以将空白设计文件保存在针织机上，在计算机中操作以添加您自己的像素艺术，然后加载回机器进行生产。至少，观看针织的发生是很有趣的，但针织服装和极客用品的粉丝们现在肯定垂涎三尺了。

我们从未放弃我们的梦想，从一天做一件东西过渡到一天做一件东西，这只是为这个目标火上浇油。
