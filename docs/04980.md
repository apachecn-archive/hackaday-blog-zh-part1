# 两足动物行走有八个伺服系统

这个小小的两足动物在运动中表现出很好的协调性。正如你所看到的，八个伺服电机负责运动，ATmega8 作为操作的大脑。这个小家伙的[第一代](http://translate.google.com/translate?hl=en&sl=lt&tl=en&u=http%3A%2F%2Frcpilotai.lt%2Findex.php%3Foption%3Dcom_idoblog%26task%3Dviewpost%26id%3D345%26Itemid%3D59%26lang%3Den)和[第二代](http://translate.google.com/translate?hl=en&sl=lt&tl=en&u=http%3A%2F%2Frcpilotai.lt%2Findex.php%3Foption%3Dcom_idoblog%26task%3Dviewpost%26id%3D325%26Itemid%3D59%26lang%3Dlt)的帖子有几个视频。我们推测一个电子表格正被用来调整预编程的动作顺序。试错，人类就是这样学会走路的，对吧？
