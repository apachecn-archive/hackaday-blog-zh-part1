# 面向恩智浦 ARM 处理器的 LUFA 开源 USB 堆栈现已推出

> 原文:[https://hack aday . com/2011/11/11/lufa-open-source-USB-stack-now-for-nxp-arm-processors/](https://hackaday.com/2011/11/11/lufa-open-source-usb-stack-now-for-nxp-arm-processors/)

![](../Images/f9b247b892b8cc4cb5ac23b833a6b558.png "lufa-nxp")

看起来 AVRs [(LUFA)的轻量级 USB 框架刚刚被移植到 ARM 微控制器](http://www.lpcware.com/content/project/nxpusblib)。恩智浦最近发布了 LPC Cortex M3 系列 ARM 控制器的封装。你不会在他们的 nxpUSBlib 描述页面上找到对 LUFA 的引用(我们认为这有点可悲)，但是如果你拿到一份 beta 代码的副本，Version.h 头文件显示它确实是该项目的一个端口。这是由 LUFA 创作者、【迪安摄像机】进一步支持的[，他与恩智浦团队进行了协商。](http://www.avrfreaks.net/index.php?name=PNphpBB2&file=viewtopic&p=889802)

该包提供了一个开源的 USB 堆栈，您可以在您的项目中将其用作 USB 主机或 USB 设备。我们是像这样的开源包的倡导者，因为它使爱好者更容易获得使用工具的帮助，并且它允许社区通过修复错误和添加功能来回馈。

我们突出了几个 LUFA 项目，像[这个键盘重映射器](http://hackaday.com/2011/10/06/hardware-based-keyboard-remapping/)和[这个 AVR 程序员](http://hackaday.com/2011/07/30/avr-programmer-modelled-after-the-mkii-uses-lufa/)。我们期待看到第一批恩智浦 LUFA 项目顺利完成！
