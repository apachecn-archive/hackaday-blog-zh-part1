# 松木赛车已经走过了漫长的道路

> 原文:[https://hack aday . com/2011/04/16/pinewood-derby-cars-have-come-a-long-way/](https://hackaday.com/2011/04/16/pinewood-derby-cars-have-come-a-long-way/)

![](../Images/90965b665c3243b032cde24b7f5a973a.png "Pinewoods01")

拿起你的石墨，骑上一辆车轮，[阿伦·霍克斯特拉]写道，用一些优秀的[松木赛车](http://hypernova.in/component/content/article/2-news/7-flashy-pinewood-derby-cars)让我们彻底难堪。为了追求这种额外的东西，艾伦和他的儿子们商量，他们想出了一些很酷的汽车创意，一个是 Tron 主题的，另一个基本上是带轮子的 Wiimote！官方的松木德比规则对电子设备只字未提，所以只要[没有什么能帮助](http://hackaday.com/2011/03/15/co2-powered-pinewood-derby-car-is-definitely-cheating/)木块在赛道上跑得更快，什么都可以。这意味着你可以自由装载任何你想要的酷灯，但是你必须通过其他方式获得你的[机器人荣誉徽章](http://hackaday.com/2011/04/13/robotics-merit-badge-just-announced/)。

[Aron]通过雕刻汽车的形状开始建造，每辆汽车的底部都有一个中空的空腔，以容纳电池和电子设备。对于 Tron Light Runner 汽车，一个连续的 EL 条在德比汽车的车身上迂回曲折，一个 AAA 电池驱动司机。[Aron]指出，覆盖这辆小车需要大约 5 英尺的电线，这比司机的额定功率多了 2 英尺。幸运的是，额外的一点点额外的电线对它的亮度几乎没有影响。

Wiimote 汽车有详细的 3d 按钮，带线性调节器的试验板，以及驱动蓝色 led 的 PIC 16F628。在大多数情况下，PIC 只是对四个 led 运行一个追踪程序，但[Aron]却不厌其烦地在 Wiimote 的启动序列中编程！

在[Hokestra]的作品上方展示的是我哥哥的松木赛车(左上)和我很多很多年前的……土豆火箭……东西……(右上)。我现在非常后悔没有考虑 led！虽然我认为当时存在的只有红色、绿色和红外。
