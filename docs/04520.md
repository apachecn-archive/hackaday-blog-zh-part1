# 机器人在啤酒桌球上把你的屁股递给你

> 原文：<https://hackaday.com/2009/12/09/robot-hands-you-your-ass-at-beer-pong/>

![](img/0bd20fbc5c1287e2dd5307f5bd56b9bc.png "pong-ball-robot")

猜猜是谁造了这个装置？你们说的对，大学生。但是尽管我们喜欢开玩笑，啤酒乒乓的主题是我们的，不是他们的。该设备使用可沿两个轴定向的气流[来控制和分拣乒乓球](http://spectrum.ieee.org/blog/robotics/robotics-software/automaton/robo-air-blower-makes-ping-pong-balls-defy-gravity)。

与致命的乒乓球发射器不同，这里的目标是优雅地控制球。他们取得了巨大的成功。休息后观看视频，看看球按颜色分类到烧杯中，转移到远处的容器中，并通过高架障碍跑道。为了给我们一点提示，你可以用它来做什么，我们在视频的最后看到机器控制着一个苹果、一个洋葱和一个水瓶。
