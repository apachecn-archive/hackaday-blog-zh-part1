# 太空入侵者按钮

> 原文：<https://hackaday.com/2009/04/14/space-invader-button/>

![spaceinvaderbutton](img/08431596c93b89e04fb8b1f9001e9dc9.png "spaceinvaderbutton")

[Marcus]看到了[Alex]的 64 像素项目,决定可以在更小的空间内实现。上图是他的[太空入侵者按钮](http://interactive-matter.org/2009/04/space-invaders-button/ "Space Invaders Button | Interactive Matter")，带有双色 LED 矩阵。控制板全部是 SMD，并搭载在矩阵上。ATmega164P 通过晶体管驱动 24 个引脚。除了动画，该板还可以做 [LED 感应](http://hackaday.com/2006/02/21/low-cost-sensing-and-communication-with-an-led/ "Low-cost sensing and communication with an LED  - Hack a Day")。这是一个非常聪明的项目，[马库斯]有一些关于使用这种微小组件的笔记。你可以在下面看到它的视频。
