# CES:海尔无线电视

> 原文:[https://hackaday.com/2010/01/09/ces-haier-wireless-tv/](https://hackaday.com/2010/01/09/ces-haier-wireless-tv/)

[https://www.youtube.com/embed/qijuE7r2CBk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/qijuE7r2CBk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们收到了一些通过电子邮件发来的参观海尔无线电视的请求。他们使用无线电源和 WHDI 展示了零线概念。当然，巨大的《2001:太空漫游》背后看起来有点壮观，但对于可以嵌入墙壁的东西，我们印象非常深刻。无线电力在这里是一件大事，我们不能等待黑客们[接管控制权](http://hackaday.com/2009/11/18/look-ma-no-wires/)并将其集成到一些消费产品中。

[Nick]-这里有几家公司正在推广无线电源产品，但这绝对是我们见过的最酷的实现之一。
