# Wii 平衡板

> 原文：<https://hackaday.com/2009/06/28/wii-balance-board/>

[https://www.youtube.com/embed/t4UciZLl2Hk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/t4UciZLl2Hk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

谷歌反垃圾邮件团队的首席开发人员 Matt Cutts 描述了如何将 Wii 平衡板连接到 linux 电脑上。他甚至展示了如何制作 GUI 来显示输入。整个项目用大约 200 行 python 代码完成。这个过程假设你已经可以和 WiiMote 建立蓝牙连接，但是如果你不能，[他也有相关的说明](http://www.mattcutts.com/blog/linux-wiimote-via-bluetooth/)。
