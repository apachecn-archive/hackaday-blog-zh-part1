# 大理石机器综述

> 原文：<https://hackaday.com/2011/05/03/marble-machines-roundup/>

[登哈]多年来一直在建造大理石机器，[决定回顾一些他最喜欢的基于大理石的建筑](http://denhaku.blogspot.com/2011/04/denha.html) ( [翻译](http://translate.google.com/translate?hl=en&sl=ja&tl=en&u=http%3A%2F%2Fdenhaku.blogspot.com%2F2011%2F04%2Fdenha.html))。有很多这样的例子，也有一些关于每一个的想法。我们最喜欢的部分是项目的数字模拟。例如，上面的图像显示了一个在物理模拟器中建造的翻转弹球机。这使得物理构建的计划变得容易得多，因为它会在你切割第一块材料之前告诉你确切的尺寸。这两张图片都是从视频中截取的，广告之后可以看到。但这并不是构建前规划的最核心部分。 [SolidWorks](http://www.solidworks.com/) ，一个最常用于为精密加工设计 3D 模型的 CAD 套件，也被用来为更复杂的机器建模。
