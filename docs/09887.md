# 登山扣帮助你磨练你的铣削技能

> 原文:[https://hack aday . com/2012/03/13/carabiner-helps-you-hone-you-milling-skills/](https://hackaday.com/2012/03/13/carabiner-helps-you-hone-your-milling-skills/)

![](../Images/27babe6671ec47d18b2cad272f35d160.png "Magnabiner-Build-your-own-Carabiner")

[Christian]正在我们认为是他当地黑客空间的地方学习使用金属铣削工具。我们喜欢公共空间，它们提供了许多探索新领域的机会。他开始了一次航行，包括参观商店里的大部分机器，因为他自己制作了一个带磁性门的登山扣。他不会被吊在山坡上。但是多亏了这个装置，他的钥匙或水瓶将会找到一个快乐的家。

这一切都始于一些草图，以建立整体设计的形状。在那里，他花了一些时间用 CAD 制作了这个铁锁的框架。他足够幸运，可以使用一个喷水器，这个喷水器拿走了 SolidWorks 文件，并为他切割出了铝框。这留下了非常锋利的边缘，所以他用带硬质合金钻头的木刳刨机将它们磨圆。

下一部分是添加门。他用端铣刀在框架上增加了一个安装区域。使用滚花工具对闸门的锁环进行纹理处理，其余部分使用简单的切割工具进行铣削。这种闸门使用磁铁使其自身居中，滚花环作为唯一的机械闭锁机构。[Christian]在休息后的视频中很好地展示了完整的登山扣。
