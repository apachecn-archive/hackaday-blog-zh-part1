# -硬播放器- 8 位音乐硬件播放器

> 原文：<https://hackaday.com/2008/10/22/hard-player-8-bit-tunes-hardware-player/>

[https://www.youtube.com/embed/zgb_NFdTHZc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zgb_NFdTHZc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

对于错过雅达利音乐黄金时代的人来说，你可以用 [-hard plAYer-](http://tolaemon.com/hplayer/) 重现 8 bit 音乐的魔力。[Tolaemon]在这个硬件播放器上花了很多心思和工作。从 AY-38912 可编程声音发生器——过去用于像 [Atari](http://www.mahalo.com/Atari) 或 [Colour Genie](http://www.classic-computers.org.nz/system-80/hardware_eaca.htm) 这样的旧电脑中——到使用 [YM 文件格式](http://leonard.oxg.free.fr/ymformat.html),每一件作品都被极其小心地组装在一起。[Tolaemon]还提供了他的设计，[固件](http://tolaemon.com/hplayer/hplayer-firmware.zip) (ZIP)，以及[零件](http://tolaemon.com/hplayer/partlist.txt)清单，这样你就可以制作自己的硬件播放器了。
