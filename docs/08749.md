# Zigroller 平衡机器人

> 原文:[https://hackaday.com/2011/10/08/the-zigroller-balance-bot/](https://hackaday.com/2011/10/08/the-zigroller-balance-bot/)

[![](../Images/b66844edff11e2ea50893f1e5fddfba4.png "zigroller")T2】](http://hackaday.com/2011/10/08/the-zigroller-balance-bot/zigroller/)

[Zigroller](http://www.logos-electro.com/blog/2011/9/25/zigroller-project-hardware.html) 是一个 Zigbee 遥控平衡机器人。虽然平衡机器人以前在[HAD]上出现过，但 Zigroller 似乎是精心打造的，该项目也有据可查。除了硬件部分，这个版本的软件在这里有[的记录。如果你想了解更多背景知识，这篇](http://www.logos-electro.com/blog/2011/10/4/zigroller-project-software.html)[【麻省理工】的论文](http://web.mit.edu/scolton/www/filter.pdf)给出了这种平衡机器人背后的理论。

这个滚轴本身被认为是今年夏天 UW 班级的一个项目。这门课是关于控制理论的，所以电子和机械设置是从[【ArduRoller】](https://github.com/fasaxc/ArduRoller)项目中复制的。为了防止这个项目影响这个“机器人”的控制代码，据称在编写新软件时没有查看代码。构建远程控制组件的过程被很好地记录下来，并且是这个项目的原创。休息之后，请观看它的视频！
