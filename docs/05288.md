# 肌电图控制假肢

> 原文：<https://hackaday.com/2010/05/04/emg-controlled-prosthesis/>

这个[假肢](http://www.youtube.com/watch?v=3aOCFlLbcnY)是一个学生项目的成果。[Amnon Demri]和他的同学们考虑到肘部以下截肢者的需要而建造了它。它使用[肌电图](http://en.wikipedia.org/wiki/Electromyography)来驱动手指和手腕。四个粘贴式传感器放置在肘部周围，以感测那里的电活动。这些信号由 PIC 16f877a 微控制器解释，然后控制伺服电机来操作假肢。这听起来像是一个非常经济的解决方案，正如你在休息后的视频中看到的，它工作得相当好。
