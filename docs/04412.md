# 乐器作为游戏——游戏作为乐器

> 原文：<https://hackaday.com/2009/11/13/instruments-as-games-games-as-instruments/>

![instruments-as-games](img/0a646a4f5b42d9c654ff4767e2ec137e.png "instruments-as-games")

这是一对截然相反的黑客。一种利用真实的乐器演奏摇滚乐队，另一种利用游戏控制器演奏真实的音乐。

[Tim]让我们知道他的朋友想出了如何通过吹笛子而不是唱歌来[演奏摇滚乐队 2 的专家水平。当然这是可行的，因为游戏只是在寻找正确的得分频率。声线可以偏移一个八度音程，但仍能正确记录，这是有道理的。我们自己不会想到这一点，但现在我们看到了她的成功，我们会尝试一下(我们的乐器技巧远远超过我们的歌唱天赋)。](http://www.youtube.com/watch?v=_oFP_TH8jYY)

看到这一点引发了与电话交谈的关联，电话交谈始于一个失明的孩子对着听筒唱一个音调，使他剩余的长途电话免费。接下来是[蓝盒子](http://en.wikipedia.org/wiki/Blue_box)，允许没有完美音高的人电子演奏音调。看看一个有天赋的长笛手(像[打拳长笛手](http://www.youtube.com/watch?v=59ZX5qdIEB0))和一个旧的电话网络能做些什么将会很有趣。

在硬币的另一面，我们有[乔丹的]项目，其中[她用《吉他英雄世界巡演》和《摇滚乐队》的 Wii 鼓](http://jordanbalagot.com/blog/2009/11/06/create-a-full-midi-drumset-with-guitar-hero-and-rock-band-drum/)创造了 midi 控制器。吉他英雄鼓是速度敏感的，这是她在设置中使用的一个功能。MIDI 数据考虑了鼓被击打的力度，产生的声音反映了这一点。这篇特别的文章概述了她在速度敏感系统中使用的接触仪，但你也可以查看她写的教程，该教程介绍了如何使用 JunXion 和摇滚乐队控制器，我们在过去的[中已经介绍过](http://hackaday.com/2008/07/02/easiest-rock-band-to-midi-setup-yet/)。

这两种控制方案的视频包含在分页符之后。我们喜欢看到人们打破“我正在通过玩电子游戏来创作音乐”的伪装，实际上以一种新的有趣的方式来使用他们的音乐天赋。
