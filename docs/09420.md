# Arduino 定日镜计算太阳的位置

> 原文：<https://hackaday.com/2012/01/03/arduino-heliostat-calculates-the-position-of-the-sun/>

[![](img/41ec12ae52c6fd8a7c9d9614a6a97128.png "heliostat")](http://hackaday.com/wp-content/uploads/2012/01/heliostat.jpg)

我们已经过了冬至，白天越来越长，但这并不意味着我们不讨厌下午 5 点左右的日落。有一种方法可以让更多的阳光透过我们的窗户——定日镜。幸运的是，[加布里埃尔]提交了他的开源太阳跟踪/定日镜项目，该项目可以在整个冬天通过我们的窗户反射阳光。

用镜子照亮房间是一种古老的做法；几千年前，定日镜被称为古语“奴隶”幸运的是，如今有一种更优雅的做事方式 Arduino。[Gabriel]想出了一个 Arduino 草图，它只使用纬度、经度和时区来计算太阳的高度和方位角。[加布里埃尔]用这个草图来驱动一对步进电机，并通过他的窗户反射阳光。

休息之后你可以看看[加布里埃尔]的定日镜演示。
