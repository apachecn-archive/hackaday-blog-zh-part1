# 万圣节黑客:比招待还有趣的放射性展示

[肖恩]正在寻找一种方式来改变他万圣节的装饰，并决定为他的前院做一个新的道具。他的车库里有一对旧油桶，他认为稍加修整后会很好看。

他把两个桶栓在一起，然后在上面涂上一层新的黑漆，再在每个桶上加一个亮黄色的辐射三叶草。一个桶包含一个打开盖子的气动活塞，露出一个点亮的南瓜，一个雾机和一组喷水嘴。第二个鼓装有空气压缩机、无线路由器和用于记录动作的 IP 摄像机。一个联网的 Arduino 触发盖子和喷雾器，这两者都可以在他舒适的家里控制。

我们认为这是对一些旧废桶的创造性利用，但我们希望他不要计划弄湿经过的“不给糖就捣蛋”的人，因为这似乎是一个让你的房子在万圣节遭到鸡蛋或其他破坏的好方法。如果喷水器可能换成一些其他酷的效果，[Shawn]就不太可能成为一些愤怒、潮湿的青少年的目标。
