# 简易自动 LED 灯

> 原文:[https://hackaday.com/2009/09/02/simple-automatic-led-lamps/](https://hackaday.com/2009/09/02/simple-automatic-led-lamps/)

![simple_led_lamps](../Images/64e6dcdaf871c7f2a0d6332c52e1540d.png "simple_led_lamps")

[woody1189]为他的衣柜组装了一些自动照明设备。九个发光二极管被分成三个灯，由一个[霍尔效应传感器](http://en.wikipedia.org/wiki/Hall_effect_sensor)控制。他在 Arduino 上做了原型，然后移植到 ATtiny85 上。虽然当前的实现可以在没有微控制器的情况下完成，但我们希望看到一些固件的改进，例如当你忘记关壁橱门时自动关闭。霍尔效应传感器似乎[出现在许多项目](http://hackaday.com/2009/06/15/rgbike-pov-2/)中，所以确保你在下一个零件订单中得到一些。休息后的视频。
