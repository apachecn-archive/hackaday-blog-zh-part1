# 廉价的机器人手

> 原文:[https://hackaday.com/2010/10/07/cheap-robotic-hand/](https://hackaday.com/2010/10/07/cheap-robotic-hand/)

![](../Images/c0407837a7daaa5d044a30da45c79e7e.png "ffcy7q")

【马自达】和我们分享，他的[廉价机器人手](http://letsmakerobots.com/node/22942)。他受到了[这个项目](http://letsmakerobots.com/node/22323)的启发，在这个项目中，有人使用一个 Arduino 和一个带有一些 flex 传感器的手套来控制一只预制的手。不过，他想自己动手做一点。他选择了 picaxe 微控制器，用细绳、一些有机玻璃和一些橡胶软管自己制作了这只手。他最终计划增加手套控制。

不幸的是，没有原理图或源代码。也许如果我们问得很好，他会分享。
