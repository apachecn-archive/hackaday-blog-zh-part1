# 谢妮等离子球

> 原文:[https://hackaday.com/2009/09/15/nixie-plasma-ball/](https://hackaday.com/2009/09/15/nixie-plasma-ball/)

![plasma_ball_pic5](../Images/a90cce80bdd9a147bc541f67cef10837.png "plasma_ball_pic5")

这本书一年前出版了，我们错过了。【Daqq】已经[用数码管](http://www.daqq.eu/index.php?show=prj_plasma_nixie)做出了等离子球。所需要的只是一个用于计算机的冷阴极变压器和一个数码管。他确实对电源做了一点小小的改动，你可以在项目页面上看到。这不是最有用的项目，因为你真的看不到太多的等离子体，但它确实很酷。我们认为这将是一个非常整洁的按钮。休息之后你可以看到一段视频。
