# 乐高主机为您所有的原型项目

> 原文:[https://hack aday . com/2010/09/13/Lego-host-for-all-of-your-prototyping-projects/](https://hackaday.com/2010/09/13/lego-host-for-all-of-your-prototyping-projects/)

![](../Images/0cfde10bdc768ec7db846777a3f0004e.png "lego-vehicle-project")

[死鸟]决定用一辆乐高 8880 超级汽车作为他所有电子修补的主机。[在他的博客](http://hexecute.blogspot.com/) ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://hexecute.blogspot.com/))中，你会发现这辆车配有一个 Arduino MEGA 接口各种原型位。它从移动的马达开始，紧接着是转向的伺服系统。从那里我们看到了一个试验板和图形液晶显示屏的增加。到目前为止，他已经学会使用 PS2 键盘作为控制器，最近他[与 Wii 双截棍](http://hexecute.blogspot.com/2010/09/nunchuk.html)接口。

我们更习惯于看到适应更广泛用途的 [NXT 套件](http://hackaday.com/2010/05/31/lego-pick-and-place/)，但如果你有一个像这样的好套件，它会成为一个很好的基础，你可以在其上添加自己的机器人元素。
