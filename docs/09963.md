# 使用 OpenCV 和 Arduino 跟踪人脸

> 原文:[https://hack aday . com/2012/03/24/following-faces-with-opencv-and-arduino/](https://hackaday.com/2012/03/24/following-faces-with-opencv-and-arduino/)

[https://www.youtube.com/embed/lD4uFD7j0AU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/lD4uFD7j0AU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Marco]在面部跟踪领域与[OpenCV 玩得很开心](http://marco.guardigli.it/2010/01/arduinoprocessing-face-follower.html)。利用一个旧的激光项目，他拼凑了一个系统，该系统将跟踪一张脸，并将激光放在上面。虽然他只是用这个作为概念验证，但不言而喻，你可能不应该在面部追踪器上安装激光器。然而，把它塞进我的 Keepon 并不是一件可怕的事情。

[Marco]在这篇文章中分享了让 OpenCV 位工作的过程，你将不得不参考他的[激光枪](http://marco.guardigli.it/2010/01/arduinopc-two-axis-controlled-laser-gun.html)项目以获得物理构建。
