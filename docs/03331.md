# 镍自动的

> 原文：<https://hackaday.com/2009/03/24/nickel-o-matic/>

![coins](img/8526068df55b490359f3460616f933a0.png "coins")

[迈克]送来这个项目。这是一个机器人，设计用来在人们观看时在木制硬币上印刷。它是在芝加哥的 iHobby 博览会上建造的。主运动由基本的 Stamp2 控制，而喷墨系统由螺旋桨驱动。整个系统有 4 个伺服系统，3 个步进电机，一个 DC 电机，一个黑客吸奶器，一个喷墨头和 5 个红外传感器。如果你没注意到，它有吸奶器。我们假设那是用吸力捡起木制镍币的部分。他表示，该项目旨在娱乐，因此有许多多余和低效的行为，正如你在休息后的视频中所看到的那样。干得好[迈克]。
