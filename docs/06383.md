# 开发电子动画平台

> 原文:[https://hack aday . com/2010/12/01/developing-an-animatronics-platform/](https://hackaday.com/2010/12/01/developing-an-animatronics-platform/)

![](../Images/949a58504e8b3e36b0f4fe69eeab4ac6.png "predator-animatronics-via-arduino")

这套掠夺者服装在肩炮中有一个电子动画元素。它跟踪面具的移动，瞄准捕食者注视的任何地方。[Jerome Kelty]被一个朋友要求帮助开发服装，他最终得到了一个可以适应许多不同用途的动画电子平台。

从 Arduino Pro Mini 开始[Jerome]设计了一个主板，它可以突破 Arduino 的引脚，使连接和驱动多个伺服电机变得容易。该板由 3.3v 锂聚合物电池供电，充电由设计中包含的 MAX1555 处理。休息之后看看展示掠夺者套装的视频。看起来[杰罗姆]是对的，他也把这个平台用在了一套带有手臂安装导弹功能的铁人三项服上。

你的下一个服装造型需要一些灵感吗？看看这个[电子动画系列](http://hackaday.com/2010/01/15/animatronics-reference/)让你开始吧。
