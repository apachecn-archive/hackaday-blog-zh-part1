# 更多手机控制的门锁

> 原文:[https://hack aday . com/2010/02/23/more-手机控制门锁/](https://hackaday.com/2010/02/23/more-cellphone-controlled-door-locks/)

![](../Images/b037e35b61719ad904f5c48ef86100eb.png "easy-door-unlocking")

[汤姆·李]和他的同事刚刚搬到一个新办公室。门的设置就像一个安全检查站，里面有电子锁扣和按钮，允许进入。这个按钮只是完成了一个低电压电路，激活了罢工，这使得它很容易接入。他们用一个小继电器搭建了一个接口板来完成电路。正如我们之前看到的， [Linksys 路由器在机箱](http://hackaday.com/2009/09/29/add-ir-control-to-your-wifi-router/)中有足够的额外空间，所以在这个小小的网络设备中容纳新电路没有问题。现在[Nicko]和他的朋友们可以使用[定制的应用程序](http://sunlightlabs.com/blog/2010/more-about-door/)来输入访问代码或从手机验证设备 ID 并获得进入许可。门仍然有钥匙锁以防停电。事实上，对系统所做的唯一改变就是在“门释放”按钮上增加了两根电线，如上图所示。休息后见视频中的一键设备 ID 认证。

这种攻击与去年的 [GSM 入口](http://hackaday.com/2009/11/28/gsm-enabled-security-door/)类似。在这种情况下，电话通过网络接口而不是 GSM 网络与门通信。
