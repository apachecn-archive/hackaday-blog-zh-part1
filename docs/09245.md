# 铝弯曲教程和一个 DIY 刹车

> 原文：<https://hackaday.com/2011/12/07/aluminum-bending-tutorial-and-a-diy-brake/>

是什么让一个项目真正与众不同？一部分是一个“哎呀，这很聪明”的角度，还有一点“这实际上是可能的。”《黑客一日》剧组真正欣赏的一件事是令人敬畏的外壳。铝锡会让你走得很远，但要想进入大联盟，你必须弯曲一些铝。幸运的是，[Rupert]发来了一篇关于弯曲外壳铝板的精彩教程。

为了制作折弯机，[Rupert]收集了一些 38 毫米的竹制工作台面废料。在用一些铰链组装了这些部件之后，他准备弯曲一些铝。

(鲁珀特)学会的一个技巧是在未来弯曲处的金属板上划线。在[Rupert]的项目中，他将 3 毫米厚的铝板放入台锯中切割 1 毫米深。当然，这只能用为有色金属设计的带有尽可能多的硬质合金齿的刀片来完成。从[鲁伯特]使用这种建造技术的自制高保真音响来看，结果是惊人的。
