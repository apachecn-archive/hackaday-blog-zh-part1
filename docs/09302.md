# 发光亚克力圣诞饰品

> 原文:[https://hack aday . com/2011/12/14/lighted-acrylic-Christmas-paints/](https://hackaday.com/2011/12/14/lighted-acrylic-christmas-ornaments/)

![](../Images/7d227a7923f22c48c429fc23cd0fb03b.png "lighted-acrylic-ornaments")

如果你有建造自己的数控铣床的麻烦，我们知道你总是在寻找用它的东西。[Boris Landoni]写来了节日的必需品；[一套带灯的亚克力圣诞饰品](http://www.open-electronics.org/diy-led-plexiglass-christmas-decorations-pic-version/)。

丙烯酸的一个有趣的特性是它在边缘发光时的反应。材质引导光线，直到它被表面的干扰反射。第一步是设计装饰品的轮廓，因为所有切割边缘都会发光。接下来，[Boris]使用 [artCAM](http://www.artcam.com/) 设计要切割的内部零件。这个应用程序翻译的救济削减，必须真正使你的设计发光(对不起，我们无法抗拒)。最好的例子就是上面看到的天使和蜡烛。

每一个压克力部件底部都有一个插槽，用来容纳一个 LED。[Boris]使用小型投影盒，带有用于二极管的 PCB，以及用于供电的纽扣电池。
