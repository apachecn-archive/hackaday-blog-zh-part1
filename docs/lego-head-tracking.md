# 乐高头部追踪

> 原文：<https://hackaday.com/2009/04/03/lego-head-tracking/>

![lego_tracking](img/e8deb0c28a098c122581d95beeede4ad.png "lego_tracking")

在这个使用乐高积木的[简单头部追踪项目中发送的【Epoch】。他做了一个定制的支架，在棒球帽上安装了 3 个乐高光传感器。然后，在用一些软盘碎片修改了他的红外摄像头后，他加载了](http://www.epokh.org/blog/?p=176)[自由轨道](http://www.free-track.net/english/)软件，并可以控制他的游戏。为了方便起见，他对乐高 Nxt 进行了编程，只在他拿着接触传感器时打开灯。休息之后你可以看到它的实际效果。这似乎与[【李炯楷】的头部跟踪](http://hackaday.com/2007/12/21/wiimote-head-tracking-desktop-vr-display/)非常相似。从视频来看，这并不顺利。
