# 抽头控制的节拍器

> 原文：<https://hackaday.com/2009/09/04/tap-controlled-metronome/>

![step13_06](img/2eaa28248fc052ca6527ebee2dc37f7f.png "step13_06")

[Adam]和他的伙伴[Matthew]送来了他们的轻触控制节拍器，或者按照他们的喜好，“有态度的节拍器”使用压电扬声器，你可以将模式和节奏输入到记忆中，它会循环地重复给你。这两个按钮允许您加快或减慢由 led 阵列指示的节拍。根据他们的要求，我们提到它完全是在 PIC 16F 上，而不是在 Arduino 上。也许最有趣的部分，我们发现绝对值得检查是他们的[令人惊讶的详细建设过程](http://wayneandlayne.com/metronome/build.php)。休息之后，看看节拍器的快速视频。
