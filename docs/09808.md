# 用 IP 摄像头远程监控你的车库门

> 原文:[https://hack aday . com/2012/03/01/用 ip 摄像头远程监控你的车库门/](https://hackaday.com/2012/03/01/monitoring-and-controlling-your-garage-door-from-afar-with-an-ip-camera/)

![remote-control-garage-door](../Images/bef4154cf784f447c0069e0a1b2c3e73.png "remote-control-garage-door")

去年，[马克·西蒙内利]的妻子问他是否能设计一些东西让她远程检查他们的车库门是否开着。[Mark]抓住机会摆弄电子设备，并围绕一个旧的 TrendNet IP 摄像头设计了一个系统。当使用 IP Cam Viewer Pro 为他的 Android 手机远程连接到摄像机时，[Mark]可以观看视频流，还可以通过他构建的一个小继电器电路触发车库开门器。

他的遥控开启器工作良好，但不幸的是，他的相机缺乏任何红外视觉/弱光功能。由于他的相机在黑暗中用处不大，他决定在远程监控他的车库时，他需要[增加一些方法来触发灯](http://www.ctsimonelli.com/2012/03/power-strip-hack/)。他认为最好的方法是用一种类似于他自己设计的电路来控制一个与电源板相连的灯，从而打开车库门。

他在五金店停下来，买了一个便宜的电源板，一到家就把它拆了，把电源开关也拆了。他给它安装了一个小的 5v 继电器，连接到相机的端子板上。虽然他承认这可能不是绝对安全的解决方案，但他可以通过简单地滑动手机屏幕来轻松控制灯和车库门。

继续阅读，看看他的遥控电源板在行动中，一定要摆动他的网站，看看更多关于他的相机控制的车库开门器的细节。
