# 令人沮丧的磁悬浮乐趣

> 原文：<https://hackaday.com/2011/07/23/frustrating-fun-with-magnetic-levitation/>

安德烈·米哈尔丘克[建造了他自己的磁悬浮装置](http://rtfms.com/rtfms-episode-8-how-to-build-a-magnetic-levitating-top.htm)你也可以…如果你有耐心的话。他没有使用电磁体，像 [Arduino 悬浮车](http://hackaday.com/2011/02/21/arduino-levitation/)或[浮动地球仪](http://hackaday.com/2010/08/31/floating-globe-hacked-to-rotate/)。相反，一对陶瓷环形磁铁和几个小时就够了。

他的设备的基础是一对非常大的环形磁铁，最常用于扬声器。在上图中很难看到它们，因为有一个倒置的塑料容器遮住了它们。选择第二个(或第三个，取决于你如何计算)环形磁铁是因为它比磁性底座中的圆形空隙小。简单地在空中平衡磁铁是不可能的，但是旋转它就是另一回事了。通过创造一个完美平衡的磁性陀螺，然后在底座的磁场中旋转，你可以让它漂浮在半空中。

休息之后请看视频。这是一个整洁的效果，但你真的必须有一个完美的设置才能工作。[Andrey]提到微调需要几个小时。如果环境条件稍有变化，整个事情就会被打乱。
