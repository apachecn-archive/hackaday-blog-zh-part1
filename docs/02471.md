# RGB 桌面时钟

> 原文:[https://hackaday.com/2008/09/01/rgb-desktop-clock/](https://hackaday.com/2008/09/01/rgb-desktop-clock/)

 <https://www.youtube.com/embed/-hxZRCponks?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span>【马克罗伊】设计了这个精致的<a href="http://www.stellarcore.com/projects/led-desktop-clock/" target="_blank"> RGB LED 台式时钟</a>。它使用 PIC16F877A 微控制器和 DS1307N 实时时钟。周边有一圈 12 个 RGB 发光二极管。小时用红色表示，分钟用绿色表示，秒钟用蓝色表示。如果双手重叠，颜色就混合了。三个按钮用于设置时间，它可以在低至 0.7 伏的电压下运行。你可以在 Stellar Core 上找到更多关于该项目的信息。他有计划在未来将其装备起来。</p> <ul><li class="readlink"><a href="http://www.stellarcore.com/projects/led-desktop-clock/">永久链接</a></li></ul> </body> </html>
