# 想学人工智能？很好。

> 原文:[https://hack aday . com/2011/09/05/want-to-learn-artificial-intelligence-good/](https://hackaday.com/2011/09/05/want-to-learn-artificial-intelligence-good/)

![](../Images/0b0375ff8cabb0eb0796b4dcf1c3cbb4.png "hal")

再过一个多月，世界各地成千上万的人将参加斯坦福大学的人工智能课程。这两个课程的注册仍然开放。“基础”课程只是观看讲座和回答问题，或者是稍微高级一点的[麻省理工学院开放式课程](http://ocw.mit.edu/index.htm)或[可汗学院](http://www.khanacademy.org/)。“高级”轨道是完整的课程，需要家庭作业和考试，并渴望斯坦福难度。

随着成千上万的人上这门课，网上肯定会出现一些学习小组。我们见过的最大的是 Reddit 上的[/r/ai class](http://www.reddit.com/r/aiclass/)和栈溢出风格 [aiqus](http://www.aiqus.com/) 。最常见的回答是“我应该从这门课中学什么语言？”是 Python，尽管有一个[在线代码仓库](http://aima.cs.berkeley.edu/code.html)，里面有 Lisp、Java、C++和 C#的文本工作代码。

如果人工智能不能让你的船漂浮起来，那么今年秋天斯坦福大学还会教授两门课:[机器学习](http://www.ml-class.com/)和[数据库介绍](http://db-class.com/)。不管你怎么看，你都可以免费上这个领域*最杰出的导师之一的课。帮自己一个忙[报名](http://www.ai-class.com/registration)。*
