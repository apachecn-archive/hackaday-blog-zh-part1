# 特罗森机器人公司举办另一场比赛

> 原文：<https://hackaday.com/2010/09/02/trossen-robotics-holding-another-contest/>

![3rd place doesn't really shoot lasers that make stuff blow up.](img/84b22a34a1d50218fcb61a26b8ef78d1.png "trcont")
