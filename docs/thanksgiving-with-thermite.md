# 铝热剂感恩节

> 原文：<https://hackaday.com/2010/11/25/thanksgiving-with-thermite/>

对于那些在美国的人，感恩节快乐。不管你是否庆祝这个传统节日，你可能还是想花点时间想想你感激什么。我们感谢我们的读者，他们驱使我们不断发布项目，并挑战我们提高技能。黑客们给我们提供了一些简单和复杂的项目。我们感谢我们的老板，他们雇佣我们做这些可怕的事情，只是偶尔打我们一下。我们感谢[铝热剂](http://hackaday.com/2008/09/16/how-to-thermite-based-hard-drive-anti-forensic-destruction/)，它燃烧起来如此明亮，看起来如此美丽。
