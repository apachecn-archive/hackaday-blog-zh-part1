# 空气清新剂零件遥控机器人玩具

> 原文：<https://hackaday.com/2011/03/07/remote-controlled-robot-toy-from-air-freshener-parts/>

[jcopro]非常喜欢 Glade 自动空气清新剂。利用其中的一对，他制作了一个简单的遥控玩具，并在周末与我们分享。你可能记得他用这些空气清新剂为他的相机建造了一个远程快门释放系统，这是我们几周前特别介绍的[。](http://hackaday.com/2011/02/04/remote-camera-trigger-built-from-air-freshener-parts/)

他没有扔掉空气清新剂的外壳，而是决定将多余的塑料用作机器人底盘。他用一对铅笔做腿，建造了他的机器人“Limpy”他从一个旧玩具上取下一对马达和控制板，用几条绝缘胶带把它安装到自己的作品上。

他承认，他甚至不愿意称这个玩具为机器人，但他很喜欢制作它，并表示这将是一个很好的初学者项目。我们同意——这对孩子们来说是一个很好的项目，尤其是如果你想重复使用他们不再玩的旧遥控玩具。
