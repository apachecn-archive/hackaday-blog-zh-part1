# 软驱乐队

> 原文:[https://hackaday.com/2011/12/30/a-floppy-drive-orchestra/](https://hackaday.com/2011/12/30/a-floppy-drive-orchestra/)

[![](../Images/fc8a3f3004215babc1306c0cd631d6d2.png "floppy_orchestra")T2】](http://hackaday.com/2011/12/30/a-floppy-drive-orchestra/floppy_orchestra/)

8 个驱动器协同工作，这个[“软驱阵列乐队”](http://silent.org.pl/home/2011/12/27/merry-xmas/)将驱动器音乐带到了一个全新的水平！似乎这还不够，[沉默]决定使用 16x64 的 LED 阵列，他一直在努力与驱动器相配合。

对于那些记得的人，我们实际上已经展示过[SileNT]，也就是[Pawel]以前的作品，我们印象深刻的是他能够在 2 张软盘上播放《星球大战》中的《帝国进行曲》。他计划在未来提供更多关于他的软驱音乐制作的信息，甚至可能是关于建立你自己的个人软驱乐队的指导。显然[SileNT]有充足的 3 1/2 英寸硬盘供应，所以也许这个壮举也能被超越…

请务必查看这 8 个驱动器运行中断后的视频。在视频中，更多的曲调被承诺“很快”，所以一定要回来查看或在 Youtube 上订阅他更多甜美的旋律。
