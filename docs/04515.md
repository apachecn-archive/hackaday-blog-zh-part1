# LEDboy 超级像素兄弟

> 原文：<https://hackaday.com/2009/12/08/ledboy-super-pixel-brothers/>

[https://www.youtube.com/embed/cB6vSHDDdhI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/cB6vSHDDdhI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们真的很喜欢这个版本的超级马里奥兄弟，这个版本是由布拉德制作的。它在 8×8 RGB LED 屏幕上播放，由 pic 微处理器供电。目前还没有很多关于构造或代码的细节，但我们预计他会很快发布。我们猜测它与他的[其他 8×8 游戏系统](http://www.bradsprojects.com/index.php?option=com_content&task=category&sectionid=4&id=13&Itemid=29)非常相似。如果你真的想快速入门，他已经出版了一些关于 pic 微处理器的[优秀教程](http://www.bradsprojects.com/index.php?option=com_content&task=category&sectionid=4&id=14&Itemid=28)。
