</span> <p>进化控制委员会多年来一直在做现场混搭表演，最近升级了他们的硬件。受[李炯楷]的<a href="http://hackaday.com/2007/12/09/wiimote-projector-whiteboard/" title="Wiimote projector whiteboard - Hack a Day">Wiimote 白板</a>的启发，他们建造了一个可以在表演中使用的背投显示器。它在<a href="http://www.ableton.com/" target="_blank"> Ableton Live </a>展示了密集的样本集合。表演者的每只手上都有一个安装在顶针上的红外发光二极管。用拇指和食指接触，LED 就会亮起。两台 Wiimotes 观察这些红外闪光来触发鼠标点击。[TradeMark G]发现 Ableton 显示屏过于复杂，无法用鼠标快速准确地导航；这种新的展示使事情变得更加容易和愉快。</p> <p>【via <a href="http://laughingsquid.com/the-evolution-control-committe-video-mashup-screen/" title="The Evolution Control Committe Video Mashup Screen |&lt;br &gt;&lt;/a&gt; Laughing Squid" target="_blank">笑鱿鱼</a></p> </body> </html>
