# Wiimote 手指跟踪音乐控制器

> 原文:[https://hackaday . com/2008/12/02/wiimote-finger-tracking-music-controller/](https://hackaday.com/2008/12/02/wiimote-finger-tracking-music-controller/)

[https://www.youtube.com/embed/Y7uLuYG62sY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&fmt=18&autohide=2&wmode=transparent](https://www.youtube.com/embed/Y7uLuYG62sY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&fmt=18&autohide=2&wmode=transparent)

进化控制委员会多年来一直在做现场混搭表演，最近升级了他们的硬件。受[李炯楷]的[Wiimote 白板](http://hackaday.com/2007/12/09/wiimote-projector-whiteboard/ "Wiimote projector whiteboard  - Hack a Day")的启发，他们建造了一个可以在表演中使用的背投显示器。它在 [Ableton Live](http://www.ableton.com/) 展示了密集的样本集合。表演者的每只手上都有一个安装在顶针上的红外发光二极管。用拇指和食指接触，LED 就会亮起。两台 Wiimotes 观察这些红外闪光来触发鼠标点击。[TradeMark G]发现 Ableton 显示屏过于复杂，无法用鼠标快速准确地导航；这种新的展示使事情变得更加容易和愉快。
