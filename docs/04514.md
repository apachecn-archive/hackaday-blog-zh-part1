# 离子飞船又名升降机

> 原文：<https://hackaday.com/2009/12/08/ionocraft-aka-lifters/>

很难相信在过去的五年里我们没有报道过举重运动员。当[泰勒]向我们透露了一个在堪萨斯城[黑客空间展示的](http://blog.cowtowncomputercongress.org/)[升降机项目](http://blog.cowtowncomputercongress.org/ionic-flyer/)时，这种意识被激发了出来。

升降机，离子飞船的临时名称，在没有燃烧或移动部件的情况下飞行。我们不打算解决是什么使飞行成为可能的细节，但是高电压是提供升力的现象所必需的。当我们问[如何处理旧 CRT 显示器](http://hackaday.com/2008/02/25/hackit-new-uses-for-old-crt-monitors/)时，第一个评论就是把它们用作升降机。如果你可以在不伤害自己的情况下接入反激式变压器，它会输出大量的电压(不，说真的，这是个问题)。

这是 CCCKC 的人使用的方法。休息后看一下视频。如果你渴望体验更多举重运动的乐趣，请到[举重项目](http://jnaudin.free.fr/lifters/story.htm)参观。
