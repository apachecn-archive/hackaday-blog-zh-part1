# Moolodeon 电动手风琴

> 原文：<https://hackaday.com/2009/12/28/moolodeon-electric-accordion/>

[https://www.youtube.com/embed/Dc1a5ig_M94?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Dc1a5ig_M94?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[李]想要一个电子音乐器和他的乐队一起使用。Melodeon 是一种半音阶手风琴，有人已经制作了电子版本，但对他来说有点太贵了。相反，他买了一个玩具手风琴，并在上面安装了电子设备。

在被遗忘的昔日 PIC 技能挫败后，他选择了 Arduino 作为控制器。两个压力传感器用于检测仪器波纹管的挤压和拉动。他做了一些扎实的工作。上面的视频使用 8 位声音[，就像我们习惯的视频游戏](http://www.youtube.com/watch?v=caIBKOztlAo)一样，休息后的视频来源于更传统的手风琴声音。
