# 简单的红外传感器简化了笔记本电脑的音频控制

> 原文：<https://hackaday.com/2011/12/13/simple-ir-sensor-simplifies-laptop-audio-control/>

![tiny-ir-tv-remote-receiver](img/72d05e38be7f8e33e40ab37013eb9311.png "tiny-ir-tv-remote-receiver")

[欧文]有一个相当大的项目正在进行中，他需要使用红外线在两个节点之间无线发送数据。他的宏伟计划的唯一问题是他从未建造过这类东西。作为一次学习练习，他决定尝试为他的笔记本电脑构建一个无线控制接口，他用它来在做作业的时候播放音乐。

他的笔记本电脑通常位于[欧文]房间的对面，连接着扬声器和放大器。他讨厌重复起床换歌，所以他想他还不如造一个红外接收器来控制 Winamp，Winamp 可以响应电视遥控器的命令。利用他的开放式逻辑嗅探器和一个旧录像机的红外接收器，他破译了他的遥控器的编码系统。然后，他编写了一个 ATtiny13 程序，对红外传感器接收到的信息进行解码，并通过 USB 将其发送到他的笔记本电脑上。

他把东西包装在一个小小的薄荷罐里，使用时挂在台灯上。现在，他只需在遥控器上按几个按钮，就可以轻松地执行 Winamp 中的任何操作。[Owen]说他对结果非常满意，现在他已经牢牢掌握了 IR 信号概念，我们迫不及待地想看看他下一步会做什么。
