# 观鸟遇到了电脑控制的水炮，令人惊叹不已

[![squirrel turret](../Images/551956116f75cff1579e8f7ce3c856ae.png "turret")T2】](http://hackaday.com/2012/03/20/birdwatching-meets-a-computer-controlled-water-cannon-awesomeness-ensues/turret-3/)

当然，松鼠可能会打扰普通的房主，但是很少有人尝试像这个自动水炮塔一样创造性地控制它们。休息后看看视频，看看这是如何完成的，但如果你只想看看松鼠对被喷的反应，快进到 16:00 左右。根据[Kurt]的说法，他确信这将是他的解决方案，然而，他的结论是“松鼠不在乎。”

至于演示，更多的是关于如何使用[【OpenCV】](http://opencv.willowgarage.com/wiki/ "Open CV")，或者开源计算机视觉。这是一个非常强大的软件，尤其是考虑到像这样的东西在正常市场上要花费数千美元。Arduino 用于将计算机的输出连接到现实世界，并控制水枪。如果你不想自己编写这样的程序，你可以像视频结束后有人建议的那样，简单地使用一根软管。
