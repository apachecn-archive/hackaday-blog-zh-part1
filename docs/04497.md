# 能上网的猫喂食器

> 原文：<https://hackaday.com/2009/12/03/internet-enabled-cat-feeder/>

[https://www.youtube.com/embed/kSCC2Z96s3g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kSCC2Z96s3g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

旅行时，照顾你的猫总是一件痛苦的事。有一些商业自动化解决方案，但它们往往价格昂贵，尤其是当需要两个时。不过这两只猫不需要担心。他们有一个[支持互联网的怪物系统](http://www.newtonnet.co.uk/catfeeder/)。

使用的系统非常独特。他们想要互联网连接的继电器，但不想把一台完整的计算机仅仅为了猫喂食器。取而代之的是，一台思科路由器被黑客攻击，运行连接到端口状态灯的继电器。它不仅可以通过互联网控制，还有一个现场直播，所以你可以看到猫作为狂欢。这些是相当幸运的猫。他们还有一个[自动猫门](http://www.newtonnet.co.uk/house/catflap/)。
