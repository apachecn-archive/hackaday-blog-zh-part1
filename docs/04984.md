# DoomBox:经典钥匙遇上小屏幕

> 原文:[https://hack aday . com/2010/03/01/doom box-classic-keys-meet-tiny-screen/](https://hackaday.com/2010/03/01/doombox-classic-keys-meet-tiny-screen/)

![](../Images/fa271491ef3f8b920bbdff557d1ef9d0.png "doomBox")

doomBox 是一款专为 ID 软件经典游戏爱好者设计的游戏装备。[JJ]用一台旧的柯达 DC290 相机制作了这个，它的镜头坏了。由于这运行了 [Digita OS](http://hackaday.com/2004/10/20/add-location-data-to-photos-latitude-and-longitude/#comment-430) ，他能够使用已经存在的毁灭端口。但是相机的出厂按钮不太适合作为控制。通过制作自己的按钮板，并使用传统的键盘按键，他获得了更加舒适的游戏体验。完成的项目位于一个非常熟悉的黑色项目框中。休息之后看他如何点燃激情。

Digita OS 页面的原始厄运似乎已经结束，所以如果你感兴趣，这里有一个替代方案。
