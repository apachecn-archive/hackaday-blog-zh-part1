# Arduino 供电的 2.4 GHz 频谱分析仪

> 原文:[https://hack aday . com/2011/07/25/arduino-powered-2-4 GHz-频谱分析仪/](https://hackaday.com/2011/07/25/arduino-powered-2-4-ghz-spectrum-analyzer/)

![](../Images/e52a1a77f4a125eeabc48a4063755f14.png "3inonespecturmamajig")

[Fred]在我们的提示行中留言，让我们了解 arduino 论坛用户[bilbo]的[最新项目:](http://arduino.cc/forum/index.php/topic,67218.0.html)三合一频谱分析仪、示波器、电压表组合。该产品由 Arduino、无线电板和诺基亚 5110 LCD 分线板组成。跳跃后的(薄)视频显示了钻机的行动。虽然焊接到一个全尺寸的 perf-board，但我们稍后可以看到，更小的电池供电版本在根除任性的蓝牙信号或只是找到丢失的微波炉方面证明是有用的。虽然[比尔博]使用与[相同的无线电](http://hackaday.com/2010/02/12/spectrum-analyzer-wedged-into-a-cellphone/)[板](http://www.mouser.com/ProductDetail/Cypress-Semiconductor/CYWM6935/?qs=eg8mrW3YyI52eYLXBaGRLQ%3D%3D)和相似的构造，但他的创造拥有几种不同的显示模式，以及兼作电压表和微型示波器。[以前的](http://hackaday.com/2011/03/16/dspic-based-spectrum-analyzer/) [频谱](http://hackaday.com/2011/01/08/pic-spectrum-analyzer-uses-fast-fourier-transform-routine/) [分析仪](http://hackaday.com/2010/07/29/graphic-calculator-as-a-spectrum-analyzer/) [builds](http://hackaday.com/2010/10/09/pulling-data-from-the-im-me-spectrum-analyzer/) 都不缺，但是这个是我们见过的第一个在 Arduino 上运行的。

谢谢你的提示[弗雷德]！如果你想在你的下一个项目中加入一些频率扫描功能，别忘了查看[比尔博]的[论坛](http://arduino.cc/forum/index.php/topic,67218.0.html) [帖子](http://arduino.cc/forum/index.php/topic,52655.0.html)以获取源代码！
