# 自动百叶窗开启器

> 原文:[https://hackaday.com/2009/03/06/automatic-blind-opener/](https://hackaday.com/2009/03/06/automatic-blind-opener/)

![blinds](../Images/7bcee21aa1b43cb489db526d5c6cda03.png "blinds")

[发明制造者]在冬天给他的窗户安装塑料绝缘材料时遇到了问题。当你从里面遮住窗户以防止穿堂风时，你接触到打开百叶窗的棍子的机会就被切断了。他的解决方案是用无绳电钻制造一个自动百叶窗开启器。他开始用家里的一些马达，但没有一个足够强劲。最终的解决方案是一个 10 美元的黑色&德克尔钻。在移除手柄和扳机以延伸它之后，他为它做了一个简单的壁挂支架。你可能会注意到他按照自己的方向把一些电池焊接进去。那些只是备用的，以防停电。休息之后你可以看到一段视频。如果您需要由警报触发的[，或者需要](http://hackaday.com/2008/11/18/alarm-clock-automated-blinds/)[不同类型的盲人](http://hackaday.com/2007/09/16/remote-control-your-blinds/)，我们或许可以帮您解决。
