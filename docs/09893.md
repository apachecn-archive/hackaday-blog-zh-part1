# 戳机

> 原文:[https://hackaday.com/2012/03/14/poking-machine/](https://hackaday.com/2012/03/14/poking-machine/)

![](../Images/34d2ccc46a433e27fea515ff4afda70d.png "poke")

我们可能在这里显示了我们的年龄，但我们不知道“戳”一下脸书到底是什么意思。无论是在线跟踪者的消极攻击性表现，还是“喜欢”按钮的延伸，我们所知道的是[Jasper]和[bartholomus]建造了一台机器，将虚拟扑克转化为我们的模拟世界。

[贾斯帕]和[巴特]称之为“戳机”，它通过蓝牙连接与你的手机同步。构建非常简单:只需要一个为 attini 运行 [Arduino 的 attini，一个蓝牙控制器(可能是来自 Sparkfun 的](http://hlt.media.mit.edu/?p=1695)[这个](http://www.sparkfun.com/products/10393))和一个伺服系统。当[贾斯帕]和[巴特]手机上的脸书应用程序收到“戳”的通知时，伺服系统就会启动，并轻轻拍打佩戴者的手臂。

关于这个项目，我们真正喜欢的一点是由几层激光切割丙烯酸螺栓连接在一起制成的外壳。这种情况下提供了一个非常干净的外观，即使它有点笨拙。我们认为这些家伙可以使用一个简单的振动/传呼马达来完成这个构建，但在这一点上它并不完全是一个拨弄机器。休息之后，请观看构建视频。
