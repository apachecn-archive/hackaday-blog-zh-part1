# ATmega8 频谱分析仪

> 原文：<https://hackaday.com/2009/12/27/atmega8-spectrum-analyzer/>

![](img/b33d51728353aa13280713cb8ab2f809.png "mega8-spectrum-analyzer")

[Manekinen]建造了一个非常灵敏的[频谱分析仪](http://www.instructables.com/id/AVR-acoustic-spectrum-analyzer)。聚会上的组件是你所期望的，ATmega8 完成了解释来自 LM324 运算放大器的数据的艰苦工作。这种构建之所以突出，是因为它速度快且可配置。事实上，对校准过程的解释是这个项目的亮点。

HD44780 模块显示光谱数据，而不是使用水的 T0。该设备目前支持几种不同的字符显示，包括 16×2、20×2、24×2 和 20×2。休息之后，我们嵌入了一个 20×4 VFD 的视频。随着视频的进行，注意弹出的波兰语单词。这相当于用 5 键键盘调节亮度和灵敏度。
