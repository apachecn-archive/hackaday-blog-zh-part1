# 蜂窝轮胎不会漏气

> 原文：<https://hackaday.com/2008/11/17/honeycomb-tire-wont-go-flat/>

一种新型轮胎已经研发出来，即使被炸了也不会瘪。Resilient Technologies L.L.C .已经开始为军方制造这些看起来非常酷的轮胎。坚硬但柔韧的塑料蜂窝起到减震器的作用，就像充气轮胎一样。与普通轮胎不同，它不会爆裂。你必须完全破坏轮胎来阻止这些。这篇文章提到这并不是第一次。[米其林生产的“tweel”](http://www.michelin.com/corporate/actualites/en/actu_affich.jsp?id=17448&lang=EN&codeRubrique=4&actu=true)用于工业用途，但你可能还记得奥迪使用它们在赛道上行驶的视频。如果没有，休息后再来看看。
