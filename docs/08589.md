# 安培小时采访[乔格兰德]

> 原文：<https://hackaday.com/2011/09/14/amp-hour-interviews-joe-grand/>

![](img/085d47c1527c228bd083584e94187fa8.png "the-amp-hour")

Amp Hour 是一个电子爱好者和专业人士的播客，刚刚对[Joe Grand]进行了一次史诗般的采访。除了主持人(克里斯·甘梅尔)和(达夫·琼斯)之外，这场讨论涵盖了硬件黑客世界的所有兴趣点。第一个短片探索了*原型的兴起、运行和衰落*、[，这是一个以工程为中心的电视节目，由【乔】和一群不同的工程师为探索频道制作](http://hackaday.com/2008/10/14/prototype-this-premiers-tomorrow/)。他礼貌地讨论了电视业务的优缺点，以及这如何影响团队深入了解他们正在建设的项目的能力。

在那里，这些人讨论了黑客空间多年来的发展。[Joe]对在这些 DIY 领域注入企业赞助以及这在长期可能意味着什么有些担忧。然后，他谈到了即将发布的为期 3 年的激光测距仪项目(我们已经看到了使用这种传感器原型的项目)。这个节目的结尾是关于硬件工厂的讨论，这些工厂[Joe]已经使用了多年，用于像 Defcon 徽章这样的项目(我们喜欢他的 [Defcon 18 徽章](http://hackaday.com/2011/01/07/a-look-back-at-defcon-18-badges/))。

这是很棒的一集，所以下载一份，留出一个小时听完整集。
