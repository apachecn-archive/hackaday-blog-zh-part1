# 带手表的机器人拳击

> 原文：<https://hackaday.com/2010/08/03/robot-boxing-with-wrist-watches/>

经典的注塑塑料 Rock 'Em Sock 'Em 机器人已经[升级为使用运动控制](http://processors.wiki.ti.com/index.php/Electronic_Rock-Em_Sock-Em_Robots)。这个项目使用了四块 [TI Chronos 手表](http://hackaday.com/2009/11/25/ti-sports-watch-for-hacking/)，两个玩家的手腕上各有一块。在休息后的视频中，我们可以清楚地看到基本单元的内部结构。我们对改造每个带有四个伺服系统的塑料机器人的高质量工艺印象深刻。电子设备包括一些铃铛和哨子，如记录分数的 SD 卡，可以通过保存的输入重播比赛。如果你手头有几块这样的手表，我们很乐意看到你移植这个项目，[让它成为一个链接控制器](http://hackaday.com/2009/05/18/punchy-punchout-improved/)。
