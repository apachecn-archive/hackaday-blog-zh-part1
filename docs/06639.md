# VR！现在有了更多的 Kinect、Wiimote 和 Vuzix

> 原文:[https://hack aday . com/2011/01/25/VR-now-with-more-Kinect-wiimote-and-vuzix/](https://hackaday.com/2011/01/25/vr-now-with-more-kinect-wiimote-and-vuzix/)

![](../Images/ecc9c6747f80945e6a00154e083252d8.png "yes, shoot it")

我们当中那些记得你可以去购物中心玩虚拟现实游戏机的人，往往会深情地记住它。发生了什么事？计算能力增长如此之快，我们现在的图形简直令人惊叹，然而虚拟现实并没有大的复兴。是的，那些头盔太大了，让你头疼，但是值得。凭借最新游戏 crazes、Wiimote 和 Kinect 的 3d 定位功能，【Nao_u】终于将这个我们都知道[应该去的地方](http://game.g.hatena.ne.jp/Nao_u/20110121#p1) ( [谷歌翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=ja&tl=en&u=http%3A%2F%2Fgame.g.hatena.ne.jp%2FNao_u%2F20110121%23p1))。嗯，也许我们会有不那么令人毛骨悚然的动漫脸飞来飞去喷墨水，但基本的东西在那里。他创建了一个虚拟现实系统，利用 Wiimote 来定位他的手，Vuzix 显示器来定位头部，kinect 来跟踪身体。即使有令人毛骨悚然的飞行头，我也想玩它，特别是在休息后看到他在视频中躲在盒子后面。VR 万岁！
