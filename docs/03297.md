# 激光蚀刻 Kindle 2

> 原文:[https://hackaday.com/2009/03/13/laser-etched-kindle2/](https://hackaday.com/2009/03/13/laser-etched-kindle2/)

![kindlesketch1](../Images/8f1bbd1556a203833dfd2871502005cf.png "kindlesketch1")

在看到 xkcd 漫画中他们称 Kindle2 为《搭车人指南》后，【Ladyada】忍不住用[激光在 Kindle 2](http://www.adafruit.com/blog/2009/03/12/first-laser-etched-kindle-2-the-hitchhikers-guide-to-the-galaxy-dont-panic/) 上刻上“不要慌”。我们认为它看起来很好，虽然有点起泡。休息后可以看到整个过程的视频。既然 xkcd 已经渗透到我们的网络、心灵和思想中，也许他可以花一点点努力去学习绘画。如果你没有激光蚀刻机，你总是可以[自己制作](http://hackaday.com/2005/03/27/cut-sheet-metal-with-your-homemade-laser/)。只是要小心，不要不小心把你的 kindle 撕成碎片。

kindle 并不是我们迄今为止见过的最奇怪的东西。我们见过用激光切割的比萨，甚至 T2 的激光纹身。
