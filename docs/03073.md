# 自动铲雪的机器人

> 原文：<https://hackaday.com/2009/01/08/robot-that-shovels-snow-automatically/>

[https://www.youtube.com/embed/xdEf60wcKcs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/xdEf60wcKcs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

你们中那些生活在多雪气候中的人会对 I 型铲流口水，这是一种电池供电的机器人，可以铲掉车道上的雪，在一个冬天里为你节省了无数个小时的辛苦劳动。它的发明者声称，尽管它的发动机动力相对不足，但它即使在大雪中也能保持车道畅通；显然，诀窍在于机器人不断监测车道上的积雪量，并在一个重要但可控的积雪层形成时立即采取行动。不幸的是，I-铲子仍然是一个原型，但如果运气好的话，你很快就能买到一个。当然，如果你没有耐心，你可以试着自己做一个。
