对于那些在课堂上没有专心听讲的人，这里是过去一周我们博客上最精彩的内容:

排名第一的是一个关于一个项目的帖子，一些雄心勃勃的黑客找到了在数码相框上运行 Linux 的方法。太棒了。

接下来是一个帖子，其中[克里斯]需要一些帮助来启动他的[1/10 规模、周期精确的 Cray I](http://hackaday.com/2012/01/10/help-chris-boot-his-cray-1-supercomputer/) “超级计算机”。

排在第三位的是一个关于[的帖子，这是一个非常精致的 737 驾驶舱模拟器](http://hackaday.com/2012/01/06/737-cockpit-will-satisfy-even-the-most-discriminating-simulator-afficiandos/)，它是在过去几年里建造的。这是一个相当紧张的建设，有很多关于它是如何完成的细节。

接下来是一个关于 [3D 打印设备的帖子，该设备类似于幻灯机镜](http://hackaday.com/2012/01/11/3d-printed-zoetrope/)，但不是使用狭缝让你连续看到模型，而是在适当的时候闪烁一些 led。

最后，排在第五位的是一篇关于 Makerbot Industries 最新发布的文章，它将让你打印出自己的 phenakistoscope。这种 3D 打印机比 makerbot 更大，可以选择配备两个挤出机，而不是一个，这样你就可以制作双色或双材料。
