# 一罐萤火虫

> 原文：<https://hackaday.com/2008/10/18/jar-of-fireflies/>

![](img/f6df25bd7f3fa5c0450beff688f4c882.png "firefly")

[Randomskk]尝试制作电子萤火虫已经有一段时间了，最终选定了一个他喜欢的设计。他的萤火虫罐使用一个 ATtiny13 来控制一组 12 个矩阵 led。绿色 SMD 发光二极管每个都焊接到一对细导线上，细导线垂到罐子里。软件随机选择一个 LED 闪烁，然后闪烁 1-3 次。每次打开罐子时，随机种子都会增加，因此您将获得 255 种不同的模式。电源只是一个标准的硬币电池。该项目在电气方面相当简单，但 LED 焊接可能会很困难。它的灵感来自这个[萤火虫罐项目](http://www.instructables.com/id/Jar-of-Fireflies/ "Jar of Fireflies")。也看看亚历克斯的[同步萤火虫](http://hackaday.com/2008/07/28/synchronizing-fireflies-ng/ "Synchronizing Fireflies NG  - Hack a Day")。下面是罐子的视频。
