# 儿童聚会的视频游戏装置

> 原文：<https://hackaday.com/2011/02/01/video-game-installations-for-kids-parties/>

![](img/eb8ade8475403f6c211372b560fe757e.png "kids-video-game-installations")

为什么不会有人为孩子着想？！事实上，一些最好的黑客来自娱乐小家伙。拿两个视频游戏版本来说。第一个是基于望远镜的控制器，用于向海盗船的投影发射虚拟炮弹。第二种是两人合作游戏，一个人开车，另一个人射门。他们都使用投影仪来显示比赛场地，红外激光器用于瞄准，NTSC 摄像机用于拾取激光点的位置。这真的很好，这要归功于物理构建的质量，以及游戏方面出色的音频和视频。休息后在视频中自己看吧。

帖子里的几个帖子[成堆的垃圾邮件]谈到了激光强度。由于有了定位系统，他想确保不会有一屋子半盲的五岁儿童。
