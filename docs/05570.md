# 可编程折纸

> 原文：<https://hackaday.com/2010/06/29/programmable-origami/>

[https://www.youtube.com/embed/vZf3lo-16wQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/vZf3lo-16wQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

麻省理工学院的研究人员想出了这个他们称之为[可编程物质](http://www.pnas.org/content/early/2010/06/24/0914069107)的巧妙演示。这种平板上覆盖着微小的箔片驱动器，可以通过编程折叠成特定的形状。上面的视频显示的是一艘船和一架飞机。利用折纸术多年来形成的概念，他们可以按照特定的排列方式将纸张分成三角形，以形成特定的形状。这个很简单，但是从我们见过的一些疯狂的折纸来看，这个会变得很酷。
