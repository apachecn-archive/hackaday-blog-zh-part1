# 这个 CheerLights 显示器有自己的想法

> 原文:[https://hack aday . com/2011/12/19/this-cheerlights-display-has-a-mind-of-own/](https://hackaday.com/2011/12/19/this-cheerlights-display-has-a-mind-of-its-own/)

![cheeriobot](../Images/a931dd1aaec0e70da566baa774ae794a.png "cheeriobot")

[Axel]想在这个假期参加 CheerLights 项目，但不是一个总是遵守规则的人，他决定让他的展示与大多数其他人有所不同。虽然他家的灯是与啦啦队灯项目同步的，但他给自己的啦啦队机器人增加了一点个性。

通常情况下，Cheeriobot 很乐意跟随世界其他地方的变化，只要 Twitter 上有指示，它就会改变自己的颜色。然而，如果事情有点慢，Cheeriobot 会不耐烦，会自己向@ CheerLights 发送一条推文，以确保它不会长时间显示单一颜色。

[阿克塞尔]还创造了一种模式，将 Cheeriobot 变成一种反向投资者。当有人发微博时，显示器的“反叛模式”会使它改变颜色，但它会随机选择一种颜色，而不是跟随其他颜色。

这绝对是 CheerLights 项目的一个有趣的转折，我们真的很喜欢这样一个事实，即如果推文流变慢了，它会让事情继续进行。
