# Midi 钢琴教师

MIDI 钢琴教练是一个工具，针对那些希望学习弹钢琴，但不想上课的人。安装在按键上方的 LED 条会亮起，准确地告诉您何时按下哪个键。我们已经在一些电子键盘上看到这种功能有一段时间了，但是这种装置可以适用于任何标准尺寸的键盘。查看页面上的图表和图片。
