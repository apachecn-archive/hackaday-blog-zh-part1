# 多米诺时钟使用机电显示器

> 原文：<https://hackaday.com/2010/12/18/domino-clock-uses-an-electromechanical-display/>

![](img/4ec70190c70e577c099c40e5996d196a.png "domino-clock")

这个时钟概念使用大的[多米诺骨牌和变化的面孔来显示时间](http://news.carbondesign.com/carbon-blog/repurposing-an-icon-introducing-carbons-domino-clock/)。据我们所知，他们还没有制造出成品，但是我们喜欢对原型中的[工程的解释。休息之后，您可以观看[Eric]解释他如何完成一个缓慢变化的数字的设计要求，该数字不使用任何功率来保持其状态，在改变状态时也使用低功率。为了实现这一点，他设计了一个翻转的圆圈，一旦设定，它会保持在白色和黑色的位置。当需要改变数字时，线圈被通电，推动他所谓的单极电机中的磁铁。不管叫什么名字，我们都想自己造一个！](http://news.carbondesign.com/carbon-blog/making-it-real-the-engineering-behind-carbons-domino-clock/)
