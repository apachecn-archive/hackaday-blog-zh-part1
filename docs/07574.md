# 不要按那个开关！

> 原文：<https://hackaday.com/2011/05/10/dont-hit-that-switch/>

![switch_box](img/20f859a07dd7d7efecfc380da62cedc1.png "switch_box")

Hackaday reader [Danukeru] [给我们发了一段视频，展示了一个基于盒子的机器人，它有着有趣的个性](http://www.youtube.com/watch?v=UmQ5LsNMXZ4)。这个盒子相当简单，从外面看似乎只有一个开关和一个 LED。然而，当开关被扳动时，盒子就活跃起来了。

当盒子被激活时，盖子打开，一只小手臂伸出来关闭开关。我们已经看过很多次了，但是这次看起来有点不同。在视频中，这个过程似乎重复了几十次，然后机器人就会生气并失控。起初，我们认为视频的结尾部分有点数字欺骗，但在查看了创作者的博客后，看起来它可能是合法的。在视频中很难看到盒子的内部，但它确实有一个遥控汽车底盘，允许它四处移动和旋转，如下所示。
