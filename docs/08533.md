# 巨大的 DIY 3D 打印机可以打印巨大的 3D 物品

> 原文:[https://hack aday . com/2011/09/08/huge-DIY-3d-printer-predictable-prints-huge-3d-items/](https://hackaday.com/2011/09/08/huge-diy-3d-printer-predictably-prints-huge-3d-items/)

好吧，如果我们需要在 RepRap 上建造巨大的东西，现在我们知道该找谁了。草根工程的[Jim]来信分享了他刚刚完成的大型 3D 打印，这是由同样令人印象深刻的[自制 3D 打印机挤出的。](http://www.grassrootsengineering.com/blog/mymachine)

他建造的部分由一束交叉图案的中空绿色管道组成，并在不到 2 天的时间内打印出来。他从来没有提到这个零件的用途，但他估计这是用 DIY 3D 打印机打印的最大的单个零件，尺寸为 376 x 376 x 250mm 毫米(14.8 x 14.8 x 9.84in 英寸)。

打印机本身是基于 RepRap 的，但显然要大得多。它使用 ABS 塑料打印，最大构建面积为 403 x 403 x 322.7mm 毫米(15.86 x 15.86 x 12.7in 英寸)。它历时 3 年建成，于 2010 年全面投入运营，但随着[Jim]需求的变化，它经历了多次改进。

这是一台外观很棒的机器，看起来打印得很好。我们当然希望在我们的车间里有一个这样的东西。
