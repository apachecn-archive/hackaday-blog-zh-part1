# 将捉人游戏带入数字时代

> 原文:[https://hack aday . com/2011/05/12/将标签游戏带入数字时代/](https://hackaday.com/2011/05/12/bringing-the-game-of-tag-into-the-digital-age/)

![tagurit](../Images/c88cf03199a63d03036fb39f50877283.png "tagurit")

你多久没玩捉人游戏了？

来自皇后大学人类媒体实验室的西尔维娅·程、和罗尔·维特加尔在经典游戏上编造了一个有趣的转折，可能会迫使你再次开始玩游戏。

他们的游戏名为 TagURIt，两名玩家穿着内置触摸传感器的发光二极管 t 恤。被称为“追逐者”的第三个玩家试图触摸其他玩家中的任何一个，以捕捉显示在玩家胸部 LED 矩阵上的标记。这个游戏是基于分数的，给追逐者奖励分数以捕捉代币，同时给其他玩家奖励分数以避免捕捉。

如果两个穿 LED 衬衫的玩家彼此靠近，代币会跳到另一个玩家身上，试图阻止追逐者。在这个游戏中，每个玩家都是一个带有位置标签的 URI，如果游戏在室外进行，则通过室内摄像头跟踪用户或通过射频传感器来确定接近度。

这绝对是一种有趣的玩捉人游戏的方式，我们想象在大群人当中会很有趣。
