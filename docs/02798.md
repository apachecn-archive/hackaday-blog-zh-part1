# 全结构软交换机

> 原文：<https://hackaday.com/2008/11/16/full-fabric-soft-switches/>

![soft_switch](img/af275ff8c523cc62710df5baff8b76c2.png "soft_switch")

当我们为 Engadget 创建我们的[背包带 WiFi 检测器](http://www.engadget.com/2005/09/13/how-to-embed-a-wifi-detector-into-a-backpack-strap/ "Embed a wifi detector into a backpack strap - Engadget")时，我们嵌入了软开关来控制设备。他们用两片铝箔，中间隔着一层打了孔的泡沫。[Plusea]采取了类似的方法，但[用导电织物](http://www.instructables.com/id/Three_Fabric_Buttons/ "Three Fabric Buttons")取代了铝箔。最终的结果是一套三个软按钮，可以很容易地清洗。看看下面的视频吧。检查后续项目，使用按钮[替换滚轮](http://www.instructables.com/id/Scroll_Wheel_Hack/ "Scroll-Wheel Hack")。
