# 灵活旋钮

> 原文：<https://hackaday.com/2010/02/19/flexi-knobs/>

![](img/01a926aef57c8b882a6719612170c314.png "flexiknobs")

这些被称为[灵活旋钮](http://tei-conf.org/10/Program/FlexiKnobsBridgingTheGapBetweenMouseInteractionAndHardwareControllers)的盒子，工作起来就像一个无线雅达利桨和鼠标集于一身。每个都有一个旋转编码器，也可以像按钮一样点击。内部是一个无线光学鼠标，它可以控制屏幕上的光标，该光标与旋钮的颜色相匹配。在休息后的视频中，你可以看到这些被用作 midi 控制。每个光标都可以锁定在一个虚拟旋钮上，给它一个物理界面。因为在一台机器上使用了几个单元，这就产生了一种抽象的多点触摸系统。这将为其他具有过多设置的应用程序提供一个良好的界面，比如 [Blender](http://www.blender.org/) 。
