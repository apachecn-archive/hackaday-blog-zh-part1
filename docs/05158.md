# 乐高变速箱–七速加倒档

> 原文:[https://hack aday . com/2010/04/08/Lego-gearbox-seven-speed-plus-reverse/](https://hackaday.com/2010/04/08/lego-gearbox-seven-speed-plus-reverse/)

![](../Images/9a8701d4c7f8e370ce229dda4c175236.png "lego-gearbox")

我们知道乐高是一种非常通用的媒介，但这超出了我们的想象。这辆乐高车的变速箱内置了七个速度档和一个倒档。这不完全是一个原创设计，但增加了在[一个十年前的乐高玩具](http://www.brickset.com/detail/?set=8448-1)中发现的五速设计。休息后，请看视频演示。设计采用了[顺序变速箱](http://auto.howstuffworks.com/sequential-gearbox.htm)；根据您想要的换档方式，通过向上或向下点击操纵杆来完成换档。如果你手头有足够的零件，你可以使用[Sheepo]发布的[组装照片](http://www.brickshelf.com/cgi-bin/gallery.cgi?f=417698)来建造它。
