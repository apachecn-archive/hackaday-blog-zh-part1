# 一个更简单的智能手机车库开门器

> 原文：<https://hackaday.com/2011/06/15/an-even-simpler-smartphone-garage-door-opener/>

![](img/a79cf8046cd9380c7e4a6f7647b58eae.png "hm1100andliftmaster")

我们过去见过智能手机车库开门器，但[Lou]的[Hack 因其简单而美丽。他的开门器解决了这个问题，不需要使用电脑、Arduinos、无线模块甚至任何基于智能手机的应用程序。这个项目只需要一个蓝牙耳机和一个晶体管。开门器使用三星 HM1100 蓝牙耳机，[Lou]对其进行了重要测试，以表明他的创作非常安全，不会意外开门。](http://btmate.com/index.php "main link")

当这款耳机连接到手机时，它会从耳塞发出嘟嘟声，因此[Lou]移除了扬声器，并用晶体管取而代之。现在，他可以使用哔哔声之前由放大器产生的电压尖峰作为他的开关信号。通过将晶体管与车库内的门按钮并联，他能够通过连接无线开门，然后立即从耳机上断开。这种设置显然非常安全，因为最初将手机与耳机连接的唯一方式是在车库内。休息之后，请观看视频，了解构建说明和演示。
