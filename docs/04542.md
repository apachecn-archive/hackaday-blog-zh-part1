# 制作您自己的 3d 显示器

> 原文：<https://hackaday.com/2009/12/14/make-your-own-3d-monitor/>

 <https://www.youtube.com/embed/NektsV1EOic?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>不要让这个粗俗的视频愚弄你。[Sprite_tm]在这方面做得非常出色。他总结出了一个相当简单的方法，可以在家里制作你自己的 3d 显示器。基本原理很容易得到。你必须同时在屏幕上提供一个物体的不同角度。[Sprite_tm]已经通过使用透明片做到了这一点，该透明片打印有仅显示特定像素的图案。当你移动头部时，你会看到一组不同的像素，因此会看到不同的图像。很简单，但是很管用。这个系统其实并不新。带偏振眼镜的传统 3d 使用非常相似的设置。然而，这个系统似乎缺乏立体感。当你移动你的头时，你看的是一个不同的角度，但仍然是它的二维图像。如果他能以某种方式呈现每只眼睛不同的视角，而不用偏光滤镜和眼镜，那将会非常酷。<a href="http://hackaday.com/2009/11/04/stereoscopic-3d-with-a-pic-micro/">我们已经看到其他人用<a href="http://hackaday.com/2009/10/09/portable-digital-stereoscopic-image-viewer/">多屏</a>做了</a>，但那变得如此笨重，会很快让大多数人头疼。我们还认为我们将会看到大量的面部跟踪透视修改，特别是随着 Natal 项目和类似技术的出现。</p> </body> </html>
