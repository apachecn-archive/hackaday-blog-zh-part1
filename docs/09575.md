# 模块化控制器，你迫不及待地向你的朋友炫耀

> 原文:[https://hack aday . com/2012/01/25/modular-controllers-you-can-wait-to-show-off-your-friends/](https://hackaday.com/2012/01/25/modular-controllers-you-cant-wait-to-show-off-to-your-friends/)

![](../Images/018aed75a2a312083dfde8a2d106a52e.png "modular-controllers")

让我们直说吧，[亚历克斯]将向我们展示如何制作这样的控制器？我们在哪里报名？即使没有看到它在行动中，我们也想要一个，但在他展示它后，建造的欲望甚至更大(休息后检查剪辑)。他是一名设计专业的学生，他做了一个开源项目，旨在使构建与几乎任何软件应用程序匹配的硬件控制器变得更加容易。

对外部控制器的需求正在上升，首先是基于音乐的应用程序，如 DJ 工具和音乐家的 Midi 控制器(我们正在考虑[单体克隆](http://hackaday.com/2010/01/12/arduinome-with-many-pots/))。但是任何可以从 USB HID 获取输入的东西都可以用这样的东西来控制。这是因为[亚历克斯]正在使用 Teensy 控制器板作为接口。只需选择您想要的输入类型——滑块、电位计、按钮、开关——然后将它们连接到微控制器引脚。如果你开始耗尽输入，他还讨论了一些附加芯片用作端口扩展器。
