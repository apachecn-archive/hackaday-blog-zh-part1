# 万圣节黑客:孩子们的鬼屋项目

> 原文：<https://hackaday.com/2011/10/15/halloween-hacks-a-haunted-house-project-for-the-kids/>

![](img/b509209447a3a514553925a79033946f.png "Halloween")


如果孩子们一直缠着你开始你的万圣节装饰，来自 BasicMicro 的[Dale]有一个简洁的互动项目，肯定能满足他们对恐怖乐趣的渴望。

他的妻子正在为这个万圣节寻找一些新的装饰品，所以他去了趟工艺品商店，找到了一个 DIY 泡沫鬼屋工具包。在说服她进行组装后，他为显示器配备了一些斯卡莱特 RGB 模块，这些模块内置了各种有趣的照明模式。当灯光变暗时，房子就活跃起来，就像你在下面的视频中看到的那样。

建造和布线并不是一项过于复杂的工作，因此对于开始对电子产品感兴趣的小心灵和小手来说，这是一个很好的起点。对于那些有一点经验的孩子来说，房子可以很容易地修改，使用伺服系统来创建摆动的门和摇晃的墓碑。

这种极限是由他们的想象力(和你的电子预算)真正定义的，所以为什么不在这个周末试一试呢？我们很想看看你和你的初露头角的黑客们都做了些什么！
