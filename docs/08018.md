# Bluescripts 让 Android 的蓝牙控制变得更加简单

> 原文:[https://hack aday . com/2011/07/08/blue scripts-makes-bluetooth-control-from-Android-a-bit-easy/](https://hackaday.com/2011/07/08/bluescripts-makes-bluetooth-control-from-android-a-bit-easier/)

![](../Images/b99cf39d8f03bc34ac036aa59ac1894c.png "android-bluescripts")

这里有一种方法可以让[使用 Android 设备](http://teholabs.com/docs/bluescripts:overview)控制你的项目。Bluescripts 是 Android market 上的一款免费应用，它使得创建发送可定制消息的界面变得更加容易。如果您的项目中有一个蓝牙接收器，连接它就像将 MAC 地址放入 Android 设备上的 XML 文件中一样简单。该文件中的每个标记都有一个名称，以及目标的地址和应该发送的消息。在接收端，您只需要确保您的项目硬件准备好接收 ASCII 消息，并根据传入的内容采取行动。

休息后请观看演示视频。如果你正在[编写你自己的 Android 程序](http://hackaday.com/2010/07/12/android-development-101-%E2%80%93-a-tutorial-series/)，这也许不像你能做的那样酷，但是我们想不出有什么东西能让 Android 界面变得如此快速和简单。
