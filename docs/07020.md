# 电视示波器

> 原文:[https://hackaday.com/2011/03/07/tv-oscilloscope/](https://hackaday.com/2011/03/07/tv-oscilloscope/)

![](../Images/acc85025fd4de10c358e40eb6617d67a.png "Fully-Functional-Television-Oscilloscope")

需要示波器？想看音乐吗？我没有钱，但我有一台旧电视。那么这种[电视到示波器的模式](http://www.instructables.com/id/Fully-Functional-Television-Oscilloscope/)可能正合你的胃口。现在还不要走开，当你在 CRT 设备内部工作时，你会暴露在电源电流、高电压和高频率下，所以需要格外小心。

如果你有你的橡胶焊接手套，并已放电你所有的脂肪电容器(包括阴极射线管)它几乎相同的魔术夫妇我们有[以前](http://hackaday.com/2009/07/29/music-visualizer-oscilloscope/) [特色](http://hackaday.com/2005/03/19/wave-vessel-audio-visualization/)。将放大的输入信号接入垂直偏转线圈，让她走，但这个 instructable 具有更详细的说明和照片，所以你有更好的机会复制这个(不太实验室级)潜在有用的设备。
