# 征用公共视频屏幕:真的还是假的？

> 原文:[https://hack aday . com/2011/03/14/commanding-public-video-screens-true-or-fake/](https://hackaday.com/2011/03/14/commandeering-public-video-screens-real-or-fake/)

![](../Images/e6ed2b1ea53ac46065752a7aaaacc915.png "times-square-video-hack")

到了大家最喜欢的评论线程游戏:真的假的？本周的版本来自于[菲尔]发来的一个关于接管时代广场的视频屏幕的方法的提示。休息后观看视频，看看黑客们使用两部分的解决方案，在繁忙的城市环境中，将 iPhone 上的视频转播到屏幕上。第一部分是一个插入 iPhone 的发射器，第二部分是一个信号中继器，当靠近视频屏幕时，它会覆盖当前与手持设备上的视频一起显示的剪辑。上面的图像显示了使用一个巨大的红色气球将中继器漂浮到大屏幕上，你可以在重播视频左侧的黑色栏中分辨出来。

我们首先想到的是，有人刚刚看了《创:遗产》,想在互联网上享受一点科幻的乐趣。我们无法想象一个硬件解决方案可以真正实现这一点，但是请在评论中分享你的想法。我们怀疑这更像是一个使用了特效的视频黑客，类似于[眼睑快门眼镜](http://hackaday.com/2011/02/22/eyelid-shutter-glasses-fake-but-still-a-hack/)视频的[停止运动蜡烛视频](http://hackaday.com/2010/06/18/candle-stop-motion-hows-it-done/)是如何伪造的。但很明显，接下来会有一个视频展示原型是如何制作的，所以我们可能是错的。
