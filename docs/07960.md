# 替代手机充电设备

> 原文:[https://hack aday . com/2011/06/30/alternative-phone-charging-devices/](https://hackaday.com/2011/06/30/alternative-phone-charging-devices/)

[![](../Images/846f859dda3f0387ea5c6de98a66a2e4.png "usb_mobile_charger")T2】](http://hackaday.com/2011/06/30/alternative-phone-charging-devices/usb_mobile_charger/)

我们都听说过给你的设备充电的太阳能电池，或者摇一摇就能充电的奇怪的手电筒，但是这种类型的充电器对几乎每个人来说都应该是新的。据报道，这种“盘式充电器”能够使用 USB 连接在 3 到 5 个小时内为手机或其他移动设备充电。它还有一个内置的收音机和提灯。如果你只是生活在一个没有现成电力的地区，这应该是一个很好的工具。

目前在非洲使用的第二个充电器是一个连接在自行车上的小型发电机。由于这种交通方式在发展中国家相当普遍，这个简单的想法无疑显示了希望。休息之后，看看自行车手机充电器的视频。
