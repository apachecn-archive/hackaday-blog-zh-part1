# 用 4016 和 555 拆下功能锁定键

> 原文：<https://hackaday.com/2012/01/07/remove-function-lock-key-with-a-4016-and-a-555/>

![](img/cb338b00213047a785cdb4836a6ab806.png "IMG_1133")

如今，许多公司试图通过过度复杂化键盘来简化生活。微软从 2001 年就开始做了。如果你喜欢你的人体工程学键盘，但讨厌“功能锁定”键，有很多选择供你尝试。

最简单的方法是修改一些 XML 或者只是在 MS 软件中设置宏，但是谁愿意每次重新安装 Windows 时都这样做呢？读者[Elco]没有，所以他在键盘内添加了一个简单的小 555 电路，如果他在快速和激烈的打字过程中碰巧击中了它，[会在三秒钟后自动重新启用羊群。](http://www.elcojacobs.com/hardware-fix-for-the-f-lock-key-on-my-microsoft-natural-ergonomic-keyboard/)

现在，无论键盘连接到什么系统，他都不必担心，如果他点击 F7，系统是否会对他的文档进行拼写检查，或者回复电子邮件，或者 F2 实际上会重命名某个东西，而不是悄悄地撤销他的工作。
