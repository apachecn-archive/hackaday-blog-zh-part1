# 用简单的方法硬重置你的福特触摸系统

> 原文:[https://hackaday . com/2012/03/08/hard-reseting-your-Ford-touch-system-the-easy-way/](https://hackaday.com/2012/03/08/hard-resetting-your-ford-touch-system-the-easy-way/)

![ford-sync-reset](../Images/b62c70cac2a10e71a370cee293c027cd.png "ford-sync-reset")

[杰夫·克莱默]拥有一辆福特福克斯，虽然他对这辆车总体上很满意，但“我的福特触摸/同步”系统有时会出问题。他每天花很多时间在车里，所以当娱乐中心锁死的时候，他不得不关掉汽车，拔掉保险丝来重置系统。由于在旅途中拔掉保险丝是非常不切实际的，他决定安装一个重置按钮，使系统重启变得轻而易举。

他从拆卸各种保险丝开始，直到他找到一个易于拆除的保险丝。一旦它成了碎片，他就将一对电线焊接到保险丝端子上，并将所有东西连接到一个常闭的瞬时按钮开关上。在添加了内嵌保险丝座并重新插入原来的保险丝后，他将按钮安装到了手套箱的背面

现在，他不再需要在每次立体声锁定时拔掉保险丝，只需按下按钮就可以上路了。希望[Jeff 的]汽车能尽快得到软件修复！
