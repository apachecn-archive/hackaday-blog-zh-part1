# 当你赢了，投币游戏就有回报

> 原文:[https://hack aday . com/2010/04/16/coin-op-pay-out-when-you-win/](https://hackaday.com/2010/04/16/coin-op-pays-out-when-you-win/)

![](../Images/c7aa2790a0a11aa46e754a3faa2f60b0.png "video-game-pays-out")

可悲的是，把硬币塞进单机游戏的日子越来越少了。[Dirk]想出了一个延长怀旧时间的方法，以一种有用的方式将货币融入到体验中。他重建了视频游戏《雷电》( Raiden )( T1 ),当你赢得游戏时，他会给你一份奖励。现在每玩一次要花一枚硬币，但是如果你坚持到最后，你可以收回这笔费用。

[德克]拿了一个原始的橱柜游戏，做了一些危险的工作来替换旧的 CRT 显示器，并改装了一台 MAME 机器来处理游戏。他使用的是 Windows，因此遇到了一些问题，但正如你在休息后看到的，事情最终都解决了。从最初的设计[到成品，吐出硬币的漏斗硬件经历了几个步骤，但它一直基于一个通过并行端口连接到 MAME 盒子的 PIC 控制器。这是一个有趣的除了](http://www.dirkdirk.com/projects/raiden/?p=34)[任何 MAME 内阁](http://hackaday.com/2009/09/26/frankenstein-has-a-thing-for-coin-op/)。
