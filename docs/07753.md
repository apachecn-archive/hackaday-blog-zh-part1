# 像训练宠物一样给机器人编程

> 原文:[https://hack aday . com/2011/06/04/programming-robots-like-you-could-train-a-pet/](https://hackaday.com/2011/06/04/programming-robots-like-you-would-train-a-pet/)

![](../Images/07e450e689a7879c250429e3473f3e36.png "dog-leash-robot-training")

[Jim]一直在与一个来自各大学的团队合作，开发一种直观的方法来指导和训练辅助机器人。他们专注于一项特殊的技术，训练一个机器人用皮带跟随，就像你训练宠物狗一样。

在阅读了 Kinect 驱动的购物卡机器人后，他受到启发，发送了一个他的研究的链接。他认为那个项目和他自己的相似，但是他的项目有几个额外的好处。第一，如果一个机器人被拴着，每个人都知道这个机器人在跟踪或帮助谁。但是有一个额外的好处，用户不需要任何培训。这是因为牵着狗散步的行为在发达国家很普遍；你可能从未养过狗，但你见过别人无数次牵着狗散步，你自己也可以这样做，无需任何训练。

皮带连接到机器人身体中心充满传感器的转台。机器人可以感知用户何时向哪个方向拉动皮带。为了增加功能，手柄上还有一个紧急切断开关。休息之后，看看一些测试视频，看看人类能多快适应这种类型的用户界面。
