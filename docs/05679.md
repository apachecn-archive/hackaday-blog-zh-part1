# AVR Guardian 过滤掉狗

> 原文:[https://hack aday . com/2010/07/19/AVR-guardian-filters-out-dogs/](https://hackaday.com/2010/07/19/avr-guardian-filters-out-dogs/)

![](../Images/3aa5e065f8475e80f2861b8d54ee714f.png "avr-guardian")

当猫和狗一起被养大时，它们可以相处得很好。这种安排的问题是让狗远离猫箱和猫粮。[Ryan Meuth]尝试了一种简单的电子屏障来让狗远离。它使用红外发射器和接收器将一束不可见光照射到猫房间的门口。在休息后的演示中，你会看到他采取措施确保猫不会触发警报。光束被设置得足够高，以至于它们的身体不会挡道，固件会测量光束被阻断的时间，以避免猫的尾巴引起的假阳性。如果狗真的试图进入房间，它会打破横梁，并引发一个高音调的报警声。

有趣的是，狗不喜欢这个声音，但猫似乎不怕它。此外，我们有一只不太凶猛的猫，它喜欢咀嚼连接两个模块的绳索。尽管如此，这仍然是一个对[Ryan]有效的解决方案，并且可以集成到自动喂食器中，当你不在家时，让狗远离喂食。
