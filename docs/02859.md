# 真空鼠

> 原文：<https://hackaday.com/2008/11/26/vacuum-micromouse/>

![micromouse](img/cacada21bfa1961f9f8af6c70bc3f8ed.png "micromouse")

从 1980 年起，日本就开始举办老鼠比赛。在这段时间里，规则集基本上保持不变。自主机器鼠在一个 16×16 的迷宫中导航，创建地图。然后，它确定未来运行的最佳解决方案。目前的记录在 6 到 7 秒的范围内。团队不得不寻找新的方法来创造更好的时机。 [Momoco08](http://www.robots-dreams.com/2008/11/all-japan-micromouse-robot-competition-2008-momoco8-video.html "Momoco8 (Video)") 用风扇将鼠标固定在赛道表面。嵌入下面你会发现它解决迷宫的视频加上真空裙的照片。
