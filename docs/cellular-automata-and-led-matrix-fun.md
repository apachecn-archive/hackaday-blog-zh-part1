# 细胞自动机和 LED 矩阵基金

> 原文：<https://hackaday.com/2008/10/28/cellular-automata-and-led-matrix-fun/>

![](img/b8b765fdc5747a02eaa7b822412d7553.png "cellular")

[Blinkomat，一个 LED 矩阵](http://www.burningdarkness.de/blinkomat)，结果相当不错。一个 Atmega 16 控制的 240 个 led 就是我们所说的像样的书架装饰。选择 12×20 的尺寸是因为微控制器有 32 条 I/O 线。使用[复用](http://en.wikipedia.org/wiki/Multiplexed_display)开关 led。由脉宽调制控制的亮度可以有 16 级变化。整体效果看起来相当流畅有趣。除了细胞自动机之外，他还编程让它做其他事情。我们个人最喜欢的是简单的正弦波。休息后再看。
