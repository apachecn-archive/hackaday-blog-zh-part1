# 借助外部 GPU 增强笔记本电脑的游戏性能

> 原文：<https://hackaday.com/2011/10/19/beefing-up-your-laptops-gaming-chops-with-an-external-gpu/>

如果你不愿意花钱买一台功能相当强大的笔记本电脑，似乎没有什么可以提高你的游戏性能。也就是说，除非你有一个空的 Express 卡插槽和正确的芯片组。

[Phatboy69]最近为他的笔记本组装了一个外接显卡，效果非常好。他的 Vaio Z128GG 板载了一个 Nvidia GT330M 显卡，很不错，但没什么值得大书特书的。使用 Express 卡到 PCIe 适配器，他为自己的系统添加了一个外部 Nvidia GTX580，他对结果非常满意。虽然以这种方式连接到他的笔记本电脑时，卡的性能受到了影响，但他声称他的图形性能提高了 10 倍，这还不算太差。

这个过程在很大程度上依赖于许多变量，但是通过适当的调整，可以获得一些出色的笔记本电脑游戏性能。也就是说，它确实将笔记本的便携性降低到了零。

如果这是你可能感兴趣的东西，一定要在笔记本评论论坛上查看[这个帖子——在那里[Phatboy69]找到了他需要的所有信息，以使他的系统正常运行。](http://forum.notebookreview.com/gaming-software-graphics-cards/418851-diy-egpu-experiences.html)
