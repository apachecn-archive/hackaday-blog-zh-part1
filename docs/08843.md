# 用家用化学品制造鲁米诺

> 原文:[https://hack aday . com/2011/10/17/making-luminol-from-home-chemicals/](https://hackaday.com/2011/10/17/making-luminol-from-household-chemicals/)

自己做什么化学发光材料？看看这个[用普通家居用品合成鲁米诺](http://www.youtube.com/watch?v=58Ve69s0qD0)的过程。你需要一些实验室设备，并且[NurdRage]提到了一些需要采取的预防措施，因为[鲁米诺](http://en.wikipedia.org/wiki/Luminol)本身是无毒的，但是在这个过程中发现的一些烟雾和中间化学物质是有毒的。

首先切下一些乙烯基手套，用一些外用酒精煮沸，提取邻苯二甲酸二乙基己酯。过滤后，水和一些下水道清洁剂一起煮沸。我们的目标是继续这个过程，直到得到纯的邻苯二甲酸酐。快好了吗？甚至还没开始。这是一个非常复杂的过程，但看起来很有趣。休息过后，你会看到完整的视频，或者一个五分钟的删节版，供那些只想体验一下这个实验的人观看。

几天前，当我们看着[量子点制造过程](http://hackaday.com/2011/10/12/brewing-up-some-quantum-dots/)时，我们要求更多的化学黑客。这正是我们正在谈论的，感谢[Rob]提供了提示。让他们继续前进！
