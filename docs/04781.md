# NXT 黑客:坦克和加热器

![](../Images/675259a27549b146d72681ffbd26de18.png "nxt-hacks-tank-heater")

这里有双倍剂量的乐高 NXT good；一个机器人鱼缸和一个[自动水族箱加热器](http://www.plastibots.com/?p=1429)。

左边的图像是一个由流行的乐高 Mindstorms NXT 套件驱动的机器人坦克。大脑位于一个管道内，包括控制器砖、超声波测距仪、陀螺仪和指南针。两套踏板围绕着管子的每一个边缘，让我们想知道哪一端是向上的？休息之后我们嵌入了这头野兽的视频。你会看到坦克在这种配置下非常灵活。

右边是一个水族箱加热器。[戴夫的]孩子们正在培育一些微小的水生生物，我们一直称之为[海猴](http://en.wikipedia.org/wiki/Sea-Monkeys)。问题是水箱需要 72-80 华氏度才能让小虾茁壮成长。他找出他的 NXT 控制器，将其与乐高温度传感器和一个 [dSwitch](http://dexterindustries.com/Products-dSwitch.html) 继电器配对。这种设置可以监控 72-78 度之间的温度，并通过开关灯来调节温度。这让他的孩子和死水一潭都很开心。

既然我们已经激起了你对 NXT 的兴趣，那就来看看由 wiimote 操作的 NXT Segway 和 T2 的 NXT 数独解算器吧。
