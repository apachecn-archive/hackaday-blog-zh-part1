# 海绵音乐

> 原文：<https://hackaday.com/2010/03/12/sponge-music/>

[https://www.youtube.com/embed/k4cYpf9zwkg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/k4cYpf9zwkg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Aaron]称这个项目为“随机”。它使用两块海绵作为音乐接口。表演者戴上腕带，然后用海绵中的水在桌子上画出不同的音符。你真的需要看视频才能完全理解这里发生了什么。

我们猜测这是基于 Arduino 的，因为他的一些其他项目也是如此。你可以试试他的另一个快速项目，[Arduino 电磁场探测器](http://www.aaronalai.com/emf-detector)。休息之后请看视频。
