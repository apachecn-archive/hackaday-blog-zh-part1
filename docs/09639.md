# 用 Kinect 代替幻肢

> 原文:[https://hack aday . com/2012/02/06/用 kinect 替换 phantom-limb/](https://hackaday.com/2012/02/06/replacing-a-phantom-limb-with-a-kinect/)

![](../Images/0e84b07d9130971afac0a9cf29a86ee7.png "Ogre")

几乎每个人都听说过幻肢综合症。它有时发生在一个肢体被截肢后，但病人的头脑仍然认为肢体是连着的。通常被认为是受损神经布线的混乱，幻肢会非常痛苦。[Ben]一直在努力寻找一种方法来减轻与幻肢相关的一些痛苦和沮丧，幸运的是，他选择了一款 [Kinect、VR 护目镜和陀螺仪。](http://blog.section9.co.uk/2012/02/03/Phantom-Limb.html)

今天，大多数治疗幻肢综合症的方法都使用一个 [Ramachandran 镜盒](http://en.wikipedia.org/wiki/Mirror_box)。镜子盒子背后的理论非常简单——如果有人最近失去了一只手，只需将一只手插入盒子的一侧，手臂残端放在另一侧。用完好的手从侧面看盒子会欺骗病人的大脑，让它认为被截肢的手还在那里。这是一种非常成功的好疗法，但[本]认为他可以做一些更身临其境的事情。

[Ben]的项目使用 Kinect 和 VR 护目镜将患者置于虚拟环境中。在几个陀螺仪的帮助下，患者可以在护目镜上看到自己的虚拟图像。这项技术与虚拟现实恐惧症治疗没有太大的不同，尽管[本]的构建中涉及了更多的电子和数学。第一个测试对象说他的疼痛正在减轻，所以看起来他可能会成功(没有双关语)。
