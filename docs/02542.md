# 24kJ 电容器组

> 原文:[https://hackaday.com/2008/09/20/24kj-capacitor-bank/](https://hackaday.com/2008/09/20/24kj-capacitor-bank/)

前几天贴出的[莱顿瓶电容](http://hackaday.com/2008/09/17/old-school-high-voltage-capacitance/)比不上[4HV.org 论坛](http://4hv.org/e107_plugins/forum/forum.php)成员之一的T3】fast MHzT5】一直在忙着打造一个 [24kj 电容放电库](http://fastmhz.com/?p=37)。该电容器组将配置为 4500v @ 2400uF，可使用微波炉变压器缓慢充电。然后，它可以通过触发火花隙在不到一毫秒的时间内释放所有储存的能量。这使得一些相当大的火花，正如在这个视频中看到的，我们不确定视频中的笑声，也许权力已经去了他的头？

[https://www.youtube.com/embed/UAgfGGjsoQM?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/UAgfGGjsoQM?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

像这样的银行可以用于[轨道炮](http://www.rollette.com/railgun/)，感应发射器，或一般爆炸的东西。[硬币收缩和罐头压碎](http://205.243.100.155/frames/shrinkergallery.html)需要比电解电容器真正能够做到的更快的放电速率，并且通常使用大脉冲电容器表现更好，然而看到这种势能的最终用途将是有趣的。
