# 计算数字:在室内使用太阳能电池

> 原文:[https://hack aday . com/2011/12/12/crunching-the-numbers-using-solar-cells-indoor/](https://hackaday.com/2011/12/12/crunching-the-numbers-using-solar-cells-indoors/)

![](../Images/fe26fd9349a06eeecac4e3e1751020c1.png "solar-cell-test-platform")

[马修]对在他的日常电子产品中使用更多的太阳能电池很感兴趣。他四处寻找，但没有找到太多关于将 pholovoltaic 用于小型室内设备的信息。我们记得几年前从[达夫·琼斯]的一篇关于太阳能计算器的电子博客文章中听到了一些相同的东西；我们能想到的唯一室内太阳能装置。

对知识的探索仍在继续，马修决定建造这个室内太阳能电池测试平台。这是一个稳定的硬件，允许他在许多不同的条件下运行可靠的测试。它被设计用来比较两个相同的细胞。一种是给锂电池充电，另一种是直接驱动负载。第二个电池独立于太阳能电路为平台供电，因此它可以用作数据记录器。他收集、转储和绘制来自测试运行的信息，然后讨论他的发现。我们不会破坏它，但结果不是很好。大多数情况下，你需要阳光来获得真实的效果，因为它比人工照明强大得多。
