# 树莓派推出

> 原文:[https://hackaday.com/2012/02/29/raspberry-pi-launched/](https://hackaday.com/2012/02/29/raspberry-pi-launched/)

树莓派[已经推出](http://www.raspberrypi.org/#top)。大约一年前，当我们[第一次听说](http://hackaday.com/2011/05/06/miniature-pc-packs-a-big-punch/)这种主板时，一台 25 美元的电脑可以显示 1080p 视频，运行 Linux，并有足够的外部世界端口来做一些*非常有趣的事情，我们持谨慎乐观的态度。现在，树莓派背后的家伙们正在向大众推出产品，我们有点兴奋，也在思考我们将要打造的所有酷东西。*

Raspberry Pi 基金会将与获得许可的制造机构合作，由 RS Components 和 T2 Premier Farnell 签约制造和分发电路板。现在严格来说是一个客户一个，但是在一个月左右的时间里，你就可以想订多少就订多少了。

目前，该基金会正专注于略贵的“B”型 Raspberry Pi，它在 25 美元的基本单元上包括以太网和 USB。“A”型号最初计划配备 128 MB 内存，但由于一些聪明的成本节约策略，该团队将基本内存增加到 256 MB。

现在我们唯一的问题是~~是否有美国经销商已经签约~~出售这款主板；我们相信，有了这个令人敬畏的小板子，一定会有令人敬畏的东西出现。评论者指出[纽瓦克](http://www.newark.com/raspberry-pi/raspbrry-pcba/dp/83T1943?Ntt=83T1943)T4 正在为美国市场销售 Raspis，但是他们缺货。最好的建议可能是等几个月，让产量赶上需求。
