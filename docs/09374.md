# 一种在假期后重复使用圣诞灯的方法

> 原文：<https://hackaday.com/2011/12/25/one-way-to-reuse-your-christmas-lights-post-holiday/>

![](img/51ffb13f5dd7116fcf4c6b6626573147.png "lighted-stage-prop")

[Andrew]向我们展示了一种方法，可以重新利用你今年用来装饰的圣诞彩灯。他有一个朋友在帮助一个当地音乐剧制作舞台道具，最后他用自己的技能制作了一个带动画功能的灯光标志。

最初的计划是用手剪出字母作为标志，并用白色圣诞彩灯将它们围起来。手工切割零件很好是可能的，但是安德鲁知道他可以用更少的时间用 CNC 商店机器人来制作零件，从而得到更好的结果。他不知道灯的间距，所以等待并用手为它们钻孔。每根绳子都连接到一个继电器，然后由 Arduino 驱动。正如你在休息后的视频中所看到的，他们表现得很好。

这是一个及时的破解，因为它使用的是普通的老式白炽灯泡，将在未来几天内上市。通常你能以一美元或更少的价格买到清仓货，所以提前计划，早点去大卖场。
