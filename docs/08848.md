# 为 PCB 生产改装喷墨打印机

> 原文:[https://hack aday . com/2011/10/17/modding-an-inkjet-for-PCB-production/](https://hackaday.com/2011/10/17/modding-an-inkjet-for-pcb-production/)

![](../Images/8e1f118eff81be5d9245da139fe4796e.png "Board")

像我们所有人一样，[瑞安]厌倦了等待董事会制作公司。为了减少周转时间，他将一台爱普生喷墨打印机改装成一台印刷电路板打印机。他建造的[指导](http://www.instructables.com/id/Converting-an-Inkjet-Printer-to-Print-PCBs/)非常彻底，看起来他从他的项目中得到了一些高质量的板子

建造从拆卸他放在家里的爱普生 C86 打印机开始。使用爱普生打印机很重要——爱普生有一个压电打印头，可以接受会堵塞其他打印机的墨水。撕掉打印机上的所有塑料后，[Ryan]开始升起打印机(或者放下床，随便什么)，然后出发去比赛。

墨盒里装满了抗腐蚀的黄色墨水和一片放在打印机上的覆铜板。印刷之后，[Ryan]用氯化铁蚀刻他的板子。可悲的是，在蚀刻过程中，一点墨水被腐蚀掉的地方，他的痕迹上出现了小针孔。他试过盐酸和 T2 过氧化物，但是这些会把他的木板变成绿色垃圾。

如果你有任何帮助[Ryan]的建议，请在评论中留下。在此之前，请查看[Ryan]发布的打印演示。
