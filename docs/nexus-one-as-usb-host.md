# Nexus One 作为 USB 主机

> 原文：<https://hackaday.com/2010/06/08/nexus-one-as-usb-host/>

[https://www.youtube.com/embed/3-bLOc1qnMM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/3-bLOc1qnMM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Sven Killig]已经设法让他的[Nexus One 进入 USB 主机模式](http://sven.killig.de/android/N1/2.2/usb_host/)。这使他能够插入各种外围设备，如网络摄像头、键盘，甚至是 displaylink 设备。这太棒了，因为它真正打开了这种设备的可能性。你可以看到他现在有一个惊人的功能和便携式系统在运行。他提到它非常稳定，除非他运行大型程序，我们想知道他是否[给它添加了任何内存](http://hackaday.com/2010/05/05/adding-ram-to-the-nexus-one-without-soldering/)。值得注意的是，[机器人也可以是 USB 主机](http://hackaday.com/2010/02/10/usb-host-mode-for-droid/)。
