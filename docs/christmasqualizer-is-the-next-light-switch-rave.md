# 圣诞节是下一个灯开关狂欢

> 原文：<https://hackaday.com/2011/10/15/christmasqualizer-is-the-next-light-switch-rave/>

[https://www.youtube.com/embed/la1OuczS_wM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/la1OuczS_wM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[凯尔]正在寻找一种方法来增加他无聊的砖墙宿舍的情趣。他发明的圣诞礼物可以点亮他的房间，也是举办狂欢派对的绝佳场所。

[Kyle]的 Christmasqualizer 中的灯串是现成的圣诞灯。根据本文的[内容，构建了一个简单的 7 波段均衡器电路。构建使用 MSGEQ7 均衡器芯片从任何来源的音频。七个均衡器频段的音量通过串行连接输出到 Arduino。](http://nuewire.com/info-archive/msgeq7-by-j-skoba/)

在 EQ 芯片连接到 Arduino 后，[Kyle]需要一种方法来开关圣诞灯串。几个固态继电器之后，他开始工作了。

github 上的[提供了所有关于圣诞节的代码。草图非常简单——按照文章连接 EQ 芯片，然后将继电器连接到 Arduino 上的输出引脚。这是一个有趣又简单的项目，能让宿舍充满活力。](https://github.com/kylehotchkiss/Christmasqualizer)
