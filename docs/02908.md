# Ripsaw MS1

> 原文:[https://hackaday.com/2008/12/05/ripsaw-ms1/](https://hackaday.com/2008/12/05/ripsaw-ms1/)

![ripsaw](../Images/c7f819eef098e6132528d440ff9cb831.png "ripsaw")

[Ripsaw MS1](http://www.howeandhowe.com/ripsaw_ms1.html "services") 是缅因州两兄弟制造的无人驾驶地面车辆。这种履带式车辆能在 3.5 秒内加速到 60 英里，最高时速可达 80 英里。在目前的形式下，它有 2000 磅的容量，这为许多不同类型的武器系统提供了可能性。控制由两个人提供:一个司机和一个炮手。他们在独立的远程工作站工作。这种锯子有可能用在通常只用于坦克的任何应用中。它可以在不危及士兵安全的情况下发起冲锋。

自 2005 年以来，我们一直在看着这个项目走向成熟，当时它被作为一个大挑战竞争对手进行营销。本周，它将在军事科学会议上演示。看看下面的运动镜头。
