# 派对滚动条 LED 显示屏

> 原文:[https://hack aday . com/2011/09/05/the-party scroller-led-display/](https://hackaday.com/2011/09/05/the-partyscroller-led-display/)

[![](../Images/0b92a9adc3f115e04b1925e5c7806a96.png "livingRoom_cropped")T2】](http://hackaday.com/2011/09/05/the-partyscroller-led-display/livingroom_cropped/)

正如[Plasma2002]所说，“音乐会上那些显示你的短信的巨型屏幕会很有趣。如果你能在自己的聚会或社交聚会上拥有同样的东西，那岂不是很棒？”这个问题的答案来自于 [this hack](http://www.plasma2002.com/partyscroller/) 的形式，这是一个滚动的字幕标志，客人只需发短信就能让它显示信息。显然，派对上的客人喜欢这个设备，谁不喜欢呢？

对[HAD]来说，更重要的是[Plasma2002]决定向所有人展示它是如何做到的。他通过一个精美的框图向我们概述了这个过程，然后将所有内容分解成实际使用的代码。谷歌语音账户被用作拨入号码，一切都是匿名的。一个“坏词”过滤器被用来保持一切都是半恰当的。

这真的是一个很酷的设备，我们希望它能出现在各地的黑客聚会上。谁知道呢，也许像这样的东西也可以享受主流的成功。休息之后，请观看视频，了解这款设备的运行情况！
