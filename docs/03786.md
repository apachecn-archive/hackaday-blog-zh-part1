# 压敏键盘

> 原文：<https://hackaday.com/2009/08/09/pressure-sensitive-keyboard/>

[https://www.youtube.com/embed/PDI8eYIASf0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/PDI8eYIASf0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

微软硬件部门的应用科学小组开发了一个有趣的工具。这是一个[压力感应键盘](http://procrastineering.blogspot.com/2009/08/pressure-sensitive-keyboard.html)。我们的思想被游戏束缚住了，显然他们也是如此。他们展示了如何根据你按下按钮的力度来给你更多的游戏控制权。还记得你第一次使用模拟手柄的时候，你肯定很难再回到 D-pad。我们想知道我们什么时候能玩这些由微软制造的[酷玩具](http://hackaday.com/2008/07/31/unmouse-cheap-multitouch-prototype/)。
