# DIY 超声波塑料焊接

> 原文：<https://hackaday.com/2012/01/20/diy-ultrasonic-plastic-welding/>

这可能是所有 reprappers、真空成型机和其他塑料制造商感兴趣的东西:塑料的[超声波焊接](http://circuitguru.com/117/diy-ultrasonic-welder-press-for-plastics/)。如果你曾经想把两块塑料连接在一起，而不是用丙酮把它们熔化在一起或者把它们拧在一起，[circuitguru]就是你要找的人。

超声波焊接机通常是为那些不介意花费数万美元购买一台设备的公司准备的。还有为[热熔](http://www.youtube.com/watch?v=wbAFxhesLkk)制作的较小版本——将塑料柱熔化成工件上的铆钉——而【circuitguru】足够幸运，价格还算合理。

因为热熔枪是手持设备，所以使用了旋转工具钻床。最终的结果是一种相对便宜的方式来连接两个塑料部件，无需螺钉、胶水或溶剂。这种纽带也非常牢固。休息后看看视频，看看[circuitguru]将两块塑料外壳连接起来，并试图将它们撕开。
