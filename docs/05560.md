# 盲人音频调谐器

> 原文:[https://hack aday . com/2010/06/27/audible-tuner for the blind/](https://hackaday.com/2010/06/27/audible-tuner-for-the-blind/)

![](../Images/d1ee429159fdcc1e8da2fc55d3bea8fa.png "audible-tuner")

[Lain Sharp]改装了这个[吉他调音器，这样他的盲人朋友](http://www.lushprojects.com/guitartuner/)就可以使用它了。在上图中，你可以看到一个白色的小盒子，里面装着额外的电子设备。内部是另一个电池和一个 ATmega168，用于连接耳机。AVR 芯片连接到调谐器上的每个 LED，并将视觉调谐计转换为音频提示。休息之后，请观看演示剪辑，了解它是如何工作的。

现在，如果我们能想出如何让我们的[琴弦与我们内置的键盘](http://hackaday.com/2009/11/19/when-an-axe-isnt-enough/)合拍。
