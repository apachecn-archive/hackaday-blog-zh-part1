# 游戏男孩脚控制器演示

> 原文：<https://hackaday.com/2008/06/15/game-boy-foot-controller-demo/>

[https://www.youtube.com/embed/lQdqudTzyBs?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/lQdqudTzyBs?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
这是《动物造型》中【乔伊·马里亚诺】的[视频](http://www.youtube.com/watch?v=lQdqudTzyBs)演示他的[游戏男孩踏板](http://filefreakout.com/hardware.html#joey_mariano)。他在 Game Boy 的背面增加了一个 D15 端口，分别与 8 个按钮相连。该端口连接到 8 个钢琴延音踏板的转接盒。盒子也为游戏机提供电力。游戏男孩正在运行 [Nanoloop](http://www.nanoloop.de/) 步进序列器。如果你想知道吉他的声音，他可能正在使用一个 [8 位模糊踏板](http://fxdoctor.com/8bit.html)。
