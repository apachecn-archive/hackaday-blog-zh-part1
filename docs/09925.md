# 同构的钢琴键盘是按钮疯狂

> 原文:[https://hack aday . com/2012/03/19/同构-钢琴-键盘-按键-疯狂/](https://hackaday.com/2012/03/19/isomorphic-piano-keyboard-is-button-madness/)

![](../Images/ce4faf67bd45bea847fc5ac0ab974ef4.png "keyboard")

[nikescar]发送了一个链接，链接到一个巨大的同构 MIDI 键盘。我们可能第一次就错过了，但这并不影响这样一个伟大的项目。

根据[项目页面](http://soundbuilders.lvl1.org/discussion/20/project-ax15-64-the-axis-64-clone/p1)，这个巨大的键盘是路易斯维尔 Soundbuilders 的工作人员努力克隆的 [AXiS-64 MIDI 控制器](http://www.c-thru-music.com/cgi/?page=prod_axis-64)。这种同构的键盘不是看起来像一个“普通”的钢琴键盘，而是以六边形的模式放置音符。这种键盘布局非常有用——和弦的指法在所有键上都是一样的——但这种键盘布局相当罕见，因此 MIDI 控制器很贵。

为了使视频中的键盘[对速度敏感，有两层 PCB。顶层使用樱桃按键开关，而底部 PCB 是一组轻触开关。测量顶部和底部按键之间的时间，可为板载微控制器提供力度信息，并将其转换为 MIDI 音符。这种设置有一些缺点，即两个键盘矩阵需要大量的开关、组件和引脚。](http://www.youtube.com/watch?v=KfjEYu79J-4)

项目页面已经几个月没有更新了，所以我们很好奇这个版本的当前状态。如果路易斯维尔 Soundbuilders 有关于这个项目的更新，[在](http://hackaday.com/contact-hack-a-day/)发送它。
