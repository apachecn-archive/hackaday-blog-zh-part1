# 构建线路图

> 原文：<https://hackaday.com/2009/02/09/build-a-wiremap/>

![wiremap](img/60c02bf1741e9d5051a4431420cc0322.png "wiremap")

对于那些不记得的人来说,[线路图是一个 3d 立体显示图](http://hackaday.com/2008/07/25/wiremap-a-volumetric-display/),由数百条微小的线构成。图像被投射到绳上以实现 3d 效果。[phedhex]已经发布了一个关于[如何建立线路图](http://www.instructables.com/id/How_to_Build_a_Wiremap/)的深入指导。他用许多好的技巧介绍了这个构建，并给出了 Windows、Mac 和 Linux 软件的链接。休息后看看视频，看看一些相当令人印象深刻的 3d 模式。
