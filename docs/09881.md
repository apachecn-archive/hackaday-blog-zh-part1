# 通过蓝牙从 Android 控制 Arduino

> 原文:[https://hack aday . com/2012/03/13/control-an-arduino-from-Android-over-bluetooth/](https://hackaday.com/2012/03/13/control-an-arduino-from-android-over-bluetooth/)

无论你是想做一些实时数据记录，还是只想用你的 Android 手机控制一个项目，[托马斯]的 [Arduino-Android 蓝牙连接](http://www.instructables.com/id/Androino-Talk-with-an-Arduino-from-your-Android-d/) instructable 一定会很有用

[Thomas]“构建使用非常便宜的 JY-MCU 蓝牙模块，可在易贝或 [dealextreme](http://www.dealextreme.com/p/jy-mcu-arduino-bluetooth-wireless-serial-port-module-104299) 上获得。该蓝牙模块直接连接到 Arduino 的 Tx 和 Rx 线路，因此可以在 Android 设备之间建立无线串行连接。在构建的 Android 方面，Android 的[Python](http://code.google.com/p/python-for-android/)和 Android 的[脚本层](http://code.google.com/p/android-scripting/)允许通过蓝牙读取无线传感器数据。

虽然使用[IOIO](http://hackaday.com/2011/04/07/a-breakout-board-for-your-android-phone/)或 [Android 开放附件](http://hackaday.com/2011/05/11/official-android-open-accessory-development-kit-its-an-arduino/)开发套件也可以将 Android 设备连接到 Arduino，但我们还没有看到太多(除了[这个](http://hackaday.com/2012/01/19/controlling-a-cute-ikea-night-light-with-android-on-the-cheap/))通过蓝牙用 Android 控制或读取简单电子设备的情况。有时候，你不需要一个很棒的开发板来构建一个简单的项目，所以我们希望[Thomas]非常好的指令性将有助于获得更多的构建。
