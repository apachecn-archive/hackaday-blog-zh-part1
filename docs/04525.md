# WiiMote + IPhone 更新

> 原文:[https://hackaday.com/2009/12/10/wiimote-iphone-update/](https://hackaday.com/2009/12/10/wiimote-iphone-update/)

![](../Images/0f10de98df39f0c01375b2fa04d4fdfa.png "wiimote_iphone_mame")

用一个 [Wii 遥控器作为 iPhone](http://www.zodttd.com/wp/2009/11/wiimote-demonstration-video-with-mame4iphone/) 游戏的控制器？太棒了。自从八月我们在[报道它以来，我们一直在等待看到这种配对的一些创造性用法。是](http://hackaday.com/2009/08/05/wiimote-iphone/)[把 MAME](http://www.zodttd.com/wiki/index.php/Mame4iphone) 移植到 iPhone 上的人。现在，他通过 [BTstack 项目](http://code.google.com/p/btstack/)在 Mame4iPhone 中添加了对 Wii remote 的支持。BTstack 试图为没有蓝牙堆栈功能或功能有限的设备(iPhone)添加蓝牙堆栈功能。

我们在用于玩 MAME 游戏的 WiiMote 中断后嵌入了视频。[ZodTTD]也是 nes4iPhone 的作者，我们希望这是他将添加 BTstack 支持的下一个项目！
