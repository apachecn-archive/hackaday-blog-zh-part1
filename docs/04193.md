# 地铁从来没有听起来这么好

> 原文:[https://hack aday . com/2009/10/07/the-subway-never-sound-so-good/](https://hackaday.com/2009/10/07/the-subway-never-sounded-so-good/)

[https://www.youtube.com/embed/ivg56TX9kWI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ivg56TX9kWI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

读者[Philippe]向我们透露了这个视频，视频显示一组[地铁台阶被变成了钢琴键盘](http://www.rolighetsteorin.se/?q=expriment/pianotrappan) ( [英文翻译](http://translate.google.com/translate?hl=en&sl=sv&tl=en&u=http%3A%2F%2Fwww.rolighetsteorin.se%2F%3Fq%3Dexpriment%2Fpianotrappan))。创造者想让走楼梯比坐自动扶梯更有趣。他们在每一步都添加了压力传感器，然后在上面覆盖上黑白键，看起来像键盘。当乘客将体重放在踏板上时，相应的音高就会被大声播放出来。

我们可能在标题上撒了谎，因为这与在你的火车站听乔希·贝尔为零钱而演奏的根本无法相比。但是在平凡的通勤中遇到这样的场景真的会让我们的一天变得明亮起来。
