# 惊人的机器人灵巧度

> 原文：<https://hackaday.com/2009/07/30/amazing-robot-dexterity/>

 <https://www.youtube.com/embed/bfdHY26E2jc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>石川小室实验室正在用机器人做一些令人惊奇的事情。上面的视频开始看起来像一个笨拙的，有点失败的运球尝试。然而，一旦它进入慢动作，我们就看到了真正的动作。这个机器人正在惊人地运球。利用 1000 FPS 摄像机，它可以在每次反弹时重新调整和击球。当球漂到机器人够不到的地方时，我们突然被这栩栩如生的动作震惊了。<a href="http://en.wikipedia.org/wiki/Personification" target="_blank">拟人化</a>可能是一个转瞬即逝的东西，当小机器人徒劳地试图去拿球时，它会如此强烈地出现，然后一会儿又消失了。如果你真的想看一些配得上<a href="http://hackaday.com/2008/09/18/crabfu-challenge-winners-announced/">蟹伏挑战</a>的拟人化表演，可以看看他们的<a href="http://www.k2.t.u-tokyo.ac.jp/fusion/tool_manipulation/index-e.html" target="_blank">多指手视频</a>中的工具操作。</p> <p>【通<a href="http://www.botjunkie.com/2009/07/30/high-speed-robot-hands-easily-outpace-humans/" target="_blank">骨瘤</a></p> </body> </html>
