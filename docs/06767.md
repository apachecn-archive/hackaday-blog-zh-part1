# 将您的计算机功率降至 8.5 瓦

> 原文:[https://hackaday . com/2011/02/08/将你的电脑研磨到仅仅 8-5 瓦特/](https://hackaday.com/2011/02/08/grinding-down-your-computer-to-just-8-5-watts/)

![](../Images/791ba0073aa6215a4bbf14a0d1b03d00.png "low-wattage-computer")

您能做些什么来确保您的系统尽可能高效地运行？从[Mux]的书里拿一页出来，他花了[大力气来测量和调整他的系统以获得最高效率](http://ssj3gohan.tweakblogs.net/blog/6097/85w-core-i3-systeem-het-bewijs!.html) ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://ssj3gohan.tweakblogs.net/blog/6097/85w-core-i3-systeem-het-bewijs!.html))。他最终得到的是空闲时 8.5 瓦的功耗和负载时约 50 瓦的功耗。幸运的是，他发布了一个包含所有细节的六部分系列。

他所做的一些改变是在软件方面，比如通过调整 BIOS 设置来降低某些硬件的电压，以及安装将屏幕置于适当睡眠模式的显示驱动程序。其他的是硬件的改变，比如用一个被黑掉的 PicoPSU 替换掉电源，从主板上去掉不必要的部分，比如 MAX232 com-port 芯片。看起来我们需要检查一下我们永远在线的 MythTV 盒子，看看我们是否可以应用这些节能技术。
