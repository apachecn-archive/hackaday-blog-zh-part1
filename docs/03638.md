# 奥托，节拍切片接口

> 原文：<https://hackaday.com/2009/06/29/otto-beat-slicing-interface/>

![otto](img/b7b89dac92a5fd29187dcc3f47c65c20.png "otto")

“创造数字音乐”在[【卢卡·德·罗索】的奥托](http://www.lucaderosso.com/otto/otto "otto «  OTTO")上有一个很棒的帖子。作为他硕士论文的一部分，这是一个独特的有形音乐界面。将样品加载到软件中，软件会将样品显示在仪器表面。然后，用户可以使用各种硬件输入操作样品，同时观察 LED 显示。该设备仅使用一个 Arduino 进行显示和输入。它与 Max/MSP 配合使用，旨在仅向表演者提供他们真正需要的信息。你可以在 Flickr 上找到更多[设备的图片，在 CDM](http://www.flickr.com/photos/luderec/sets/72157619927348386/ "OTTO - a set on Flickr") 上找到一张[内脏的图片。下面嵌入的是“入门”视频，展示了它的使用。](http://createdigitalmusic.com/2009/06/29/otto-beautiful-original-hardware-for-beat-slicing-in-circles/ "Create Digital Music » OTTO: Beautiful, Original Hardware for Beat Slicing in Circles")
