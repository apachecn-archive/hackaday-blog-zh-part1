# 真实生活超级马里奥硬币块

> 原文:[https://hack aday . com/2011/03/23/real-life-super-Mario-coin-block/](https://hackaday.com/2011/03/23/real-life-super-mario-coin-block/)

![mario_coin_block](../Images/3618d2e011e5e8c5842c29134fb394f1.png "mario_coin_block")

Instructables 用户[Bruno]最近制作了一个有趣的小玩具，将一点马里奥怀旧情绪从视频游戏世界带到了我们的世界。他的超级马里奥硬币块从第一款马里奥游戏中就能立即辨认出来，并且表现出你所期望的那样。击打或轻敲方块底部，一次释放一枚硬币，并伴有直接来自游戏的声音。

硬币块由厚纸板制成，包裹在游戏块的彩色模型中。在里面，一个装有弹簧的硬币管放在发射臂上方，发射臂也与弹簧相连。一个伺服驱动臂拉下发射臂，将一枚硬币从它的管中落到发射臂上，一旦伺服臂旋转到足够远，发射臂就会从盒子的顶部抛出。当这种情况发生时，内置的 MP3 播放器被触发播放游戏中的“硬币声音”。555 定时器用于确保每次激活时伺服致动臂旋转一周，基于 LM386 的放大器用于增加 MP3 播放器的输出音量，两者都使用可充电电池操作。

请务必查看一些内部工作以及下面嵌入的视频中的最终产品。
