# 使用水的多点触摸

> 原文：<https://hackaday.com/2010/06/26/multitouch-using-water/>

[https://www.youtube.com/embed/cuihQVRpRrQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/cuihQVRpRrQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[井上太极]又回来了，这次是带着一个使用水作为触摸表面的多点触摸系统。该装置包括一个放置在 LCD 顶上的水箱、一盏灯和一个网络摄像头。当有东西打破水面时，网络摄像头会捕捉到反射的光线。就计算机而言，它与我们在许多[自制的多点触摸系统](http://hackaday.com/category/multitouch-hacks/)上看到的斑点识别没有什么不同。与[的雪风号驱逐舰](http://hackaday.com/2010/06/25/yukikaze-music-visualizations/)混合在一起，这个家伙可能会拥有世界上最轻松的计算机系统。
