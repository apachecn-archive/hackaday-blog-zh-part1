# 玻璃回收游戏

> 原文:[https://hackaday.com/2009/10/17/the-glass-recycling-game/](https://hackaday.com/2009/10/17/the-glass-recycling-game/)

![bottle-bank-arcade](../Images/5fb57f6dc5958b86011f8c3b95076554.png "bottle-bank-arcade")

在玻璃瓶不像塑料或铝那样被回收利用的前提下，因为这种努力没有报酬，Fun Theory 的团队开始增加一个非金钱激励。这是他们努力的成果，瓶子银行拱廊。现在回收玻璃瓶有点像游戏。

回收箱上的一个孔上方会有一盏灯闪烁，如果瓶子插入得足够快，就会得到奖励。西蒙说，这种类型的概念在休息后嵌入的视频中非常受欢迎。我们会想象视频中的一些孩子让他们的父母带着一些空瓶子把他们拖回家，这样他们就可以玩了。

一个晚上，这个改造过的瓶子库被使用了 100 多次，而附近一个低技术含量的仓库只被使用了两次。这是一个很好的方式来使用一些技术知识，对黑客的热爱，以及传播快乐的愿望，以使社区变得更有趣，并帮助增加最终被回收的垃圾量。这是同一个团队制作了[音乐地铁舞步](http://hackaday.com/2009/10/07/the-subway-never-sounded-so-good/)，我们希望他们保持这种趋势！
