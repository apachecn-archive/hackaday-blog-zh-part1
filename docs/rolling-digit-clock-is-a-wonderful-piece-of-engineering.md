# 滚动数字钟是一项了不起的工程

> 原文：<https://hackaday.com/2012/03/01/rolling-digit-clock-is-a-wonderful-piece-of-engineering/>

![](img/f4efc713f810b4fc5d2bc24beae07918.png "DIGIT")

很久以前，在一次他几乎不记得的乞讨旅行中，[Victor]从一个旧的荷兰铁路时钟上买了一个四位数字。这些古董显示器使用一条涂有塑料的布，在马达的帮助下绕着自己滚动，以显示数字 0 到 9。已经过去很多年了，但是【Victor】终于找到时间[用这些个位数的显示器建造了一个时钟](http://www.hackvandedam.nl/blog/?p=171#)，我们很喜欢这个结果。

因为这些显示器是在机械设备为王的时代制造的，[Victor]必须稍微修改每个数字，以便它们可以在连续旋转伺服的帮助下显示数字。四个伺服系统由 Arduino 控制——每个数字一次改变一个以减少电流消耗——每个数字都添加了一个磁铁和簧片开关，以便数字可以重复显示。

在[维克多]用金属齿轮取代塑料伺服齿轮之前，这个时钟噪音很大。从那以后，他把每个数字放在一个钟罩(实际上是一个颠倒的花瓶)下面，我们不得不说[Victor]手上有一个漂亮的时钟。检查视频的时钟改变数字显示休息后的时间。
