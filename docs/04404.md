# 基于枫树种子空气动力学的单翼飞行

> 原文：<https://hackaday.com/2009/11/11/single-wing-flight-based-on-maple-seed-aerodynamics/>

![one-winged-flight](img/5184ba4bacbe7258cfaf86ea42f25c38.png "one-winged-flight")

萨马拉微型飞行器是马里兰大学航空航天工程自主飞行器实验室三年多工作的成果。萨马拉是 [DARPA 纳米飞行器项目](http://www.darpa.mil/dso/thrusts/materials/multfunmat/nav/index.htm)的申请人。与我们在 7 月看到的[扑翼飞机不同，这种飞行器只用一只翅膀飞行。安装在垂直于机翼的杆上的小螺旋桨提供旋转。机翼的俯仰改变为爬升、下降或悬停。](http://hackaday.com/2009/07/01/first-hovering-ornithopter-nav/)

休息之后你可以看到飞行测试的视频。萨马拉发出的声音让我们想起了经典的外星人入侵电影，在飞行测试中使用威尔第的安魂曲作为背景音乐(2:43)似乎非常合适。大约 5 点 45 分，有一些机上视频片段，只是房间旋转的模糊画面。如果每秒钟有几帧是在车辆旋转的同一点上拍摄的，这将更加有用。
