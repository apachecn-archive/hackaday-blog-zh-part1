# 让蝾螈保持健康和凉爽

> 原文：<https://hackaday.com/2011/12/21/keeping-axolotl-healthy-and-cool/>

[![](img/f8e19302ab8c25501574d4ebf28934dd.png "axolotl")](http://hackaday.com/wp-content/uploads/2011/12/axolotl.jpg)

上面看到的真实生活中的口袋妖怪是一只蝾螈，一种类似蝾螈的动物，只生活在墨西哥城附近的一个湖里。这些可爱的动物可以人工饲养，但是饲养它们是一个挑战。[LRVICK]决定他不想花几百美元买一个水族箱冷却器，所以他[用通常用来保持电脑良好低温的零件做了他自己的](http://familab.org/blog/2011/12/thermo-electric-axolotl-cooling/)。

符合要求的商用水族箱冷却器起价约为 300 美元，并会继续上涨。不想花那么多钱，[LRVICK]花 5 美元找到了一个 77 瓦的 Peltier 冷却器，并认为他可以让它工作。水冷 CPU 的现成部件用于构建水族箱冷却器——冷侧是一个水块，热侧是一个巨大的散热器和风扇，一束管道通向水箱。

现在[LRVICK]有一只美西螈生活在一个看起来非常专业的水族箱里，水温稳定在 65 度。他身材很好，美西螈看起来很开心。
