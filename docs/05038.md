# 翻盖便携式游戏魔方

> 原文:[https://hackaday.com/2010/03/14/flip-top-gamecube-portable/](https://hackaday.com/2010/03/14/flip-top-gamecube-portable/)

![](../Images/c45679250a142ecabd77c2d2932a03a6.png "flip-top-gamecube-portable")

这是另一个便携式家用游戏机。[Techknott]制造了这个[闪亮的 GameCube 掌机](http://www.youtube.com/watch?v=UnPzaUNtEPg)。你可能还记得他在[便携式 Dreamcast](http://hackaday.com/2009/12/16/techknotts-portable-dreamcast/) 和[无线 Xbox 360 接口](http://hackaday.com/2008/09/13/xbox-360-wireless-visual-interface/)上的工作。这一次他完成了镜像；概念上的一个好主意，但即使是他的演示图像也已经被污迹困扰。但是如果你能把手指放在塑料按钮上，这将是一个引人注目的设计。我们喜欢的一个部分是隐藏光驱的翻盖屏幕。这是一个比我们在[【hail Razer 的】GC 便携](http://hackaday.com/2009/11/08/gamecube-to-go/)上看到的曝光镜头好得多的解决方案。一如既往，休息后的视频。
