# NES On ZipIt

> 原文：<https://hackaday.com/2009/09/03/nes-on-zipit/>

[https://www.youtube.com/embed/uwxGQhtf4uU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/uwxGQhtf4uU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
[甚至比第四代还要早](http://hackaday.com/2009/09/02/neo-geo-case-mod/)，【凯文】向[透露了这个项目](http://hunterdavis.com/archives/123)由【亨特·戴维斯】。这是为 Zipit 的 Z2 即时信息设备收集的软件模块。其中一些包括允许它运行(移植)Fluxbox 窗口管理器、FCE 超 NES 模拟器和 Java 框架。开发的下一个飞跃，一个为 Zipit 优化的 SNES 模拟器，正在进行中。Z2 开箱即可运行 Linux。这意味着你不再需要刷新设备来添加软件，比如 DOSbox(也是由 Hunter Davis 引入的)。请务必查看上面 Z2 上的 FCE 超视频演示。
