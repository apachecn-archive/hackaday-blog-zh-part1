# 灯升级让你头脑发热

> 原文:[https://hack aday . com/2012/01/30/lamp-upgrade-make-you-a-hot-head/](https://hackaday.com/2012/01/30/lamp-upgrade-makes-you-a-hot-head/)

![](../Images/9274ff05052d8a25c6fc2af7b8257186.png "head-lamp-upgrade")

[卡梅伦]决定给他的 20 岁的头灯改头换面。他在外出跑步时使用它，并希望有更多的光线来看他要去的地方，以及背面的红色尾灯。股票设计在头带的前面使用白炽灯泡，在后面使用电池组。他设法[将该设备转换为输出 700 流明](http://cameroncharles.blogspot.com/2012/01/petzl-zoom-led-conversion-final-chapter.html)，而没有对该设备的形状因子进行重大改变。

他决定的第一个变化是使用 Cree XLamp，它通过消耗大约 9.5 瓦的功率提供 700 流明的光。显然，原始电池组在这种负载下不会做得很好，所以他还采购了 5000 毫安的锂电池。一点电路设计和 PCB 布局给他提供了两个四元件 LED 模块的驱动芯片，一个电池充电电路，以及一个驱动头灯和闪烁红色 LED 尾灯的 ATtiny13。休息后看视频里的 blinky good。
