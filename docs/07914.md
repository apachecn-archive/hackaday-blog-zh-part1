# 《一万年钟》听起来像印第安纳·琼斯的电影——让我们怀疑我们是否被套牢了

> 原文:[https://hackaday . com/2011/06/23/10000 年的时钟——听起来像印第安纳·琼斯电影——让我们怀疑是否被控制/](https://hackaday.com/2011/06/23/10000-year-clock-sounds-like-an-indiana-jones-flick-makes-us-wonder-if-were-being-trolled/)

![](../Images/20050b342eee5825d1a35c412c075a71.png "10000-year-clock")

因此，当你听说有人正在建造一座可以运行一万年的钟时，你会想‘哦，太棒了’。然后你开始观察它，意识到它是在美国偏远地区以山的规模建造的，事情开始变得有点奇怪。尽管这听起来像科幻小说(或一些创造性的套索)，Long Now 基金会正在建造和安装一个在未来一万年内每年报时一次的时钟。

这座目前正在建造中的钟将有 200 多英尺高，位于德克萨斯州西部石灰岩山上的一个竖井中。当我们读到竖井将从上往下钻，然后安装一个带有机械臂的竖井，在石墙中磨出一个螺旋楼梯时，脑海中浮现出[印第安·琼斯]的典故。这不是唯一一个计划中的时钟。内华达州的第二个站点已经被购买。

有很多有趣的功能，其中最重要的是一个“编钟引擎”，每年都会播放一首独特的曲子，永远不会重复。[Alex]向我们发送了一篇连线文章的原始提示，这篇文章以令人难以置信的细节报道了项目。但我们也发现了一个长达一小时的 SETI 对话视频。休息之后你会发现这一点。
