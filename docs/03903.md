# XR-噪音盒

> 原文:[https://hackaday.com/2009/08/28/xr-noise-box/](https://hackaday.com/2009/08/28/xr-noise-box/)

[https://www.youtube.com/embed/3l5DOGn_rFw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/3l5DOGn_rFw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[denha]用一个 [XR-2206](http://www.exar.com/Common/Content/ProductDetails.aspx?ID=XR2206) 多波形[函数发生器](http://en.wikipedia.org/wiki/Function_generator)组装了一个他称之为 [XR-NOISE](http://www.youtube.com/watch?v=3l5DOGn_rFw) 的噪音盒。输出具有令人印象深刻的可控设置数量，并使用一组 led 来指示音量和速率。XR-NOISE 使用 1/4”插孔用于输入和输出，也可以通过位于盒子前面的 tap-sensitive mic 进行控制。不幸的是，似乎没有任何进一步的文档或示意图来提供背景，但似乎[这个函数发生器芯片](http://www.mouser.com/search/refine.aspx?N=4294014055&Keyword=2206)也已经用于其他音频黑客项目，包括使用电阻压力传感器的 [scratch-synth](http://vimeo.com/1462491) 。
