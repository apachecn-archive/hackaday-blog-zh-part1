# 3D 打印窥镜

> 原文:[https://hackaday.com/2012/01/11/3d-printed-zoetrope/](https://hackaday.com/2012/01/11/3d-printed-zoetrope/)

正常情况下，29 个男人走在一个无止境的圆圈里会是[M·C·埃舍尔]版画的素材。在一些发光二极管和 3D 打印机的帮助下，Tobias 将这一切变成了现实。

像他早期的项目一样，[托拜厄斯]为自己建造了一个漂亮的小型频闪观测镜，通过精确的间隔闪烁 led 来保持运动的错觉。代替平面的 2D 图像，[托拜厄斯]去寻找一个行走的 3D 人物，随着计时器电路的节拍前进。这些图形本身是通过 Shapeways 打印出来的。

电子设备在这次迭代中得到了改进。以前，[托拜厄斯]用的是 555 和一大堆辅助部件。这个版本的电路被改进为使用施密特触发器和一个光学编码器。易于在 perfboard 上构建的原理图和布局图[可用](http://drehkino.de/platine_v2_en.php)，因此您可以随意为自己构建一个。

[托拜厄斯]“佐伊特罗普”和去年在《燃烧的人》中看到的[巨大的卡戎雕塑](http://hackaday.com/2011/09/16/burning-man-2011-peter-hudsons-charon-strobe-sculpture/)没什么不同。当然，它没有 40 英尺高，但仍然是一件不错的作品。
