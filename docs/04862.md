# 打印你自己的比萨饼

> 原文：<https://hackaday.com/2010/02/03/print-your-own-pizza/>

[https://www.youtube.com/embed/XJ7gD0nH-xA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/XJ7gD0nH-xA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

如果你认为在你的生活中电脑数控永远不够用，也许披萨绘图机应该是你的下一个项目。这是一个大型的双轴机器，将加压的调味汁喷到比萨饼皮上。这是一个食品级的 [RepStrap](http://hackaday.com/2008/10/19/lasercut-reprap-kit/) ，似乎使用了一个[花园喷雾器](http://www.kmart.com/shc/s/p_10151_10104_043W731495690001P?vName=Lawn%20&%20Garden=&cName=OutdoorTools&Supplies=&sName=Sprayers%20&%20Spreaders=&psid=FROOGLE&aff=Y&sid=KAx20061023x272xTier3) r 作为蓄水池。当一个松动的软管夹把酱汁喷到房间里时，他们得到了教训。我们认为这有点像披萨制造工厂的重新发明，但还是很有趣。
