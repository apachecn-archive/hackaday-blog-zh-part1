# 虚拟特雷门琴

> 原文:[https://hackaday.com/2009/12/05/virtual-theremin/](https://hackaday.com/2009/12/05/virtual-theremin/)

[https://www.youtube.com/embed/eo5IRWvznWc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/eo5IRWvznWc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Mojo]通过创建一个虚拟的特雷门琴去掉了许多复杂的电路。一个[特雷门琴](http://en.wikipedia.org/wiki/Theremin)是一种电子乐器，通常有两个天线，可以感知玩家的手与乐器的接近程度，并做出相应的反应。

这种设计被称为 AirDeck，它使用 Wii 遥控器作为红外传感器，并使用两只装有红外 led 的手套。来自 Wii 遥控器的数据由一个定制的 Java 应用程序处理，然后转换成声音。[Mojo 的]界面还包括一些[替代选项](http://hackaday.com/2009/06/15/theremin-controlled-mario/)，比如转台 scratch 界面。

最后，这仍然只是一个合成器/midi 控制器，不能经得起真正的东西。然而，如果你不是一个有成就的玩家，你可能永远不会注意到不同。
