# 用知识增强力量

> 原文:[https://hackaday.com/2011/08/15/power-up-with-knowledge/](https://hackaday.com/2011/08/15/power-up-with-knowledge/)

LM317 是许多想要快速、廉价、可靠和可调电源的人的最爱。它只需要几个部件就能安装好，而且它的工作做得相当好。有时候，你只需要一个电源。虽然有上百万个教程，但没有几个像[Phil]在他的两个最新视频中的[那样深入。](http://jumperone.com/)

涵盖从墙上插座到最终输出的所有内容，[Phil]一步一步地解释每个部分，说明它的作用以及产生高质量结果背后的数学和公式。然后，他走到一个工作模型前，查看每个部件，在示波器上显示其真实输出，如果你自己没有示波器，这非常方便。

在第二个视频中，他将这些知识构建成一个专业的台式模型，带有 LCD 仪表读数和清漆纸，以完成正面外观。如果你想制作你的第一个长凳，或者想更好地了解你现在拥有的长凳上到底发生了什么，你应该在这两个高质量作品的休息后加入我们。
