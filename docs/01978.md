# 大功率 LED 闪烁电路

 <https://www.youtube.com/embed/mNDu56oGNck?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <br/>邪恶疯狂的科学家 Labs 为我们带来了这款<a href="http://www.evilmadscientist.com/article.php/ledblinker" target="_blank">轻松制作 LED 闪烁电路</a>。这个想法是把一个 LED 和一串圣诞灯上一个闪烁的小白炽灯泡串联起来。灯泡有一个内部双金属片，当它变热时会弯曲变形，切断电路。当它足够冷却时，它会恢复到原始形状，并再次闭合电路，使灯泡和 LED 打开。由于双金属片仍在预热，两个灯在最初通电时都有短时间的持续发光。</p> <p>该项目使用 5W 蓝色 LED、前述灯泡和装有 3 节 AAA 电池的 6V 电池组。电池组和灯都固定在一小块穿孔板上。复制这个项目应该很容易，并提供一个非常亮的 LED，但要让一个 5W 的 LED 发出最亮的光，一个更大的灯泡和散热器将是必要的。</p> </body> </html>
