# 建筑起重机之舞

> 原文：<https://hackaday.com/2009/10/04/dance-of-the-construction-cranes/>

[https://www.youtube.com/embed/gP_RhEstx_c?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/gP_RhEstx_c?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

黑客就是以一种非常规的方式使用零件和资源。如果把一对[高耸的起重机](http://en.wikipedia.org/wiki/Crane_(machine)#Fixed)和 [LED 圣诞灯](http://hackaday.com/2009/01/23/choreographed-christmas-lights/)串在一起，让它们随着一段古典音乐起舞，这不是标新立异，那么我们不知道什么是标新立异。昨晚几乎每个小时都有一刻钟的时间，两名起重机操作员和两名“舞蹈指导”以协调的方式摆动和旋转起重机。该项目由[[Brandon Vickerd](http://www.brandonvickerd.com/)创建，作为 Nuit Blanche 的 T10 部分。它的目的是展示建造我们生活的城市的技术。一个来自不同视角的实时视频流被投射到起重机下方的现场。
