# 如何构建一个每次都正确的真空表单表

> 原文：<https://hackaday.com/2011/08/31/how-to-build-a-vacuum-form-table-that-gets-it-right-every-time/>

[James]在家里制作各种机器人和超级英雄服装复制品，所以他总是在寻找更好的方法来使用他的真空工作台获得一致的结果。很多人用他们的烤箱或电煎锅的加热线圈来加热塑料片，但[James]对走这条路并不感兴趣。他说，他宁愿不在烹饪食物的烤箱里加热塑料，也不喜欢暴露加热元件的想法。

相反，他选择了稍微贵一点，但完全合理的设置，每次都能产生一致的结果。正如你在下面的视频中看到的，大部分成型工作台都是用中密度纤维板制成的。他的加热设备是设备中最昂贵的部分，因为它是现成的石英房间加热器。他将加热器放在背面，用铝箔作为反射器，通过一个 MDF 框架将热量向上引导。安装在顶部的塑料板受热均匀，很快，他就有了一个完美的真空成型的道具，可以开始上漆了。

当然，它可能比我们之前看过的其他真空成型器要贵一点，但是如果你问我们的话，花更多的钱来获得一致的结果是值得的。
