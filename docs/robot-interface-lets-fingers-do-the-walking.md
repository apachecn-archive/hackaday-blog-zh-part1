# 机器人界面让手指行走

> 原文：<https://hackaday.com/2009/11/12/robot-interface-lets-fingers-do-the-walking/>

[https://www.youtube.com/embed/945Z2xtdEBE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/945Z2xtdEBE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们把这个放在“最佳界面实现”下面。这个[机器人由 iPod Touch 表面的手指手势](http://www.robots-dreams.com/2009/11/walky-using-natural-finger-gestures-to-control-humanoid-robots-video.html)控制。它可以根据从你汗湿的手指接收到的输入向前走、转身、回避、跳跃和踢球。不同于由 iPhone 控制的汽车[(或由动力轮](http://hackaday.com/2009/10/24/use-iphone-to-run-yourself-over/)控制的[)，这有一些潜力。尤其是考虑到](http://hackaday.com/2009/11/10/remotely-control-your-crappy-car-dangerously/)[多点触控设备](http://hackaday.com/2009/10/10/multi-touch-lcd-from-leds/)在我们日常生活中不可避免的扩散。
