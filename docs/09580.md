# 用硬盘合成

> 原文:[https://hack aday . com/2012/01/27/synthese-with-a-hard-drive/](https://hackaday.com/2012/01/27/synthesize-with-a-hard-drive/)

如果你想要一种产生嗡嗡声合成器声音的伪机械方式，[gijs]就是你要的人。他只用一个旧硬盘和一些组件做了一个小型合成器。

每当手动旋转磁盘时，驱动器内部的主轴电机会在其连接上产生一些异相正弦波。合成器将这些正弦波进行比较和放大，然后发送给扬声器。结果是一个奇怪的嗡嗡作响的 chiptune 式的琶音。

构建电路直接焊接到硬盘驱动器外壳[曼哈顿风格](http://hackaday.com/2011/09/04/a-ham-radio-receiver-manhattan-style/)。因为主轴电机的输出产生异相正弦波，[gijs]认为如果他能利用一些相位干扰来改变他的合成器的音色，这将是一个好主意。整个建筑安装在墙上，一侧有铰链，这样扬声器就可以四处移动。变化不大，但我们可以看到一些波形相互抵消。
