# 厨房黑客:Sous Vide 构建不需要花费一只胳膊和一条腿

我们并不是没有见过廉价的 Sous Vide 建筑。我们只是喜欢[Kelvin's]的[Sous Vide machine 赋予闲置物品新的生命。烹饪器皿是他花 3 美元买的一个瓦罐。他把它放在一个大的泡沫聚苯乙烯盒子里，这个盒子是他通过当地的自由循环项目免费得到的。循环泵是一个 0.99 美元的鱼缸部件，每小时推动大约 10 加仑。](http://www.flakshack.com/forum/index.php?topic=14079.0)

他甚至四处寻找控制电路的最佳价格。PID 控制器显然是最重要的部分，因为它将调节烹饪温度。他找到了一个折扣很大的模块，花了他 30 多美元。它甚至有一个自学功能，听起来很方便(虽然不确定是否所有这些都有)。

休息之后请看视频。我们喜欢使用他的旧 RAM 散热器来帮助消散驱动加热元件的固态继电器的热量。由于 SSR 在泡沫盒内部，我们可以看到热量成为一个问题。这样它就消散了，但没有浪费。
