# BB313 试验板平台使 ATtiny 原型制作变得轻松

> 原文:[https://hack aday . com/2012/03/01/bb 313-试验板-平台-制作-at tiny-原型制作-无痛/](https://hackaday.com/2012/03/01/bb313-breadboard-platform-makes-attiny-prototyping-painless/)

![Johngineer's BB313](../Images/439e437246a75849681d77d34d36a130.png "Johngineer's BB313")

[John De Cristofaro，又名 Johngineer]在他的电子项目中使用了各种 ATmega 微控制器，但他发现自己更多的时候使用 ATtiny2313 或 ATtiny4313。他厌倦了每次开始一个项目都必须连接引脚接头、电容等，所以他花了一些时间围绕芯片设计了一个易于使用的试验板平台。

受 LadyAda 的 Boarduino 的启发，他的 BB313 板具有 FTDI 引脚接头、ISP 编程接头、复位按钮，以及可直接插入任何试验板的分线引脚。除了分享相似的布局，[约翰]说，相似之处仅此而已。他的开发板是为用 C 或 C++编程的设计人员设计的，所以 Arduino 代码在没有一些实质性修改的情况下无法运行。

该板看起来像一个非常方便的台式工具，我们很肯定它会为任何频率使用这些芯片的人节省大量时间。[John]说，他组装电路板的成本约为 6.50 美元，但价格可能会略有不同，这取决于 PCB 的制造地点。

如果设计看起来有帮助，一定要去他的网站看看。他已经做好了图纸，材料清单，以及其他所有可以拿走的东西。
