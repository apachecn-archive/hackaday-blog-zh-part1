# 只需几个简单的步骤，就可以在钻床上安装激光瞄准器

> 原文:[https://hacka day . com/2012/03/10/只需几个简单的步骤即可将激光瞄准器添加到钻头上/](https://hackaday.com/2012/03/10/adding-a-laser-sight-to-your-drill-press-in-just-a-few-easy-steps/)

![drill-press-laser-sight](../Images/67a86ad97b6ef1031f399fb507e604ba.png "drill-press-laser-sight")

[Derek]正在使用他的 Dremel 钻床准备一堆 PCB，发现每次都很难将注意力集中在旋转的钻头上，以将其与电路板上的焊盘对齐。他认为激光瞄准器将有助于推进这一过程，但由于没有现成的解决方案可供他的印刷机使用，他自己制造了一个。

他买了一个便宜的台灯，有一个可弯曲的金属灯颈，他把台灯拆开，省下了可弯曲的金属护套。他在颈部的一端安装了一个管夹，在另一端安装了一个激光模块。[Derek]然后将激光臂安装在印刷机的“乌鸦巢”上，瞄准钻头的尖端。

正如你在下面的视频中看到的，使用激光轻松定位钻头的能力有助于他快速加工任何 PCB。
