# 用旧电脑制作切割激光器

> 原文:[https://hack aday . com/2010/09/30/build-a-cutting-laser-from-an-old-PC/](https://hackaday.com/2010/09/30/build-a-cutting-laser-from-an-old-pc/)

[德雷克·安东尼]使得从个人电脑制造切割激光器看起来很容易，而且看起来确实很容易。几乎所有你需要的东西都可以在一个死气沉沉的桌面单元里找到。二极管是从 DVD 刻录机(16x 或更快)上拔下来的，同时还使用了电源装置、处理器和 GPU 的散热器。你还需要一个聚焦镜头(只需要几美元)、一些热胶、一个 LM317、一个电阻和一副与激光二极管波长匹配的护目镜。

他将二极管安装到透镜中，然后将组件粘合到处理器散热器上的一个钻孔中。使用 LM317 可变调节器、电阻、电源和 GPU 散热器构建驱动器，以保持冷却。休息后，请观看视频，了解激光切割胶带、燃烧塑料和点燃火柴的过程。[https://www.youtube.com/embed/zihZiGUKUS8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zihZiGUKUS8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T3】
