# 打造你自己的腕表

> 原文:[https://hackaday.com/2009/10/19/build-your-own-wristwatch/](https://hackaday.com/2009/10/19/build-your-own-wristwatch/)

![surface-mount-wristwatch](../Images/89f171162d9bd6acc7627a8ae12774e4.png "surface-mount-wristwatch")

[rgbphil]完成了一项伟大的工作[，详细描述了他是如何制造他的微点手表](http://www.instructables.com/id/Microdot---wrist-watch-LED-pattern-timepiece)。这个项目比我们上个月看到的 [pong 手表](http://hackaday.com/2009/09/28/pong-on-your-wrist/)要平易近人得多。如果你已经制作了一些印刷电路板，但还没有尝试过使用表面贴装元件，这是一个很好的尝试方法。

零件数量相当少，几个开关，电阻，电容，发光二极管，手表晶体，和 PIC 16F88 微控制器。[rgbphil]是使用 charlieplex 的[，因此不需要单独的移位寄存器来驱动所有的 led。他详细讲述了布置电路的过程。他遇到的一些问题包括如何以简单的方式管理所有的 charlieplex 连接，如何在电路板上对芯片进行编程，以及如何对设备进行控制布局。](http://hackaday.com/2008/12/03/intro-to-charlieplexing/)

在休息后我们嵌入的视频中，显示屏看起来很棒。我们将把这些组件添加到下一个零件订单中，并使这个项目成为帮助我们度过未来漫长寒冬的计划的一部分。
