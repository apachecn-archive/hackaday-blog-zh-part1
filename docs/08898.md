# 万圣节技巧:闪烁的南瓜灯

> 原文：<https://hackaday.com/2011/10/23/halloween-hacks-flickering-jack-o-lantern/>

![](img/8a4123db22326d47a0737a46a4e9af08.png "Halloween")

![](img/1cd9f50c3d08e29cca0538fc8580a88b.png "punkin")

一元店总是一个发现一些奇怪东西的好地方，所以当[jethomson]发现一个闪烁的鬼火时，他认为它会成为 [74xx 逻辑竞赛](http://dangerousprototypes.com/open-7400-logic-competition/)的一个伟大项目。

不使用闪烁南瓜自带的闪烁白炽灯泡，[jethomson]决定围绕 74HC14 六角反相施密特触发器 IC 重建一个闪烁电路。通过在输入到地之间增加一个电阻和电容，芯片被用作张弛振荡器。经过一点元件选择和一些计算，他有一个红色和蓝色的 LED 闪烁在 2，6，9 和 15 赫兹。

结果是一个看似随机的光图案，看起来像鬼魅般的蓝色，就像手持的鬼火一样。虽然它可能不是 74xx 竞赛中最复杂的之一，但它在我们的书中因其独创性而获得加分。

虽然[jethomson]说他的相机不能很好地捕捉到他的项目，但他确实上传了一段鬼火的视频。休息之后来看看。
