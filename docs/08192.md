# 红外通信教程

> 原文:[https://hackaday.com/2011/07/29/ir-communications-tutorial/](https://hackaday.com/2011/07/29/ir-communications-tutorial/)

![](../Images/91d12cebf1c1b8c6ab40f9ed9f872124.png "wireless_infrared_complete")

在看到我们通过 light 发布的[通讯后，读者【Chris】在我们的收件箱里留下了这个方便的小链接。一个非常好的](http://hackaday.com/2011/07/28/microcontroller-communications-using-flashing-lights/)[教程，关于使用红外线来实现 2 个 pic 微控制器之间的通信](http://www.pyroelectro.com/tutorials/wireless_infrared_communication/parts.html)。

本教程涵盖了您需要的所有器件、物理布线和原理图，并附有详细说明电路各部分的注释。接下来是基本的红外原理，以及一个简化的按钮电路，你可以看到它实际上正在工作。一旦你在一些试验板上建立了练习，他就做一些软件并从中获得一些结果。

最终，这个小设备的传输速度达到了 9600 波特左右，但是它们必须靠得非常近，并且有一条直线。这些通常被认为是红外技术的两个缺点。红外从未真正消失过，但在很多设备上使用。你知道的越多，对你越有利。
