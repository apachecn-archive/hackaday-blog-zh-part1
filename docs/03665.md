# 电池脱硫器

> 原文：<https://hackaday.com/2009/07/05/battery-desulfator/>

![mini-d](img/e5bfafd630604a756dd5e0e5c7c6b30c.png "mini-d")

Hack a Day favorite [Mikey Sklar]带着新项目回来了。Mini-D 是一种电池脱硫器。如果 12V 铅酸电池的电压低于 12.3V，铅板上将开始形成硫晶体。这种晶体生长增加了内阻，最终使电池无法使用。电池脱硫器通过电池发送高频脉冲，产生共振，打破晶体。对于 60 磅的汽车电池，完全脱硫大约需要三周时间。你可以在他的网站上找到 ATmega169 的原理图和十几行代码。下面嵌入了一个视频，他在视频中解释了该设备和其他技术，如负载测试。
