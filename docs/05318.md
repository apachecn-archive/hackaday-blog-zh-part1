# 托比:轮式割草机机器人进化了

> 原文:[https://hack aday . com/2010/05/10/Toby-reel-mower-bot-evolved/](https://hackaday.com/2010/05/10/toby-reel-mower-bot-evolved/)

![](../Images/238df000d9d222296d142859385c7f78.png "robo-reel-mower-driver")

格雷森·西格勒推出了他的机器人割草机的新版本，他称之为托比。[之前的设计](http://hackaday.com/2010/05/03/remote-control-reel-mower/)给轮式割草机增加了马达，但他在牵引力方面遇到了麻烦。新设计更像是一个多功能机器人平台，用于牵引后面的卷轴电机。有了更好的车轮，更稳定的底座和充足的动力，这是一个重大的改进。

自从我们上次登记后，他的零件订单就来了，现在他已经完全实施了 RC。休息之后请看视频。
