# 模拟墨盒

> 原文:[https://hackaday.com/2011/02/02/emulating-ink-cartridges/](https://hackaday.com/2011/02/02/emulating-ink-cartridges/)

![](../Images/10e8faa4b7405580ec31e6e5a6ea33d3.png "Reverse-Engineering-to-Emulate-Ink-Cartridges-for-")

[Smartie_on_computer]想用爱普生打印机做一些试验。在得到一个有点拆卸的，第一步是简单地让它运行。不幸的是，其中一个墨盒不见了，如果没有安装好所有的墨盒，这些打印机几乎什么也做不了。[Smartie_on_computer]没有去购买他们不打算实际使用的昂贵墨盒，而是选择使用微控制器模拟墨盒。在对墨盒上使用的协议进行了一些搜索后，信息最终出现在[专利](http://www.freepatentsonline.com/7660008.html)中。[Smartie_on_computer]现在有了一台功能打印机，在不久的将来，它注定会成为一台 3d 打印机。休息之后你可以看到一段视频。
