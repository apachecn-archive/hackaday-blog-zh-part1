# 厨房窍门:微波炉播放与你的烹饪时间相匹配的 YouTube 视频

> 原文：<https://hackaday.com/2011/11/18/kitchen-hacks-microwave-plays-youtube-videos-matched-to-your-cooking-time/>

看看 uWave，[这是一台在烹饪时播放 YouTube 视频的微波炉](http://uwaved.com/howitworks.html#materials)。[Kevin]和宾夕法尼亚大学的三个同学为 2011 年 PennApps 黑客马拉松开发了这个项目。它用平板电脑取代了众所周知的无聊的老式旋转食物展示微波炉。现在，Arduino 读取烹饪时间，并通过其以太网屏蔽将该信息发送到服务器。然后，服务器在 YouTube 上搜索与烹饪时间大致匹配的视频，然后将该视频推送到平板电脑上开始播放。休息后嵌入的视频演示显示了这一点，以及机器在过程开始时发送的 tweet。

这是一个有趣的概念，我们认为用于将视频推送到平板电脑的代码有很多其他应用程序(我们将这一个保留为书签)。另一方面，我们想知道公共微波炉需要多长时间才能得到广告支持？我们认为，销售抗酸剂、胃酸倒流药、Cup 'o 汤和热口袋的公司很难抗拒这个机会。
