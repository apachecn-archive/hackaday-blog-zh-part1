# 国际象棋桌:网络游戏的物理界面

> 原文:[https://hack aday . com/2010/07/11/chess-table-physical-interface-for-internet-games/](https://hackaday.com/2010/07/11/chess-table-physical-interface-for-internet-games/)

![](../Images/6396e7f7705d2e2be853b9fd6646f5a9.png "internet-chess-table")

想要回到玩真正的棋子上，[托马斯·波托特施尼格]建造了一个与互联网接口的国际象棋桌。工作台使用自定义设置将电路板投影到工作台的磨砂表面上。你的网络对手的棋步在棋盘上显示为箭头，这样你就可以移动棋子。你的动作会被表面下的网络摄像头检测到，因为这款桌子实际上是一个多点触摸显示屏。根据描述，这款投影仪似乎是由 128×64 图形液晶显示器制成的。采用 64×64 像素区域，下方为 LED，上方为镜头。这非常有效。休息之后自己看吧。

想要一些其他的国际象棋设置选项吗？你可以[和机器人手臂](http://hackaday.com/2010/04/14/robotic-chess-opponent/)对战，或者如果你有 144 平方英尺的额外空间，你可以[为偶尔的游戏建造一个真正的大棋盘](http://hackaday.com/2010/06/14/monster-chess/)。
