# 教程:更换坏电容

> 原文:[https://hack aday . com/2012/01/17/tutorial-replacing-bad-capacitors/](https://hackaday.com/2012/01/17/tutorial-replacing-bad-capacitors/)

![](../Images/b1b34abec7782a641f2e003d16fb9849.png "cap")

[glytch]发来了一篇关于在主板上更换垂死电容器的教程，老实说，我们想不出一个更好的入门教程主题。没有什么比让你的朋友认为你是一个从坟墓里把坏掉的电子产品带回来的巫师更好的了。

对于维修，一个死主板是[glytch]的猎物。看了一眼板子后，他发现了几个鼓鼓囊囊的快要爆开的瓶盖。在订购了几个具有相同电压、电容和尺寸的电容后(相信我们，您会想要相同尺寸的电容)，[glytch]将一个熨斗和脱焊编织物拿到电路板上，并更换了几个电容。

迟早，所有的电容器都会坏掉。这并不总是一件坏事——在[电容器故障](http://en.wikipedia.org/wiki/Capacitor_plague)之后的几年里，我们捡了不少“坏了的液晶显示器”,用价值几美元的电容修好了它们。我们的[80 年代末的电脑](http://68kmla.org/wiki/Capacitor_Replacement)中的许多盖子已经被更换，而这些机器仍然在突突前进。
