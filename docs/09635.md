# 机器人蚀刻草图绘制灰度图像

> 原文:[https://hack aday . com/2012/02/06/robotic-etch-a-sketch-draws-gray-images/](https://hackaday.com/2012/02/06/robotic-etch-a-sketch-draws-grayscale-images/)

![](../Images/3553a6c60fc99dd08ca58977d464d13a.png "Untitled")

[Patrick] [决定做一个电脑控制的蚀刻草图](http://www.patrickmccabemakes.com/PatrickMccabeMakes/da_Vinci.html)。虽然这个想法是[而不是新的](http://hackaday.com/?s=etch-a-sketch)，但总有不同的方式来实现目标。Arduino 用于控制一对步进电机，这两个电机的价格非常便宜，甚至自带驱动程序。接下来，使用泡沫板模拟了一个支架，这有助于确定所有部件应该放在哪里。

接下来是一种将步进器连接到旋钮的方法，将使用齿轮，并采购用于模型飞机的夹头来实现齿轮和轴之间的机械连接。通过泡沫板和纸打印件，一切就绪后，就可以开始制作薄胶合板了。胶合板通过激光切割机制成大部分支架和齿轮。现在一切都是软件了，一个为 OSX 设计的程序将低分辨率的图片转换成曲线，非常适合蚀刻草图在屏幕上绘制。

结果相当令人印象深刻，休息后加入我们的快速视频。
