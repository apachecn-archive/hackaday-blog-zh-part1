# Uzebox 视频播放器

</span> <br/>大家最喜欢的开源游戏主机，<a href="http://belogic.com/uzebox/" target="_blank">的 Uzebox </a>(也被克隆为<a href="http://hackaday.com/2008/11/25/fuzebox-open-source-gaming/">的 Fuzebox </a>)，刚刚被黑进了一个新功能——<a href="http://uzebox.org/forums/viewtopic.php?f=5&amp;t=317&amp;hilit=video+player" target="_blank">一个视频播放器</a>。在降低质量(8 位颜色)的情况下，Uzebox 能够以 30fps 的速度播放 SD 卡上的“黑客帝国”,外加 15kHz 的音频。当考虑到它是在 4096 字节的 RAM 上运行时，这是一个令人印象深刻的壮举。视频文件必须首先通过 Photoshop 宏转换成一系列图片才能播放。除了一个<a href="http://www.avrfreaks.net/index.php?name=PNphpBB2&amp;file=viewtopic&amp;p=612093" target="_blank">超频</a> <a href="http://www.atmel.com/dyn/resources/prod_documents/8011S.pdf" target="_blank"> ATmega644P </a>和<a href="http://www.analog.com/static/imported-files/data_sheets/AD725.pdf" target="_blank"> AD725 </a>(在某些版本中已经跳过<a href="http://www.dutchtronix.com/UzeboxConsole.htm" target="_blank">)之外，一个 Uzebox 可以用几个电阻来构建。</a></p> </body> </html>
