# Uzebox 视频播放器

> 原文：<https://hackaday.com/2009/09/10/uzebox-video-player/>

[https://www.youtube.com/embed/hWWsSn_QKLM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/hWWsSn_QKLM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
大家最喜欢的开源游戏主机，[的 Uzebox](http://belogic.com/uzebox/) (也被克隆为[的 Fuzebox](http://hackaday.com/2008/11/25/fuzebox-open-source-gaming/) )，刚刚被黑进了一个新功能——[一个视频播放器](http://uzebox.org/forums/viewtopic.php?f=5&t=317&hilit=video+player)。在降低质量(8 位颜色)的情况下，Uzebox 能够以 30fps 的速度播放 SD 卡上的“黑客帝国”,外加 15kHz 的音频。当考虑到它是在 4096 字节的 RAM 上运行时，这是一个令人印象深刻的壮举。视频文件必须首先通过 Photoshop 宏转换成一系列图片才能播放。除了一个[超频](http://www.avrfreaks.net/index.php?name=PNphpBB2&file=viewtopic&p=612093) [ATmega644P](http://www.atmel.com/dyn/resources/prod_documents/8011S.pdf) 和 [AD725](http://www.analog.com/static/imported-files/data_sheets/AD725.pdf) (在某些版本中已经跳过[)之外，一个 Uzebox 可以用几个电阻来构建。](http://www.dutchtronix.com/UzeboxConsole.htm)
