# 占领波士顿的太阳能充电器

> 原文:[https://hack aday . com/2011/10/13/solar-charger-for-occupy-Boston/](https://hackaday.com/2011/10/13/solar-charger-for-occupy-boston/)

![](../Images/b10103a4b7a6ec0d80dd0d88210f2120.png "solar")

造反实验室的团队一听说占领华尔街运动要来波士顿，他们就决定拿起烙铁支持抗议人群。他们发明了一个太阳能充电 USB 盒来给手机和数码相机充电。

这个案子直接来自一家军队剩余物资商店。最初，这个盒子被荷兰军方用来测试空气中的毒气，并在可以安全摘下防毒面具时发出信号——希望这不是即将发生的事情的预兆。

盒子顶部是五块太阳能电池板，每块输出两瓦。一块试验板包含一个 7805 电压调节器、几个电容和一个二极管。这种调节输出进入 USB 集线器，并为充电提供电源。

对于我们没有武器的兄弟来说，这是一个便宜又简单的项目，我们想不出比占领波士顿帐篷城更好的项目了。当然，今年冬天的占领抗议活动将需要供暖设备，但至少现在抗议者有办法为他们的通信设备供电。
