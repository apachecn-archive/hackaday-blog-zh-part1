# BAMF2011:从头开始打造键盘

> 原文:[https://hack aday . com/2011/05/21/bamf 2011-keyboards-从头开始构建/](https://hackaday.com/2011/05/21/bamf2011-keyboards-built-from-scratch/)

![](../Images/2912a7af6cb3616fe1123e072aec75ef.png "keyboards")

作为计算机和程序员之间最直接的接口，键盘可以是一个非常私人的，有时几乎是宗教的东西。一些人在他们的老式[IBM M 型](http://hackaday.com/2011/05/18/uber-keyboard-hides-security-tools-in-plain-sight/)或豪华[皮革键盘](http://hackaday.com/2008/12/14/leather-keyboard/)中找到安慰，但是制造者【Carol Chen】把事情完全掌握在她自己手中。[Carol]的 Maker Faire 展览有六个有趣的商业触觉和人体工程学选项的样本……但[Chen]的个人键盘，她作为一名全职程序员致力于她的工作，已经按照她自己的严格规格进行了制作。

*Cherry switches* are available in different colors corresponding to varying tactile feels. These can either be scrounged from existing keyboards (as [Chen] does for the key caps), or vendors such as [Digi-Key](http://hackaday.com/2007/11/16/how-to-where-to-find-parts-for-your-projects/) can source the different colors if you want custom pressure for each finger (as on some commercial ergonomic models). A [laser-cut acrylic](http://hackaday.com/2011/03/02/open-source-laser-cutter-v2/) backing plate holds the key switches in place, while wiring was soldered in point-to-point fashion rather than [etching a PCB](http://hackaday.com/2011/03/03/simple-pcb-etchant-made-from-chemicals-you-can-put-in-your-mouth/). Custom-fabbing the keyboard allows [Chen] to tailor the layout to her *vim*-heavy coding style. Note the miniature space bar, and curious placement of modifier and macro keys around the periphery.
