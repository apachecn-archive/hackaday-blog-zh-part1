# 无线电控制球

> 原文：<https://hackaday.com/2009/02/05/radio-controlled-sphere/>

[https://www.youtube.com/embed/8twa_zqT9Vg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/8twa_zqT9Vg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[prabbit22m]写了一篇关于如何建造无线电控制球体的说明书。该机制相当简单，一个驱动电机，一个伺服系统和一个稳定的陀螺仪。为了转向，伺服机构将重心移向一侧。你可以在上面的视频中看到这个系统运行得非常好。如果它没有那个陀螺仪，那将是疯狂的，相信我们，[我们已经做了我们自己的实验](http://www.youtube.com/watch?v=1od46xkVlPg)。如果你喜欢这个，但想要更多的功能，看看这个[有一个相机，无论走到哪里都能拍照](http://hackaday.com/2005/07/30/defcon-day-2-hamster-ball/)。我们也不能忘记 Swarm。自主的机器人球体群。可能是所有想法中最好的。把它打扮成一个普通的球来扰乱人们。
