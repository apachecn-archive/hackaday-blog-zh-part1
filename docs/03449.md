# 人力电动仪器

> 原文:[https://hack aday . com/2009/04/28/human-powered-electric-instruments/](https://hackaday.com/2009/04/28/human-powered-electric-instruments/)

![music](../Images/4c30ba73b156c6431a13ae15576ccde6.png "music")

[诺亚]向我们透露了他用[身体接触电子仪器](http://web.media.mit.edu/~nvawter/projects/ExertionInstruments/)的工作。他正在制造仪器，虽然是电子的，但需要物理动作来驱动它们。在休息后的视频中，你可以看到一个例子，以波尔卡的形式。他们拿着的那部分(打印头？)连接到为仪器供电的发电机上。这种效果使乐器更像是一种必须弹奏的传统弦乐器。我们也可以很容易地想象一个类似于手风琴或手风琴的界面的功能。在他的网站上的研究报告部分有一些图表，但是你必须通过一两个 pdf 文件才能找到它们。我们希望看到以一种更容易理解的形式对这些工具及其组件进行分类。
