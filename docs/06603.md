# 磷光激光绘画

> 原文：<https://hackaday.com/2011/01/18/phosphorescent-laser-painting/>

这里有一个简单而有趣的想法，可以增加激光扫描仪图像的视觉持久性。利用夜光涂料，[ [Daito Manabe](http://www.daito.ws/en/work/uvlaserfadeout.html) ]准备了一个表面，以便激光的强光留下一个随着时间慢慢褪色的痕迹。他利用这个想法在处理过的表面上打印单色图像，从最暗的区域开始，到最亮的区域结束。这种效果非常有趣，因为图像开始看起来很抽象，但随着时间的推移，它会显示出更多的细节。

正如测试视频中所证明的，激光扫描的脉冲与油漆的褪色率相匹配。因此,“写入”图像所花费的时间似乎与最终图像的期望视觉持久性成正比。我们想知道，通过结合巧妙的定时和可变的激光强度，你能更快地书写图像吗？用这个拍电影会有多难？有了创建自己的[微型激光投影仪](http://hackaday.com/2007/05/22/tinyprojector-mini-pov-laser-projector/)，甚至是 [RGB 扫描仪](http://hackaday.com/2009/10/04/rgb-laser/)的能力，这个想法对于令人兴奋的视觉效果肯定有很大的潜力。使用磷处理的投影屏幕增加便携性！
