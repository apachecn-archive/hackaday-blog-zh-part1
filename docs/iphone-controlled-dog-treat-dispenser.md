# IPhone 控制的狗狗零食分配器

> 原文：<https://hackaday.com/2008/12/05/iphone-controlled-dog-treat-dispenser/>

[https://www.youtube.com/embed/8L3hdPxMa_o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/8L3hdPxMa_o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Stephen Myers]一直在摆弄一些测试版 ioBridge 硬件。他决定制造一个遥控狗零食分配器。 [ioBridge](http://www.iobridge.com/store/ "ioBridge | Store") 的硬件是专门为使网络支持项目变得容易而构建的。主控制板有四个 I/O 通道与附加模块通信。它在主板上有一个以太网端口和一个易于配置的网站。

[Stephen]在他的项目中使用了伺服插件板。分配器是由一个附在伺服机构上的废光盘心轴制成的。他可以从他的 iPhone 上发出命令，iPhone 上会显示狗舍的实时视频。他将基于这个系统构建其他几个自动化项目。
