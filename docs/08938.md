# 爬行的僵尸令人毛骨悚然

> 原文:[https://hack aday . com/2011/10/27/crawling-zombie-is-shocking-creeping/](https://hackaday.com/2011/10/27/crawling-zombie-is-shockingly-creepy/)

![](../Images/ca8e9b5f69654efd059b05bdadad7be9.png "crawling-zomie-prop")

当[马克]发来关于这个爬行僵尸道具的提示时，他说它听起来并不可怕，但警告我们当你看到它时它是可怕的。他是绝对正确的，休息后的视频显示了一些明显的不死运动。

这个[爬虫实际上是无线电控制的](http://www.teambac.com/web_hall/crawler.html)。细节是简短的，但有大量的图片和硬件构建教程的开始。木头框架是僵尸的锁骨和脊椎。连接到脊柱的是两个电机，允许独立的肩部操作。我们敢打赌，现实的运动是由于一个天才的操作员在控制，但如果你玩一会儿，它不会太难掌握。看起来最初的构建是无头的，但是我们认为僵尸头的加入真的完成了这个项目！
