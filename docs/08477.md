# 修理旧的拾放机

> 原文:[https://hack aday . com/2011/09/01/repair-an-old-pick-and-place-machine/](https://hackaday.com/2011/09/01/repairing-an-old-pick-and-place-machine/)

![repairing_juki_placemat_360](../Images/4900a75b615cfc3efbb0de263c5608c9.png "repairing_juki_placemat_360")

专业级的拾取和放置机器非常昂贵，所以当 Null Space Labs 的工作人员只花了 1200 美元买了一个旧的 Juki Placemat 360 时，他们非常兴奋。然而，当他们最终在内部得到它时，他们意识到卖方对“工作”的定义与他们的有点不同。这台机器的压缩机坏了，还有各种各样的其他零件需要仔细检查，然后才能让它们重新运转起来。他们整理了一份“建造日志”,展示了他们的工作，因为这台机器是从一个古老的、大部分时间都在工作的遗迹变成了一台精干、吝啬的拾取和放置机器。

到目前为止，他们已经把它拆了下来，检查了机器这些年来的升级和损坏情况。工作人员开始进行一些小的维修，甚至用机器视觉取代了该装置的激光器，这在测试期间似乎工作得很好。

虽然你的车间里可能没有任何类似的机器，但看着他们一点一点地拆卸和改造它还是很有趣的。
