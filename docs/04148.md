# 发电背包

> 原文:[https://hackaday.com/2009/10/01/power-generating-backpack/](https://hackaday.com/2009/10/01/power-generating-backpack/)

[https://www.youtube.com/embed/UL_CUbN9F7E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/UL_CUbN9F7E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在军队里，你必须携带成吨的东西。这是我们经常听到和看到的事情。他们总是试图想出办法来减轻你带进地里的东西的重量或数量。这个[发电背包](http://www.defensetech.org/archives/005043.html)利用你脚步的自然上下运动来发电。这可以大大减少带入现场的电池数量。反正他们是这么说的，士兵一般随身带几节电池？除此之外，我们认为这很酷。我们可以用它来为我们的手机或 GPS 在长途徒步旅行和野营中充电。你也可以[建立自己的东西](http://hackaday.com/2009/01/15/tape-measure-generator/)来类似地工作。
