# 感觉就像我们用低音吉他音箱一样

> 原文:[https://hackaday . com/2011/04/15/feel-like-we-do-a-bass-guitar-talk-box/](https://hackaday.com/2011/04/15/feel-like-we-do-with-a-bass-guitar-talk-box/)

![dino_comes_alive](../Images/5c287797ee9d748c358f9ea3b4f4ce61.png "dino_comes_alive")

黑客(Dino Segovis)来信分享了他的 HackAWeek 系列的最新黑客技术，这一次他为他的低音吉他建造了一个对话盒。如果你足够老，你可能还记得在“弗兰普顿活了”专辑中，talk box 成为主流音乐的时候。

对话盒的概念非常简单。一个小型扬声器内置在一个密封的外壳中，通过一根塑料管将音乐从音乐家的吉他上传到他的嘴里。管子放在音乐家的嘴里，靠近麦克风。当他的嘴动的时候，来自吉他的声音被修改并反射到麦克风中。

[迪诺]用他的低音吉他和一个放大器从一个旧的磁带走带设备组装了一个类似的系统。他最初遇到的问题是，由于低音频率低，声音无法一直传到电子管。他啊哈了一声！瞬间，将扬声器安装在麦克风上，而不是放在地板上，这似乎已经解决了这个问题。
