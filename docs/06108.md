# 用旧零件制作横卧三轮车

> 原文：<https://hackaday.com/2010/10/12/building-a-recumbent-trike-from-old-parts/>

这辆横卧三轮车是用三辆回收自行车的零件组装而成的，没有焊接。这些自行车在背部和颈部比传统的骑行位置更容易一些。这一个也允许一个更短的踏板曲柄，这是创作者关心的问题，[巴里米尔曼]。他不仅在制作方面做得非常出色，而且还详细地分享了这个项目。

这是一个很好的建设。它不会赢得轻质设计奖，因为它包括一大块胶合板。但是，如果焊接不是一个选项，这是值得的重量打击。为了您的观赏乐趣，休息后，您会发现一个简短的停车场试驾的成品横卧。
