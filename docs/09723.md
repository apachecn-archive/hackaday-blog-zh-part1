# 手持式数控加工

> 原文：<https://hackaday.com/2012/02/20/handheld-cnc-fabrication/>

![](img/2171bad28e4925969ca1f9607331b82a.png "tiger")

虽然将 3D 模型加载到 CNC 程序中并让机器在一片股票上运行是很棒的，但手工雕刻项目的艺术性、工艺和触觉反馈还有很多可说的。[Amit Zoran]和[Joe Paradiso]用他们的手持式但仍然是数字控制的铣削设备在这两种方法之间建立了一座漂亮的桥梁，他们称之为[自由 D](http://web.media.mit.edu/~amitz/Amit_Zoran_home_page/the_freeD.html) 。

免费的 D 看起来像你的普通手持 Dremel 工具，带有雕刻附件和一些额外的伺服系统。这些额外的部分服务于一个目的:该工具实际上跟踪它自己在 3D 空间中的方向。借助工件下方的几块磁铁，Free D 将其方位发送回运行 CNC 程序的计算机。当计算机检测到雕刻附件过于接近所需的形状时，Free D 会自动收回自己的工具头。

考虑到建造一个拥有六个自由度的磨坊的疯狂和花费，免费的 D 看起来像是一些模型制作者工具箱中的有用工具。休息之后看看免费 D 的演示视频。
