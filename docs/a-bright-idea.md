# 一个好主意

> 原文：<https://hackaday.com/2011/05/30/a-bright-idea/>

![](img/1a90fec9e61fda883759ab9f43a69d68.png "Screenshot - 05302011 - 01:10:20 PM")

[Jeri Ellsworth]有了一个好主意——[一个漂浮在你头顶上的大脑激活灯泡](http://www.youtube.com/watch?v=Quz78xRxv1Y)。外出时，她看到一个家伙背包里伸出一根金属棒，上面系着一个电子游戏图标。这根杆子让图标看起来像是漂浮在他的头上(想想模拟人生)，这是这款 LED 供电灯泡的灵感来源。灯泡连接到一根金属杆和一个金属环上，金属环的弹性足以让一对电极紧贴在你的头上。

这些电极以及用于噪声基准的第三个探针连接到仪表放大器 AD620。在运算放大器的帮助下，它调制附着在灯泡背面的红色或绿色 led。最终的结果是一种有趣的方式来显示大脑活动，同时在问答小组上被烤，或者只是在 Maker Faire 上漫步欣赏所有令人惊叹的景象。
