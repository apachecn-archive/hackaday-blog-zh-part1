# 更有激光射击场的美好；现在有了鸭子

> 原文:[https://hack aday . com/2012/01/11/more-laser-shooting-range-good-now-with-duckies/](https://hackaday.com/2012/01/11/more-laser-shooting-range-goodness-now-with-duckies/)

![](../Images/7445d5db0dea463eb598c9befdcf2606.png "laser-shooting-range")

这是另一个项目，[让我们想起了嘉年华上的射击游戏](http://dma.sjsu.edu/art103/duckhunt/)。这实际上是受视频游戏“猎鸭”的启发，由圣何塞州立大学的四名学生共同完成的一个班级项目。它使用看起来像橡胶鸭子的移动玻璃靶。玩家用激光枪射击他们基地的传感器。鸭子发出蓝光表示被直接击中。

[Lananh Nguyen]主修商业营销，但他也辅修工作室艺术，多年来一直在吹玻璃。我们认为他将一直有一份制作和销售玻璃的副业，因为那些鸭子看起来棒极了。[Michael]和[Chris]一起工作，建造了一个可以前后移动目标的振动平台，并在 Arduino 上连接了光传感器。一个绿色激光二极管被添加到一个丙烯酸枪来完成这个项目。休息之后看看游戏视频，看看这一切是如何结合在一起的。

如果你错过了上周我们介绍的另一个激光射击场，你会想再去看看那个用锡罐做靶子的项目。
