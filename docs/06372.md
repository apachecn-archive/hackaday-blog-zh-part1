# 攀登自行车存储挫败小偷？

> 原文：<https://hackaday.com/2010/11/29/climbing-bike-storage-thwarts-thieves/>

如果你有一辆昂贵的自行车，并且不介意在你的快递袋中携带一大堆额外的重量，你会喜欢这个概念。一个设计团队[在大约 14 天内建造了一个爬杆自行车架](http://www.conrad.de/ce/de/content/ti_fahrradschloss/ti_fahrradschloss)。休息后的视频显示了原型制作过程以及使用中的成品“锁”。这是雇用设计师的公司的广告，但这是一种我们不介意看的广告。

方形通道构成了设备的主体，一组直排轮抓住灯杆，并使用三个 12V 齿轮马达进行攀爬。控制器是一个无线遥控器，类似于汽车上用于无钥匙进入的遥控器。在视频中，你可以听到运载工具达到其完成高度时汽车警报设置的陈词滥调。很好。
