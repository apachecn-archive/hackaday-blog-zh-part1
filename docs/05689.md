# 打开双层生态机器人三号便便

> 原文：<https://hackaday.com/2010/07/21/break-out-the-two-ply-ecobot-iii-poops/>

 <https://www.youtube.com/embed/qU6zi1_aZiw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>想象一下当<a href="http://www.brl.ac.uk/projects/ecobot/ecobot%20III/index.html" target="_blank">生态机器人 III </a>上的<a href="http://www.botjunkie.com/2010/07/21/ecobot-iii-eats-poops-moves/" target="_blank">这篇文章</a>和上面的<span style="text-decoration:line-through;">恶心</span>视频出现在我们的 feed 中时，我们有多惊讶。机器人理论上可以永远自给自足，只要它有食物来源。是的，你没看错，食物。</p> <p>依靠 grub 的典型<a href="http://www.botjunkie.com/2009/02/02/eatr-robot-runs-on-soylent-green/" target="_blank">机器人燃烧生物质来产生热量/蒸汽/能量，但 Ecobot III 实际上使用微生物燃料电池进行消化，并以氢气的形式提取能量。</a></p> <p>这个过程还不是很有效率，当然废物必须被排出体外，但是我们离我们的机器人霸主不可战胜的那一天越来越近了。该项目已经停止(我们无法想象为什么)，但你仍然可以在进程上阅读<a href="http://www.brl.ac.uk/projects/ecobot/main.html" target="_blank">，并认识生态机器人的兄弟:</a><a href="http://www.brl.ac.uk/projects/ecobot/ecobot%20II/index.html" target="_blank"> II </a>和<a href="http://www.brl.ac.uk/projects/ecobot/ecobot%20I/index.html" target="_blank"> I </a>。</p> <p>相关:我们都会死，<a href="http://hackaday.com/2009/06/26/carnivorous-robots/">食肉机器人</a>。</p> </body> </html>
