# 打开双层生态机器人三号便便

> 原文：<https://hackaday.com/2010/07/21/break-out-the-two-ply-ecobot-iii-poops/>

[https://www.youtube.com/embed/qU6zi1_aZiw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/qU6zi1_aZiw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

想象一下当[生态机器人 III](http://www.brl.ac.uk/projects/ecobot/ecobot%20III/index.html) 上的[这篇文章](http://www.botjunkie.com/2010/07/21/ecobot-iii-eats-poops-moves/)和上面的恶心视频出现在我们的 feed 中时，我们有多惊讶。机器人理论上可以永远自给自足，只要它有食物来源。是的，你没看错，食物。

依靠 grub 的典型[机器人燃烧生物质来产生热量/蒸汽/能量，但 Ecobot III 实际上使用微生物燃料电池进行消化，并以氢气的形式提取能量。](http://www.botjunkie.com/2009/02/02/eatr-robot-runs-on-soylent-green/)

这个过程还不是很有效率，当然废物必须被排出体外，但是我们离我们的机器人霸主不可战胜的那一天越来越近了。该项目已经停止(我们无法想象为什么)，但你仍然可以在进程上阅读[，并认识生态机器人的兄弟:](http://www.brl.ac.uk/projects/ecobot/main.html) [II](http://www.brl.ac.uk/projects/ecobot/ecobot%20II/index.html) 和 [I](http://www.brl.ac.uk/projects/ecobot/ecobot%20I/index.html) 。
