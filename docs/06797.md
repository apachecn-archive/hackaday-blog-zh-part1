# RFID 触发展示箱

> 原文：<https://hackaday.com/2011/02/11/rfid-triggered-presentation-box/>

![blasphemous_bible_box](img/36ca19525d3bf6973160fde341636244.png "blasphemous_bible_box")

[蕾丝]需要为他的大学艺术课建立一个“盒子项目”,并认为他可以通过添加一些电子元件来增加一些趣味。他的项目被称为“[亵渎神明的圣经盒](http://izationlabs.com/2010/03/25/blasphemous-bible-box/)”，由一本圣经组成，打开了《启示录》中讨论旧雪茄盒中野兽印记的部分。盒子通常是锁着的，但被编程为当他通过他手中嵌入的 RFID 芯片时解开内部闩锁。使用更简单的电路就可以实现这种效果，但内置的 Arduino 似乎可以很好地完成这项工作。[蕾丝]没有提到他是否考虑过修改盒子，但如果第二版建成的话，用磁性锁固定的弹簧加载外部闩锁会产生很好的效果。添加一个伺服系统来慢慢打开盒子，以及包括一个扬声器在打开时大声播放卡米娜·布拉纳-福尔图娜将会非常酷(提示，提示)。休息之后我们有盒子运行的视频。
