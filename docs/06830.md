# 尖叫娃娃恶作剧玩具

> 原文:[https://hackaday.com/2011/02/15/screamer-doll-prank-toy/](https://hackaday.com/2011/02/15/screamer-doll-prank-toy/)

![screamer_doll](../Images/aea98ba083f8a4ca83047daae4c84a3f.png "screamer_doll")

虽然现在只是二月中旬，但这可能是你开始考虑愚人节计划的好时机。迪诺的尖叫娃娃是他提交给即将到来的 T2 555 大赛的作品。这个有趣的小电路可以很容易地用来骚扰你的立方或室友，是所有愚人节恶作剧者的完美选择。他把所有的部件都装进了一个小塑料玩具中，用三个光电池代替了娃娃的眼睛和嘴巴，并用一个明亮的 LED 灯替换了鼻子。当光线照射到光电池上时，会发出响亮的高音尖叫，LED 会猛烈闪烁，直到光源被移除。将设备遮挡光线会导致声音和 LED 都变慢，但 LED 的亮度可以确保玩具在被覆盖时仍然会发出噪音。

如果能看到这个项目的一个版本，在黑暗中完全沉默，允许它在晚上隐藏起来，在早上迎接它的受害者，那将是非常棒的。

如果你想看设备运行的视频，请继续阅读——只是要确保你的扬声器音量保持在合理的水平！
