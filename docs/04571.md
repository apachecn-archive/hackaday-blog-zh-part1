# 佩吉 2 X2，带视频

> 原文：<https://hackaday.com/2009/12/19/peggy2-x2-with-video/>

[https://www.youtube.com/embed/-ImjfHxXGfc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/-ImjfHxXGfc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

作为闪光灯的狂热爱好者，我们总是喜欢看闪光灯工作。上面的视频显示了另一个改进，即[两个 peggy2 单元作为一个](http://littlebird.posterous.com/multiple-peggy2s-showing-serial-video-via-the)单元一起工作。[iservice2000]将两者链接在一起，为显示编写了新的代码。他用一台 Arduino 来驱动这一切，让他们像一个人一样行动。虽然[在钉子上的视频并不新鲜，但这是我们第一次看到它们被拴在一起。最终结果将是一个滚动的标志，可以通过网络更新，或者可以显示推文。我们确实注意到了一点撕裂，是来自相机还是软件？](http://hackaday.com/2008/07/09/live-full-motion-video-on-a-peggy/)
