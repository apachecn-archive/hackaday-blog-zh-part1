# 黑客日链接:2012 年闰日

> 原文：<https://hackaday.com/2012/02/29/hackaday-links-leap-day-2012/>

地球每 365.256 天绕太阳一周。因为这个数字不是一个整数，所以每四年都会在二月多加一天，除非这一年能被 100 整除，除非这一年能被 400 整除，或者类似的情况。为了纪念这个日历黑客，这里有一些在过去一周左右的时间里出现的东西。

#### 大理石音序器

![](img/70925d5682d8a52acc6a4cb510d064a3.png "marbles")

[Brian]发送了这个基于弹珠的音序器,听起来像有人在将 MIDI 运行到 Atari 2600 中。那里的某个地方有光敏电阻，这真的让我们想起了那些[机器人](http://bleeplabs.com/thingamagoop2/)。

#### YouTube CLI

![](img/cb2c574a7cbeaf0d4646c847627f6b14.png "youtube")

[Mike]使用 YouTube 作为他的音乐库。虽然这是一种完全可以接受的听音乐方式，但是用户界面很糟糕。为了解决这个问题，[Mike]正在从命令行下载视频，自动转换成 MP3，并通过扬声器播放。它与 SSH 配合得很好，所以我们称之为胜利。

#### 钥匙卡锁

![](img/c59bf46dc333efcdb0fa78188b98e1b8.png "lock")

[valenitn]刚刚加入麻省理工学院媒体实验室，但他的钥匙出了严重问题——进入大楼需要身份证，但进入他的办公室需要钥匙。他不再需要钥匙了，至少自从他改装了办公室的门之后。看看[的视频](http://vimeo.com/37089736)。

#### 到处都是流行蛋挞猫

![](img/11ef9f5b740148ba1c99ec2644e90903.png "Basic")

[skywodd]在 Maximite Basic 计算机上看到了我们的文章，认为他可以提交一个他一直在做的项目。他让他的 Maximite 唱 [nyan 猫歌曲](http://skyduino.wordpress.com/2012/02/02/mmbasic-duinomite-mega-et-nyan-cat-ftw/)，然后创造了一个基本的[音乐播放器](http://skyduino.wordpress.com/2012/02/04/mmbasic-lecteur-de-partition-musical-epik-win/)。干得好，[skywodd]。

#### 不知道是聪明还是疯狂

![](img/f13903d26e34e2fece046b316cd2915a.png "not sur")

[Vikash]偶然看到一个[论坛帖子](http://www.computerhope.com/forum/index.php?topic=119445)，其中一个名为[I Shooter]的用户描述了他的 Windows 和 Linux 双启动设置:[I Shooter]将数据线连接到一对 SATA 硬盘，一个加载 Windows，另一个加载 Linux。电源电缆使用继电器切换*，因此一次只有一个驱动器通电。*【I Shooter】在创造力上得了很多分，但这种强力硬件双启动设置并不常见是有原因的。我们希望有这张照片。
