# 强力链接控制器

> 原文：<https://hackaday.com/2009/02/03/punchy-punchout-controller/>

![punchy_mcpunchpunch](img/045f8ce6b077ac34a4bd70475c95a771.png "punchy_mcpunchpunch")

我们喜欢另类的输入。他们可以让一个古老的经典重现活力，或者给大多数平凡的任务增添一点趣味。在这里，我们看到一个完美的例子，游戏[为 NES 打卡是由一个出气筒类型的东西](http://www.instructables.com/id/Interfaces_for_Games_PunchOut/)控制的。碰撞传感器是手工制作的，并连接到电脑游戏手柄上。它们被安装在一些泡沫上，形成一个漂亮的糊状冲压表面。也有一些反馈，当你的角色被击中时，一个自定义脚本会检测到精灵颜色的变化，并向 Arduino 发送信号。现在，它只是点亮一个 LED，但目标是发出频闪信号，让你望而却步。这看起来会很有趣，尤其是如果你想让它更有冲击力的话。休息后可以看到他们轻击它的视频。
