# Nerf 哨兵炮塔

> 原文:[https://hackaday.com/2010/06/18/nerf-sentry-turret/](https://hackaday.com/2010/06/18/nerf-sentry-turret/)

随着考试的结束，亚当·格雷格有时间制作一把 Nerf 哨兵枪。把所有的东西放在一起其实很容易。他有一个运行[动作](http://www.lavrsen.dk/twiki/bin/view/Motion/WebHome)的上网本，这是一个开源的动作感应程序，用于网络摄像头。当检测到移动时，通过 USB 电缆连接的 Arduino 会启动伺服系统来扣动枪的扳机。炮塔本身的电池升级提高了射击速度。看到用几支铅笔和满满一拳头的电缆扎带完成硬件原型制作是很有趣的。休息之后来看看。

这个特殊的玩具，Nerf N-Strike Vulcan，已经成为 turrent 项目的一个非常受欢迎的起点。我们已经看到了一个使用机动基座的[和另一个作为](http://hackaday.com/2010/05/12/nerf-sentry-gun-build-in-progress/)[一部分的](http://hackaday.com/2009/05/17/nerf-centry-gun-with-image-recognition/)。
