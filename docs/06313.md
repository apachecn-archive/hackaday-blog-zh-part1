# 最新的硬件奖励，开放激光雷达项目

> 原文：<https://hackaday.com/2010/11/17/newest-hardware-bounty-the-open-lidar-project/>

受到 Adafruit 成功发布的 Kinect 赏金的启发，RobotBox 社区的[gallamine]发布了自己的~~【200 美元~~[【400 美元】赏金](http://robotbox.net/blog/gallamine/open-lidar-project-hack-neato-xv-11-lidar-200-bounty)，奖励第一个能够黑掉 Neato Robotic 的 XV-11 vacuumbot 的扫描激光雷达的人。这种传感器对任何机器人制造商都特别有用，因为即使是真空吸尘器的全零售价也低于大多数独立激光雷达设备的成本，后者往往超过 1000 美元。奖金似乎每天都在增加，从 200 美元开始，由于其他几个感兴趣的团体，奖金翻了一番。

幸运的是，据我们所知，该传感器从未被设计成防黑客(甚至可能是秘密的友好黑客？)，因为该传感器的主要开发者之一是某家 Brew 机器人俱乐部的成员。当[公司对黑客](http://hackaday.com/2010/11/10/monotron-openly-monophonic/)好的时候，我们很高兴，我们希望在未来看到更多这样的例子。不确定 XV-11 是什么？请务必在休息后查看视频，了解有关真空及其扫描激光雷达的信息。
