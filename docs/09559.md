# 通过 Android 和 Launchpad 远程登录

> 原文:[https://hack aday . com/2012/01/24/remote-entry-via-Android-and-launch pad/](https://hackaday.com/2012/01/24/remote-entry-via-android-and-launchpad/)

![](../Images/58679f7e93567d5bd9bcde4f3b140098.png "Untitled")

[MarkoeZ]的门锁有钥匙断裂的问题，而正常人可能会尝试简单地更换所有的锁，[MarkoeZ]决定用“黑客日”的方式，给他的门加一把蓝牙锁。他的门已经设置好了，有人可以走过去，呼叫内部对讲机，然后有人按门铃让你进去。从内部来看，对讲机由 TI Launchpad 改装而成，带有“deal extreme”蓝牙模块和继电器。

通过使用运行“BT_Serial_Tester”的 android 手机来授予访问权限，这是一个简单的应用程序，允许您通过蓝牙发送字符。只要输入一个密码，在蜂鸣器响起之前抓住门，你就进去了！MSP 代码和原理图的起点在他的博客上。休息之后，请加入我们，观看一段快速演示视频。
