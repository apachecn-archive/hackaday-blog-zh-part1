# DIY 脉搏血氧仪

> 原文：<https://hackaday.com/2010/05/02/diy-pulse-oximeter/>

这台[脉搏血氧仪](http://translate.google.com/translate?js=y&prev=_t&hl=en&ie=UTF-8&layout=1&eotf=1&u=http%3A%2F%2Fwww.elektroda.pl%2Frtvforum%2Ftopic1655378.html&sl=pl&tl=en)结果非常好。它基于一个飞思卡尔微控制器，可以检测脉搏和血液中的氧饱和度。该传感器由两块木头制成，允许两种波长的光穿过你的手指。一个传感器接收到你粗短手指另一侧的光线，然后比较读数来计算饱和度。休息后检查完成的项目。

几个月前，我们看到了基于 Arduino 的血氧计。这种生物识别黑客在这里很少见。如果你有一个[有据可查的](http://hackaday.com/2009/09/19/how-to-make-your-project-an-internet-sensation/)项目，别忘了[告诉我们关于它的](http://hackaday.com/contact-hack-a-day/)。
