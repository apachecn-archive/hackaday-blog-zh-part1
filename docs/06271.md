# 单声道的

> 原文:[https://hackaday.com/2010/11/10/monotron-openly-monophonic/](https://hackaday.com/2010/11/10/monotron-openly-monophonic/)

![](../Images/0de26eb8e9dfa252a8aed8b2adfd65ee.png "Picture 6")

著名的 synth 制造商 Korg 已经[发布了他们小小的 Monotron 的原理图](http://www.korg.com/monotron) ( [如果你不喜欢填写表格](http://www.korg.co.jp/Support/Manual/download.php?id=512)就把它们放在这里，让所有人都可以看到和使用！这对于任何想从头开始建立合成器或者对现有的 monotron 进行电路改造的人来说都是一个好消息。将滤波器电路单独添加到现有的电子设备中会很有趣。

诚然，已经有[许多](http://www.youtube.com/watch?v=MgHmxKy-pLk)[monotron MODS 的例子](http://www.youtube.com/watch?v=9WcF819PX9I)，但这不应该阻止你试验你自己的变体。现在，有了原理图，你就可以在自己的 CAD 软件中对 synth 的架构进行根本性的改变。想要更多振荡器吗？失真？一切都等着你去探索。我们很想知道人们会在这方面走多远。感谢 Korg 认识到黑客的价值！
