# 大狗的两足兄弟

> 原文：<https://hackaday.com/2009/10/27/bigdogs-bipedal-brother/>

 <https://www.youtube.com/embed/67CUudkjEG4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>波士顿动力又来了。这一次，他们创造了一个具有自然步态的令人毛骨悚然的两足动物。它可能看起来非常类似于 BigDog，因为它实际上是几乎相同的系统。这个名为 PETMAN 的双足系统被设计用来帮助测试化学防护服。这个机器人可以通过在充满毒气的房间里行走、奔跑，甚至爬行来给宇航服施加压力。PETMAN 不仅能走、跑、爬，还能流汗、变体温。那很酷。像大狗一样，最令人印象深刻的部分是当他们推它一把<a href="http://www.youtube.com/watch?v=W1czBcnX1Ww" target="_blank">时，它以一种几乎是有机的动作恢复过来</a>。</p> </body> </html>
