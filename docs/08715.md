# 可持续发展秘诀:瘦客户机服务器

![](../Images/01965a188f5eb89e4da18b2ee674c374.png "thin")

虽然不环保，但是我们大多数人都是 24 小时运行小型家庭服务器的。一个小型服务器是一个有用的工具，不幸的是浪费了大量的能源。[kekszumquadrat]的[瘦客户端家庭服务器](http://fomori.org/blog/blog/2011/09/29/cheap-home-server-introducing-the-thin-server/)实际上是一个还过得去的灯箱，不会消耗大量的电力。

[kekszumquadrat]在开始他的任务时开始查看 [SheevaPlug](http://de.wikipedia.org/wiki/SheevaPlug) ，但是有点担心[的电源故障](http://chemicaloliver.net/electronics/sheevaplug-why-globalscale-suck/)。在寻找替代品时，他在易贝遇到了很多廉价的瘦客户机。价格合适，而且*的一切*都运行 Linux，所以几天后他就有了一台[惠普 t5710](http://h10010.www1.hp.com/wwpc/ca/en/sm/WF06b/12132708-12133552-12133552-12133552-12133560-12133572-39506253.html) 瘦客户机。

这台小电脑在连接到 IDE 端口的闪存驱动器上安装了一个嵌入式版本的 XP。抛弃了那个“操作系统”，[kekszumquadrat]连接了一个 USB 硬盘并安装了 [Arch Linux](http://www.archlinux.org/) 。经过几次更新和软件包安装后，他有了一台连接到互联网的有用的机器。

与 SheevaPlug 的 7 瓦功耗相比，15 瓦的瘦客户机是一个耗能大户。与我们临时搭建的服务器相比，[kekszumquadrat]做得非常出色。回收旧硬件也不会伤害任何人。
