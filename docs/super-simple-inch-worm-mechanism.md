# 超简单英制蜗杆机构

> 原文：<https://hackaday.com/2010/08/10/super-simple-inch-worm-mechanism/>

[https://www.youtube.com/embed/dvplpWJ9CcI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/dvplpWJ9CcI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

坚持“机器人”定义的人应该转移你对视频片头标题的注意力。[Randofo]发布了这个[漂亮简单的英寸蜗杆机构](http://www.instructables.com/id/Simple-Bots-Inchworm/)，仅使用了一把尺子、一些连接器、一个开关、一个伺服系统、一把梳子、一些电池和几个特百惠容器。它英寸，因为它的设计，非常好。我们特别喜欢使用梳子作为一个容易修改的开关激活器。
