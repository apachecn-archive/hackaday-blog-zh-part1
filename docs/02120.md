# 啤酒机器人会给你倒一杯冰的

> 原文:[https://hack aday . com/2008/07/02/beer-bot-will-pour-you-a-cold-one/](https://hackaday.com/2008/07/02/beer-bot-will-pour-you-a-cold-one/)

我们爱[啤酒](http://www.hackaday.com/2008/05/28/beverage-cooling-contraption-contest/)，我们爱[机器人](http://www.hackaday.com/category/robots-hacks/)，所以猜测我们对这个[机器人酒保](http://www.channelflip.com/2008/07/02/asahi-beer-robot/)的感觉应该是显而易见的。这个被称为“朝日先生”的机器人一边打开瓶子、倒啤酒，一边泰然自若地用欢快的英国口音接受你的订单。它还有一个定制的懒人苏珊，上面有瓶子槽和机器人手可以抓住的槽口。

令人惊讶的是，这并不是朝日第一个倒啤酒机器人。不过，那个要求你做大部分的跑腿工作，所以这个是一个显著的改进。最棒的是，当一个比你更有魅力的人走进酒吧时，它不会忽视你，也不会因为你给了一个糟糕的小费而冷落你。我们现在担心酒精引发的机器人暴行…它们每天都变得更像人类。休息后的视频。
