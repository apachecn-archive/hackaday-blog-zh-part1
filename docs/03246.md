# 假特雷门

> 原文:[https://hackaday.com/2009/02/23/false-theremin/](https://hackaday.com/2009/02/23/false-theremin/)

![fth](../Images/6e7cf38ba1e29420dd146c76f5ee261e.png "fth")

[vilxes91]送来了这个很酷的小[假——特雷门琴](http://vlxs.wordpress.com/) ( [译](http://74.125.67.132/translate_c?hl=en&sl=es&tl=en&u=http://vlxs.wordpress.com/&prev=_t&usg=ALkJrhjyh_qQAARVh1NJn9dWyAwBqj3jhQ))。这是一个[非常简单的电路](http://hackaday.com/2008/10/25/tiny-optical-theremins/)，可以安装在非常小的地方。这不是一个真正的调光器，因为它利用从盖子周围射入的光量来进行调制。要演奏它，你打开它，打开它，盖子越开，音调越高。休息之后你可以看到一段视频。
