# Kinect、鼠标和 Nerf Gun 组合用于《死亡之屋》

> 原文:[https://hack aday . com/2011/03/21/Kinect-mouse-and-nerf-gun-combine-for-house-of-the-dead/](https://hackaday.com/2011/03/21/kinect-mouse-and-nerf-gun-combine-for-house-of-the-dead/)

![](../Images/ad773191824cf747db17fcb0e6c95a9f.png "house-of-the-dead-controller")

【托尼·布兰奇】为玩《死亡之屋》自己做了[动作控制器。它应该适用于任何遵循“轨道”类型游戏的射击游戏(你的角色不是自由行走，而是沿着你无法控制的设定路径移动)。](http://www.instructables.com/id/Using-a-Nerf-gun-and-Kinect-to-play-House-Of-The-D)

两个部分共同促成了这一切。第一个是你在上面看到的 Nerf 镖枪。安装在塑料外壳顶部的电路板来自五键无线鼠标。这些按钮用于感应玩家的扣扳机动作。控制器的第二部分是 Kinect。它已被设置为与 Windows 7 机器一起工作。[Tony]使用灵活动作和关节骨骼工具包( [FAAST](http://projects.ict.usc.edu/mxr/faast/) )来绑定和跟踪枪支控制器，在屏幕上移动鼠标光标以匹配武器的移动。休息后看看视频，看看这个系统的反应有多快。

这与我们在之前见过的[枪械控制器有着非常有趣的区别。](http://hackaday.com/2010/05/10/fps-controller-hacks-getting-easier/)
