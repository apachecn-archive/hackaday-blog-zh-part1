# 机器人手臂和爪子完全由 ShapeLock 雕刻而成

> 原文:[https://hack aday . com/2011/08/19/完全由 shapelock 造型的机械臂和爪子/](https://hackaday.com/2011/08/19/robotic-arm-and-claw-sculpted-entirely-from-shapelock/)

![shapelock_robotic_claw](../Images/ac4b4b9473957135d8691e84462858ba.png "shapelock_robotic_claw")

[阿列克谢]在[中写道，分享一个他已经努力研究了很长时间的机械爪](http://robocraft.ru/blog/mechanics/583.html) ( [谷歌翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=ru&tl=en&u=http%3A%2F%2Frobocraft.ru%2Fblog%2Fmechanics%2F583.html))。虽然很多人会求助于某种 3D 塑料打印机，如 MakerBot，如果他们需要塑料部件的话，[Alexey]没有机会。相反，他精心制作了聚己内酯的整个机制，或者更普遍的说法，Shapelock。

他使用了从吹风机、刀子到照明设备的各种工具，用塑料一点一点地手工雕刻出爪子及其控制臂。我们对他能够从塑料中切割出的传动装置印象特别深刻，这有时会很挑剔。

正如你在下面的视频中看到的，爪子通过一些 Arduino 驱动的伺服系统模仿他用控制臂做的每个动作。一切似乎都运行得相当好，尽管谷歌进行了粗略的翻译，但我们认为这是一个伟大的项目。如果你自己也想做类似的事情，他的网站上有很多图片，这应该会给你一个很好的想法，知道这些东西是如何组合在一起的。
