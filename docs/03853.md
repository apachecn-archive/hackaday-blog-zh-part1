# 人形机器人+家酿沃尔多=大笑脸

> 原文：<https://hackaday.com/2009/08/24/humanoid-robot-homebrew-waldo-big-smiles/>

[https://www.youtube.com/embed/WCTvggKuPvU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/WCTvggKuPvU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

机器人发烧友维塔利娅斯·诺瓦建造了这个平台，可以让仿人机器人实时模仿自己的身体动作。他把自己的人机界面称为“主从服”，但在其他地方，这种界面往往被称为 1942 年罗伯特海因莱茵有先见之明的中篇小说之后的随想。这个[项目页面](http://blog.elektronika.lt/robotai/humanoidas/)没有太多细节，大部分都是用他的母语立陶宛语写的，但图片很能说明问题，通过谷歌翻译(Google Translate)的一点帮助，我们可以了解到一些基本事实:这个机器人本身就是一个商用套件，也就是来自日本的[近藤 KHR-1HV](http://www.kondo-robot.com/product/khr-1hv.html) 。这种定制的挽具使用一系列苏联时代的军用电位计(购自易贝)读取他的肘和肩的位置，然后基于 ATmega8 的接口板将这些读数转换成运动命令，发送到机器人的车载控制器。一些额外的注释和代码可以在[robosavy 论坛](http://robosavvy.com/forum/viewtopic.php?t=3768)上找到。

有用吗？看着吧。随着视频的进展，他的笑容很有感染力！

Hack a Day 之前已经介绍过其他的 Waldos，但是最新的这款以其轻便简洁而赢得了时尚积分。
