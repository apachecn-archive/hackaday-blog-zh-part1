# 毒品系统

> 原文:[https://hackaday.com/2009/03/28/the-narcisystem/](https://hackaday.com/2009/03/28/the-narcisystem/)

![biometric](../Images/125871f4a7ec34265698adad8c8cd35f.png "biometric")

麻醉系统是艺术展览的一部分，在展览中，埃里克将自己绑在尽可能多的生物传感器上。该系统的核心是一个[漏斗 IO](http://www.sparkfun.com/commerce/product_info.php?products_id=8957) ，它包括一个 Arduino、Xbee 插头和 LiPo 充电电路。它从心率监测器、脑电图仪、[酒精测试仪、](http://www.mahalo.com/Breathalyzer "breathalyzer - Mahalo")、指南针和加速度计收集数据。这些数据被发送到一台[笔记本电脑](http://www.mahalo.com/Laptop "laptop - Mahalo")，然后被发送到不同的显示器。休息之后，您可以在视频中看到该设置的运行情况。红光是他的心跳，蓝光是他面对的方向。你看不到的是他每走一步的大功率低音重击声。脑电图数据本应影响音乐的节奏，但它失败了，被放弃了，就像基于他血液酒精含量的烟雾机器一样。他指出他想做得更多，但是缺少硬件。
