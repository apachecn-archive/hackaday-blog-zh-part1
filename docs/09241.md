# 支持网络的 Kinect

> 原文：<https://hackaday.com/2011/12/06/web-enabled-kinect/>

![](img/32a84fa0f83b849173a3833809c532c1.png "kinect")

有 Kinect hacks 用于机器人视觉、3D 扫描仪，甚至伪激光雷达设置。到目前为止，这些构建的一个限制因素是要求设备上有一台成熟的计算机来处理深度图并进行所有必要的处理和计算。这似乎不是什么大问题，因为[wizgrav]发布了 Kinect 的 HTTP 接口 [Intrael](https://code.google.com/p/intrael/) 。

[Eleftherios]在他的[本地黑客空间](http://hackerspace.gr/wiki/Main_Page)赶上了[wizgrav]，在那里他做了一个关于 Intrael 的[简短教程](http://www.youtube.com/watch?v=sIXIGeJSuAg&feature=youtu.be)。[wizgrav]的项目提供了来自 Kinect over HTTP 的每一帧，这些帧被封装在 JSON 数组中。除了声音，Kinect 输出的任何东西现在都可以通过互联网轻松获得。

该项目旨在将计算机视觉从台式机和机器人笔记本电脑领域转移到网络领域。[wizgrav]对他的项目的用途有一些想法，例如智能安全摄像头和各种交互式表面。

休息之后，看看 Intrael 的入门[wizgrav]演示(对我们来说是希腊语，但有字幕)，以及一些 Intrael“所见”的演示。
