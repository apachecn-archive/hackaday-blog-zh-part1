# Commodore 64 笔记本电脑

> 原文：<https://hackaday.com/2009/04/06/commodore-64-laptop/>

![c64](img/3ef9ba71fbd160a1d693ba222c046b53.png "c64")

[Ben Heck]刚刚完成了他的一个更独特的笔记本电脑游戏控制台。这一次是准将 64 号，他从 2006 年就开始尝试了。最近，他放弃了一切，重新开始了迄今为止最快的构建。虽然它看起来和他的其他笔记本电脑很相似，但他花了很大力气让它从米色到质地看起来像 80 年代的电脑。他使用了原始的 C64C 主板，因为这是最后的和最小的修订版，并与原始键盘相结合。一个 [1541-III-DTV](http://www.pyrofersprojects.com/blog/?page_id=38 "Pyrofer’s Projects   » 1541-III-DTV") 允许使用 SD 卡作为软盘设备。只需将任何磁盘映像拖到卡上，它就准备好了。下面看看它的使用视频。
