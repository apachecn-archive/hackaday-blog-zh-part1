# 自由烤面包机免费分发自由/开源软件

> 原文:[https://hack aday . com/2012/03/06/freedom-toaster-dispenses-FOSS-for-free/](https://hackaday.com/2012/03/06/freedom-toaster-dispenses-foss-for-free/)

![](../Images/c93151916de70b887ed3f40d9c22f10f.png "freedom-toaster")

塞内卡学院 Linux 俱乐部想出了一个绝妙的方法来帮助向更广泛的受众推广 Linux。他们拿了一些多余的硬件[做了一个开源软件自动售货机](http://cdot.senecac.on.ca/projects/toaster/)。这不是文字游戏。该项目本身是一个开源项目，目标是以 CD 和 DVD 的形式分发其他开源软件。

他们的构建页面分享了所有的细节。他们购买了一台旧的服务器机柜，该机柜即将被 IT 部门淘汰。它足够大，可以容纳一个人，这有点大材小用，但这也减少了有人试图偷走它的可能性。在里面你会发现一台电脑，两个显示器(一个是触摸屏供消费者使用，另一个只是额外隐藏在里面进行维护。

必须自带空白 CD-R 或 DVD-R(但刻录是免费的)。你可以在上面油漆一新的亭子里看到齐腰高的 DVD 架。我们认为这里唯一缺少的是一个 USB 端口，用于酝酿一个可启动的 USB 棒。
