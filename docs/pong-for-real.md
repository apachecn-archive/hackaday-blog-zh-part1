# 乓，真的

> 原文：<https://hackaday.com/2010/01/06/pong-for-real/>

[https://www.youtube.com/embed/Z5mjx1iYt5g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Z5mjx1iYt5g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在 EvilMadScientistLaboratories.com，他们问了一个我们很多人从未想过要问的问题。Pong 到底应该是什么？他们没有像我们其他人一样假设这是乒乓球，而是决定建立一个字面上的物理解释。他们可能采取了一些自由，使用螺线管作为桨，但最终效果相当好。看他们打球看起来既有挑战性又有趣。有一篇关于建设的很棒的文章，所以一定要查看项目页面。
