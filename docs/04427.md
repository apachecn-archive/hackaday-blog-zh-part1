# 全向车

> 原文：<https://hackaday.com/2009/11/17/omni-car/>

[https://www.youtube.com/embed/QB15-uSNvHA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QB15-uSNvHA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们偶然发现了[尼克·梅尔顿]的一个项目，一辆全向车。它是全向的，这意味着它可以在任何时候去任何方向，而不必转弯。身体是他设计的，然后用 3D 打印机打印出来。不过，我们感兴趣的是控制方案。他找到了一种简单的接线方法来完成这项工作。当然，你可以看到它在试图走直线时遭受了一些非常糟糕的“漂移”,但总的来说，我们认为他做得很好。

你可能想花点时间看看他的项目页面。这家伙已经做了一堆有趣的东西，像三角洲机器人，奇怪的混合轮/腿机器人，特斯拉线圈，和手臂安装喷火器。从视频来看，他也很年轻。我们认为他的人前途光明。
