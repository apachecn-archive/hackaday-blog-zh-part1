# 150 美元的数控铣床有点慢，但非常坚固

> 原文:[https://hack aday . com/2011/11/11/150-CNC-mill-is-a-tad-slow-but-very-solid/](https://hackaday.com/2011/11/11/150-cnc-mill-is-a-tad-slow-but-very-solid/)

![diy-150-cnc-mill](../Images/a4ecd5df8c84a8c555627b3995978746.png "diy-150-cnc-mill")

就像其他人一样，[亚当]认为数控机床很酷——酷到他决定从头开始制造一台自己的。

数控机床主要由中密度纤维板和废木料制成，抽屉滑轨用于平稳的龙门移动。一个非品牌的旋转工具被用来做实际的切割，[亚当]拿起几个 Sparkfun 步进电机来驱动机器。

组装没有太多的麻烦就完成了，但是[Adam]说为工厂编程是一个漫长而令人沮丧的过程。起初切割很粗糙，不太准确，但渐渐地他把事情做得相当好了。正如你在下面的视频中看到的，虽然剪切看起来很棒，但速度的提高是以牺牲为代价的。他说，这台机器可以通过重新设计来加快速度，如果有空的话，他会抽出时间来做这件事。

这不是我们见过的绝对[最便宜的 CNC 版本](http://hackaday.com/2010/06/21/100-cnc-mill/)，它非常接近。经过一些调整，它肯定会是一个预算友好的有力竞争者。
