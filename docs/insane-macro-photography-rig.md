# 疯狂的微距摄影设备

> 原文：<https://hackaday.com/2011/11/14/insane-macro-photography-rig/>

[fotoopa]刚刚上传了一个他的 3D 微距摄影设备的 Flikr 构建日志，他用这个设备为[拍摄飞行中的昆虫](http://www.flickr.com/photos/fotoopa_hs/sets/72157623127172132/)。在好莱坞或*国家地理*之外，我们从未见过如此疯狂的摄影设备。

[fotoopa]的构建基于两台相机——一台尼康 D200 和 D300。这些相机通过两个镜子指向主题昆虫，从而为 3D 图像提供良好的立体分离。当然，麻烦的是当一只昆虫飞到钻机前面时抓拍照片。

对于快门控制，[fotoopa]使用了两个红外激光笔指向两个相机会聚的地方。钻机上方镜头中的光电二极管检测到这个红外点并触发快门。为了加速尼康相机上极其缓慢的 50 毫秒快门，增加了一个[高速快门](http://www.uniblitz.com/product.aspx?productName=VS14&housing=S1&bfID=1&mount=&so=&hto=&tab=0)，以便在 3 毫秒内捕捉图像。

[fotoopa]的 2011 年[钻机](http://www.flickr.com/photos/fotoopa_hs/sets/72157625073020642/)把事情降低了一个档次；今年他只使用一台摄像机。尽管他今年没有获得任何 3D 图像，但制作如此令人敬畏的钻机的技能令人印象深刻。
