# 机器人感应

> 原文:[https://hackaday.com/2010/02/17/robo-vibe/](https://hackaday.com/2010/02/17/robo-vibe/)

![](../Images/8850a4ea205b377721b6977234ba3873.png "robot-vibrophone")

听起来像性玩具的东西实际上是这些音乐家没有练习的原因。Marv 是由[蒂姆·奥基夫]、[迈克尔·麦金泰尔]和[布洛克·罗兰]制造的[MIDI 驱动的机器人电颤琴](http://www.robovibes.com)。每个键下面都有一个螺线管。这里的美妙之处在于，除了四个用于安装的小孔，电颤琴没有任何改动。螺线管被安置在仪器的外部边缘，但是还有一个隐藏的秘密。在两排钥匙之间安装了一套减震器。这些是用来停止阻尼铃声键后，音符应该已经停止。

这些家伙展示了一些美丽的工艺。休息后看看视频，如果你有机会，在 2010 年的 BarBot 上亲自见 Marv。如果你真的参加了机器人盛会，不要错过享受[吸奶器驱动鸡尾酒](http://hackaday.com/2010/02/17/drink-making-unit/)的机会。
