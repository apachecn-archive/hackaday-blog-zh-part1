# 用这些简单的制作技术制作一个间距测量仪

> 原文:[https://hack aday . com/2012/01/09/build-an-interval meter-with these-simple-fabrication-techniques/](https://hackaday.com/2012/01/09/build-an-intervalometer-with-these-simple-fabrication-techniques/)

![](../Images/58dfd54b53912393659b8388c64353b1.png "intervalometer-using-simple-fabrication-techniques")

[L]刚刚完成这个 intervalometer 的制作，他的冗长的项目文档包含了所有东西的一点点内容。制作使用普通的原型材料和简单的技巧，即使对于初学者也很容易掌握。

硬件基于 ATmega8 微控制器。在网上四处窥探后，L 想看看一些相机遥控快门控制中的基于分压器的对焦和快门命令是否适用于他的模型。对商业快门释放的研究向他展示了这是如何做到的，所以他将这一点融入到他的设计中。说到设备的固件，我们非常喜欢他对菜单系统的解释。有很多设置，他在规划用户界面方面做得很好，让它们都能在成品上工作。

原理图和电路板布局由 Eagle 完成。在布局过程中，他选择使用条形板简化组装，甚至指导我们完成切割走线和添加跳线的步骤。它在这个透明的塑料盒中完美完成，并在休息后的视频中演示。
