# 可怜人的热成像相机

> 原文：<https://hackaday.com/2009/07/01/poor-mans-thermographic-camera/>

![thermographic camera](img/e428c4455ccf4946e9c5b5ae09ffa218.png "thermographic camera")

使用红外温度计，有两种方法来制作热成像相机。第一个使用云台。当计算机控制从左到右的平移时，模拟扫描线，从每一步采集温度样本。垂直分辨率是通过倾斜实现的。另一种方法是在温度计上安装一个网络摄像头。当计算机记录视野时，温度计的激光指针与温度注释一起被捕获。我们认为最好的结果是两种方法的结合。下面嵌入的视频展示了结果。这将是对[自主彩弹哨兵](http://hackaday.com/2008/08/24/autonomous-paintball-sentry-gun/)的一个很好的补充。
