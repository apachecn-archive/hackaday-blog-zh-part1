# 为 3D 扫描构建您自己的线激光器

> 原文:[https://hack aday . com/2012/03/02/build-your-own-line-laser-for-3d-scanning/](https://hackaday.com/2012/03/02/build-your-own-line-laser-for-3d-scanning/)

![diy-line-laser](../Images/4755b0cf5c4db4936996972b35f078da.png "diy-line-laser")

[Valentin]想要对他在房子周围的一些物体进行 3D 扫描实验，但他说他不想为这个项目购买线激光器，因为它们非常昂贵。幸运的是，他的零件箱里有一些随机的组件，而且他不用花很多钱就能制造出自己的线激光器。

他的教程实际上涵盖了构建线激光器的两种不同方法，这两种方法都使用了你手头可能已经有的部件。

他的第一次建造包括将一个小方形镜子粘在一个平台上，然后将它安装在一个回收的 DC 发动机上。一旦马达开始旋转，他瞄准镜子的廉价激光指示器就会在他扫描的任何介质上画出一条完美的线。

他的二线激光使用从他不再使用的旧硬盘捐赠的部件。他从底盘上取下驱动器的读取头，在启动激光器之前，在驱动臂上安装了一面小镜子。当激光瞄准镜子时，他给马达施加了一个不确定的交流电流，使它振荡并画出一条线，类似于他的第一次设置。

虽然它们可能不是专业制造的扫描激光器，但正如你在他的网站上看到的，Valentin 的努力产生了一些不错的图像。
