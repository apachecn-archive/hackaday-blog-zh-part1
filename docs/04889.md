# Porta Touch:便携式多点触控

> 原文：<https://hackaday.com/2010/02/09/porta-touch-portable-multitouch/>

 <https://www.youtube.com/embed/lHpZ5tM7sN8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>我们刚刚发现了这个很棒的<a href="http://nuigroup.com/log/porta_touch/" target="_blank">便携式多点触控设备，叫做 portatouch </a>。该系统由 NUI 集团网站[portatouch]的一名用户制造，使用一个剥离的 LCD 作为显示器，红外 led 边缘照亮其前面的触摸表面。安装在 LCD 下方的摄像头拾取 led 的反射光，并将其转换为触摸点。虽然实现不是什么新东西，但是这个包真的很棒。如果你想学习如何设置所有的技术方面，前往<a href="http://nuigroup.com/" target="_blank"> NUI 集团网站</a>，你会找到你想要的一切。我们很想看到他的装备更详细的分解。便携性和快速构造非常棒，似乎不需要大量定制工作就可以复制。</p> </body> </html>
