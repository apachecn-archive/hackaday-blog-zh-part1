# USB 加速度计控制器

> 原文：<https://hackaday.com/2009/11/20/usb-accelerometer-controller/>

[https://www.youtube.com/embed/szZv2YYQm0U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/szZv2YYQm0U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

正如你在上面看到的，那个配件方向盘里没有 wiimote。取而代之的是一个自制的加速度控制器，通过 USB 连接到电脑。基于 PIC 18F2550 和 2 轴加速度计，该设备被 windows 检测为标准控制器。原理图和源代码可以在他的网站上找到。他说它也可以被用作“运动鼠标”。休息之后你可以看到一段视频。

当我们第一次看到视频时，我们认为它可能与加速度计控制的迷宫项目是同一个人，因为 wiimote 方向盘外壳。
