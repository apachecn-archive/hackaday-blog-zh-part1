# SNES 烤面包机

> 原文:[https://hackaday.com/2009/03/22/snes-toaster/](https://hackaday.com/2009/03/22/snes-toaster/)

![nintoaster](../Images/c24fa1f9dc8c9cedd33d3dc6993c452c.png "nintoaster")

从带给你 [NES 烤面包机](http://www.stupidfingers.com/projects/nintoaster/ "STUPIDFINGERS - Projects (Nintoaster)")的同一个人那里，诞生了[超级烤面包机](http://forums.benheck.com/viewtopic.php?f=18&t=28971 "benheck.com Forums • Information")。看起来最困难的部分是扩展盒式磁带连接器。滑动按钮相当于电源按钮。烤面包控制现在改变发光的红色发光二极管的亮度。下面嵌入视频介绍。
