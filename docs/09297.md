# 鲁道夫·托伊被黑宣布收到电子邮件

> 原文:[https://hack aday . com/2011/12/14/rudolph-toy-hacked-to-announce-incoming-email/](https://hackaday.com/2011/12/14/rudolph-toy-hacked-to-announce-incoming-email/)

![email-reading-reindeer](../Images/1571dd255e512ed0b867e0800a2224b6.png "email-reading-reindeer")

现在是黑客的季节，GadgetGangster 的[Nick McClanahan]当然是在用他最近的作品展示他的圣诞精神。他有一个电子动画红鼻子驯鹿玩具鲁道夫，觉得把它变成电子邮件阅读机器会很有趣。

他撕开玩具，取出内部零件，从原来的 PCB 上拆下内置扬声器和伺服系统。然后，在重新组装玩具之前，他从身体外部的这些组件中伸出电线。驯鹿主要使用螺旋桨平台进行控制，带有一个 E-Net 模块和一个小型音频放大器，分别负责网络通信和音频输出。

大部分工作都是由[尼克]正在使用的软件完成的，该软件允许鲁道夫定期检查他的 Gmail 收件箱，看是否有新邮件。当信息数量增加时，驯鹿开始行动，在宣布发件人的名字之前移动并点亮它的鼻子。

他使用一个音素语音合成器来输出，这就完成了工作，尽管如果我们不得不整天听它，我们会发疯的。由于驯鹿连接到他的局域网，在个人电脑上通过一个更强大的语音合成器运行数据，返回一个更好的声音片段进行回放可能是可行的。
