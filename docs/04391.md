# 敬畏:机器人墙

> 原文：<https://hackaday.com/2009/11/09/awe-a-robotic-wall/>

[https://www.youtube.com/embed/CaMuS3wZ-bM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/CaMuS3wZ-bM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[AWE 是一个有趣的项目](http://www.aweproject.org/)，你办公室的墙壁是一个有用的机器人。反正就是这个目标。这堵墙是铰接的，可以根据你的需要重新设计形状。你可以在他们的视频中看到，他们想出了 AWE 在不同位置的几种具体用法。我们想要敬畏，我们看到那里有潜力。视频并没有赢得我们的心，只是没有像投影仪支架这样的简单设置带来足够的额外好处。我们认为真正的好处并不明显。当我们看到女孩站了起来，墙壁直觉地后退时，我们的注意力又回来了。你们觉得它有什么潜在用途？
