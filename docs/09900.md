# 一个更好的防尘裙为您的数控轧机

> 原文:[https://hack aday . com/2012/03/14/a-better-dust-skirt-for-your-CNC-mill/](https://hackaday.com/2012/03/14/a-better-dust-skirt-for-your-cnc-mill/)

![](../Images/8cdc7af994a71c64a6b0ea08233f9be7.png "a-better-cnc-dust-skirt")

[Joshendy]想更好地看看他的数控铣床上的切割头。问题是，旋转的叶片会抛出很多你不想在商店里乱飞的垃圾，所以它们通常被连接到商店吸尘器的护罩包围着。他刚才的[胎具是自己的透明防尘裙](http://www.youtube.com/watch?v=0YHAqdXDTdM)解决的问题。

最初的防尘罩使用黑色鬃毛刷来容纳切割产生的废物。除了模糊了你对刀具的看法，这并没有很好地包含碎片。右图所示的解决方案使用透明、柔韧的 PVC 作为裙边。休息后的视频详细介绍了构建过程。[Joshendy]切下一个替换板，然后安装磁铁连接到切割器上。裙部用一系列螺钉固定在那个板上，这样一旦磨损就很容易更换。
