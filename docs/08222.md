# 谁知道 Thinkpad 电池需要跨接启动？

> 原文:[https://hack aday . com/2011/08/01/who-know-ThinkPad-batteries-require-a-jump-start/](https://hackaday.com/2011/08/01/who-knew-thinkpad-batteries-require-a-jump-start/)

锂电池组到了使用寿命，通常还会有很大的剩余电量。这是因为它们由多个电池单元组成，只要其中一个电池单元出现故障，整个电池就会瘫痪。我们听过的最有趣的例子之一是丰田普锐斯，但这是另一个故事了。在这种情况下，[Mika]想要从他的 IBM Thinkpad T40 中复活电池。他找到了有问题的电池并更换了它，但在修复后无法从电池中获得任何电量。

他在输出上测量 0V，但是可以测量电池而不是控制电路，并且得到超过 11V。显然，控制电路不允许输出。我们完全理解这里的概念(想想关于笔记本电脑电池爆炸的真正负面报道)。当控制电路断电时，似乎有一个锁定机制。[Mika]设法通过将电压短路到控制电路中来克服这一点，这是他在休息后的视频中喜欢的方法来启动汽车。
