# Arduino Shield 包括除厨房水槽以外的所有东西

> 原文:[https://hack aday . com/2012/03/21/arduino-shield-include-everything but-the-kitchen-sink/](https://hackaday.com/2012/03/21/arduino-shield-includes-everything-but-the-kitchen-sink/)

![arduino-3g-shield](../Images/0e1d5b0f6f023537b0f89ca370691818.png "arduino-3g-shield")

萨拉戈萨 Cooking Hacks 的工作人员来信分享了他们为 Arduino 平台开发的新盾。有 100 种不同的屏蔽可以做这做那，但是我们认为他们的 3G/GPRS 屏蔽看起来很巧妙。

base shield 包含一个兼容 WCDMA 和 HSPA 的 3G 调制解调器以及一个 GPS 接收器。该调制解调器可以与 Arduino 配合使用，或者通过 USB 直接插入电脑，让它以您喜欢的任何方式使用。由 Cooking Hacks crew 提供的更高级的套件具有 VGA 摄像机，以及可以播放和录制音频的音频套件。

他们很快指出的一个简洁的特性是，并不是所有的网络通信都需要由 Arduino 来处理。简单的事务(HTTP/FTP/POP3/SMTP)都可以通过向调制解调器发送标准 AT 命令来执行。

现在，这些套件并不便宜，但我们不禁有兴趣看看人们在这个盾牌的指导下梦想出什么样的应用程序。烹饪黑客是 Libelium Communcations 的一个部门，但由于他们是该公司的“开放硬件部门”，如果你有兴趣转而使用自己的 3G 调制解调器，可以使用图表。
