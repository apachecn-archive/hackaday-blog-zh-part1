# 春分钟

> 原文:[https://hackaday.com/2010/09/07/equinox-clock/](https://hackaday.com/2010/09/07/equinox-clock/)

![](../Images/627a9764721cad2b935d05bb128f71b0.png "equinox-clock")

春分钟由简单的零件组成，但精细的设计和精确的结合使它成为钟表中的瑰宝。内部由一个 Arduino、一个 DS1307 实时时钟、12 个 LED 驱动器和 60 个 RGB LEDs 组成。这些与电容式触摸界面相结合，使用三个发光块来显示时间，一个用于显示分钟，一个用于显示秒。休息之后自己看吧。

让我们高兴的是，[Bram Knaapen]分享了案件的细节。构成车身的黑环是激光切割和喷漆的。他用经过喷砂处理的小块丙烯酸树脂来漫射光线。这也是利用不同电路板之间的互连实现清洁电路的一个很好的例子。

我们总是喜欢看钟表[,不管它的完成程度如何](http://hackaday.com/2010/06/22/morse-code-clock/),但是伟大的设计让我们想把一个项目挂在墙上，而不是放在零件箱里。
