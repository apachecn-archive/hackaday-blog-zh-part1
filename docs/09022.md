# 宣布我们的下一个主题:厨房黑客

> 原文:[https://hackaday . com/2011/11/07/announcing-our-next-theme-kitchen-hacks/](https://hackaday.com/2011/11/07/announcing-our-next-theme-kitchen-hacks/)

![](../Images/16574d783e81e3cfded6e5f180bf806f.png "Theme banner Kitchen Hacks")

这是一个让 Hackaday 员工非常兴奋的主题；厨房黑客。这是一个广泛的主题，可以包括任何与食物、食物准备、厨房用具和增强功能有关的东西。

我们已经看到了很多关于这个主题的精彩例子。一个混合鸡尾酒的厨房岛怎么样？也许你更喜欢[一个跟踪你购物清单的人工智能](http://hackaday.com/2010/05/24/build-hal-into-your-kitchen/)或者仅仅是[一个集成得很好的电脑亭](http://hackaday.com/2010/01/23/iphone-look-alike-on-your-kitchen-wall/)。当然，已经有一个 iPod 基座可以满足这个需求了！你可能正在寻找一个高压锅，或者在你的烹饪库中添加一个真空吸尘器。我们不应该遗漏[在吐司上“打印”图像的能力](http://hackaday.com/2009/06/23/the-holy-toaster-kit/)。

说到烤面包，我们仍在等待有人能造出一台激光面包切片机，边切边烤面包。你明白了。女士们，先生们，拿起电脑，记录下你的厨房秘籍，然后[发送给我们](http://hackaday.com/contact-hack-a-day/)！
