# 双离合器变速器模型

> 原文：<https://hackaday.com/2010/01/04/double-clutch-transmission-model/>

![](img/6a58dec6a4590b2c3aa4101e2a2e7edd.png "model-gearbox")

[艾伦]做了一项非凡的工作，制造了一台电脑控制的模型变速箱。他几年前的项目是基于为大众和奥迪汽车开发的双离合器直接换档变速箱。他的设计使用了一个齿轮头马达来为这个传动装置提供动力。换档由计算机通过串行电缆控制，伺服电机提供物理运动来换档。在休息后看到剪辑中所有这些移动的部分可能会让你有点头晕。

这是一些极端的模型建设。这让我们想起了那个在 12 月份风靡互联网的建造了铝制飞机模型的家伙。
