# NES 墨盒中的仿真器–如此干净，看起来像是工厂制造的

> 原文：<https://hackaday.com/2009/10/12/emulator-in-nes-cartridge-so-clean-it-looks-factory-made/>

![nesP_NES-emulator-in-game-cartridge](img/867618bb5c203a9c1b9e68da44b80c8e.png "nesP_NES-emulator-in-game-cartridge")

我们在本·赫克论坛上向[airz]表示祝贺。他[组装了一个 mod，将一个模拟器安装到一个原始的 NES 盒式磁带](http://www.forums.benheck.com/viewtopic.php?f=18&t=33844)中，并使用了一个被销毁的原始 NES 控制器；他干得非常出色！

他使用的是便宜但功能齐全的仿真器板。它配有 4g 内存，但也有一个 SD 卡插槽。NES、Game Boy 和 Game Boy 彩色光盘都可以在 2.8 英寸彩色液晶显示器上播放，但该系统还具有一个电视输出连接器，可用于更大的屏幕。

[airz]在外壳上的切割非常惊人，轻而易举地超过了我们看到的最后一款[卡式模拟器 mod](http://hackaday.com/2008/07/30/nintendo-entertainment-system-hacks/) 。控制装置的孔看起来就像塑料是这样模制的。为了逼真，他还切掉了墨盒商业端的 PCB 接口，并用胶水粘好。显然，它需要三个墨盒，两个控制器，和两个模拟器，使它这么好，但如果你想做一个煎蛋卷…
