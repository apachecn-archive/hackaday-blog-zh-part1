# 设计出一只狡猾的猫

> 原文:[https://hack aday . com/2011/01/31/out-engineering-a-sneaky-cat/](https://hackaday.com/2011/01/31/out-engineering-a-sneaky-cat/)

![](../Images/d56a1b5cfbaaea273371aab34fc5d3cd.png "CatBuzzer")

猫做自己想做的事情，很少和主人想让它们做的事情不谋而合。在[杜米特鲁]的案例中，他女朋友的猫[普夫]比预期的更频繁地出现在寒冷的户外。他决定组装一个定制的寻猫项圈，而不是满足于普通的铃铛，因为即使猫没有行为不端，铃铛也会变得令人讨厌。他使用 PIC 微控制器作为大脑，并使用温度和光线传感器来判断猫是否溜进了寒冷的黑夜。一旦猫被标记为在外面，蜂鸣器和 LED 就会定时响起，直到它回到安全的室内。

[Dumitru]的网站和他的 YouTube 视频都是罗马尼亚语的，尽管提供的图表和源代码说明了一切。他出色地完成了整个设计过程，包括花在 IDE 和 EAGLE 设计电路板上的时间。YouTube 已经设法为大部分细节添加了字幕，但是我们认为这篇文章将会是对任何罗马尼亚语爱好者的一次真正的款待。请务必在休息后观看这两个视频。
