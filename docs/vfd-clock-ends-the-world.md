# VFD 时钟(结束世界)

> 原文：<https://hackaday.com/2009/12/26/vfd-clock-ends-the-world/>

![](img/a8c025d0807c255996b5f41256f6040e.png "4210430248_530fd9fd0b")

当我们看到 RSS 时，我们真的认为[[Jason]的 VFD 时钟](http://www.vonnieda.org/archives/1406)是某种形式的新恐怖袭击。谢天谢地，我们与俄国的关系不像以前那么疯狂了(T2)。

主要组件是一个带有 MAX6921 驱动器的 IV-18 VFD，对于未经训练的人来说，这听起来令人惊讶地具有威胁性。然而，Arduino 让我们的心平静下来，并向我们保证这只有闪烁 VFD 的潜力。虽然主要代码、原理图和 CAD 目前还不可用(开源即将在你附近的影院推出)——你可以看看[杰森]的灵感,[冰管钟，它运行许多相同的组件。](http://www.ladyada.net/make/icetube/make.html)

休息之后，请欣赏它的视频。我们最喜欢“倒计时”功能。
