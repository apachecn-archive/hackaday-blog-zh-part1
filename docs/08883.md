# 盒子里的电子动画

> 原文:[https://hackaday.com/2011/10/20/animatronics-in-a-box/](https://hackaday.com/2011/10/20/animatronics-in-a-box/)

![](../Images/85b1a52f15cae865f46df706f5bf1c98.png "animatronics-to-go")

[Knife141]建造了一个令人印象深刻的电子头。他主要将它用于志愿者活动，比如让学生对技术感到兴奋。他制作了一个刚好适合木偶的行李箱，便于存放和运输。

他开始用硬纸板制作头部的零件，以确保它们能够装配在一起并正常工作。然后这些被用作模板，从半英寸厚的胶合板上切割下来。一系列直接连接或通过连杆连接的伺服系统通过伺服控制板移动嘴、眼睛、眼睑、眉毛和脖子。

声音由一台叫做 RAPU 的单板计算机通过一对计算机扬声器播放。该板也是向伺服控制器发送命令的部件。当[Knife141]想要为电子动画创造一种新的行为时，他首先编写对话，然后让一个文本到语音转换程序将它转换成 MP3。然后，他经历了为对话编排木偶的乏味过程，这个过程通常每运行一分钟就要花费他一个小时。虽然这是值得的，但休息之后，你可以通过观看他在视频中的一段表演来亲眼看看。
