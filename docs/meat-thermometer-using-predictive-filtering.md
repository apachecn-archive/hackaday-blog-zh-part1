# 使用预测滤波的肉类温度计

> 原文：<https://hackaday.com/2009/06/17/meat-thermometer-using-predictive-filtering/>

![meatthermoafteradjustment](img/82e08b2036fbeeb90b95cab982621dc8.png "meatthermoafteradjustment")

NerdKits 的家伙们用预测滤波技术在肉类温度计上制作了一个非常有用的视频，如下图所示。视频、[补充文本和代码](http://www.nerdkits.com/videos/meat_thermometer/)可在他们的网站上获得。温度计由一个 [LM34 温度传感器](http://www.national.com/mpf/LM/LM34.html)组成，连接在一根 12 号实心铜线上。温度计信号在 ATmega168 微控制器上处理，并使用 python 的 pygame 库可视化。该项目的真正亮点是他们对预测过滤的出色解释，这可以很容易地用于大量的项目。
