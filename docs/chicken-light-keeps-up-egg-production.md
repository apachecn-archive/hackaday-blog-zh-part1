# 鸡光保持鸡蛋产量

> 原文：<https://hackaday.com/2011/11/13/chicken-light-keeps-up-egg-production/>

事实证明，随着白天变短，鸡下蛋越来越少。但是你可以用人造光欺骗他们继续生产。[Jpitz31]决定[建造他自己的定时鸡舍灯](http://www.joestechblog.com/2011/11/13/chicken-light-relay-timer/)来弥补缺口，直到阳光充足的日子回来。

他已经有了一个 LED 野营灯，但需要找到一种方法来给它供电，并按照时间表开关它。他选择了 ATmega328 作为后者，因为他有一堆多余的东西。至于电力，鸡舍所在的地方没有交流电，所以他选择了 12V 的铅酸电池，希望未来能增加太阳能充电功能。

开关由继电器控制，由 DS1307 实时时钟保持精确时间(就是上面看到的红色 PCB)。主板上的一切都很好，我们有几个未来改进的功能建议。线性调节器会消耗一些额外的功率，因此改用开关调节器将有助于节省电量。此外，添加光传感器也非常容易，这样只有当环境光下降到预设水平时，灯才会打开。这样他就不需要随着白天长度的变化而打乱时间表。
