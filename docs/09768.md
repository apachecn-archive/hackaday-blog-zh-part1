# 为您的座驾增加脚坑灯和手套箱灯

> 原文:[https://hack aday . com/2012/02/26/add-the-foot well-and-the-glove box-lights-to-your-ride/](https://hackaday.com/2012/02/26/adding-footwell-and-glove-box-lights-to-your-ride/)

![](../Images/e2b636ee86b538617caf7749e235ddad.png "footwell-and-glove-box-illumination")

[KonaStar]向我们展示了为您的汽车内部增加一些灯光并不难。只需找到一些未使用的空间，将线缆布线到看不见的地方。在这里，他为他的汽车的搁脚空间和手套箱添加了 LED 照明。

他设法在汽车的模压仪表板上找到了一些凹陷，大小正好适合一个小的四引线 LED。他为这些导线钻孔，并在仪表板内侧焊接一些原型板。这样就不会有什么未完成的东西吸引你的眼球，而且原型板提供了一个区域来放置电阻和互连。

手套箱里有开关，关闭时可以关灯。这些也适用于脚部空间。此外，LED 照明线束连接到停车灯上，因此当车灯关闭时，它们不会亮起。因为他是从一个用来给车辆增加环境光线的照明线束开始的，所以这个系统也会对车门的开启做出反应。

如果你不介意在安装过程中取出几块内饰，这是一个不错的补充。我们更喜欢类似于[这个转向灯黑客](http://hackaday.com/2011/10/03/animated-turn-signals-add-a-little-bit-of-cylon-to-your-automobile/)的东西。
