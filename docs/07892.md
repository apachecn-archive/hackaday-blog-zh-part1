# 锁相环 101

> 原文:[https://hackaday.com/2011/06/21/pll-101/](https://hackaday.com/2011/06/21/pll-101/)

[Jeri Ellsworth]和前 Commodore 计算机工程师和现任全职修补匠[Bil Herd]在 skype 上聊了聊这个一小时长的视频中的 101 个锁相环。PLL 对于许多应用都很方便，但它们的基本用途是保持时钟信号同步。

涵盖的主题包括:我们为什么关心，CD4046 的基本解释，捕获范围和元稳定性。从模拟电视到时钟恢复、寻找下降沿和 FPGA 的例子。这东西塞满了信息。

随着未来剧集的讨论和[Bil]长凳的快速游览，这是不容错过的。广告之后加入我们的视频吧！
