# 硬盘 RGB 时钟

> 原文:[https://hackaday.com/2009/09/07/hard-drive-rgb-clock/](https://hackaday.com/2009/09/07/hard-drive-rgb-clock/)

[https://www.youtube.com/embed/XK9en2H7rBA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/XK9en2H7rBA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们在过去给你带来了一个硬盘时钟，但是[mb1988]试图通过完整记录(用波兰语)把[的项目带到另一个层次。受](http://www.elektroda.pl/rtvforum/viewtopic.php?p=6984995)[【dzgdzzh】的版本](http://www.youtube.com/watch?v=3zh64ndyHxg)相同发明的启发，【mb1988】决定对其进行逆向工程(以及做一些改动)并基于[强大的 ATmega128](http://www.atmel.com/dyn/resources/prod_documents/2467S.pdf) 。因为这个时钟和大多数硬盘时钟一样，只能显示半径实线，所以不能用来显示文本。但是，可以使用遥控器切换配色方案和动画。住房本身也令人印象深刻。外壳的后部是用 Rayobond 模制而成的[，而前部只是原始的 HDD 外壳，但喷涂了黑色并蚀刻了一些原始的艺术品。源代码、原理图和 PCB 布局可以在项目的线程上免费下载，但是你必须](http://www.alcoa.com/aap/north_america/en/home.asp)[登录论坛](http://www.bugmenot.com/view/elektroda.pl)才能访问它们。
