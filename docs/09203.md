# 用计算器制作美妙的软盘音乐

> 原文:[https://hack aday . com/2011/12/01/making-sweet-floppy-drive-music-with-a-calculator/](https://hackaday.com/2011/12/01/making-sweet-floppy-drive-music-with-a-calculator/)

![floppy-music-ti83-style](../Images/59c443e1c8406827f401ac373dd6307b.png "floppy-music-ti83-style")

[Chris]说他最近非常忙，几乎没有时间进行黑客攻击。然而，他确实设法找到了一点时间来完成一个小项目，这个项目已经占据了他的待办事项清单一段时间——一个软驱音乐控制器。

我们以前见过使用微控制器来驱动软驱马达的[黑客，但是我们不记得有任何使用计算器来代替的。虽然微控制器给你提供了大量的 I/O 引脚，但[Chris'] Ti-83+只有两个。](http://hackaday.com/2011/10/02/star-wars-imperial-march-played-by-dual-floppy-drives/)

即使有计算器的 I/O 限制，他也不觉得这项任务太难，因为他只需要将驱动器的一对引脚保持在低电平，同时给另外两个引脚施加脉冲。他修改了为 Ti 计算器编写的媒体播放器，以输出必要的控制信号，然后他弹奏了一些曲子。

正如你在下面的视频中看到的，他的简单设置工作得很好——对于仅仅几个小时的工作来说还不错。
