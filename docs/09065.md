# Arduino 的起源

> 原文:[https://hackaday.com/2011/11/15/the-origin-of-the-arduino/](https://hackaday.com/2011/11/15/the-origin-of-the-arduino/)

[![](../Images/68eea82b063f5ea3d4194fe589557e47.png "arduino03-1319574149091")T2】](http://hackaday.com/2011/11/15/the-origin-of-the-arduino/arduino03-1319574149091/)

如果你想知道 Arduino 是如何形成的，可以看看[IEEE Spectrum]的文章，标题是“[Arduino 的形成](http://spectrum.ieee.org/geek-life/hands-on/the-making-of-arduino/0)”Arduino 起源于意大利北部，正如[HAD]上展示的大量项目所示，它已经成为 DIY 处理能力的首选处理器。它的成本(大约 30 美元)和易用性是它获得如此巨大成功的最大因素。

许多人对 Arduino 感兴趣的一点是，它是完全开源的，在[知识共享许可](http://creativecommons.org/)下获得许可。这本身就很有创新性，因为 CCL 通常适用于音乐和写作等艺术作品。尽管[Banzi]和他的团队决定放弃这一设计，但这种主板已经在全球售出超过 25 万块，还不包括它们的许多克隆产品。

如果你想知道它的名字“Arduino”是怎么来的，它是以意大利北部城镇伊夫雷亚一家名为“热地·阿尔杜伊诺酒吧”的酒吧命名的。
