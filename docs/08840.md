# 用振动器护罩加入 Pen15 俱乐部

> 原文：<https://hackaday.com/2011/10/16/join-the-pen15-club-with-a-vibrator-shield/>

![](img/b5b697f8ad1eb59e1f034833ebb18906.png "wingnut")

对于所有的电视爱好者，Arduino 有一个新的[振动器护罩](http://slashdong.org/2011/10/15/pen15-board/)。更好的是:你可以将 Pen15 shield 与 Kinect 配合使用，在家中享受健康自然的乐趣。

体面和“安全工作”的风格阻止我们把我们知道的一切放在头版，所以休息后继续阅读。

我们就直接跳到这一个。 [Pen15 板](http://slashdong.org/2011/10/15/pen15-board/)是一个 Arduino 兼容的盾牌，“专为性技术和[远程控制](http://en.wikipedia.org/wiki/Teledildonics)研究而造。”电路板非常简单，只需要一个 MOSFET、几个电阻和一个插孔来连接振动器。Pen15 板是《slashdong.org[的作者【qDot】多年渴望的结果。](http://slashdong.org/)

下周，[qDot]将在卡耐基梅隆大学的 Art & & Code 3D 举办一场研讨会，在那里他将把 Pen15 和 Kinect 进行混搭。我们不知道这需要什么，但我们很确定这看起来会像《T2》中剪草人的一个场景。

当然，Pen15 shield 在没有 Kinect 的情况下完全可以使用。[qDot]写了 [Deldo](http://www.youtube.com/watch?v=D1sXuHnf_lo) ，Emacs 的振动控制和遥控模式。是的，你现在可以和 Emacs 做爱了。

非常感谢发送这封邮件的所有人。真的，你帮了大忙。
