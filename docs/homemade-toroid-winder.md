# 自制环形绕线机

> 原文：<https://hackaday.com/2009/06/20/homemade-toroid-winder/>

[eclipsed78]，造了一个[自动环形绕线机](http://waterfuelcell.org/phpBB2/viewtopic.php?t=1034) ( [互联网档案](http://web.archive.org/web/20090623064924/http://waterfuelcell.org/phpBB2/viewtopic.php?t=1034))。为了装载圆环，滚筒裂开。然后，电线被缠绕在卷筒上，就像缠绕其他线圈一样。当滑块将金属丝拉出卷筒时，卷筒旋转，同时旋转进出圆环。在操作过程中，侧向张力可防止钢丝松弛。为了控制匝数比，缠绕线圈随着滚筒旋转而步进。T4 根据原理图设计了一个步进驱动器，这样他就可以驱动电机。你可以通过一系列视频来观看[操作中的绕线轮。第一个嵌在下面。如果你曾经需要缠绕一个巨大的环形变压器，这个项目就是为你准备的。](http://www.youtube.com/user/eclipsed78)
