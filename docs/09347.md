# [本·克拉斯诺]在他的隐形眼镜上贴发光二极管只是为了刺激

> 原文：<https://hackaday.com/2011/12/20/ben-krasnow-sticks-leds-in-his-contacts-just-for-kicks/>

[Ben Krasnow]写道，经常会有新闻报道研究人员在隐形眼镜中嵌入单像素 LED 显示屏的项目。他最近看到的一篇文章中有一只戴隐形眼镜的兔子，他不是那种害怕[以科学的名义伤害自己身体的人](http://hackaday.com/2011/08/15/controlling-muscles-with-high-intensity-magnetic-pulses/)，他决定在自己身上做实验。

他首先将一个微小的 0402 SMD LED 焊接到手绕线圈上，将显示器层压在一对普通隐形眼镜之间。在尝试用水将镜片粘在一起后，他选择用一把热镊子将边缘钉在一起，这样做的眼镜更牢固，但不舒服。LED 由他组装的简单感应线圈供电，该线圈使用火花隙发射器来使 LED 闪烁。

如果你没有被人们往眼睛里塞东西吓到，一定要看看下面的视频，看看[本的]增强型隐形眼镜在起作用。虽然这可能不会给他像终结者一样的视觉，但考虑到他是在业余时间在工作室里拼凑的，这还是很棒的。
