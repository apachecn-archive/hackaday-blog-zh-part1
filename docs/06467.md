# 使用光线不足的设备拍摄微距照片

> 原文:[https://hack aday . com/2010/12/21/macro-photos-using-an-under-light-rig/](https://hackaday.com/2010/12/21/macro-photos-using-an-under-light-rig/)

![](../Images/183475971f9d38450b7a353112a231da.png "SAMSUNG")

[Aki Korhonen]想要加强他的微距摄影设置。他已经有了用于这项工作的相机，但他想要一个完全可调的目标平台，可以很容易地点燃。他想出的是一个夹具来固定相机并微调每张照片的主题。它使用一个带有一块玻璃的框架，玻璃与镜头的距离可以通过旋转一个旋钮来调节。在玻璃下面，一个反射面将 LED 手电筒的光线向上引导穿过平台，照亮了他拍摄的雪花。下一次演出的计划中有一个固定的 LED 光源。
