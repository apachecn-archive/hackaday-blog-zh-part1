# 加速度计控制的乒乓

> 原文:[https://hack aday . com/2008/09/23/accelerometer-controlled-pong/](https://hackaday.com/2008/09/23/accelerometer-controlled-pong/)

[https://www.youtube.com/embed/jYVgXYtxk4E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/jYVgXYtxk4E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

佩里的令人敬畏的 [AcceLED Pong](http://separaterealities.com/projects/acceled_pong/) 项目通过添加基于加速的控制赋予了经典游戏新的生命。通过向左或向右倾斜电路来移动乒乓板。运动由一个 [ADXL203](http://www.analog.com/en/mems-and-sensors/imems-accelerometers/adxl203/products/product.html) 双轴加速度计测量，一个 [ATMEGA32](http://www.atmel.com/dyn/Products/Product_card.asp?part_id=2014) 微控制器将加速度转换成球和桨的运动。游戏显示器为三色 [SparkFun 8×8 LED 矩阵](http://www.sparkfun.com/commerce/product_info.php?products_id=759)带串行接口。

[Perry]也使用类似的设置制作了一个 USB [LED 频谱分析仪](http://separaterealities.com/projects/spectrum_analyzer/)，由 Linux [XMMS 媒体播放器](http://www.xmms.org/)供电。
