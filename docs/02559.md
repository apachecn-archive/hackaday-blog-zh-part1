# 加速度计控制的乒乓

> 原文：<https://hackaday.com/2008/09/23/accelerometer-controlled-pong/>

 <https://www.youtube.com/embed/jYVgXYtxk4E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>佩里的令人敬畏的<a href="http://separaterealities.com/projects/acceled_pong/" target="_blank"> AcceLED Pong </a>项目通过添加基于加速的控制赋予了经典游戏新的生命。通过向左或向右倾斜电路来移动乒乓板。运动由一个<a href="http://www.analog.com/en/mems-and-sensors/imems-accelerometers/adxl203/products/product.html" target="_blank"> ADXL203 </a>双轴加速度计测量，一个<a href="http://www.atmel.com/dyn/Products/Product_card.asp?part_id=2014" target="_blank"> ATMEGA32 </a>微控制器将加速度转换成球和桨的运动。游戏显示器为三色<a href="http://www.sparkfun.com/commerce/product_info.php?products_id=759" target="_blank"> SparkFun 8×8 LED 矩阵</a>带串行接口。</p> <p>[Perry]也使用类似的设置制作了一个 USB <a href="http://separaterealities.com/projects/spectrum_analyzer/" target="_blank"> LED 频谱分析仪</a>，由 Linux <a href="http://www.xmms.org/" target="_blank"> XMMS 媒体播放器</a>供电。</p> </body> </html>
