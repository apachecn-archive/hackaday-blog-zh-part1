# 用旧液晶显示器制作隐私显示器

> 原文:[https://hack aday . com/2011/11/25/making-a-privacy-monitor-from-an-old-LCD/](https://hackaday.com/2011/11/25/making-a-privacy-monitor-from-an-old-lcd/)

![privacy-screen](../Images/f22f95b52d4e19037cc710731fede082.png "privacy-screen")

[dimovi]有一个备用的液晶显示器，他认为把它变成一个“隐私”显示器会很棒。

这个过程对任何喜欢拆卸电子产品的人来说都很简单。他拆开了显示器的塑料框架，用美工刀将偏光膜切掉。一旦薄膜被移除，他花了一些时间使用 Oops 清洁剂和油漆稀释剂的组合来移除玻璃面板上的薄膜粘合剂。

他重新组装了显示器，现在不管屏幕上实际显示的是什么，显示器都会发出明亮的白光。他从一副影院 3D 眼镜上取下镜片，用从显示器上取下的薄膜代替塑料。

现在，[迪莫维]是唯一一个可以看到他在电脑上做什么的人，这正是他喜欢的方式。

虽然这个过程背后没有太多的魔法，但我们认为这是重用旧显示器的好方法。
