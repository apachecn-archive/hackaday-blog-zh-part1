# 基于 Arduino 的 EATC 替代品

> 原文:[https://hack aday . com/2010/05/19/arduino-based-eatc-replacement/](https://hackaday.com/2010/05/19/arduino-based-eatc-replacement/)

![](../Images/325b89c4f6531fef38b8f387508cc859.png "eatc-arduino-replacement")

[丹·马托克斯的] 2000 辆福特·金牛座汽车上的电子自动温度控制模块坏了。该地区的垃圾场没有匹配的替代品，新的也很难得到，所以他用 Arduino Mega 造了一辆 EATC 替代品。它包括一个电磁控制板，用于通风选择器、鼓风机控制和控制电动车窗的新开关。他让系统启动并运行，这很重要，因为在移除了坏掉的 EATC 后，赛车被卡在全速吹 90 度的空气中。他整理了一个演示和一个安装视频，我们在休息后嵌入了这个视频，但是还有[一个相册](http://picasaweb.google.com/114351322941159318025/CID#)你可以翻阅。我们开发的控制系统的草图[在 pastebin](http://hevymetal87.pastebin.com/dKs8vKhp) 上，所以趁热拿吧。
