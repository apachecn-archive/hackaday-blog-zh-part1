# 定制自行车的 3D 打印凸耳

> 原文:[https://hack aday . com/2012/02/24/3d-printed-lugs-for-your-custom-bike/](https://hackaday.com/2012/02/24/3d-printed-lugs-for-your-custom-bike/)

![](../Images/abfe641e3b25a26c31b9be8dcefa95a3.png "3d-printed-lugs")

我们还没有听到太多关于使用不锈钢作为介质的 3D 打印，但这正是 t [he 用于组装自行车架](http://vimeo.com/34293503)的凸耳所发生的事情。它们是使用激光加工制造的，激光加工是使用[选择性激光熔化](http://en.wikipedia.org/wiki/Selective_laser_melting)生产的零件的商标名称。休息后的视频将向您概述如何清理这些部分。

激光熔化金属粉末以固化最终零件所需的区域。就像我们在 RepRap 或 Makerbot 上看到的爱好印刷一样，完成印刷工作需要结构支撑，这些需要在激光完成工作后移除。这是大部分劳动力进入的地方。你会看到一吨的废料从笼状的吊耳中拉出，我们肯定不会缺少锉磨和抛光来完成。但是哇，多么有趣的结果。我们只需要弄清楚是否有人已经找到了一种划算的方法来组装这些金属粉末打印机。
