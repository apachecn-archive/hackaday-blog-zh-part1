# 瓶子火箭 POV 视频

> 原文:[https://hackaday.com/2011/07/03/bottle-rocket-pov-video/](https://hackaday.com/2011/07/03/bottle-rocket-pov-video/)

![pov_fireworks_video](../Images/a7625782255a26397fa5cddcc71b42db.png "pov_fireworks_video")

这是一个假日周末，和你一样，我们也需要一点时间来放松，在与朋友和家人聚会时喝几杯。显然，本周末一个更大的事件是你的城市或你喝醉的邻居举办的烟火表演。

撇开罗马蜡烛战争不谈，你有没有想过从烟火的角度看 7 月 4 日是什么样子？我们做了，[，【Jeremiah Warren】](http://laughingsquid.com/videos-shot-from-cameras-mounted-on-the-tips-of-fireworks/)也做了，他制作了一个很棒的视频，展示了当你点燃导火线，像小女孩一样跑开后会发生什么。

这段令人眼花缭乱的视频是在火箭发射前，他用一对钥匙链相机直接绑在火箭上拍摄的。它非常有趣，所以如果你有几分钟的空闲时间，一定要去看看。

这可能不太符合被认为是黑客的标准，但由于爆炸和疯狂的观点视频，我们不得不传递它。
