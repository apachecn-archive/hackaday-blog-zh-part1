# 史诗木制大理石运行为儿童的房间

> 原文：<https://hackaday.com/2010/11/27/epic-wooden-marble-run-for-kids-room/>

一些父母在艺术或工艺博览会上为他们的孩子购买活动雕塑。不是史蒂夫·莫斯利，他把他孩子的小屋变成了一个雕塑，他用大理石环绕整个房间。它足够大，有如此多的功能，以至于找到一个横幅图像有点困难。休息后，我们嵌入了一个视频，你会看到一个货车轮升降机，大量的重力馈送曲线，循环，倾斜，摇杆，楼梯升降机，和…好了，你得到了想法。

考虑到这个项目的规模，它非常便宜；木头大约 70 美元，玻璃弹珠 40 美元，其他东西大约 60 美元。我们很高兴他如此详细地分享了他的建筑方法。你需要一个库存充足的商店。像这样的精细工作需要木工常用的工具，但我们敢打赌带锯和摆动主轴砂光机是天赐之物。
