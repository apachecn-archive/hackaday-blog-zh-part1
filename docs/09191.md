# 用锂聚合物替代工具电池的利弊

> 原文:[https://hackaday . com/2011/11/29/用锂聚合物替代工具电池的利弊/](https://hackaday.com/2011/11/29/pros-and-cons-of-replacing-tool-batteries-with-lithium-polymer/)

![](../Images/6d49969b9e8bfb4874f07d9066606095.png "lithium-polymer-battery-replacement")

[HammyDude]厌倦了为他的电动工具购买替换电池。他手头上有一些锂聚合物电池，并决定拿走他的一个报废的电钻，并把报废的电源组换掉。

你在上面看到的橙色电池组上有一个 deans 连接器，用于遥控车辆。通过打开钻机外壳，[HammyDude]能够添加配对的 deans 连接器。现在，这种替代品很容易插入钻头，甚至可以放入手柄体中。

这种电池由几节电池组成，一个便宜的充电器可以单独给每节电池充电，以实现平衡充电。在休息后的视频中，[HammyDude]指出 Makita 充电器将电压施加到所有串联的电池上。它不能平衡充电，所以当一个电池没电时，电池就完蛋了。我们以前在 Makita tools 中遇到过[这个问题。](http://hackaday.com/2010/01/12/makita-battery-pack-repair/)

在视频的最后要注意的一个缺点是:这种替代没有任何低电压截止。电池电量过低会永久损坏电池。必须有一个简单的电路可以作为安全措施。如果你知道其中一个，[给我们一个提示](http://hackaday.com/contact-hack-a-day/)。
