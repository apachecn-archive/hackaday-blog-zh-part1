# 乐高摆钟

> 原文:[https://hackaday.com/2011/03/04/lego-pendulum-clock/](https://hackaday.com/2011/03/04/lego-pendulum-clock/)

![](../Images/207e24fd276492c2af607da30273a2a1.png "lego-pendulum-clock")

在它周围放一个盒子，它将是一个落地钟，但现在它是一个由乐高积木制成的摆钟。休息后的视频展示了一个很好的构建概述。你可以从几个不同的角度看到它的工作原理，也可以看到一个加速显示重量随时间变化的片段。一个重物，由废 AA 电池制成，驱动时钟，另一个重物开关上弦马达。当驱动装置的重量达到其绳索的末端时，该马达会自动给时钟上弦。

这是一个很好的偏离，从我们看到的大多数时钟项目，因为它利用机械概念，而不是电子。[Pmroskelly]的大部分构建细节都是作为 Picasa 相册上的评论分享的，可在上面的链接中找到。还有其他一些视频，比如展示擒纵机构工作原理的视频。
