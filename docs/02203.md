# 我放弃蓝牙手机

> 原文：<https://hackaday.com/2008/07/12/igiveup-bluetooth-handset/>

![](img/085747a7dbecbfd7f13c55891fdae908.png)
Instructables 用户【ManaEnergyPotion】贴出了一个相当幽默的[蓝牙手机黑客](http://www.instructables.com/id/Bluetooth-Handgun-Handset-for-your-iPhone-iGiveUp/)。他只是简单地拆开了一把气枪和一个蓝牙耳机，然后将这些部件整齐地放在盒子里。耳机实际上在枪膛里，而麦克风在手柄里。你扣动扳机接听电话，或者结束通话。最棒的是，他们把这个带到了 iPhone 产品发布会上，并发布了人们对此的反应视频，作为实际的产品概念。休息之后你可以去看看。
