# 555 视频游戏

> 原文:[https://hackaday.com/2011/02/24/555-video-game/](https://hackaday.com/2011/02/24/555-video-game/)

![](../Images/6daf506e870a7237eb313944841e97a4.png "555game")

[刷希巴]又回到了一些 555 定时器疯狂，这一次是他的那种[躲避球视频游戏](http://kaput.retroarchive.org/555game/)。以 23 个 555 定时器、运算放大器(LM324)和 atari 拨片为特色，稍加修改，使电位计充当分压器。

输出是单色复合电视信号，游戏的玩法类似于躲避球游戏，目的是避开球并干掉对方。也可以发射自己的球，偏转迎面而来的攻击。当被击中时，被击败的玩家从屏幕上消失 3 秒，然后另一轮开始。

就我个人而言，这个项目切中要害，因为我计划的项目是一个基于 555 的 pong 游戏，在截止日期前几天，我只有一个不稳定的同步发生器，所以看到一些接近我想象的东西，并工作，让我微笑，对我的精神状态感觉好一点。

休息之后，看看一个短视频，以及[刷希巴]之前的 555 项目， [Synthanola](http://hackaday.com/2011/02/16/punch-tape-musical-synth/) 。
