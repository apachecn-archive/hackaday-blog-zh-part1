# Debian 再次出现在 G1

> 原文：<https://hackaday.com/2009/01/15/debian-on-the-g1-once-again/>

![g1](img/722353fb53805c388dd1df8afb83f247.png "g1")

[ghostwalker]顺便访问了我们之前的 Debian Android 帖子，让我们知道他已经[简化了安装过程](http://www.androidfanatic.com/cms/community-forums.html?func=view&catid=9&id=251#251 "Debian on Android installer released. - AndroidFanatic Community Forums")。第一次，很快就很难完成这个过程，因为固件更新已经取消了 root 访问权限。黑客们已经想出了如何从 RC30 降级[并](http://androidfanatic.com/cms/community-forums.html?func=view&id=219&catid=9 "Rooting Procedure for RC30 (US)! - AndroidFanatic Community Forums")[安装 BusyBox](http://androidfanatic.com/cms/community-forums.html?func=view&catid=9&id=228 "Got root? Need more? Install Busybox! - AndroidFanatic Community Forums") 。你需要做的就是从[ghostwalker]下载这个包，然后运行安装脚本。这在技术上不是一个端口，因为 Debian 已经支持 ARM EABI。如果你可以访问整个 Debian 软件包树，你会在你的手机上运行什么？下面嵌入了 Debian 启动的视频。
