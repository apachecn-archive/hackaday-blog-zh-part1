# 用垃圾做的激光螺旋浆

> 原文：<https://hackaday.com/2009/09/13/laser-spiro-made-from-junk/>

![P6080017](img/ad3ba7770e4e84b2d0b253bdf3cddfc3.png "P6080017")

有时，我们需要像[FireMyLaser]这样的人来告诉我们，你真的不需要定制 PCB、昂贵的工具或激光切割机等东西来进行黑客攻击。他的激光肺活量主要依靠牙签、热熔胶和决心。这是一个双电机螺旋浆，带有一个基本电机和一个安装在旋转笼中的电机。笼子通过电刷获得能量，很像[我们已经报道过的其他项目](http://hackaday.com/2009/06/13/persistence-of-vision-propeller-clock/)。休息后请观看视频。
