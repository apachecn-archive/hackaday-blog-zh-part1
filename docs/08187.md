# 时尚、无磁盘的 GameCube 手持设备

> 原文:[https://hack aday . com/2011/07/29/sleek-disc-less-game cube-hand held/](https://hackaday.com/2011/07/29/sleek-disc-less-gamecube-handheld/)

![sd_card_gamecube_handheld](../Images/cd29b8381ab9b6288d5341b5c3444aca.png "sd_card_gamecube_handheld")

控制台黑客[techknott]的技能很可能是首屈一指的。我们确实喜欢[本·赫克]，认为他的便携式游戏机非常棒，但你必须看看这个便携式 game cube[tech knott]组装在一起。

虽然构造细节相当稀疏，但下面的视频展示了便携式“魔方”的最佳功能。他的新掌上电脑比我们过去推出的[翻盖 GameCube](http://hackaday.com/2010/03/14/flip-top-gamecube-portable/) 或 [Dreamcast](http://hackaday.com/2009/12/16/techknotts-portable-dreamcast/) 便携式电脑小得多，屏幕更宽，而且完全没有光盘。虽然从 SD 卡启动游戏备份的合法性我们不会深究，但我们喜欢这个概念。

控制台本身可能只有标准 GameCube 控制器宽度的 1.5 倍，虽然它没有内置电池组，但我们不会拒绝。此外，谁想在外面玩游戏立方体？有了这个，我们非常乐意把糊状的自己留在室内，非常感谢。

我们这里唯一的抱怨是缺乏建造细节。[techknott 的]掌上游戏机非常神奇——我们只是希望能够看到这种神奇是如何创造的！

一定要看看下面的视频，看看控制台的运作。
