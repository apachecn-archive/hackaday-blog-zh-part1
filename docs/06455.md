# Wii 的 SD 活动指示器

> 原文:[https://hack aday . com/2010/12/17/SD-activity-indicator-for-Wii/](https://hackaday.com/2010/12/17/sd-activity-indicator-for-wii/)

![](../Images/28c19533c47a91abdfece0e2e174d4ce.png "SD-activity-LED-Wii")

[DeadlyFoez]想知道他任天堂 Wii 里的 SD 卡什么时候在用。[他使用 PICAXE 08M 构建并指示 LED](http://www.wiihacks.com/wii-customizations/76129-my-sd-card-activity-led-mod.html#post482624) ，并将其添加到 SD 插槽旁边。他使用微控制器的一个引脚来监控 SD 卡插槽的一个引脚上的电压。当卡空闲时，该引脚有一个特定的值，当它在使用时，该值上升。他没有分享何频采样的细节，也没有分享他的源代码中的神奇数字实际上代表了什么。但是如果你想自己做这个，这个概念应该是一个足够的开始。休息之后，观看它在剪辑中闪烁。
