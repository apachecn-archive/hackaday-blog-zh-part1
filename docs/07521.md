# Papercraft Strandbeest 是一个伟大的雨天项目

> 原文：<https://hackaday.com/2011/05/01/papercraft-strandbeest-is-a-great-rainy-day-project/>

在中西部这里，有时似乎春天永远不会，嗯…春天。我们知道“四月的阵雨带来五月的花”，但是几乎上个月的所有时间这些地区都是寒冷和多雨的。虽然情况正在改善，但我们认为准备一些有趣的项目总是好的做法，以防万一你和孩子的计划被取消。

我们认为 hack aday reader[[DOM beef]paper craft strand beest 是一个雨天下午的完美主意。供应清单很短，除了组成 strandbeest 主体的厚纸之外，只需要一些剪刀、钳子、回形针和胶水。根据他的说明书中包含的 PDF 模板，纸张被切割成小块，并通过小块回形针固定在一起。](http://www.instructables.com/id/Paper-Strandbeest-Mk-2)

一旦所有的腿都建好了，剩下的一个回形针就成了主轴，所有的东西都在 strandbeest 身体的主要部分下面连接在一起。

正如你在视频中看到的，腿工作得很好，尽管 strandbeest 可能在短期内受益于手摇曲柄。计划在他的作品中添加一个小马达，一旦完成，应该可以让 strandbeest 快速移动。

如果你正在寻找更多有趣的项目与孩子们一起做，看看这个[纸工艺陀螺仪](http://hackaday.com/2011/03/26/papercraft-gyroscope/)或这些[粘糊糊的电路](http://hackaday.com/2011/04/15/squishy-circuits-for-tiny-tinkerers/)就够了。
