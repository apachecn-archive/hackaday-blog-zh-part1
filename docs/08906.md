# 你想要多少钱买一个手机充电器？

> 原文:[https://hackaday . com/2011/10/24/你想要多少钱买一个手机充电器/](https://hackaday.com/2011/10/24/you-want-how-much-for-a-phone-charger/)

![](../Images/24a28373013363e1e063dd27aee95d84.png "Exif JPEG")

人们很快发现我是个呆子，他们的下一个问题通常是“你为什么拥有那部又旧又笨的手机？”。老实说，我不喜欢电话。经过十年的 Palm Pilots 和 Windows CE 设备，我也真的不喜欢触摸屏(胖子手指和一个嚼指甲的坏习惯无济于事)。我也不喜欢为了买一台带收音机的高档 PDA，你通常必须注册一个数据套餐，或者全额付费。

现在滚出我的草坪！说真的，我真的只需要用手机做两件事，打电话和发短信。如果我有一个愿望清单，我唯一想要的其他东西是 MP3 文件的大容量存储和蓝牙。很自然地，当我开始新的工作时，我在部门里找到了这个极客，在我被问到我的基本 LG 翻盖手机后不久。

经过几天的审问，我开玩笑地回了一句“既然你这么担心，为什么不给我一个更好的手机呢！”经过一点暗示和午餐时“瑞士面包卷”的贿赂，我得到了一部装有 Windows Mobile 5 的旧 HTC 手机。

虽然它不完全是 iPhone 或 Android，但它比我的更有特色，它有一个迷你 SD 卡插槽和蓝牙！唯一的问题是，他找不到充电器。我们不知道这东西是否能用(他从未见过这东西被打开)，也不知道电池的状况。

作为一个优秀的小黑客，我还是接受了它，休息后加入我，看看我如何让它燃烧起来，并在这个过程中节省了相当多的变化。

有问题的手机是 Cingular 2125(是的，Cingular，所以它已经有 5-6 年的历史了，但我的翻盖手机也是)，它在电池下面标有 HTC innovation。似乎 HTC 网站上的每个页面都包含“创新”这个词。它还可以有其他几个名字，比如 Wizard 1xx 和 Faraday，但不管怎样，最有用的信息来自 cingular 页面。

![](../Images/177f075ad4fa769b2806d794ea51fc44.png "Exif JPEG")

在那一页上，它说手机可以通过迷你 USB 端口从 5-9 伏充电。您可以使用标准 USB 电缆充电，但如果电池电量低于一定水平，您需要使用壁式充电器。由于这部手机可能至少 4 年没有见过充电器，所以可以说电池没电了。

我在谷歌上搜索了一段时间，但从未真正找到任何关于充电器的信息，除了它们的成本，人们可以花几美元买到一个通用的 USB 充电器，但我怀疑这会有什么好处，因为如果它可以通过 2 线 USB 连接充电，它也会在我的电脑上充电。然而，官方的充电器是 29.99 +运费，坦率地说，我没有办法支付这个费用，并且为一部我甚至不知道如何工作的手机等了一个星期。

按照我的一个注册商标“猜测和奇思妙想”，我决定用很多摩托罗拉手机正在使用的 Adafruit minty boost 讨论中发现的一张图纸的方式连接一个 USB 插头(谁在乎我是否油炸它)。首先是找到一个男性迷你 USB 插头，挖来挖去，我发现了完全无用的 USB 线来与我的 TI lauchpad。我用一把美工刀在插头侧面划了一道口子，慢慢切开塑料保护罩，迅速将插头从外壳中取出。

![](../Images/bb655c3940209e264c1e39858935529b.png "motoplug")

![](../Images/708d3c3eb07e70775e35f0ab01d2100d.png "Exif JPEG")

移除塑料保护罩后，我拆除了连接 USB 插头的 4 根电线，并焊接在一些长带状电缆上。很自然，我需要的一个针脚被剪掉了，他们经常这样做是为了防止不熟练的工人进行不正确的连接，但这非常讨厌。

![](../Images/2bb63bfcc510050056a66d2d6e32dfc1.png "Exif JPEG")

![](../Images/d24fcfbcd9ec2016debe1e479975d724.png "Exif JPEG")

为了解决这个问题，我只是用我的烙铁挖出引脚周围的塑料，并用小刀清除所有熔化的糊状物，留下足够的金属让我获得良好的焊接连接。一旦导线被焊接上，我将另一端焊接到一组引脚头上，这样它就可以插入我的试验板。接上电源，手机开始充电，几个小时后，电量只充了一半。

![](../Images/a1357d4fa5732bd2b9a7851b4ee20e96.png "Exif JPEG")

我让它在电脑 USB 端口上通宵完成，尽管周末我打了几个电话，发了许多短信，用 NES 模拟器打败了“超级马里奥兄弟”。在所有这些之后，电池电量仍然只有满电量的 75%左右。所以我知道手机工作，电池在这一点上仍然是好的。

![](../Images/dc5e91f0444089e6e7ef2e0ddb1686b4.png "Exif JPEG")

我本来可以就此打住，但如果我让电池放电低于其 USB only 阈值，我会重新做一遍，为了防止这种情况，我决定将我的 3 线和电阻从试验板上移到一个适当的电源上，并制作自己的充电器。

虽然试验板仍然连接着，但我测量了手机的电流，使用这种设置，手机将消耗不到 700 毫安(准确地说是 678 毫安)，这足以拧紧 USB 端口，但足够低，我不得不质疑 HTC 的意图，因为他们真的可以放弃这一点。

无论如何，我翻遍了我的垃圾箱，拿出了一个 LG 充电器，我从来没有见过的手机，额定为 5 伏 800 毫安，这是完美的。现在只需将充电器连接到 USB 端口，并适当地拉下 2 个引脚。

切断 LG 充电器的电线，我收到了 4 根电线，因为 24(？)末端的引脚连接器只是用螺钉固定在一起，我打开它，将它与谷歌上的引脚排列进行比较。奇怪的是，红色是 5 伏，绿色和黑色是地，黄色是 ID 线(类似的电阻设置，我想除了它似乎是我需要的)。

![](../Images/341cd7dbaa6800301972c88a05eba4ce.png "Exif JPEG")

它从来没有失败过，如果我查找引出线，这些线就会逻辑匹配，如果我不查找它们，那么它就是各种各样的疯狂。它只是值得查找或探测它们，因为电线的颜色在大多数情况下没有任何意义，无论如何你都必须查找它们。

我剪下绿色和黄色的电线，因为我不需要它们，为了防止两端短路，我拿起我的剪线钳，稍微拉了一下绝缘层，这样电线就回到了绝缘层里面。然后，我在两端滴上一滴液体强力胶，这样它们就不会戳出来了。

![](../Images/319280203848771c289d9f6c080e0471.png "Exif JPEG")

同时，在 USB 插头上，我焊接了一个 220K 欧姆的电阻，将引脚 2 和 4 连接在一起，并将另一端接地。最后，我将电源和地线连接到 USB 插头的第 1 和第 5 针，为了使事情变得更紧密，我拉了拉电源线的外套，它隐藏了 2 根剪断的线。

![](../Images/56482e7abf7097131d319bc30e311305.png "Exif JPEG")

现在我有了一个合适的手机壁灯，但是它很脆弱。真的没有任何东西将金属 USB 屏蔽固定住，焊点上也没有应力消除。为了解决屏蔽问题，我只是简单地用了一层薄薄的凝胶强力胶。它有一个小突起，所以靴子能支撑住它。为了解决开机问题，我拿出了我最喜欢的小黑客工具，腻子环氧树脂。我所要做的就是标出我不想让油灰去的地方，然后用胶带把它粘住，这样它就不会挤在塞子里面了。

![](../Images/0b8432aadf4e197c8c846ad3c25a8040.png "Exif JPEG")

如果你从来没有使用过环氧腻子，它只是环氧树脂和固化剂的腻子形式。你所要做的就是把它们混合成统一的颜色(我的是蓝色和白色，所以它是紫色的),然后应用。你可以用不同的添加剂得到不同浓度的这种东西，但我只是用沃尔玛便宜的“5 分钟品牌”。由于这种材料硬化很快，你必须快速移动，将它混合成均匀的颜色，然后涂抹。

![](../Images/11874907bf22650fe53df46b714e71ff.png "Exif JPEG")

一旦你把它滴在上面，你可能想弄光滑表面，因为你会有指纹，或者环氧树脂粘在手指上的小刺(即使戴着乳胶手套)。最好的方法是用一些普通的旧植物油沾湿你的手指，然后抹平。它永远不会是 100%完美的，但也足够好了。

![](../Images/f8f75de9930e947f497f6ca1fdbc84b5.png "Exif JPEG")

最后，在它安装之前，我需要修剪插头末端，这样它就不会覆盖在需要插入插头的地方。使用美工刀，刀刃在植物油中擦拭，简单地向下切割，直到你碰到金属，稍微倾斜一点，这样你就可以把切割的部分去掉。在这一点上，环氧树脂开始变硬，所以它会感觉不像油灰，更像沙子，只要继续做小切口，最终当你开始看到胶带移动一点，因为你现在正在撬它的边缘，你已经走得足够远了。

现在我已经为下一个五年做好了准备，而不是被问到“你为什么用那个又脏又旧又笨的手机”，我会被问到“你为什么用那个又脏又旧的智能手机……哦，它能播放塞尔达吗？好吧，那很酷”

我觉得不错。
