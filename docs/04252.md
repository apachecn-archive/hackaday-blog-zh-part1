# 变形机器人在 IROS 展示

> 原文：<https://hackaday.com/2009/10/15/morphing-robot-demonstrated-at-iros/>

![morphing-robot](img/1829510d1399abafd044795c6b19b74b.png "morphing-robot")

一个变形机器人在本周的 IROS 会议上展示。这个球体没有刚性结构，但使用某种类型的“膨胀”系统来移动。这个机器人概念是由 iRobot 公司提供的，作为他们正在进行的 DARPA 计划的一部分。“膨胀”实际上是皮肤中的一种物质，可以从液态转化为固态。他们称之为“干扰概念”，并在我们插播的视频中给出了外行人的解释。

当移动时，这个白色的球是一个翻腾、旋转、膨胀的恐怖物质。《异形》中即将孵化的豆荚，或者《毁灭战士 3》中的某样东西浮现在脑海中。我们昨天报道的来自 IROS 的六足动物令人惊叹，但这真的让我们毛骨悚然。更重要的是，这是一年前 iRobot 原型的片段。新的东西可以做得更多，比如把几个这样的东西组合成一个单元。

我们很高兴[DarwinSurvior]给我们发来了这个问题的提示,但现在我们晚上会睡不着觉了。
