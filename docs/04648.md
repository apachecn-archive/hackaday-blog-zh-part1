# 弹球构建抛出了挑战

> 原文：<https://hackaday.com/2009/12/31/pinball-build-throws-down-the-gauntlet/>

![](img/1a392c11acb3f9844f19047c349ba18c.png "pinball-build-off")

[Jeri Ellsworth]正在建造她自己的弹球机。她的构建日志在[以每日视频](http://www.youtube.com/user/jeriellsworth)的形式发布，展示整个过程。除了看到错综复杂的坡道，陷阱和控制器，她概述了她的建设技术。这些包括基于旧弹球机复制零件，用[定制工具](http://www.youtube.com/watch?v=WQGoKUVfe6g)或[烤箱](http://www.youtube.com/watch?v=gHLS3P820ZM)弯曲丙烯酸树脂。

她计划使用的驱动程序是一个 Altera FPGA，带有一堆 fet 来控制重载组件。没有太多关于实际电子产品的信息，但我们在创意领域的组件中看到了很多乐趣。到目前为止，我们最喜欢的是第 7 天的电视节目。屏幕是半透明的，背面投影有图像。当它前面的斜坡升起时，弹球就可以跳过屏幕了！

我们找不到这方面的项目页面，但我们在休息后嵌入了[Jeri 的]视频。 <https://www.youtube.com/embed/Acy3EIu16d8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
