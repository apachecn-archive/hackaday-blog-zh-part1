# 一个电视-B-乱世与图片扭曲

> 原文:[https://hack aday . com/2012/02/23/a-TV-b-gone-with-a-pic-twist/](https://hackaday.com/2012/02/23/a-tv-b-gone-with-a-pic-twist/)

![pic-tvbgone](../Images/21e85cff9e8e33ec91207ed3084b0d09.png "pic-tvbgone")

[Kayvon]认为 TV-B-Gone 是一个有趣的小设备，并想制造一个，但他没有 AVR 程序员。他决定锻炼一下自己的 PIC 技能，制作一个自己的微芯片衍生品，而不是拿起一些 AVR 工具简单地制作一个复制品。

基于 PIC 的 TV-B-Gone 与其基于 AVR 的兄弟非常相似，以 PIC24F08KA101 为特色，而不是 ATTiny。他的 TV-B-Gone 版本可以无限期打开，这使他可以将设备放在一个方便的隐藏位置，只要他喜欢，就可以大肆破坏。

[Kayvon's] TV-B-Gone 以不到 7 美元的价格完成了原厂产品的所有功能，比 Adafruit 套装便宜不少。如果你不反对 perfboard 的构建，一定要看看 Adafruit 论坛上的构建线程。[Kayvon]已经为您完成了大部分繁重的工作，您需要做的只是构建它。
