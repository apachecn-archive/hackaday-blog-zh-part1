# VHS 面包机

> 原文:[https://hackaday.com/2009/02/16/vhs-toaster/](https://hackaday.com/2009/02/16/vhs-toaster/)

虽然据说这个灵感来自于一个关于年轻一代的片段，但我们都知道这最终会发生。[莱蒙尼]把一台 [VHS 卡座变成了烤面包机](http://www.instructables.com/id/How_to_make_a_VHS_video_toaster/)。他们做了一个了不起的工作，它看起来几乎完美的股票。我们可以想象，保持录像机的外观是相当困难的，尤其是让一切都正确排列。最后，我们有一个使用我们的旧 VHS 甲板。休息之后你可以看到一段视频。
