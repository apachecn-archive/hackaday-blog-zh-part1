# 来自甜甜圈和茶的太阳能电池

> 原文：<https://hackaday.com/2009/03/19/solar-cells-from-donuts-and-tea/>

[https://www.youtube.com/embed/bVwzJEhMmD8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/bVwzJEhMmD8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这听起来难以置信，但却是真的。你可以从甜甜圈粉和茶叶中收获一个简单的太阳能电池所需的大部分东西。粉状油炸圈饼含有纳米颗粒的[二氧化钛](http://en.wikipedia.org/wiki/Titanium_dioxide)，这是一种“高带隙半导体”。这意味着它可以用来制造太阳能电池。茶叶只是用来给材料染色，这样它就能吸收可见光谱。这个过程有点太复杂了，不是你可以随便在野外扔在一起的东西，但它仍然很可爱。
