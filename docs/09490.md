# 满载的电子实验室让您的项目轻而易举

> 原文:[https://hack aday . com/2012/01/12/full-loaded-electronics-lab-makes-your-projects-a-breeze/](https://hackaday.com/2012/01/12/fully-loaded-electronics-lab-makes-your-projects-a-breeze/)

![](../Images/94a17b9f83798328d112d49d7f8f46d2.png "fabricating-and-arduino-shield")

这个黑客真的没什么特别的。[Craig Hollabaugh]需要一个 Arduino 护罩来安装 Pololu 电机驱动器和连接外部硬件。真正让我们花太多时间看他的帖子的是，[克雷格]记录项目的叙述风格令人愉快，我们羡慕他的电子实验室。该链接指向四个项目页面中的第一个。下一页链接在每页的底部，或者您可以在休息后找到收藏。

[Craig]从在 Eagle 中设计单面盾开始。他制作自己的 PCB 已经很多年了，他花了大约四次尝试才把墨粉转印正确(我们也是抗蚀剂镜像错误的受害者！).当需要钻销头时，[Craig]使用他的 3D 打印机制作一个支架，让 Dremel 安装到钻床上。这里有一个关于从 Harbor Freight 购买硬质合金钻头的好建议；我们以为易贝是唯一能买到这些的地方。我们中的许多人都需要订购零件，但这个车间里有他需要的一切物品，而且摆放有序。他还有锡膏和 PID 配套的烤箱来回流电路板。哦，当他忘记添加一个电阻时，返工站就会自动添加一个。

明白我们的意思了吧……工具再多也不嫌多。
