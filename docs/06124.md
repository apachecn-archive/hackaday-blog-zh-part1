# 球形多点触摸装置

> 原文:[https://hackaday.com/2010/10/15/spherical-multitouch-rig/](https://hackaday.com/2010/10/15/spherical-multitouch-rig/)

![](../Images/36b2094b7cecb92355984ff10ee5a60f.png "globe")

我们都喜欢多点触摸，但我们已经看到了太多的设置，它变得有点不那么令人兴奋了。这个将以其独特的形状吸引你的注意力。它是一个使用所有开源软件的[球形多点触摸](http://hml.queensu.ca/node/302)。好吧，因为极点是不可用的，它可能只是环形，或圆柱形，但它仍然令人印象深刻。他们使用安装在磨砂球体最上点的凸面镜来反射安装在底座底部的投影仪。一个指向同一面镜子的网络摄像头从几个发射器接收反射的红外线。休息之后你可以看一段视频。
