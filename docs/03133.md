# 向 Macbook Pro 添加右键单击

> 原文:[https://hack aday . com/2009/01/23/adding-right-click-to-a-macbook-pro/](https://hackaday.com/2009/01/23/adding-right-click-to-a-macbook-pro/)

![mbp_rc](../Images/7839e4a6baff5be67fa8bbc8d040b17b.png "mbp_rc")

令人惊讶的是，我们从试用 MAC 的人那里听到的最常见的抱怨之一是没有右键单击。最新版本的 unibody 有一个解决这个问题的选项，但是旧版本没有这个选项。虽然你可以随时插入 USB 鼠标，但这很难成为黑客的解决方案。[spiritplumber]给我们发来了他做的这个模型，[给他的 Macbook Pro](http://www.instructables.com/id/Adding_a_right_mouse_button_to_a_Macbook/) 增加了右键功能。值得注意的是，这是针对 2006 年至 2008 年版本的 Macbook Pro 的。你得靠你自己了。

[spiritplumber]指出，在跟踪板的背面有模拟某些事件的测试点。其中一个，刚好是右键。他向我们展示了如何将它连接到触控板右下角的自制接触按钮上。这可能对您的 macbook 有潜在的危险，所以要小心并遵循他的焊接技巧。如果你愿意，你可以对你的左键点击的另一边做同样的事情，或者就让它保持原样。休息之后你可以看到一段视频。
