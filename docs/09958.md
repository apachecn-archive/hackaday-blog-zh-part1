# 纸板框架三翼飞机

> 原文:[https://hackaday.com/2012/03/23/cardboard-framed-tricopter/](https://hackaday.com/2012/03/23/cardboard-framed-tricopter/)

![](../Images/277bc63b55573ef375b05857b4dfafa4.png "cardboard-frame-tricopter")

谈到降低建造成本，这款三翼机使用纸板作为框架，比它的四轴机亲戚少一个马达。除了休息后在视频中分享的细节外，几乎没有其他细节，所以我们只是根据我们看到的来猜测(请在评论中分享您自己的见解)。

这个集成着陆垫的平滑曲线使我们认为这个框架是用数控设备或忍者刀切割的。三个电机支架中的两个看起来就像上面显示的那样，但是第三个有一个连接到伺服电机的铰链安装支架。这样，螺旋桨可以围绕平行于支撑臂的轴线倾斜。我们敢打赌，这个功能主要是为了调整飞机的偏航。

视频评论提到，当油门在 45%时，这可以悬停，表明在需要时有很多升力可用。直到你真的通过在螺旋桨周围增加塑料笼子来减轻重量。这是一种整洁看到的东西'坚持'到天花板上的剪辑结束时，驾驶油门大开，并使用笼作为顶端起落架。
