# 电话控制太阳能充电罐

> 原文：<https://hackaday.com/2009/12/03/phone-controlled-solar-charged-tank/>

[https://www.youtube.com/embed/RmLU4GS7zAI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/RmLU4GS7zAI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这看起来像一个有趣的小项目。[克里斯]建造了一辆[坦克，他用他的 iPhone](http://www.projectallusion.com/1/post/2009/11/iphone-controlled-solar-powered-arduino-tank.html)控制着这辆坦克。坦克主要由现成的部件构成，如 Arduino 和 Xbee 无线单元，通过[处理](http://www.processing.org/)进行控制。你可以看到，当他改变马达的速度时，它的反应相当灵敏。你不需要 iPhone 来做这个，真的要做他正在做的事情，任何多点触摸输入都可以。你可以从他的网站下载代码。

我们对太阳能充电有点好奇。这要用多少电？那块面板看起来是不是有点小？充电需要多长时间？
