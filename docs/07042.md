# 电子学和 Arduino 的初级概念

> 原文:[https://hack aday . com/2011/03/08/初学者-电子概念-和-arduino/](https://hackaday.com/2011/03/08/beginner-concepts-in-electronics-and-arduino/)

![arduino_tutorial_series](../Images/dd4e170744f207ff46a68aee8e06ca92.png "arduino_tutorial_series")

[Jeremy]不仅介绍 Arduino，还帮助人们开始使用电子产品。他在自己的网站上发布了一系列视频，讨论使用 Arduino 的基础知识。刚刚出版了他的第十本教程，他给我们写了一封短信让我们知道。他的视频涵盖了广泛的主题，包括源代码、原理图和完整的零件清单，以帮助您跟随他。

虽然他的课程的核心特别关注如何使用 Arduino 执行特定的任务，但他也向观众介绍了许多不同的概念。他涵盖了 PWM、按钮去抖、中断、SPI 通信、模拟输入等，所有这些都是超出 Arduinos 实验范围的必读主题。

这些视频的长度从 6 分钟到 20 分钟不等，所以如果你打算看的话，你应该留出一段时间。

如果你不喜欢 Arduino 特定的视频，他的网站上还有一系列[通用技术相关的视频](http://jeremyblum.com/tag/techbits/)，你可能会感兴趣。否则，留下来看看他的系列教程中的最新内容。
