# 焊接和铸造 ABS

> 原文：<https://hackaday.com/2011/12/16/welding-and-casting-abs/>

![](img/cc5159760120f4a7105979c4d497263f.png "glue")

任何一个有 3D 打印机的人身边总是有一吨无用的塑料。一些塑料可能来自无用小玩意，但大部分是坏的印刷品、短的细丝和一般的废料。[卢克]找到了一种有趣的方法[来利用这些 ABS 废料](http://www.protoparadigm.com/2011/12/abs-glue-weld-cast-texture-and-more/)，并最终将垃圾变成了有价值的塑料部件。

丙酮通常作为洗甲水出售，它会把任何由 ABS 制成的东西变成一滩塑料。[Luke]使用相同的过程制作胶水——他在一个小容器中装入一半丙酮，另一半装入一小块 ABS。一天左右后，他有了一种很好的薄胶水，它会变干成为固体 ABS。[Luke]用这个做了一个 400 毫米长的挤压 t 型槽。我们不知道它是否适合建造一个儿童 RepRap，但它肯定会是一个有趣的实验。

[卢克]还用他的 ABS 胶做了一点选角。用更浓的 ABS 和丙酮溶液，他设法制造出了这个“东西”。整个过程在 [Thingiverse](http://www.thingiverse.com/thing:14490/) 得到了解释，我们迫不及待地想看看这东西能做些什么。
