# 打印您自己的超级跑车

> 原文：<https://hackaday.com/2012/03/20/print-your-own-supercaps/>

[Gil]最近来信告诉我们加州大学洛杉矶分校正在进行的一些令人敬畏的研究。显然，通过将一些氧化石墨层叠到 DVD 上，并将其扔进 lightscribe 刻录机，就有可能打印出自己的超级电容器；一些相当大容量的。

对于那些不知道，超级电容器通常是由两个电解质浸泡，活性炭板由离子渗透膜分开。由于活性炭具有令人难以置信的表面积，可以达到巨大的能量密度，在某些情况下为 1kJ/lb。

激光成型的石墨海绵取代了研究人员印刷电容器中的活性炭。休息后的一段视频适度详细地讨论了整个过程，同时更多的细节可以在关于这个主题的[他们的](http://hackaday.com/?attachment_id=69854)两篇[论文](http://hackaday.com/?attachment_id=69855)中找到。

第一个打印出晶体管的人会得到一袋 mosfets！
