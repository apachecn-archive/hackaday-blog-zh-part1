# 二进制计算器

> 原文:[https://hackaday.com/2010/09/04/binary-calculator/](https://hackaday.com/2010/09/04/binary-calculator/)

![oooh, love those blue LEDs](../Images/bbb4cd936b360e5a026585600af1df16.png "binary_calculator")

[Syst3mX]本周早些时候给我们发了一封关于他的二进制计算器的电子邮件，它几乎溜进了我们收件箱的深处。幸运的是，我们今天在那里注意到了它，并认为我们应该分享。

他希望能够计算二进制值，而不必经历重重困难或启动计算机，所以他建造了自己的二进制计算器。至于大脑，他选择了阿蒂尼 2313。他能做基本的运算，如非、与、或、异或、加、减、乘、除和取模。它看起来也不坏。休息过后，你可以观看它的视频。
