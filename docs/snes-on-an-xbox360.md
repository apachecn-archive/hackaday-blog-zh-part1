# Xbox360 上的 Snes

> 原文：<https://hackaday.com/2009/08/17/snes-on-an-xbox360/>

[https://www.youtube.com/embed/HBDYbRxYax4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/HBDYbRxYax4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

令人惊讶的是 Xbox 360 并没有看到太多的自制动作。[tmbinc]主动将 Snes9x 模拟器移植到它的上。没有漂亮的图形用户界面，也不能运行在 linux 或其他系统之上。这是本机的，直接针对硬件仿真。他认为这是第一次这样做。不幸的是，它不能在最新的 Xbox 360s 上工作。
