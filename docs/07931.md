# 使用 Arduino 作为控制器玩的数字化复古 LCD 游戏

> 原文：<https://hackaday.com/2011/06/26/digitized-retro-lcd-games-played-using-an-arduino-as-a-controller/>

这个有点抽象。还记得那些在 20 世纪 90 年代流行的液晶游戏吗？你知道，那些只有一个游戏，通常是一个角色可以来回移动，以捕捉，击中或阻止从天上掉下的物体或由一些恶棍扔出的物体？[Tobie Nortje]当然记得他们，并建立了一个 Arduino 控制器来玩游戏的虚拟版本。

他首先找到了一个将旧游戏数字化的网站。也就是说，他们已经拍摄了 LCD 的每个状态的图像，然后使用这些图像和原始物理的近似来实现游戏。与 NES 或世嘉主系统游戏不同，这些设备的 rom 不能因为特殊的屏幕而被丢弃。取而代之的是，一个虚拟版本的硬件被嵌入到一个网络界面中。

[Tobie 的]黑客的一部分是使用一个 Arduino 和几个按钮作为控制器。它很容易设置，我们认为试验板控制器的大小和重量非常接近 LCD 游戏。休息后看看视频，在评论中告诉我们你的想法。
