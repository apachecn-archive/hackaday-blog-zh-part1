# 尽你所能，史酷比不会摔倒

> 原文:[https://hack aday . com/2010/07/10/try-as-you-powery-scooty-wont-fall-over/](https://hackaday.com/2010/07/10/try-as-you-might-scooty-wont-fall-over/)

![](../Images/a0a19cceabaf02bc0be73e516a7c6c15.png "scooty-balancing-bot")

Scooty 是一个基于 PIC32 的平衡机器人。这个项目是通往可骑行版本的漫长道路上的第一次迭代。休息后的视频展示了一些不可思议的承诺。史酷比与一个大橡胶球对抗。两者相互滚动，但由于强大的处理器，惯性传感器和测距仪，Scooty 保持平衡。

史酷比对球并不公平。我们想看史酷比和球平衡机器人的对决。
