# 汽水瓶天窗

> 原文:[https://hackaday.com/2011/03/08/soda-bottle-skylights/](https://hackaday.com/2011/03/08/soda-bottle-skylights/)

![](../Images/7d2bfc4e262bcc52e93c6fb10b83d768.png "soda-bottle-skylight")

这里有一种环保的方法可以照亮封闭的空间。利用装满水的瓶子来利用太阳能。很简单，它们是两升装的汽水瓶。它们已经装满了水和两瓶盖的漂白剂，以防止微生物进入。然后用一个薄膜罐将帽子盖上，以防止阳光照射。它们通过屋顶上的洞安装，在阳光充足的情况下，它们发出相当于 50 瓦白炽灯泡的光。

我们首先想到的是避开天气，但这将在休息后的视频中解决。通过适当的防风雨密封，它们不会泄漏。我们可能还没有在房子里安装它们，但是对于那个没有电并且似乎吞噬院子里工具的黑暗小屋来说，这是一个多么好的补充啊。也许我们最终能够找到所有丢失的手抹子。
