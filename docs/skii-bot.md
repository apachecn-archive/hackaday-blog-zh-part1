# 滑雪机器人

> 原文：<https://hackaday.com/2009/10/21/skii-bot/>

[https://www.youtube.com/embed/luzs63dGHQs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/luzs63dGHQs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

博扬·内梅克发明了一个比我们滑雪更好的机器人。并不是说滑雪比我们好是一个非常崇高的目标。机器人能够使用标准的现成滑雪板滑雪。它有一个双重计算系统。一个系统只做转向和平衡，而另一个做所有的视觉和控制。他正在使用[雕刻](http://www.youcanski.com/en/instruction/carving.htm)滑雪技术，表示普通滑雪对机器人来说太难了。这对我们来说听起来是个挑战。休息之后，你可以看看比我们更好的 blooper reel。
