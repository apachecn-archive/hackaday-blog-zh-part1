# sugru–可塑硅胶粘合剂

> 原文:[https://hack aday . com/2009/12/04/sugru-molded-silicone-adhesive/](https://hackaday.com/2009/12/04/sugru-moldable-silicone-adhesive/)

![](../Images/b5e974229d6b2990f38f4181818f4c72.png "sugru-glasses-hook")

读者[James]告诉我们一个考虑到黑客而开发的新产品。 [Sugru 是一种硅树脂粘合剂](http://www.sugru.com/)，可在室温下固化。它是可塑的，一旦硬化，它会保持轻微的弹性。你可以在上面的图片中看到，它已经被用来制作一个钩子，但发明者展示了一系列其他用途，如替换椅子上缺失的脚，成型手柄和防水。最吸引人的一个方面是，Sugru 将与光滑的金属形成化学键。

该产品让我们想起不久前用于加固电子产品的两部分耳塞材料。不同的是，Sugru 是一部分，是粘合剂。它是一个装满单独大小的小包的背包。[使用方法](http://sugru.com/instructions/)，选择你需要的量，打开包装露出产品，然后揉捏和塑造口香糖状的物质以适应你的需要。休息后请观看演示视频。
