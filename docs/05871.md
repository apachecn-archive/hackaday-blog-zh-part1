# 我们可以在一个月内清理海湾

> 原文:[https://hackaday . com/2010/08/29/seas warm-we-can-clean-up the gulf in-a-month/](https://hackaday.com/2010/08/29/seaswarm-we-can-clean-up-the-gulf-in-a-month/)

![](../Images/25ae79ecc3ae14f177509c6d6542aa56.png "seaswarm_prototype1")

想在一个月内清理完墨西哥湾漏油？Seaswarm 说[用 5000 个漂浮机器人](http://senseable.mit.edu/seaswarm/)就能做到。

顾名思义，该项目使用群体机器人技术。每个单元从太阳获取能量，并拖着一条不会被水弄湿的吸油材料 [纳米织物](http://senseable.mit.edu/seaswarm/ss_nano.html)的传送带。一旦织物被原油浸透，就可以加热去除；这不是蜂群自己能完成的任务。但是听听这个:从纳米纤维中分离出油后，两者都可以再次使用。这意味着你获得了清理墨西哥湾的环境效益，而不是扔掉你的收集介质，石油再次成为可用的商品。听起来像很多高承诺，但休息后看看视频，自己决定。
