# PSP 加第二棒，相机，32 个

> 原文：<https://hackaday.com/2009/10/06/psp-plus-second-stick-camera-32-gigs/>

![jobless_punk_psp_mod](img/ecf384521f7b5f107a7293a85db64d3c.png "jobless_punk_psp_mod")

[失业朋克]最近完成了他的[PSP 应该是什么样的模式](http://acidmods.com/forum/index.php?topic=29518)。他内置了一个摄像头，并在身体中添加了一个开关，以在使用摄像头或使用 USB 端口之间切换。还有一个额外的模拟棒，并通过 USB 端口增加了充电功能。他还额外配备了 32 GB 的闪存。该设备当然运行定制固件，这有利于无处不在的模拟器和自制应用程序群。

我们同意最初的 PSP 是一个非常强大的手持设备，但它的潜力从未得到充分发挥。随着下一代 [PSP Go](http://hackaday.com/2009/09/29/psp-go-gets-butchered-teardown/) 的即将发布，我们希望旧设备的价格和可用性会带来更多这种类型的 mod。休息之后请看视频。

这里有一个来自[JoblessPunk]的制作精良的视频，详细介绍了他的工作。前半部分是宣传，以帮助寻找未来涉及这些模块的商业风险的合作伙伴。后半部分展示了新的特性、连接器、控制等。当你看到视频的第 25 秒时，试着抑制住想要窒息的冲动；从那里开始都是上坡路。
