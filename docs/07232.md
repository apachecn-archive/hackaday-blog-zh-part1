# 动作感应《我的世界》爬虫会吓走你的鹤嘴锄

> 原文：<https://hackaday.com/2011/03/28/motion-sensing-minecraft-creeper-will-scare-the-pickaxe-offa-ya/>

![](img/ab018407dfe97333436e81e0e88f8d57.png "scienceNooo")

[Chris]写信告诉我们他的动作感应， [Arduino 驱动的爬行器](http://www.thenewhobbyist.com/2011/03/motion-activated-attack-creeper/)。好像这些家伙在游戏中还不够可怕似的，[克里斯]不得不去制作最接近(合法的)真实世界的复制品。爬行器利用一个带有防波罩的 Arduino Uno 来播放爬行噪音，PIR 传感器来检测受害者，RC 汽车作为机动推车。爬行器坐在那里等待来自 PIR 探测器的信号，当它看到运动时，RC 遥控器被触发，爬行器发出噪音，然后(我们假设)出现恐慌。

我们可能已经放弃了玩具车的整个钢筋混凝土部分，找到了 H 桥电机控制器，但使用钢筋混凝土遥控器有潜力。整个触发机制可以远程放置，允许爬行器从某种覆盖物中跳出。

《我的世界》还不够吗？如果你对更多的砍树活动感兴趣，可以看看我们在《我的世界》的其他项目。

一个视频的爬行在行动中可在跳后！
