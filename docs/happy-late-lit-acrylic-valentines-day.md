# 快乐晚点燃丙烯酸情人节

> 原文：<https://hackaday.com/2012/02/16/happy-late-lit-acrylic-valentines-day/>

[![](img/d23783da7aca8df07af7303538172371.png "acrylic_valentine")](http://hackaday.com/2012/02/16/happy-late-lit-acrylic-valentines-day/acrylic_valentine/)

虽然他准时带着他的自制情人节礼物，但这篇文章对你来说有点晚了。然而，随着心形图案的改变，它可能是一个完美的“对不起，我又忘记了情人节”礼物，所以它可能仍然有用。

这个概念并不复杂，只是在一块丙烯酸树脂周围有一条 LED 灯。电池座和开关完善了这一建设。这是一种点亮事物的巧妙方式，但我们认为特别有趣的是它用最少的传统工具雕刻和切割的方式。

当然，[丹曼]可以使用带锯，但至于实际雕刻轮廓，他使用了一个改良的电动牙刷！我们很想看到那栋大楼被报道出来。如果这还不够的话，用他的话来说，刻字是用压缩空气喷嘴、笔管和装满陶瓷粉末的聚苯乙烯泡沫杯“犹太区爆破”的！Macgyver 会很骄傲的！
