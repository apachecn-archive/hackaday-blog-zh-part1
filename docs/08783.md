# 线性马达可以用垃圾制造

> 原文:[https://hack aday . com/2011/10/09/垃圾造直线电机/](https://hackaday.com/2011/10/09/linear-motor-can-be-built-from-trash/)

![](../Images/314b4123007d404174ed8d458f68c2eb.png "linear")

[Raul]用一个抽屉滑轨和一个多余的平板扫描仪制作了一个便宜的线性致动器。

我们见过的一些产品，像 [PCB 钻床](http://hackaday.com/2011/06/25/semi-automatic-pcb-drill-press/)或[效率极低的恒温器](http://hackaday.com/2011/09/29/sustainability-hacks-automatic-window-control/)，都使用线性致动器作为它们产品的关键部件。与我们通常使用的其他部件相比，这些线性致动器相当昂贵，因此走自制路线来降低成本总是一个受欢迎的想法。

[Raul]建造的第一步是在一块木头上安装抽屉滑轨。一个步进电机连接到致动器的一端，一条同步带沿着组件的长度被拉紧。在一个支架连接上抽屉滑轨和皮带后，你就有了一个非常便宜的线性马达。

在我们翻垃圾箱的日子里，我们数不清有多少次错过了不需要的家具的抽屉滑轨。一周中的任何一天，你都可以在口袋里找到老式 ScanJets 的步进电机、齿轮系和同步带。对于廉价而精确的直线运动，我们无法找到比劳尔做得更好的了。
