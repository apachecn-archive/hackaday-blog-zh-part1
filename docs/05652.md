# RBD(机器人啤酒递送)

> 原文:[https://hackaday.com/2010/07/13/rbd-robotic-beer-delivery/](https://hackaday.com/2010/07/13/rbd-robotic-beer-delivery/)

![](../Images/00ce1fdaca32e5801524da1679361612.png "robotic-beer-delivery")

柳树车库的那些疯狂的程序员又来了。这一次，他们创造了一个机器人丫头来送啤酒。当口渴的时候，你可以打开一个网络界面，把你的啤酒图片拖到购物篮里。一旦你提交了你的订单，机器人就会突袭你的冰箱，带着你的泡沫食物回来。它甚至会为你打开瓶子，但是，正如你在休息后看到的，这是你的啤酒真正起泡的地方。

我们已经看过了 PR2 的台球游戏，现在是作为一名酒吧女招待。柳树车库刚刚加入了我们希望我们工作过的地方名单中的 [SparkFun](http://www.sparkfun.com/tutorial/news/sparkfun-freeday.jpg) 。 [https://www.youtube.com/embed/c3Cq0sy4TBs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/c3Cq0sy4TBs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
