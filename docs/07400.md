# 光纤鼠标防止杂散磁场

> 原文:[https://hack aday . com/2011/04/15/fiber optic-mouse-prevents-stray-magnetic-fields/](https://hackaday.com/2011/04/15/fiberoptic-mouse-prevents-stray-magnetic-fields/)

![](../Images/6b827eba2093afabe3d506b2f0914e01.png "mouse")

[Ben]需要一个能在电信号和磁场无法忍受的地方工作的输入设备，所以他最终用光纤代替了鼠标的电源。

[Ben]把一些玻璃纤维从鼠标连接到正交编码器来得到 x 和 y 的速度。鼠标点击是通过用一个小快门修改现有按钮来阻挡光线穿过按钮框来读取的。这不是[本]第一次将光纤用于输入设备。去年，他还利用同样的原理制造了一个光纤操纵杆。

上个月我们报道了【Ben】的 DIY 电子显微镜[，我们想知道这两个项目是否有关联。他的](http://hackaday.com/2011/03/23/diy-scanning-electron-microscope/)[项目日志](http://benkrasnow.blogspot.com/2011/04/diy-scanning-electron-microscope-image.html)称，他从冷却风扇和加热器产生的电场中得到了失真的图像。也许他解决了那个问题，现在只是在追踪最后一个不需要的电磁辐射。
