# 万圣节宿醉:会说话的脑袋

> 原文:[https://hack aday . com/2011/01/10/Halloween-hangover-talking-heads/](https://hackaday.com/2011/01/10/halloween-hangover-talking-heads/)

这是万圣节剩下的道具。[克里斯]在他的门廊上用去年的恐怖装饰建造了他自己版本的唱歌头。演员唱着恐怖歌曲的投影图像赋予了四个聚苯乙烯泡沫塑料人体模型中的每一个都有自己的数字生命。门廊楼梯上有一个光学传感器，会被不知情的寻糖者绊倒。Arduino 监控传感器，并使用 AutoHotkey 在上网本上启动视频剪辑。休息后看看视频，看看完成的显示器在天黑后做它的事情。
