# 弹出式蜻蜓机器人可能是名片的未来

> 原文:[https://hackaday . com/2012/03/05/pop-up-dragonfly-robot-be-the-future-of-business-cards/](https://hackaday.com/2012/03/05/pop-up-dragonfly-robot-could-be-the-future-of-business-cards/)

想要在求职面试中给人留下深刻印象的工程师们，最好从钱包里拿出一个这样的弹出式机器人。这个工程奇迹使用[一种层压构造技术来建造一个机器人作为弹出式组件](http://www.youtube.com/watch?v=VxSs1kGZQqc)。你可以看到在这个过程中使用的底座，它是一个六边形，在层压过程中用作脚手架，包括便于组装的机械连接。

该设计要求多层材料按照严格的规格进行激光切割。一旦所有的部分都完成了，就用棒将它们对齐，然后融合在一起。激光切割机再走一次就完成了铣削，机器就可以装配了。但是对于这么小的零件，你需要一个可靠的方法来把所有的东西组合在一起。当脚手架的两个部分分开时，我们前面提到的连接允许这样做。作为一张名片，唯一不可能做到的是需要在焊料池中走一趟，以将碎片粘合到位。但是也许某种类型的夹紧机构可以在将来消除这种需要。

不要错过休息后解释整个过程的视频。你甚至可以看到小家伙扇动翅膀。这就是目前它所做的一切，还没有任何可用的转向机制。
