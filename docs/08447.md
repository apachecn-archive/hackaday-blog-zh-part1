# MindFlex 观察你仅凭视觉判断他人

> 原文:[https://hack aday . com/2011/08/29/mind flex-watches-as-you-judge-others-by-sight-alone/](https://hackaday.com/2011/08/29/mindflex-watches-as-you-judge-others-by-sight-alone/)

![mindflex_headset](../Images/7511464e830abaf909e4fd27b8287859.png "mindflex_headset")

真的很想知道他的大脑在想什么。

不，真的。他知道所有来来去去的想法，但他想知道在他有意识的思想流之下，他的大脑中发生了什么。带着一个 MindFlex 耳机和一个耳机，他开始解码他脑子里到底在想什么。

他花了一个月的时间抓取了 3500 万份谷歌个人资料，将每个用户的照片下载到 MySQL 数据库中。Teensy 连接到 MindFlex 传感器板上，通过串行连接收集耳机的所有输出。

他的实验包括让监视器上的每张个人资料图片闪烁一秒钟，记录下 MindFlex 上 11 个可用脑波通道中的 2 个。然后这些值被标绘出来，这样他就可以看到耳机捕捉到的“注意力”和“冥想”值。一天结束时，他发现，有趣的是，看着狗最让他放松！

我们很想知道从他收集的数据中可以得出什么样的关联，以及什么样的“隐藏”思想在其他人的大脑中流动。它最终肯定会成为一把双刃剑，揭示潜意识偏见和其他类似的事情，但至少可以说这是一个非常有趣的实验。
