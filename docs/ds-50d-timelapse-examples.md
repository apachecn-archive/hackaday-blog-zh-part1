# DS + 50D 定时拍摄示例

> 原文：<https://hackaday.com/2008/12/19/ds-50d-timelapse-examples/>

[https://www.youtube.com/embed/MwKHABKL_dY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/MwKHABKL_dY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们在 9 月份报道了[曹思德]的佳能 DSLR 的任天堂 DS 控制器。后来他改进了软件，使它有了一个日落/日出的计时器。在温哥华期间，他还拍摄了大约 30GB 的延时摄影图像(T3)，并制作了几个测试视频。他仍在研究处理方法，以充分利用 1500 万像素的图像。我们期待着未来的结果，因为 YouTube 现在正在使用一个[更大的播放器来播放高清内容](http://www.youtube.com/watch?v=xtROejpxrlo&feature=hd "YouTube - Hot Tech Gifts for the Holidays on Mahalo Daily!")。
