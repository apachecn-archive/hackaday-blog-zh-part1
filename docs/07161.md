# 人跟踪球演示让我们想建立自己的

> 原文:[https://hack aday . com/2011/03/21/people-tracking-orb-demo-makes-us-want-to-build-our-own/](https://hackaday.com/2011/03/21/people-tracking-orb-demo-makes-us-want-to-build-our-own/)

![kinect_eye](../Images/5e5efe2d3034565eca55337af3792815.png "kinect_eye")

本周早些时候，我们看到了一段视频，基于 Kinect 传感器收集的数据，一个基于球体的眼球会在房间里跟着你。尽管我们尽了最大努力，但除了视频之外，我们找不到更多的东西，但似乎这个项目背后的人已经在最近的一篇博客文章中直言不讳了。

英国设计工作室的乔恩·乔治(Jon George)解释说，人跟踪眼球可视化是使用 PC、Kinect 和一种名为 Puffersphere 的产品构建的，该产品可以在玻璃球体内部投影 360 度图像。通过应用将图像扭曲成圆形的过滤器，全景图像被转换以供球体内部的特殊透镜使用。

创建图像后，一个简单的 Windows 应用程序与 OpenNI 框架结合使用，使图像可以在房间里跟随你。

这个有趣的小项目的唯一问题是球体的价格——我们不确定它到底是什么，但请放心，它超过了我们愿意为这样一个玩具支付的价格。我们认为必须有一种方法来使用更便宜的硬件在某种程度上模拟球体的效果。有可能使用这种[球面镜投影构建](http://paulbourke.net/miscellaneous/domemirror/)的小规模 DIY 版本来完成，尽管它由[凹半球](http://paulbourke.net/miscellaneous/domemirror/iDome/)而不是完整的球体组成。

与此同时，看看这两个视频的球体在行动。别担心——我们知道你完全是在想索伦之眼，所以第二个视频不会让你失望。
