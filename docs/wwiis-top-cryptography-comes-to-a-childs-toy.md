# 二战的顶级加密技术出现在儿童玩具上

> 原文：<https://hackaday.com/2011/02/15/wwiis-top-cryptography-comes-to-a-childs-toy/>

这个玩具有一些升级的内部部件，使它变成了一个谜机。我们绝对喜欢这个想法，因为它采用了一个你的孩子可能已经长大的玩具，并用它来提供历史和数学的可教时刻。但是我们在骗谁呢？我们想做一个只是因为它是一个有趣的项目。

[Sketch]从旧货店拿了这个玩具，因为它有一个完整的键盘，他可以用它来制作自己的机器。它由 Arduino 驱动，四行字符液晶显示器取代了原来的显示器。他的帖子涵盖了他用来解决键盘布线的方法，也包含了对[英格玛机](http://en.wikipedia.org/wiki/Enigma_machine)如何工作的粗略概述。休息后，观看已完成项目的视频。

如果这满足了你的胃口，还可以去看看我们在 Hackaday 的第一年报道过的纸谜机。
