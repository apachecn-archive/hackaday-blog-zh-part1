# 通过 Twitter 实现家庭自动化

我们已经看到大量的东西被连接起来，当事情发生时向 twitter 发送更新。当我们第一次从[Matt]那里读到这个技巧时，这正是我们所期望的。我们惊喜地发现，他实际上正做着相反的事情。他在 IoBridge 模块上安装了几个项目，他可以通过向他的账户发送推文来控制它们。在视频中，你可以看到他发出蜂鸣器的声音，并改变液晶显示器上的显示。很高兴看到 twitter 被用作控制的一部分，而不是每次有人刷新时[就更新我们。](http://hackaday.com/2009/05/05/twittering-toilet/)
