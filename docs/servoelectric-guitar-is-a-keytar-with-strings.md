# 伺服电吉他是一种带弦的键盘手

> 原文：<https://hackaday.com/2009/11/06/servoelectric-guitar-is-a-keytar-with-strings/>

![servoelectric-guitar](img/c0d607e23250a740ac043a282f24c763.png "servoelectric-guitar")

[Keith Baxter]进行了一些吉他改装的“珠穆朗玛峰”。他正在开发一种伺服电吉他，用音品换键盘。它仍然是一把吉他，因为它有琴身、琴弦和拾音器来感知琴弦的振动并将振动传递给放大器。传统上，左手会根据需要通过将琴弦压向[音档](http://en.wikipedia.org/wiki/Fret)来缩短琴弦，现在，左手可以通过键盘来改变琴弦的音高。这是传统吉他和 80 年代现象的有趣融合，即[键盘手](http://en.wikipedia.org/wiki/Keytar)。

每个串连接到不同的伺服电机。当键盘上的一个键被按下时，相应的伺服系统会调整该弦的张力，使其调整到新的音高。他最初的设计包含了很多定制电路，但是他[改进了这个项目，加入了一个 Arduino 控制器](http://www.servoelectricguitar.com/arduino.htm)。第二代产品不仅简化了控制电路，还对其进行了改进。

休息之后我们嵌入了一些视频。在第一个例子中，你可以看到琴弦根据每个新音高进行调整。第二，看看吉他手背后…你认为他对那些**巨型**电容器有什么计划？
