# Arduino +增强现实

> 原文：<https://hackaday.com/2009/08/28/arduino-augmented-reality/>

[https://www.youtube.com/embed/sxr8oaRUq6k?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/sxr8oaRUq6k?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在这个视频中，你可以看到 Arduino 数据收集单元和增强现实系统的结合。这是由 pachube.com 的人建立的，一个分享你所在位置的传感器信息的网站，这是一个有趣的想法。我们可以看到每个单元都有自己的模式，因此它可以将正在收集的数据以 3d 形式叠加在其上。虽然这看起来很酷，但我们仍在试图弄清楚这有什么用？谁会带着一台连在电脑上的相机在办公室里闲逛呢？也许这意味着更多的手机，所以你可以得到快速的单位读数，而不必去访问他们的日志。因为我们知道你们有多喜欢 Arduino，我们想你们可能也会对我们看到的这幅[比真人还大的肖像](http://dailyduino.com/archives/639)感兴趣。
