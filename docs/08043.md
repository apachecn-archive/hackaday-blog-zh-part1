# 如何建立自己的脉冲激光手枪

> 原文：<https://hackaday.com/2011/07/10/how-to-build-your-own-pulsed-laser-pistol/>

![build_your_own_pulsed_laser_gun](img/a74047edd3bccb9c18c4fe60160ad2ac.png "build_your_own_pulsed_laser_gun")

自称疯狂科学家和 Instructables 用户的[Trevor Nestor]最近制作了一把脉冲激光手枪，并决定分享他的制作过程，这样你也可以在家里制作一把射线枪。除了他在易贝花了大约 100 美元购买的钕:钇铝石榴石激光头之外，这把枪主要由废弃的部件组成。然而，他确实说，如果你在退役的艾布拉姆斯坦克库存附近闲逛，你可以用旧测距仪获得 SSY-1 激光。

[Trevor]带你了解回收旧一次性相机的过程，以获得足够的电容器组组件，以及如何构建充电电路。他还讨论了激光头需要如何布线，然后在一个厚脸皮的视频中展示了成品激光手枪，嵌入下面。

虽然这种枪缺少我们展示给你的这种激光手枪的光洁度，但在穿透塑料、木头和薄金属时，它的威力丝毫不减。如果你想建造自己的激光器，这是一个很好的起点，为改进留有余地，例如更快的充电电路。
