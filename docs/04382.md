# 迪吉里杜管被改装成包括电子操纵

> 原文:[https://hack aday . com/2009/11/06/didgeridoo-moded-to-include-electronic-manipulation/](https://hackaday.com/2009/11/06/didgeridoo-modded-to-include-electronic-manipulation/)

![didgeridoo-sound-synthedsizer](../Images/ccb6c11bf626edad65948a50b6e79cac.png "didgeridoo-sound-synthedsizer")

这不是一个火箭筒，但这种经过改造的工具让玩家看起来像是在吸食某种武器。[Kyle]想把迪吉里杜管发出的已经很神秘的声音转换成电子信号。

这个项目的实体建筑非常漂亮。他在仪器外面安装了几块弯曲的控制板。控制功能包括六个按钮、五个拨动开关和六个与 Arduino 接口的电位计。声音被设备拾取，然后通过蓝牙与开关设置一起发送到计算机。然后，电脑施展魔法，在休息后创造出视频中听到的邪恶音效。

上面链接的他的文章包括几个详细描述合成过程的图表。它们有点超出我们的理解范围，但是如果你知道发生了什么，请在评论中分享你的见解。
