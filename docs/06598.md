# 诺基亚互联网密钥外部天线

> 原文:[https://hack aday . com/2011/01/17/Nokia-internet-key-external-antenna/](https://hackaday.com/2011/01/17/nokia-internet-key-external-antenna/)

![](../Images/9d2a3f2a7b74089da1c871c6768bfa80.png "nokia-wind-external-antennae")

[Maurizio]的无线网络出现了一些接收问题，他着手[为 USB 加密狗](http://www.spazioalchimia.it/laboratorio-di-sperimentazione/10-do-it-yourself/134-internet-key-antenna-esterna) ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://www.spazioalchimia.it/laboratorio-di-sperimentazione/10-do-it-yourself/134-internet-key-antenna-esterna))添加一个外部天线。他之前曾在诺基亚互联网钥匙内部探查过，发现内部天线是一个包裹在塑料盒周围的柔性电路基板，通过弹簧连接器与主电路板接触。如上图所示，这种塑料框架非常适合将 SMA 连接器安装在合适的位置，使其伸出外壳的末端。这给了他更好的范围，但由于速度取决于无线节点的流量，这并不能保证在这次黑客攻击后你会得到更快的连接。
