# DaScratch 多点触控 DJ 接口

> 原文：<https://hackaday.com/2008/09/20/dascratch-multitouch-dj-interface/>

![](img/0c7793ba15b983c6bccb9a79179308d8.png "stanton_dascratch")

斯坦顿发布了一款新的笔记本电脑 DJ 控制器外设。DaScratch 是一个 USB 连接的 MIDI 设备，设计用于模拟录音交互。它有一个很大的触摸区域，用户可以在那里进行刮擦、滑动和按钮按压动作。这款小巧的设备预置了 Traktor、Serato 和 Ableton Live 等软件，但可以与任何支持 MIDI 的软件一起工作。多个单元可以使用磁铁配对在一起。

正如下面的视频所示，有很多不同的交互方式。不过，我们真的很想看到这个设备被拆除。我们有一种明显的感觉，它的设计看起来比底层硬件实际上更令人印象深刻。[https://www.youtube.com/embed/6PUVHgKnrts?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/6PUVHgKnrts?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
