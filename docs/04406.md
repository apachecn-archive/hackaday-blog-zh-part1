# 线性光学编码器

> 原文:[https://hackaday.com/2009/11/12/linear-optical-encoder/](https://hackaday.com/2009/11/12/linear-optical-encoder/)

![dsc03989](../Images/060ac551abb0a2eb187db527d5391de5.png "dsc03989")

光学编码器并不新鲜；从鼠标到打印机，它们无处不在。它们可以让 DC 汽车公司知道他们的确切位置，甚至当前的方向。如果这听起来像旧帽子，这是因为我们已经向您展示了旋转版本之前。

[克里斯]使用相同的概念，但生产了一个[线性光学编码器](http://www.pyroelectro.com/tutorials/simple_motor_encoder/index.html)而不是[旋转](http://hackaday.com/2008/08/17/scratch-built-jog-wheel/)。他的设置很像非步进电机 CNC 和 RepRap mills 中使用的设置，允许普通 DC 电机知道它们在平面内的位置。这是一个快速教程，但我们喜欢细节，它提醒我们，我们需要完成 DC 汽车厂仍然是壁橱里的一堆零件。休息之后，请观看[克里斯]的视频。
