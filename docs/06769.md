# 使用 LED 矩阵模块的 Pong 时钟

> 原文:[https://hack aday . com/2011/02/09/pong-clocks-using-led-matrix-modules/](https://hackaday.com/2011/02/09/pong-clocks-using-led-matrix-modules/)

![](../Images/bbf64db2fcb8ac7e0462fe2453c2abee.png "led-matrix-pong-clock")

我们喜欢乒乓球钟，因为它们是展示品。这个特别的产品[被称为智慧时钟](http://timewitharduino.blogspot.com/2011/01/first-release-of-wise-clock-3-software.html)，是这个项目的[第三次硬件修订](http://timewitharduino.blogspot.com/2011/01/introducing-wise-clock-3.html)。LED 显示屏非常简单，因为他们使用的是 Sure Electronics 的 32×16 双色模块。如果你不想[设计和建造自己的多路复用显示器](http://hackaday.com/2011/01/31/how-to-build-a-ping-pong-ball-display/)，这是一个有点廉价和高性能的选择。

休息之后，你可以看到船桨的动作看起来非常逼真。它们像人类玩家一样移动，但这些时钟通常不是这样。如果你想知道这是怎么做到的，看看最初由[123led]为另一个项目开发的代码。
