# 戴尔 Streak 的 SNES 控制器坞站

> 原文:[https://hack aday . com/2010/11/04/snes-controller-dock-for-Dell-streak/](https://hackaday.com/2010/11/04/snes-controller-dock-for-dell-streak/)

![](../Images/5456b12d5f7a7c8678b515602de3913e.png "snes-controller-dock")

戴尔 Streak 是一款安卓平板电脑。[Collin Meyer]希望使用原装 SNES 控制器在设备上玩模拟游戏。他想出的是一个控制器，它是手持设备的一个接口。

要做到这一点，需要几个因素的共同作用。Streak 使用标准的 PDMI 坞站，通过 USB 连接与电脑相连。[Collin]通过连接坞站连接器上的几个引脚，重新调整了同步电缆的用途，从而强制设备使用 USB 主机模式。从那里，他用一个小小的微控制器将 SNES 控制器转换成一个 USB 设备(非常类似于[这个黑客](http://hackaday.com/2010/04/09/snes-controller-usb-accelerometer/))。短小精悍的同步电缆在 SNES 控制器体内找到了新家，在休息后的视频中，看起来他使用了类似 sugru 的[来为 Streak 增加一点支持。](http://hackaday.com/2010/10/11/oogoo-a-home-made-sugru-substitute/)
