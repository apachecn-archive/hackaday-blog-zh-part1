# EOS 底座上的佳能 FD 镜头

> 原文：<https://hackaday.com/2011/11/06/canon-fd-lenses-on-an-eos-mount/>

![](img/c9cdb00532da6f99fcf7b71f1d4dec70.png "lens")

相机镜头很贵，如果你像我们一样，你可以很容易地找到你的相机无法使用的非常便宜的镜头。[山姆]有一个佳能 EOS 和一堆老派的 FD 镜头在他的处置。但有一个问题:使用这些旧镜头和适配器意味着聚焦在无限远是不可能的。谢天谢地，他上传了几个视频([第一部分](http://vimeo.com/31400082)、[第二部分](http://www.youtube.com/watch?v=3GXCKDqVtjw)和[第三部分](http://www.youtube.com/watch?v=A1OUrrV5LnU))，展示了为他的新相机修改 FD 镜头的过程。

为了对 FD 镜头进行修改，所需要的只是一些环氧树脂、一个螺丝刀和一个 M42 到 EF 适配器。拆下 FD 镜头的背面后，[Sam]将 M42 适配器安装在他的相机上，举起镜头检查最小对焦距离。一点研磨或几个金属垫片确保镜头在正确的位置。

下一步是确保光圈仍然可以控制。[Sam]在他的视频的第二部分中讲述了这一点。一点点梦想就能搞定所有的困难工作。镜头最后用*微小的*环氧树脂连接到 M42 适配器上，转换完成。

虽然[山姆]可以放几张他的特写照片，但他很好地完成了这个过程的每一步。休息之后看看嵌入的视频。
