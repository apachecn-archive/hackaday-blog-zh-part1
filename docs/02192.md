# IPhone 开发团队发布越狱视频

> 原文：<https://hackaday.com/2008/07/11/iphone-dev-team-releases-jailbreak-video/>

[https://www.youtube.com/embed/R63lKICiWfg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/R63lKICiWfg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
因越狱 iPhone 而臭名昭著的 iPhone 开发团队刚刚[发布了一段 iPhone 3G 被黑的视频](http://www.youtube.com/watch?v=R63lKICiWfg)。保持关注，并在他们的博客上关注[的发布。这是对先前版本的 iPhone 已经可用的](http://blog.iphone-dev.org/) [PwnageTool](http://forums.macrumors.com/showthread.php?t=462648) 的重大更新。

他们增加了许多新功能，例如:固定的网络搜索、定制的安装程序配置和定制的根分区。他们承诺将很快发布，但声明不会在本周末发布。你可以从我们的镜子中获得视频的[高分辨率版本。](http://mahalo.dreamhosters.com/hackaday/pwnage_final.mov.qt)
