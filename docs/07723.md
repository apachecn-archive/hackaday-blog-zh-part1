# 买卖和交换黑客硬件

> 原文:[https://hacka day . com/2011/06/01/buying-selling-and-bartless-hacking-hardware/](https://hackaday.com/2011/06/01/buying-selling-and-bartering-hacking-hardware/)

![](../Images/57c14efd57f664935c7d79efdc064cce.png "the-junkbox")

普通读者[Grenadier]来信让我们了解他新发表的名为垃圾箱的黑客硬件交易帖子。显然，当他不在[炸掉电容器](http://hackaday.com/2011/05/18/blowing-up-capacitors/)或[建造自己的](http://hackaday.com/2011/05/10/roll-your-own-capacitors-high-voltage-edition/)高压电容器时，他会腾出时间编写一个网站，在那里你可以买卖和交换剩余的元件。

我们在 Hackaday 就有类似的东西。右边一栏的美元符号图标，就在特色文章横幅下面，会带你去我们的分类广告区，在我写这篇文章的时候，那里有九篇文章。但是这些东西潮起潮落。不时查看一下[新发布的链接](http://classifieds.hackaday.com/customlist/3)，看看这个数字有没有增长。

无论是在垃圾箱还是在我们的分类广告区，我们认为最大的问题是在你需要的时候找到你需要的东西。这不是什么新鲜事。如果你以前错过了，我们在休息后嵌入了一个旧的 EEVblog 集，其中[达夫·琼斯]参观了 Apex 盈余。这就好像一个有囤积倾向的黑客在四十年前买下了一家商店，只是不停地把越来越多的商品堆到天上。
