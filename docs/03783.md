# 全方位研究平台

> 原文：<https://hackaday.com/2009/08/08/omnidirectional-research-platform/>

 <https://www.youtube.com/embed/-bcoAqKhdL4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>[Shachar Geiger]发来一个有趣的项目，他和[Tal Avivi]在耶路撒冷的比撒列学院合作。他们的任务是设计一辆单人电动车。他们从麻省理工学院的 Transology 得到一些线索，设计了 T2 全方位研究平台 T3。上面嵌入了一段视频。它可以使用三种不同的输入方式来驱动:加速度操纵杆、传统游戏手柄或身体质量转移。他们从 Arduino 开始，但需要更多的 I/O，不得不改用接线板(这是在 Mega 之前)。这个平台大部分是用废料建造的。加速度计被放在一个旧的微软 Sidwinder 里。标准操纵杆来自 Sega Mega Drive。重量传感器不是廉价的家用秤。</p> </body> </html>
