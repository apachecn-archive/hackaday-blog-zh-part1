# 愚蠢的巨大视点显示

> 原文:[https://hackaday.com/2008/11/22/stupidly-huge-pov-display/](https://hackaday.com/2008/11/22/stupidly-huge-pov-display/)

![pov_gear](../Images/ef64fb7e02a5190b843a6afcd194df19.png "pov_gear")

[Mario Mauerer]和他的朋友被委托为一个聚会(至少是在一个高压实验室里)建造这个 [2m 视觉暂留显示器](http://translate.google.com/translate?u=http%3A%2F%2Fmauerer.m.googlepages.com%2F&hl=en&ie=UTF-8&sl=de&tl=en "//mauerer.m.googlepages.com/")。被称为“来自地狱的显示器”，它使用 100 个蓝色 SMD LEDs 来生成[视点](http://hackaday.com/tag/pov/)图像。它们通过移位寄存器连接到 ATMega64。他们的目标速度是 600 转/分钟，以获得无闪烁的图像，这意味着螺旋桨尖端以 140 英里/小时的速度移动。该板可以通过 IRDA 无线更新，增加 SD 存储的计划已经到位。你必须在下面嵌入的视频中看到和听到这个东西。
