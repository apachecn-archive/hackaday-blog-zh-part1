# 使用“网络摄像头”的自动驾驶汽车

> 原文:[https://hack aday . com/2011/06/13/an-autonomous-car-using-a-web cam/](https://hackaday.com/2011/06/13/an-autonomous-car-using-a-webcam/)

![](../Images/6be11ca16118eea74794a7bb00417c8e.png "autonomous car")

这辆来自康奈尔大学学生的自动遥控式汽车是为那里的高级工程课程设计的。它的主要“传感器”是一个低分辨率的网络摄像头。正如休息后的视频所示，这辆车利用它的视觉处理在白色表面上保持在两条黑线内做得很好。它还有一个红外传感器，可以探测到汽车前方的物体，并在不撞车的情况下停下来。

所有“视觉”计算都由 Atmel Mega644 MCU 处理，这是一款 8 位处理器。由于这种芯片的处理限制，必须做大量的工作来使这一过程在计算上高效。这些学生对他们的项目进行了令人难以置信的详细描述，重点是代码和电气设计。休息之后看看他们赛车的视频。
