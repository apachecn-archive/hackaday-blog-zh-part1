# 很明显，我们在玩雅达利美洲虎

> 原文:[https://hack aday . com/2011/12/24/were-on-an-atari-jaguar-kick-apparely/](https://hackaday.com/2011/12/24/were-on-an-atari-jaguar-kick-apparently/)

[![](../Images/db21e8d2ae9135e7e97bb83ebf2687cb.png "jag")T2】](http://hackaday.com/wp-content/uploads/2011/12/jag1.jpg)

我们不知道为什么，但雅达利捷豹本周受到了很多关注。[10P6]刚刚推出的这个 [Jaguar/CD 组合](http://www.atariage.com/forums/topic/191715-working-unseen-jaguar-jag-cd-combo-discovered/)让我们想起了雅达利在 1993 年可能推出的产品。

一旦你过了[kevincal]在雅达利时代论坛上开的“愚人节”玩笑，建造本身就相对简单了。[10P6]拿了一个普通的捷豹 CD 驱动器，在捷豹外壳上割了一个洞。整个案例 mod 花了不到三个小时，但[10P6]给了我们很多关于雅达利在 1993 年可以和/或应该建造什么的评论。

[10P6]表明，如果 CD 驱动器在基本单元上有库存，并以稍高的价格发布，这种类型的 Jaguar 会节省 Atari 的资金。这将削减手推车槽的成本，并减少制造过程中的塑料用量。[10P6]还谈到 Atari 工程师如何在系统时钟增加的情况下放弃 68000 协处理器。我们不太喜欢这个想法(问我们关于我们的纹身)，但逻辑确实有道理。

当然，这款车紧随我们几天前看到的 [Jaguar Portable](http://hackaday.com/2011/12/22/handheld-jaguar-makes-32-or-64-bits-portable/) 而来。老实说，我们不知道捷豹是怎么回事。
