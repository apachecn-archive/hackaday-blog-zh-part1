# Cat，去见 Arduino

[https://www.youtube.com/embed/eUO3vn-ivqo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/eUO3vn-ivqo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

你是否曾经觉得家里需要一台便携式条形码扫描仪？没有吗？嗯，[Mkanoap]做了，[所以他做了一个](http://www.instructables.com/id/Make-a-portable-barcode-scanner-with-an-arduino-an/)。他已经将他的 [CueCat](http://en.wikipedia.org/wiki/CueCat) 连接到他的 Arduino 上，以捕获条形码数据并将其存储在 SD 卡上。他把它作为个人图书馆的库存工具。以前，他不得不随身携带笔记本电脑进行扫描，或将书籍拖到办公桌前，而现在，他可以随心所欲地进行扫描。

你可能会注意到视频中一些愚蠢的声音。请注意，这些是它播放的实际声音。摄像机拍得不够好，所以他给他们配音。你不会想错过整个经历吧？
