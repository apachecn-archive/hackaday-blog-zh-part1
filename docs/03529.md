# 自动骰子

> 原文:[https://hackaday.com/2009/05/26/dice-o-matic/](https://hackaday.com/2009/05/26/dice-o-matic/)

[https://www.youtube.com/embed/7n8LNxGbZbs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/7n8LNxGbZbs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

管理 gamesbyemail.com。他最大的障碍之一是为游戏产生真正的随机数。他尝试了各种方法，如 math.random 和 random.org，但不断收到关于随机数质量的投诉。他的解决方案是建造一个自动掷骰子的机器。他的[最初的尝试是用乐高](http://gamesbyemail.com/DiceGenerator)做的，从来没有足够的可靠性被放进系统中。然而，Dice-O-Matic 已经被证明是一个产生随机数的怪物。它有 7 英尺高，每天能掷出 130 万次骰子。哇哦。
