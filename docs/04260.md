# 老鼠跑过虚拟现实迷宫

> 原文:[https://hackaday.com/2009/10/16/mouse-runs-through-vr-maze/](https://hackaday.com/2009/10/16/mouse-runs-through-vr-maze/)

![vr_mouse_setup](../Images/c260f98a1ec09f929e20b2b54602fb05.png "vr_mouse_setup")

为了确定动物在运动中的大脑，科学家们建造了这个奇妙的装置。这是一个鼠标的虚拟现实吊舱。虽然我们非常肯定/希望你们没有人需要鼠标虚拟现实系统，但我们认为这个装置足够有趣，可以独立存在。这似乎是一个凸面镜设置，投射在一个圆顶。这个控制器很有趣，因为它看起来像一个巨大的轨迹球黑客。他们使用的是一个装在漂浮在气垫上的球上的光学鼠标。这使得鼠标移动起来更加容易。休息之后有一段全程录像。
