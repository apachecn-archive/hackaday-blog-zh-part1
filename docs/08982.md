# 这是你的飞车

> 原文：<https://hackaday.com/2011/11/02/heres-your-flying-car/>

![](img/725a9f8641a3324577792885a8c73210.png "chopper")

在过去的几年里，我们已经看到了大量的四轴飞行器。我们从来没有见过大到足以举起一个人的电梯，直到现在。

几年来，e-volo 的[Thomas]、[Stephan]和[Alexander]一直致力于开发一种巨大的载人多翼飞机。几天前，他们的原型机搭载一名全人类飞行员升空。关于它们的构造没有太多的细节，但从我们所知，这架飞机完全是由电池驱动的。

测试车看起来是一项极简主义的研究。起落架看起来像一个重新设计的瑜伽球，底盘只是四块铝管焊接成一个十字架。原型的动力装置是车辆每个象限的四个无刷电机。没错——有 16 个马达围绕着飞行员旋转。

这不是我们第一次看到基于机器人医生设计的建筑。今年早些时候，中国的一些家伙建造了一个非常漂亮的死亡陷阱八翼直升机。e-volo 团队在安全考虑方面绝对有优势——他们有实际的[设计和工程研究](http://www.e-volo.com/Design_Studies.html)

好消息是，e-volo 团队希望改进他们的原型，并将其出售给大众。对美国人来说，坏消息是联邦航空局对电动飞行器不太感冒。该团队正在开发混合驱动版本，只要重量保持在较低水平，我们总能获得超轻认证。
