# 交互式数字壁炉非常适合寒冷的冬夜

> 原文：<https://hackaday.com/2011/11/03/interactive-digital-fireplace-is-great-for-those-chilly-winter-evenings/>

![video-fireplace](img/51d66af47378c74d1c53110fed3d569d.png "video-fireplace")

我们都见过它们——那些壁炉 DVD 似乎会出现在杂货店的端盖上，并在白象礼品交易所进行交易。如果你家里没有壁炉，他们可能会提出一个合理的解决方案，但来自 Gadget Gangster 的[Nick]认为，如果你能创建自己的互动数字壁炉会好得多。

他的壁炉使用 guggant Propeller USB 平台和 ProtoPlus 板，根据用户输入将各种视频片段合并在一起，并在任何配备 RCA 插孔的电视上播放。这个过程相当简单，包括通过 ProtoPlus 板将开关以及音频和视频输出连接到 Propeller 板。剩下的工作是使用软件完成的，需要用户选择和编码视频片段以存储在 SD 卡上。[Nick]不过，如果你想走更快的路线，确实提供了一整套剪辑供下载。

在对视频剪辑进行编码和分类以符合“添加木材”和“添加木材”用户输入按钮之后，Propeller 完成剩余的工作，在需要时从适当的类别中随机选择剪辑。

不可否认，这个结果有点低保真度，尽管我们认为它很酷，更愿意称之为“复古”。这可能不是一个高清壁炉录音，但依偎在一个手工制作的壁炉上比你事后拿起的 DVD 要好得多。
