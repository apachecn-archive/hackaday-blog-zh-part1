# 传感器设备可以发现汽车司机是否与骑自行车的人共用道路

> 原文:[https://hack aday . com/2012/03/07/sensor-rig-find-out-if-driver-is-share-the-road-with-cycleers/](https://hackaday.com/2012/03/07/sensor-rig-finds-out-if-motorists-are-sharing-the-road-with-cyclists/)

![](../Images/870790f53c4da519affd1ea0537ef4a2.png "sensor-rig-for-sharing-road-with-bikes")

显然，在德克萨斯州奥斯汀市，自行车和汽车共享道路是有原因的。(克里斯托弗·斯坦顿)来信告诉我们，现在有一项法律要求司机在超过骑自行车的人时留出 3 英尺的空间。这是很难执行的，因为没有像雷达枪提供的可靠证据来证明速度限制。上面的硬件正在通过收集传球习惯的数据来寻求帮助。[它测量并记录每辆经过你的车的距离](http://www.ghisallofoundation.org/projects/2012/0003/)。目标不是给更多的司机开罚单，而是收集可能有助于改变危险驾驶习惯的统计数据。

如您所见，前端设备机架承载硬件，便于在自行车上安装。它有一个臂，该臂延伸到侧面的距离与手柄杆的距离相同。带有广角镜头的高清摄像头设置为拍摄照片，可用于确定车道上自行车和车辆的位置，以及传感器的距离读数。

我们当然有兴趣看到在交通繁忙的城市环境中平均通行许可的数字。即使有自行车道，在繁忙的一天，事情也会变得很紧张！
