# 醒来先生，醒来就跑

> 原文：<https://hackaday.com/2009/10/28/mr-wake-wakes-up-and-runs/>

[https://www.youtube.com/embed/Kdeqd-e0mjI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Kdeqd-e0mjI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

【同位素】一定是真的恨自己。他建造了一个纯粹邪恶的刑具。[威克先生](http://letsmakerobots.com/node/11550)，他的闹钟机器人，应该算是一个刑具。这个可爱的小机器人，有着明亮多彩的时钟和整洁的塑料管框架，是一个闹钟。闹钟一响，威克先生就开始监视你。如果它探测到你的存在，它就会起飞，只等你睡眼惺忪地跌跌撞撞地回到它现在的位置。你知道当你追上它时它会做什么吗？它又起飞了。你为什么要造一个你知道自己会在一阵兽性的愤怒中被摧毁的机器人呢？

事实上，我们更喜欢《醒来先生》而不是《时钟先生》，仅仅是因为它的粗糙感。如果你真的很难起床，你可能要考虑一些[更激烈的](http://hackaday.com/2009/06/26/pneumatic-bed-alarm-clock/)。
