# 用莫尔斯电码键盘做好它

> 原文:[https://hack aday . com/2012/02/23/用莫尔斯电码键盘做正确的事/](https://hackaday.com/2012/02/23/doing-it-right-with-a-morse-code-keyboard/)

![](../Images/1cd8b083b09637027d3108463e29fb9d.png "iamb")

多年来，Hackaday 已经看到了几十种莫尔斯电码键盘，但纽约电阻器公司的[Hudson]最终设法给这个想法以应有的公正。他用职业选手使用的相同类型的电报键制作了一个 USB 莫尔斯电码键盘。

为了他的项目，[Hudson]得到了一个通常挂在 CW 钻机上的美妙的抑扬格桨。与我们之前见过的[莫尔斯](http://hackaday.com/2012/01/31/telegraph-key-makes-for-a-fantastic-twitter-input/) [键盘](http://hackaday.com/2010/05/27/usb-morse-code-keyboard/)不同，【哈德森】使用了[抑扬格拨片](http://en.wikipedia.org/wiki/Telegraph_key#Iambic_.28dual-lever.29_Paddles)，一个电报键，一个杠杆用于 dits，另一个杠杆用于 dahs。由于 dits 和 dah 是独立的电气连接，微控制器——一个小小的[解析莫尔斯电码并向计算机发送正确的字母非常容易。](http://www.pjrc.com/teensy/)

[Hudson]还为 dits 和 dahs 添加了一些音频反馈，并为 Teensy 和 speaker 设计了一个激光切割外壳。休息之后，请观看抑扬格键盘的视频。
