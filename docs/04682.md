</span> <p>[迈克]正在制造他自己的脉搏血氧仪,它利用光来测量血液中的氧饱和度。这种测量的一个附带好处是可以从相同的数据中计算出脉搏率。用于探测器的<a href="http://tinkerish.com/blog/?p=166" target="_blank">部分包括一个红色 LED、红外 LED 和一个 TSL230R 光强测量芯片。正如上面的视频中所解释的，每个 LED 通过你的指尖照射到光传感器上。红外 LED 被用作基线，并与红色 LED 进行比较，红色 LED 的一些强度被手指中的红色血液吸收。这是一个非常容易理解的生物识别概念，所以在进入</a><a href="http://hackaday.com/2009/03/28/the-narcisystem/">更复杂的生物识别界面</a>之前，你可能想从这里开始。</p> <p>[谢谢罗斯]</p> </body> </html>
