# 在高尔夫球车上放太多的马

> 原文：<https://hackaday.com/2010/12/13/putting-too-many-horses-in-a-golf-cart/>

![](img/f4b137221932bb2331a216f995292bbd.png "100-hp-golf-cart")

想要[把 100 匹马放进你的高尔夫球车](http://www.youtube.com/watch?v=x1HYkBD5UBU)里，然后把它带到跑道上吗？为什么不呢？[Mac McAlpine]就是这样做的，他把一辆 2007 款 GSXR 600 电喷车扔进了一辆 1987 年的燃气驱动雅马哈高尔夫球车上。休息之后，你可以看到一个测试运行的视频，以及交换引擎时所做的修改的概述。[Mac]提到，即使升级了动力，该设置仍然使用普通自行车链条和千斤顶。然而，他升级了前桥和后桥，分别安装了卡钳和鼓式制动器。车顶上还有一个[明日纳斯卡赛车](http://en.wikipedia.org/wiki/Car_of_Tomorrow)翼，帮助车轮保持在路面上。

还好他戴着那个头盔。没有安全带的话，撞破挡风玻璃会更舒服一点。开车小心点[Mac]！
