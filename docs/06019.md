# 一天工作一小时

> 原文：<https://hackaday.com/2010/09/27/work-for-hack-a-day/>

![](img/6aaeac0acf964e0b683334a3c5567908.png "It is hard work, but someone has to do it.")

HackADay.com 是一个著名的受人尊敬的博客，致力于硬件和黑客相关的所有事情，正在寻找知识渊博和充满激情的作家加入他们的专业和专注的自由作家团队。这是一个有报酬的自由职业者的职位，需要在写作风格和交付方面的专业性，一致性和可靠性。我们渴望找到并引进软件/硬件/技术和黑客世界中一些最有激情的人。我们支付给我们的作者一个稳定的每篇文章的价格，我们准备把 HackADay 带到一个新的水平，提供更多的内容，涵盖更广泛的兴趣和专业知识。如果你是一个好作家，喜欢我们谈论的这类东西，那么我们希望**你**在这个家喻户晓的名字在黑客领域的发展和主导地位背后。

鼓励所有硬件/黑客/技术爱好者申请，但在以下领域有经验的人将获得加分:

–电路设计
–机器人制造
–移动设备编程
–全桌面应用编程
–计算机安全
–粗糙的硬件制造
–修改现有系统

如果你准备好加入 HackADays 的顶尖自由撰稿人团队，请通过我们的[在线求职板](http://bit.ly/cWWYX9)申请。正式简历是受欢迎的，但是带有博客链接、写作样本或任何作品集的求职信才是我们真正想要看到的。这里有一个包括什么的想法:

*   关于你自己的简短简历
*   3 个以 Hack a Day 风格撰写的每日帖子示例
*   3 你想在网上看到的软件或硬件指南。对于我们在过去所做的工作的例子，看看我们的一些项目总结，操作和软件教程。
*   两句关于如何通过功能或内容来改进网站的话
*   你为什么会成为 Hack a Day 的合适人选，还有其他原因吗

感谢您的关注！
