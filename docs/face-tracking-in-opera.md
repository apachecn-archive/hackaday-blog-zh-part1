# 歌剧中的人脸跟踪

> 原文：<https://hackaday.com/2009/04/30/face-tracking-in-opera/>

[https://www.youtube.com/embed/1ioV2Dj56iw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/1ioV2Dj56iw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

受今年来自歌剧的[愚人节玩笑的启发，【杰森】已经让](http://labs.opera.com/news/2009/04/01/)[面部表情识别](http://smert.net/2009/04/23/opera-face-gestures-beta/)实际工作了。虽然这可能看起来像一个愚蠢的项目，但它可以真正帮助一些人。对于有运动控制限制的人来说，这可能是一个[伟大的辅助工具](http://hackaday.com/2009/04/20/twitter-brain-interface/)。他说它现在有一些问题，最明显的是长期使用的性能问题，所以他希望得到一些聪明人的意见。
