# 基于 DS 的盲人阅读器

> 原文：<https://hackaday.com/2009/11/30/ds-based-reader-for-the-blind/>

![](img/c6d0e6f1cfb270885d0d27a774a99ad1.png "DS-voice-reader")

[Epokh]发布了一些自制软件，用任天堂 DS 作为文件的语音阅读器。这对于通常使用屏幕阅读器，但可以在旅途中利用这项技术阅读书籍、文档和电子邮件的盲人和视障人士来说非常有用。未来的版本希望添加一个电子邮件客户端，并通过相机实现 [OCR](http://en.wikipedia.org/wiki/Optical_character_recognition) ，以便在旅途中阅读文档。

利用 [flite 包](http://www.speech.cs.cmu.edu/flite/index.html)来提供文本到语音的功能。我们对这个包很熟悉，从休息后的视频来看，它在[到 DS 硬件](http://sourceforge.net/projects/mydsreader/)的端口没有丢失任何东西。[Epokh]指出，[类似的阅读器](http://www.intel.com/pressroom/kits/healthcare/reader/index.htm?iid=pr1_marqmain_reader)可能要 1500 美元，而 DS 售价约为 130 美元。我们迫不及待地想看到最终版本的充实！
