# 办公室游戏节目蜂鸣器保持事情公平公正

> 原文：<https://hackaday.com/2012/02/03/office-game-show-buzzer-keeps-things-fair-and-square/>

如你所料，[Chuck 的]办公室的销售团队是一群非常有竞争力的人，他们决定要一个能让他们在每周例会上互相挑战的系统。比赛包括回答经理提出的问题，但举手只起了有限的作用——[他们需要一种明确的方式来判断谁先“进来”回答问题。](http://chuckontech.com/?p=166)

由于[Chuck]只有很短的时间和很少的预算，他选择寻找最简单的问题解决方案，这是一个基于 Arduino 的游戏节目蜂鸣器系统。游戏显示器是由一个 Arduino、一些 led 和一个 Altoids tin 制成的，而蜂鸣器按钮是从一个旧的无线电广播控制台中抢救出来的。

现在，当提出一个问题时，销售人员可以回答，因为他们知道只有最快的人点击按钮才会被记录。到了下一个问题的时候，主持人只需点击他的蜂鸣器来重置控制台。

虽然它没有我们不久前推出的这个游戏蜂鸣器系统那么花哨，但[Chuck]说它可以完美地完成工作，而且价格便宜。
