# 全彩色激光电视

> 原文:[https://hackaday.com/2011/12/31/full-color-laser-tv/](https://hackaday.com/2011/12/31/full-color-laser-tv/)

[![](../Images/9c25b6c5b6b88c14d8c0684b9aa392cf.png "TV")T2】](http://hackaday.com/wp-content/uploads/2011/12/tv.jpg)

早在 2001 年，[Helmar]用一个红色激光笔和一个旋转的 18 面镜子制作了一个令人惊叹的单色视频显示器。蓝色和绿色激光比十年前便宜得多，所以[Helmar]决定用他的激光投影仪来实现全彩色。(在德国，所以启动 Chrome 或获得[谷歌翻译](http://translate.google.com/translate?sl=de&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.laser-tv.eu%2F&act=url))

[Helmar]的纯绿色投影仪的古老网站介绍了操作原理。一束激光照射到多面的多棱镜上。这被反射到另一面镜子上，该镜子为视频帧中的每一行提供反射。今年早些时候，[Helmar]破解了一个红色和蓝色激光器，以补充先前存在的绿色激光器。最终的结果是一个由该死的激光驱动的 RGB 投影仪。

据我们所知，投影仪只有复合输入；附带的 DVD 播放器为此提供了所有信号。令人惊讶的是，[Helmar]没有在电路中使用微控制器。所有的电子设备都是简单的逻辑门。如果你问我们，真的很神奇。
