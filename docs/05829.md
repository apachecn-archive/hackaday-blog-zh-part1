# Modchip 通过 USB 端口攻击 PS3

> 原文：<https://hackaday.com/2010/08/19/modchip-hacks-ps3-via-usb-port/>

[https://www.youtube.com/embed/4jOEbZEkp9A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/4jOEbZEkp9A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

已经过去很久了，但上面的视频显示了一个 modchip 通过从硬盘运行游戏来绕过 PlayStation 3 的安全性。销售 modchip 的网站[Ozmodchips.com](http://www.ozmodchips.com/ps-jailbreak-worlds-first-ps3-modchip-plug-and-play-p-68.html)和已经确认可以工作的 psx-scene.com[现在都因为繁忙的交通而不稳定。但我们知道的是。该设备名为 PS Jailbreak，可用于将 PS3 游戏转储到运行最新固件(3.41)的 PlayStation3 的硬盘上。转储的游戏可以通过从 modchip 生成的菜单中选择它们来从硬盘上玩。不过，这要花你不少钱。目前的预订价格为 169.99 澳元或 147.47 美元，预计交付日期为 8 月 27 日。](http://psx-scene.com/forums/showthread.php?t=64935)
