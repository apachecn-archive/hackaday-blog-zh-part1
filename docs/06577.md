# 销头定径机

> 原文:[https://hackaday.com/2011/01/13/pin-header-sizing-machine/](https://hackaday.com/2011/01/13/pin-header-sizing-machine/)

![](../Images/76c26e3341b20640fc02cb204a19ec6b.png "header breaker md")

[Ammon Allgaier]制造了一种工具，它可以以很高的精度拆开销头。在休息后的视频中，他演示了内置功能。它们包括一个可调节的停止装置，以选择你想要在每个切段中的针数。输入侧还有一个小凹槽，收割台的塑料框架就在其中。插入直至接触到挡块，然后向下推，在正确的位置折断割台。几个弹簧将切割齿返回到其静止位置，使您可以快速切割 40 针坯料。这台机器将成为那台[自动线切割机](http://hackaday.com/2010/07/26/more-automated-wire-cutting/)的好伴侣。

这对单接头来说很好，但我们一直在寻找一种可靠的方法来折断双引脚接头。当我们试图使用两把钳子时，经常会犯错误。如果你知道更好的方法，请在评论中分享你的方法。
