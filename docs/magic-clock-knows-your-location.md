# 魔法时钟知道你的位置

> 原文：<https://hackaday.com/2009/11/29/magic-clock-knows-your-location/>

[https://www.youtube.com/embed/p-A4ZNdkmEs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/p-A4ZNdkmEs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

直接出自《哈利·波特》小说的是[魔法时钟](http://www.themagicclock.com/build-log)。就像小说《这个钟》(即使它不报时，它还是一个钟吗？)显示家庭成员的当前位置，从家到医生的办公室，甚至到致命的危险(我们听说每年的这个时候很好)。

时钟指针由 [4 个独立的伺服电机](http://www.themagicclock.com/build-log/movement)驱动，伺服电机由[一个 Arduino](http://www.themagicclock.com/build-log/the-brains) 维护。家庭成员的位置通过 Twitter 无线更新。我们认为为每个成员的 GPS 手机编写的脚本可能更值得信赖，但它目前似乎工作得很好。
