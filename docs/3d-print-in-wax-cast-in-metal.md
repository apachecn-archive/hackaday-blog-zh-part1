# 蜡制 3D 打印，金属铸造

> 原文：<https://hackaday.com/2012/01/31/3d-print-in-wax-cast-in-metal/>

![](img/3c7dcd07304562069ba6244e491e9732.png "Untitled")

干冰、酒精、蜡珠和搅拌器有什么共同之处？这是第一次尝试为这个[蜡 3D 打印机制作介质，这是【Andreas】一直在建立的](http://andreasbastian.com/3dp/3dp.html)。想要用金属生产 3D 打印物体，发现[直接金属激光烧结](http://en.wikipedia.org/wiki/Direct_metal_laser_sintering)机器仍然遥不可及，他开始寻找一种不同的方式。

在尝试了几种自己制作粉状蜡的不同方法后，他认为直接购买更省时。使用市场上可买到的混合有活性炭的粉末蜡和定制的打印机，用适度高功率的激光喷蜡。在新烧结的层上涂上更多的蜡粉，3d 零件向上构建。一旦他有了蜡制的零件，他就可以做一个模具，用[失蜡铸造](http://en.wikipedia.org/wiki/Lost_wax_casting)的方法铸造金属。

虽然质量并不完美，你仍然需要一个大约 2500 美元的激光装置(从他的学校借来的),但这无疑是迈向未来的一步。
