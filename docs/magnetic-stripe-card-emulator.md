# 磁条卡仿真器

> 原文：<https://hackaday.com/2006/01/02/magnetic-stripe-card-emulator/>

![magnetic](img/8b1ed96c2f40feac7dc3d9c99062eea5.png)

这是一个概念证明磁条卡模拟器。Adron 将一条薄金属条嵌入磁条的第二轨道。该条的末端用细铜线包裹以形成螺线管。螺线管由 PIC 微处理器和一些晶体管驱动以增强信号。提供的软件可以脉冲测试模式和你提供的任何额外的卡号。
