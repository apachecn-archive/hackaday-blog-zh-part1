# Python PIC 在连接到计算机时非常有用

> 原文：<https://hackaday.com/2012/02/28/python-pic-useful-when-attached-to-a-computer/>

![](img/ff76609767f981570e40899060c4b5c2.png "pyMCU")

[Richard]发送了一个链接，链接到他正在研究的由 Python 控制的微控制器。与我们之前看到的[便携式 Python 板](http://hackaday.com/2011/09/26/python-on-a-microcontroller/)不同，【Richard】认为他的 pyMCU 不是最好的自主使用。该板仅在连接到计算机时使用，作为计算机的数字世界和我们的模拟世界之间的桥梁。

我们已经看到运行[轻量级 Python 解释器](http://wiki.python.org/moin/PyMite)的主板，但是我们对这种主板只有在插入计算机时才有用的想法相当感兴趣。板载 PIC 16F 芯片具有足够的数字、模拟和 PWM 引脚，几乎可以完成任何可以想象的任务，如果您希望通过微控制器获得一些输出，还可以使用 16 引脚 LCD 显示头。

[Richard]说他从事图片工作的时间比 Arduino 出现的时间还要长，但是根据兴趣程度，他会考虑开发 pyMCU 的 Arduino 版本。我们所知道的是 pyMCU 将会是*令人敬畏的*来教年轻人电子和编程，并且当他们使用它的时候，我们肯定会为该板找到更多的用途。
