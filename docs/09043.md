# 皮卡丘来找你了(尤其是在地毯上)

> 原文：<https://hackaday.com/2011/11/09/pikachu-is-coming-for-you-especially-on-carpet/>

![](img/8af3650b260e2d4b87306008a47f1452.png "pikachu-carpet-bot-v5")

如果你仔细观察，你会发现皮卡丘并没有展示一对时髦的投掷星，而是悬挂在两者之间。我们毛茸茸的朋友正在给这个叫做地毯猴 V5 的[地毯漫游机器人装上一张笑脸。它已经工作了多年，这只是原型开发过程中的又一站，因为版本 6 的开发已经在进行中。](http://www.societyofrobots.com/robot_carpet_monkey_v5.shtml)

该项目是一个证明，什么可以完成使用所有的设计工具在您的处置。动力机制被认为是腿的质量和使用轮子的便利性的结合。每个附件都覆盖着有策略地放置的点，用来抓住地毯，并允许“轮子”在机器跳过它们时抓住物体。你可以看到，每个都有一个弹簧机构，以进一步促进与车轴的每一圈抓地力。这似乎远远超出了通常由爱好机器人产生的东西，我们认为这是一件伟大的事情！

休息过后，会有一段视频展示这些夹子的所有部件是如何组装的。大约 3 分钟后，看到机器人在房间里巡游。
