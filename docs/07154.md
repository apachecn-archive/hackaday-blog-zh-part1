# EeePC 触摸屏改造

> 原文:[https://hackaday.com/2011/03/20/eeepc-touchscreen-retrofit/](https://hackaday.com/2011/03/20/eeepc-touchscreen-retrofit/)

![](../Images/4ddbfeceb265abfca395cf93b3c6ce66.png "eeepc-touchscreen-upgrade")

给你的电脑增加触摸屏功能真的不是很贵，但是要让所有东西看起来都像它应该的样子却是一项巨大的工作。[Deadbird] [写了一份分步指南，帮助你安装触摸屏硬件](http://deadbird.legtux.org/?p=130&lang=en) *和*让你的上网本像新的一样装回去。

硬件由两部分组成。覆盖屏幕的透明膜和读取输入的驱动板。薄膜本身的背面有一层粘合层，可以粘在液晶面板上。但是要安装它，你首先必须从挡板上取下面板。你还需要一个地方来放置驱动板。[Deadbird]不知何故在机箱内为控制器找到了足够的空间，但他不得不拆下键盘和主板来将其安装到位。这相当于完全拆卸您的 eeePC。但是，如果你习惯了触摸感应设备，并且曾经发现自己触摸液晶显示器并想知道为什么计算机不跟随链接，这对你来说可能是值得的。休息之后，您可以在一个剪辑中看到最终产品。
