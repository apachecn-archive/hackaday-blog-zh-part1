# 大众需求:OK Go Rube Goldberg 机器

> 原文:[https://hack aday . com/2010/03/20/by-popular-demand-the-ok-go-rube-Goldberg-machine/](https://hackaday.com/2010/03/20/by-popular-demand-the-ok-go-rube-goldberg-machine/)

![](../Images/c3484ffc046b102ef8031b9ce3093034.png "OK-Go-Rube-Goldberg")

我们收到了许多关于 OK Go 视频的提示，视频中有一台 Rube Goldberg 机器。如果你还没看过的话，休息之后请观看他们的视频。这是一个罕见的例子，YouTube 视频中的音轨得到了录制它的艺术家的完全认可。

我们看到这个的第一反应是什么？谁是得到报酬设计和建造那个装置的幸运儿？你不必对此感到困惑，乐队已经发布了一个由四部分组成的视频系列，讲述了这台机器，并记录了设计会议和建造过程(这些视频也在休息之后)。Syyn 实验室的工程师们被派去完成细致的破坏工作，我们认为他们做得非常出色。有很多关于这项工作的报道，但是[我们最喜欢的是《连线》的](http://www.wired.com/gadgetlab/2010/03/ok-go-rube-goldberg/)，因为它详细描述了过程，而不是最终产品。

鲁布·戈德堡机器公司最棒的地方就是问“为什么？”是错误的问题。纯粹的建造乐趣使得[接管一栋房子](http://hackaday.com/2010/02/20/rube-goldberg-ftw/)或[过度复杂化满足饥饿](http://hackaday.com/2009/09/28/the-incredible-breakfast-machine/)是值得的。
