# DyIO 是一个巨大的机器人开发板

> 原文:[https://hack aday . com/2012/03/03/dyio-is-a-huge-robotics-development-board/](https://hackaday.com/2012/03/03/dyio-is-a-huge-robotics-development-board/)

![](../Images/e0b7036e68b73f93c7adb3553670210b.png "DyIO")

[Kevin]来信告诉我们他在过去几年中致力于开发的机器人开发平台。他称他的设备为“DyIO ”( T1 ),看起来这是一种让机器人快速启动并运行的非常简单的方法。

因为 DyIO 代表动态输入和输出，[Kevin]认为在他的构建中放置 24 个独立的 IO 引脚很重要。这些引脚可以作为 24 个数字输入或输出，一些模拟输入和 PWM 输出，甚至 DC 电机控制。

真正有趣的是[Kevin]和他的团队选择构建的 SDK。有了这个 SDK，你可以用 Java 或者任何你想要的语言来编写 DyIO。[凯文]和他的团队已经围绕 DyIO 建立了一些有趣的项目，如[六足机器人](http://wiki.neuronrobotics.com/NR_Hexapod_kit)和[电子动画口袋妖怪](http://wiki.neuronrobotics.com/Animatronic_Pokemon)。虽然我们确信一些超乎想象的令人敬畏的东西正等待着用 DyIO 平台来构建，但您可以在休息后查看这些已经完成的构建。
