# 数控蛋糕装饰机

> 原文:[https://hackaday.com/2011/01/05/cnc-cake-decorator/](https://hackaday.com/2011/01/05/cnc-cake-decorator/)

AutoFrost 数控蛋糕装饰器是奥林工程学院学生项目的成果。[Tara Krishnan]和她的同学设计并建造了硬件，以及一个 Python 接口，用于绘制你想要打印在糖霜上的设计。X 和 Y 轴由步进电机控制，Z 轴可手动调节。该软件可以设置蛋糕的大小，这使得它比我们上次看到的机械化装饰师[更加通用。GUI 还允许一次应用一种颜色的多种颜色，机器暂停以便操作员切换彩色糖霜容器。总之，这是一个很好的构建，但是下一个版本应该去掉第二个 Arduino。我们不认为代码复杂到足以保证其中两个。休息后请观看演示视频。](http://hackaday.com/2010/10/16/computer-aided-cake-decoration/)
