# 内置无线耳机

> 原文:[https://hackaday . com/2009/09/28/internal-wireless-headphones/](https://hackaday.com/2009/09/28/internal-wireless-headphones/)

[https://www.youtube.com/embed/upYgHgMgTpk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/upYgHgMgTpk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

无线耳机是帮助大多数台式电脑系统清理混乱的好方法。然而，在将壁式电源插头和耳机插孔插入电脑后，使用的电线数量实际上增加了一倍。在[帕克]找到一套旧的 [JVC 900 Mhz](http://www.amazon.com/JVC-HAW300RF-900mhz-Wireless-Headphones/dp/B00001OPK7/) 无线耳机(和一个普通的 900 Mhz 发射机)后，他撬开箱子，看看他能用它们做什么。意识到发射器使用的是 12V DC 电源，他用一个[未使用的软式连接器](http://www.playtool.com/pages/psuconnectors/connectors.html#floppy)(提供+12V、+5V 和两条地线)为其供电。他还将音频输入线直接连接到他的声卡头上，而不是在后面提供一个耳机插孔。然后，他用塑料将整个东西包起来，以防止不必要的短路，并将其放回他的个人电脑中，给他留下了一个非常实用的无线系统。休息后的详细照片。
