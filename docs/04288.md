# Ps3 笔记本电脑由本·赫克提供

> 原文：<https://hackaday.com/2009/10/21/ps3-laptop-courtesy-of-ben-heck/>

![ps3s_hero1](img/c105c36ef5285cedf84f6610b2220da9.png "ps3s_hero1")

本·赫克可能是有史以来最酷的人(拿亚当·沙维奇来说)。你可能会想，在生产了这么多笔记本版本的游戏机后，他会厌倦，或者我们会对它们感到厌倦。但是没有！我们喜欢每一部新作品——包括他最新的笔记本电脑化 Ps3 。没有太多的“新”来自他的最新文章，但它确实重申了一些事情，包括:为什么要制造你能买到的东西，廉价的泡沫片应该按吨购买，当你失去 Ps3 原件时，使用过热的 Xbox 360 的螺丝。哦，我们只是在开玩笑，你知道我们会永远爱你。休息之后看一段视频。
