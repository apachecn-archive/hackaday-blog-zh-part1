# 8 位 MIDI 合成器

> 原文:[https://hackaday.com/2011/02/18/8-bit-midi-synthesizer/](https://hackaday.com/2011/02/18/8-bit-midi-synthesizer/)

![](../Images/656e5351b74bae825aba8568c3d6a767.png "8-bit-midi-synth")

[德里克·伊诺斯]努力开始产生效果。他一直在研究一个由 MIDI 控制的 8 位合成器，他称之为去调制器。正如他在休息后演示的那样，该设备具有正弦波和方波功能，可以产生相当悦耳的声音。但它也提供了录制您自己的样本的选项，然后根据来自您选择的设备的 MIDI 命令进行修改。在这种情况下，他使用了摇滚乐队 3 键盘(或者 [keytar](http://hackaday.com/2009/11/19/when-an-axe-isnt-enough/) ，如果你愿意的话),以一种比它最初的目的更有创造性的方式。

现在，我们将不得不满足于演示视频和功能列表，因为没有其他细节。但是开源代码和硬件信息在他的任务清单上。
