# 万圣节道具:高科技南瓜灯

> 原文：<https://hackaday.com/2009/10/29/halloween-props-techy-jack-o-lanterns/>

[https://www.youtube.com/embed/UeYS_Tjf7pQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/UeYS_Tjf7pQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

万圣节就在这个周末。如果你仍然有一些时间和零件可用，你可能会寻找香料你的南瓜灯。我们发现了几个项目，我们认为分享一下可能会很好。他们中没有一个人有资格单独发表一篇文章，所以我们想把他们聚集起来，一起分享。它们似乎都是由 Arduino 驱动的，我们知道这会带来一些评论。只是澄清一些问题，他们没有付钱给我们做 Arduinos 广告。人们只是用它们做了很多项目。

首先，你可以在上面看到的[傻弦射击鬼火](http://www.instructables.com/id/Arduino-controlled-Silly-String-shooter/)。他使用一个连接到 Arduino 和运动传感器的单一伺服系统。当它探测到运动时，它会喷出一小段细绳。您可以从项目页面下载代码。我们可能建议您以避免直接喷到一些孩子眼睛里的方式来安排。

休息之后请关注接下来的两首。

[https://www.youtube.com/embed/fF5Q5mIKb2U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/fF5Q5mIKb2U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

第二，我们有[闪烁的 LED 南瓜灯](http://www.instructables.com/id/Arduino-Powered-Pumpkin/)，它会根据你的移动而变化。同样，由 Arduino 和接近传感器驱动，它不会变得更简单。

[https://www.youtube.com/embed/n3K02Rjbexs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/n3K02Rjbexs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

最后，我们带给你[音乐同步 LED 南瓜](http://www.instructables.com/id/Music-Synchronized-LED-Pumpkin/)。这个有一个很好的雕刻，前面有一个鼓，随着音乐的节奏从后面点亮。同样，这是由 Arduino 控制的。设置相当简单，您可以从项目页面下载源代码。
