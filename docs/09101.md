# 极坐标笔绘图仪绘制巨大的图像非常慢

> 原文：<https://hackaday.com/2011/11/17/polar-pen-plotter-draws-huge-images-very-slowly/>

[euffy]刚刚在 11 月发布了一个关于他的 Polargraph 绘图机的说明性的,它能够比糖蜜还慢地绘制巨大的图像。这台绘图仪只使用两个步进电机来控制笔的位置，而且几乎可以完全由废旧零件制成——他的这台绘图仪造价仅 150 英镑。

Polargraph 在一个大平面的顶角使用两个步进电机。一个加重的笔架用串珠绳连接到两个马达上，这在窗帘上很常见。通过控制从托架到每个马达的距离，可以精确地控制笔的位置。这不是一种非常快速的绘制图像的方式(查看[实时视频](http://www.youtube.com/watch?&v=YwZWg_JDoIc))，但看起来肯定很有趣。

还有一些其他的绳索链绘图仪，如 [Der Kritzler](http://hackaday.com/2011/09/04/material-of-choice-felt-pen-on-glass/) 和 [Hektor](http://hektor.ch/) 。[Euphy]的工作是我们见过的最好的文档构建之一，他还发布了[代码](http://code.google.com/p/polargraph/)和[网站](http://www.polargraph.co.uk/)。

当我们想画一些[白板](http://hackaday.com/wp-content/uploads/2011/11/board1.jpg) [艺术](http://hackaday.com/wp-content/uploads/2011/11/board2.jpg)的时候，我们真的可以用【Euphy】的绘图仪。当我们在垃圾箱里寻找一些小型步进电机时，请在休息后查看 Polargraph 的延时视频。
