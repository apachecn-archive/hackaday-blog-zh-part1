# Charlieplex 显示器和电路板布局提示

> 原文:[https://hack aday . com/2011/02/07/a-Charlie plex-display-and-a-board-layout-tip/](https://hackaday.com/2011/02/07/a-charlieplex-display-and-a-board-layout-tip/)

![](../Images/b0f354348a56311fc79c22947f5b229b.png "charlieplex-conways-game-of-life")

[Ben]正在让自己跟上微控制器的速度。他接手了这个 Charlieplex LED matrix build 项目，从而进入了深水区。正如你在休息后看到的，他不仅让显示器工作了，还编写了康威的生命游戏在驱动设备的 ATtiny85 上运行。你在上面看到的是原型版本，[本]用来确保他的硬件刚刚好。他看到项目通过一个制造板，这就是布局提示的来源。为了确保他有足够的空间来放置所有的组件，他打印出电路板插图，用胶带粘在一些聚苯乙烯泡沫塑料上，然后插入所有的通孔部件。现在，他可以确信这个设计在物理上是可行的，我们将祈祷一切在电气上也是合理的。
