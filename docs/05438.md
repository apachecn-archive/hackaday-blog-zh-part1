# LabVIEW 的简单数据输入

> 原文：<https://hackaday.com/2010/06/02/easy-data-input-for-labview/>

[https://www.youtube.com/embed/9cKBdn4uHyY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/9cKBdn4uHyY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

道具去【Michael Nash】在[国家仪器的 labVIEW](http://www.ni.com/labview/whatis/) 和一个 Arduino 之间[建立一个接口](http://web.me.com/iklln6/automation/LabVIEW.html)(上面是一个使用电位计的例子视频)。就我个人而言，从我们被迫使用 labVIEW 的那一次[开始，我们就讨厌它的每一秒钟。](http://hackaday.com/2010/03/30/the-first-robotics-competition/#more-22835)

它如此糟糕的一个原因是[数据采集模块](http://www.ni.com/dataacquisition/)的成本高达数百美元，然而文档和帮助资源却非常匮乏。通过使用 Arduino 代替模块，价格和难度降低了很多。这就引出了一个问题，为什么花了这么长时间才得到一个像样的(并且如此简单的)设置？
