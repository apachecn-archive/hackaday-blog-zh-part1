# 用你的思想控制一个播放列表

> 原文：<https://hackaday.com/2012/02/18/control-a-playlist-with-your-mind/>

![](img/ddc75d546d5f32896fff2f1baadba347.png "eeg")

因为切换应用程序来改变一首歌曲是一个如此艰难的考验，[奥斯卡·塞尔玛]和[清-陈为]决定用他们的集体智慧来最后改变。用脑子 FM 播放列表。他们称他们的项目为 Buddhafy，其工作原理是采用现成的 EEG 硬件，并将其与音乐流 API 相结合。

在制作过程中，这些人使用了一个神经天空意识波来读取奥斯卡大脑中的阿尔法波。来自思维波的数据被传入一个 [Python 脚本](https://github.com/akloster/python-mindwave)，该脚本向最后一个发送请求。FM 和 Spotify APIs。脑波模式中的高α波与专注或深度冥想状态相对应。如果[奥斯卡]非常努力地集中注意力，他会得到平静和放松的曲调。如果[奥斯卡]失去焦点，音乐就会切换到[有史以来最好的歌曲](http://www.youtube.com/watch?v=oHg5SJYRHA0)。

这些人上传了他们上周在旧金山 MusicHackDay 上演示的幻灯片。还有一个他们建造行动的视频；休息之后你可以去看看。
