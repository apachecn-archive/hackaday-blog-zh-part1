# PS3 控制器欺骗突飞猛进

> 原文:[https://hack aday . com/2011/05/25/PS3-控制器-欺骗-突飞猛进/](https://hackaday.com/2011/05/25/ps3-controller-spoofing-advancing-with-leaps-and-bounds/)

![](../Images/84f9bd8a022de9e5bc33f6e7059d1081.png "sixaxis-status-gui")

(Matlo 的)PlayStation 3 控制器欺骗项目的开发进展相当顺利。这是一个[包，允许你将一台 PC 识别为 PS3 控制器](http://diy-machine.blogspot.com/2011/01/new-english-tutorial.html)。我们知道你在想什么:你为什么要这么做？大约一年前，当[我们最初关注项目](http://hackaday.com/2010/05/05/custom-ps3-controller-software-emulation-version/)时，我们提到这允许你使用任何 Linux 友好的外设作为 PS3 控制器。在下面嵌入的剪辑中，你会看到，当谈到第一人称射击游戏时，没有什么能打败一个好的键盘和游戏鼠标。[Matlo 的]解决方案不仅允许您使用替代控制硬件，而且几乎有无限的可配置性。

说到配置，他在 GUI 上做了大量的工作。在初始软件包安装之后，不需要输入终端来配置系统。一旦就位，您就可以设置蓝牙加密狗的 MAC 地址来欺骗您的 SixAxis 控制器的地址。从那里你可以设置按钮映射，校准鼠标硬件等等，甚至编程宏(太棒了)。现在出去 pwn 死亡竞赛的每个人，现在 PlayStation 网络已经恢复运行。
