# 活火半衰期

> 原文:[https://hackaday.com/2010/06/23/live-fire-half-life/](https://hackaday.com/2010/06/23/live-fire-half-life/)

![](../Images/8a412b84e7e3bce2ffa12374fa19c312.png "live-fire-half-life")

我们不知道当它第一次出现时我们是如何错过的，但有一个黑客将. 22 口径的手枪与视频游戏半衰期结合在一起。简单是最好的，这句格言用在这里。一堵墙被建下来用作投影屏幕。安装在干墙上的加速度计报告来自子弹撞击的振动数据，该数据用于对其位置进行三角测量。这个目标数据然后被发送到游戏界面。

正如你在休息后的视频中看到的，这非常有效。开发这个的[滑铁卢实验室]人员也负责[那个 iPhone 控制的汽车](http://hackaday.com/2009/11/10/remotely-control-your-crappy-car-dangerously/)。我们在那个项目中目睹的滑稽动作延续到了这个项目，因为他们在剪辑中的 2:12 用一对铲子演示了使用设置来播放半衰期。T3
