# DIY 花束保证让你在情人节被甩

> 原文:[https://hackaday . com/2011/03/10/DIY-flower-bouquet-guarantee-to-get-you-dump-on-valentines-day/](https://hackaday.com/2011/03/10/diy-flower-bouquet-guaranteed-to-get-you-dumped-on-valentines-day/)

![led_flower_bunch](../Images/772006520b7d4323b35ec64ca0e939df.png "led_flower_bunch")

[埃里克]是一个破产的工程师。

当刚刚过去的情人节来临的时候，他没有钱给他的女朋友买礼物，所以他必须变得有创意。每个女孩都喜欢花，但是除非他打算从杂货店的垃圾箱里拿出一些旧的雏菊送给他的女友，否则他必须自己动手做。

他开始把一些 T 恤衫衣架捆在一起，弯成杆状。然后，他用一些 Cat-5 电缆的绿色和绿色/白色线对将它们包起来，给它们一些颜色。牺牲了一个旧键盘来创造花瓣和叶子，大概是在粘在衣架末端之前用马克笔上色。他在花盆上加了一个簧片开关，当一个写着“爱你的埃里克”的磁铁纸板放在花盆附近时，这个开关就会点亮他安装在花瓣下的发光二极管。

他的女朋友喜欢他们吗？这么说吧，她已经不在照片里了。他没告诉我们花束是不是原因，只说她走了。(尽管我们碰巧认为这很酷)。
