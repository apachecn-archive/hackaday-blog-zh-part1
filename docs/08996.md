# 啤酒安全系统让吃白食的人远离你的储藏

> 原文:[https://hack aday . com/2011/11/03/beer-security-system-keepers-freeloaders-out-of-your-stash/](https://hackaday.com/2011/11/03/beer-security-system-keeps-freeloaders-out-of-your-stash/)

密尔沃基黑客空间的工作人员对他们的啤酒相当认真。他们曾经有一个装满易拉罐的冰箱，供 hackerspace 的所有人使用，但他们决定加强这一功能，创建一个安全的啤酒分发系统。

像我们见过的许多其他人一样，他们的 kegerator 内置在一个旧冰箱中，门上还装有一个水龙头。为了确保闯入者远离他们珍贵的啤酒，他们使用 Arduino 和 RFID 标签来确保冰箱的安全。他们使用相同的 RFID 钥匙 fobs 成员进行访问的空间跟踪啤酒消费，解锁水龙头每当一个有效的标签刷过传感器。

他们仍在调整和修改系统，但到目前为止看起来还不错。这是一个让不速之客远离啤酒储藏地的好方法，同时也给他们一个跟踪消费的方法。一旦事情完全结束，我们期待看到更多的细节和代码。
