# CES: 3d 激光投影系统

> 原文：<https://hackaday.com/2010/01/09/ces-3d-laser-projection-system/>

![](img/9dcdba752c5a9d15bfa4b26317a4f2d1.png "3d-laser")

我们真的很想看到这个系统。据说它在空中制造了一个 3D 投影。他们在地板上没有系统，但邀请我们到他们的酒店房间进行私人展示。我们设法找到了房间，并被带进了一个更小的单元，它在一个圆柱体内做了一个大约 4×4 英寸的投影。

这是真正的 3D。这不是老式的有角度的玻璃把戏。他们不让我们拍照或录像。那家伙根本不会回答任何问题。我们推测那只是交叉激光。你可以清楚地看到顶点，就像你在上面的图片中看到的，并且有肉眼可见的扫描刷新效果。我们真的希望我们能看到投射到空中的大装置。
