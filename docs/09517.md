# 使用 Pez 分配器的增强现实

> 原文:[https://hack aday . com/2012/01/17/augmented-reality-using-pez-dispensers/](https://hackaday.com/2012/01/17/augmented-reality-using-pez-dispensers/)

![](../Images/56284a1a97b68c90a903f75de7adc321.png "CARS")

[Johan]真的让自己摆脱了与这个的困境。

在他的儿子开始玩 [AppMates](http://www.appmatestoys.com/) 后，他迫切地想要必要的数字来完成这个体验。AppMates 是一款 iPad 游戏，在这款游戏中，一名儿童驾驶一辆小型塑料汽车在触摸屏上显示的虚拟环境中行驶。对[约翰]的儿子来说不幸的是，官方许可的 [*汽车*人物](http://store.apple.com/us/product/H6925B/A/disney-cars2-appmates?)至少要三个月才能到达他在瑞典的家。[Johan]做了下一个最好的事情，做了他自己的。

【Johan】拿了一对*车* Pez 掌柜，脱图。人物的底部被 Pez 分配器底部的一点塑料和一点导电泡沫覆盖，并被添加到身体上。现在[Johan]的儿子可以开着他的玩具在散热器泉周围转，就像他可以开着官方的小雕像一样。还有就是不要让一个孩子为了一点塑料等上几个月，所以我们称这一次为[Johan]的胜利。
