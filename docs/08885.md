# 万圣节技巧:立体模型

> 原文:[https://hackaday . com/2011/10/20/Halloween-hacks-diorama-Rama/](https://hackaday.com/2011/10/20/halloween-hacks-diorama-rama/)

![](../Images/8a4123db22326d47a0737a46a4e9af08.png "Halloween")

![](../Images/a27d38889fc7e6599b69e6cb95c39fc9.png "castle")

路易斯维尔黑客空间 LVL1 的人要举办万圣节派对。为了展示他的建筑技能，LVL1 成员[JAC_101]组装了一个万圣节立体模型，展示了弗兰肯斯坦博士实验室的内部运作。

有一堆非常整洁的部件使这个版本非常棒。首先出场的是 [LVL1 等离子标志](http://hackaday.com/wp-content/uploads/2011/10/lvl1on.jpg)。这个标志是四个电路通过氙闪光灯泵送高压电荷。一束非常神圣的氙等离子体穿过灯管，而不是明亮的闪光。该标志由四个一次性照相机闪光灯电路构成。

几个闪烁的 LED 手电筒照亮了弗兰恩斯坦博士的巢穴，而[怪物](http://hackaday.com/wp-content/uploads/2011/10/monster_glow.jpg)是一个麦当劳快乐套餐玩具，包裹在手术胶带中，涂有紫外线反应涂料。

为了重新利用现有材料，从~~【九分之七】的再生室~~ LVL1 的 rave 供应柜中取出了一个[等离子圆盘皮带扣](http://hackaday.com/wp-content/uploads/2011/10/plasma.jpg)，提供了一种合适的“疯狂科学家”美学。为了更好地测量，一点 EL 线和一些黑光灯一起被扔了进去，以激活 UV 涂料。

虽然弗兰肯斯坦的实验室墙上少了一个滑稽的超大闸刀开关，但[JAC_101]仍然完成了一个伟大的构建。
