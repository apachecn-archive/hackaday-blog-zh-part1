# 受《星际迷航》启发的口袋门

> 原文:[https://hack aday . com/2010/12/06/star-trek-inspired-pocket-doors/](https://hackaday.com/2010/12/06/star-trek-inspired-pocket-doors/)

![](../Images/09a105d46ecb9e342576fda0d0e5b539.png "star-trek-sliding-door")

你对自己的黑客能力有足够的信心来在你家的墙上建一个项目吗？[马克]用他的技能为他的卧室建造了一扇气动滑动门。它类似于你在企业号上找到的滑动门，两个部分很好地滑入墙壁让你通过。虽然上面的图片显示了内部，但他还是照着做了，最终得到了一个看起来很棒的完全完工的房间。阁楼上的压缩机提供移动门部件所需的压力。它是自动化的，但使用按钮或键盘组合来运行，而不是检测运动。当然，因为他使用 PIC 微控制器来驱动系统，所以总有未来变化的空间。休息过后，看看视频中的成品有多棒。
