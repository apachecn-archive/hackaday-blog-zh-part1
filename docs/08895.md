如果你第一次错过了它们，这里是我们过去一周最受欢迎的帖子。

我们本周最受欢迎的帖子是关于一个表面镶嵌了 256 个 led 矩阵的球，可以显示各种图案。

接下来是一个关于 Arduino 的[振动器护罩的帖子。如果你一定要把性和书呆子气混在一起，这是一条通往幸福的路。这个帖子对于工作来说是安全的，但是链接可能就不安全了。](http://hackaday.com/2011/10/16/join-the-pen15-club-with-a-vibrator-shield/)

在那之后，我们发布了一个关于拍摄全景照片的新方法的帖子。在这篇文章中，你会看到一个嵌入了 36 个摄像头的球。如果你把它扔到空中，它会在行程的顶点拍一张照片。

第四，我们有一篇文章描述如何[给你的笔记本电脑](http://hackaday.com/2011/10/19/beefing-up-your-laptops-gaming-chops-with-an-external-gpu/)添加一个外部 GPU。如果你只有一台笔记本电脑，并且想玩最新最好的显卡，这是你可以选择的一条路线。

最后，如果你喜欢发光的东西，这款适合你。这篇文章链接到一个视频，向你展示如何用家用化学品制造发光氨。
