# 宣布我们的下一个主题——可持续发展秘诀

> 原文:[https://hackaday . com/2011/09/26/announcing-our-next-theme-sustainability-hacks/](https://hackaday.com/2011/09/26/announcing-our-next-theme-sustainability-hacks/)

![](../Images/1c68783e4a3a1085d32c6b2d3944fb4c.png "sustainability hacks theme")

我们的最后一个主题，ATtiny hacks，收到了非常好的响应，但现在是时候继续前进了。今天，我们宣布我们的下一个主题，这将是可持续发展的黑客。在这个主题中，我们将展示能让我们减少足迹的项目。这可能是基于可再生资源的项目，控制系统的项目，允许我们使用更少的能源，如自动风扇，以取代更积极的冷却需求。我们还对展示将功耗推到极限的项目感兴趣。像我们的其他主题一样，我们需要你的帮助才能成功。如果你有一个你认为我们可能感兴趣的项目，请通过[我们的热线](http://hackaday.com/contact-hack-a-day/)告诉我们。

作为精神食粮，驾驶对我们大多数人来说只是生活的一部分。尽管有办法让它对地球产生更少的影响。这方面的一个例子是尾随，虽然现在并不常见，但随着燃料价格的上涨，这种情况可能会变得越来越普遍。这是一个重塑车辆尾部使其更符合空气动力学的过程。你可以在休息后看到一段视频，很好地展示了他们的过程，弥补了声音的不足。
