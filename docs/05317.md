# 多材料 3D 打印

毫不奇怪，我们热衷于 3D 打印，[尤其是【Devlin】](http://hackaday.com/2010/01/12/how-to-make-a-printable-ces-badge/)。现在，我们对上面视频中的[这种多材料 polyjet 机器已经完全失去了理智。在我们走得更远之前，值得一提的是，这个帖子不是广告，我们只是认为这台机器是难以置信的。](http://www.youtube.com/watch?v=D4Yq3glEyec)

它能够同时使用多种材料打印 600 dpi 的 3D 图像。两种类型的刚性材料，一种像 ABS，另一种像聚丙烯，以及七层软材料都存在于同一个打印头上。它们可以与支撑材料同时沉积。在视频中，您可以看到打印机中的附件，硬塑料外壳中已经安装了类似橡胶的填充物。他们甚至展示了打印后完全组装好的自行车链条。这些机器的成本？我们不想知道，只是梦想能不受限制地接触到一个很有趣。
