# 淘气的鸭子将会是你工作场所的神秘圣诞老人的终结

> 原文:[https://hackaday . com/2011/12/13/naughty-duck-will-the-end-of-secret-Santa-at-your-place-of-work/](https://hackaday.com/2011/12/13/naughty-duck-will-be-the-end-of-secret-santa-at-your-place-of-work/)

![](../Images/217c162c521ece5d33639a7bc2c9b4bd.png "naughty-duck")

啊，他是不是很可爱？外表是会骗人的，因为如果你让他开始，[这鸭子会说一些粗俗的话](http://gigavolt.blog.com/naughty-duck/)。[Gigavolt]发现这个小家伙被遗弃在慈善商店，并认为它可能有一些黑客潜力。他是对的。这个普通玩具已经可以一边拍打着它的嘴和翅膀一边唱歌了。在[Gigavolt]的巢穴呆了一段时间后，这只鸭子将被列入淘气名单。最棒的是，由于一个秘密的圣诞老人交换，这将最终落在别人手里。

上面链接的构建文章对于你在工作中阅读是安全的，但休息后嵌入的视频肯定不是。[Gigavolt]开始用自己的 PIC 16F628 微控制器替换内部的集成电路。他使用新的音轨，由 SOMO-14D 音频播放器板播放。这两者使用不同的输入电压水平，这有点麻烦，但这是一个待机功耗，一直令人烦恼[Gigavolt]他使用 DorkbotPDX 命令滚动自己的电路板，但不明白为什么电流消耗如此之高。看看这只诅咒的鸭子，然后看看你是否能解决它的电路问题。
