# Scratchbot:络腮胡前来救援

> 原文：<https://hackaday.com/2009/07/02/scratchbot-whiskers-to-the-rescue/>

[https://www.youtube.com/embed/GTekO_RQCzE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/GTekO_RQCzE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

Scratchbot 被设计成一个救援机器人，去能见度低的地方。它最明显的特征是它用“胡须”来感觉东西。我们觉得这有点噱头。如果是在低能见度的情况下，红外或音频，可能声纳会更有效吗？它如何区分不同的物理障碍？胡须真的是新的吗？难道它们真的只是碰撞传感器吗？也许他们有更复杂的事情要做。还有另一个最近的机器人[，它利用胡须](http://www.botjunkie.com/2009/06/19/video-friday-psikharpax/)并比较不同的触觉轮廓来确定它正在触摸什么。
