# IPhone 上的导弹命令与真正的导弹

> 原文：<https://hackaday.com/2009/12/24/missile-command-on-iphone-with-real-missiles/>

![](img/3f0ae105239095e79e0d18b4900dbdee.png "iphone-missile-launch")【杰夫】和他的团队完成了这个 [iPhone 控制的火箭发射器](http://www.prism.gatech.edu/~jlumish3/ECE4180/Index.html)，作为他们在佐治亚理工学院最终项目的一部分。两个伺服系统提供由机载电子罗盘参考的旋转，以及发射的仰角控制。这些使用几块[电路板与](http://www.phidgets.com/) [eBox 2300](http://www.embeddedpc.net/eBox2300/tabid/110/Default.aspx) 接口。

查看下面的发布视频。太糟糕了，他们用一个商业解决方案来实现伺服控制，而不是自己构建(特别是考虑到它是一个嵌入式系统类)。但是它仍然是一个很好的建筑。现在他们需要在火箭上安装一些成像设备，然后就可以投入使用了。
