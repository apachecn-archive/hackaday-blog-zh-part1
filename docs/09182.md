# diy 汽车照明照亮了设计师忽略的领域

> 原文：<https://hackaday.com/2011/11/28/diy-car-lighting-illuminates-areas-the-designers-overlooked/>

[戴夫]似乎就是对改装他的新车看不够。在他住的地方，下班回家的路上通常很黑，他发现自己经常把东西掉在汽车地板上。日产决定不在 Juke 的脚坑或手套箱中安装照明设备，因此[Dave]认为他需要纠正这种情况。

通过[他之前的修补工作，你可能已经注意到了，](http://hackaday.com/2011/11/11/led-strip-and-uc-add-some-flash-to-your-tail-lights/) [Dave]是 led 和 ATTiny 微控制器的忠实粉丝。他在司机和乘客的脚坑里添加了一些乐高支持的发光二极管，在手套箱里安装了一个额外的灯。

Juke 的仪表板上有几个可用的空白点，所以[戴夫]在其中一个空位上安装了一对电容式触摸传感器和一个小开关。触摸时，最上面的电容引脚对会点亮脚灯，而下面的引脚对会打开手套箱中的 led。底部的开关是用来打开后座上的灯的，那是他不久前安装的。

这是一个简单的加法，但当他在黑暗中在地板上四处搜寻时，就完全不同了。我们对自动点唱机少了多少便利设施感到有点惊讶，但根据他的喜好调整它们似乎让[戴夫]很开心。
