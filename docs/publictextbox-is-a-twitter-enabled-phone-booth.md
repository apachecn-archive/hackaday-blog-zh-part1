# @publictextbox 是一个支持 Twitter 的电话亭

> 原文：<https://hackaday.com/2012/01/27/publictextbox-is-a-twitter-enabled-phone-booth/>

![](img/9403b45795b58dbfb1ed68750e719fd2.png "booth")

为了在共享空间进行公开交流，[dan]制作了一个公共文本框，作为@publictextbox twitter 账户的终端。我们可以看到像这样的东西在黑客空间或其他地方很有用，可以向世界宣布当地的制造商和建筑商发生了什么。

软件安装非常简单，可以在任何你可能放在角落里不用的旧电脑上运行。该应用程序是用 Processing 构建的，[代码](https://gist.github.com/1267563)极其简单，易于修改。尽管这个盒子是一个可爱的纸板数字，但 Twitter 盒子可以装扮成任何可以想象的形式。我们很想看到一个漂亮的 TARDIS blue，但我们会把它留给[dan]。

休息过后，你可以看看 Twitter 电话盒的演示。或者，您可以转发这篇文章，并参加@publictextbox 的负载测试。
