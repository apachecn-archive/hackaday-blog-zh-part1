# 快速项目:硬盘系统计量器

> 原文:[https://hack aday . com/2011/05/18/quick-project-hard-drive-system-meter/](https://hackaday.com/2011/05/18/quick-project-hard-drive-system-meter/)

![](../Images/554ea0d111b99ffb81121a70f93fd03a.png)

[金]送来这个有趣的小项目。他给自己 3 个小时的时间来完成一次破解(不包括研究时间),并且[设法想出了这个很酷的活动计](http://www.headfuzz.co.uk/?q=3HrHDHack)。他处理整个项目就像它是某种竞赛。基本规则已经制定，要求最终产品的实用性，最小的投资，和几乎完全使用垃圾堆件。

使用一个旧的硬盘作为项目的框架以及计量器的“拨号”部分，他将系统负载/硬盘驱动器和 proc 活动计量器组装在一起。该项目的大脑是一个 AVR，他甚至实现了一些 PWM 来平滑事情。他对这个东西的构造做了一些相当详细的描述(你的构建时间中包含了文字记录吗？-50 分！).即使他用的是他自己制造和销售的产品( [OSIF](http://www.robotfuzz.co.uk/OSIF) )，你也可能知道没有它该怎么做。
