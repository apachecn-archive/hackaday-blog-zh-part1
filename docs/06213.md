# 众所周知，万圣节的房子会突然唱起歌来

> 原文:[https://hackaday . com/2010/11/01/Halloween-house-has-known-to-burst-into-song/](https://hackaday.com/2010/11/01/halloween-house-has-been-known-to-burst-into-song/)

![](../Images/d4eaa48972bf07554dc550a512067ecf.png "house-sings-thriller")

万圣节刚过一天，迈克尔·杰克逊的替代者就以一个非常有才华的音乐屋的形式出现了。这所房子不仅接近迈克尔·杰克逊的舞蹈动作，而且能很好地模仿他的声音。这个房子也因制作达路德的[怪兽大杂烩](http://bit.ly/monstermashhouse)和[沙尘暴(Techno)而闻名。YouTube 的 KJ92508](http://bit.ly/SanstormDa) 上传了他的万圣节战利品供大家欣赏。到目前为止，他还没有在光天化日之下做一个如何做的视频，甚至做了一个视频，但这里是希望他会，因为许多人要求一个偷偷的高峰。他提到他使用了“4 个会唱歌的南瓜脸、墓碑、手工雕刻和吹模南瓜、闪光灯、洪水和数千盏灯。”我期待着明年的装饰。这是日常生活中的技术和一点点努力就能做到的又一个例子。休息之后，一定要看看这家公司制作的《颤栗》视频。
