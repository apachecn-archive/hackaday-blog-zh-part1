# 僵尸[杰克逊]机器人随着颤栗起舞

> 原文:[https://hack aday . com/2011/12/31/zombie-Jackson-bot-dances-to-thriller/](https://hackaday.com/2011/12/31/zombie-jackson-bot-dances-to-thriller/)

![](../Images/336a889bffd36c4d28fecddefc5e0c19.png "zombie-jackson-bot-dances-thriller")

[马克·克莱恩]制造了这个小家伙，他称之为机器人温德尔。但是，除非你用它做些有趣的事情，否则像这样的电子动画有什么好处呢？这就是为什么你可以在休息后的剪辑中看到音乐视频惊悚片中与[迈克尔·杰克逊的]编舞相匹配的动作。

这是为[马克]打造的基础。他开始使用 Inkscape 为每个木质部分设计模板。打印出来后，他将每张纸粘在一块 1/4 英寸的胶合板上，用带锯沿着线条切割。我们没有太多的粘合剂喷涂经验，但他提到罐子应该有临时粘合的说明，以便切割后可以移除模板。

在组装过程中，他确保添加伺服喇叭，以便在添加电机时轻松连接。他总共用了五个。两个给轮子，两个给手臂，一个给脖子。protoboard shield 使它们很容易连接到用作控制器的 Arduino。
