# 自动鸡舍门是太阳能的

> 原文:[https://hack aday . com/2010/11/27/automated-chicken-coop-door-is-solar-powered/](https://hackaday.com/2010/11/27/automated-chicken-coop-door-is-solar-powered/)

![](../Images/30858b9aba452a448728c83e19e33267.png "automatic-chicken-coup-door")

[Fileark]最近一直忙于黑客攻击。这一次，他建造了一个太阳能鸡舍门，早上开门，晚上关门。单个电机通过一圈弹簧绳将门打开和关闭。门框两侧都有限位开关，以确保正确定位。上面看到的灰色盒子里是硬件；屋顶线上的太阳能电池板的调节器，一个坏了的不间断电源的电池，以及驱动板本身。运行 Arduino bootloader 的 AVR 芯片监控光电晶体管以检测日出和日落，使用一对继电器适当地驱动门电机。

休息之后，请查看演示和硬件概述。[Fileark]在看到[闹钟鸡笼门](http://hackaday.com/2010/09/01/alarm-clock-learns-new-trick-opening-doors/)后受到启发，制作了他的。我们不知道他是否有机会看看[的垂直鸡笼门](http://hackaday.com/2010/05/03/automated-chicken-coop-door/)，但我们认为他机械上不太复杂的解决方案同样优雅。
