# Twitter、会说话的鱼等等

> 原文：<https://hackaday.com/2009/09/24/twitter-talking-fish-and-more/>

[https://www.youtube.com/embed/Y6kECR7T4LY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Y6kECR7T4LY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

【丹·罗斯】和 mbed 团队派出了他们的黑客[推特大嘴巴比利·巴斯](http://mbed.org/esc/)。最初，黑客只是拥有一个 mbed 微控制器和一个 SD 卡，sd 卡上有一些 wav 文件，允许 Billy 说出经典语录。然而，为了更进一步，他们使用 mbed 的 HTTPClient 库来获取 Twitter 更新，然后让 Billy 大声说出来！查看 mbed 团队在[食谱](http://mbed.org/projects/cookbook/)中的其他一些很酷的项目。

但是等等，什么是 mbed？我们也没有任何线索！

一些研究显示它是一个功能强大的新型微控制器和原型板。具有一些功能，如直接以太网和 USB 连接，以及 PWM、串行、模拟、[和更多](http://mbed.org/tour/)。真正让我们兴奋的是它的行为很像一个 USB 拇指驱动器。拖放你的二进制文件，它的编程！唯一的缺点是，与一张 Arduino 图片相比，100 美元的价格可能更值得投资。这会是击败所有其他工具的下一个大型原型工具吗？让我们在评论中听听吧！

[更新:mbed 的限时预购价格为 60 美元，而不是 100 美元——感谢 Dan 和 mbed 团队！]
