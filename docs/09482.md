# 修复售后前照灯总成的控制方案

> 原文:[https://hack aday . com/2012/01/12/fixing-the-control-scheme-for-a-after-market-head light-assembly/](https://hackaday.com/2012/01/12/fixing-the-control-scheme-for-an-aftermarket-headlight-assembly/)

![](../Images/795da31ea93f170fa9ae6fae8f2d78ea.png "vw-headlight-hack")

[比尔·波特的] 2004 年帕萨特的前灯外壳因年久而发黄，无法发出应有的光线。他决定用一些也包含 LED 灯条的售后模块来代替它们。当他们到达时，他惊讶地发现很容易就到位了。但是在测试时，他肯定对它们的工作方式不满意。白天驾驶模式使用全功率的 HID 灯泡，工厂装配在白天将它们调暗。他开始按照自己喜欢的方式改变电子设备的工作方式。

总是疯狂的科学家，[比尔]开始制作一个真值表，显示灯如何对点火和头灯开关的各种状态做出反应。他想出的是一个由继电器和二极管组成的与门。它允许他将 led 作为运行灯打开(根本没有 HIDs 打开)，而其他功能不受影响。
