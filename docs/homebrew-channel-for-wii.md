# Wii 的自制频道

> 原文：<https://hackaday.com/2008/05/26/homebrew-channel-for-wii/>

[https://www.youtube.com/embed/6ji2imug_bc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/6ji2imug_bc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

玩够了任天堂自制游戏了吗？我们也没有。尤其是现在自制酒频道已经打开了自制酒的大门。到目前为止，你可以在未安装的 Wii 上运行自制程序的唯一方法是[的《暮光之城黑客](http://wiibrew.org/index.php?title=Twilight_Hack)，它利用了《暮光之城公主拯救游戏》中的一个缺陷。家酿频道让你用一个有用的图形用户界面启动各种家酿应用程序，而不是每次你想运行它们时都进行破解。它可以访问存储在 SD 卡上的应用程序，同一网络上的电脑，甚至是 [USB Gecko](http://usbgecko.com/) 。目前没有 USB 闪存驱动器或 DVD 支持。

通过运行 Twilight Hack(不要担心，这可能是最后一次)将自制频道文件加载到 SD 卡的根目录中，可以将自制频道加载到 Wii 上。Wii 将重新启动，然后该频道将出现在列表中。我们自己测试了一下，发现所有东西都可以从 SD 卡上正确加载(我们没有尝试其他来源)。我们确实遇到了一个问题，如果 Gamecube 存储卡在插槽中，它无法加载我们的任何自制应用程序，甚至无法正常重启，但这很容易解决，只需将其拔出即可。

这一版本背后的开发团队希望尽可能使事情变得简单和容易，所以他们包括了下载链接到 Twilight Hack，Homebrew Channel，甚至一个 Homebrew 软件包来帮助你开始。如果你想要更多的自制软件，去 Wiibrew 吧。
