# 用汽水罐做盒子

> 原文:[https://hack aday . com/2011/01/21/用汽水罐制作盒子/](https://hackaday.com/2011/01/21/making-boxes-from-soda-cans/)

这个闪亮的小盒子是由一个汽水罐制成的。你不需要太多就能完成这件事；一个铝罐，砂纸，剪刀，圆珠笔，直尺，还有一些时间。浮雕是用笔尖完成的，但是有一点小技巧。设计首先从铝的下面压入金属。然后将它翻转过来，描摹轮廓，完成后从底部最后描摹一次形状。我们认为你会同意，这导致了一个令人印象深刻的设计救济。

这将成为你随身携带的结婚戒指的一个很好的项目。或者这正是你下一个项目所需要的外壳。休息之后你会看到一段教学视频。
