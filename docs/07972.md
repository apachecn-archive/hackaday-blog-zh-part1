# 电活性聚合物

> 原文:[https://hackaday.com/2011/07/01/electro-active-polymers/](https://hackaday.com/2011/07/01/electro-active-polymers/)

![](../Images/646a14376d3787a51806d756ea214f73.png "electro-active-polymers")

你说什么？你有接触电流时会改变形状的刚性材料？为我们报名吧！虽然制造过程看起来有点令人生畏，但我们喜欢用电活性聚合物工作的结果[。当果汁打开时，这些塑料片可以通过向一个方向收缩而弯曲。它的效果非常类似于肌肉线，但分布在更大的区域。](http://www.youtube.com/watch?v=4XGVMXCxBNA)

从休息后的视频中我们看到，这似乎不是最有弹性的材料。一些测试镜头打破了玻璃，但我们相信这将随着时间的推移而改善。看起来有一些关于制造你自己的 EAP 的信息，但是这个过程看起来并不比在研究水平上进行的更容易。我们可能会坚持[建立我们自己的空中力量](http://hackaday.com/2008/12/21/scratch-built-air-muscles/)，直到 EAP 更容易为项目采购。
