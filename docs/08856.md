# 无电池酸铝阳极氧化染色

> 原文:[https://hack aday . com/2011/10/17/阳极氧化-染色-无电池铝-酸/](https://hackaday.com/2011/10/17/anodizing-and-dying-aluminum-without-battery-acid/)

虽然许多人已经在家里尝试过阳极化铝，但也有很多人宁愿把它留给专业人士，因为这个过程需要高度浓缩的硫酸。[Ken]开始思考这个过程，并想知道是否有一种方法可以使用更容易获得和处理的化学物质来获得类似的结果。

通过一些实验，他发现硫酸氢钠(NaHSO4)是一种硫酸的钠盐，可以很容易地用在它的位置上，并取得很好的效果。这种化学物质通常在五金商店和泳池店被宣传为“Aqua Chem ”,而且价格非常合理。当与适当的 DC 电流以及阴极配对时，硫酸氢钠很容易阳极化铝工件，并使其易于用 RIT 染色，RIT 是一种容易获得的布料染料。

我们对结果印象深刻，当查看[Ken 的]测试件时，似乎用硫酸氢钠染色的金属颜色更均匀，条纹更少。还值得一提的是，[Ken]发现在阳极氧化之前使用氢氧化钠溶液蚀刻铝是相当容易的，这对于喜欢更哑光表面的人来说是非常好的。
