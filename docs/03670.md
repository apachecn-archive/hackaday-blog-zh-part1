# 各种坎特纳建筑

> 原文:[https://hackaday.com/2009/07/07/various-cantenna-builds/](https://hackaday.com/2009/07/07/various-cantenna-builds/)

这是一个用于增强无线信号强度的经典项目。Cantennas 使用波导聚焦，就像放大镜聚焦光线一样。[罗伯特]做了一个[天然淡啤酒罐](http://img269.imageshack.us/img269/3633/0629091907.jpg)，左上图。他的方法使用了三个啤酒罐、一个纸巾架和一个浴帘杆。在[终端](http://hackaday.com/contact-hack-a-day/)，他注意到信号从 11Mbps 提升到 54Mbps。如果我们的房间缺乏足够的信号，这肯定是我们可以一起对付的。休息后阅读抛物线和寻求版本。

右下角的图片是一个使用旧卫星天线建造的抛物线汤罐的例子。虽然结构值得注意，但似乎没有引用任何数字。过去，我们已经成功地使用了[的老式卫星天线](http://www.engadget.com/2005/11/15/how-to-build-a-wifi-biquad-dish-antenna/)。右上和左下的图片是扫描天线。他们使用伺服系统，通过移动和检查信号强度来寻找更强的连接。下面是左下[坎特纳扫描](http://www.youtube.com/watch?v=Z5XwqB709kk&NR=1)时的在线视频。也许你想知道[如何搭建一个锡罐波导 WiFi 天线](http://www.turnpoint.net/wireless/cantennahowto.html)。所有的说明都提供了一个粗略的成本细分，该网站有一个计算器来确定截止频率给定一个罐的直径。
