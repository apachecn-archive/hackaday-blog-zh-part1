# 万圣节道具:变声蠢朋克服装

> 原文：<https://hackaday.com/2010/10/31/halloween-props-voice-changing-daft-punk-costume/>

![](img/2af55e6ca519e9caad4a32b79c7a8679.png "voice-changing-helmet")

[韦斯特博士]和我们分享了他的万圣节服装；一个受蠢朋克启发的变声头盔。他戴着摩托车头盔盯着看，在后面割出一个洞来放超低音扬声器。里面有一个旧的电脑麦克风和便携式立体声系统的放大器电路。Arduino 用于从麦克风中拾取佩戴者的声音，并执行数字信号处理。一旦完成更改，信号将被发送到一个 R-2R 梯形电阻以执行数模转换，并传送到放大器进行广播。休息后在视频中听到结果。

头盔的其余部分是装饰性的。他发现了一种叫做 flex-edging 的汽车车身修复产品，用作金属毛发。那些鳍被红色和蓝色的发光二极管串所强调。面板使用立体声系统的扬声器和有色护目镜来完成外观。

他不想要一个复制品，但是我们认为他的作品应该和原作看起来一样。
