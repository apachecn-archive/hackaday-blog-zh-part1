# 廉价的吸声板 DIY

> 原文:[https://hackaday.com/2011/10/21/cheap-acoustic-panel-diy/](https://hackaday.com/2011/10/21/cheap-acoustic-panel-diy/)

![](../Images/4d43e182a3c4ce6b5720c37896a5efe3.png "acousticpanels")

[Eric Wolfram]写信给我们，让我们了解他自己组装的简单而便宜的[吸声板](http://acousticsfreq.com/blog/?p=62)。安装家庭影院时，音响效果往往被忽视(尤其是如果你在电视上花了太多钱，买不起家具的话)，导致聆听质量下降，环绕声系统的空间声音成像不佳(也是家具问题的原因)。吸音板的添加有助于控制房间的音响效果，甚至可以提升房间的档次。这些对于家庭工作室来说也很方便，因为那里的混响水平较低。

这种面板生产起来相对简单，只需要一张 2 英寸厚的致密玻璃纤维板，粘在木框上，外面覆盖一层透声织物。[Eric]在材料选择过程中为你提供了很多帮助。这个项目最好的部分(除了它明显的实用性)是除了布料之外，所有的材料都可以在普通的家装店里便宜的买到。[Eric]提到如果需要，您可以用彩色粗麻布代替。一旦面板组装好并粘好，就可以像一个大相框一样挂在你选择的墙上。这当然有助于改善音响效果，减少仓库/商店里的回音。我们周末可能要试试这个。
