# 防止他人窥探您的数字生活

> 原文：<https://hackaday.com/2011/12/20/keep-others-from-snooping-in-your-digital-life/>

![](img/b2dd0fa0204a91d0b768103a53ec4554.png "computer-security-for-the-occupy-movement")

在我们的数字时代，窥探的目光无处不在。可悲的是，它们甚至可能属于你自己的政府。但是不管是谁，你都可以做一些事情来尽可能保证你的私人数字设备和内容的安全。

上面的链接指向[Jerry Whiting]关于这个话题的讨论。他当然是一个有趣的演讲者，但确保你在工作中使用耳机，因为语言有时会有点闷热。他针对占领运动的教训，但它是一个有趣的听任何阴谋论者在那里。这些主题涵盖了所有领域，从物理访问的幽灵开始，然后转移到通过流量分析和使用密钥对来保护您的网络。这个*安全 101* 片段分为两部分(第一部分是在休息后嵌入的)，每部分都超过三十分钟。他计划发布第二课，内容涉及哈希和加密。
