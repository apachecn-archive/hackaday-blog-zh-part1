# 当音乐是最重要的东西时，DIY 音频播放器

> 原文:[https://hackaday . com/2011/11/18/a-DIY-audio-player-for-when-all-the-matters-the-matter-the-music/](https://hackaday.com/2011/11/18/a-diy-audio-player-for-when-all-that-matters-is-the-music/)

![orange-mepod](../Images/caafe2aa77fbd4ad812bc7f2d086cf19.png "orange-mepod")

说到个人音频播放器，[Grissini]运气不佳。他估计这些年来他已经经历了半打 iProducts/iknockoff，最终加起来相当于一大笔钱被白白浪费掉了。格里西尼决定，如果自己造一台音乐播放器，他会过得更好，而不是拿出他冰冷的现金购买另一台电池没电或屏幕破裂的音乐播放器。

他的橙色 mePod 并不是最吸引人或最时尚的音乐播放器，但[Grissini]说它非常好用。Arduino Uno 为设备供电，他使用 Adafruit 波盾来处理音频回放。电源通过 4AA 电池提供，可以在合理的时间内保持曲调，并让他能够毫不费力地更换电池充电。

这名球员被用一些剩余的硬纸板包裹起来，用亮橙色的胶带包裹，然后被绑在(格里西尼的)腰带上。他说，当他出门在外时，他得到了很多关注，这是你可以从这样一个独特的设计中期待的。
