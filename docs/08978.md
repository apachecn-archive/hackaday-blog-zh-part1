# 非常逼真的骷髅道具

> 原文：<https://hackaday.com/2011/11/01/amazingly-realistic-skeleton-prop/>

![](img/2fb54624efae0dd65e31108041e98575.png "realistic-motorized-skeleton")

[Cjmekeel]花了几周时间准备今年的万圣节展览。他的核心产品是这个全尺寸的机动骨架。但是他还做了一些其他的宝石来赞美它。有一台老式收音机，它的表盘神秘地移动，播放一条新闻，警告一名逃跑的精神病人。他还花了几美元给一只乌鸦装上了发光的红眼睛和伺服电机。

但是令人毛骨悚然的骨架意味着你可能甚至不会注意到其他的道具。他从一个看起来相当无聊的普通塑料头开始，变了一些真正的魔法来构建腐烂的肉体和裂开的伤口。那双穿透的眼睛也不疼。磁头在几个伺服电机上移动，这些电机使用随机值和睡眠周期来进行令人不安的不平稳运动。休息后看看视频，看看是什么让孩子们在万圣节远离他的房子。

这只是一个构建日志，不幸的是还没有显示成品的帖子。如果我们能收集到足够的信息，我们会试着跟进。
