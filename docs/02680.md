# USB 认证的死栓锁

> 原文:[https://hack aday . com/2008/10/22/USB-authenticated-dead bolt-lock/](https://hackaday.com/2008/10/22/usb-authenticated-deadbolt-lock/)

![](../Images/b61e0cdd24a4673db47b90241fc38e63.png "usb_auth")

制造商 local 256 向我们发送了这个 [USB 认证的死栓项目](https://256.makerslocal.org/wiki/index.php/USB_Auth)。这些人花了大约 60 美元建立了一个认证系统，可以读取 USB 存储设备中芯片的序列号。USB 设备中存储器上的实际内容根本不会被使用。他们使用 Freeduino 板来控制它的行为。它有一个磁性传感器，防止它在门打开时启动锁。他们提到他们使用透明铝作为外壳，我们假设他们指的是[星际旅行品种](http://en.wikipedia.org/wiki/Transparent_aluminum)，而不是[氧氮化铝](http://en.wikipedia.org/wiki/Aluminium_oxynitride)。休息后请务必观看视频。
