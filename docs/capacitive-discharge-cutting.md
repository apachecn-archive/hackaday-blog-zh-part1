# 电容放电切割

> 原文：<https://hackaday.com/2009/07/04/capacitive-discharge-cutting/>

![capacitive discharge cutting](img/5609cbd15bd8de3927d3c258462ac901.png "capacitive discharge cutting")

电容放电切割提供了比线性变压器版本更多的控制。一个非常大的电容器被充电到一个精确的电压，然后通过材料放电产生一个受控的切割。同样的设备也可以用于点焊和凸耳焊接。下面嵌入了铜制屋顶防水板被切割的视频。线性变压器的例子可以在我们的 [How-to:建立你自己的点焊机](http://hackaday.com/2009/06/23/how-to-build-your-own-spot-welder/)中找到。
