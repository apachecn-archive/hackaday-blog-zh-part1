# 素描步骤

> 原文：<https://hackaday.com/2010/01/04/step-a-sketch/>

![](img/3db415e6d6f7a64c975e3e96fbdc4d15.png "step-a-sketch")

[Chris]正在尝试计算机数字控制，从[一个蚀刻草图界面](http://poweredbynerd.com/designs/step-a-sketch/)开始。这是一个很好的开始方式，因为项目中真正困难的部分已经在玩具内部了。他用步进电机取代了两个白色旋钮，并通过 mosfet 网络将它们连接到 PIC 16f84a。PIC 然后通过并行端口从计算机获取命令。

休息后可以看到数控机床的视频。他需要添加一个框架来提高图像绘制的精确度，但这第一次尝试是相当不错的。我们更喜欢让计算机来负责设计，因为用鼠标控制草图并不会让我们的图纸更好。T3
