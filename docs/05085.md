# XBMC 躲在一个 SNES 控制器里

> 原文:[https://hack aday . com/2010/03/23/xbmc-hiding-in-an-snes-controller/](https://hackaday.com/2010/03/23/xbmc-hiding-in-an-snes-controller/)

![](../Images/b0dbbbc653c6178f912c0d8e8cdc1cc9.png "XBMX-inside-SNES-controller")

这是另一个 [SNES 控制器，被改造成一个 USB 系统](http://www.youtube.com/watch?v=XVFaY2ETQ_Y)。我们上次看到的[使用了一个套件作为控制器的适配器，但这个版本使用了自制的 PCB 和 ATmega8 微控制器，](http://hackaday.com/2009/12/22/snes-system-in-the-controller/)[是 NES 和 SNES 控制器的开源适配器](http://raphnet.net/electronique/snes_nes_usb/index_en.php)的最新版本。正如你在休息后看到的，[Atarity]建立了适配器，然后添加了 USB 集线器和拇指驱动器，这样他就可以从控制器上运行 XBMC 的副本。现在他有了 XBMC，可以为这些经典游戏推出模拟器，也可以播放传统媒体。

你很快会看到更多这种类型的 mod。有人向我们透露，一个关于 SNES 控制器黑客的深入教程正在编写中，尽管这与[Atarity]的工作无关。
