# DIY 激光绘图仪

> 原文：<https://hackaday.com/2009/10/25/diy-plotter-with-laser/>

![plotter-with-300w-laser](img/5922289429d306dfe679c073e0ca5f54.png "plotter-with-300w-laser")

[Viacheslav] [造了一个又快又准的绘图仪](http://sensi.org/~svo/motori/)。他想更进一步，用激光代替了笔。这个 300 兆瓦的装置在烧木头方面做得很好，但除了薄膜，它还不足以作为激光切割机。我们想知道它是否可以用来切割[表面贴装 PCB 生产](http://hackaday.com/2009/10/13/how-to-populate-a-surface-mount-pcb/)的锡膏模版。

在休息后的视频中，你可以看到一些使用钢笔的绘图。除了在纸上书写之外，[Viacheslav]还测试了将这种方法应用于 PCB 生产中覆铜板的抗蚀剂。他能够实现 0.8 毫米的间距，但生产过程受到抗蚀剂笔易于快速磨损的限制，并且只能在短时间内防止蚀刻(与墨粉转移抗蚀剂相比)。

就像他的[触摸感应键盘项目](http://hackaday.com/2009/09/18/touch-sensitive-keypad/)一样，他花时间彻底[记录他的工作](http://sensi.org/~svo/motori/)。构建说明、图片、CAD 文件和源代码都可供您阅读和使用。
