# RFID 智能卡读卡器

> 原文:[https://hackaday.com/2011/05/17/rfid-smart-card-reader/](https://hackaday.com/2011/05/17/rfid-smart-card-reader/)

![](../Images/13643babdbb6a6e65f2350c977f5c42e.png "rfid")

[纳维克]来信向我们展示他的最新项目。这是一款[便携式智能卡读卡器，配有漂亮的液晶显示屏](http://sites.google.com/site/airwavershr/Home/rfid-smart-card-reader)。他只是碰巧有一个基本的邮票 2px，智能卡读卡器，和智能卡读写器坐在周围等待使用。有什么比手持智能卡阅读器更好的用途呢？在一个漂亮的项目外壳中把这些碎片拼凑在一起，[Navic]搜索他可用的代码，拼凑出他需要的东西。现在，当你滑入一张卡片，你会在那个漂亮的蓝色屏幕上得到一个很好的数据读数。不幸的是，如果你在读取完成前拔下卡，一切都会停止。

你可以在休息后看到最终的视频，也可以在 youtube 上看到一些链接的正在进行的视频。他问是否应该增加写作的能力，我们说是。存储数据，然后写入(复制)到另一个设备。
