# 当斧头不够用的时候

> 原文：<https://hackaday.com/2009/11/19/when-an-axe-isnt-enough/>

![](img/67baf2fc9d301fbba97699d2d618cf02.png "guitar-plus-keyboard")

[克里斯]想要一把带键盘的吉他，但不想要键盘。像任何优秀的黑客一样，他拿了一把便宜的吉他和一个小键盘，然后[把他们介绍给彼此](http://plentyprojects.blogspot.com/2009/11/secret-project.html)。他移动控制旋钮为键盘腾出空间，并将控制电路从键盘添加到吉他主体的顶部。一个 9v 的电池用来为按键供电，一种叫做“克里斯·柯林斯的变压器戏法”的东西用来放大声音。如果你知道这个变压器的细节，请留下评论，我们会更新帖子。休息后看一下视频，看看[克里斯]用琴键和吉他表演最后的倒计时。

这把吉他是中国制造的电视转播克隆品，我们认为他是第一个发现不使用全尺寸键的键盘的现实用途的人。[Chris]对缺乏构建照片表示歉意，但我们给他一个通行证；他在人群中冲浪时丢失了手机，这是丢失照片的最好借口。一旦他有了足够的练习来把所有的东西组合在一起，我们迫不及待地想看他用这把粗糙的斧子的另一场表演！
