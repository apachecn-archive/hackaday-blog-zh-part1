# RCA DSB772WE 的拆卸表明，这种廉价的流媒体盒可能有潜力

> 原文:[https://hackaday . com/2011/12/03/RCA-DSB 772 we-tear down-shows-this-cheap-streaming-box-may-have-potential/](https://hackaday.com/2011/12/03/rca-dsb772we-teardown-shows-that-this-cheap-streaming-box-might-have-potential/)

![RCA-DSB772WE](../Images/44c6691636fb3bd2d8b555b642611797.png "RCA-DSB772WE")

[大卫·安德斯]来信分享了他在当地沃尔玛买到的一个便宜小玩意的一些细节。他以 48 美元的价格获得了 RCA DSB772WE 媒体流盒，到目前为止，它看起来可能是他客厅的一个有前途的附加物。

他为盒子创建了一个项目页面，详细介绍了他迄今为止的一些发现。该设备基于 MIPS，开箱即可运行 Linux 内核版本 2.6.28.9。网络组件基于 Broadcom BCM7615 芯片组，尽管[David]认为以太网插孔在生产过程中被移除了。

到目前为止，他已经设法在设备上运行了一个串行控制台，以及一个额外的 USB 主机连接。这就是他迄今为止所做的所有探索，但鉴于这个盒子可以通过 HDMI 输出 1080p 信号，它可能是苹果电视或类似设备的廉价替代品。

如果你家里碰巧有一个，或者正打算买一个，一定要看看他的项目页面，并提供你可能从中收集到的任何信息。我们确信[大卫]会喜欢它，我们当然也期待看到他的黑客冒险之旅还会有什么收获。
