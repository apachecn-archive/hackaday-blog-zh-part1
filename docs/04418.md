# 遥控割草机

> 原文：<https://hackaday.com/2009/11/14/rc-lawnmower/>

![FJI64OAG1XBRNEO.MEDIUM](img/9855e7513be510ce36fe915ee44051d5.png "FJI64OAG1XBRNEO.MEDIUM")

[Johndavid400]建造了这个[看起来难以置信的遥控割草机](http://www.instructables.com/id/Arduino-RC-Lawnmower/)。在花了一些时间修理坏掉的遥控汽车后，他想转向更有用和更强大的东西。他决定制造一台割草机。他使用易趣上的收发器，Arduino 解释信号并输出到他的定制马达板上。在休息后的视频中，你可以看到割草机看起来反应迅速。然而，他确实指出，在这个过程的早期，他遇到了一些小故障，导致割草机失控。我们几年前报道过一个[非常相似的割草机](http://hackaday.com/2007/02/11/remote-control-lawn-mower/)，它使用轮椅作为底座。
