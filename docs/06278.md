# 日本微型飞机

> 原文:[https://hackaday.com/2010/11/11/japanese-micro-planes/](https://hackaday.com/2010/11/11/japanese-micro-planes/)

![](../Images/578f3ea2300b8167ee92775da0e85c95.png "indoapueen")

一些设计非常好的微型飞机已经在网络上嗡嗡作响。这里的目标是超轻重量。这些竖起来的纸飞机有着惊人的目标重量，大约为 [10 克](http://www.kyoto.zaq.ne.jp/tkd/meIndoorAirPlane.htm) ( [翻译过来就是](http://translate.google.com/translate?u=http://www.kyoto.zaq.ne.jp/tkd/meIndoorAirPlane.htm&sl=ja&tl=en&hl=&ie=UTF-8))。微型飞机越轻，速度就越慢，机动性就越强，这将导致一些非常有趣的应用。对于控制，看起来许多飞机都使用红外接收器/发射器(很像你在电视遥控器上看到的提示)。尽可能获得最轻的飞机迫使设计师们想出一些非常巧妙的技巧。例如，他们使用缠绕在稀土磁体上的线圈来控制襟翼，而不是使用包装好的伺服系统。休息之后，你可以看到这些自制的“伺服系统”在工作。

一些人采用了更经典的方法，用[橡皮筋动力](http://www.youtube.com/watch?v=73KHyvpuMz0)代替 Li-po/马达组合。
