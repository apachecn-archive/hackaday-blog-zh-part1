# 拖放编程让孩子们很早就开始

> 原文：<https://hackaday.com/2011/04/15/drag-and-drop-programming-gets-kids-started-early/>

虽然为 Arduino 编程对于那些已经在这个街区呆过几次的 EEs 来说是小菜一碟，但是仍然有一些人会发现很难开始使用 IDE。它被吹捧为易于使用，但有一个陡峭的学习曲线，如果你是 5 或 6 岁。[Julian da Silva]已经努力工作了一段时间，以使 Arduino 比以往任何时候都更容易使用。

今天早些时候，我们发布了一个关于[可塑油灰的故事，孩子们](http://hackaday.com/2011/04/15/squishy-circuits-for-tiny-tinkerers/)可以用它来构建基本的电路，使他们能够在年幼时进入爱好电子产品的有趣世界。[Julian 的]项目“Minibloq”旨在对 Arduino 做同样的事情。作为一项正在进行的工作，Minibloq 使用图形界面一次“构建”一块 Arduino 代码。代码组件被拖放到屏幕的一边，而源代码是在另一边生成的。这有助于温和地向那些 Arduino 新手介绍如何编写实际代码，一次一点。

[Julián]正在努力确保他的应用程序在 OLPC 和其他面向教室的计算机上运行良好，以确保它能够达到尽可能广泛的受众。我们认为这将是一个伟大的介绍世界的微控制器为儿童以及那些谁从来没有修补与电子产品在任何时候在他们的生活。
