# 安全锁:生物识别打字安全

我们已经看到了一些绕过生物安全措施的方法，但是这里有一个我们认为很难欺骗的新产品。[安全锁系统与密码](http://www.youtube.com/watch?v=_vMb9JUhC1g)结合使用，以识别特定用户。这个软件记录你的打字风格，包括击键间隔时间、按键时间和按键压力数据。然后将这些信息标准化，并与最初设置密码时存储的用户信息进行比较。如果您不符合与存储数据相匹配的规范，即使使用正确的密码也无法进入。

锦上添花的是 Safelock 会寻找恶意用户。如果你输入了错误的密码，它会开始记录和分析你的打字风格。如果你做了足够多不正确的尝试，你将被贴上安全威胁的标签，并被系统完全锁定。我们只能想到一种可靠的方法来避免这种情况，那就是使用中间人的方法来记录合法用户的键盘输入，以便以后播放。

这是一个创新的用户识别系统，我们不是唯一这样认为的人。在 2009 年用户界面软件和技术研讨会上， [SMU](http://www.smu.edu/) 的学生【杰夫·艾伦】和【约翰·霍华德】获得了[学生创新大赛](http://www.acm.org/uist/uist2009/program/sicwinners.html)的一等奖。
