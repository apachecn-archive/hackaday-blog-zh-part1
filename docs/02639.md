# Playstation 坦克控制器

> 原文:[https://hack aday . com/2008/10/12/PlayStation-tank-controller/](https://hackaday.com/2008/10/12/playstation-tank-controller/)

![](../Images/3e3c2cdd9f6fea2366bda8202ac0b582.png "playstation_tank")

[stuart]想要一种比他一直使用的软件解决方案更直观的方法来控制他的[漫游车平台](http://www.daphreak.com/projects/rover.html)。他决定使用带 AVR 的 [Playstation 控制器来无线操作漫游车。他发现一对](http://www.daphreak.com/projects/ps2-int.html)[夫妇](http://www.gamesx.com/controldata/psxcont/psxcont.htm) [引用](http://www.curiousinventor.com/guides/ps2)在 PIC 上使用 SPI 与控制器对话，并将其用于他的 ATMega88。代码可以在他的网站上找到。他从控制器上拆下隆隆马达，把芯片和发射器包塞进去。Linx TX/RX 对安装在可插拔板上，因此他可以在其他项目中使用它们。跳跃之后是坦克运动的视频。
