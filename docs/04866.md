# 作为乐器或 Midi 控制器的吉他英雄

> 原文:[https://hackaday . com/2010/02/04/guitar-hero-as-a-an-instrument-or-midi-controller/](https://hackaday.com/2010/02/04/guitar-hero-as-an-instrument-or-midi-controller/)

[https://www.youtube.com/embed/azfgasnWtW0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/azfgasnWtW0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Robert]用 Max/MSP 写了一个程序，让他用他的吉他英雄控制器创作音乐。休息之后还有另一个视频，他介绍了各种功能，但这里是它的要点。这可以在 Mac 和 Windows 上工作，并允许一种“现场播放”或 midi 映射模式。在 midi 模式下，每个键都可以根据您的要求进行配置。他的例子使用选择栏滚动不同的样本，绿色按钮播放它们，红色按钮停止。

直播模式让我们更加投入。在软件中，你可以选择音阶的类型和你想弹奏的键。这弥补了控制器缺乏足够的品，使其成为半音阶乐器，这些设置可以从控制器中调整。有一个向上的拾音偏移，使拾音条的向上移动与向下移动成为不同的音符。运动控制也可以用作输入。他[用那种方法演示了弯音和截音](http://www.youtube.com/watch?v=-1AnfCwl260#t=4m56s)。

这看起来很有趣。他需要和 Joran 合作，在混音中加入鼓点，组成一个比你在商店里能买到的更有创意的摇滚乐队。
