# 把起重机游戏带回家

> 原文:[https://hackaday.com/2010/02/24/bring-the-crane-game-home/](https://hackaday.com/2010/02/24/bring-the-crane-game-home/)

![](../Images/283e587a09b474d73a59fd77eca9499d.png "home-crane-game")

每个人都熟悉四分之一狼吞虎咽的起重机游戏。经常会有一个孩子在附近乞求父母给他们更多的硬币，这样他们就可以试着去抓玩具。马克。Cryan]通过[制作一个家庭版的起重机游戏](http://www.instructables.com/id/CRANE-GAME)来更好地利用他的住处。

[马克]在他家的拱门上安装了一个门架。起重机小车骑在该龙门架上，并使用一个卷轴来升高或降低爪的系绳。赢得丰富的风格点，他用老老鼠的情况下形成了爪。Arduino 控制系统中不同的马达，一个玩具被重新设计成控制器。正如你在休息后看到的，它比电影大厅版本的游戏更有趣，你的孩子可以免费玩它。
