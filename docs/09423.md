# 由塑料碗制成的便携式电子鼓套件

> 原文:[https://hack aday . com/2012/01/03/portable-electronic-drum-kit-made-from-plastic-bowls/](https://hackaday.com/2012/01/03/portable-electronic-drum-kit-made-from-plastic-bowls/)

![portable-drum-kit](../Images/9cd237874e6da0b949839de714ea01a5.png "portable-drum-kit")

(伊恩·科尔的)儿子正在学习用电子鼓打鼓，他想在频繁拜访爷爷奶奶家期间继续练习。[Ian]在今年夏天早些时候买了一个 Spikenzielabs“全包鼓套件”(DKKAI)，并着手制作一套易于运输的鼓。

DKKAI 配备了一个基于 ATmega168 的电路板和一组压电元件，可用于记录点击量。剩下的工具由[Ian]提供，所以他去宜家寻找便宜耐用的鼓面。他拿着一把 1/2 升的塑料碗回来了，他把这些碗放在一个 PVC 管鼓架上。

压电装置安装在薄铝盘上，铝盘又粘在碗盖的背面。压电通过 PVC 管连接到 DKKAI 套件，信号最终馈入 iPad 运行的车库带。[Ian]说他的便携式架子鼓效果很好，虽然有些地方需要改变，但他的儿子对他的新练习套非常满意。
