# 手工制造的机械手

> 原文:[https://hackaday.com/2010/05/07/hand-made-manipulator-arm/](https://hackaday.com/2010/05/07/hand-made-manipulator-arm/)

![](../Images/5372fc930e0778afa5f7b829ba41d04b.png "hand-made-manipulator-arm")

我们非常习惯于看到 CAD 在大多数东西的设计过程中使用。这是一个有点令人震惊的遇到一个项目，这涉及，这执行得很好，没有使用 CAD。

[安东]花了 100 个小时[手工打造这个机械手](http://letsmakerobots.com/node/18504)。他把零件画在苯乙烯上，然后用剪刀剪下来。他已经开始用 AutoCAD 构建第二版，但从休息后的视频中我们可以看到，对原始设计的改进将是微小的。增加了磁性编码器的伺服系统的速度和流动性有助于优雅的机器人舞蹈；我们很想成为它的[国际象棋伙伴](http://hackaday.com/2010/04/14/robotic-chess-opponent/)。 <https://www.youtube.com/embed/1qpc4G3NQhk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
