# 简单的 PCB 腐蚀剂，由可以放进嘴里的化学物质制成

> 原文:[https://hack aday . com/2011/03/03/simple-PCB-腐蚀剂-由化学品制成-你可以把它放进嘴里/](https://hackaday.com/2011/03/03/simple-pcb-etchant-made-from-chemicals-you-can-put-in-your-mouth/)

![etching_test](../Images/eb131729322f75f4a15b377508268b5e.png "etching_test")

[Stephen]经常发现有必要在家里制作自己的 PCB，当他最近有做蚀刻的冲动时，他意识到他刚刚摆脱了“氯化物和官僚酸”他没有被空的化学药品柜吓倒，而是在厨房里四处摸索，将任何可能有能力从 PCB 上剥离铜的东西混合在一起。

现在，我们不一定推荐这种做法，但似乎他终于找到了一个赢家。他发现了一种可以在家里用简单安全的家庭原料制作的配方，效果非常好。不过，一个合理的警告是，使用这种溶液时，需要遵循标准的氯化铁处理程序。

如果你想知道他在厨房里调制了什么，以及背后的化学反应，你必须访问他的网站，我们不会破坏你的网站。然而，你可以在我们下面发布的视频中看到解决方案。
