# 用 Mbed 构建的吉他回声踏板

> 原文:[https://hack aday . com/2010/03/28/guitar-echo-pedal-build-with-mbed/](https://hackaday.com/2010/03/28/guitar-echo-pedal-built-with-mbed/)

![](../Images/1e6a5af4883b5b9ba9cc391f1569790f.png "mbed-echo-pedal")

[Vsergeev]为吉他或其他音频处理应用程序构建了[回声踏板。他在这个项目中使用了 mbed 微控制器。你可能还记得黑客作家[菲尔]](http://dev.frozeneskimo.com/embedded_projects/audio_echo_effect) [给 mbed 贴上了“类固醇上的 Arduino”](http://hackaday.com/2009/11/21/review-mbed-nxp-lpc1768-microcontroller/)的标签，它确实很好地处理了这种音频处理。休息之后，我们加入了回声效果的剪辑。在设计过程中，[Vsergeev]使用 [LTspice](http://www.linear.com/designtools/software/#Spice) 来模拟模拟电路，并在投入物理电路之前做出正确的事情。
