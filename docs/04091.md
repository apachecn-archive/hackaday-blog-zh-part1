# 本·赫克变得邋遢，我们喜欢这样

> 原文：<https://hackaday.com/2009/09/23/ben-heck-gets-sloppy-and-we-love-it/>

![sc_7](img/fe491338e8936130645002fde549c0f6.png "sc_7")

[Ben Heck]是游戏系统黑客的代名词。他的项目因其质量和独创性受到了全世界人民的称赞。事实上，他非常优秀，他的许多项目已经远远超出了我们通常认为的黑客行为。它们看起来和感觉上都像商业产品。虽然这是一个了不起的成就，但我们对真正被黑客攻击的东西情有独钟。他最近做的工作就是一个很好的例子。控制器需要用单手工作，所以他破解了它。他很匆忙，所以它没有得到他通常的专业完成。我们有点喜欢这样。这单手控制器模式可以在休息后看到行动。
