# 见见克拉皮杜先生

> 原文:[https://hackaday.com/2011/05/04/meet-mr-clappidoo/](https://hackaday.com/2011/05/04/meet-mr-clappidoo/)

![mr_clappidoo](../Images/6669a93a396677fd750c4eaed6c4f9f9.png "mr_clappidoo")

[Laurence]经常在家里修修补补，制作各种有趣的互动玩具给他的孩子玩。克拉皮杜先生是他不久前完成的一个玩具，它与[有着惊人的相似之处，我们最近向你展示了他的一个项目，帕皮杜](http://hackaday.com/2011/04/26/papydoo-is-watching-you/)。这不是一个巧合，因为 Papydoo 是在 Clappidoo 先生建成后创建的，借鉴了他的前任的许多特征。

你爸爸是谁，他是做什么的？

你这么问是件好事。Clappidoo 先生使用红外运动传感器来检测附近的物体，唤醒并与任何经过他身边的物体进行互动。他有四种不同的随机情绪，从愤怒到轻浮。他通过改变发光二极管眼睛的颜色以及播放简单的声音来表达这些情绪。一个轻木箱子组成了 Clappidoo 的身体，他用一个小伺服反复地开合他的~~盖子~~嘴巴，因此得名。

像他的其他项目一样，[劳伦斯]集中努力确保用于为 Clappidoo 供电的三节 AA 电池尽可能长时间地工作。他说，在适度使用的情况下，该设备可以在需要更换之前用掉同一组电池几个月。

这是个有趣的小玩意儿，肯定会让孩子们高兴的。留下来看看 Clappidoo 的快速视频。
