# 家具机器人，改造

> 原文:[https://hackaday.com/2010/06/07/furniture-bots-transform/](https://hackaday.com/2010/06/07/furniture-bots-transform/)

![](../Images/3920b93a5d37fe9f308549c437d411a3.png "transforming-table")

这张[机械化桌子](http://www.dbfletcher.com/capstan-table/)自动从六人座扩展到十二人座。我们试图用上面的三张图片来捕捉这个动作，但不要错过休息后视频中的转变。唉，我们在现实生活中永远不会看到这样的东西，因为它住在一艘值得[罗宾·利奇](http://en.wikipedia.org/wiki/Robin_Leach)关注的游艇上。我们复制桌面部件的几何形状不会有问题，但必须有一些严肃的设计工作来完成控制运动的结构。没有列出确切的价格，但创作者指出，建设成本在数万英镑。目前，我们将坚持使用我们的[宜家家具品牌](http://hackaday.com/2010/06/07/did-that-table-just-move)。
