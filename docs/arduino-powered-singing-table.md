# Arduino 电动歌唱桌

> 原文：<https://hackaday.com/2009/09/30/arduino-powered-singing-table/>

[https://www.youtube.com/embed/vc4PSgXDp64?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/vc4PSgXDp64?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这部音乐剧《懒惰的苏珊》或《疯狂的亚当》是由麻省理工的学生带给我们的。基本上是边转边放【魂二魂】的《起死回生》。用他们的话说:“通过与唱歌的懒苏珊的互动，我们发现了每个人独特的饮食模式和行为，它们反映了我们的个性和兴趣。用餐体验扩展到一个新的领域。”只有我们觉得这很傻吗？Arduino 不仅在这方面有些矫枉过正，它如何反映我们的个性和兴趣呢？我们知道，有人会说艺术不需要有意义，但这只会很快变得令人讨厌。想出一个主意并把它变成现实，干得好。请不要把它带到我们下次的办公室聚会上。同样值得注意的是[音乐懒人](http://www.blujay.com/?page=ad&adid=3035416&cat=7110000)并不是一个新的想法。
