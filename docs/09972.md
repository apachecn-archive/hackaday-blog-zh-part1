# 按下磁带上的播放键进行配置

> 原文：<https://hackaday.com/2012/03/26/press-play-on-tape-for-your-fignition/>

![](img/b08f776292426b9e19da72fbc7ddc02e.png "fignition")

[朱利安·斯基德莫尔]一直在忙着改进 Fignition，这是一台小型 AVR 供电的教育计算机，支持从盒式磁带加载程序。

我们第一次看到这幅图是在 [BBC 决定报道一个致力于用一台简单的“裸机”电脑改善计算机教育的老派黑客](http://hackaday.com/2011/04/29/bbc-covers-an-old-school-hacker/)之后。[Julian]的发明让人回想起像 BBC Micro 和 TRS-80 这样非常简单的计算机的时代，并鼓励学生用窥视和戳来代替我们的笔记本电脑和台式机上积累了几十年的垃圾。

因为这个配置是为黑客和学生设计的，所以完全有可能[建立一个键盘](http://hackaday.com/2011/12/04/a-keyboard-for-your-fignition/)，甚至[在条形板上建立一个配置](http://hackaday.com/2011/11/18/fignition-flint-is-a-stripboard-build-of-the-simple-computer/)。现在，这些学生有了更好的硬件，可以将程序保存和加载到磁带(或任何录音机)上，甚至还有分辨率为 160×160 的图形视频模式。

我们知道这看起来有点不可思议，但今年高中毕业的学生都出生于 1994 年，他们很可能从未见过 Commodore 64、Sinclair Spectrum 或整整一代人学习过的其他 80 年代微型计算机。这项发明试图阻止那些无知的大众，他们不知道计算机在过去 30 年里取得了多大的进步，我们喜欢它。
