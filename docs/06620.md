# 使用空气清新剂硬件为种植的水族箱施肥

> 原文:[https://hack aday . com/2011/01/21/施肥-种植-水族馆-使用-空气清新剂-硬件/](https://hackaday.com/2011/01/21/fertilizing-a-planted-aquarium-using-air-freshener-hardware/)

![](../Images/b41c24fd2db7ee3cc4e6208335f5ac1e.png "aquarium-auto-doser")

让水族箱里的活植物保持快乐是一件相当麻烦的事。其中一个常见的仪式是添加肥料，这被称为剂量。[Majstor76]想出了一个创造性的方法来自动使用空气清新剂。他去掉了装香水的罐子，重新设计了一个洗手液泵来输送富含营养的液体。休息后，你可以看到启动泵的动力并不短缺，电动空气清新剂底座有一个延迟电路，允许几个不同的时间释放选项。只要体积输出相当一致，我们认为你可以稀释你的肥料来微调剂量。
