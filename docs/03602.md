# VR 游戏枪

> 原文：<https://hackaday.com/2009/06/21/vr-game-gun/>

[https://www.youtube.com/embed/7Ym0tZSWGMc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/7Ym0tZSWGMc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

你们中的许多人可能已经尝试过不同的游戏界面。这里有一个做得很好的小黑客，他们创造了一把装有显示器的枪，用于视频游戏。一眼就能看出他们到底做了什么。他们在枪上安装了一个旋转鼠标用于追踪，并去掉了按钮键盘。我们必须给他们一些信任，他们似乎已经包装得很好了。由于这是一个标准的鼠标和键盘，它只需插入电源，就可以运行任何游戏。正如你在他们的版本中看到的，屏幕在这种配置下工作得非常好，几乎看起来像一个巨大的范围。休息之后，你可以看到无线版本，听到他们对基于投影仪的模型的渴望。
