# 电路弯曲介绍

> 原文:[https://hackaday.com/2011/01/11/intro-to-circuit-bending/](https://hackaday.com/2011/01/11/intro-to-circuit-bending/)

![](../Images/0f6bf7108262c4ceab6179f49391cd2a.png "Banner")

电子乐器对黑客来说是一种乐趣，因为只要有少量的工具、专业知识和好奇心，它们就很容易被修改。与任何黑客一样，主题总是有可能被破坏，所以不值得冒险在你的千元专业合成器内乱搞。对我们所有人来说幸运的是，在世界各地的路边和二手商店里，有满满一铲的旧电子音乐玩具。这些有趣的小设备为熟悉音频电子和电路弯曲技术提供了充足的机会。

定义说明:在音频电子领域,“电路弯曲”一词与“硬件黑客”同义，我们已经看到一些关于哪个术语更适合某个项目的争论。我们欢迎你在评论中分享你的观点。

继续阅读开始。

**材料**

那么，你听说过电路弯曲，你想试试吗？对于这个介绍，你至少需要以下材料:

*   带电池或交流适配器的电子乐器(bendee)
*   鳄鱼夹(用于临时连接)
*   各种电阻器和/或电位计
*   耳朵

如果您还拥有以下信息，这将非常有帮助:

*   示波器
*   台式电源
*   照相机

对于我们第一次尝试电路弯曲，我们将使用雅马哈 PSS-14 键盘。我们在路边发现了它，它被遗弃了，孤独，在这个世界上没有一个朋友。就像疯狂的科学家在孤独的被遗弃的动物身上进行疯狂的基因实验一样，我们将重建这个可怜的生物，让它变得比以前更好、更强、更奇怪！

**背景研究**

由于我们的学术标准很高，我们将从研究键盘开始。你们中更有冒险精神的人可以跳过这一步，直接进入有趣的部分。从[雅马哈的网站](http://usa.yamaha.com/products/musical-instruments/keyboards/digitalkeyboards/portable_keyboards/pss14/?mode=model)中，我们可以看到这款机型拥有“100 种高级波形记忆声音”——这是他们对“100 种预先编写的声音文件”的搞笑营销术语——这使得它被称为“波表合成器”。波表合成是创建声音的一种非常简单而廉价的方法，因为你可以简单地将一串声音复制到芯片的存储器中，然后逐个样本地读取它们，改变采样率以改变音高(或者为每个音高值设置单独的样本，这取决于你必须使用多少存储器)。

进一步的研究表明，我们不是第一个弯曲这种特殊键盘的人。这个例子和[这个](http://www.youtube.com/watch?v=CGTxYDAEKOU)展示了一些有趣的可能性，到本文结束时，我们会对他们所做的有更好的了解。但说够了，让我们打开这个宝贝！
![](../Images/44a7f1e81e7cb711fd8c328f05d281d4.png "1-KeyboardFront")
这里我们看到了 PSS-14 的原始状态:可操作，但缺少外壳螺丝(当我们找到它时，它是用胶带粘在一起的)。或许之前的主人自己做了一些探索？预编的歌曲涵盖了音乐的所有主要类别:回忆、热门音乐、最喜爱的音乐、欢乐时光。当我们年轻的时候，我们总是听很酷的热门音乐，但后来它成为主流，所以现在我们主要进入地下记忆场景。如果你没有听过《勇敢的猪》的新混音版，你就没有活过。
![](../Images/d095b395e677c2fe152f2893787950c8.png "3-Voices")
有二十种键盘控制的声部可供选择，大部分声音都差不多。这里看到的音量控制会在你按下它们时发出很大的“邦戈”声，不管音量是否尽可能低。

**看引擎盖下**

很明显，这个东西可以更好，所以让我们打开它，看看我们可以改善它的操作。
![](../Images/a8ff54e1ee6b6b9d71cc6648e670b074.png "4-Board1")
引擎盖下的电路板相当稀疏，这并不奇怪，因为它是一个波表合成器，因此大多数有趣的东西都在右边的微控制器内。如果你能找到数字时代之前的旧电子音乐玩具，你就能接触到更多的声音生成的本质。不幸的是，在路边很难找到这些东西。
![](../Images/eaf145096251700fa913af16fca5401d.png "5-BoardPower")
在电路板的左侧，我们可以看到明显标记的 Vcc 和 GND 连接，从电池端子上很容易找到。键盘需要 4 节 AA 电池，这意味着它需要 6 伏电源供电。我们没有这个键盘的交流适配器，所以我们现在用我们的台式电源供电。
这几乎不需要说，但如果你要使用交流供电设备，一定要非常小心。我们正在使用的台式电源有一个限流器，但壁式变压器可能会将危险的电子流推过你的身体，我们知道这是一种不舒服的体验。
![](../Images/017d55f600729e1bb5fd5c0274763bf6.png "7-BoardChip")
在右边，我们可以看到更多的电阻和圣杯一样的时钟元件(就是 IC 左边的蓝色斑点)！在数字合成器上，这通常是乐趣的主要来源。
![](../Images/a643efdc7629e8c97c68ade7120587ea.png "8-BoardMiddle")
在电路板的中间有一簇电容器，看起来像一个多晶体管封装。当我们把板子转过来开始探测时，我们会弄清楚这是怎么回事。
![](../Images/1dc9f3b3c1b13717170cc0045a4b8fc6.png "9-GreenBoard")
电路板焊接和印刷的一面看起来有趣多了。你看到的黑色斑块是导电墨水——这是一种非常常见和廉价的传感器技术，用于从简陋的 NES 控制器到高端的罗兰电子钢琴的所有东西。这是一种被称为[力感电阻(FSR)](http://www.sensorwiki.org/doku.php/sensors/force-sensitive_resistor?s[]=fsr) 的形式，它存在严重的非线性、迟滞和可重复性问题。另一方面，它非常容易实现，可以印刷在电路板上。
![](../Images/49f7f47ba4dfaf691550e11a386293ac.png "10-CPUClose")
在 CPU 的下侧我们可以开始特征化引脚功能。许多引脚连接到不同的按键和按钮。我们在上面看到的许多晶体管也专用于这个关键矩阵。

**查看范围**
![](../Images/7ed54738bff141fcfe9e6a148005deb9.png "11-TDM")
通过进一步调查，按钮/按键状态基于全局激励时分复用到脉搏波信号上，如图所示。[根据这个家伙](http://www.hollis.co.uk/john/bent/)列出的一个电路弯曲的 PSS-15(与此型号相同，但有一个银色的控制面板)，将音频输出的一部分连接到键盘矩阵返回可以重新触发按钮或按键，使“循环”。非常有趣，请看:
![](../Images/999029f913a345b0ee9c7776bfddd804.png "12-PWMDAC")
键盘使用基于 PWM 的 DAC，与音频输出进行比较。同样，这是一项非常便宜的技术(你可以很容易地为你的 arduino 制作一个)，你可以在这里得到一个简单的解释。我们马上可以看到，这种技术的一个缺点是，不同电压电平之间的转换时间可能难以控制，可能导致失真。除此之外，将 PWM DAC 输出连接到一个或多个键盘矩阵回路也很有意思。
![](../Images/ca659bf28ab610fe0e22be250da134ef.png "13-HackPoints")
这里是这个键盘上两个有趣的“黑客点”的特写。我们将改变右边的电阻，看看它对信号有什么影响，我们将把现有的 8MHz 时钟改为不同的时钟。
