# 一位处理器

> 原文:[https://hackaday.com/2012/03/01/a-one-bit-processor/](https://hackaday.com/2012/03/01/a-one-bit-processor/)

戴上抽象思维帽，拿出纸和笔，花些时间弄清楚这个一位处理器是如何工作的。有一天,[Strawdog]在上班的路上想出了这个主意(别担心，他坐火车……比[达夫·琼斯的]令人恐惧的开车时间播客安全多了)。他在纸上画出草图，以确保他知道他将项目进行到哪里，然后与 Legion Labs 合作，在处理中实现它，作为可视化其功能的一种更简单的方法。因为它是一位的，所以只能容纳一条指令。该指令是复制指令，然后是分支-if 指令。它将当前位复制到一个地址，如果该位是 1，它将转移到第二个地址。

对你来说太快了？我们认为这个描述相当不错，但是如果你不能从这篇文章的描述中很好地把它放在一起，你可能想要[建立这个 2 位纸处理器](http://hackaday.com/2010/09/23/2-bit-paper-processor-teaches-how-they-work/)并且首先学习它如何工作。它应该教你理解 1 位版本所需的基本概念。正如您在上面的图像中所看到的，在处理示例中还有一个单步功能，它让您可以分析程序执行期间每个指令的效果。
