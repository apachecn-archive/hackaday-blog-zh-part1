# 2011 年 CCC 市场徽章

> 原文:[https://hackaday.com/2011/07/19/2011-ccc-r0ket-badge/](https://hackaday.com/2011/07/19/2011-ccc-r0ket-badge/)

![](../Images/972434f3bbf29743cdefaf38fafdf9f6.png "2011-ccc-badge-sneak-peek")

[Geekabit]来信询问我们是否已经看到了 2011 年的 CCC 徽章。答案是否定的，我们没有*见过*他们，因为上面的图片是我们在他们的破证书网站上唯一能找到的偷窥图片。但我们很高兴他与我们分享了这个链接，因为它确实讲述了今年的徽章上将会有哪些硬件和固件功能。

我们需要马上为他们的几件事鼓掌。最值得注意的是，3.7 伏 600 毫安时的 LiPo 电池可以通过 USB 端口充电。它拥有一个 ARM Cortex M3 处理器，运行他们所谓的“不可中断”引导程序，通过 USB 端口编程。你可以看到有一个 LCD 屏幕，我们猜测它大约是 128×128 像素(如果你知道其他的，请纠正我们)。你可以通过射频收发器使用 5 向按钮进行交互，也可能使用光学接口，但我们不确定该功能是否会在最终设计中实现。他们还推出了额外硬件的屏蔽系统，这样你就可以在到达目的地之前设计自己的附加设备。

像往常一样，如果你得到了其中的一个，我们希望听到你的项目的所有信息，并得到股票徽章及其功能的概述，所以不要忘记[给我们写信](http://hackaday.com/contact-hack-a-day/)。
