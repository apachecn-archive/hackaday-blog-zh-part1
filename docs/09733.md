# 用电子猫耳朵活出你内心的万物有灵论

> 原文：<https://hackaday.com/2012/02/21/live-out-your-inner-animist-with-animatronic-cat-ears/>

![](img/ed8634dcb407277c4b268606065dac78.png "cat ears")

虽然有一只大白鲨作为你的精神动物听起来可能很酷，但我们发现戴着一副巨大的机械颚在公共场合走来走去有点妨碍。[abetusk]没有这个问题；他可以在任何他喜欢的地方戴着他那令人敬畏的电子猫耳朵。

这款耳机的设计灵感来自于[这款极具日本特色的](http://www.youtube.com/watch?feature=player_embedded&v=w06zvM2x_lw#!)电子耳，装有脑电图硬件，可以读取佩戴者的情绪状态。[abetusk]认为拆开一些[大脑扫描硬件](http://hackaday.com/2009/11/07/mindflex-teardown/)是太多的工作，没有足够的好处，所以他决定由按钮控制的伺服系统就可以了。

在建筑的电子部分，附在头带上的伺服系统由一个附件 13 控制。一个按钮对耳朵来说会经历三种状态:长时间按下按钮是惊喜，长时间按住轻敲是生气或悲伤，轻敲三次是“眨眼”所有的代码都在 GitHub 上[，休息过后，你可以在视频中与【abetusk】可爱的助手一起查看这些猫耳情绪。](https://github.com/abetusk/kears)
