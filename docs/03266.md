# 六足直升机

> 原文：<https://hackaday.com/2009/03/02/hexapod-helicopter/>

[https://www.youtube.com/embed/RMhrSwatSiE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/RMhrSwatSiE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

建造了这架六条腿的小直升机。你可以看到它相当灵活。你也可以看到它并没有很好地离开地面。他一定从中获得了更多的能量，因为他在拍摄后的第二天就毁了它。令人惊讶的是，这六条小腿给遥控直升机增添了多少特色。这个东西几乎和[螃蟹](http://www.crabfu.com/)里的东西一样可爱。
