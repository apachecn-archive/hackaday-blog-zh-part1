# 游戏手套了解你的弱点

> 原文:[https://hack aday . com/2010/03/10/game-glove-learn-your-weakness/](https://hackaday.com/2010/03/10/game-glove-learns-your-weakness/)

[史蒂夫·霍弗]为无依无靠的人设计了一个伟大的黑客。这只[手套将与你玩一场激烈的石头剪刀布](http://grathio.com/2010/03/rock-paper-scissors-training-glove.html)游戏。[Steve]意识到中指和无名指是所有需要被监控的，以决定你在做三个手势中的哪一个。他用这些手指背面的柔性传感器作为输入。还有一个加速度计来判断拍摄前的三次震动。

你看到的小屏幕显示了手套的选择，它本身就是一个黑客。这个想法改编自邪恶的疯狂科学家项目，用三片丙烯酸蚀刻出不同的图标，并用发光二极管边缘照明。所有这些，连同一个扬声器和记分牌，都连接到一个 Arduino。锦上添花？[Steve]编写了一个自适应学习算法来观察你的比赛风格以获得优势。

休息过后，请观看实际操作。一旦你掌握了石头剪子布，你应该考虑建造[其他基于手套的](http://hackaday.com/2008/07/14/diy-joystick-glove/) [外围设备](http://hackaday.com/2009/05/01/cell-phone-glove/)。
