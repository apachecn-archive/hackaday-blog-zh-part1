# 研究步进电机的生成特性

> 原文:[https://hack aday . com/2012/01/23/investigating-the-generative-properties-of-a-a-stepper-motor/](https://hackaday.com/2012/01/23/investigating-the-generative-properties-of-a-stepper-motor/)

![](../Images/7452271712726fe203406cc7a8142fc8.png "stepper-generator-board")

你可能知道，如果你(机械地)旋转一个马达，它就会在正常情况下的输入上产生电力。当您关闭旋转的电机时，这可能会成为一个问题，这也是电机驱动电路内置保护二极管的原因。但是[迪诺]对驱动马达不感兴趣，他想看看[他能用旋转步进马达](http://hackaweek.com/hacks/?p=595)产生的电做些什么。

他为此建造了你在上面看到的试验台。在前景中，一个 12V 的 DC 马达被一个电导管夹固定住。这连接到步进电机被测试使用一段橡胶管。DC 发动机为他的实验提供了可靠的输入，但在未来可能会被螺旋桨取代，使其成为风力发动机，或被水轮取代。休息后看看视频，看看[迪诺]从中获得了什么，以及它如何用于给 led 供电，给电池充电，或驱动马达。
