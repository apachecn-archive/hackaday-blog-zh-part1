# 实践中的多晶型

> 原文:[https://hackaday.com/2010/01/19/polymorph-in-practice/](https://hackaday.com/2010/01/19/polymorph-in-practice/)

[https://www.youtube.com/embed/gxy22VfZq0E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/gxy22VfZq0E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Leesam]向我们透露了这个很酷的关于变形术的教程。对于那些没有看过的人来说，这是非常酷的东西。这是一种小颗粒塑料。你可以在相对较低的温度下将它熔化，然后塑造成你需要的形状。我们已经看到它被用在几个项目上，最著名的是 [CrabFu 的 swashbot3](http://www.crabfu.com/swashbot/swashbot3.html) 。一般来说，我们看到它的手形，所以它是相当有机和不完美的。但是它可以用于更精确的形状。本教程带领我们通过最简单的方法来制作要剪切和弯曲成括号的材料，以及获得最佳效果的一些有用的提示。
