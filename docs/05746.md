# 灯泡形状系数计算机

> 原文:[https://hack aday . com/2010/08/02/light-bulb-form-factor-computer/](https://hackaday.com/2010/08/02/light-bulb-form-factor-computer/)

![](../Images/defe80795f6112fe4577d6dd12ada225.png "light-bulb-form-factor-computer")

这台[电脑可以安装在任何标准灯泡插座](http://fluid.media.mit.edu/people/natan/current/luminar.html)中。它使用一个微型投影仪和一个摄像头，无论你在哪里照射它，它都会产生一个触摸显示。上面的照片和休息后的视频显示了电动灯臂上的灯泡，但这只是烟雾和镜子，灯泡本身才是核心概念。我们认为自制版本有很大的潜力。我们已经看到类似的触摸[显示器安装在笔记本电脑的侧面](http://hackaday.com/2010/04/28/bonfire-interactivity-using-pico-projectors-and-cameras/)，但是为什么还要有电脑呢？放弃无线 USB 连接，让它连接到您的家庭服务器以获得处理能力。对于传统上对计算机不友好的地方，它成为了一个完美的解决方案。例如，当显示屏投影在砧板上时，你不想用沾满面团的手触摸的[厨房电脑](http://hackaday.com/2010/01/23/iphone-look-alike-on-your-kitchen-wall/)就变得可以清洗了。 <https://www.youtube.com/embed/XV5V-dQW8CI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
