# 带着你的 PS3 环游世界

> 原文:[https://hack aday . com/2011/01/28/take-your-PS3-on-world-tour/](https://hackaday.com/2011/01/28/take-your-ps3-on-world-tour/)

![](../Images/75d85e86408741fbd894fffc3be610af.png "ps3-hardcase-mod")

[Lossfresnos65]一定在计划做很多旅行，因为[他把他的 PlayStation 3 埋在一个鹈鹕盒子里](http://psx-scene.com/forums/f177/ps3-portable-78759/)。在里面，你会发现一些菱形板挡板覆盖控制台本身，用于包围安装在机箱盖上的 19 英寸高清电视。在上面的图片中，你可以看到门可以打开，露出一个电视遥控器和两个六轴控制器的泡沫塑料切口。没有电池和无线互联网，但电源线、以太网、分量视频和 HDMI 的连接在外壳的底部有一个外部端口。还有两个风扇来保持一切凉爽，在前面你会发现两个 USB 端口和一个耳机插孔。休息后，他展示了成品，但没有内部建设的照片被发现。我们在顶部链接了他的论坛帖子，问他，也许他会发货。
