# 可持续性黑客:生物柴油摩托车速度记录

> 原文:[https://hack aday . com/2011/10/03/sustainability-hacks-bio-diesel-motorcycle-speed-record/](https://hackaday.com/2011/10/03/sustainability-hacks-bio-diesel-motorcycle-speed-record/)

有时候你不需要很大的马力来赢得速度记录。幸运的是，在 alt 燃料级别中没有其他人参赛，[约翰]的[生物柴油摩托车](http://www.suckindiesel.com/thingy/viewtopic.php?f=31&t=1891&start=0&sid=a46c55918f9a1fda086e6a830ed51fe7)在去年夏天的[LTA 活动](http://lta-lsr.com/default.aspx)中创造了新的陆地速度记录。

[约翰]的自行车是一个垃圾场 1978 年川崎 KZ400。股票引擎被替换为一个洋马风冷柴油发动机的中国山寨。燃料是普通的旧植物油。从排气管的外观来看，我们认为[约翰]的车库有一股浓郁的炸薯条味。

与公路速度相比，[约翰]的陆上速度纪录有点荒谬——当你是班上唯一的司机时，这是一个不错的奖励。第一次通过 42 英里/小时有点令人失望，所以[约翰]拆除了挡泥板，尾灯和刹车。除去所有不必要的重量后，最高速度——和新的记录——是 56.5 英里/小时。

将一辆柴油车改装成使用炸薯条油是很棒的，而且比燃烧液化恐龙对环境好得多。无论如何，一辆绿色的摩托车比 2000 磅重的汽车移动不到 200 磅重的人要好得多。休息之后，看看[John]的陆上速度跑。
