# 如何种植自己的 EL 线 DNA 螺旋灯

> 原文:[https://hack aday . com/2011/12/22/how-grow-your-own-wire-DNA-helix-lamp/](https://hackaday.com/2011/12/22/how-to-grow-your-own-el-wire-dna-helix-lamp/)

![el-wire-helix-lamp](../Images/97257af5d9238204add399edf5532f7d.png "el-wire-helix-lamp")

[LucidMovement]正在寻找一些基于水晶的艺术品，但似乎找不到任何符合要求的东西，所以他决定自己制作一些东西。

他台灯的灵感来自我们都很熟悉的东西，DNA 双螺旋结构。为了培养晶体，他用镍铬合金和 EL 线制作了一个螺旋形的生长基质，将它们浸入一种温暖的明矾溶液中。一旦他有了一套漂亮的水晶，他就把它安装在一个丙烯酸管中，用透明硅胶填充空气空间来密封显示器。然后，他将填充硅胶的管子安装在他为该项目切割的旋转丙烯酸支架上。该支架由几片丙烯酸制成，包含移动齿轮以及从底部照亮显示器的 RGB LEDs。

这盏灯闲置时看起来很棒，但当他打开它时，它真的会发光(没有双关语)。[LucidMovement]为 lamp 投入了大量的工作，并提供了各种提示、技巧和注意事项，供任何想要打造自己的 lamp 的人参考。请务必查看他的文章以了解更多细节，并留下来观看一段关于这种灯的视频短片。
