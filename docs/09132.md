# 频闪仪项目使用光驱电机和 Arduino

> 原文:[https://hack aday . com/2011/11/21/stroboscope-project-uses-optical-drive-motor-and-arduino/](https://hackaday.com/2011/11/21/stroboscope-project-uses-optical-drive-motor-and-arduino/)

![](../Images/2aafb774d15331b14cdffeb7ba22921f.png "arduino-stroboscope")

在寻找一个由 DVD 刻录机零件制成的 diy 激光切割机的过程中(黑客仍在工作中)，这个家伙最终得到了一个装满光驱残羹剩饭的垃圾箱。他通过建造这个频闪仪很好地利用了一些剩余资源。随着媒体的旋转，前景中失焦的白色 LED 闪光灯将小黑人定格在同一个地方。正如在休息后的视频中看到的那样，这种效果是由视错觉产生的舞蹈图形。

这与令人惊叹的 3D 赛艇骨架的概念相同，但比例大大缩小。上面看到的每个剪影都略有不同，显示了构成整个动画帧的一个姿势。它们是激光切割的，但是一些精心制作的纸艺可能也能达到同样的效果。假设你能让它们在高速旋转时不翘曲。
