# 全自动弩

> 原文：<https://hackaday.com/2009/12/08/fully-automatic-crossbow/>

[https://www.youtube.com/embed/QS4RKoRyTik?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QS4RKoRyTik?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们收到了[诺亚]发来的一封激动人心的电子邮件，内容是关于一位使用十字弓和十字弓手枪的绅士。我们不情愿地看了一眼，我们所看到的让我们开心了一天。[TheDuckman666]一定对所有弩的东西都有贪得无厌的爱。他的网页上有他制作的七个不同模型的细节和图片。他的 [YouTube 频道](http://www.youtube.com/user/TheDuckman666)展示了另外三个模型，都有多个螺栓的杂志。

看到电动扳起扳机的自动[射弹发射器](http://hackaday.com/2009/10/08/maggy-locked-and-loaded/)满足了我们的胃口，但我们确实希望有更多关于建造过程的背景信息。
