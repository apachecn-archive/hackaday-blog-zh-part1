# 会说话的数字卡尺让工程更容易理解

> 原文:[https://hack aday . com/2011/11/09/talking-digital-calipers-make-engineering-more-accessible/](https://hackaday.com/2011/11/09/talking-digital-calipers-make-engineering-more-accessible/)

![talking-digital-calipers](../Images/9eaa90db3cc7ad61f799e0fe0a44f549.png "talking-digital-calipers")

NerdKits 的团队最近组装了一种设备，旨在帮助残疾人更容易使用测量事物的过程。[Terry Garrett]是一名机械工程专业的学生，该领域的任何人都知道，这是一门需要进行大量测量的学科。由于[Terry]看不见东西，他经常在实验室里请同学帮忙测量物品，但当他在附近的一家设计工作室找到工作后，他知道他必须想办法自己测量。

输入 NerdKits。

[Humberto]来信分享了他和他的团队如何构建一套会说话的数字卡尺来帮助[Terry]完成日常任务。他们的设计基于他们之前从事的一个项目，从一套卡尺中获取数字读出数据。DRO 信息被输入到 ATmega382p 中，atmega 382 p 将预先录制的声音片段组合在一起，宣布被测物体的大小。

正如您在下面的视频中所看到的，该系统看起来运行得非常好，[Terry]对他的新谈话工具非常满意。我们喜欢看到这种类型的黑客，因为他们真正改变了人们的生活——干得好！
