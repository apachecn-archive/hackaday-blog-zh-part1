# DPAC 让你的闹钟相形见绌

> 原文:[https://hack aday . com/2010/05/01/dpac-让你的闹钟蒙羞/](https://hackaday.com/2010/05/01/dpac-put-your-alarm-clock-to-shame/)

![](../Images/22ffab4ea6b877efbaf464f9c5a07f72.png "dpac")

DPAC，[动态可编程闹钟](http://egaertner.com/dpac/)，远远超出了你对闹钟的期望，但我们发现它的所有功能都很有用。你可以看到底部有四个按钮控制菜单滚动。左边第二个当前显示“同步”，这是时钟每 10 分钟使用一次的功能，但可以手动强制。这将检查你的谷歌日历，安排下一个事件的提醒，同时考虑驾驶距离、交通和天气情况。它有一个用于收音机和 iPod 操作的音频系统，但也包括一些家庭自动化选项。使用 X10 通信协议，它可以打开灯，启动咖啡机，打开百叶窗，作为温和唤醒周期的一部分。所有这些都可以通过时钟本身或 web 界面进行配置。原型是在 Arduino 上完成的，但最终版本使用 AVR ATmega324 以及漫游网络 RN-134 WiFi 模块([数据表](http://www.rovingnetworks.com/documents/rn-134-ds.pdf))进行连接。看看[Eric Gaertner]和他的开发伙伴在休息后拍摄的演示视频。
