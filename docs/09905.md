# Gamelatron，一个完全机器人化的印尼 Gamelan 管弦乐团。

> 原文:[https://hack aday . com/2012/03/15/gamelatron-a-full-robotic-indonesian-gamelan-orchestra/](https://hackaday.com/2012/03/15/gamelatron-a-fully-robotic-indonesian-gamelan-orchestra/)

![](../Images/715e2988e1e2308992207c6dd5014051.png "gamelatron")

去年夏天，我们和[亚伦·泰勒]聊了聊他的自动化 Gamelan 管弦乐队。机器人管弦乐队的特点是收集了大量的印度尼西亚锣、[金属琴](http://en.wikipedia.org/wiki/Metallophone)、木琴和钹，通过连接在槌棒上的简单拉动螺线管来驱动。Gamelatron 的定制控制器使用 MIDI 激活各种 24V 螺线管，整个事情本质上是一个巨大的 MIDI 乐器，可以通过任何排序设备播放。

[Aaron]有多种方法将 MIDI 输入控制器，包括“Padma Bhuwana”，一个有 16 个街机按钮的木箱，连接到 Arduino。Arduino 既可以在运行 Ableton live 的计算机上激活序列，也可以直接从 Arduino 中抽取 MIDI 序列，用于计算机自由交互安装。[Aaron]还将他的 Akai MPD32 连接到计算机上进行现场表演，或者他可以让笔记本电脑完成所有非交互式安装工作。

拥有 170 个左右简单驱动的仪器，真正有趣的是能够将它们展开并填满空间的每一个面。一个很好的例子是 2011 年火人节的过渡神殿，那里的铜锣和其他东西会横跨多层。这里有一个最近的 [Wired magazine](http://www.wired.com/video/gameletron-robotic-orchestra/1481332065001) 发布的视频，其中[Aaron]给出了一个快速的设置概述，或者如果你对广告太不耐烦了，可以查看几个 Gamelatron 在《燃烧的人》和《PEX 夏日祭》中的视频。我们还包括了[Aaron]的 kickstarter 视频，其中有一些关于设置的更多细节(以及关于 kickstarter 项目的无关内容，该项目已经过期)。有了所有疯狂的 [midi 乐器](http://hackaday.com/?s=midi+instrument) hacks，我们在这里看到它有很多互动的潜力是不难想象的。
