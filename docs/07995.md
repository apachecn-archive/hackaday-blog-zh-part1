# 带着这个飞行遥控超级英雄飞向天空

> 原文：<https://hackaday.com/2011/07/05/take-to-the-skies-with-this-flying-rc-superhero/>

是一只鸟！是一架飞机！这是…

一个会飞的 RC 超级英雄？

不，你没有胡思乱想。制造者[Greg Tanous]既喜欢遥控飞机也喜欢超级英雄，所以他认为将他的两种爱好结合成一个壮观的玩具会很棒。超级英雄有两种型号，身高分别为 75 英寸和 57 英寸。较大的飞机可以作为套件购买，但较小的单位只有平面图形式。

这位飞行超级英雄重约三磅，由泡沫、碳纤维和各种其他轻质材料制成。当使用推荐的马达时，遥控超级英雄可以从站立位置发射，不需要跑道。该飞行器看起来就像任何常规的遥控飞机一样易于操纵，使得任何经验丰富的飞行员都可以轻松地拿起控制器并开始飞行。

遥控飞机很酷，但我们必须承认，垂直发射、飞行的超级英雄听起来很有趣！
