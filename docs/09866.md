# 金枪鱼罐头和其他垃圾变成了斯特林发动机

> 原文:[https://hack aday . com/2012/03/09/tuna-can-and some-other-trash-turned-a-sterling-engine/](https://hackaday.com/2012/03/09/tuna-can-and-some-other-trash-turned-into-a-sterling-engine/)

下次你给自己做金枪鱼三明治时，试着想想如何用下脚料制造斯特林发动机。如果你能像[Killerlot]一样成功，我们会说你已经赢得了你的[黑客徽章](http://hackaday.com/2011/12/25/hackaday-links-december-25-2011/)。

这个项目中使用的罐头实际上是番茄酱中的沙丁鱼，但前者的内容是没有实际意义的。这个罐子是斯特林发动机的蒸汽室。凸轮杆、活塞和气门都是由回形针制成的，还有将它们固定在适当位置的支撑结构。罐子里面是一块潮湿的海绵。当酒精灯放在罐子下面时，它加热里面的~~水~~空气，这在活塞上产生压力，向上推动活塞，直到凸轮打开阀门，刚好及时释放压力，以便循环再次开始。动量是这一机制的必要组成部分，这就是裁谈会飞轮的用武之地。休息后，看它在剪辑中突突前进。
