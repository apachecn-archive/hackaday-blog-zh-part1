# 在你的会议上播放“透明手榴弹”广告

> 原文:[https://hack aday . com/2012/02/21/data-broa casting-transparency-手榴弹-广告-奇思妙想-你的会议/](https://hackaday.com/2012/02/21/data-broacasting-transparency-grenade-ads-whimsy-to-your-meetings/)

![](../Images/52f069d8b1a7fb80036f8ea737d006ed.png "transparency-grenade")

如果你厌倦了闭门进行的秘密交易，也许你需要开始随身携带这个透明手榴弹。外壳模仿苏联时代的 F1 手榴弹。但它没有装满炸药，不会在房间里散播致命的弹片。取而代之的是，当插针被拔出时，它开始录制音频并嗅探网络数据包，然后将二者广播到远程服务器。也许你可以认为这是发送到世界各地的数据碎片。

上面的零件分解图显示了正在使用的硬件。该设备的核心是一个 Gumstix 板，它使用 WiFi 模块来嗅探和广播数据。您在完全组装好的设备中看到的 LED 条形图显示了无线信号强度。

听起来好像外壳本身是由 Tusk2700T 半透明树脂 3D 打印而成的，但我们对这部分硬件描述有点困惑。我们不太需要传输我们会议的录音，但我们很想将这个案例设计用于 MP3 外壳。
