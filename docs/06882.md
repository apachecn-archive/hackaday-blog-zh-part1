# 亮闪闪的头饰

> 原文:[https://hackaday.com/2011/02/21/blinky-headgear/](https://hackaday.com/2011/02/21/blinky-headgear/)

![](../Images/61850dd0fd9b88bff2747992bda0daaa.png "blinky-visor")

[Everett Tom] [在他的护目镜](http://weekendhobbyist.blogspot.com/2011/02/led-visor-msp430-final-build-into-visor.html)上增加了一些闪烁的 led，同时磨练他的 PCB 设计技能。他开始用 [TI eZ430-F2013](http://hackaday.com/2011/01/30/hands-on-with-ez430-f2013/) 制作闪烁电路及其模式切换按钮的原型。一旦解决了这个问题，他就使用 BatchPCB(一种[低成本专业电路板制造](http://hackaday.com/2010/08/01/hackaday-links-august-1-2010/)选项)来制造电路板。最终设计包含一个 MSP430 F2013 微控制器以及电压调节器电路，该电路从隐藏在电路板旁边的头带内的 9V 电池中获取电能。
