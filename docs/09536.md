# 管状视点显示

> 原文：<https://hackaday.com/2012/01/20/tubular-pov-display/>

![](img/2095236b8ce8845db43aa1fe438dc702.png "goomba")

[Ryan]的[圆柱体视点显示](http://rucalgary.hackhut.com/2012/01/19/rgb-pov-display/)是一件令人惊叹的作品。现在坐在他的工作台上令人印象深刻，但我们确信挂在舞池中间会令人惊讶。这个显示器上有 64 个 RGB LEDs，它们的亮度足以照亮任何空间。

电力是通过滑环提供的。接地连接到[Ryan]在汽车配件商店买到的马达轴上。这是一种有效的方式，但显示器只能由存储在 ATMega1284 闪存中的图像来控制。[Ryan]承认这不是一个理想的设置，所以他正在研究 ZigBee 或蓝牙连接。

我们已经看到了一些[惊人的 spinny](http://hackaday.com/2010/11/13/lots-of-spinning-pov-goodness/) [POV 缸](http://hackaday.com/2010/08/09/spin-peggy-get-3d-pov/)，但【瑞安】的建设看起来令人惊讶的专业。所有的电路板文件，原理图和代码都上传了，还有一个 BMPs 和 PNGs 的图像转换器。休息后请欣赏演示。
