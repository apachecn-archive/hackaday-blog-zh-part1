# 反向地理缓存盒看起来很棒，功能齐全

> 原文:[https://hack aday . com/2012/01/19/reverse-geo cache-box-looks-great-and-packed-with-features/](https://hackaday.com/2012/01/19/reverse-geocache-box-looks-great-and-packed-with-features/)

![](../Images/cbb89d0ef406c0b0f334b25c14504f33.png "beautiful-reverse-geocache")

[护林员鲍勃]精心制作了这个看起来很棒的反向地理藏宝箱。我们最喜欢的功能是顶部的黑色丙烯酸片。它是激光切割的(不确定字母是否被雕刻)，给人一种很棒的成品外观，同时隐藏了一些东西。

橙色盒子是一个金属钱箱，盖子上有一个光滑的凹痕，当不被携带时，把手就在那里。[Bob]取下把手，在那个空隙里安装了 GPS 模块。但是旁边还有一个有机发光二极管显示器。正如你在休息后的演示视频中看到的那样，屏幕足够亮，可以透过覆盖在凹陷处的烟熏丙烯酸树脂清晰地看到。

这个项目让[Bob]有机会订购他的第一个专业制造的电路板。他在 Eagle 中完成了设计，设法保持在 Seeed Studio 最便宜的 Fusion PCB 选项的 5cmx5cm 范围内。该板承载 PIC 18F87J50，负责处理屏幕、GPS 模块、输入按钮和 USB 端口。电力来自内部锂电池。

我们推出了许多反向地理缓存盒，它们仍然是我们最喜欢的项目之一，因为在设计和构建过程中投入了太多的爱。这是我们随机挑选的另一个供你娱乐。
