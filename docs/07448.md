# Rube Goldberg Machine 让工程系学生创下了纪录

> 原文:[https://hack aday . com/2011/04/21/rube-Goldberg-machine-puts-engineering-students-in-the-record-books/](https://hackaday.com/2011/04/21/rube-goldberg-machine-puts-engineering-students-in-the-record-books/)

![rube_goldberg](../Images/2b54644590c86d8e2ce0c7bd3204f3d6.png "rube_goldberg")

鲁布·戈德堡机器在这里一直是粉丝们的最爱。它们真正体现了过度工程的概念，并且与我们通常在黑客日报道的内容有所不同。

早在二月份，来自普渡大学两个校园专业协会的工程学生合作建造了一台创世界纪录的鲁贝·戈德堡机器。他们在普渡地区鲁贝·戈德堡机器竞赛中的参赛作品不仅为他们赢得了地区冠军，而且有可能将他们载入书籍，因为他们创造了最复杂的设备。

他们的装置被称为“时间机器”,可以再现地球历史上的事件。它从大爆炸开始，经历了不同的时期，包括石器时代、古埃及和中世纪。它还在更近的时期停留过几次，包括第二次世界大战和冷战，然后在启示录中自毁。

总的来说，这台机器包含了 244 个给植物浇水的步骤，比之前的记录保持者多了 14 个。继续阅读，观看一段视频，其中重点介绍了这款机器的一些更有趣的功能，并且一定要去[看看](http://hackaday.com/2010/07/13/a-pair-of-rube-goldberg-builds/)这些[鲁布·戈德堡机器](http://hackaday.com/2010/02/20/rube-goldberg-ftw/)我们已经[在过去报道过](http://hackaday.com/2010/06/15/great-ball-contraption/)。
