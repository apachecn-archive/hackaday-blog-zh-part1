# 由废料制成的激光射击场

> 原文：<https://hackaday.com/2012/01/03/laser-shooting-gallery-made-from-scrap/>

当主持电子音乐广播节目时，加斯顿·克拉雷斯(Gaston Klares)并不十分出色，而是在工作室里忙着创作一些令人惊叹的东西。他最近建造的一个[激光枪射击馆](http://web.me.com/fernandeklares/gastweb/Lasergun.html)将一个经典的嘉年华游戏带到了他的后院。([谷歌翻译](http://translate.google.com/translate?sl=de&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fweb.me.com%2Ffernandeklares%2Fgastweb%2FLasergun.html&act=url)链接)

激光枪本身由 95%的回收部件组成。激光枪的目的很简单——只要在扣动扳机时制造声音效果并发射激光。射击场是所有乐趣发生的地方。五个狗粮罐排列在靶场的远端。每个罐子上都有一个小洞，每个罐子里都有一个光电晶体管，可以激活“翻倒”机制。当所有的易拉罐都被撞倒后，挡风玻璃刮水器电机会将所有的易拉罐重新直立起来。

感谢[克里斯]发送这一个。休息之后，看看正在进行中的射击馆。
