# 自主彩弹哨兵枪

> 原文:[https://hack aday . com/2008/08/24/autonomous-paint ball-sentry-gun/](https://hackaday.com/2008/08/24/autonomous-paintball-sentry-gun/)

![](../Images/97ec6a00ad345d51e40932bca38507e8.png)
【艾伦】指出这把[伟大的商业彩弹哨兵枪](http://zero-op.com/mercenary)。这把枪有一个运行 Ubuntu 的低功耗主板，连接到一个 firewire 摄像头。两台高扭矩电机控制 Tippman 98 custom 的平移和倾斜。控制箱便于安装。您可以更改敏感度、颜色跟踪和拍摄模式。它还有一个遥控开关。周围都是好看的机器，但在$3K，我们怀疑你会买一个。休息后嵌入测试视频。
