# C64 操纵杆适配器

> 原文:[https://hackaday.com/2011/10/18/c64-joystick-adapter/](https://hackaday.com/2011/10/18/c64-joystick-adapter/)

![](../Images/a6cb8bc74ac12496b175838a22d8e9da.png "c64-joystick-adapter")

[Marcus Gritsch]想用复古硬件做他的复古游戏…或者至少用一些复古硬件。虽然他在模拟器上玩他的 Commodore 64 游戏，但他认为使用原装控制器会大大增加怀旧感。这是一个老式的竞赛专业操纵杆，有按钮和一个类似于街机硬件和 DE-9 连接器质量的操纵杆。他设法通过[建立自己的 USB 到 C64 操纵杆适配器](http://dangerousprototypes.com/forum/viewtopic.php?f=56&t=2971)来连接新的和旧的。

他的项目从基于 PIC 24FJ64GB002 微控制器的电路试验板开始。这就完成了所有的工作，具有本机 USB 支持，从旧硬件读取和转换信号没有问题，旧硬件只是每个内部开关的导体，当启动时拉至地。一旦开始工作，他就把所有的东西焊接到一些原型板上；两端各有一个连接器、芯片本身、一个稳压器和一些无源元件。这是一个强健的体格，应该能给他带来多年的乐趣。
