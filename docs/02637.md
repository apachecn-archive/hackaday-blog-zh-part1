# 电动自行车

> 原文：<https://hackaday.com/2008/10/11/electric-bicycle/>

[https://www.youtube.com/embed/6wNo5SM2da4?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/6wNo5SM2da4?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
【garygadget 15】英国有一个有趣的 [youtube 页面](http://www.youtube.com/user/garygadget15)展示了一辆电动自行车。在他名为“绿色省钱”的页面上，他用一辆电动自行车取代了他的通勤车。然后，他拍摄两人的通勤视频，并与结果进行比较。他使用的 DIY 电动自行车套件是由 Cyclone[制造的](http://www.cyclone-tw.com/newkits.htm)有多种瓦数，从 180 瓦到 1500 瓦不等，他们在展示转换步骤方面做得很好。他们有足够的细节，如果你喜欢的话，你可以用打捞上来的零件制作你自己的。
