# 激光追踪器回放竞技攀岩

> 原文:[https://hack aday . com/2011/04/19/laser-tracker-replays-competitive-rock-wall-clippens/](https://hackaday.com/2011/04/19/laser-tracker-replays-competitive-rock-wall-climbs/)

![laser_climbing_tracker](../Images/ad303ad2e10bc4abf19846d4f61914b8.png "laser_climbing_tracker")

Instructables 用户[PenfoldPlant]是一个室内攀岩的狂热爱好者，当他看着其他人进行高难度的攀岩时，他经常想知道他是否可以沿着同样的路线攀岩。不幸的是，除了观察其他登山者并希望记住他们走过的路，他发现你没有太多的办法来确保你精确地复制攀登。

他想了一会儿，想出了一个激光跟踪系统，可以用来记录登山者的攀登过程，然后回放任意次。这使得登山者能够复制其他登山者的路径，并在计时赛中相互竞争。

这很像大多数赛车游戏中的“幽灵”功能，尽管这个过程是半手动/半自动化的。当攀登者攀登时，通过用激光笔手动跟踪攀登者的路线来记录最初的攀登。该路径被记录下来，然后可以重放，这要归功于机载 Arduino。

这确实是一个简洁的系统，虽然它已经运行得相当好了，但我们认为仍有改进的空间。让攀登者戴上某种可以用网络摄像头或其他记录设备追踪的灯塔并不十分困难，这样就不用手工劳动了。然而，在这种情况下，我们认为 Arduino 需要换成更强大的东西。
