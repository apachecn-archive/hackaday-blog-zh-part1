# 升级太阳能灯为 IPad 充电

> 原文:[https://hack aday . com/2011/10/28/升级太阳能灯为 ipad 充电/](https://hackaday.com/2011/10/28/upgrading-a-solar-lamp-to-charge-an-ipad/)

![ikea_sunnan_upgrade](../Images/4e23d50c6d3d1cde14af532cc03aca02.png "ikea_sunnan_upgrade")

[菲利普]不久前，伏打系统公司的工作人员看了宜家的孙楠太阳能台灯，虽然他们认为它非常有用，但他们确实想改变一些东西。

他们首先修改的是增加电池包的容量。拆开灯，拧开包装的盖子，露出了一组 3 节 AA 电池，他们换成了更高容量的型号，额定瓦时超过两倍。

增强电池是一个好的开始，但灯的微型太阳能电池板没有希望在死亡谷之外补充电池。为了确保他们获得良好的充满电，一个小插孔被连接到电池组，允许该小组连接任何大小的外部太阳能电池板。

最后，[菲利普]和他的公司希望能从灯的电池组为 iPad2 充电。他们黑进了一个小 USB 连接器和一个稍微修改过的 MintyBoost 板，为他们的平板电脑提供一点额外的能量。

虽然他们仍在测试这些修改，但他们说一切都很好，并提到额外的电池容量和充电能力是一个很好的补充。
