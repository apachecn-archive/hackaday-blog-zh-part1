# 在家制作荧光棒

> 原文:[https://hackaday.com/2010/07/18/making-glow-sticks-at-home/](https://hackaday.com/2010/07/18/making-glow-sticks-at-home/)

[https://www.youtube.com/embed/tItOOpyJP5k?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/tItOOpyJP5k?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

即使你对自己制作荧光棒不感兴趣，你也可能会喜欢这个关于自己制作荧光棒的教育性指导。由一个很短的化学品清单组成，[全部在线](http://redirectingat.com/?id=487X782&xs=1&url=http%3A%2F%2Fwww.alfa.com%2F&sref=http%3A%2F%2Fwww.instructables.com%2Fid%2FMake-Glow-Sticks-The-Science%2F)，过程相当简单。如果你想承担一个稍微复杂一点的化学项目，你也可以[让 TCPO 组件成为你自己的](http://www.instructables.com/id/How-To-Make-TCPO-for-glow-sticks/)，可能会节省一些钱，因为单个组件比最终产品便宜。正如他们所说，买荧光棒更便宜也更容易，除非你正在大批量生产。
