# 低成本视频聊天机器人

> 原文：<https://hackaday.com/2011/02/09/low-cost-video-chat-robot/>

![remote_control_chat_robot](img/d3f777acdb22fa6571d857aaaf92af79.png "remote_control_chat_robot")

[约翰尼·李中]最近从西雅图搬到了山景城，他想找一种方式与他的未婚夫保持联系，因为他的未婚夫在未来几个月内不会搬家。虽然我们大多数人可能会考虑购买一对网络摄像头来保持联系，但他决定以自己的方式做事。他用一台 iRobot Create 和一台上网本，每台大约 250 美元，建造了一个遥控视频聊天机器人，他可以在 1000 英里之外的他以前的住所周围导航。虽然一开始 500 美元似乎很贵，但[Johnny]提醒我们，商业版本可能会达到数千美元。

整个设置由定制软件控制，以管理机器人的移动，可以与免费的视频会议应用程序结合使用，如 Skype。他还修改了 iRobot 的充电站，以便同时为机器人和上网本充电——他解释了这个过程，但在此之前发表了几项免责声明。就像我们已经介绍过的他以前的一些项目一样，他在他的网站上提供了这个项目中使用的 C#源代码，以及控制软件和 dock 修改的文档。
