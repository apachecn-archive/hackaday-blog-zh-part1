# 蚀刻素描电脑是一个惊人的简单黑客

> 原文:[https://hackaday . com/2011/01/31/etch-a-sketch-computer-is-a-simple-hack/](https://hackaday.com/2011/01/31/etch-a-sketch-computer-is-a-surprisingly-simple-hack/)

![](../Images/13d69a6b46dbc2429648785a80520cdf.png "etch-a-sketch-computer")

我们不确定是否称这个[为蚀刻草图升级或计算机界面降级](http://northackton.stdin.co.uk/blog/2011/01/etch-a-sketch-controlled-computer/)，但不管怎样，它都是独一无二的。[Martin Raynsford]在他的电脑显示器上添加了一个熟悉的红色框架，带有经典的白色旋钮，可以控制水平和垂直光标移动。甚至还有点击选项，通过按下按钮，正如你在休息后看到的，修改产生了一个完美可用的数字蚀刻草图。我们已经看到了许多由电脑控制的玩具版本[，它们使用了奇特的部件，需要相当多的技巧来制作。这种对功能的模仿很容易构建，这种想法非常简单。[马丁]把编码器轮和鼠标分开。他把每一个放在一个旋钮上，把传感器和微型开关的电线接回到原来的 PCB 上，PCB 贴在显示器的背面。从计算机的角度来看，它看起来和行为就像一个普通的鼠标，但这更有趣(但效率更低)。](http://hackaday.com/2010/02/04/propeller-takes-step-a-sketch-to-a-new-level/)
