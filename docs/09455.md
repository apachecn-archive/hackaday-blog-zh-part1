# DIY 菲涅耳反射镜

> 原文:[https://hackaday.com/2012/01/09/diy-fresnel-reflector/](https://hackaday.com/2012/01/09/diy-fresnel-reflector/)

![](../Images/52a207cee1d2dbeaff19db331b8b0ac5.png "fresnel")

就像用放大镜摧毁一个蚁群一样，没有什么比用一个漂亮的抛物面反射器煮热狗和烤棉花糖更好的了。当然，用聚酯薄膜或铝带覆盖一个旧的圆盘式卫星电视天线并不需要太多技巧，不管它有多酷。[Uwe]想出了一个更具技术含量的方法[建造一个菲涅尔反射器](http://www.dr-iguana.com/prj_flatpackmirror/)，它可以在几秒钟内煮好你的热狗，但只能在晴天。

当他意识到一个扁平的纸板环去掉一部分后可以连接成一个圆锥截面时，他引导了一点阿波罗尼奥斯。产生的截面看起来就像菲涅耳反射镜中的一个同心环。【Uwe】用 Visual Basic 写了一个小程序，计算他的圆锥曲线的必要直径和角度。

一块硬纸板被剪了下来，用一些非常反光的铝带拼凑起来。由此产生的菲涅尔反射镜将正常太阳辐射的 117 倍集中到一个小点上。在建筑纸上烧个洞绰绰有余，但我们今天的午餐将使用微波炉。
