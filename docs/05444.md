# 超级探针

> 原文：<https://hackaday.com/2010/06/04/superprobe/>

Superprobe 是一个逻辑分析仪、万用表以及更多有趣的构建项目。[本·里维斯]没有提出[最初的想法](http://mondo-technology.com/super.html)，但他肯定拿了一个好东西，并把它变得更好。您可以使用它来测试逻辑，将逻辑注入电路，读取电容和电阻，测试频率，从 1 线器件读取器件地址，等等。可互换的探头、内部或外部电源的选择、简单的双按钮操作以及强大的 PIC 微控制器都使它成为您的电子工作台的绝佳工具。休息之后，看看[本]为展示他的修补成果而制作的高质量视频。
