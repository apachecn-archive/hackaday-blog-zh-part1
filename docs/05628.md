# 自主电池交换

> 原文：<https://hackaday.com/2010/07/09/autonomous-battery-exchange/>

这看起来像一个摩天轮，但它却是小型机器人的电池更换站。当电池电量不足时，[号机器人](http://mobots.epfl.ch/marxbot)会前往电池站。一旦进入支架，两侧的手臂将机器人固定在适当的位置，同时将低电量电池吸出，并从 15 槽传送带中插入一个新电池。在转换过程中，机器人不会断电，而是依靠一些大电容器储存的电力。休息之后再看。

不久前，我们看到了一个可以把自己插到墙上插座的机器人。这很好，因为机器人不必回到充电站，但它仍然需要等待电池充满。有了几个战略性放置的电池站，就可以轻松保持机器人正常运行，几乎不需要停机更换电池。
