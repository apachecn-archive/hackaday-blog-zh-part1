# 把你的照相手机变成盖革计数器

> 原文：<https://hackaday.com/2012/01/15/turn-your-camera-phone-into-a-geiger-counter/>

下次当你在机场的安检口排队等候时，为什么不拿出你的智能手机来数一数那些人体扫描仪和 x 光机发出的辐射呢？[有一款针对这种情况的应用，由【罗尔夫-迪特·克莱恩】先生提供。](https://market.android.com/details?id=com.rdklein.radioactivity&feature=search_result#?t=W251bGwsMSwyLDEsImNvbS5yZGtsZWluLnJhZGlvYWN0aXZpdHkiXQ..)

该应用程序的工作原理是用一片胶带或塑料阻挡所有进入手机摄像头传感器的光线。因为高能辐射会在手机内部的 CMOS 摄像头传感器上产生伪像，辐射将被捕捉为微小的白光斑点。这篇文章的标题图片是从慕尼黑亥姆霍兹研究中心的照相手机拍摄的，当时手机正沐浴在每小时 10 西弗的铯-137 衰变产生的伽马辐射中。

我们必须注意到，来自 CMOS 相机传感器的“坏数据”并不罕见。这些可能来自传感器本身的电气异常，甚至来自电池的热量。[Rolf]的应用程序读取本底噪声，并从计数器中减去它。放射性衰变产生的β粒子，如香蕉中的钾-40 或花岗岩柜台中的铀，并没有真正记录下来，尽管[Rolf]在氯化钾和长测量时间方面取得了一些成功。尽管如此，这是一个非常酷的方式把手机变成一个三录相机。
