# 构建一个简单的 FM 发射器 Bug

> 原文:[https://hack aday . com/2011/07/16/building-a-simple-FM-transmitter-bug/](https://hackaday.com/2011/07/16/building-a-simple-fm-transmitter-bug/)

![simple_fim_transmitter_hack_a_week](../Images/ad25a692a6158da8a224630fabe7bd19.png "simple_fim_transmitter_hack_a_week")

[Dino]通过一个朋友得到了一个 FM 发射器“bug”套件，并认为它会使[成为一个简单而有趣的每周黑客项目](http://hackaweek.com/hacks/?p=283)。该套件是简单的双晶体管半波调频发射机，制造商建议可以用来窃听一个房间，因此得名。在对说明书开了一点玩笑之后，[迪诺]开始着手建造发射器，不到一个小时就完成了。

一旦他把所有的东西都焊接在一起，他会花一些时间来测试错误，并解释电路板的各个部分如何一起工作来传输调频信号。他提到，增加一个偶极天线可以很容易地扩展发射机的范围，并简要地调侃了下周的一集，他计划在那里建造一个类似的双级发射机。

这种 FM 电路是初学电子课程中最先看到的几个简单项目之一，所以如果你认识刚开始尝试的人，一定要把这一集传下去。

继续阅读，看[Dino]解释他的 FM bug 发射器的来龙去脉。
