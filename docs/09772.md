Wii 双截棍控制的 Monotron

![](../Images/4ce77d268dd0b6cabf732ef99fa949ac.png "links-wii- nunchuk-monotron")

给你的音乐合成器添加一点运动控制变得非常容易。这里有一个 Wii 双截棍用来控制[单晶体](http://hackaday.com/2010/11/10/monotron-openly-monophonic/)的例子。[谢谢约翰]

**黑客登上月球和其他太空相关目标**

![](../Images/b3fa3dffe4820dd308739cbf99ba5e36.png "links-hackerspace-global-grid")

是的，显然一个非政府支持的登月探险正在进行中。但是在你能爬之前，你必须先走，这个过程的第一步是启动一个黑客空间支持的卫星网络，名为[黑客空间全球网格](http://shackspace.de/wiki/doku.php?id=project:hgg)。看看这个[对该倡议创始人之一](http://www.thepowerbase.com/2012/02/hackers-in-space-hackerspace-global-grid-interview/)【哈迪斯】的采访。[感谢 MS3FGX]

**激光笔和毛玻璃**

![](../Images/19efd1167494dc5667929b0ea22faa72.png "links-laser-pointers-frostd-glass")

我们的印象是激光表演需要精确校准的硬件。但是【Jas Strong】用[激光笔和缓慢旋转的玻璃](https://plus.google.com/116592986723950087833/posts/Q6iT8RE2Bgk)做出漂亮的颜色，证明我们错了。[谢谢迈克]

**MSP430 Twitter 报价器**

![](../Images/1242352008012a217393d083fad95b09.png "links-msp430-twitter-ticker")

[Matt]使用 TI Launchpad 构建了一个 Twitter ticker。它在 LED 矩阵或有机发光二极管显示器上工作，并带有处理 API 的 Python 脚本。

安卓软驱黑客

![](../Images/81d2e024d7e366848c6e6cb3697db6e6.png "links-android-floppy-drive")

[Pedro]向我们展示了他如何用他的 Android 平板电脑读取软盘。硬件包括一个坞站，用于为平板电脑添加 USB 端口，以及一个集线器和 USB 软驱。在软件方面，DOSbox 的 Android 移植完成了剩下的工作。
