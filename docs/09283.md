# 准备好演奏一些邪恶的竖琴吧

> 原文:[https://hack aday . com/2011/12/12/get-ready-to-play-some-wicked-air-harp/](https://hackaday.com/2011/12/12/get-ready-to-play-some-wicked-air-harp/)

![](../Images/2777a07acef31b577862b05eaac98693.png "air-harp")

当你可以在空中挥舞双手来制作音乐时，谁还需要触觉界面呢？ [Air String 使这成为可能](http://people.ece.cornell.edu/land/courses/ece5760/FinalProjects/f2011/yk579_vm285/yk579_vm285/index.html)令人惊讶的是，它没有使用 Kinect 传感器。

在上面的图像中，你可以看到两个绿色的标记帽被用作拨子来从不存在的弦中拉出音乐。明智地栖息在那本用 VHDL 编写的*数字系统分析与设计教科书上的是一台记录玩家图像的摄像机。该信号由 FPGA(因此是教科书)实时处理，并显示在右边的监视器上。一组引导线覆盖在图像上，因此演奏者知道从哪里弹奏她期望的音符。*

该程序被设计成拾取亮绿色作为输入。正如你在休息后的视频中所看到的，它非常有效。负责该项目的康奈尔大学学生团队还提到了一些可能的改进，如添加距离传感器(超声波测距仪？)以便深度可以用于声音的动态。
