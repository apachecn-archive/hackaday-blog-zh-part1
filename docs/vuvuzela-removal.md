# 呜呜祖拉移除

> 原文：<https://hackaday.com/2010/06/15/vuvuzela-removal/>

![](img/aaf342d9b95b56571a1688ad253c5389.png "vuvuzela1-thumb-550xauto-40955 (Custom)")

世界杯期间，我们到处都能听到对呜呜祖拉这种噪音制造者的抱怨。无论你是不是这项运动的粉丝，当一个黑客同伴感到恼火并开始攻击时，你都可以欣赏。[Tube]已经创建了一个软件过滤器，设法从视频中删除呜呜祖拉的声音。他分享了使用 Logic Express 和 Mac mini ( [谷歌翻译](http://translate.google.com/translate?js=y&prev=_t&hl=en&ie=UTF-8&layout=1&eotf=1&u=http://www.surfpoeten.de/tube/vuvuzela_filter&sl=auto&tl=en))创建这一切的过程。也许这也能缓解呜呜祖拉持续不断的抱怨。
