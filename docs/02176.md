# 磁性电影

> 原文：<https://hackaday.com/2008/07/10/magnetic-movie/>

磁场线可能肉眼看不见，但如果我们能看到它们，它们的行为方式会让我们大吃一惊。半导体公司的[Ruth Jarman]和[Joe Gerhardt]希望让每个人都能看到它们，所以他们制作了[磁性电影](http://www.semiconductorfilms.com/root/Magnetic_Movie/Magnetic.htm)，这部电影结合了动画、理论模型和整个地球磁力的实际 VLF 记录，以生动的颜色制作了一部显示磁场在空中移动和跳跃的电影。

这部电影一部分是艺术项目，一部分是科学实验，但我们可以在两个层面上欣赏它，因为观看磁力线的路径和运动既美丽又有益。休息之后你可以一睹为快。
