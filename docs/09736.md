# 隐藏书架开关

> 原文:[https://hackaday.com/2012/02/21/hidden-bookshelf-switch/](https://hackaday.com/2012/02/21/hidden-bookshelf-switch/)

![](../Images/f02553c5712b0f43b1c14a7c7f4fa6c0.png "Hidden-Bookshelf-Light-Switch")

所以你的房子里没有任何秘密通道，但是如果你有一个书架，这个秘密开关可以给你的日常生活增加一些乐趣。[Brandon]在订购时，他看到了一个已经脱销的商用版本，所以他开始自己制作。

他在用开关操作一盏灯。黑客的捐赠部分是一个灯调光器，你可以在大盒子商店找到。这其实只是一个带延长线的穿墙插头。通过将调光器模块从延伸部上切下，可以使用按钮来连接和断开线路中的一个导体。确保使用额定电压或电源电压的按钮！

为了使按钮开关与书一起工作，弯曲一个支架，它将滑入精装书的书脊。我们喜欢他自制的压弯机(角铁，坚固的铰链，和一大块 2×4)用于成型支架。一旦一切就绪，没人会知道这些书有什么特别之处。
