# Delta 机器人 3D 打印机

> 原文:[https://hackaday.com/2011/09/04/delta-robot-3d-printer/](https://hackaday.com/2011/09/04/delta-robot-3d-printer/)

![](../Images/b964479a8acaab15c0d9a53452fcdcfd.png "delta")

有时候，并不是所有的构建都如我们所愿。这就是发生在[Rob]身上的事情，以及他尝试制造一个进行立体光刻的[Delta 机器人](http://www.robhopeless.com/2011/08/delta-robot-3d-printer-sounds-cool.html)。德尔塔机器人能够非常快速和精确地移动，所以[罗布]在手臂末端安装了一个激光模块。在激光前放一些紫外线固化树脂后，他留下了一滴粘性物质，我们正在试图找出原因。

[Rob]认为公认的糟糕的打印质量是由于衍射和反射构建板。如果是这样的话，我们同意在树脂中加入一些染料会有所帮助的评估。[Rob]博客上的一些评论者认为他的激光运行速度太慢了。很遗憾[Rob]放弃了他的构建，把它变成了一个简单的 X & Y 轴构建。Delta 机器人可能真的非常快，增加一台打印机可能意味着打印只需要几分钟而不是几个小时。有几个人[正在努力让德尔塔重新起飞](http://www.reprap.org/wiki/Delta)，但在此之前，这个项目还有一两个原型。休息之后，看看[Rob]在 Delta 机器人立体造型方面的尝试。
