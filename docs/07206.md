# 便携式 NES 控制台得到它哦，所以正确

> 原文:[https://hack aday . com/2011/03/24/portable-nes-console-gets-it-oh-so-right/](https://hackaday.com/2011/03/24/portable-nes-console-gets-it-oh-so-right/)

![nes_portable](../Images/6f09ba03ae3c76ce9ede77658176244a.png "nes_portable")

moddedbybacteria 论坛上的[Doug]一直在努力开发一款看起来很棒的便携式 NES 控制台,并写信告诉我们它已经完成了。

该控制台最初是一个简单的想法，使用任天堂的芯片(NOAC)板，以创建一个便携式控制台，播放原始的 NES 盒式磁带。一旦一个塑料盒和一些电池以及 NOAC 板被找到，这个项目就顺利进行了。

[道格]最终决定，他宁愿在项目中使用一个真正的任天堂主板，所以 NOAC 被废弃…暂时。他不小心烧坏了任天堂的主板，所以它回到了 NOAC，但是升级了线路，以解决他在早期构建中遇到的一些干扰问题。一个原始的 NES 控制器被拆卸下来用于控制台，这给了便携式一个真实的感觉。让箱子看起来恰到好处是一个漫长的过程，但我们认为他做得很好。

如果您想查看他的整个构建日志，请注意它有 23 页长。如果你不耐烦，想直接进入发布会，这发生在第 22 页。

请继续阅读，看看[Doug]在制作过程中和制作完成后放在一起的两个视频。
