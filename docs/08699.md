# 用马里奥启动 1989 年的苹果电脑

> 原文:[https://hack aday . com/2011/09/29/booting-a-1989-MAC-with-Mario/](https://hackaday.com/2011/09/29/booting-a-1989-mac-with-mario/)

![](../Images/f264f3cf19de2ad47258a154d32ab7f3.png "iici")

作为 [68k 麦金塔解放军](http://68kmla.org/forums/viewtopic.php?f=29&t=16544)的新兵，【dougg3】真的是在炫耀自己的硬件黑客能力。他想出了一个替代他的 Mac IIci 的 ROM SIMM，[让它在开机时播放马里奥主题](http://www.youtube.com/watch?v=Yen0omvBo2Y)，而不是正常的编钟。

在这些旧的苹果电脑中更换只读存储器并不是一件不寻常的事情。在一些 68k 机器上，有一个 SIMM 插槽来替换或扩展焊接的 ROM。事实上，[从 IIsi](http://home.earthlink.net/~gamba2/os8_se30.html) 中取出 ROM SIMM 并将其放入[万王之王的计算机](http://www.tuaw.com/2009/03/06/what-every-superhero-needs-watchmen-black-se-30/)中以使 SE/30 32 位干净是相当常见的。我们从来没有见过这些旧 Mac 的可重写 ROM SIMM，所以我们很确定[dougg3]只是从垃圾箱里放了一个 Mac IIsi。

现在，整个 68k 解放军都在叫嚣着要一个[dougg3]的可重写 rom(我们有现金)，如何处理它的问题就出现了。当然，SE/30s 现在可以不用安装 [MODE32](http://en.wikipedia.org/wiki/MODE32) 和[新的启动编钟](http://www.youtube.com/watch?v=bRJtMMYCf0E)就可以 32 位干净了。我们真的很想看到一些硬核 ROM 黑客正在进行，比如在 Quadra 950 中安装 68060。
