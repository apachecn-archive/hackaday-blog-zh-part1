# Kinect +《我的世界》三连胜

> 原文:[https://hackaday.com/2011/03/25/kinect-minecraft-trifecta/](https://hackaday.com/2011/03/25/kinect-minecraft-trifecta/)

![](../Images/abf3be915266b4b9ea30b98a1a68cb9c.png "minefecta01")

今天，我们有一个特殊的款待，三个项目结合了“最快销售的消费电子设备”，Kinect，和“最快销售的独立 java 游戏，曾经让我们整个周末睡不着觉”，《我的世界》！

[Sean Oczkowski]写信告诉我们，他在使用 Ubuntu 上的 OpenKinect Java wrapper 来玩 Kinect 上的[《我的世界》时所做的努力。在维基百科的帮助下，这段代码用了大约 4 天时间完成。使用直方图在视野中定位玩家，脚本计算身体的中心质量，并为占据屏幕象限的肢体定义交互。[Sean]出色地完成了贯穿整个过程以及过程中所做决策的工作。整个事情有点像在](http://code.sketchydomainname.com/2011/03/how-to-playing-minecraft-or-other-games.html)[地方](http://www.youtube.com/watch?v=NzGvbYuwF4o)跑步，我们无法想象在不可避免的爬行遭遇中会发生的扑打。

接下来，我们有请[韦德·麦吉利斯]和他获奖的[《我的世界》Kinect 控制器](http://www.youtube.com/watch?v=dS_N-qIPGY4#t=2m30s)。[韦德]在[的网站](http://www.wademcgillis.com/?page_id=442)上提供源代码和可执行文件。这个版本的控制使用骨骼跟踪数据来感知用户的手势。这仍然包括像僵尸一样伸出你的手，但它更灵活一些，因为你可以把手臂放在自己的身体前面。

最后，[内森·维尼克尼斯]一直在做一些非常有趣的工作，用 Kinect 将巨大的三维模型导入游戏世界。[Nathan]然后再多做一点，给这些数字做动画！看看下面的视频，看看真正令人印象深刻的结果。我们 Hackaday 认为这是这项技术最恰当的用途，并且可能开始在公共服务器上建造我们自己的巨大雕像。
