# 一个什么都有的贸易展览摊位，一个软式飞艇！

> 原文:[https://hackaday . com/2011/11/29/for-the-trade-show-booth-that-everything-a-blimp/](https://hackaday.com/2011/11/29/for-the-trade-show-booth-that-has-everything-a-blimp/)

[![](../Images/5e909bec126b580b6cb0a677fd94372b.png "leetzeppelin")T2】](http://hackaday.com/2011/11/29/for-the-trade-show-booth-that-has-everything-a-blimp/leetzeppelin/)

商展都是为了吸引注意力，让人们了解你的产品，那么还有什么比定制的遥控飞艇更好的呢？当然，你可以只买一个，但是那有什么意思呢？经过几次设计迭代后，[Tretton37]提出了一种被称为[LeetZeppelin]的飞艇，由 Arduino、XBee 模块以及连接到计算机的 Wiimote 控制器控制。

黑客本身就是一个很好的例子，它将现成的材料转化为更有趣、更独特的东西。除了上面列出的组件之外，业余爱好伺服系统还被改装成可以与乐高玩具一起控制齿轮和“轴台轴承”他们的网站上提供了这次黑客攻击中使用的“重要”部件的列表，以及一段视频，也是在休息之后。

至于这次黑客攻击吸引了贸易展的注意力，Fredrik Leijon 是这样说的:“我们认为所有凝视天空和半张大嘴的行为证明这是一个巨大的成功！”
