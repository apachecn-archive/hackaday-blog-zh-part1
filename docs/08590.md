# 打造自己的 CT 扫描仪

> 原文:[https://hackaday.com/2011/09/15/build-your-own-ct-scanner/](https://hackaday.com/2011/09/15/build-your-own-ct-scanner/)

![](../Images/67aa2134bdc398edcf99f137a62381a6.png "mouse")

[Linas]为奖学金竞赛自己造了一个 x 射线发生器。我们认为这对[Linas]来说还不够，因为在 x 射线发生器完成后，[他用他的项目来模拟 3D 物体](http://forum.elektronika.lt/viewtopic.php?t=70636&postdays=0&postorder=asc&start=0) ( [谷歌翻译链接](http://translate.google.com/translate?sl=lt&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fforum.elektronika.lt%2Fviewtopic.php%3Ft%3D70636%26amp%3Bpostdays%3D0%26amp%3Bpostorder%3Dasc%26amp%3Bstart%3D0))。这是一个惊人的构建，让我们为那个在自制 CT 扫描仪中名列第二的家伙感到遗憾。

CT 扫描仪背后的原理相当简单——[围绕旋转轴](http://www.youtube.com/watch?v=rDA-sDNLqSQ)拍摄一系列物体的 x 射线。从那里开始，数字化 x 光图像以产生 3D 模型是一件相当简单的事情。困难的部分是制造 x 射线发生器。[Linas]使用了定向 x 射线管、几个电源以及我们可以从中收集 x 射线的胶片，而不是 CCD 传感器。这部电影被扫描进电脑，然后重新组合成 3D 图像。

[Linas]似乎不太热衷于向互联网上的任何老小丑透露他的建筑示意图，因为他的建筑有高电压和辐射组件。尽管如此，这是一个惊人的建设。

看看 YouTube 上[Linas]的 CT 扫描仪对一只旧电脑鼠标成像的演示，以及休息后在 MATLAB 中对相同数据的重建。
