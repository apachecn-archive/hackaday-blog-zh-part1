# 球形机器人获得一些新技能

> 原文：<https://hackaday.com/2009/10/15/sphere-bots-get-some-new-skills/>

[https://www.youtube.com/embed/OWkK-o4Vq-A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/OWkK-o4Vq-A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

球形机器人，或者在这种情况下，遥控车辆是非常通用的。它们在大多数地形上行驶的速度都差不多，有些情况下包括水面。不过，这并不是说它们在这些地形上行驶得特别好。常见的问题是他们不能很好地爬过颠簸，[直到现在](http://www.botjunkie.com/2009/10/15/spherical-robot-can-climb-over-obstacles/)。我们已经见过[几个](http://hackaday.com/2008/08/06/autonomous-swarm-at-large/) [版本](http://hackaday.com/2009/02/05/radio-controlled-sphere/)的球体机器人，但是它们似乎都需要相当平整光滑的表面，除了[沉入水中的那个](http://www.botjunkie.com/2009/07/21/video-rotundus-groundbot-capable-of-amphibious-assaults/)。虽然我们没有看到任何真正有必要爬楼梯的魅力。事实上，我们还没有看到，但他说可以在休息后的采访中看到。
