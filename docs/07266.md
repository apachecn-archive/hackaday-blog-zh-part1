# 没有任何基板的 PCB

> 原文:[https://hackaday.com/2011/03/31/pcbs-without-any-substrate/](https://hackaday.com/2011/03/31/pcbs-without-any-substrate/)

[Kimio Kosaka]通过 7 号奥巴卡项目将自由形式赛道的概念推向了一个新的高度。这是一个全功能的 Arduino 板，没有板。迹线就在那里，为硬件的其余部分提供所有的支持。你可以想象这个包裹有多脆弱，所以得知 O'Baka 项目意味着“愚蠢”的项目也就不足为奇了。

我们并不认为这种制造工艺完全没有价值。上面看到的“电路板”看起来很有趣，在你想展示电路时，这可能是个好主意。[Kimio]描述了他实现这一造型的过程。第一步，设计单面电路板；他用了 EagleCAD。打印出痕迹后，他用 0.46 毫米的钢丝描绘出每个连接。现在加热烙铁——[ Kimio]建议使用高酸性焊剂来简化焊接到钢材上的过程。

我们认为这将很好地补充像 [LED 立方体](http://hackaday.com/2011/03/20/third-times-a-charm-512-led-cube-kicks-it-up-a-notch-with-rgb-leds/)这样使用电路本身作为支撑结构的项目。休息之后，在视频中从几个不同的角度查看这个 Arduino 替代方案。
