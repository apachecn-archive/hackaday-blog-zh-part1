# 更新:6809 计算

> 原文:[https://hackaday.com/2010/12/15/update-6809-computing/](https://hackaday.com/2010/12/15/update-6809-computing/)

![](../Images/ef3f4f7895bbee0dfa68c58ba52420fa.png "8bit")

马特·萨诺夫正在从头开始设计他自己的 8 位计算机。这意味着不仅要设计硬件，还要编写自己的内核和定制库。自从我们[最后一次看到这台 8 位机器](http://hackaday.com/2010/07/27/6809-computing/)以来，他增加了视频和声音输出，这使他能够开始为他的计算机开发一些软件(休息后看它玩康威生活游戏)。

坚持他的电脑的复古主题，他使用 TMS9918 芯片输出视频，使用 YM2149 输出音频。YM2149 是用于[雅达利 ST](http://en.wikipedia.org/wiki/Atari_ST) 的音频芯片，允许他通过一点黑客技术播放为该系统生成的歌曲，以解决雅达利 ST 运行在 8MHz 而他的[摩托罗拉 6809](http://en.wikipedia.org/wiki/Motorola_6809) 仅运行在 2MHz 的事实。
