# 旧弹球游戏增加了奖励系统

> 原文:[https://hack aday . com/2011/08/08/old-pachinko-game-tweaked-to-add-a-reward-system/](https://hackaday.com/2011/08/08/old-pachinko-game-tweaked-to-add-a-reward-system/)

![restored_gumball_vending_pachinko_machine](../Images/4af233c20f6b3669d570e0a75a27b504.png "restored_gumball_vending_pachinko_machine")

蒂姆·希金斯(Tim Higgins)在一次车库拍卖中为他的妻子买了一个旧弹球游戏，但它在车库里闲置了几年。当他最终把它掸掉灰尘时，他决定要为它修复并建造一个漂亮的橱柜，尽管他认为这个想法有点蹩脚。

他说他喜欢在他的项目中使用某种 CPU，尽管这有点过分，但他把在游戏中加入某种微处理器作为自己的目标。他不想破坏机器的原始美学，所以他决定可以使用 Arduino 来驱动一个针对熟练弹球玩家的奖励系统。

他用一些 PVC 管建造了一个由 Arduino 控制的加料斗。当玩家赢了，微控制器触发一个小的业余爱好伺服系统，分发口香糖球/糖果等。

[Tim]说他的妻子喜欢这份礼物，他对礼物的外观也很满意。点击他的博客了解更多的建造细节，并确保查看我们在下面嵌入的修复照片幻灯片。
