# 用电视遥控器控制闪光灯的功率

> 原文:[https://hackaday . com/2010/09/03/用电视遥控器控制闪光灯的电源/](https://hackaday.com/2010/09/03/controlling-the-power-of-a-flash-with-a-tv-remote/)

![](../Images/8aa877f112fb96faff40a09b9febc3a6.png "diy_flash_power")

对于那些喜欢摄影的人来说，设置灯光是每次会议最大的任务之一。有一些闪光灯可以通过遥控器来控制，以帮助简化这一过程，但它们的成本会高得多。[Dsvilko]与我们分享了一个相当简单的电路，它允许你使用电视遥控器来控制你的闪光灯。他用的是 picaxe 08m，他说在这个项目之前，他从未用过微控制器。我们说你的第一个微控制器项目做得很好。休息过后，你可以观看它的视频。
