# 聚碳酸酯鱼使用三个伺服系统游泳

> 原文:[https://hack aday . com/2009/10/17/聚碳酸酯-鱼用-三伺服-游泳/](https://hackaday.com/2009/10/17/polycarbonate-fish-uses-three-servos-to-swim/)

![polycarbonate-fish](../Images/546fa5496a5f704a75967c782e388fd6.png "polycarbonate-fish")

[Amnon]正在艰难地学习水和电子并不总是喜欢很好地一起玩。他一直致力于创造一条会游泳的鱼，它使用三个伺服系统来弯曲一张鱼形的聚碳酸酯。这张照片并没有真正做到项目公正，但你可以通过休息后观看视频来更好地了解他的成就。

三个伺服系统以及一些用于避障的距离传感器都由 PIC 16F877A 微控制器控制。[Amnon]尝试了三种不同的防水方法；将器件涂上清漆，浸入热胶中，然后浸入环氧树脂中。前两个导致电子设备进水损坏，但第三个设法工作。它阻止了水进入，但也阻止了控制器的重新编程。

尽管没有成功，我们还是很想看到将鱼浸入一大桶熔化的胶水中的过程。一旦完善，这可能是[携带我们的毁灭武器](http://hackaday.com/2009/10/12/underwater-laser-just-add-shark/)的完美平台。
