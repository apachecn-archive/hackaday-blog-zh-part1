# 自动宿舍门

> 原文：<https://hackaday.com/2009/07/06/automated-dorm-room-door/>

[https://www.youtube.com/embed/46s7nE72nvI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/46s7nE72nvI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Chris]来信告诉我们他住在宿舍时做的这个项目。他建立了一个系统来自动化他的宿舍门。它通过 iPhone，秘密敲门，甚至钥匙来处理解锁和打开/关闭门。锁/旋钮部分由伺服机构操纵，而打开/关闭动作是液压的。在和它一起生活了一年后，他说它永远不会变旧，但还是有一些小毛病。显然它有时会在半夜随机打开。如果你有兴趣做这样的事情，但不损坏门，也许你应该检查一下 [RFID 宿舍门锁项目](http://hackaday.com/2009/01/02/rfid-dorm-room-door/)。
