# 第一个 Raspberry Pi 构建是 MAME 机器

> 原文:[https://hack aday . com/2012/03/22/the-first-raspberry-pi-build-is-a-mame-machine/](https://hackaday.com/2012/03/22/the-first-raspberry-pi-build-is-a-mame-machine/)

Raspberry Pi 在近一个月前推出，但这些奇妙的廉价单板计算机仍在从中国到全球黑客和建筑商工作台的途中。虽然他们还没有发货，但很多人都迫不及待地想用 Raspi 做些有用的事情。[Nicholas]认为他应该马上投入运行，所以他[模仿树莓派](http://1337technophile.blogspot.com/2012/03/compiling-xmame-106-on-rasberry-pi.html)为他的新玩具到来时他将建造的 MAME 机器做好一切准备。

[Nick]在官方的 Raspberry Pi 论坛上找到了一张 Raspi VirtualBox 图片。在启动网络浏览器并在控制台上敲几下键盘后，他将注意力转向了 MAME 模拟器。这是一个相对简单的安装(虽然编译花了六个小时)，但是我们确信 Raspi 将会出现在相当多的 MAME 版本中，所以这是值得的。

当然，你一个月前订购的树莓派可能现在正在大海中央的集装箱船上，但这并不意味着你不能开始规划你的制作。只需加载一个 VirtualBox 映像，查看几个教程，就可以开始了。
