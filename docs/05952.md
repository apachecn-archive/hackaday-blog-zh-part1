# 激光字幕投影仪

> 原文:[https://hackaday.com/2010/09/15/laser-marquee-projector/](https://hackaday.com/2010/09/15/laser-marquee-projector/)

![](../Images/a423d1e8c4df8c674a3b66ffb6f9730b.png "laser-message-scroller")

这个[激光信息滚动器](http://codinglab.blogspot.com/2010/09/diy-laser-projector.html)是用便宜的零件制造的。[劳尔]系统的核心是一个旋转的药丸盒，上面有八面镜子。每个都将激光重定向到投影表面的不同垂直部分。该设备上有八个小臂，每个小臂在光学传感器旋转时打断它的光束，从而促进正确生成投影图像所需的精确同步。在休息后的视频中，我们可以看到一个看起来像 Arduino 的东西在控制系统。这是有意义的，因为它很容易连接激光指示器和传感器，USB 连接允许消息流向系统。
