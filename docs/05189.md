# 由激光指挥家指挥的螺线管乐队

</span> <p>这个视频让我们喜笑颜开。格里芬·米尔萨普正在用螺线管乐器乐队创作现场音乐。每个螺线管被设置成撞击一个物体，例如碗或杯子。触发机制是乒乓球内部的光传感器。仪器的收集由安装在发动机上的绿色激光器进行。当光束经过一个乒乓球时，里面的光电晶体管启动螺线管，发出一个音符。音高比<a href="http://hackaday.com/2010/02/17/robo-vibe/">的 Robo-vibe </a>更随机一点，但听到[格里芬]取得的成绩令人高兴。</p> </body> </html>
