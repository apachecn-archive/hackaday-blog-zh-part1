# 重新点亮仪表组

> 原文:[https://hackaday.com/2011/12/29/relighting-a-gauge-cluster/](https://hackaday.com/2011/12/29/relighting-a-gauge-cluster/)

[![](../Images/9bb879dd4569c3b7c415aabebf0c019a.png "gauge")T2】](http://hackaday.com/wp-content/uploads/2011/12/gauge.jpg)

当[加勒特]卡车仪表板上的几盏灯烧坏时，他正看着一张高额的修理账单。修理店必须更换巨大的印刷电路板来更换几个焊接灯泡，所以他看到了一张 500 美元的修理账单。点亮 LED 灯是每个人的第一个项目，所以[加勒特]决定[换成 LED 灯泡](http://www.garrettgalloway.com/index.php?p=projects&e=4)来节省几美元。

修复非常简单——在移除表盘和针之后，[Garrett]发现了一个巨大的 PCB，上面有几个烧坏的灯泡。他拿了一个万用表到每个灯泡的焊盘上，用一个发光二极管和电阻替换每个灯泡。完成的项目看起来像是从工厂出来的，比最初在他的卡车上发现的丑陋的琥珀色灯泡有了巨大的改进。[Garrett]还上传了一张他建造的漂亮的[指示图，展示了焊接得很好的](http://www.instructables.com/id/06-07-Duramax-LED-Gauge-Cluster/)[灯替换件。](http://www.instructables.com/file/F00SE7RGW88U2ED/)
