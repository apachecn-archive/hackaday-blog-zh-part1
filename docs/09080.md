# 在家制作气凝胶

> 原文：<https://hackaday.com/2011/11/16/making-aerogel-at-home/>

[本]超越了自己。他在自己的车库里成功制作了硅气凝胶单片。气凝胶是一种被称为“硬空气”的轻质固体，在 T2 非常昂贵，尤其是非颗粒状的。

生产气凝胶的技术已经在互联网上流传了很长时间。生产需要一些不常见的化学物质和一个超临界干燥室，这意味着在家里制造硬空气需要很多专业知识。不知何故，[本]得到了一些四甲氧基硅烷，很难得到的成分，并用管件和液态二氧化碳制作了一个超临界干燥室。

最终，[本]能够制造出几小片气凝胶。他的作品的大小受到他的“模具”(实际上是一个注射器)和他的干燥室的大小的限制。[本]很有可能建造一个更大的超临界干燥室，制造更大的气凝胶块，并以数百美元的价格出售。
