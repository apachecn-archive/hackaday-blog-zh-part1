# 生日快乐游戏机

为了庆祝 Gameboy 的 20 岁生日，[stupid ventions 发布了一段视频，展示如何更换屏幕](http://www.youtube.com/watch?v=NN5duEPO_vc)。它们往往是一个共同的弱点。本质上，他们只是买了另一个，我们假设坏了，Gameboy，并把它们换了出来。这不是一个非常复杂的技巧，但是很高兴知道它如此简单。我们对 Gameboy 有着美好的回忆，当他吹进弹夹时，这些回忆如潮水般涌来。生日快乐游戏机。
