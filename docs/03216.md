# 语音控制的家庭自动化

> 原文：<https://hackaday.com/2009/02/14/voice-controlled-home-automation/>

![stephanie](img/aea32f4709e9afad309ea72a805958d8.png "stephanie")

[Brian]提交了关于他的[语音控制家庭自动化系统](http://absolutemaximumratings.com)的文章。从语音识别系统微软 SAPI 开始，他编写了一些基本的家庭自动化程序。为了让这个项目更加令人敬畏，他决定为他的自动化系统建立一个物理模型。这个没有实体的头是“斯蒂芬妮”。她对自己的名字有反应，有一个随着单词中的音节移动的铰接下巴，甚至可以根据命令弹出她的“大脑托盘”。我们想要一个。

在他的网站上有很多关于相关电路的信息，还有源代码和视频。休息之后你可以看视频。
