# SheevaPlug PBX

[[Greg]一直在开发 Debian/free PBX/Asterisk](http://plugpbx.org/)的一个版本，让人们能够使用 SheevaPlug。如果你还没看过，SheevaPlug 是一个装在墙上插座里的微型电脑。当他们去年宣布的时候引起了一些轰动，我们[已经](http://hackaday.com/2009/02/25/sheevaplug-tiny-linux-server/)提到他们[几次](http://hackaday.com/2009/08/21/openpogo-an-alternative-to-pogoplug-software/) [次](http://hackaday.com/2009/04/29/hackit-sheevaplug/)。[Greg]正在为 SD 卡提供预配置格式的操作系统，因此您可以下载并将其放入您的 SheevaPlug 中。是的，在它启动之前还有一些工作要做，你可以在上面的视频中看到。感谢[Greg]发送此邮件，继续努力。
