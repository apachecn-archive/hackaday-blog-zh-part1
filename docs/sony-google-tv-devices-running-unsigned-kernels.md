# 运行未签名内核的索尼谷歌电视设备

> 原文：<https://hackaday.com/2012/02/20/sony-google-tv-devices-running-unsigned-kernels/>

![](img/f832adb6e7bab7b49053e96736adbe00.png "sony")

“我是 root”的自豪叫声再次响起，这一次是在索尼谷歌电视设备上。虽然在以前的固件版本中发现了低级别的漏洞，但是[降级过程让你可以在更新的电视或互联网流媒体设备的蓝光型号上运行未签名的内核](http://gtvhacker.com/index.php/About_Sony_Downgrade_%2B_Rebooter_(Root))。

这些系统是基于 Android 的，目前运行 3.1 版本的蜂巢。这个版本修补了以前的漏洞，但有了三个不同的 u 盘，你可以降级，利用和升级到最新内核的改变和未签名的黑客。这给了你渴望已久的 root 访问权限，但是除了在论坛帖子中讨论的[特性之外，这个漏洞还没有太多变化。](http://forum.gtvhacker.com/announces/topic645.html)

我们一直在寻找在客厅设备上运行的开源项目，并希望有一天我们能看到 GTV 的 XBMC 分支。在那之前，我们只能祈祷拉斯伯瑞皮·XBMC 的生存能力。
