# 吉他成为一个改进的彩色器官

> 原文:[https://hack aday . com/2011/07/24/guitar-bees-a-improved-color-organ/](https://hackaday.com/2011/07/24/guitar-becomes-an-improved-color-organ/)

[![](../Images/60593d77e9b660c0136691d92027b130.png "guitar")T2】](http://hackaday.com/wp-content/uploads/2011/07/guitar.png)

[查尔斯]想在他的吉他里放些发光二极管。他还希望单个音符输出特定的颜色，但他找不到任何基于音调的算法将声音转换为颜色的项目。经过大约一年的工作，他的[彩弦吉他](http://cnlohr.blogspot.com/2010/11/colorchord-sound-lighting.html)诞生了。

与我们见过的其他颜色器官不同，音符的颜色与音符的绝对音高无关。相反，颜色映射在一个音乐键内。I 和弦永远是黄色的，IV 和弦永远是紫色的，V 和弦永远是蓝色的。弹奏 C 调将使 led 分别为 C、F 和 G 和弦输出黄色、紫色和蓝色。

现在，吉他音频输出的处理是由个人电脑来完成的。[Charles]说延迟是一个大问题，所以我们猜测这是他尚未转向嵌入式解决方案的原因。他还没有完全弄清楚如何自动确定一首歌曲的基调，但他正在努力。

[查尔斯]的造型让我们想起了 70 年代早期的里肯巴克灯展。将里肯巴克与 lucite Strat 复制品进行比较是错误的，但我们要说的是，ColorChord 优于里肯巴克的[手绘汽车尾灯灯泡](http://www.the-music-connection.com/331.htm)。
