# 在家阳极化钛

> 原文:[https://hackaday.com/2012/02/26/anodize-titanium-at-home/](https://hackaday.com/2012/02/26/anodize-titanium-at-home/)

![](../Images/b26628d1162b9c387543a58ab71dbe10.png "anodizing-titanuim-at-home")

我们周围真的没有钛，这不是一种便宜的材料。但是这个技巧向你展示了如何在你的家庭实验室(或厨房)阳极氧化钛，它可能有助于金属进入未来的项目。这一过程似乎并不太困难或危险，而且在完成时有可能获得许多不同的颜色。

在上图中,[PinkFlute]使用无糖汽水可乐零度作为化学试剂。附在器具上的鳄鱼夹提供正电压，浸在饮料中的黄线是负电压。饰面颜色由提供的电压决定。基于大约 100 伏到 20 伏的电压范围，可以选择绿色、紫色、黄色和蓝色的各种色调。

这是两种阳极氧化方法之一。另一种是用泡沫刷蘸苏打，负极引线夹在上面。你只要刷上带电物质就可以改变野营车的漆面。
