# 层流水射流解释

[戴夫]组装了这个[层流水射流](http://www.youtube.com/watch?v=Ijn98G0I99E)，主要来自 PVC 和吸管。没有项目页面，但他确实深入解释了它是如何工作的。水从底部进入，被一系列海绵减缓速度，然后被迫通过一列吸管。然后，它在顶部汇集，然后被迫通过一个非常光滑和尖锐的喷嘴。我们设法找到了这个[的另一个视频，制作了一个 15 美元](http://www.youtube.com/watch?v=OV-IazRk0sU)的视频，里面有大量的信息和链接。多久我们才能在我们的读者中看到一个完整的音乐同步喷泉？
