# A.R.T .帮你整理可回收物品

> 原文:[https://hack aday . com/2012/02/04/a-r-t-sorts-your-recyclables-for-you/](https://hackaday.com/2012/02/04/a-r-t-sorts-your-recyclables-for-you/)

![](../Images/a90ee64ab8225d014c34c34f5e808fb2.png "automated-recycling-sorter")

[Yuhin Wu]来信让我们了解一下多伦多大学的一群同学制作的自动回收分类器。他们参加了学校的学生设计竞赛，我们很高兴地宣布它获得了第一名。

成角度的滑轨设计用于分离玻璃、塑料和金属容器。第一次分拣发生在入口区。一套[力臂](http://en.wikipedia.org/wiki/Moment_(physics))用于剔除玻璃瓶。因为有几个在一排，一个更大更重的塑料容器不会被错误地分类，同样的情况也适用于较小的玻璃瓶。

将玻璃从混合物中取出后，团队继续将金属和塑料分开。阿尔杜伊诺被用于此目的。它感应到金属罐通过溜槽时引起的电气干扰，并启动活板门对其进行分类。塑料对该传感器没有影响，会滑过收集器，进入自己的分类箱。
