# 真棒光纤 LED 海盗头盔

> 原文:[https://hack aday . com/2011/09/01/awesome-fiber-optic-led-Viking-helmet/](https://hackaday.com/2011/09/01/awesome-fiber-optic-led-viking-helmet/)

![octobrite_led_mohawk_helmet](../Images/d667bdeed7454b21b27ff2bbeec9e775.png "octobrite_led_mohawk_helmet")

在 MaceTech 的一个朋友找到了他，他需要一个安装在维京人头盔上的发光莫霍克发型，他马上就要。

现在，[加勒特]在 LED 方面做了大量的工作，但不是每天都有人要求你做一个声音响应 LED 莫霍克发型。他手头上有各种发光二极管和其他部件，但找到构成莫霍克发型的光纤花了一些时间。

经过一番搜寻，他找到了一些廉价的散装光纤玩具棒，并忙着将它们切开，取出光纤束。这些纤维被粘在一个激光切割的塑料组件中，在那里它们与手头的几个 OctoBrite CYANEA 模块配对。他从 SparkFun 购买了一些组件，包括控制设备的 Arduino Pro Mini，以及处理音频输入/过滤的驻极体麦克风和图形均衡器芯片。

他把莫霍克发型的代码部分包起来，交给了他的朋友，他的朋友说“这个头盔是+99 到 epic awesomeness”，这对我们来说听起来像是一个响亮的认可。
