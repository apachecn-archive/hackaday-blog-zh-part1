# 2010 年见证新的船闸运行速度记录

> 原文：<https://hackaday.com/2010/05/28/2010-sees-new-lock-impressioning-speed-record/>

[https://www.youtube.com/embed/Bj9KEmLWRek?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Bj9KEmLWRek?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Jos Weyers]向我们透露这个[锁印象视频](http://www.youtube.com/watch?v=Bj9KEmLWRek)。它显示了他在今年的 SSDev 大会上的最后一轮锁印象锦标赛。尽管他比去年在 T2 创下的 87 秒单锁世界纪录缩短了 15 秒，但他还是获得了第三名，因为比赛平均要进行几轮。

这种开锁的方法是在检查钥匙坯上因试图开锁而留下的微小痕迹后，用锉刀来制造正确的齿。我们已经看到了[箔压印](http://hackaday.com/2010/02/03/foil-impressioning-defeats-security-locks/)以及[电子压印](http://hackaday.com/2010/02/15/electronic-key-impressioning/)，但是[的比赛视频](http://hackaday.com/2008/11/10/impressioning-at-lockcon/)让这成为我们最喜欢的方法。
