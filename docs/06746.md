# 彩弹计时器

> 原文:[https://hackaday.com/2011/02/06/paintball-chronograph/](https://hackaday.com/2011/02/06/paintball-chronograph/)

![](../Images/5da56edcdb7f0bfce3c1e07acae6c227.png "PaintballBallistic-Chronograph")

这个[光滑的小计时器](http://www.instructables.com/id/PaintballBallistic-Chronograph/)可以告诉你你的彩弹的速度，以及你的射速。在这本由[Klash69]编写的说明书中，你可以看到如何用不到 40 美元为自己建造一个。计时表本身通常不太有趣，但我们认为他在这方面做得很好。你有一个漂亮的紧凑型包装，配有一个明亮的大显示屏。它现在真正需要的是一个平滑的外壳。至于技术细节，他在枪管处使用间隔 4 英寸的红外传感器进行探测。我们不是专家，但我们认为由于枪口的闪光，这可能不会在枪上很好地工作，实际上知道的人应该在评论中让我们知道。大脑是 PIC18F13k50，你可以在说明书上下载完整的零件清单和原理图。
