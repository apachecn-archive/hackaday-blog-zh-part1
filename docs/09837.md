# 用家用产品酿造火药

> 原文：<https://hackaday.com/2012/03/06/brewing-up-gunpowder-with-household-products/>

当僵尸来袭时，你不会想跑去商店买更多的弹药。但是你可以随时给曾祖父的火枪装上自制的火药。玩笑归玩笑，火药的发明和扩散对世界产生了深远的影响。在这里，你可以看到用我们现代世界中常见的化学物质来制作它是多么容易。

这个实验中用到的两种化合物是硝酸铵和氯化钾。你从哪里能得到这些材料？速溶冰袋使用硝酸铵和水来启动吸热反应。氯化钾可以在杂货店找到，作为食盐的替代品。

这些化学品需要按重量来计量。[William Finucane]手头没有电子秤，所以他用一把木尺、两个塑料抽屉和一个 Bic 打火机作为支点来平衡。用大约相等的两种材料，他开始溶解在水中，过滤，加热混合物产生硝石。把这个和糖粉混合，你就得到了火药。不相信有效果？你可以在休息后的片段中看到火热的善良。
