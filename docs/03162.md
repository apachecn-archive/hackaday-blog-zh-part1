# 伪 3D 聊天

> 原文:[https://hackaday.com/2009/01/29/pseudo-3d-chat/](https://hackaday.com/2009/01/29/pseudo-3d-chat/)

![3d_video-conference](../Images/82deb7dcd64b284831457fad701fde22.png "3d_video-conference")

[Chris Harrison]和[Scott E. Hudson]构建了一个新颖的系统，用于伪造 3D 视频聊天会话。它们的实现将聊天参与者的图像与背景分开。然后，他们会根据观众头部的移动动态调整视频的位置。他们使用 OpenCV 库来做面部识别(就像[的笑人演示](http://hackaday.com/2008/12/31/laughing-man-in-processing/))。3D 效果与你在[李炯楷]的[中看到的非常相似。下面嵌入了伪 3D 聊天的视频。](http://hackaday.com/2007/12/21/wiimote-head-tracking-desktop-vr-display/ "Wiimote head tracking desktop VR display  - Hack a Day")
