# 用 FPGA 构建时间陶笛

> 原文：<https://hackaday.com/2011/12/13/building-the-ocarina-of-time-with-an-fpga/>

![](img/b45e56b5954a8b26d8feaa2b2ee677a3.png "ocarina")

[Joe]和[Evan]想在康奈尔大学的 FPGA 课程中找点乐子。当面对学期初要做的事情时，他们认为加法合成是一个值得追求的目标。他们最终为他们的最终项目制作了时间陶笛。

这些人从录制一个真实的陶制卵形笛开始，并计算出每个谐波的相对功率水平。因为任何声音都可以由一束正弦波合成，让他们的 Altera FPGA 板复制这些频率可以产生一个很好的陶制卵形笛声音

[乔]和[埃文]的陶制卵形笛有一个“吹口”，就是一个小麦克风。这个麦克风连接到 FPGA 板上，控制音量。可悲的是，这些人没有时间拆开 N64 控制器，所以 6 个红色按钮作为手指孔。

从休息后的视频来看，[乔]和[埃文]真的把一些听起来像林克的陶笛的东西拉在了一起。干得好，伙计们。
