# 可变螺距四轴飞行器倒立飞行

> 原文：<https://hackaday.com/2012/02/24/variable-pitch-quadrocopter-flies-upside-down/>

![](img/657d0e2bc12743e99f7ccbad6b13b8ca.png "quad")

由[马克·卡特勒]和[乔纳森·p·豪]设计的[可变螺距四旋翼飞行器](http://acl.mit.edu/projects/vpitch_quad.html)直接来自航空航天控制实验室。虽然真正的全尺寸直升机总是有可变桨距旋翼，但在遥控飞机上改变叶片的桨距是一种相当不常见的改装。然而，如果操作得当，能够轻松改变螺旋桨的推力方向会带来非常酷的飞行，就像[让飞机机头朝下悬停。](http://www.youtube.com/watch?v=sN7jz7P7p1Y)

[Mark]和[Jonathan]发现了可变螺距四旋翼飞行器可以带来的两种有趣的技术。首先是轨迹生成——由于增加了机动性，他们的四旋翼飞机可以在遵循预编程路径时执行更积极的倾斜转弯。他们设计的第二个好处是快速减速。在休息后的第一个视频中，您可以比较可变螺距和固定螺距四轴飞行器的减速率。虽然固定螺距的四轴飞行器在被命令停止后继续爬升，但配备可变螺距旋翼的四轴飞行器可以在瞬间停止。

我们仍然在等待相当于红牛飞行比赛的四轴飞行器，但当它到来时，我们知道什么会赢得障碍赛。
