# 手腕上的乒乓

> 原文:[https://hackaday.com/2009/09/28/pong-on-your-wrist/](https://hackaday.com/2009/09/28/pong-on-your-wrist/)

![pong_wristwatch](../Images/de6879283cc4ba89e0da8813c4728788.png "pong_wristwatch")

[John]想要一个 [pong clock](http://hackaday.com/2008/09/14/pong-clock/) 并把它放入手表中。花一个下午的时间研究他的[详细构建日志](http://www.maushammer.com/systems/Watch/Build_Log/Build_Log.html)。这个历时多年的项目完成得一丝不苟，干净得让我们嫉妒。他自己加工表壳和按钮，实现了专业的外观，相当于或超过了一些市售“游戏”手表的质量。该项目围绕一个由 [TI MSP430F2013](http://focus.ti.com/docs/prod/folders/print/msp430f2013.html) 处理器驱动的有机发光二极管显示器展开。不要错过休息后的视频，包括原型制作、PCB 加工、表壳打磨和腕表使用。目前，这是第三代开发，但对于这样一个令人兴奋的项目，你真的完成了吗？
