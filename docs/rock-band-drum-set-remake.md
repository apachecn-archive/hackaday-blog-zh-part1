# 摇滚乐队鼓组翻拍

> 原文：<https://hackaday.com/2009/09/11/rock-band-drum-set-remake/>

[https://www.youtube.com/embed/uQhAx4l0fvs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/uQhAx4l0fvs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们已经看到了流行视频游戏摇滚乐队的一些令人印象深刻的改装，从[新钹](http://hackaday.com/2008/12/23/guitar-hero-cymbal-remake/)到[气动脚踏](http://hackaday.com/2009/03/02/breath-controlled-guitar-hero-kick-pedal/)，但我们不能说我们已经看到有人像 EDrums 的人一样走得那么远。他们开始制作自己的[网头](http://www.edrums.info/rock_band.htm)，一个[接线盒](http://www.edrums.info/diyj5.htm)来连接所有东西，然后一个[底座](http://www.edrums.info/rock_band_rack.htm)来将所有东西固定在一起。这绝对是一些奉献和辛勤工作的设置，只会用在电视机前的客厅。休息后，查看更多的视频，以及与原始摇滚乐队鼓组的比较。
