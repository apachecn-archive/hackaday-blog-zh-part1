# 大脑汽车接口

> 原文:[https://hackaday.com/2011/02/18/brain-car-interface/](https://hackaday.com/2011/02/18/brain-car-interface/)

![](../Images/356ee1bfe99defaaa7b33f8731a31cc9.png "Untitled")

AutoNOMOS 实验室项目已经找到了一种新的方式来操纵它的车辆，即你的大脑。我们已经看过了之前的版本，它使用了一辆大部分由电脑控制的货车，由 iPhone 通过[遥控。然而，这个名为“大脑驾驶员”的机器人将操作者放在驾驶座上，头部绑着脑电图仪。](http://hackaday.com/2009/10/24/use-iphone-to-run-yourself-over/)

为了获得更运动的外观，目前的车辆是一辆线控大众帕萨特旅行车，充满了有趣的玩具，如激光雷达/雷达传感器技术，相机和专门的全球定位系统。EEG 接口是一种商业上可用的 Emotiv 模型，在安全的地面上进行几轮训练后，驾驶员被置于对汽车的控制中。

在一个演示中，汽车接近一个四向交叉路口，司机只需向左或向右思考，汽车(智能地)在正确停车并检查障碍物后导航转弯。在第二个演示中，汽车和司机被放在一个未使用的机场来测试反应能力。

如果你喜欢大脑、汽车、机器人和旋转激光器，请在视频休息后加入我们。
