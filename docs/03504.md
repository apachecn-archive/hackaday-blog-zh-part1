# 迪士尼的魔杖

> 原文：<https://hackaday.com/2009/05/17/magic-wands-for-disney/>

[NRP]给了我们一些他的项目。其中最引人注目的是迪士尼资助的一个学校项目。他们将为排队等候乘车的人们提供某种互动娱乐。他们决定采用魔杖风格的界面。每根魔杖都有一个加速度计、一个用于跟踪的红外 LED、一个 XBee 单元和几个用于交互的按钮。他们编写了一些自定义游戏和一个多人白板来测试这一切。休息后，你可以在视频中看到这些动作，以及一个以太空为主题的乒乓游戏。尽管这是由迪士尼资助的，你仍然可以找到所有的[源代码和原理图](http://gitorious.org/fantasia)，都是免费的。
