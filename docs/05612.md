# 多层显示器用水代替屏幕

> 原文:[https://hack aday . com/2010/07/07/multi-layer-display-uses-water-inst-of-screen/](https://hackaday.com/2010/07/07/multi-layer-display-uses-water-instead-of-screen/)

![](../Images/21b6b0da2570a6467a138e023ed767c6.png "multi-layer-display")

这个[多层显示器使用水滴](http://www.cs.cmu.edu/~ILIM/projects/IL/waterDisplay2/index.html)作为投影媒介。通过这种方式，可以看到几个不同的投影区域，形成不太 3D 的分层效果。诀窍在于同步设备的各个方面。有三个歧管，每个歧管有 50 根不锈钢针，用于生产水滴。一个电磁阀驱动液滴，一台照相机在半空中拍摄它们，一台计算机用投影仪同步这些点的图像。在休息后的视频中，你可以看到 SIGGRAPH 2010 演示，其中包括对过程的描述以及动作镜头，包括 3 层版本的俄罗斯方块。
