# 一个会说话的逆向地图拼图盒

> 原文:[https://hack aday . com/2012/01/20/a-talking-reverse-geo cache-puzzle-box/](https://hackaday.com/2012/01/20/a-talking-reverse-geocache-puzzle-box/)

![](../Images/0146f169ba06f99320db8114e29812df.png "talking-reverse-geocache-puzzle-box")

这是一个会说话的逆向地图拼图盒，【Erv Plecter】为他的朋友们做的结婚礼物。这个盒子本身并不是真正的礼物，而是一个惊喜的递送系统，可以从这对夫妇的朋友圈子里收集现金，用于蜜月旅行。我们认为这比得到一个厚厚的账单信封要有趣 50 倍。一旦你最终到达目标地点，谁会真的指望在里面找到现金呢？

与我们最近看到的另一个地理藏宝箱不同，这个没有显示屏来显示目的地的线索。相反，它会播放[Erv]自己录制的音频片段。他们是在开玩笑，这是另一种很好的个人接触。可以移除盒子右侧突出的大头针来播放线索并检查位置。它由一条小链子连接到一个 5 欧元的硬币上，硬币上隐藏着盖子的紧急释放装置。该设备由锂电池供电，可以通过侧面的 USB 端口在不打开盒子的情况下充电。

休息之后我们嵌入了盒子的视频演示。
