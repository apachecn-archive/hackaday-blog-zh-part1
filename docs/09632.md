# 打造你自己的无人驾驶汽车

> 原文:[https://hack aday . com/2012/02/05/build-your-own-driving-car/](https://hackaday.com/2012/02/05/build-your-own-self-driving-car/)

如果你曾经想要一辆自己的无人驾驶汽车，[这是你的机会](https://www.udacity.com/course/artificial-intelligence-for-robotics--cs373)。[斯坦福人工智能班](http://hackaday.com/2011/09/05/want-to-learn-artificial-intelligence-good/)的联合讲师【巴斯蒂安·特龙】(和伟大的【彼得·诺维格】)正在开设一门新课程，将在七周内教会每个报名的人如何编写自动驾驶汽车的程序。

机器人汽车课程与 CS 101 [“编程入门”](http://www.udacity.com/cs#101)课程一起教授。如果你不知道解释器和编译器之间的区别，这个类适合你。你将在七周内学会如何从头开始制作一个搜索引擎。“构建搜索引擎”课程由弗吉尼亚大学的教授(Thrun)和(大卫·埃文斯)教授。无人驾驶汽车课程仅由[Thrun]教授，他凭借他的机器人汽车赢得了 2005 年【DARPA 大挑战赛。

如果你想知道这是否会是另一个像在线人工智能课程一样的一次性交易，不要担心。辞去斯坦福大学终身教授的职位，专心致力于网络教学。他仍然留在斯坦福大学担任副教授，但现在他把时间花在了他的在线大学 [Udacity](http://www.udacity.com/) 上。看起来他可能正忙于他的新项目；到目前为止，计算理论、操作系统、分布式系统和计算机安全等课程都计划在 2012 年开课。
