# Nimbus，便携式 N64

> 原文:[https://hackaday.com/2010/05/08/nimbus-portable-n64/](https://hackaday.com/2010/05/08/nimbus-portable-n64/)

![](../Images/89356632cec25f9801a990ca46e78d6d.png "Oh N64, such...interesting...memories you bring back.")

我们喜欢漂亮而成功的 N64 便携式 mod(如果事实不是[已经很明显的话](http://hackaday.com/?s=n64))。今天，我们将把[唐宁的] Nimbus N64 添加到我们的最爱列表中。

底座由真空成型塑料制成，而按钮来自改良的 superpad，屏幕为 5 英寸的 Zenith。我们无法找到关于电池寿命或重量的消息，但对于那些喜欢打猎或有其他问题的人来说，[建造日志](http://forums.benheck.com/viewtopic.php?f=5&t=36890) [是可用的](http://moddedbybacteria.freeforums.org/downing-s-2nd-n64p-the-nimbus-64-t52.html)。跟随跳转到光轮在行动中的视频。
