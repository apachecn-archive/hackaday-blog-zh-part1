# Android 示波器

> 原文：<https://hackaday.com/2010/10/14/android-oscilloscope/>

这是为你的安卓手机准备的双通道示波器。它使用由 dsPIC 驱动的基本模块进行信号处理。从那里，LMX9838 蓝牙模块将数据广播到手机，以便可以显示波形。[Yus]将他一直使用的一些 Python 代码移植到一组 Java 和 XML 文件上，以便让它在 Android 上工作。这实际上是他第一次与 SDK 一起工作…我们会说他学得很快。休息之后看看它的实际效果。
