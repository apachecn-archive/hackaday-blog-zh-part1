# 迷你四轴飞行器太棒了

> 原文:[https://hack aday . com/2011/04/29/mini-Quadro copter-is-crazy-awesome/](https://hackaday.com/2011/04/29/mini-quadrocopter-is-crazy-awesome/)

![crazyflie_quadrocopter](../Images/fb0b3059f99e28a0cdca5cab9dc194dc.png "crazyflie_quadrocopter")

四轴飞行器最近风靡一时，虽然我们已经看到了相当多的大型设备，[Arnaud Taffanel，Tobias Antonsson 和 Marcus Eliasson]一直在尽职尽责地努力抵制这种趋势。他们的 CrazyFlie 是一个微型四轴飞行器，它使用 PCB 作为设备的主要结构。

由于目标是使用 PCB 作为框架，直升机从一个电机边缘到另一个电机的足迹只有 8 厘米，重量只有区区 20 克！整个平台运行在一个皮质-M3 CPU 上，该 CPU 从一个加速计和一对陀螺仪接收输入，以帮助保持平衡。无线通信通过一个 2.4Ghz 的无线电发射机进行处理，quadrocopter 的电源由一个从 R/C 飞机上收集的 110 mAh 的微型 LIPO 电池组提供。

所有的控制和遥测都由一台电脑处理，它将从飞行员的游戏手柄接收到的控制信息传递给 CrazyFlie。如果一切都由四轴飞行器自己处理，我们很想看看他们是否能保留这个小脚印。不管怎样，这个东西很棒——我们绝对想要一个！
