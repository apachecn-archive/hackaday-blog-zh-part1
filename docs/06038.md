# Mac 平板电脑保持了捐赠者 Macbook 的外观

> 原文:[https://hack aday . com/2010/10/01/MAC-table t-keeps-the-look-of-the-donor-macbook/](https://hackaday.com/2010/10/01/mac-tablet-keeps-the-look-of-the-donor-macbook/)

![](../Images/d2c52bb5bf40ec80d50db936f71f90b2.png "mac-tablet")

大约在 2007 年,[谜企鹅]用 core 2 duo Macbook 制造了一台平板电脑。电池爆炸，损坏了外壳和里面的几个部件。但是平板电脑的新生还是有希望的。他卸下屏幕，通过使用视频输出端口进行测试，以确保没有屏幕电脑仍能正常工作。

事实证明，将整个东西组装成平板电脑有些令人头疼。他解决了在 LCD 和边框之间添加触摸屏引起的问题，并在屏幕翻转的情况下实现了连接。成品看起来很好，因为边框没有像其他平板电脑一样翻转[。取而代之的是，在屏幕后面的地方开了一个观察窗。](http://hackaday.com/2009/10/14/dell-mini9-tablet/)

还有一点要做。上面看到的黑色 USB 电缆是他计划内置的触摸屏的连接器。磁性睡眠传感器已被重新定位到耳机插孔。插入一个插头(不需要电线)可以让它休眠以便运输。还有一个屏幕键盘和将屏幕方向从横向改为纵向的能力。休息之后，我们嵌入了[谜企鹅]的一个演示。[https://www.youtube.com/embed/QZGskdaJww0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QZGskdaJww0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T3】
