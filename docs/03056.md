# 平行停车系统

> 原文：<https://hackaday.com/2009/01/04/parallel-parking-system/>

![parking](img/f718ccc2b7efdbfbd1f60f7f36271b7b.png "parking")

Autoblog [挖出了](http://www.autoblog.com/2009/01/04/from-the-archives-students-build-their-own-parallel-parking-sys/ "Students build their own parallel parking system - Autoblog")这个 2006 年的经典机械工程项目。一个由五名托莱多大学学生组成的团队构建了一个帮助平行泊车的系统。首先，你把机头朝前。然后，液压油缸将驱动轮从行李箱中取出，抬高汽车后部。单轮也是液压驱动的，将汽车移入空间。他们有一个[博客记录了六周的建设](http://utppd.blogspot.com/ "UT Senior Capstone Project Group #16")。看看下面的演示视频。
