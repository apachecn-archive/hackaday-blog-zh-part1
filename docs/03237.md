# 平板步行机

> 原文:[https://hackaday.com/2009/02/20/flatpack-walker/](https://hackaday.com/2009/02/20/flatpack-walker/)

![flatpack](../Images/bedb29a5ebd00f641189e19390f90798.png "flatpack")

如果你对机器人感兴趣，你可能已经熟悉了双马达步行机。这种设计通常用于英国生物工程学会机器人学，是制造四足步行机器人的一种非常简单的方法。[Edwindertien]用这些[平板步行机计划](http://www.instructables.com/id/simpleWalker_4_legged_2_servo_walking_robot)让设计变得稍微容易一点。他使用 4 毫米厚的桦木，通过激光切割使他的助行器成型，但这种设计几乎适用于任何材料。他的步行机是 Arduino 驱动的，所以它可以被[编程为各种行为](http://www.mahalo.com/Arduino "Arduino - Mahalo")，特别是如果你添加一些传感器的话。休息之后你可以看到他的机器人在工作。
