# 高电压:建立自己的 84 千伏避雷针

> 原文：<https://hackaday.com/2011/08/24/high-voltage-build-your-own-84-kv-lightning-stick/>

有句谚语说“说话要轻，拿着大棒”。现在，这种棍子可以采用高压形式。上面的装置，让我们想起了末端粘着一个可乐罐的侧柄警棍，是一个便携式范德格拉夫发电机。

虽然在评论中有争议，但这个黑客的创造者声称，你可以使用该设备用 84 千伏的电电击某人。当然，作为一种武器，它是缺乏的，因为我们谈论的是静电；电压可以通过屋顶，但电流极低。尽管如此，你还是可以用它们做一些有趣的事情。休息后的视频显示，随着灯光变暗，它发出火花。[Yardleydobon]在他的教程的最后还包括了一些其他的技巧。他用另外两个汽水罐制作了一套富兰克林铃铛，其中一个的铝拉环悬挂在它们之间。当他给电池充电时，电池片来回跳动，发出铃声。一旦充好电，只需对其中一个罐子放电就可以重新开始振铃。
