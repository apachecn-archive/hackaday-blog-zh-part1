# 再造第一台个人电脑

> 原文:[https://hackaday.com/2011/09/30/recreating-the-first-pc/](https://hackaday.com/2011/09/30/recreating-the-first-pc/)

![](../Images/20ce6282661467babbb13c04166de62c.png "kenbak")

如果你正在寻找一个简单的 Ardunio 项目，为什么不[复制第一台个人电脑](http://www.funnypolynomial.com/software/arduino/kenbak.html)？

在发现 Arduino 之后，[Mark]意识到重建真正的旧电脑将是一个有趣的项目。一台 Altair 8800 摆在桌面上，但是大量的闪光灯、开关和英特尔 8080 CPU 使得这个项目相当困难。经过一番搜寻，【马克】发现了 [Kenbak-1](http://www.old-computers.com/museum/computer.asp?c=1259&st=1) ，它被广泛认为是第一台个人电脑。Kenbak 还有一个额外的好处，那就是它的 I/O 非常少，而且完全是用 TTL 元件构建的。

由于肯巴克-1 是一台极其简单的计算机，[马克]的构建最终相当小。原理图只是一个 ATmega328，几个移位寄存器和一个实时时钟，用于 OG 计算机没有的一些附加功能。通过按下按钮将机器代码输入到 mega 的 RAM 中，完成的构建被编程，然后被执行。[Mark]已经想出了几个程序——一个用二进制计数的程序，一个“赛昂之眼”,一个 BCD 和二进制时钟。虽然 Kenbak-uno 没有原版那种令人敬畏的复古外壳，但它仍然是一种非凡的构造。
