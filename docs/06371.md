# 暖管钟

> 原文:[https://hackaday.com/2010/11/29/warm-tube-clock/](https://hackaday.com/2010/11/29/warm-tube-clock/)

![](../Images/e312234f187ed568d576e635d2a0e024.png "warm-tube-clock")

暖管钟是数码管钟的新成员。它从冰电子钟中获得灵感，并使用与[相同的电压驱动电路。但是这个用了四根管子，而不是那种很难找的单根管子。它还有其他一些锦囊妙计。容纳管子的屏蔽设计成两种不同的类型。它还为每个灯管配备了一个 RGB LED，增加了上面看到的绿色辉光，并有几个小型霓虹灯灯泡，用作小时和分钟之间的冒号。](http://hackaday.com/2009/08/25/learn-from-the-ice-tube-clock/)

驱动板以运行约 3000 行代码的 ATmega328 为中心。固件提供了许多选项，包括声音反馈，以及对每个时钟，日历，闹钟和 LED 颜色切换的设置。您可以亲自观看插播在广告之后的设置视频，它会带您浏览菜单的每个阶段。我们不得不认为你需要一本操作手册来设置这个东西。
