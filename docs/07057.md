# 你会用 1MW 的激光脉冲手枪射瞎你的眼睛

> 原文：<https://hackaday.com/2011/03/10/youll-shoot-your-eye-out-with-a-1mw-laser-pulse-pistol/>

特种部队用它们来对抗眼镜蛇的邪恶势力。汉·索洛在莫斯·艾斯利酒吧开了他的第一枪。多年来，手持脉冲激光枪一直是只存在于卡通和电影领域的东西…直到现在。

德国黑客[Patrick Priebe]最近建造了一个看起来非常好的激光脉冲枪,它可以很容易地从好莱坞电影布景中获得。它光滑的白色和黑色的外表增加了神秘感，但是对于这种枪的实际威力却没有多少警示。

它装有 Q 开关 Nd:YAG 激光器，一旦电容器充满电，它就会发出 1 MW 的红外光。激光脉冲的持续时间接近 100 纳秒，所以他无法用相机捕捉到它，但它的影响在他射击的任何介质中都很容易看到。激光可以刺破气球，射穿塑料，甚至在刀片上炸出一个洞。

[Patrick]说他非常乐意帮助任何寻找零件来源的人，并为他们自己制造一个零件，那么您还在等什么呢？
