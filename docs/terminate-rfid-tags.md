# 终止 RFID 标签

> 原文：<https://hackaday.com/2009/12/22/terminate-rfid-tags/>

![](img/beaf93bcbe1fead88b7bb819fb5ccb68.png "rfid-zapper")

[这把枪](http://codeninja.de/rfiddler/)只猎杀 RFID 标签。带给我们[激光手势控制](http://hackaday.com/2006/12/17/gesture-laser-pointer-control/)的【mnt】，制造了这个 RFID Zapper，但包含了更多。任何好的武器听起来都必须刻薄，他通过在步枪上安装 MP3 播放器实现了这一壮举。发射 RFID 标签的线圈由一个闪光装置供电，但为了获得视觉反馈，他还有第二个闪光装置来发出你的德国护照失效的信号(见休息后的视频)。

很难相信我们还没有涉及 RFID Zappers。这个概念来自几年前的混沌通信大会。这种方法的工作原理是通过 RFID 标签发送一个非常强的电磁场，使其烧毁。有一个关于 RFID Zappers 的 wiki 帖子，但是当我们加载它时，Firefox 抛出了一个证书警告；[阅读风险自担](https://events.ccc.de/congress/2005/static/r/f/i/RFID-Zapper%28EN%29_77f3.html)。
