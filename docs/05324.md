# Open3DP 着眼于普通材料的 3D 打印

> 原文:[https://hack aday . com/2010/05/11/open 3dp-looks-at-3d-printing-in-common-materials/](https://hackaday.com/2010/05/11/open3dp-looks-at-3d-printing-in-common-materials/)

![](../Images/1a06da4e2be0f571ca58937133be7999.png "3d-glass-printing")

[巴迪·史密斯]给我们发了一个 Open3DP 的链接，他称之为“真正的 3d 打印黑客”。Open3DP 展示了华盛顿大学索尔海姆快速原型实验室的项目。他们正在研究可以普遍获得的材料的 3D 打印，为此[他们公布了在糖、陶瓷和玻璃等材料上进行粉末打印的配方](http://open3dp.me.washington.edu/?cat=6)。看看他们的档案。我们发现[微波窑](http://open3dp.me.washington.edu/?p=632)上的帖子很有趣，还有上面看到的[关于 Shapeways 玻璃印刷](http://open3dp.me.washington.edu/?p=656)的文章。休息之后，我们还嵌入了一个关于 Open3DP 工作的短片。

**更新:**【[马克·甘特](http://www.me.washington.edu/people/faculty/ganter/)】给我们留言，澄清 Open3DP 是大约一年前第一个开发可打印玻璃的公司，名为 Vitraglyphic。他们还将在 [Rapid2010](http://www.sme.org/cgi-bin/get-event.pl?--001887-000007-home--SME-) 上发表论文，并宣布一种新的可打印材料。
