# 纳夫狙击手

> 原文：<https://hackaday.com/2009/03/01/nerf-sniper/>

![snerf](img/a5a0848c72df9071ddac4e3932abe90b.png "snerf")

一个真正的黑客知道，你不能用一件普通的设备就击败竞争对手。Nerf 战争也不例外。这些家伙做了他们必须做的事情，成为办公室战争中的主导力量。他们制造了自己的高威力 Nerf 步枪。使用一些聚氯乙烯，一个洒水阀和一个自行车打气筒，他们制作了一个 Nerf 枪，可以发射大约 300 英尺。休息之后你可以看到一段视频。
