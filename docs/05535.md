# PlayStation 2 便携版

> 原文:[https://hackaday.com/2010/06/22/playstation-2-portable/](https://hackaday.com/2010/06/22/playstation-2-portable/)

![](../Images/d72e134561acbefc61d4ab8a6f823fed.png "ps2-portable")

[Raizer04]刚刚完成他的 [PlayStation 2 便携版](http://moddedbybacteria.freeforums.org/post523.html#p523)。他觉得 PS2 的硬件比 PSP 提供的要多得多，这就是为什么他选择把 PS2 的超薄硬件塞进一个便携的盒子里。他从一个电子玩具开始，充当案例捐赠者，并使用 bondo 为控制器、扬声器、灯和屏幕形成开口。一个漂亮的油漆工作和一些金属工作导致了上面看到的愉快的完成。在背面，您会发现一个带灯的机箱风扇、硬盘和 USB 端口。没有光驱，因为游戏是从拇指驱动器加载的。休息之后看一下演示视频，但是帮你自己一个忙，先把你的声音关小。

如果这不能满足你对便携式游戏机项目的渴望，你也可以看看这个 N64 版本的。
