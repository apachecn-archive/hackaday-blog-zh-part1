# 将路由器转变为 APRS 网关

> 原文:[https://hack aday . com/2011/06/09/turning-a-router-into-an-aprs-gateway/](https://hackaday.com/2011/06/09/turning-a-router-into-an-aprs-gateway/)

![](../Images/89be3ee35b2702ec0136a4571732be0c.png "WRT")

[Chris Kantarjiev]是一名业余无线电爱好者(呼号 K6DBG)，在 APRS 做了很多工作。我们认为他的构建，[将 WRT54gl 路由器变成 APRS 网关](http://www.dimebank.com/cak/k6dbg/k6dbg_igate.html)将对我们一直在讨论的 [APRS 追踪器构建](http://hackaday.com/2011/04/20/trackuino-%E2%80%93-an-open-source-arduino-aprs-tracker/)非常有用。

在 APRS 建立互联网网关，通常需要一台“真正的”电脑。[Chris]不喜欢这个想法，所以他采用了用于嵌入式设备的 igate 软件 apr4r，并将其删减到适合 WRT 的 4MB 闪存和 16MB 内存。

实际的 APRS 硬件通过焊接在 WRT54gl 板上的接头连接，这些接头连接到一个基于 PIC 的小型 TNC。[Chris]认为 APRS 的“主干”很棒，但网络上没有足够的节点来实现全面覆盖。我们认为这将是一个伟大的方式，把廉价的硬件放在野外，以弥补 APRS 网络中的差距。

休息后，请查看视频，了解改装后的 WRT54g 的概要。如果你对业余无线电感兴趣，两周后[将会是比赛日](http://www.arrl.org/field-day)。[找一个当地的俱乐部](http://www.arrl.org/field-day-locator)，看看业余无线电能做些什么。
