# 循环时钟一圈又一圈地旋转来报时

> 原文:[https://hack aday . com/2011/09/13/recycled-clock-spins-round-and-round-to-tell-time/](https://hackaday.com/2011/09/13/recycled-clock-spins-round-and-round-to-tell-time/)

![pop_bottle_recycled_clock](../Images/278602f5c2d0ed16a46eae17d035e7db.png "pop_bottle_recycled_clock")

[duckcrazy]最近分享了他制作的一个时钟的细节，该时钟使用回收部件来显示时间。

他从拆除几个精心挑选的汽水瓶和一个旧钟开始了他的项目。瓶子的底部和中部被保存了下来，他证实了它们可以很容易地相互插入。时钟的底座由一张 CD 组成，上面安装了时钟的马达组件。

他制作了两个带有小时和分钟标记的开口纸筒，然后将各自的时钟指针粘在里面。圆筒和时钟指针被重新安装到时钟的马达上，整个东西被封装在汽水瓶内。

这是一种建造时钟的新颖方式，暂时超越塑料瓶和纸，基于他的工作，有很多潜力可以实现一些更酷的设计。我们想象由微型和连续旋转伺服系统驱动的激光蚀刻圆柱体会非常可爱，尽管这只是冰山一角。
