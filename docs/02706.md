# PSP SNES 控制器模型

> 原文:[https://hackaday.com/2008/10/29/psp-snes-controller-mod/](https://hackaday.com/2008/10/29/psp-snes-controller-mod/)

![](../Images/2a7a5eb7d8b8eefbd81e6381d311832e.png "snes_psp_mod")

[foo]提交了这个非常棒的模型来给 PSP 添加一个 SNES 控制器。有人联系他，请求为一个单手使用受限的人改装一款 [PSP](http://www.mahalo.com/PSP_Hacks "PSP Hacks - Mahalo") 。PSP 控制对她来说太难了，但奇迹般地，她能很好地玩 SNES 控制器。mod 的质量非常好。[foo]在背面增加了一个控制器插入的端口。除了背面的端口，PSP 看起来完全正常，功能也很好。插上电源后，电缆和插头也可以作为 PSP 的支架。休息后请观看视频了解更多信息。
