# 月球着陆器重新制造

> 原文：<https://hackaday.com/2009/07/16/lunar-lander-remade/>

[https://www.youtube.com/embed/CnKzeHPgWy8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/CnKzeHPgWy8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

对于那些足够幸运还记得它的人来说,《月球登陆者》是一款非常棒的游戏。虽然它有简单的矢量图形和高度重复的游戏，但它让我们着迷。我们可能在与朋友的竞争中失去了整整几周的时间。好吧，现在我们可以通过游戏的[实体版本来重温这种体验。[Lain]建立了这个奇妙的街机风格的游戏，完全复制月球着陆器的游戏玩法。该项目的风格是梦幻般的，巨大的模拟仪表和表盘提供实时反馈。如果你完成了所有 3 个级别，你甚至会得到一份奖品。通过浏览](http://www.lushprojects.com/lunarlander/)[的博客](http://lushprojects.com/blog/?cat=6)，你可以获得大量的建造细节。也许他应该和建造[阿波罗登陆计算机复制品](http://www.galaxiki.org/web/main/_blog/all/build-your-own-nasa-apollo-landing-computer-no-kidding.shtml)的人一起建造终极模拟器。
