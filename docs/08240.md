# 专业黑客为他的 Commodore 64 增加了 USB 功能

> 原文:[https://hack aday . com/2011/08/03/dedicated-hacker-adds-USB-capabilities-to-his-commodore-64/](https://hackaday.com/2011/08/03/dedicated-hacker-adds-usb-capabilities-to-his-commodore-64/)

![usb_basic_commodore_64](../Images/275fb3856555c4b06c258493362aff8f.png "usb_basic_commodore_64")

说 64 号准将的粉丝是一个专门的群体[是很保守的说法](http://www.luigidifraia.com/c64/usb-64/index.html)。仍然有相当多的人花费各种时间为 C64 构建和编程，以使使用它们变得愉快，并跟上当前的技术。

[Luigi]就是这些人中的一员。他想要一种快速而便宜的方法，在他的 PC 和 C64 之间轻松传输文件。对[Luigi]来说，这意味着 USB 文件传输，这将需要相当多的工作来实现。他从开发自己的基本解释器开始，最终可以扩展到支持 USB。使用他的 BASIC-Plus 解释器，他能够实现 USB 内核，它可以通过 USB 到串行适配器以 1.2 KB/s 的速度传输文件。想要更快的文件传输，他建立了一个 USB 转并行适配器，这使得速度提高了近 8 倍。

所以，如果你一直渴望在你的 C64 上有 USB 功能，不用再找了，[Luigi]正好有你需要的！
