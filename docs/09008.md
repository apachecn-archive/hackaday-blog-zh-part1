**为 Raspberry Pi 原型构建详细信息**

![](../Images/ea79d39e40cb39bf10956b1d7e0b22e0.png "prototype-raspberry-pi")

随着 Raspberry Pi 的发布，开发团队发布了大约五年前的原型细节。该板最初基于 ATmega644，构建在某个 perfboard 上。

**“零能耗装置”挑战**

![](../Images/5cb991d6e9ae3c4091494cd418094d8c.png "links-zero-energy-device-challenge")

我们在这个标题上称之为 BS，但是[寻找不使用一次性能源的设备](http://www.greencheck.nl/index.php?/archives/839-Zero-Energy-Device-Challenge.html)的目标在我们的书中是一件好事。例如，我们能得到更多使用长寿命电容器而不是电池的东西吗？

**命令粘贴公告牌等的粘合剂。**

![](../Images/ef5ee844277b2a901944535c8e048fdb.png "links-command-hook-bulletin-boards")

这似乎是显而易见的，但我们多年来一直使用钉子来安装公告板和白板。问题是，当你把图钉插在一边时，另一边会从钉子上掉下来。[Zhanx]正在使用 3M Command Hooks [的粘合剂将他的粘在墙上](http://tinyfabland.blogspot.com/2011/10/updated-my-white-board-space.html)。

**伺服驱动抓取手**

![](../Images/94ac2c7cc205ad7ac6535f58c83d7b1b.png "links-robot-gripping-hand")

[纳维克]一直在努力研发这个机器人手。有一些细节，但他表明它可以抓住一磅以下的物体，他在测试期间一直在测量安培数。

**手机紧急充电**

![](../Images/e680d22e113948573b3bb0bd2e511f92.png "links-power-outtage-cellphone-charging")

这一次可能不是紧急情况，但是在新英格兰的暴风雪使他的手机断电后，克里斯确实找到了给他手机充电的方法。他将提灯电池连接到 7805 调节器上，然后将其接入 USB 集线器，以连接他的手机。紧要关头还不错！
