# “愚蠢的专家”为即将到来的僵尸末日制造了一把弯刀弹弓

> 原文：<https://hackaday.com/2011/03/30/stupid-expert-builds-a-machete-slingshot-for-the-impending-zombie-apocalypse/>

有时人们建造东西只是为了简单的挑战。这是其中的一个案例。

上图中的这位先生是弹弓频道的 rg Sprave。他自称是弹弓方面的“超级专家”,并将自己对以荒谬速度投掷物体的热爱提升到了一个全新的水平。

他的最新作品是弯刀弹弓，确切地说是弯刀弩。测量超过六英尺长，令人印象深刻的装置使用厚橡胶带发射一把特别改造的弯刀，具有疯狂的力量。在下面的视频显示的试射中，弯刀被嵌入到六层非常厚的纸板中，需要相当多的工作才能移除。

正如他在视频中挥舞着沾满鲜血的前臂所说，建造这样的设备是极其危险的，应该仅限于“愚蠢的专家”。无论如何，你都不应该试图建立一个你自己的。
