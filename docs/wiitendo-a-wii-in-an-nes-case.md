# Wiitendo:装在 NES 盒子里的 Wii 游戏机

> 原文：<https://hackaday.com/2008/08/11/wiitendo-a-wii-in-an-nes-case/>

[https://www.youtube.com/embed/jTs8c4r8m94?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/jTs8c4r8m94?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
【马特】已经设法将一台 [Wii 塞进一台旧 NES](http://www.nintendo-hacks.com/Wii-Mods) 。这篇文章很短，但确实提到了一些细节。他安装了一些游戏方块控制器端口和一个游戏方块存储卡。他增加了一些连接到 Wii 电源按钮的外部发光二极管。他为此付出了很多努力，因为他说内部外壳的几乎每个区域都需要一些认真的 dremel 操作，才能让 Wii 适合内部。他甚至承诺会写一份更详细的报告，所以请密切关注。

你可能还记得《黑客日》前情提要中的 Wii 笔记本电脑。
