# 没有根的三星生机勃勃

> 原文:[https://hack aday . com/2010/07/30/tethering-the-Samsung-vibrant-without-root/](https://hackaday.com/2010/07/30/tethering-the-samsung-vibrant-without-rooting/)

![](../Images/c6b60b84c16dd897750005eaaf862fa4.png "samsung-vibrant")

如果你有一部三星 Vibrant，并想利用这个无限制的 3G 账户，你可以 [连接而不用启动手机](http://samsungvibranthacks.com/samsung-vibrant-hacks/how-to-usb-tether-your-samsung-vibrant/) 。此方法使用 USB 电缆为 Windows XP 和 Windows 7 计算机提供互联网访问。三星自己的 Kies 软件处理网络共享，只要你有一个神奇的号码就可以连接到 T-Mobile 美国网络；“epc.tmobile.com”作为 APN 姓名，而“*99#”作为电话号码。[Zedomax]在休息后制作了视频，带你体验拴绳仪式。
