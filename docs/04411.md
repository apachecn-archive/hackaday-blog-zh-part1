# 什么东西？Thingamagoop2！

> 原文:[https://hackaday.com/2009/11/12/thingamawha-thingamagoop2/](https://hackaday.com/2009/11/12/thingamawha-thingamagoop2/)

![ard](../Images/d8da3379f9f636474b69d1b5675d2f45.png "ard")

介绍一下 [Thingamagoop2](http://bleeplabs.com/thingamagoop2/) ！还记得那个[外观笨拙的盒子](http://hackaday.com/2008/07/03/thingamakit-make-your-own-thingamagoop/)吗？它有两个 LEDacles 伸出来，还有一堆奇怪的按钮和旋钮，通过一些[电路弯曲](http://hackaday.com/2008/08/19/how-to-hack-a-thingamakit/)合成不同的声音，让你的邻居连续几个小时睡不着觉？现在它回来了！更小，少了一个 LEDacle，但挤满了更多的功能-就像我们个人最喜欢的:通过 Arduino 编程的能力。休息之后，请观看“音乐”的视频。
