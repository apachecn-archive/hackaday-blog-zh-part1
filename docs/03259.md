# BMOW:国产 Cpu

> 原文:[https://hackaday.com/2009/02/27/bmow-a-home-made-cpu/](https://hackaday.com/2009/02/27/bmow-a-home-made-cpu/)

![bmow](../Images/fb98f32ab2d2aceda4a93aa8777dcf0a.png "bmow")

构建自己的 CPU 听起来是一项艰巨的任务。[使用手工绕线制作自己的 CPU](http://www.stevechamberlin.com/cpu/)超越了成为一种艺术形式的困难。[史蒂夫]通过[手动缠绕](http://en.wikipedia.org/wiki/Wirewrap)每一根电线制造了一个 CPU。那是 1253 根电线，或者 2506 个缠绕的末端。就算不行，看着也挺好的。但它确实工作，你可以看到一个演示视频显示休息后的音频功能。该系统现在被封装在宏基 x-terminal 机箱中，因此它不那么漂亮，但仍然是一个相当不错的项目。你可以跟随他构建每个部分，视频，声音，甚至键盘界面。看到它被分解成最基本的形式真是太神奇了。
