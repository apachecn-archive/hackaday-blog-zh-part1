# 用 CPU 冷却器和啤酒冰箱给自己降温

> 原文:[https://hacka day . com/2010/05/25/用 cpu 冷却器和啤酒冰箱给自己降温/](https://hackaday.com/2010/05/25/cool-yourself-with-a-cpu-cooler-and-beer-fridge/)

我们又一次来到了一年中身无分文(或无聊)的黑客试图找出如何在不购买空调的情况下保持这个地方凉爽的时候。[Paul Stacey]向我们发送了他的解决方案，将一个 [CPU 冷却器套件与一个啤酒冰箱](http://www.youtube.com/watch?v=UTQ9r_rGNkg)配对。CPU 散热器从液体冷却套件中取出并丢弃。取而代之的是一圈塑料管进入啤酒冰箱的冷冻室，在那里交换来自蓄水池的盐水。冷液体通过风扇套件的散热器循环，并通过上面看到的风扇单元释放其冷量。这种方法将一个冷空气风扇放在你的正前方，LCD 上显示数字温度和风扇速度读数。

我们最担心的是这会加热冰箱里的啤酒。尽管如此，这比使用自制的沼泽冷却器更加自动化。话又说回来，我们心中一直对[情有独钟，这是我们最喜欢的重力通风冷却方式](http://www.gmilburn.ca/2005/06/15/geoffs-original-homemade-air-conditioner/)。无论如何，休息之后看看[Paul 的]构建方法，他让它看起来又快又简单。
