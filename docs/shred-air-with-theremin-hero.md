# 用特雷门英雄撕碎空气

> 原文：<https://hackaday.com/2010/07/13/shred-air-with-theremin-hero/>

[https://www.youtube.com/embed/sBhOB7CfUoY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/sBhOB7CfUoY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

还记得那些易趣拍卖的几千美元的空气吉他吗？我们也不知道，但是《特雷门英雄》(更多信息见 [YouTube 描述](http://www.youtube.com/watch?v=sBhOB7CfUoY))是你能在空气中实际摇摆的最合法的游戏。

就像使用[特雷门琴控制马里奥](http://hackaday.com/2009/06/15/theremin-controlled-mario/)一样，垂直天线充当品板，而水平天线检测弹奏。将特雷门琴的输出与一些定制软件(尚未发布)和吉他英雄结合起来，你就有了特雷门琴英雄空气吉他。
