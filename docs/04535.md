# 双向屏幕、屏幕上(和屏幕外)多点触摸

> 原文：<https://hackaday.com/2009/12/12/bidi-screen-on-and-off-screen-multitouch/>

[https://www.youtube.com/embed/kXuxK6IeQfo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kXuxK6IeQfo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

麻省理工学院首次展示了他们最新的技术进步，这是一个多点触摸屏，也可以作为手势界面。多点触摸方面并不新鲜，该团队解释了使用[led](http://hackaday.com/2009/10/10/multi-touch-lcd-from-leds/)或[相机系统](http://hackaday.com/2008/05/20/multitouch-project-roundup/)的传统界面如何工作，但无法识别屏幕外的手势。

手势是相对较新的亮点，引入了像[纳塔尔](http://www.xbox.com/en-US/live/projectnatal/)或[视角跟踪](http://hackaday.com/2009/12/11/19177/)这样的项目，但在离屏幕更近的距离上无法工作。麻省理工学院已经完成了看似不可能的事情，通过结合和修改这两者，制造出了有史以来第一个多点触摸近距离手势显示器。

想想看，就在几个月前，同一所学校还在玩立体书。
