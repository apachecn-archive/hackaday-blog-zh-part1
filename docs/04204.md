# 玛吉，锁定和加载

> 原文：<https://hackaday.com/2009/10/08/maggy-locked-and-loaded/>

![maggy](img/0f0ee6303f60dd32ad55333e5506e63a.png "maggy")

一些读者可能记得[Paul]的项目[Jak，21 点机器人](http://hackaday.com/2009/04/09/jak-the-blackjack-robot/)；但是他的兴趣已经转向了线圈枪的发明。[玛吉](http://sites.google.com/site/paulscoilguns/Home/maggy)，他的最新武器，可能不是我们见过的[最漂亮的](http://hackaday.com/2009/05/07/portable-coil-pistol/)线圈枪，也不是[最环保的](http://hackaday.com/2009/03/20/making-a-rail-gun/)，但看起来确实很有前途。它采用三级逻辑加速器，而不是典型的单级加速器，可以发射 10 克的弹丸(理论上)达到 85 公里/小时！在他的网站上看看他是如何从一把手枪变成现在的怪物的，以及他跳跃后正在进行的工作的视频。
