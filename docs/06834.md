# 流浪的 Arduinobot

> 原文:[https://hackaday.com/2011/02/16/the-wandering-arduinobot/](https://hackaday.com/2011/02/16/the-wandering-arduinobot/)

![](../Images/58ff4f2d3f05a219489eaf8c829a5550.png "wanderinger-arduinobot")

仅仅因为轮子和第三条腿的支撑，你几乎不得不爱上这个基于 Arduino 的机器人。仔细看，你认得他们吗？是的，它们是卡普塞拉的零件！这是一个来自过去的爆炸，我们想知道为什么我们没有看到过去的奇怪的建筑玩具在更多的黑客中使用？

但是我们跑题了，这个小家伙使用连续旋转伺服系统来移动。栖息在身体顶部的是第三个伺服系统，它来回扫描红外测距仪，寻找其路径上的障碍。身体本身是一个黑色和 Decker 可充电电池组，称为口袋电源，其中包括一个 USB 端口，用于调节 5V 电源。有一件事是肯定的，这个小家伙在避免比萨饼和啤酒方面是神奇的…你将不得不在休息之后亲自看到。
