# 垃圾箱电脑让你用 Neato XV-11 清洁和原型

> 原文：<https://hackaday.com/2011/08/30/dustbin-computer-lets-you-clean-and-prototype-with-a-neato-xv-11/>

所以你给自己买了一个 Neato XV-11，你的地板从来没有这么干净过。唯一的问题是，你想在不失去你的扫地小弟的情况下使用硬件。[Hash]通过在垃圾箱模块内部构建一台计算机[找到了问题的解决方案。](http://random-workshop.blogspot.com/2011/08/neato-xv-11-dustbin-computer.html)

你可以在触摸屏上方的图像中心看到。通常这只是空白塑料，因为它是你扫地的可移动容器，但[Hash]的灵感来自模块化设计。由于该绑定是可移动的，所以这是使附加硬件可移动的完美方式。他所需要做的就是找到一种方法来连接 Neato 自己的电子设备。解决方案是非标准的 USB 电缆。

他用 Insignia Infocast 3.5 的内脏(他在圣诞节清仓时捡了几个)为触摸屏磨了一个开口，添加了一个冷却风扇，并连接了一个拨动开关(上图未显示)，该开关可以为 USB 电缆输入的 14-17 伏电压供电。Infocast 是一个品牌不同的 Chumby，因此有大量基于 Linux 的功能，并且支持 WiFi。休息后观看[Hash's]的视频，了解这一巧妙概念的所有内容。

我们还没有看到太多利用 Neato XV-11 的黑客。[Hash]就是那个黑了单位上的激光雷达的家伙，但肯定还有其他人在做令人印象深刻的项目。如果你知道的话，不要犹豫[发送一个提示](http://hackaday.com/contact-hack-a-day/)。
