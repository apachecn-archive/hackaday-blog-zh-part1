# 了解 DMX，展望圣诞装饰

> 原文:[https://hack aday . com/2011/11/01/looking-on-Christmas-decor-by-learning-about-DMX/](https://hackaday.com/2011/11/01/looking-toward-christmas-decor-by-learning-about-dmx/)

![](../Images/4c08ee62b6d329b0bad593a9c3098850.png "dmx-tutorial-series")

所以万圣节终于来了，我们希望你有足够的时间来完成你最初设想的可怕壮举。现在又是时候展望未来，开始计划今年的圣诞装饰了。灯光总是一个受欢迎的主题，今年你可能想看看 DMX 的灯光控制，并决定这是否是你想走的路线。[Akiba]支持你，他刚刚整理了一组视频[解释 DMX 照明协议以及如何使用 Arduino](http://freaklabs.org/index.php/Tutorials/Software/Light-Sequencing-and-Decoding-DMX-with-an-Arudino.html) 。

这里的想法是，Arduino 可以用作一种 DMX 中枢，直接连接到运行开源控制器硬件的计算机。它可以发送 Arduino 解码的命令，决定是否只是将它们传递给 DMX 兼容设备，或者做它最擅长的事情，并控制其他通常无法通过照明命令协议访问的硬件。对控制程序来说，4 美元一束的 LED 灯看起来和 1000 美元的舞台灯没有什么不同，这使得在你的前院建造你自己的娱乐节日表演变得便宜和容易。休息之后，请观看本系列的第二个视频，其中[Akibo]详细介绍了他的系统的硬件设置。其他部分可通过上面的链接获得。

小心，这可能是一个耗费精力的努力。不相信我们？环顾四周，你会发现 DMX 并不缺少专为圣诞灯建造的大型。
