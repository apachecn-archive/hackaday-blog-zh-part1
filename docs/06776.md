# 四轴摄像头安装在电线上

> 原文：<https://hackaday.com/2011/02/09/four-axis-camera-mount-rides-on-a-wire/>

![](img/3cdd82438b79973a5656c96c9aab7c0c.png "four-axis-camera-gimball")

这是[bad wolf]团队与他们的大学项目合影。他们用 CAD 设计了一个相机的四轴万向支架，用激光切割零件，然后把它们磨成规格。在上面的图片中，有一个小型的傻瓜相机安装在悬挂的环内，但这个装置足够强大，可以支持任何尺寸的相机。

安装环可以像时钟的指针一样旋转，但它也可以绕水平轴旋转。支撑环的支架可以在垂直轴上旋转，整个组件沿着支撑它的金属丝移动。休息后，你可以看到一些测试镜头，显示钻机通过手持无线电控制器操作。

这种设置可以让摄像机尽可能地远离电缆。但是如果你想要的东西能让你拍摄非常高的物体，你需要使用不同的设置。
