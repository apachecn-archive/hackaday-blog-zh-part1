# 机器鸟拍走了最后一点隐私

> 原文：<https://hackaday.com/2011/03/30/robotic-bird-flaps-away-last-bits-of-privacy/>

这确实给天空中的眼睛一个逼真的外观。如果你担心你生活的每一部分都没有被监控摄像头记录下来，Festo 仿生学习网络已经推出了一款无人机，这款无人机将很难与野生动物区分开来。

休息后看视频。我们不能 100%确定它不是假的，但它看起来足够真实(真正令人惊叹的设计的标志)。你会看到机器鸟从地面上的一个固定点和设备头部后面的摄像机视角振翅而去。它通过拍打和旋转翅膀来推动自己，能够自主起飞、飞行和着陆。

它比为 DARPA 开发的蜂鸟无人机更大，但我们认为它看一眼就不会那么显眼。没有关于该设备的预期用途的消息，但我们确信你们中的一些人正在享受《诸神之战》中机械猫头鹰的怀旧之情，这就是我们想要一个的原因。
