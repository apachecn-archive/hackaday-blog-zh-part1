# C64 可视化调试器

> 原文：<https://hackaday.com/2009/08/29/c64-visual-debugger/>

![c64ICU](img/2462041c057a76ebfd864ef83831cea8.png "c64ICU")

Root Labs 写了一篇关于 ICU64 的文章，这是一个[的 Commodore 64](http://en.wikipedia.org/wiki/Commodore_64) 模拟器，具有一些不寻常的特性。其中最特别的是能够显示系统的整个工作 RAM。每个 RAM 地址在被访问时都会亮起。如果需要，用户还可以放大或更改每个地址的值。这听起来很复杂，但是演示视频展示了这些能力的威力。这也可以作为低级代码的[内存管理](http://en.wikipedia.org/wiki/Memory_management)的很好的入门。可惜的是《ICU64》的作者[[math figure](http://www.youtube.com/user/mathfigure)]还没有向公众发布这个，不过应该很快就会发布了。

[ICU64 已经发布！](http://icu64.blogspot.com/)

[感谢 mathfigure 跟进此事]
