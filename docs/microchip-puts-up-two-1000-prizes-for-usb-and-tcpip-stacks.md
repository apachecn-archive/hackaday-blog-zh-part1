# Microchip 为 USB 和 TCP/IP 协议栈设立了两个 1000 美元的奖项！

> 原文：<https://hackaday.com/2011/09/15/microchip-puts-up-two-1000-prizes-for-usb-and-tcpip-stacks/>

![microchip_call_for_open_source](img/c9844a252959efb3d7cd7fec74054d02.png "microchip_call_for_open_source")

几周前，我们发表了一篇名为[解决 Microchip 的开源问题](http://hackaday.com/2011/08/31/addressing-microchips-open-source-problem/)的文章，其中我们谈到了他们在开源代码方面的一些缺点，特别是 TCP/IP 协议栈和 USB 协议栈。不出所料，这些评论相当负面。有趣的是，微芯片实际上是在倾听。如果你通读了所有的评论，你会对微芯片的内部情况有所了解。在最后，Microchip 的[Marc]留下了一些评论，概述了 TCP/IP 和 USB 独立移植堆栈的两个奖项。由于法律原因，Microchip 不能完全开放他们拥有的，所以他们需要开发社区的帮助，他们为每个人提供 1000 美元，以证明他们是认真的。如果你点击[这个链接](http://www.microchip.com/stacks/)，你将会看到一个概述比赛规则的页面。

战书已经放下了！你有能力完成这件事并为自己赢得整整 1000 美元吗？
