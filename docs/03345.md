# 自动化鼓

> 原文：<https://hackaday.com/2009/03/29/automated-drum/>

![snaredrum](img/db515033958a6346b09cdae13669f6f7.png "snaredrum")

一个由三名博士生组成的团队建造了这个“[多槌自动击鼓乐器(马迪人)](http://www.wired.com/gadgets/mods/multimedia/2009/03/gallery_instruments?slide=15&slideView=7 "New Musical Instruments Battle for $10K in Prizes")”。他们的[表达机器乐器](http://www.expressivemachines.org/ "Expressive Machines Musical Instruments")网站致力于制造这样的乐器，他们最近在第一届年度[古特曼乐器比赛](http://www.wired.com/gadgets/mods/multimedia/2009/03/gallery_instruments "New Musical Instruments Battle for $10K in Prizes")上展示了他们的作品。乐器的“低赌注 X 奖”。25 名申请人被选中展示他们独特的乐器，奖品为$10K。我们喜欢车队的马迪人，因为它改编了一种传统乐器，然后将它推向了极限。这让我们想起了 2005 年的疯狂 J 机电吉他。你可以在下面看到马迪人队比赛的视频。
