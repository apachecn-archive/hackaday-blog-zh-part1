# 用锤子和钉子建造电路

> 原文:[https://hack aday . com/2011/04/06/用锤子和钉子建造电路/](https://hackaday.com/2011/04/06/circuit-building-with-a-hammer-and-nails/)

![real_breadboarding](../Images/e2499a42c915c8a657ed8d6ec13204ab.png "real_breadboarding")

Make 的[Collin Cunningham]最近完成了另一个版本的“Collin 的实验室”——这一次，主题是试验板。他首先讨论了一种常见的无焊试验板，毫无疑问，您对此很熟悉。然而，你可能不知道的是，试验板是如何得名的。

很久以前，在全国的每一个购物中心都有 RadioShack 之前，像无焊试验板这样的花哨的原型制作用品是不存在的。业余无线电操作员会在木板上制作电路原型，通常使用周围的任何东西作为基板。很多时候，这意味着家庭的砧板最终成为一个临时的原型工作站。

一种流行的构建电路的方法是在试验板上钉上小钉子，用绕线把东西连接在一起。[Collin]在视频中演示了这一技术，构建了一个简单的 LED 闪光灯电路。

他说，这个过程工作得相当不错，而且做起来很有趣。然而，他确实提到，构建任何需要集成电路的电路都是不可能的。
