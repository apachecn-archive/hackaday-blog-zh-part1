# 2008 年机器人世界杯的微型机器人

> 原文：<https://hackaday.com/2009/04/14/microbot-for-the-robocup-2008/>

[https://www.youtube.com/embed/dMJJ2FJqsdI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/dMJJ2FJqsdI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[tioguerra]帮助为 2008 年在中国举行的世界机器人世界杯制作了这些微型机器人。在[混合现实论坛](http://odin.pcag.fh-wolfenbuettel.de:8080/mixedreality/)上有大量的信息，但是你可能会对[机器人部分](http://odin.pcag.fh-wolfenbuettel.de:8080/mixedreality/index.php/board,5.0.html)特别感兴趣。有不同方面的故障，如充电和红外通信，以及一些伟大的单位连同一个较小的照片。机器人的构造看起来非常完美，有人花了很长时间才把设计做好。由于我们没有看到 2009 年机器人的新部分，我们猜测他们会再次使用这种设计。
