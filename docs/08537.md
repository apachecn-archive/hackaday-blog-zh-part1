# 这个盒子里的侧滚游戏不需要 25 美分硬币

> 原文:[https://hackaday . com/2011/09/09/no-quarters-required-for-this-side scrolling-a-box-game/](https://hackaday.com/2011/09/09/no-quarters-required-for-this-sidescrolling-game-in-a-box/)

![teagduino_game_in_a_box](../Images/d577451ea162a8249c6dda3739e08e45.png "teagduino_game_in_a_box")

来自蒂格实验室的[Adam]来信分享了他们制作的一个新工具，以帮助展示蒂格杜伊诺的功能。他们放在盒子里的桌面视频游戏是由他们周围的一堆电子元件、苏打水吸管、大量画家胶带和一些冰棒制成的。

当有人拉动盒子前面的绳子时，一个伺服机构自动打开它，第二个伺服机构开始旋转游戏卷轴。随着卷轴的移动，玩家会看到一组要躲避的障碍物，通过旋钮控制的伺服系统来引导“英雄”。当通过任何障碍物时，连接在角色背部的霍尔传感器被触发，障碍物用磁带连接在卷轴上。当英雄撞上障碍物时，游戏结束并自动关闭，这让玩家非常懊恼。

正如你在下面的视频中看到的，这是一个非常有趣和具有挑战性的游戏。
