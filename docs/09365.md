# 动画 LED 布基球

> 原文:[https://hackaday.com/2011/12/23/animated-led-buckyball/](https://hackaday.com/2011/12/23/animated-led-buckyball/)

![](../Images/c5d1b567de5a28e212be4883ed91ac7d.png "led-buckyball")

这个[动画 LED 巴基球](http://tim.cexx.org/?p=843)与模拟碳分子关系不大，但却能为你的派对装饰增添一些刺激。[Tim]认为 LED 立方体黑客已经完成了他们的使命，所以接受了由五边形和六边形组成的球体的挑战。

与许多 LED 项目一样，找到一个好的扩散器至关重要。[Tim]决定使用热胶棒，它可以很好地漫射和传输 RGB LEDs 发出的光。这种选择的意外后果是形状在 90 个胶筒的重量下下垂。

在每个胶水段的末端，你会发现一个微小的表面贴装 RGB LED 和控制器组合。这是[Tim]自己的设计，它将每个节点的成本降低到大约一美元。在一个足球作为装配夹具的帮助下，他完成了构建，并编写了一些代码来制作休息后看到的赏心悦目的动画。
