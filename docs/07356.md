# Retrode 获得升级

> 原文:[https://hackaday.com/2011/04/11/retrode-gets-an-upgrade/](https://hackaday.com/2011/04/11/retrode-gets-an-upgrade/)

![](../Images/44e26eb3dce2ae01feee87c8a3e9141c.png "golden-carts-thumb")

自从 YouTube 上的一个[模糊视频](http://hackaday.com/2009/06/19/usb-reader-for-snes-game-carts/)我们发誓这是一个精心策划的骗局后，我们就一直在跟踪这个[回溯](http://www.retrode.org/)。现在，[马蒂亚斯]告诉我们，它正在进行第三次重大升级，它真的开始像一个商业项目。该视频展示了 Retrode II 的新原型案例，该案例已由 [3d 打印](http://www.thingiverse.com/)。普通黑客也能获得如此先进的防护设备，这实在令人难以置信。新的 Retrode II 将内置端口，因此 SEGA 和 SNES 控制器可以插入。自推出以来，该社区一直在合作构建[插件板](http://www.retrode.org/plug-in-adapters/)，允许人们直接在他们的计算机上玩虚拟男孩、雅达利 2600、GBx、Turbografix-16、 [Neo Geo Pocket、](http://ponrevival.blogspot.com/2011/01/retrodeplug-in-neogeopocket-adapter.html)甚至 N-64 卡带。非常酷。
