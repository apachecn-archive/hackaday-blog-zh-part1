# 太阳能热水器

> 原文：<https://hackaday.com/2009/12/14/solar-water-heater/>

![](img/f51518818b19b9447c79876e87203d1e.png "diy-solar-water-heater")

[杰克]为自己建造了一个太阳能热水器。该系统有一个 82 加仑的隔热水箱，并使用聚氯乙烯管制成的太阳能收集网格。随着水的加热，它上升到收集器的顶部，流入水箱，同时将较冷的水从水箱底部拉回收集器。

一个阳光明媚的日子气温约为 130 华氏度。在阴天，水仍然可以加热到大约华氏 90 度。这可能比你期望的要低 19 度，但这一差异是由一个出水时会发热的电动淋浴头弥补的。[杰克的]系统，在休息后的视频中可以看到，允许他在晴天放弃用电，在阴天减少一半的用电。 <https://www.youtube.com/embed/KOY49bH7Ocg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
