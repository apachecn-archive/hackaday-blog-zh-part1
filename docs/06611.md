# 用废料制作你自己的里程表

> 原文:[https://hack aday . com/2011/01/19/用废品制作自己的里程表/](https://hackaday.com/2011/01/19/make-your-own-odometer-from-scraps/)

![](../Images/3c55ab601c3e386788846462e6e423bf.png "Operation")

对于那些喜欢快速和有趣的周末项目的人来说，这个由[PeckLauros]制造的里程表是为你准备的。它由最简单的材料制成，包括一些纸板、一个计算器、电线、胶水、热熔胶、磁性驱动键、一张旧 CD 和一个阅读器，以及一根橡皮筋。当磁铁连接到 CD 上时，在计算中，当磁铁闭合电路时，计算器增加 0.11 米。[PeckLauros]指出，由于它是一个自制的设备，因此它确实存在一些缺陷，例如当光盘旋转速度太慢时，它会两次添加 0.11m。只要跑得更快，就很容易解决这个问题。视频在休息下面。
