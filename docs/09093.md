# 我是 Root！— Kindle Fire 版

> 原文:[https://hackaday . com/2011/11/16/I-am-root-kindle-fire-edition/](https://hackaday.com/2011/11/16/i-am-root-kindle-fire-edition/)

![](../Images/cd6a6a0d02b896d9104300c290c86dd0.png "kindle-fire-root")

亚马逊的新平板阅读器 Kindle Fire 已经扎根。今天早上[Death2All110]贴出了他获得 root 权限访问他的设备的步骤(这款设备刚开箱，闻起来还是新的)。繁重的工作由[一个名为 SuperOneClick](http://forum.xda-developers.com/showthread.php?t=803682) 的包来完成，该包旨在为所有运行 Android 的手机和设备提供基础。

这比必要的一次点击多一点，但不会太多。使用 Android Developer Bridge 结合 SDK，您需要输入一个将被识别为 VID 的值。在那里，打开安装来自未知开发者的应用程序的功能，在你的 PC 上重新枚举设备并运行一键软件包。

你能用这个做什么？嗯，它完全开放了 Android 操作系统，所以你可以随心所欲地使用它。我们还没有看到任何演示，但它应该比我们看到的索尼 PRS-T1 更好。
