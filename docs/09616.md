# Kinect Windows 版发布

> 原文:[https://hack aday . com/2012/02/02/Kinect-for-windows-released/](https://hackaday.com/2012/02/02/kinect-for-windows-released/)

![](../Images/4a0fcecbf8a873deea9d6d209066994f.png "Kinect")

尽管这些年来我们已经看到了数十起 Kinect 黑客事件，但 Kinect 硬件本身还是存在一些问题。Kinect 传感器的范围*始于*三英尺，这不利于 3D 扫描仪的构建。此外，不可能将一台以上的 Kinect 连接到一台电脑上——这将导致我们现在几乎无法想象的构建。

[不要害怕](http://blogs.msdn.com/b/kinectforwindows/archive/2012/01/09/kinect-for-windows-commercial-program-announced.aspx)，因为微软刚刚发布了 Windows 版的 Kinect。基本上，它是专门为黑客设计的。Windows 的 Kinect 可以可靠地“看到”40 厘米(16 英寸)以外的物体，并且支持连接到同一台计算机的多达四个 Kinect。

微软将 Windows 版 Kinect 的价格定为 250 美元。这对我们来说是一个交易破坏者——一个新的 XBox Kinect 售价大约是它的一半。如果你能让微软相信你是一名学生，Windows 版 Kinect 的价格就会降到 150 美元。如果你把价格和新的 XBox Kinect 相比，这还不算太差。

我们预计大多数开发者已经从他们当地的 Craigslist 或 Gamestop 上买了一两台 Kinect。如果你还没有(而且有最重要的教育折扣)，这可能是一个值得购买的。
