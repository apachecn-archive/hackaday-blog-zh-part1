# “Effervo”Kinect 粒子效果机

> 原文:[https://hack aday . com/2011/10/17/the-eff voo-Kinect-particle-effect-machine/](https://hackaday.com/2011/10/17/the-effervo-kinect-particle-effect-machine/)

这是 Xbox Kinect 的一个名为“Effervo”的新技术。这是一个使用 Openframeworks 构建的非常酷的效果。Kinect 设置在用户面前，投影仪将图像放在用户屏幕前。微软的传感器可以捕捉到这个人及其动作的三维数据。正如它所描述的，Effervo 程序“使用简单的迭代规则来控制它的运动，并给人以群体行为的印象。”这可能不是一个“Haloween Hack”，但我们肯定可以看到类似的东西用在一个闹鬼的房子。也许它可以用血滴代替微粒？

也许这个黑客会激励其他人追随杰森的脚步。他把自己描述为“程序员变成艺术家”我们愿意认为所有的工程和编程工作都是一种艺术形式，但休息后这段视频尤其引人注目。
