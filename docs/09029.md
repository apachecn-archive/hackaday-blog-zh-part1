# 平移延时钻机

> 原文:[https://hackaday.com/2011/11/08/panning-time-lapse-rig/](https://hackaday.com/2011/11/08/panning-time-lapse-rig/)

![](../Images/8061443e02228efc54aea1b8ffc9044b.png "panning-time-lapse-rig")

这里有一个简单的相机设置，可以让你[制作自己的平移延时视频](http://mesalioasi.blogspot.com/2011/11/timelapser-pan-time-lapse-con-lumix-lx5.html)。它使用一对由 Arduino 驱动的电机来拍摄连续的静态图像，这些图像随后可以转换成视频格式。

[Acorv]对他的新 Lumix LX5 没有内置定时选项感到不高兴。但幸运的是，它的顶部有一个标准的外部闪光灯连接器。他在一个论坛帖子上看到有人制作了一个夹具，安装在闪光灯支架上，使用伺服电机按下快门按钮。他重新制作了这个，并完成了一半。

摇摄部分是由大猩猩促成的。这种特殊型号具有旋转功能。这是通过用一根绳子将它连接到步进电机上来实现的。当步进器转动时，线就缠绕在卷轴上，逐渐转动相机。简单，而且看起来效果很好。休息后，请查看视频，看看日落时在湖边拍摄的测试。

如果你有一台提供红外遥控快门释放功能的相机[，那么延时部分可以用红外间隔计](http://hackaday.com/2009/10/30/time-lapse-courtesy-of-arduino/)来处理，使机械结构变得更简单。
