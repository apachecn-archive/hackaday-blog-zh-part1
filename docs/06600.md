# 3D 立体显示的一瞥

> 原文：<https://hackaday.com/2011/01/18/glimpses-of-a-3d-volumetric-display/>

![](img/5089c3df1baa3db35033a2212ae6d358.png "3d-volumetric-display")

定制展示看起来很有趣，但这是我们希望在贸易展上看到的东西，而不是在别人的餐桌上。[Taha Bintahir] [构建了一个 3D 立体显示器](http://mtaha.wordpress.com/2011/01/04/3display/)，并在上面的图像中展示它，使用的是从 Autodesk 导出的超人徽标的 3DS 文件。在休息后的视频中，您可以看到显示器是一个透明的金字塔，允许观众从显示器周围的任何视点看到内部的 3D 对象。自从第一次发布以来，他[还添加了一个 Kinect 到组合中](http://mtaha.wordpress.com/2011/01/14/kinect-3d-volumetric-display/)，允许用户通过身体运动来控制 3D 物体。

[塔哈的]帖子上基本上没有关于显示器硬件的信息，所以我们向他询问了这个问题。它的工作原理是首先获取一个 3D 模型，然后从四个不同的摄像机角度进行渲染。他使用了一个定制设计的棱镜来显示，最初的渲染被扭曲以匹配棱镜的尺寸。这些渲染被投射到棱镜上，以产生 3D 物体漂浮在其中心的幻觉。

我们希望听到更多关于这是如何设计的细节，以及使用了什么硬件。如果[塔哈]分享更多信息，我们将发布后续消息。
