# 通过简单的调整赋予旧电器第二次生命

> 原文:[https://hackaday . com/2012/02/25/通过简单的调整让旧电器重获新生/](https://hackaday.com/2012/02/25/giving-old-appliances-a-second-life-with-simple-tweaks/)

![blender-light](../Images/2868bec4a94405d732fe78cefd845ee7.png "blender-light")

巡游房产销售可能完全是一场赌博——有时你会发现一座金矿，有时除了垃圾什么也没有。[约翰·奥恩比]最近在这样的拍卖中发现了一个看起来很光滑的旧搅拌机，并决定把它带回家。镀铬底座和凹槽玻璃立即引起了他的注意，但他没有买搅拌器，所以他可以制作普通的冷冻饮料—[他想要一盏灯。](http://www.instructables.com/id/Blender-Lamp-1/?ALLSTEPS)

转换相当简单，只需要他把机器的运动部件，包括电机和刀片，都取出来，换上一个小的白炽灯烛台底座。虽然他的修改本身并不具有突破性，但更进一步，将会创造出一些非常酷(和实用)的复古家居设备。

如果你愿意的话，请允许我花一点时间，想象一下用一些 LED 灯条甚至 EL 线代替简单的白炽灯泡。用一个小扬声器代替搅拌机的盖子，你可以把几个这样的扬声器放在一起作为复古的环绕卫星。

我们肯定可以支持他重复使用搅拌机，否则很可能会被扔进垃圾填埋场。很高兴看到坚固耐用的电器获得了第二次生命，即使是以意想不到的方式。你有没有像[John]一样从垃圾堆里抢救出什么东西，或者你对可能遇到类似商品的黑客同伴有其他想法？请在评论中告诉我们。
