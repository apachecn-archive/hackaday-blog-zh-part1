# 农场机器人

> 原文：<https://hackaday.com/2011/02/21/farmbot/>

![](img/20341a723823c664492cd6fe946a88b8.png "planter_front_c")

普洛斯彼罗是一个自动微型种植机的工作原型，它将作为一个群体被释放。使用安装在 [Lynxmotion AH3-R](http://www.lynxmotion.com/c-92-ah3-r.aspx) 六足机器人身上的视差推进器，尽管如果你发现身体的价格超出预算，我们对六足机器人有很多不同的[想法。](http://hackaday.com/page/1/?s=hexapod)

机器的输入是相当标准的机器人食物，如红外线和超声波。另一方面，输出更加有趣，例如喷漆、可伸缩钻、种子分配器，当然还有一堆伺服系统。

用钻头和一罐喷漆武装一个机器人听起来像是一个享受几个小时乐趣的食谱，但它确实有工作要做。四处走动，机器人停下来检查它下面的土地，如果土地可以播种，一个小螺旋钻就会从机器人的腹部翻下来。钻完一个洞后，撒下一粒种子，然后在钻头后面用铲子盖上。最后，播种点用白色油漆标记，机器人继续前进。

虽然它是一个原型，还没有完全成型，但它是一个有趣的东西，所以休息后加入我们的视频。
