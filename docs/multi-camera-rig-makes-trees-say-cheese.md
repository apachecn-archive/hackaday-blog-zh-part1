# 多相机装备让树说奶酪

> 原文：<https://hackaday.com/2009/09/30/multi-camera-rig-makes-trees-say-cheese/>

![redwood_camera_rig](img/52650dba58671d675e842e689da54a7c.png "redwood_camera_rig")

[Nick Nichols]着手拍摄一棵完整的红杉树。这里有一个问题，这些红杉有 300 多英尺高，它们不仅仅矗立在一片广阔的沙漠中。如果照片是从远处拍摄的，你只会捕捉到这些雄伟美景的顶部。你怎么能近距离拍摄呢？建立一个定制的装备来拍摄多张照片，并将它们拼接在一起，形成一张合成照片。

[Nichols]搭建了一个平台，可以放置三个摄像头，分别聚焦在树的左边、中间和右边。该框架包括一个陀螺仪，以保持相机稳定。通过将相机从红杉树的顶部降低到底部，他们能够捕捉 84 张照片来组合最终的镜头。这一结果被刊登在国家地理杂志的[十月号上。我们已经包括了最终的图片，并在休息后嵌入了钻机的视频。](http://ngm.nationalgeographic.com/2009/10/redwoods/bourne-text)
