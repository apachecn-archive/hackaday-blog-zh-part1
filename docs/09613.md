# 塑料汽水瓶的镜头适配器

> 原文:[https://hack aday . com/2012/02/01/lens-adapter-from-a-plastic-soda-bottle/](https://hackaday.com/2012/02/01/lens-adapter-from-a-plastic-soda-bottle/)

![](../Images/b51482411fd3ce44b48c4c9fe676bcee.png "soda-bottle-lense-adapter")

如果你正在寻找没有内置安装选项的相机接口，这种镜头适配器非常有意义。它[使用汽水瓶](http://binaryhead.blogspot.com/2012/01/lente-macro-del-hombre-pobre.html) ( [翻译](http://translate.google.com/translate?hl=en&sl=auto&tl=en&u=http%3A%2F%2Fbinaryhead.blogspot.com%2F2012%2F01%2Flente-macro-del-hombre-pobre.html))的盖子和螺纹颈部，使镜头可调和可拆卸。

在过去，我们已经看到这种黑客使用[一个钻有孔的镜头盖](http://hackaday.com/2010/05/06/building-removable-epoxy-mounting-brackets/)作为安装支架。但这只有在你选择的镜头确实有镜头盖的情况下才有用。这种方法可以让你把汽水瓶盖切掉，然后装在相机上。现在，每个镜头都可以固定在瓶子的螺纹颈部，通过拧入或拧出附件，可以对焦点进行一些调整。

显然，这对微距镜头或鱼眼镜头来说效果很好。但是还有各种各样的其他选择，比如添加一个显微镜镜头适配器，或者在你的设备上安装高质量的光学元件。
