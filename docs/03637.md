# 芭比的网络漫游器

</span> <p>[亨特、凯尔和迪伦]给我们发了一些关于他们的芭比网络漫游车的信息。这是一辆老式的芭比动力轮吉普车，已经被改造成了一辆网络遥控汽车。他们拆掉了旧的传动系统和转向系统。后轮现在独立驱动转向。它使用 Arduino 来控制电机，使用装有 linux 的宏碁 Aspire 来实现更高的功能。他们提到最远的测试在 1600 英里以外，这很酷，但是当它支持 web 时，距离真的很重要吗？</p> <p>他们提到凉爽因素与尺寸成比例，我们不得不同意，只要他们保持足够小，不会造成任何真正的伤害。如果你不想放弃你的笔记本电脑，你可以通过一个路由器再多花一点力气就能建立一个网络漫游器。</p> </body> </html>
