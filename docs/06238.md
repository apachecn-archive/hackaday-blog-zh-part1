# 制造点接触晶体管

> 原文：<https://hackaday.com/2010/11/05/making-point-contact-transistors/>

 <https://www.youtube.com/embed/vmotkjMSKnI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>[Jeri Ellsworth]又回来了。我们在这里似乎报道了她的很多作品。她的最新视频涵盖了她如何从 1N34 锗猫须二极管创建一个点接触晶体管。打开二极管上的玻璃外壳后，她用普通电连接器上锋利的磷青铜金属作为收集器和发射器。一个 330 微法的电容器充电到 20 伏，然后通过一个 680 欧姆的电阻放电到基极和集电极引线，形成集电极区。她的测试夹具是一个简单的振荡器电路，这样一个适当形成的晶体管将启动电路振荡，并发出可听见的声音。我们期待更多关于电子设备的深奥知识引起我们的注意。</p> </body> </html>
