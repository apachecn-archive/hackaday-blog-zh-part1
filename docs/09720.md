# 紧急按钮让你远离糟糕的音乐

> 原文:[https://hack aday . com/2012/02/19/the-panic-button-saves-you-from-bad-music/](https://hackaday.com/2012/02/19/the-panic-button-saves-you-from-bad-music/)

![](../Images/3d93d6809a96a8f288ecf0198b25b7b5.png "panic")

随机播放公共音乐收藏真的很有趣。你体验新的音乐，听到老的最爱。然而，并不是每个人都有相同的口味。有时候，为了所有相关的人，你绝对需要跳过即将播放的歌曲。紧急按钮就是这样诞生的。

从一个会发出警报声的恶作剧紧急按钮开始，他们迅速将内脏取出。他们插入了一个 minimus 板，它有一个 AT90USB162 大脑，并包括板载 USB 硬件。当你按下紧急按钮时，它会模仿按键进入下一首曲目，从而将每个人从任何威胁他们耳膜的怪物手中拯救出来。
