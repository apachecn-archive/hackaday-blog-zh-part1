# 我们要薯条。

> 原文:[https://hackaday.com/2009/11/01/well-take-the-fries/](https://hackaday.com/2009/11/01/well-take-the-fries/)

![fjdkalfjdklajfdaflj](../Images/1a37ce28548fbe5e01f1afa6d9008f92.png "fjdkalfjdklajfdaflj")

我们都经历过。我们做过的最艰难的决定。大学和债务还是自由但没有事业？成家立业还是独自在外生活？2 号餐加一小杯奶昔或薯条？！

[Leon]发送了他的[随机数生成器](https://mywebspace.wisc.edu/lnmaurer/web/minirng/minirng.html)，奇怪的是不只是生成一个数字，而是帮助我们完成那些改变生活的决定。使用类似设置的[噪声二极管](https://mywebspace.wisc.edu/lnmaurer/web/minirng/minirng.html#Random)，他能够产生随机位，然后由 ATtiny24 转换并显示在 7 段 LED 上。他的发电机也有能力设置上限和下限。要避免选择的自由似乎需要做大量的工作【T4 悖论】，但是我们仍然喜欢这个项目。休息之后看一段视频。
