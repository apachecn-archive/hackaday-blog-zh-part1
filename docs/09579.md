# 一个 8×8 的游戏

> 原文:[https://hackaday.com/2012/01/27/sixty4racer-an-8x8-game/](https://hackaday.com/2012/01/27/sixty4racer-an-8x8-game/)

![](../Images/471bb4be10723cb7f8965ec16e1fd41f.png "Untitled")

【皮特】有一个很酷的新[教程，为基于](http://10rem.net/blog/2012/01/22/my-first-real-pix-6t4-game-sixty4racer) [PIX-6T4](http://www.pix6t4.com/) 微控制器的游戏系统创建了一个对雅达利经典作品的重新想象。PIX 是一个在更大的板上的 netduino，具有 2 个模拟控制器、一个扬声器、一个 sd 卡和一个 8×8 单色 LED 显示屏。这么低的分辨率，可能会让老 VCS 看起来像一个 360，但在这么低的水平上制作游戏还有很多东西要学。

[Pete]整理的教程涵盖了概念、游戏性、进度目标、屏幕处理和一船代码来展示这一切是如何一起进行的。尽管这是针对一个基于 C#的系统，但许多基础知识适用于你能想象到的任何系统。因此，如果你想学习如何处理 C 语言中的图形、精灵动画、碰撞，或者随机地将瓷砖块放在一起，那么你应该看看。
