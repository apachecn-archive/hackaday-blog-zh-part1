# Wiimote 投影仪白板

> 原文:[https://hack aday . com/2007/12/09/Wii mote-projector-white board/](https://hackaday.com/2007/12/09/wiimote-projector-whiteboard/)

[李炯楷]带着他的 Wiimote 交互式白板又回来了。这些东西的商业版本又贵又重。他的技术甚至不需要投影仪，只需要一台电脑、一台 Wiimote 和一支简单的红外发射笔。这种笔只是一支笔尖装有红外 LED 的手写笔。休息后点击视频，你可以从他的网站[这里](http://johnnylee.net/projects/wii/)下载软件。
