# GATARI“2600”乐器

> 原文：<https://hackaday.com/2012/01/02/the-gatari-2600-musical-instrument/>

[![gatari](img/22e0d98560a1a3096c1454c1cdc77e04.png "gatari")](http://hackaday.com/2012/01/02/the-gatari-2600-musical-instrument/gatari/)

你有雅达利 2600 在收集灰尘吗？如果你有一台现代化的游戏机，也许你会想要一件乐器来占用你玩视频游戏的时间。嗯，看看 GATARI 2600 就知道了！

[cTrix]用一个定制的 EPROM 芯片作为盒式磁带插入制作了这个设备。虽然构建的细节有些模糊，但这种定制芯片允许为设备编写音乐。一切都由操纵杆控制，告诉 GATARI 生成所需的轨迹。从这个基本轨道开始，使用三个踏板对声音进行修改，包括均衡器、凸缘踏板和保持踏板。

休息之后，请观看视频，了解它是如何构建的简要说明。如果你更想在日本的[Blipfest]上看到它的运行，请跳到 1:05！
