# 旋转视点时钟做得太好了

> 原文：<https://hackaday.com/2010/10/03/spinning-pov-clock-done-oh-so-right/>

![](img/1e9401b43b39b453bf4716807a7298a3.png "spinning-pov-done-right")

[Kizo]建造了一个非凡的视觉暂留钟。该设计使用 PC 冷却风扇来旋转螺旋桨状的 PCB。当它四处走动时，霍尔效应传感器同步发光二极管的照明以绘制显示。旋转电子设备的电力通过底座上的变压器和旋转板上的线圈无线传输。最终版本使用运行在 20 MHz 的 ATmega324 微控制器，并具有用于改变设置的 IR 接收器。这 3000 行代码带来了许多花哨的功能，包括一个具有大量设置的菜单系统，从调整时钟显示到滚动消息的字体选择。休息之后看一下演示。双面板看起来很难在家里蚀刻，但正如你从[论坛帖子](http://www.elektronika.ba/forum/viewtopic.php?t=9243) ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://www.elektronika.ba/forum/viewtopic.php%3Ft%3D9243))中看到的，[Kizo]从头到尾在这个构建上做得很好。 <https://www.youtube.com/embed/Q1saYuGyw7A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
