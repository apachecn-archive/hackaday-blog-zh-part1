# 电影搬运工:移动电影院

> 原文：<https://hackaday.com/2009/12/04/movie-mover-a-mobile-theater/>

[https://www.youtube.com/embed/po8F6lIFhkk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/po8F6lIFhkk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[电动独轮车]组装了这个[无线电控制的移动影院系统](http://hackedgadgets.com/2009/12/04/mobile-movie-theater-robot/)来娱乐他的邻居朋友。它是一个投影仪，看起来像一个电动轮椅底座。他还设法在里面安装了一套像样的音响系统，这需要第二块电池。这是非常酷的，虽然我们可以看到加固它多一点，以便能够在田间驾驶。他说“这是附近最快的电影院之一”。这让我们想知道竞争是什么样的。
