# 简易高压电源

> 原文:[https://hack aday . com/2008/11/10/easy-high-voltage-power-supply/](https://hackaday.com/2008/11/10/easy-high-voltage-power-supply/)

![hv_supply](../Images/134a0cbcdd22aa99950a18167ebd15a3.png "hv_supply")

[rocketman221]写了一个最简单的方法来建立一个高压电源。这个特别用在了他的[钢卷](http://hackaday.com/?s=coil+gun "Hack a Day")上。他没有建造定制电路，而是使用一次性相机的闪光充电板。六个 450V 470uF 电容器并联在一起构成一组电容器。两个充电器板连接到一个开关，以启动充电过程。四个额外的板被连接到用于第二充电阶段的第二开关。它的器件成本非常低，只需 3.3V 输入电压就能达到 450V。这篇文章中有很多关于高压危险的警告；你需要清除所有的焊剂残渣，以防止电路板上产生电弧。下面嵌入的是银行通过几个物体放电的视频。
