# 自动天空跟踪捕捉不明飞行物

> 原文：<https://hackaday.com/2012/03/16/automated-sky-tracking-to-catch-ufos/>

![](img/04c1daca25a1046f6e2a66bfcfc533f0.png "UFO")

如果你想拍一张不明飞行物的照片，你会怎么做？坐在内华达州 51 区附近的路边？绳子上的馅饼罐？一个法国 UFO 爱好者小组用迪斯科灯和闭路电视摄像头组装了一个自动 UFO 探测设备，这样长时间观察天空就可以自动化了。

该建筑使用一个带有高度和方位支架的迪斯科灯不断扫描天空，寻找奇怪的，无法解释的光。连接到这个旋转支架的是一个摄像机和一个闭路电视摄像头，它将视频流传输到指挥和控制笔记本电脑进行图像分析。

除了物体追踪，闭路电视摄像头前还有一个衍射光栅。这个项目背后的团队以前用这个来做一些[非常低技术的光谱学](http://www.ufo-science.com/wpf/?page_id=594) ( [翻译](http://translate.google.com/translate?sl=fr&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.ufo-science.com%2Fwpf%2F%3Fpage_id%3D594&act=url))来识别光源中的发射线。具有包括氧和氮的特征的光可能是电离空气，而不太常见的元素可能是“高级推进”的特征。

虽然这个建筑将会探测到大量的卫星和流星，但是很有可能在视频中捕捉到一个无法解释的现象。
