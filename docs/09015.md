# 破解“魔棒”远程控制灯光显示

> 原文：<https://hackaday.com/2011/11/07/hacking-a-magic-wand-to-remotely-control-light-displays/>

Hackaday reader [Kieran]在一个名为“幻灭森林”的户外鬼屋景点做志愿者。一名志愿者带领与会者穿过闹鬼的森林，他帮助每个人保持在预定的路线上。这条步道通常由该团队建造的小型 LED 灯具照明，但组织者希望这一次让灯光更具互动性。

另一位组织者给了[基兰]一根[哈利·波特]魔法蜡烛，让他可以用红外线魔杖点着玩具。他被告知“让它做一些很酷的事情”，所以他仔细观察它，以确定一切是如何工作的。

使用一个 Arduino 克隆体和一些借来的红外代码，他能够让魔杖与森林的步道照明一起工作，但是在挥动魔杖和触发灯光之间有很大的延迟。在第二次尝试中，[基兰]复制了玩具使用的红外协议，加快了速度，大大增加了魔杖的射程。现在，导游只需轻轻一抖手腕就能点亮和熄灭步道照明。
