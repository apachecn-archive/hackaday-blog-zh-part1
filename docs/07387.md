# 基于 FPGA 的 Turbografx 16 克隆

> 原文:[https://hack aday . com/2011/04/14/FPGA-based-turbografx-16-clone/](https://hackaday.com/2011/04/14/fpga-based-turbografx-16-clone/)

[Gregory]来信分享了他最近的项目，[一个 PC 引擎/Turbografx 16 控制台](http://torlus.com/index.php?2011/04/10/237-pc-engine-turbografx-16-in-a-fpga)的 FPGA 克隆。你可能还记得去年的他，当时我们谈到了[他的 SEGA 创世纪 FPGA 克隆](http://hackaday.com/2010/07/16/sega-genesis-cloned-with-an-fpga/)。他不能就此罢休，决定以 FPGA 的形式复活另一台 16 位机器。

他已经在这个项目上工作了大约三个月，但是他做得很快，一切都准备就绪并开始运行。几周前，这个项目还处于非常不稳定的 alpha 阶段，但是在解决了一些 bug 之后，他现在可以随心所欲地渲染任何游戏了。

这个克隆人使用 Altera DE1 板，就像他以前的版本一样，他已经能够模拟构成 Turbografx 逻辑板的所有三个主要芯片。他还在研究可编程的声音发生器，但这将在不久的将来实现。虽然 FPGA 目前将 ROMS 存储在其闪存中，但他计划增加从 SD 卡加载游戏的能力。

继续阅读，看看一对展示他的主机克隆在行动的视频，令人印象深刻。
