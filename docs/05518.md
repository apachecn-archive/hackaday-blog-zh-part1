# 会说话的扑克计时器

> 原文：<https://hackaday.com/2010/06/18/talking-poker-timer/>

[https://www.youtube.com/embed/k8iIuPnXzm8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/k8iIuPnXzm8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这个有趣的按钮盒是一个会说话的扑克锦标赛计时器。充满了有用的工具，如二进制时间显示，智慧的话，倒数到游戏结束，甚至是很好的旧的“两人自行车”模式(大约 1:20)。虽然我们发现它很难理解，但我们还是为它的功能列表鼓掌，尤其是这首歌。他使用了一个带语音屏蔽的 Arduino，所以电子方面没有太多东西，但你可以从他的网站下载他的源代码。
