# 啤酒冰箱挟持 IPad

> 原文:[https://hack aday . com/2010/08/12/beer-冰箱-货舱-ipad-人质/](https://hackaday.com/2010/08/12/beer-fridge-holds-ipad-hostage/)

![](../Images/8fea33992a9b2b39933bf126f3e8d399.png "beer-fridge-with-iPad")

Yelp 的工程师们举办了一次黑客马拉松，生产了这款以 iPad 为中心的啤酒冰箱。一个口渴的家伙吃饱了撑着来到这个装置前，刷了他或她的 RFID 卡，然后分发了一杯多泡沫的美味。iPad 显示屏会显示关于正在播放的内容的信息，并允许您给它评分。这是基于 [kegbot](http://hackaday.com/2009/12/14/monitor-keg-volume-co2-and-temp/) 的，但是它有一些额外的技巧。这个版本不是通过重量来测量还剩多少啤酒，而是在啤酒管道上使用一个流量传感器。温度数据由固定在桶上的模拟传感器记录，整个桶通过连接到 iPad 串行端口的 Arduino 连接在一起。你甚至可以在网上查看啤酒。休息之后自己看吧。
