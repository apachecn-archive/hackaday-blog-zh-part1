# 铣削互锁木环

> 原文:[https://hack aday . com/2011/12/01/milling-interlocking-wood-rings/](https://hackaday.com/2011/12/01/milling-interlocking-wooden-rings/)

![](../Images/c543785c1a0927ca88a3546138633c11.png "milling-wooden-rings")

[Johan von Konow]想做点特别的东西作为结婚礼物送给妻子。他认为一对互锁的微型戒指是最好的纪念品。他开始寻找从一块实心木头上磨出木环的方法，并记录了他的旅程供我们欣赏。

这个项目提出了一个有趣的挑战。大多数数控铣床提供三个轴的自由度，但他只有一个 2D 铣床意味着路由 PCB。这意味着切割只能在一个深度从上到下进行。为了制作戒指，他需要从多面切割。随着更多的研究，[Johan]发现在环完成之前，必须从八个不同的角度切割木材。

解决这个问题的方法是先磨一个夹具来固定木材。它在每个不同的角度持有股票。开始切割前的最后一步是研磨原料本身，以完全适合他的定制夹具。我们认为结果很棒，部分归功于手工锉磨、打磨和抛光，以消除铣削留下的痕迹。
