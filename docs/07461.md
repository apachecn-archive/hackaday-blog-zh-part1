# 继电器式计算器

> 原文:[https://hackaday.com/2011/04/22/relay-calculator/](https://hackaday.com/2011/04/22/relay-calculator/)

![](../Images/10cab99324946a58dcbbfdd14426b80f.png "Screenshot-16")

计算器是几乎每个应用程序中都有的便利工具。我们今天经常认为它们是理所当然的，但即使在我还是个孩子的时候，它们仍然是你想买的那种昂贵的设备。[619 团队]带给我们的这个令人敬畏的[接力计算器](http://www.carobotics.org/projects/calculator/)说明了我们已经走了多远。

具有光学滑块输入系统，用户可以选择任何两个 4 位数，并可以增加或减少他们。逻辑由几个继电器执行，这些继电器被设置为“与”、“或”、“异或”门，然后连接在一起构成加法器。

输出也是二进制的，以光的形式，虽然我们真的不知道这些是某种形式的管子，还是它们只是末端发光的棒。无论哪种方式，如果你需要大量的啃咬数学，并希望一个对话的开始，这适合法案相当不错。否则，你可以继续连接越来越多的继电器，也许有一天[可以制造你自己的继电器计算机](http://hackaday.com/2010/11/18/electromechanical-computer-built-from-relays/)。
