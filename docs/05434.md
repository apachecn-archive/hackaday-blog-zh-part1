# 没有 NXT 零件的乐高打印机

> 原文：<https://hackaday.com/2010/06/02/lego-printer-built-without-nxt-parts/>

[https://www.youtube.com/embed/zX09WnGU6ZY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zX09WnGU6ZY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Squirrelfantasy] [用乐高](http://www.b3ta.com/links/Lego_printer)积木搭建了一台打印机。这不是一个 Mindstorm 项目，而是依赖于某种类型的开发板和原型板上的一些辅助组件。我们无法得到足够好的外观来告诉到底是什么组成了电子设备，所以在评论中开始辩论。我们觉得这是一台打印机，而不是绘图仪，因为当纸张通过它时，触针只在一个平面上移动，但这也是值得商榷的。

我想这回答了[为什么我们不制造自己的打印机](http://hackaday.com/2010/01/26/why-arent-we-building-our-own-printers/)这个问题？有些人是。
