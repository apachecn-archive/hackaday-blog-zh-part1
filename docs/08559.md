# 世界时钟模拟白天和黑夜

> 原文：<https://hackaday.com/2011/09/12/world-clock-simulates-night-and-day/>

[Justin] [一直想要一个地质时钟](http://www.instructables.com/id/Night-and-Day-Clock/)，但是由于它们的价格在[几千美元](http://www.geochron.com/)的范围内，他很确定他永远不会有机会拥有一个。他毫不气馁，想出了一个为自己制作一个小型时钟的方法，并写了一篇文章来分享他是如何做到的。

他首先从 FlorinC 购买了一个 [Wise 时钟 3](http://hackaday.com/2011/02/09/pong-clocks-using-led-matrix-modules/) ,但是他肯定不会按照最初的意图使用这个时钟。他没有用数字显示时间，而是把怀斯钟拆开，把一张世界地图的羊皮纸打印本夹在面板中间。经过调整的固件映像允许他使用 Wise Clock 的 LED 阵列模拟白天和黑夜。他还对时钟进行了编程，以考虑季节性光照模式，正如你在下面嵌入的视频中看到的那样。

我们认为这是一个伟大的想法，虽然我们可能会使用普通的白色发光二极管，如果我们建立一个，明智的时钟中的 RGB 发光二极管肯定会提供一个整洁的效果。
