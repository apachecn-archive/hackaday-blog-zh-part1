# 在 Lifehacker 学习编码

> 原文:[https://hackaday . com/2011/01/26/learn-to-code-at-life hacker/](https://hackaday.com/2011/01/26/learn-to-code-at-lifehacker/)

![](../Images/2d518797a696b0c19c2f4f40333a6dc8.png "lhns")

lifehacker 的[Adam]正在整理一个关于编程原则的系列，名为[“学习编码”](http://lifehacker.com/5736011/learn-how-to-code-part-i-variables-and-basic-data-types)。他们使用 Javascript 作为基础来教授基础知识，这将允许你用任何类似的语言完成工作，比如 actionscript。在你掌握了这些基础知识之后，即使转向面向对象语言也不会太难。我们绝对热爱教育资源，并希望他们继续这个系列相当远。他们已经发布了第一部和《T2》第二部。我们在休息后提供了每个人的视频，但是在链接上有更多的信息。
