# 声控 led 标志

> 原文:[https://hackaday.com/2009/11/10/voice-controlled-led-sign/](https://hackaday.com/2009/11/10/voice-controlled-led-sign/)

[https://www.youtube.com/embed/KiYYJ5WMG6I?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/KiYYJ5WMG6I?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

120 个 led 和 NerdKit，检查。Python 和适当的谷歌语音模块，检查。在高温下搅拌 2 分钟，在 400 华氏度下烘烤一个小时。你的蛋奶酥做好了，哎呀，我们是说你的[声控 led 标志](http://thedavisblog.com/blog/?p=691)做好了。在戴维斯的[谷歌语音](https://www.google.com/voice)账户上留下一个以“消息”开头的语音邮件，它会在几分钟内显示给所有人阅读。我们认为:把它变大，在你可以留言之前加一个安全码，你就有了一些[超大消息乐趣](http://www.youtube.com/watch?v=hFqnDjyDI9o)的完美配方。
