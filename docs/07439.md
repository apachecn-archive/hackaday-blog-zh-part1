# IPhone 监视你的每一次呼吸，每一个动作

> 原文:[https://hackaday . com/2011/04/20/iphone-观看每一次呼吸，采取每一次行动](https://hackaday.com/2011/04/20/iphone-watching-every-breath-you-take-every-move-you-make/)

![iphone_data](../Images/efc333abd4348569832fd095f180a674.png "iphone_data")

大多数人倾向于享受一定程度的隐私。除了我们都愿意在网上以论坛帖子、Twitter 活动等形式分享的数据。人们通常喜欢独处。

如果你发现你的 iPhone(或任何 3G 设备)正在跟踪和记录你的每一个动作，你会怎么想？

这正是两位来自英国的研究人员所宣称的。他们表示，手机会不断使用手机信号塔记录你的位置，将信息放入带有时间戳的数据库中。该数据库未加密，并且每次与 iTunes 同步时都会拷贝到您的电脑上。此外，如果您更换手机，该数据库会复制回您的新手机。

我们知道许多 iPhone 应用程序使用位置感知来增强用户体验，如果有必要，执法人员应该能够从你的手机中提取数据——我们对此完全没有意见。然而，当你去过的任何地方都是在没有任何通知的情况下被秘密地明文登录时，我们就有点警惕了。至少，苹果应该考虑加密文件。

虽然这些数据不像你的社会安全号码或银行密码那样敏感，但如果落入坏人之手，同样是危险的。即使是一个中等技能的小偷，在找到或偷了一部 iPhone 后，也可以轻松地倾倒其中的内容，并拥有一个强大的数据集，显示你住在哪里，何时离开——所有这些都是完美的入室盗窃的要素。

继续阅读，看看两位研究人员讨论他们的发现的相当长的视频。
