# 开源武器让你呕吐

> 原文：<https://hackaday.com/2009/09/28/open-source-weapon-makes-you-puke/>

![](img/228a10404a895833fd781a97c5465811.png "The Bedazzler")

[Adafruit Industries](http://www.adafruit.com/) 的 [Limor](http://ladyada.net/) 和[冰管钟](http://hackaday.com/2009/08/25/learn-from-the-ice-tube-clock/)做出了自己的[开源非致命武器](http://ladyada.net/make/bedazzler/):眩目器。在参加了 DHS 的一次会议后，她看到了这部大制作的电影，她决定制作自己的电影。三十六个发光二极管，六个开关场效应晶体管，一个[电路板](http://www.adafruit.com/index.php?main_page=product_info&cPath=19&products_id=72)，以及一个以前的手电筒，这个炫目的东西比武器更好的狂欢玩具。它不能像现在这样工作，但是我们认为有人黑掉它让人们……嗯，黑掉它只是时间问题。休息后看视频。
