# 老派 LED 灯光秀

> 原文:[https://hackaday.com/2011/03/01/old-school-led-light-show/](https://hackaday.com/2011/03/01/old-school-led-light-show/)

![programmable_lightshow](../Images/5666d6ba5a2bed00b8423bb71e3beadd.png "programmable_lightshow")

如今，构建可以显示各种不同图案的 LED 阵列相当容易。挂上一个 Arduino，做一些 charlieplexing，然后你就可以走了。当[维克多]年轻的时候，他没有你们这些孩子现在拥有的那些花哨的微型控制器和电路模拟软件。事实上，据我们所知，他不得不在雪地里走着上山去上学。

这并没有阻止他在 1987 年建造这个项目的瑰宝。他的 LED 追踪器/灯光秀根本不使用任何微控制器，而是依靠 EPROM 来存储预定义的显示程序。一系列开关安装在闪光器的前面，使他可以轻松地在程序之间切换，一个罐子安装在设备的前面，以控制发光二极管的速度。

即使对于一个 20 多年前建造的项目来说，他的灯光秀也是相当流畅的。有时候你就是打不过一个优秀的老派黑客。
