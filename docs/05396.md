# 斯塔比先生的故事

> 原文:[https://hackaday.com/2010/05/26/the-story-of-mr-stabby/](https://hackaday.com/2010/05/26/the-story-of-mr-stabby/)

斯塔比先生曾经是一个穷困潦倒的流浪汉，坐在路边，在城市垃圾堆里等待永恒。幸运的是，有人发现了他，并把他带到了纽约电阻黑客组织。他们立即进行了一些修改，使他复活了。

他现在是一个快乐的幸运刺机器人，眼睛闪闪发光，脚步轻快。他的报告包括刨、砍，当然还有刺。他可以通过手机控制，有一个漂亮的语音界面。
