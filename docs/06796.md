# 意大利面条制成的热喷枪

> 原文：<https://hackaday.com/2011/02/11/thermic-lance-made-from-spaghetti/>

[弗罗格兹]发来一段视频，他发现了用意大利面条制成的热喷枪。如果你不熟悉的话，热喷枪通常由一个装满铁棒的铁管组成，然后使用高压氧气燃烧。然而，这个矛是通过用铝箔紧紧包裹一捆意大利面条，并将其连接到氧气罐上而制成的。虽然热喷枪通常用于需要切割厚钢的重型建筑，[latexiron]和他的朋友们用它们来切割椅子。虽然我们不一定会宽恕醉酒破坏无辜的庭院家具，但我们不禁一遍又一遍地观看这个视频，惊讶于日常面食的令人难以置信的新奇用途。跳完之后，你也可以加入醉酒狂欢。如果你喜欢以食物为基础的切割火把，一定要看看这个[培根长矛](http://www.popsci.com/bacon)。
