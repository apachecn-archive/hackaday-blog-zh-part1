# 模块化视点最终项目

> 原文:[https://hackaday.com/2009/05/06/pov-final-project/](https://hackaday.com/2009/05/06/pov-final-project/)

![pov](../Images/c6f7a881be13bd33743de86f9e43546d.png "pov")

[乔纳森]把他 ECE 476 的最终项目发给了我们。他的团队建造了一个可以从电脑无线更新的 LED 视点显示器。他们以某种模块化的方式建造了它，LED 灯条是一种易于更换的模块。这符合他们建造 3D 版显示器的计划。他们已经使它能够轻松地从内部编程切换到直接的计算机控制，允许他们在时钟模式下离开它，或者控制它玩。更新似乎真的很快，你可以在休息后的视频中看到它们的效果。所有的源代码和原理图都可以在网站上找到。我们喜欢这些最终项目，它们通常有大量的数据和很棒的图片。让他们继续来。
