# 另一个固定自行车虚拟现实装备

> 原文:[https://hack aday . com/2009/11/19/another-stationary-bike-VR-rig/](https://hackaday.com/2009/11/19/another-stationary-bike-vr-rig/)

![](../Images/4d7b252cb0213442b97671bddd79c92f.png "bike-vr-rig")

[Shingo]分享了他将一辆[固定自行车作为虚拟现实](http://www.youtube.com/watch?v=DG1FWa-0XrE)界面的实现。这与我们一周前报道的[谷歌街景设置](http://hackaday.com/2009/11/11/biking-through-google-street-view/)类似，但更进一步。他们在自行车电脑上安装了补丁，以检测自行车车轮的旋转，并增加了一个加速计来控制方向。这种设置可以在街景中导航，但休息后的视频还详细介绍了与谷歌地球的界面，甚至可以在第二人生中导航，跟随你的化身一起骑车。使用可穿戴显示器远远优于像[冲浪架](http://www.surfshelf.com/)这样的东西，并且真正给你一个目标，而不仅仅是一些寒冷天气的锻炼。所以接受这个想法，把它安装到一个可穿戴的电脑上，你就拥有了一个与我们承诺的未来世界相称的锻炼系统。
