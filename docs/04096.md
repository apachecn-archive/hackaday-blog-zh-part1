# 爱迪生汽缸录音需要更多的牛铃

> 原文：<https://hackaday.com/2009/09/24/edison-cylinder-recordings-need-more-cowbell/>

![bruderhofer2-1](img/3ed13631965fc9fe78d6c237ffd28ce9.png "bruderhofer2-1")

[诺曼]花了三年时间开发和建造自己的[爱迪生圆筒留声机，配有电动拾音器](http://www.christerhamp.se/phono/bruderhofer2.html)。我们很高兴他做到了，而且他与世界分享了它，因为这个产品是一个美丽的东西。每一部分都是干净和精确的，有足够的空间进行调整，以适应不同的媒体。他重新使用了录像机的磁头，并把它连接到一个数控加工的聚丙烯芯轴上。针通过精密的被动驱动支架与针筒连接。它由一根一端带弹药筒的铝棒和另一端的两个轮子组成。轮子沿着一根精密的杆移动，由针沿着蜡上的凹槽推进。精彩！

我们嵌入了一段设备播放苏萨 19 世纪晚期的《El Capitan 》( T1)的视频。虽然我们熟悉这些柱面录音，但我们惊讶于一个柱面上可用的录音空间如此之少。
