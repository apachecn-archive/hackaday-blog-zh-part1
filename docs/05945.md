# 银翼杀手雨伞军刀

> 原文:[https://hack aday . com/2010/09/14/blade-runner-umbrella-saber/](https://hackaday.com/2010/09/14/blade-runner-umbrella-saber/)

![](../Images/ac5d001e207a79ca151d9512ad0e28b1.png "blade-runner-umbrella-saber")

这是一把银翼杀手伞，做得有点太对了。它的光剑防雨外观带来了双倍的极客感，但我们认为它越界的地方是内置的音频系统。当你打开它时，它会播放《银翼杀手[中的流行歌曲，这可能不会在公共场合播放。但是与我们去年看到的](http://en.wikipedia.org/wiki/Blade_Runner)[一元店发光雨伞](http://hackaday.com/2009/11/12/raid-the-dollar-store-for-an-illuminated-umbrella/)相比，质量是在不同的星系中。

[Erv' Plecter]用透明的聚碳酸酯管替换了伞的中央支撑杆。一根光缆蜿蜒穿过空心管，由手柄中的 Luxeon LED 照明。定制的 PCB 和 900 毫安的电池都装在那里。休息之后，看一看(听一听)演示。我们需要在[光剑电影复制版](http://hackaday.com/2010/08/27/devote-your-life-to-replicating-a-lightsaber/)之后将它添加到我们未来的项目列表中。
