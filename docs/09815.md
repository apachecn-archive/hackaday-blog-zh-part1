# 星球蛇用你周围的环境做了一个游戏板

> 原文:[https://hack aday . com/2012/03/02/snake-the-planet-make-a-game-board-out-of-your-environments/](https://hackaday.com/2012/03/02/snake-the-planet-makes-a-game-board-out-of-your-surroundings/)

![](../Images/fae0554698a7a0374f93fbb3705f512c.png "snake-the-planet")

这是周五晚上，这些家伙开车在镇上四处寻找一个好地方玩面对面的蛇游戏。这并不是说他们需要一个可以坐的地方(为此他们带着沙发和落地灯旅行),而是说[他们使用投影仪和摄像机在他们的周围制作一个游戏](http://mpulabs.com/#snake-the-planet)。

一辆白色的神秘机器风格的货车有足够的空间来放置他们需要的一切东西，使这个移动的拱廊成为现实。移动电源为摄像机和投影仪提供电力。首先，该系统对其前方的表面进行高对比度的黑白拍照。出现在白色阈值以下的所有东西都成为游戏板上的一面墙，其他所有东西都是可玩的区域。障碍是由窗户、门口、管道、标志、挂在墙上的泡沫板组成的，甚至是你的身体，如果你在扫描时挡道的话。从那里，每个人抓住一个操纵杆，玩黑客最喜欢的贪吃蛇游戏。
