# 消防喇叭

> 原文：<https://hackaday.com/2009/09/02/fire-horn/>

![firehorn](img/f8c07546901e9a984ef1102cb2740733.png "firehorn")

艺术家阿里埃勒·施莱辛格(Ariel Schlesinger)成功地用丁烷燃料替换了 T2 号(air horn)T3 的压缩空气罐。这个黑客设法稍微改变喇叭的音调，同时制造一个非常吵的喷火器。虽然不像[和](http://hackaday.com/2009/08/22/the-serpent-mother/)[其他](http://hackaday.com/2009/08/16/colored-pyrotechnics/) [火焰喷射器](http://hackaday.com/2008/12/11/walkdrive-a-dogmotorcycle/)那么令人印象深刻，但这可能是最容易实现的。目前，包含 how-to 的网站[已经关闭，但是我们还是建议不要在家里这样做。跳跃后的视频。](http://www.vvank.com/08/fire-horn/)
