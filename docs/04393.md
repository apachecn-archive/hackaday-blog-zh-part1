# 过山车控制的 HTPC

> 原文：<https://hackaday.com/2009/11/09/coaster-controlled-htpc/>

![RFID](img/9625c96d59111d5a142e3d3a0d78900c.png "RFID")

如今，HTPCs 变得越来越普遍，然而优雅地控制内容可能是一个令人烦恼的问题。Roteno 实验室提出了一个奇妙的解决方案，他们称之为 RF idj T1。与我们之前报道的 [RFID 手机](http://hackaday.com/2009/03/14/rfid-controlled-phone-dialing/)类似，他们使用了一套相框杯垫，并在每个杯垫上安装了描述性图片和独特的 RFID 标签。当一个杯垫被放置在传感器区域时，服务器开始传输特定的选择，包括当地新闻，本周科技播客中的[，以及其他特定的专辑。Roteno Labs 甚至设法加入了一个“shuffle”标签，可以从库中随机播放内容。最终的结果是非常好地放在一起，出色地记录，甚至有一个工作视频后休息。](http://twit.tv/)
