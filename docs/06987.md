# LED 艺术品就在你眼前消失了

> 原文:[https://hack aday . com/2011/03/02/led-artwork-消失-就在你眼前/](https://hackaday.com/2011/03/02/led-artwork-disappears-right-before-your-eyes/)

![take_a_picture_demo](../Images/8c5f345e02dc500f700b90de14d4cb1d.png "take_a_picture_demo")

如果你走进一家美术馆，看到墙上除了空白的画布什么也没有，你可能会被迫要求退款，或者认为你发现了世界上最懒的艺术家。然而，如果这个画廊碰巧在展示(布拉德·布吕歇尔和凯尔·克莱蒙特)的作品，那你就错了。

这两位艺术家合作创作了一系列名为“[拍照](http://www.kyleclements.com/tap/)”的作品。当用肉眼观看时，他们制作的每一幅画看起来都像一张空白的画布。然而，如果你用手机或数码相机给画布拍照，一个全新的世界将展现在你的眼前。他们的作品使用红外 led 构建，肉眼无法看到，但市场上几乎任何 CMOS 或 CCD 传感器都可以看到。这些图像从简单的笑脸和物体到抽象的几何形状。

这是一个非常简单而新颖的方法，我们碰巧认为它很酷。艺术家们还没有说他们对这个项目未来的计划，但我们很乐意看到它使用更大的 LED 阵列来显示更高分辨率的图像，甚至短片。

请继续阅读，看看他们是如何创作这些艺术作品的，以及展示这种效果的宣传片。
