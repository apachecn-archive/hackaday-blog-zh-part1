# Pyxis 3 打 Beta 更名为 Gadgetos

> 原文:[https://hackaday . com/2011/12/09/pyxis-3-hits-beta-rebranded-as-gadget OS/](https://hackaday.com/2011/12/09/pyxis-3-hits-beta-rebranded-as-gadgetos/)

![](../Images/8e87c45273da7028bf72e2bcb57610be.png "gadgetos-pyxis3-beta")

Pyxis 3 的测试版现已推出。Skewworks 继续开发 ARM 操作系统，随着向版本 3 的过渡，他们给它起了一个新名字:Gadgetos。与 [Pyxis 2](http://hackaday.com/2011/03/04/net-os-for-your-arm/) 的一个很大的不同是新的内核是闭源的。但是他们已经采取措施确保操作系统仍然是独立于硬件的。这是通过修改内核以允许在运行时从 SD 卡加载驱动程序来实现的。

用户界面也有所改变。Gadgetos 完全依赖于基于上下文的菜单系统。最低输入硬件要求一个触摸屏 LCD 和一个触摸按钮。按住那个按钮，屏幕中央会弹出一个菜单。这个菜单根据它被调用的位置而不同(这是上下文出现的地方)。如果你在桌面屏幕上，你可以选择载入程序等等。如果你在运行应用程序时打开菜单，你会看到该应用程序可用的选项。想了解更多关于新导航系统的信息，请在休息后观看视频。
