# 555 合成器

> 原文:[https://hackaday.com/2011/03/09/a-555-synthesizer/](https://hackaday.com/2011/03/09/a-555-synthesizer/)

![](../Images/294d3bcc227616cd07425737cdcc6bcd.png "synth")

受到童年电子设备记忆的启发，[Frank]决定制作一个新的 [555 合成器](http://www.frank-buss.de/555/index.html)，并将其加入 555 竞赛。[弗兰克的]翻拍是用唱针演奏的，带有一个启动和释放的包络电路，装在一个快速但有效的丙烯酸外壳中。

使用一个 555 定时器，一只装满电容器的手，两个晶体管，以及适量的电阻器和电位计，由于可选的滤波、包络和大量微调，声音令人印象深刻。完整的原理图、物料清单、公式和电路板文件以及一些更专业的电位计的来源均可用。

休息后加入我们，观看一个 13 分钟长的视频，非常酷，因为它显示了从原型到最终产品的一些快进动作，尽管你只是想要 555 synth 快进到 10:44 的演示
