# 加重的同伴立方体永远不会威胁到你，事实上，它是一个低音炮

> 原文:[https://hackaday . com/2011/09/10/the-weighted-companion-cube-never-threat-to-stab-you-in-fact-a-subwoofer/](https://hackaday.com/2011/09/10/the-weighted-companion-cube-will-never-threaten-to-stab-you-and-in-fact-is-a-subwoofer/)

![](../Images/c9ebaf2e508a9b4a50b4474a383fd31b.png "cube")

来自 techPowerUp！论坛来了[一个可怕的加权伴侣立方体低音炮](http://www.techpowerup.com/forums/showthread.php?p=2386648#post2386648)建立的转喻用户命名为[立方体]。

这个建筑一开始是一个咖啡桌，有一个椭圆形的玻璃桌面(没有字，如果边缘是蓝色或橙色)。立方体的内部取自一个 400 瓦的潜艇。正如任何一个好的子建设者会，[立方体]保持空气量和港口调整的捐助者盒。

在之前，我们已经看到过[一个配套的立方体潜艇，它以 EL 线为特色，具有“发光立方体”的效果，但是【cube】可能做得有点过火，在立方体的每一侧都包括了](http://modsandends.com/?p=285)[发光环](http://hackaday.com/wp-content/uploads/2011/09/glowing.jpg)。这些圆环由安装在有机玻璃片上的 2500 个发光二极管点亮，并通过电线点对点连接。虽然[Cube]声称自己“不是电子专家”，但他无疑有很大的耐心来组装这些灯。
