# 开源逻辑分析仪

> 原文：<https://hackaday.com/2009/07/23/open-source-logic-analyzer/>

[https://www.youtube.com/embed/kqwtzUUPqu8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kqwtzUUPqu8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[杰克·加塞特] [把他一直在研究的逻辑分析仪](http://www.gadgetfactory.net/gf/project/lax/)送进来。逻辑分析仪拥有一系列令人印象深刻的功能，它可以采样 100MHz 的 32 个通道，200MHz 的 16 个通道，SPI，UART，I2C 等。分析器的最大样本量目前是 4K，它支持 RLE 以减少内存消耗。分析仪通过 USB 连接到标准 PC 上的 java 客户端。基于 Xilinix FPGA 的[开源硬件](http://www.gadgetfactory.net/gf/project/butterfly_main/)可以以 100 美元的价格在[购买预组装，这使得它成为我们今年早些时候评论过的](http://shop.gadgetfactory.net/index.php?main_page=product_info&cPath=1&products_id=5)Salea 逻辑分析仪的直接竞争对手。
