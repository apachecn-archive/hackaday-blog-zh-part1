# 女服务员巴士跟踪系统

> 原文:[https://hack aday . com/2009/05/03/waitless-bus-tracking-system/](https://hackaday.com/2009/05/03/waitless-bus-tracking-system/)

![waitless](../Images/dd880220ef0e9b8ab3ad3ecc82d7501d.png "waitless")

校园里的公交系统常常令人沮丧。你站在车站等着，你不知道走着会不会更快。如果你在你的车站有一个[无等待跟踪系统](http://www.ece.gatech.edu/academic/courses/ece4007/09spring/ece4007l04/dk2/)，你可以准确地看到公共汽车在哪里，并使这个决定变得容易得多。该单位是自给自足的，太阳能，并配备了无线互联网。它以 Arduino 为核心，通过点亮地图上的 LED 来显示公交车的当前位置。休息之后你可以看到一段视频。
