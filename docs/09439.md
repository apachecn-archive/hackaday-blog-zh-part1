# 需要帮忙弄干手套吗？

> 原文:[https://hack aday . com/2012/01/06/need-a-hand-drying-these-gloves/](https://hackaday.com/2012/01/06/need-a-hand-drying-those-gloves/)

![](../Images/03699143cd09b7020bab08f5cc555ea3.png "glove-drier")

在雨中滑雪时，[安德鲁]只剩下一副湿淋淋的手套。让它们风干没有什么好处，因为 3 天后它们仍然是湿的，用风扇吹它们对这种情况没有什么帮助。幸运的是,[Andrew]已经想了一段时间用标准的管道配件来制造一个强制通风手套干燥机。

原型与消费者模型相似，手套套在管子的末端，虽然这对干燥手掌很有效，但对湿手指的情况没有任何帮助。为了解决这个问题，一个新的设计出现了，它有四个固定的手指和一个由铜管制成的可移动的拇指。稍微钻孔，焊接，然后金属手用胶带粘在吹风机的末端，在大约三个小时内将浸湿的手套变成完全干燥的手套。
