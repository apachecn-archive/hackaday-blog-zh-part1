# 把一个蹩脚的创玩具变成一个很酷的万圣节服装道具

> 原文:[https://hackaday . com/2011/08/27/converting-a-lame-tron-toy-into-a-cool-Halloween-costume-prop/](https://hackaday.com/2011/08/27/converting-a-lame-tron-toy-into-a-cool-halloween-costume-prop/)

![sweet_souped_up_tron_identity_disc](../Images/5d57431bee68a1dfd2b38ff7c6b6b92b.png "sweet_souped_up_tron_identity_disc")

Instructable 用户(cubeberg 的)的女儿在今年早些时候看了《创:遗产》,并当场决定她想在万圣节打扮成葵拉。作为一个令人敬畏的父亲，[他决定亲自为她制作服装](http://www.instructables.com/id/Custom-Tron-Disc-Mod)，并去商店寻找一张身份光盘来完成这个造型。

这个玩具相当乏味，缺乏一个合适的创道具应有的照明。他认为自己有能力让它变得更好，所以他收集了一些工具，一束 led，然后出发去他的工作室。他掏空了光盘，切掉了他能找到的任何无关的塑料碎片。他在光盘的内环和外环之间连接了 64 个 led，他使用 ATmega 328 和 Max7221 显示驱动器来控制这些 led。

他没有事先展示任何玩具的照片，但最终产品看起来很棒。我们打赌他的女儿一定很期待万圣节的到来——我们知道我们会的。
