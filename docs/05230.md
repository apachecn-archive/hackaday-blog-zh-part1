# 苹果 IIe 推特股票

> 原文：<https://hackaday.com/2010/04/24/apple-iie-twitter-ticker/>

![](img/1e34247c520fcd6c23ef9ecd6e7e24ba.png "twittering-apple-IIe")

一个手动输入引导程序和一个定制的通信协议让 [Apple IIe Twitter ticker 焕发生机](http://www.atomsandelectrons.com/blog/post/Apple-t.aspx)。[Chris Yerga]在跳蚤市场花 20 美元买了一台几十年的旧机器。刚刚完成他的 [TweetWall](http://hackaday.com/2010/04/20/twittering-pub-hanging/) 后，他决定将这个想法应用到 1 MHz 机器上。他手动输入一个 50 字节的引导程序，让他通过操纵杆端口将程序转储到内存中。从那里他用 USB FTDI 电缆给[装配了一个连接。现在，图像和文本由现代机器处理，并以 3600 波特的速度传送到 Apple IIe。休息过后，请观看实际操作。](http://hackaday.com/2009/09/22/introduction-to-ftdi-bitbang-mode/)
