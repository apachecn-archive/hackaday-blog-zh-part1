# 一个不需要花费一只胳膊和一条腿的假肢。

> 原文:[https://hacka day . com/2011/06/11/a-prosthetic-arm-that-cost-an-arm-a-leg/](https://hackaday.com/2011/06/11/a-prosthetic-arm-that-doesnt-cost-an-arm-and-a-leg/)

![](../Images/ce9e54baddf94795e7af02e3d95ee540.png "arm")

在一次摩托车事故后，他失去了一只手臂，他的儿子选择了不放弃。他们利用他们丰富的机械技能创造了一个替代的机器人手臂，使马丁能够开始做一些他最初安装的假肢无法做的简单事情。没有太多的细节，但似乎这个手臂有两个自由度，通过下巴控制来释放他的另一只手臂。对于任何对类似项目感兴趣的人，你应该看看[开放式修复项目](http://openprosthetics.org/ "open prosthetics")。广告之后有一小段视频讲述了马丁的故事。如果你没有手臂，造一个，在我们看来是完全合理的。干得好伙计们！我们期待看到下一个版本。
