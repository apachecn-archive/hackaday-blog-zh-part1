# PushingBox 提醒你你的猫的漫游习惯

> 原文:[https://hack aday . com/2012/03/09/pushing box-alerts-you-of-you-you-cats-roaming-habits/](https://hackaday.com/2012/03/09/pushingbox-alerts-you-of-your-cats-roaming-habits/)

![](../Images/b7441b47b88c9bb16bcfde2f0206a6c5.png "pushing-box-cat-door-notifications")

狗可能是人类最好的朋友，但猫肯定是黑客最好的灵感来源。他们为项目提供了如此多的“理由”，比如这个猫门，它被用来监视他的猫朋友的进进出出。他正在使用一种我们之前从未听说过的叫做 [PushingBox](http://pushingbox.com/) 的网络服务，从监控那扇门的 Arduino 发送通知，比如推文和电子邮件。

上图中贴在猫门上的两个白色矩形是通常用于入口门监控的磁铁。使用一对簧片开关可以让系统区分进来的猫和出去的猫。Arduino 与网络连接，运行 PushingBox API 来管理通知消息。休息之后，请观看视频中的系统演示。

这将是对我们看到的[迪诺]建造的猫门的一个很好的补充。当然，如果你真的想全力以赴与猫黑客的下一个项目应该是[一个 GPS 跟踪项圈](http://hackaday.com/2011/10/11/see-where-socks-has-been-using-a-gps-tracking-collar/)。 [https://www.youtube.com/embed/jxlVAa3VC5U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/jxlVAa3VC5U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
