# 圣诞老人徒步比赛的获胜者

> 原文:[https://hackaday.com/2011/02/28/santa-pede-contest-winners/](https://hackaday.com/2011/02/28/santa-pede-contest-winners/)

![](../Images/b5b3a88050b906b1562ad4c9fdd8cc4f.png "spw")

我们收到了几份精彩的参赛作品。我们笑着，哭着，困惑地盯着我们的显示器，这太棒了。现在我们在这里宣布圣诞老人比赛的获胜者。

如果你还记得的话，规则非常简单。找一个跳舞的圣诞老人(或者其他同样在节日跳舞的烦人的玩具)，把它拆开，用碎片做一些东西。我们意识到，其中大多数都有一个非常相似的设置，一个或两个伺服系统和一点塑料。这样每个人的起点都差不多。我们对用这么少的钱所做的事感到惊讶。

继续阅读，找出三个类别中的每一个:最过分的，没有多余部分的最佳步行机，和最佳演示。

**最佳步行者**得主，唯一使用可用零件的是【乔舒亚·杰克逊】。
[https://www.youtube.com/embed/VJWKxHpJ6i8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/VJWKxHpJ6i8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
他没有添加任何额外的部件，并设法获得了一个不错的步态。干得好。[约书亚]将会收到一件[薄荷糖](http://www.adafruit.com/index.php?main_page=product_info&cPath=10&products_id=14)和一件[hack adayt 恤 T10。](http://calebkraft.com/hackaday/index.php?route=product/product&product_id=49)

最佳展示奖的获得者是【Jonathan Thompson】和他的[弗兰肯斯坦圣诞老人](http://jethomson.wordpress.com/2011/01/31/franken-santa/)。

他的文章写得很详细，也拍得很好，他还花时间装饰他的圣诞老人，让他在人群中脱颖而出。他将收到一个[抽屉](http://www.adafruit.com/index.php?main_page=product_info&cPath=28&products_id=197)和一件[休闲 t 恤](http://calebkraft.com/hackaday/index.php?route=product/product&product_id=49)。

最夸张奖的得主是[【亚当·奥特勒】和他的愤怒的圣诞老人](http://forums.hackaday.com/viewtopic.php?f=3&t=100)，展现了他惊人的努力和坚韧。

[https://www.youtube.com/embed/J77CKDWHwD0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/J77CKDWHwD0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[亚当]全力以赴了。从一个实际上做得很好的快速行走器开始，然后继续添加一个具有良好网络界面的微控制器，允许从他的 android 手机进行控制。他的圣诞老人不仅有行走的能力，而且还有一种“狂暴模式”，在这种模式下，他四处狂怒，发射他的火箭发射器，直到你切断电源才能停止。干得好[亚当]，你绝对踢了这场比赛的屁股。
【亚当】将收到[大脑机器](http://www.adafruit.com/index.php?main_page=product_info&cPath=48&products_id=287)，以及 [Hackday 衬衫](http://calebkraft.com/hackaday/index.php?route=product/product&product_id=49)和[带发光灯泡的 LED 衬衫](http://calebkraft.com/hackaday/index.php?route=product/product&product_id=65)。
