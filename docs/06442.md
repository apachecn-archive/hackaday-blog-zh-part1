# 发明机器人运动员

> 原文:[https://hackaday.com/2010/12/14/inventing-robot-athletes/](https://hackaday.com/2010/12/14/inventing-robot-athletes/)

![](../Images/ce54e69d06c970f5d07be1a5c6979c3f.png "athletic-robot-leg")

从工程学的角度来看，人体是一个神奇的工具。复制它的一部分被证明是极其困难的，但是这些运动腿显示了很大的希望。这是一位名叫[Ryuma Niiyama]的日本研究人员的工作。他多年来一直致力于这项设计，现在正在使用气动执行器来模仿人类腿部的肌肉。腿的下部使用弹簧机构，类似于目前使用的一些跑步假肢。这些就像弹簧一样储存能量，并通过反弹到地面来重新利用。他试图教他的机器人使用这些腿；让它经历一个学习过程，这是使用大腿驱动器进行运动和平衡所必需的。休息后，我们惊讶于视频中的动作如此逼真。即使摔倒，动作也非常逼真。我们认为[小狗](http://hackaday.com/2009/07/17/little-dog/)的动作足够真实，令人毛骨悚然，这个机器人可能足够接近我们自己的言谈举止，以至于落入[恐怖谷](http://hackaday.com/2010/08/15/hackaday-links-august-15-2010/)。
