# 便携式灯箱，适合移动中的摄影师

> 原文:[https://hacka day . com/2011/10/05/便携式灯箱-供移动中的摄影师使用/](https://hackaday.com/2011/10/05/portable-light-box-for-photographers-on-the-go/)

![](../Images/701c58ab35af54a20b245a7838861269.png "portable-light-box")

为了让高质量的灯箱更便于携带，[Hharry]设计了一个可折叠的版本，配有可调节的侧灯。

轻型帐篷被摄影师用作拍摄小件物品的舞台。多种光源的使用，以及漫射它们的织物，意味着阴影的减少，否则可能会破坏照片。这种设计始于一个浅盒状的中密度纤维板底座，从左到右有一些挡板。抽屉滑轨将灯杆连接到这些挡板上，在搭建帐篷时，可轻松拉出四个光源。

舞台上的白色布料边缘缝有口袋，用来装传力杆。这些不是永久固定的。当楔入适当位置时，它们会拉动织物以保持帐篷拉紧，但很容易折叠起来存放在底座的空腔中。最后，手提箱的顶部向下折叠，一个抽屉拉手用作提手。

浅色帐篷并不是对抗照片中阴影的唯一方式。看看[这种利用镜子](http://hackaday.com/2010/08/25/take-better-pictures-of-your-projects/)调节照明条件的方法。
