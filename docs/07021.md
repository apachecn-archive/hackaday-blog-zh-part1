# 明蒂调频发射机

> 原文:[https://hackaday.com/2011/03/07/minty-fm-transmitter/](https://hackaday.com/2011/03/07/minty-fm-transmitter/)

![](../Images/15522f1ffe9c752f9eeced2386e6ab67.png "complete_test_board")

[Chis]想要一个装在薄荷罐里的调频发射机窃听器,这正是他做的。它只有 17 个独立的部件，使用 3 伏硬币电池，体积小得足以放入塑料薄荷盒中，但仍有足够的空间放薄荷糖。

使用简单的设计，音频通过驻极体麦克风进入设备，并撞击晶体管进行放大，然后信号被传递到电路的振荡器部分，该电路采用 LC 槽式设计，产生载波频率，并将其与室内约 100 英尺范围内的信号混合。

如果你想知道晶体管、导线和电容如何构成 FM 发射器，或者如果你想完成最终项目，还提供原理图、pcb 文件和组装说明，每一部分都被分解为多个步骤，并通过动画、理论和仿真进行详细说明。
