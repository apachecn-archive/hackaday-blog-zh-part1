# 发光的接线电缆

> 原文:[https://hackaday.com/2009/03/14/glowing-patch-cables/](https://hackaday.com/2009/03/14/glowing-patch-cables/)

[https://www.youtube.com/embed/zi3lpVzM-bM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zi3lpVzM-bM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Sleepydog]刚刚发来了这个他用内置 [EL 线](http://en.wikipedia.org/wiki/El_wire)制作的跳线[酷视频](http://www.youtube.com/watch?v=zi3lpVzM-bM)。他使用以太网供电路由器来控制哪些端口有电。他表示，这将使在混乱中识别特定电缆变得容易。虽然概念验证看起来完全可行，想法也不错，但我们不得不怀疑投入所有额外硬件的成本是否值得。每根电缆都必须有自己的逆变器，这不仅增加了成本，还可能增加干扰。这并不意味着我们不想这么做，我们想。但是我们想要它只是因为它看起来很酷。他现在需要编排一些音乐，让他的整个服务器房间成为一个别致的展示。
