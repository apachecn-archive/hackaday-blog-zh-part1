# 引擎黑客:添加燃料喷射到骑割草机

> 原文:[https://hackaday . com/2011/08/30/engine-hacks-adding-fuel-injection-to-a riding-lawn mower/](https://hackaday.com/2011/08/30/engine-hacks-adding-fuel-injection-to-a-riding-lawnmower/)

你是否厌倦了每年春天摆弄草坪设备上的引擎，试图让它们再次运行？约翰决定做点什么，把他的骑马剪草机改装成燃油喷射。在从他的货车上偷了一个燃料喷射器并用螺栓固定在割草机的汽化器上后，他撒了一点 Arduino 魔法。通过少量的代码调整，他已经准备好~~推出~~ mow。你可以找到一个原理图和他的代码，如果你跟随论坛链接到第二页。

要了解更多燃油喷射的优点，请查看这个项目，在这个项目中，史蒂夫升级了他的奥斯汀希利，使用通用汽车的燃油喷射系统。这个构建日志非常全面，显示了系统中的每个元素，描述了它的作用，并显示了它应该去哪里。如果你是一个汽车迷，这绝对是一个检查。

最后，如果你想真正了解汽车燃油喷射的细节，看看 [Megasquirt 燃油喷射计算机系统](http://www.bgsoflex.com/megasquirt.html)，它可以让你根据具体的汽车和型号进行微调。
