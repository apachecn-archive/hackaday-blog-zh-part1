# 五款概念鼠标增加了多点触控功能

> 原文：<https://hackaday.com/2009/10/06/five-concept-mice-add-multi-touch-control/>

</span> <p>微软展示了增加鼠标功能的五个概念。它们都试图通过电容传感、视频识别、传感器连接或激光扫描，用触摸灵敏度取代传统的移动和点击。我们对其中一些功能的前景感到兴奋，但同时也想知道这对这种被滥用的外设的价格有何影响。休息之后，我们将讨论每一款设备，以及上面嵌入的视频的时间参考。<span id="more-16786"/></p> <p><strong>电容式鼠标(0:00) </strong></p> <p>第一个产品是最接近我们现在拥有的。这是一个标准的鼠标，有一个电容垫覆盖身体。这个小键盘检测并报告任何触摸，通过手接触的屏幕上的斑点来证明。我们认为这是一个笔记本电脑触摸板包裹着一个桌面鼠标。包起来运出去，我们要一个！</p> <p><strong>视频多点触控鼠标(1 分 52 秒)</strong></p> <p>下一个概念是使用一片弯曲的丙烯酸树脂作为你通常会找到按钮的部分。感应的秘密是一个内置的摄像头，它将图像数据传送给计算机。通过分析图像数据来提供触摸灵敏度。我们有点怀疑这一过程是否能在老鼠体内完成，但我们认为这是 R&amp;D 的研究人员要解决的问题。此外，丙烯酸树脂需要有多干净才能获得良好的处理图像？</p> <p><strong>关节式鼠标(3:28) </strong></p> <p>绝对是这群人中最奇怪的。这是三合一的鼠标，一个传统的鼠标传感器放在你拇指的关节垫下，另一个放在你的食指上。第三个传感器在你手掌所在的鼠标体上。触觉按钮可以添加到两个卫星垫。我们确信有人对此有很大用处，但它可能不会比轨迹球更受欢迎。</p> <p><strong>圆球鼠(4:56) </strong></p> <p>球体是另一种基于相机的设计。它使用了一个圆顶形的片，而不是弯曲的丙烯酸片。视频分析也存在同样的问题，但请务必观看用作单手 FPS 控制器的演示。</p> <p><strong>侧鼠标(6:34) </strong></p> <p>这款鼠标可以让你的桌子变成多点触控区域。它使用红外激光扫描来采集鼠标主体周围的手指按压和运动。这让我们想起了<a href="http://www.virtual-laser-keyboard.com/" target="_blank">激光 qwerty 键盘</a>，可能也一样酷，谁知道呢？我们预计使用这种设备需要一定程度的学习过程。</p> <p><strong>结论</strong></p> <p>我们希望看到新的和改进的输入设备以低廉的价格上市。这些目标可以实现吗？这真的是在发明一种更好的老鼠(陷阱)。</p> <p>[via <a href="http://mashable.com/2009/10/06/microsofts-multitouch-mice/" target="_blank"> Mashable </a></p> <p>[谢谢亚历克斯]</p> </body> </html>
