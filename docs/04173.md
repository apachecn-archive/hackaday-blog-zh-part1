# 万圣节道具:低成本弹出

> 原文：<https://hackaday.com/2009/10/05/halloween-props-low-cost-popup/>

[https://www.youtube.com/embed/XJoedH9thds?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/XJoedH9thds?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Backroads]整理了这篇非常详细的文章，解释了如何制作一个低成本的弹出道具。他用一个气动阀和一个自制的聚氯乙烯活塞来升降一个可怕的面具。他使用现成的 110 伏交流阀门控制器来控制阀门。一盏闪烁的灯、一个“尖叫者”和一台造雾机帮助完成了这个项目。结果相当不错。我们可能会在它前面放置一个压力传感器来优化惊吓时间。
