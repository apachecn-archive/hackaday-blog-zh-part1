# 视频最差的传输方式是什么？

> 原文:[https://hack aday . com/2009/12/04/what ' s the worst-way-to-transmit-video/](https://hackaday.com/2009/12/04/whats-the-worst-way-to-transmit-video/)

![](../Images/13dd83654a9f36e5d9606f26a1d7aaba.png "worst-video-transmission")

也许传输视频最糟糕的方式是并行发送所有像素数据。这正是[Gebhard sengüller 的] [并行图像设备](http://www.gebseng.com/08_a_parallel_image/)的工作原理。公平地说，这是一件名为“平行影像”的艺术作品，讲述了如果串行数据传输从未实现，我们会在哪里的概念。伴随展览的[宣传册](http://www.gebseng.com/08_a_parallel_image/a_parallel_image_brochure.pdf) (PDF)给出了我们一直在寻找的有趣细节。

该装置由光传感器单元和显示单元组成。两者都是由环氧树脂制成的一平方米的条板面板。传感器单元由一个 50×50 网格的光导体组成，光导体与显示单元上 50×50 网格的灯泡相对应。当你扔进连接 2500 像素传感器单元和显示单元的 7500 米长的磁线时，事情开始变得相当疯狂。

你应该能弄清楚这是如何工作的。传感器接收光线，然后影响相应灯泡的亮度。结果是一个有趣的图像，一个会让任何电视修理工喝酒的网络色情噩梦。
