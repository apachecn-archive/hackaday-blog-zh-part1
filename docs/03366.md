# 雅达利 2600 的 s 视频

> 原文:[https://hackaday.com/2009/04/05/s-video-from-an-atari-2600/](https://hackaday.com/2009/04/05/s-video-from-an-atari-2600/)

![atari](../Images/c6e8eb27a9e981d5168a8fbd73020655.png "atari")

[Ben Heck]发布了这篇关于从 Atari 2600 中获取 S-Video/composite 的文章。这实际上是[长角牛工程师]的黑客，他在最近的一次活动中向[本]展示了它。如果你们中的任何人试图在现代电视上播放这些经典，你可能会发现这很难。如果你设法通过适配器之类的东西把它连接起来，你可能仍然会有视频问题。这完全缓解了这个问题。在你焊接这个之后，你的 Atari 就有了原生的复合/S-video。正如你在休息后的视频中看到的，它似乎工作得相当好。
