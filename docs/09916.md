# 时尚导致了精神控制的裙子提升装置

> 原文:[https://hacka day . com/2012/03/16/fashion-leads-to-mind-controlled-skirt-lifting-syntaction/](https://hackaday.com/2012/03/16/fashion-leads-to-mind-controlled-skirt-lifting-contraption/)

![](../Images/29d7f75837ff9ea31aab34e449be1655.png "mind-controlled-skirt-lifting-conraption")

这一定是世界碰撞的一个例子。谁能想到 Mindflex 和 Arduino 的极客风格能进入高级时装领域？果然，[这件衣服是根据模特](http://vimeo.com/38547109)的精神集中而变形的(这里必须忍住不开玩笑)。

细节有点稀疏，但你可以在休息后看看视频中的原型。没有裸体；一条较大的裙子遮住了一个更朴素的版本。那个裙边连接着某种由 Arduino 驱动的马达系统。当帽子中的脑电图传感器检测到一定程度的脑电波活动时，外裙被提起并拉到衣服的后面，露出下面更紧的版本。

[Lorenzo]来信分享了这个服装黑客的链接。他提到 Lilypad 和 Mindflex 在这里工作。深入查看这位艺术家的网站,我们发现这并不是唯一生产的科技服装。有一种孕妇装备可以感知婴儿的心跳，并收集关于母亲和婴儿的其他数据，以及其他一些数据。

我们认为这种日常装备不会有太大的前途，但是[更实用的版本](http://hackaday.com/2008/06/22/turn-signal-jacket-how-to/)已经出现了，所以我们认为可穿戴技术的前途无量。
