# 粘连使物体感觉很响

> 原文:[https://hacka day . com/2011/11/18/synescage-makes-objects-feel-loud/](https://hackaday.com/2011/11/18/syneseizure-makes-objects-feel-loud/)

联觉是大脑线路的混乱，感觉输入的感知不同于“正常人”通常的体验。有联觉的人可以将视觉输入映射到大脑中的听觉感知，或者在一条高速公路上行驶，在那里最近有一只臭鼬被撞死，可以闻到“响亮”的气味。这是一种有趣的感知世界的方式，通常大多数人都无法接触到，但[synescage](http://www.youtube.com/watch?v=BhmfGMqn-8o)试图复制这种看待世界的方式。

有很多类型的联觉(齐柏林飞船感觉是紫色的，或者苹果比葡萄闻起来更远)，但是[Greg]和他的团队需要选择一种亚型来降低他们项目的复杂性。他们选择将视觉输入映射到触觉。这是通过将十几个扬声器贴在测试对象的脸上来实现的。一个摄像头记录下了被摄对象正在看的地方，通过处理草图，摄像头被缩小为 4×3 像素的灰度网格。每个像素的强度对应于每个扬声器的嗡嗡声强度。剩下要做的就是给实验对象戴上面具，让他们四处走动。

syne search 是为旧金山科学黑客日建造的，最终赢得了人民选择奖。项目网站上有一堆照片和一篇很棒的文章，所以一定要去看看。
