# 自动平衡传输由 Arduino 控制

> 原文：<https://hackaday.com/2012/03/16/self-balancing-transport-is-arduino-controlled/>

[Nick Thatcher]已经制造了几个版本的自制赛格威，最新的版本令人印象深刻。在开发这个项目的时候，他认为这个东西根本不可能工作，这就导致了它的名字，No-way。

休息过后，你可以观看[尼克的]试乘视频。看来这辆两轮车已经可以日常使用了。你只能在聚碳酸酯机身的右侧辨认出一个红色的关闭开关。如果事情失去控制，或者当你骑完它的时候，你可以切断电源。但也有一个安全开关，我们认为它使用两个传感器，你的脚放在外壳的顶面上。手柄上有一些内置的指示灯，每个拇指下还有用于转向的按钮。控制电路包括一个 Arduino UNO，它从 SparkFun 读取陀螺仪/加速度计传感器板。两个 7.2 Ah 电池为一对电动滑板车电机提供 24V 电压，用于驱动手推车车轮。

我们喜欢看这些赛格威克隆项目。所以，如果你正在做你自己的，不要忘记记录你的进展！
