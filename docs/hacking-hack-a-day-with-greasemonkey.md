# 用 Greasemonkey 黑进一天

> 原文：<https://hackaday.com/2012/02/25/hacking-hack-a-day-with-greasemonkey/>

自从 Hack a Day 于 2004 年首次出现以来，该网站的设计一直非常一致。黑色的背景，绿色和白色的文字，虽然看起来有点可疑，但对我来说很好。然而，对其他人来说，网站的设计无论从形象上还是字面上都是一个持续的眼中钉。[詹姆斯·利顿]是这些读者中的一员，他写了一篇文章来分享一个技巧，这个技巧可以帮助他阅读最新的黑客技术，而不会伤害他的眼睛。

[詹姆斯]使用火狐浏览网页，所以他创建了一个小的 Greasemonkey 脚本，一旦到达浏览器，就可以调整 Hack Day 的样式表。他的剧本颠倒了背景，同时改变了一些其他的项目，使得阅读更加舒适。总的来说，我们认为这个变化是相当合理的，但是你自己去判断吧——你可以在他的网站上看到更详细的前后截图。

[James]还指出，对于那些更喜欢 Chrome 浏览器的人来说，这个脚本在 Chrome 上应该可以很好地工作。

所以，如果你的眼睛有点敏感，请随意拿起他的剧本并进行定制——我不认为我们会很快改变主题。
