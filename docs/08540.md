# 旋转紫外光刻录机

> 原文:[https://hackaday.com/2011/09/09/spinning-uv-light-writer/](https://hackaday.com/2011/09/09/spinning-uv-light-writer/)

![uv_light_writer_ring](../Images/601a97500a1fa160f2d1908cd63d95f5.png "uv_light_writer_ring")

自从在 Make 上看到一篇关于紫外发光二极管的文章后，他就一直想用紫外发光二极管在磷光表面上写字。他无意中购买了紫外线发光二极管，当他打算购买紫色的，所以他认为他的错误是所有的理由，他需要给紫外光写作一个尝试。

他使用 led 制作了一个 PIC16F628 UV POV 板，虽然手动在黑暗表面的各种辉光上滑动书写器很酷，但他希望 POV 板保持静止，而不是移动书写介质。他买了一些磷光塑料，但发现它不太灵活，这意味着他不能使用传送带方式来展示。有一天，他突然意识到一个乙烯基环可能会工作得很好，于是他用一个旧卡带播放器的马达，建造了你在上面看到的紫外线记录器。

尽管在紫外线环上有一个小缺陷，但它似乎工作得很好，虽然[大卫]对结果很满意，但他已经为第二次修订准备了很多想法。
