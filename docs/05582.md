# 微控制器访问网络

> 原文：<https://hackaday.com/2010/07/01/microcontrollers-access-the-web/>

[https://www.youtube.com/embed/ExoJDxBGEBU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ExoJDxBGEBU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[肯尼斯·芬尼根]带着另一个视频回来了，展示了他正在做的一些很酷的事情，将他的微控制器连接到互联网上。通常，我们看到像 iobridge 这样的预建模块可以做到这一点。[Kenneth]正在使用一个微芯片 ENC28j60 模块进行通信，他设法把它全部塞进一个小小的[电路板](http://electroboards.com/)上。[肯尼斯]开始成为[这里的常客](http://hackaday.com/?s=finnegan)。
