# 谁知道雅各布的梯子有这么多选择？

> 原文：<https://hackaday.com/2011/08/17/who-knew-jacobs-ladder-builds-had-so-many-options/>

[掷弹兵]对高电压情有独钟，正如你所见，他找到了多种方法来建造 HV 玩具的图标-一架雅各布的梯子。

这三种看起来很相似，但它们使用不同的方式产生电压，这种电压是让火花在空气中跳跃所必需的。左边的展示使用了霓虹灯变压器，中间的展示基于 x 光机的变压器，右边的展示使用了微波炉变压器。[Grenadier]讨论了每种方法的优缺点，然后链接到他关于使用每种方法的深入文章。也有三者的视频。休息之后我们已经嵌入了微波炉变压器的视频。那个版本的雅各布之梯需要某种方式来启动火花，在视频中他是手动进行的。总是可以选择增加一个螺线管来完成这项工作，但他确实提到，你不能只是让它运行，因为廉价的变压器不久就会烧坏。

如果你喜欢你在这里看到的，也许当[他跑过一些汽水罐](http://hackaday.com/2011/06/14/turning-beer-cans-into-bells-with-35kv/)时，你会很开心。
