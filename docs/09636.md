# DIY Windows 8 平板电脑

> 原文:[https://hackaday.com/2012/02/06/diy-windows-8-tablet/](https://hackaday.com/2012/02/06/diy-windows-8-tablet/)

![](../Images/caa41f3abe9ff73c44f56e8f74b661a9.png "Untitled")

来自 instructables 的[hackitbuildit]为我们带来了一款 [DIY windows 8 平板电脑](http://www.instructables.com/id/Windows-8-Tablet/)。为了制作平板电脑，使用了一台符合 windows 8 预览版最低要求的旧笔记本电脑，一个触摸屏转换套件，当然还有软件本身。笔记本电脑首先是通过移除屏幕周围的外壳来准备的，如果你只看图片，看起来有点像他在把它拆开！但是如果你看视频，螺丝被移除了。

屏幕翻转过来，放在键盘上，中间有几个垫片，因为许多笔记本电脑将键盘区域用作散热器。安装触摸屏，外面热粘一些木条，填补屏幕和底座的缝隙。只需一点点油漆，你就能得到一个大而实用的 windows 8 平板电脑，并开始开发。
