# 使用 Arduino 恢复自动存储塔

> 原文:[https://hack aday . com/2011/08/17/restoring-a-jukebox-with-an-arduino/](https://hackaday.com/2011/08/17/restoring-a-jukebox-with-an-arduino/)

[Jim]刚刚使用 Arduino 为他的父亲修复了一个旧的 Seeburg USC1 点唱机,替换了一个机电老鼠窝电线。这一堆 45 张唱片换成了带 [Sparkfun MP3 播放器保护罩](http://www.sparkfun.com/products/10628)的 Arduino Mega 2560，点唱机的灯光现在由 74595 个移位寄存器控制。因为他的点唱机不收钱，美元纸币识别器被修改成一个“跳过歌曲”按钮，当点唱机队列中没有歌曲时，SD 卡上有 500 首额外的歌曲将随机播放。

我们以前见过[吉姆]的一种体型。今年早些时候，他使用相同的 Arduino + MP3 shield 设置修复了一台三十岁的弹球机。看起来[吉姆]很擅长修复笨重的旧电子产品。点唱机的修复很棒，比我们仍在投入资金的[联网触摸屏怪物](http://www.youtube.com/watch?v=VH27fhqNGc0)更有档次。
