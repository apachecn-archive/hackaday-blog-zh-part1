# 触觉 4 位迷宫

> 原文:[https://hackaday.com/2011/10/17/tactile-4-bit-maze/](https://hackaday.com/2011/10/17/tactile-4-bit-maze/)

![](../Images/6edd036b9eac9b04661c45be435eb673.png "maze")

[奥斯卡]做拼图已经有一段时间了。2000 年，他做了一个小型机电 4 位迷宫，玩起来真的很有趣。不过最近，他一直在研究一个改进版本，这可能是一个商业产品的开端。

早期的[机电迷宫](http://clickmazes.com/fourbit/ix4bit.htm)(你可以在页面上的小程序中玩它)只是一个连接到电磁铁和开关的微控制器。要完成这个迷宫，找到将所有东西从 0 移动到 1 的位的模式。这有点像[的狐狸鸡纹拼图](http://tvtropes.org/pmwiki/pmwiki.php/Main/FoxChickenGrainPuzzle)，只是稍微复杂一点。

[Oskar]的最新版本使用电动推子来表示比特的 0 和 1 状态。机电版同样的逻辑在最新版。Ardunio 负责运动控制和游戏逻辑。

作为一个微小的逻辑游戏玩具，这是一个伟大的想法；每个人在一生中的某个时候都需要亲自动手操作卡诺图。查看下面的视频，观看 4 位迷宫的演示。
