# 深入 ZigBee 网络

> 原文:[https://hackaday.com/2012/02/20/get-neck-deep-into-zigbee/](https://hackaday.com/2012/02/20/get-neck-deep-into-zigbee/)

![](../Images/41250c71e596efb1164920a768b040a1.png "zigbee-tutorial")

这里有一个庞大的教程，将[完善你对 ZigBee 无线通信](http://webdelcire.com/wordpress/archives/1714) ( [翻译](http://translate.google.com/translate?hl=en&sl=auto&tl=en&u=http%3A%2F%2Fwebdelcire.com%2Fwordpress%2Farchives%2F1714))的理解。该协议非常适合业余电子项目，因为它使用低功率短程无线设备来构建网状网络。该指南涵盖了硬件和软件，但也花时间解释了硬件在后台做什么。

正如您所看到的，XBee 模块的几种不同版本被用作示例。它们几乎都依赖于一系列 SparkFun 分线板，每个分线板都有不同的用途。一旦您获得了这些模块，就需要有相当多的选择来配置它们，使它们能够很好地相互配合。我们阅读了教程的大部分内容(我们将把剩下的留到以后再享受)，即使没有硬件或者不能使用我们所学的界面，我们也没有任何问题。

每当我们谈到 XBee 模块时，我们总是喜欢提到，将这些模块用于具有[的远程传感器非常容易，不需要额外的微控制器](http://hackaday.com/2010/12/14/garage-monitor-has-an-extra-arduino/)。
