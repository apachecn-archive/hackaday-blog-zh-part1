# 三位二进制时钟

> 原文:[https://hackaday.com/2011/02/20/three-digit-binary-clock/](https://hackaday.com/2011/02/20/three-digit-binary-clock/)

![](../Images/8cb29f12d84bcb64f659b9163aa91c69.png "binary-clock")

这是维克多设计的三位数二进制时钟。它利用 PIC 16F628A 一次驱动一个数字。休息后的视频显示它在滴答作响，用蓝色 led 显示小时、分钟和秒。您可能想知道为什么这些 led 没有与电路板齐平？[Viktor]在这个项目上比大多数二进制时钟项目更进一步，设计了一个 PCB 以适应旧的笔记本电脑 PSU 的外壳，然后制造了电路板。有了[像 DorkbotPDX groups orders](http://hackaday.com/2010/08/01/hackaday-links-august-1-2010/) 这样的选项，做这件事已经变得相当便宜，而且当你需要为未来的超级奇妙项目设计一个高度复杂的电路板时，这确实是一个很好的实践。
