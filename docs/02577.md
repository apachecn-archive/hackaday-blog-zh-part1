# DIY 垂直轴风力发电机

> 原文：<https://hackaday.com/2008/09/28/diy-vertical-axis-wind-turbine/>

![](img/d78f585c3687bab24252b6fc7105408d.png "dyi-wind-turbine")

这个[垂直轴风力涡轮机](http://www.instructables.com/id/V10_vawtsavoniusvertical_axiswindturbineamete/) (VAWT)使用五个切成两半的 3”PVC 管，用于在三个儿童自行车轮上旋转的叶片，以旋转一个 [Ametek 38 伏马达](http://www.windstuffnow.com/main/ametek_motor.htm)或一个[风蓝色交流发电机](http://www.windbluepower.com/category_s/1.htm)。整个东西在一个 12 英尺高、2 英尺见方的盒子里旋转，这个盒子可以放在他的甲板上。总共花了他大约 125 美元加上时间，如果你使用风蓝色交流发电机多一点。
