# 任天堂把 nes 和 snes 塞在同一个盒子里

> 原文：<https://hackaday.com/2011/10/26/nintenduo-stuffs-an-nes-and-snes-in-the-same-case/>

![](img/7415c875cdcd9ae32870baf93dcb9abc.png "nintenduo")

[lovablechevy]喜欢她的复古双人控制台，特别是因为它比它取代的 NES 和 SNES 占用的空间少。不过有一个小问题:复古二人组与她的旧任天堂墨盒不是 100%兼容。对她来说，battle toad 是一个交易破坏者，所以她建造了[任天堂](http://forums.benheck.com/viewtopic.php?f=18&t=44383)，这是一个 NES/SNES 的游戏机，使用了所有原始的任天堂硬件。

邦多女王开始了她的项目，从顶部装载的 NES 和较小的 SNES 开始。有一个[摄影画廊](http://s246.photobucket.com/albums/gg107/sharonrenees/nintenduo%20-%20tssap/)展示了可爱地放置在他们新的塑料家中的内脏。

[lovablechevy]不仅可以播放经典的*报童*、*大金刚国家*和*超级马里奥 RPG* 与复古二人组[不兼容](http://en.wikipedia.org/wiki/Retro_Duo)，所有的配件如 Zapper 和电源垫现在都可以工作。

完成的构建非常小；比 SNES 2 大不了多少，和巨大的 NES 子弹相比简直是小巫见大巫。她上传了一段视频，视频中她试图用她的任天堂游戏机不去拍那只愚蠢的猎鸭狗。休息之后来看看。
