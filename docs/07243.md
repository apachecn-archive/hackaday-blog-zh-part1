# 超级乒乓球台让乐趣倍增

> 原文:[https://hack aday . com/2011/03/29/super-pong-table-doubles-the-fun/](https://hackaday.com/2011/03/29/super-pong-table-doubles-the-fun/)

![super_pong](../Images/339ad1652d5d30958c41ad029f3e7b34.png "super_pong")

虽然乒乓球传统上是两个人之间的游戏，但 Instructables 用户[Brad]已经将[组合在一起，使乐趣倍增](http://www.instructables.com/id/Super-Pong-Coffee-Table)。他的 Pong 咖啡桌可以同时容纳四个人，成为客厅的焦点。

该桌子由中密度纤维板制成，包含 900 个发光二极管，全部由 PIC18 微控制器控制。MCU 安装在他设计的控制板上，还有驱动 led 阵列所需的其他附加位。一套旧的 Atari paddle 控制器被拆卸下来，安装在桌子周围，这是一次真正的复古 Pong 体验。

正如你在视频中看到的，动作相当疯狂。在游戏结束之前很难判断谁是赢家，但[布拉德]说记分牌将在未来的修订版中出现。
