# 光敏 MIDI 手套

> 原文:[https://hackaday.com/2011/02/22/light-sensitive-midi-glove/](https://hackaday.com/2011/02/22/light-sensitive-midi-glove/)

![](../Images/1b08910bb3d6fe4bb73d74facff34ba1.png "light-sensitive-midi-glove")

基于手套的噪声发生器的最新产品放弃了常用的柔性传感器，转而采用光敏电阻。[Bruno Ratnieks]负责[这个音乐手套](http://www.instructables.com/id/SNF-Drumming-MIDI-Glove-using-Arduino-and-light-se/)，他的方法将很容易重现。他使用 Arduino 与它连接，同时提供与音频软件的 USB 连接。传感器本身再简单不过了，当每个光敏电阻与一个固定值的电阻结合时，就会产生一个分压器。这就是你需要的所有硬件，通过一些创造性的编码，你可以让它做得比休息后在视频中听到的效果更多。

有些人会说[布鲁诺]只是在他的项目设计中没有使用足够多的胶带。我们喜欢他如何将金属丝编织进这些针织手套的网眼中，使其牢牢固定。
