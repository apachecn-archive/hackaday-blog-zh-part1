# Snega2usb 更新:Usb Snes 和世嘉墨盒阅读器

> 原文：<https://hackaday.com/2009/09/07/snega2usb-update-usb-snes-and-sega-cartridge-reader/>

[https://www.youtube.com/embed/QNBg_jWjBmI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QNBg_jWjBmI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当我们第一次发布[Matthias_H]的用于 SNES 游戏车的[USB 阅读器时，它受到了热烈的欢迎。snega2usb 允许您在 pc 上玩 SNES 和世嘉游戏。最新的修订版比第一版更令人惊叹。[马蒂亚斯]增加了读取](http://hackaday.com/2009/06/19/usb-reader-for-snes-game-carts/)[世嘉 Genesis/Mega Drive](http://en.wikipedia.org/wiki/Sega_genisis) 卡带的能力，以及将游戏直接保存到卡带的能力。该板也已从过去的鼠窝升级为外观智能的双面 PCB。到目前为止，马蒂亚斯还没有遇到任何读取墨盒的问题，即使是带有 [SuperFX](http://en.wikipedia.org/wiki/Super_FX) 芯片的墨盒。[Matthias]还为项目开通了一个网站，在那里可以找到关于项目发展的最新信息。[马提亚斯]正在接近生产版本，它将具有更好的固件，控制台质量连接器和闪亮的外壳。
