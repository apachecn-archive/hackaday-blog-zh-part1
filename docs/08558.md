# 玻璃印刷电路板

> 原文:[https://hackaday.com/2011/09/12/glass-pcbs/](https://hackaday.com/2011/09/12/glass-pcbs/)

我们的朋友[Jeri]向我们透露了 youtube 上这个很酷的视频，视频中作者使用一些非传统材料制作了自己的“透明”PCB。在网上可以找到一盎司的铜箔和一些透明的显微镜载玻片，然后用一些薄的紫外线固化胶将铜箔粘在载玻片上。一旦这两个部分匹配，整个事情是弹出到一个 eeprom 擦除其强烈的紫外光，然后多余的修剪。

墨粉转移的正常程序用于将电路图案复制到包铜玻璃上，并在标准氯化铁中蚀刻。铜被移除了，但是固定它的 UV 胶仍然存在，需要特别注意，因为这种材料对温和的溶剂都很脆弱，你不希望你的痕迹剥落。接下来，不用干净的焊锡膏，用热风枪焊接零件，保持玻璃均匀受热以防止破裂。这就给你留下了一块看起来像磨砂玻璃的板，为了在消除磨砂效果的同时保护胶水，使用了一些聚氨酯来填充所有的小突起，并使表面光滑到几乎 100%透明。
本视频中的最终应用是一个触摸感应板，它可以透过玻璃背面正常工作，并为用户提供一个光滑的界面。视频休息后加入我们。
