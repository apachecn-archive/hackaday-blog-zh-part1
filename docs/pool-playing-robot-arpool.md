# 撞球机器人+ ARpool

> 原文：<https://hackaday.com/2009/09/20/pool-playing-robot-arpool/>

[https://www.youtube.com/embed/AENJxqR0g48?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/AENJxqR0g48?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

喜欢在台球上输吗？嗯，皇后大学的人让这一切变得简单多了。[深绿色机器人](http://rcvlab.ece.queensu.ca/~deepgreen/DeepGreen/main.html)被创造出来的目的是玩一个完美的游戏，让它能够打败最熟练的人类玩家。关于这个项目已经写了不止一[两篇](http://rcvlab.ece.queensu.ca/~greensm/papers/ACG11.pdf)的研究论文。一个安装在天花板上的[佳能 350D](http://hackaday.com/?s=350d) 跟踪所有球的位置，此外还有另一个安装在球杆上的凸轮(为了更高的击球精度)。使用一堆计算和一台电脑(可能比 Arduino 更先进的[),深绿能够制定战略和进行游戏。很好。](http://hackaday.com/category/arduino-hacks/)

积极的一面是，来自皇后区的另一个团队正在从事一个独立但[相关的项目:ARpool](http://rcvlab.ece.queensu.ca/~jordans/arpool.php) (就像在增强现实中一样)。它的创建是为了让打台球变得更容易。该网站没有提供太多的信息，但它似乎将不同的动作投射到台球桌上，让一个没有经验的球员知道是否有可能击球。
