# BBox MIDI 鼓序列器

> 原文：<https://hackaday.com/2009/09/14/bbox-midi-drum-sequencer/>

![bbox](img/c34493213795cfb0ae390e517a0a9dd1.png "bbox")

在之前，我们已经[讨论过](http://hackaday.com/2008/07/15/pspseq-30-psp-sequencer-release-and-interview/)[，但是读者【Johan】发来了他最新的项目，这是一个更简约的方法。被称为 BBox](http://hackaday.com/2009/06/29/d-touch-tangible-drum-interface/) 的他将他的鼓发生器基于 Arduino 和 LCD 显示器。Arduino 不是合成声音，而是输出 MIDI，然后由他的 Roland Juno-D 解释。在构建该设备时，他使用了我们最喜欢的技巧[来保持界面整洁。然后他找到了一个很棒的香蕉盒作为案例。虽然，这个项目可能没有其他项目那么实用，但它确实很有天赋。休息后的视频。](http://hackaday.com/2009/01/29/led-push-buttons/)
