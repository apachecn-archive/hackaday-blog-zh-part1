# 牧田电池组修理

> 原文:[https://hackaday.com/2010/01/12/makita-battery-pack-repair/](https://hackaday.com/2010/01/12/makita-battery-pack-repair/)

[Rob]厌倦了他的 Makita 电动工具电池组的老化，所以他想出了自己修理的方法。广告之后的视频向我们展示了从打开箱子开始的整个过程。里面有一个控制板和一个十节电池。[Rob]已经确定这些电池故障只发生在第一节电池上，这是通过用万用表测量电池电压来确认的。演示电池中的第一个电池读数为零伏，需要更换。出于某种原因，他手头上有很多这样的电池，在视频的最后，他展示了一大块提供半千瓦小时电力的电池。

为了完成复活，他将控制电路从集成 PCB 上拆下。似乎是电池的 PCB 上的微控制器监控它，并在它认为该单元的寿命已经结束时将它们砖块化。通过破解充电器，他现在可以对改造后的电池组进行平衡充电，并在它们进入垃圾填埋场之前得到更多的利用。
