# 倒数到新年

> 原文:[https://hack aday . com/2011/12/26/新年倒计时/](https://hackaday.com/2011/12/26/counting-down-to-the-new-year/)

![](../Images/c1b3ddd6e7a973b95c34aa532aea4d53.png "cardboard-count-down")

[Rod]可以在新年前夜提供社区服务。他在海边的一个亲戚家过夜。午夜会有烟火，但是人群没有办法倒数到最后期限。今年，他建造了自己的倒计时显示器，这样每个人都可以在最后几秒加入进来。

这是一个临时建筑，所以每个数字都装在一个纸板盒里。[Rod]首先在正面画出七段数字的轮廓，然后在每个段中添加三个 led 的孔。他用 12V 的电压给这些段供电，因此能够和一个电阻一起串联运行发光二极管，用一个晶体管切换每个段。他选择了一个 Arduino 来驱动显示器，因为他周围有两个，所以他使用两个，而不是抓住一个移位寄存器作为端口扩展器。

如果你正在寻找一个更持久的解决方案，我们真的很喜欢为这个记分牌设计的[数字显示。但是如果你想在节日前完成的话，你必须自己蚀刻木板。](http://hackaday.com/2010/07/30/scoreboard-from-scratch/)
