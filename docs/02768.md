# 自主饼干怪兽

> 原文：<https://hackaday.com/2008/11/10/autonomous-cookie-monster/>

![cookie_monster](img/168d76ccb84e6bceeb70c78e03fa79c9.png "cookie_monster")

建造了[自主 Wall-E](http://hackaday.com/2008/10/28/autonomous-wall-e/ "Autonomous Wall-E  - Hack a Day") 的【DJ Sures】带着另一个创作回来了。他的新[自主饼干怪兽](http://forums.trossenrobotics.com/showthread.php?t=2609 "[Contest Entry] My Autonomous Cookie Monster (LOL) - Trossen Robotics Community")当然是一个有趣的构建。他已经有了饼干怪兽毛绒玩具，所以第一步是剥去蓝色怪兽的皮，并插入一个骨架。他用了另一个机器人。车轮有两个伺服系统，每个手臂和脖子各有一个。嘴里有一个距离传感器。他为 PIC18F4685 微控制器构建了一个定制板，该板运行与他之前的机器人相同的 2D 映射代码。看看下面的视频吧。
