# 亚克力弯曲加热器

> 原文:[https://hackaday.com/2010/12/02/heater-for-bending-acrylic/](https://hackaday.com/2010/12/02/heater-for-bending-acrylic/)

![](../Images/66435747b5ff4bf14be26af33995cd00.png "heater-for-bending-acrylic")

我们喜欢在我们的项目中使用丙烯酸，但有几个棘手的技术，特别是获得胶合边缘的干净切口和将材料弯曲成曲线。[Giorgos Lazaridis]对后者有一个很好的解决方案，[一个专用的丙烯酸加热器](http://pcbheaven.com/projectpages/Acrylic_Glass_Folding_Heater/)。他没有使用烤箱来加热材料进行弯曲，而是使用一台旧激光打印机的高功率灯产生的局部热量。他的解决方案的下一部分是保持丙烯酸树脂的受热面积尽可能小。这是通过在灯泡的两侧制造散热器来实现的。上面看到的金属条有水流过，有助于将软化的材料分离成窄条。休息过后，看看这个系统在视频中的表现。
