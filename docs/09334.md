# [杰克]的太阳能时钟

> 原文：<https://hackaday.com/2011/12/19/jacks-solar-powered-clock/>

忙碌了一天的【杰克·巴芬顿】[终于完成了](http://www.youtube.com/watch?feature=player_embedded&v=ijMOdhYEILA#!)他为 buildlounge.com 激光切割机赠品制作的太阳能钟。

[Jack]一直在他的博客上张贴[构建日志](http://blog.robotbrigade.com/?p=51)，现在这个项目终于完成了。这座钟完全靠太阳能运转。这个钟取代了花哨的电子设备，给非常古老的学校日晷增添了新的活力。[杰克]屋外的一个盒子捕捉阳光，并将其聚焦到一组光纤上。这些纤维将阳光透过墙壁传输到钟面。在任一时刻，只有一部分光纤被点亮，并且这些光纤与一天中的时间相对应。由于有许多令人困惑的光纤布线，时钟可以通过点亮钟面来指示一天中的时间。

当这个项目在[杰克]的眼中还是一线曙光时，我们就报道了它的萌芽，我们必须把它交给他。这是一个非常有创意的项目，而‘夏令时’的加入更是锦上添花。休息之后看看视频，了解时钟实际上是如何工作的，一定要在 [BuildLounge](http://www.buildlounge.com/2011/12/19/contest-entry-solar-clock/) 上看看这个项目。
