# 发电厂乐团灵感 LED 领带

> 原文：<https://hackaday.com/2011/09/01/kraftwerk-inspired-led-tie/>

如果你在最后一次面试后没有得到这份工作，可能是因为你没有戴上这条可爱的发电厂乐团风格的领带。尽管我们自己的[Caleb Kraft]坚持认为邪恶疯狂科学家实验室的人最近的这项发明是向他致敬，但[Ralf Hütter 和 Florian Schneider]不敢苟同。

领带的灵感实际上来自发电厂乐团 1977 年为“机器人”拍摄的视频，视频中乐队戴着内嵌滚动发光二极管的黑色领带。这种效果与拉森扫描仪非常相似，尽管发电厂乐团的领带只向一个方向照亮发光二极管，而且不会淡入淡出。

EVMSL 发布了他们商店出售的拉森扫描仪的固件更新，复制了发电厂乐团效应，他们还整理了一个快速教程，展示如何构建自己的硬币电池供电的 LED 领带。我们并不建议任何人冲出去购买他们的工具包，因为它可以很容易地复制-我们只是觉得它很酷。

所以，如果你正在寻找一件复古的万圣节服装，不要再搜索了——邪恶的疯狂科学家实验室已经为你准备好了。

继续阅读他们领带的视频演示，以及激发灵感的原始视频。
