# 使用 Arduino 的远程温度计

> 原文：<https://hackaday.com/2009/03/03/remote-thermometer-using-arduino/>

![temp](img/1b31319722517a964add3c80abc748d0.png "temp")

[Peter]发来了这篇关于如何[制造一个远程可访问的温度计](http://solderintheveins.co.uk/?p=64#more-64)的文章。硬件方面很简单，你只需要一个 Arduino，一个电阻和一个热敏电阻。软件是主要的焦点。您可以通过命令行检查温度，也可以通过电子邮件。它还可以使用主机声音硬件告诉你温度。休息之后你可以看到一段视频。我们不得不奇怪[为什么](http://hackaday.com/2009/01/24/wattcher-twittering-kill-a-watt-plans-posted/) [他](http://hackaday.com/2009/01/02/twittering-washing-machine/) [没有](http://hackaday.com/2008/12/16/twittering-toaster/) [让](http://hackaday.com/2008/12/12/baby-twitters-via-kicks/)它碎碎念。
