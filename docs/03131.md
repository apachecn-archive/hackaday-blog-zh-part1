# 自动射击场

> 原文:[https://hackaday.com/2009/01/22/automated-shooting-range/](https://hackaday.com/2009/01/22/automated-shooting-range/)

![targets](../Images/4dd9549d771d7b7647f3c8c427293bdf.png "targets")

[Steve]是在和一个朋友讨论气枪的时候想到这个主意的。他的朋友哀叹他们的气枪爱好缺乏“行动”风格的目标。[史蒂夫]以此为契机，制作了自己的[自动目标系统](http://www.polymythic.com/2009/01/arduino-iobridge-airsoft-target-range/)。靶子本身由[构造](http://en.wikipedia.org/wiki/Construx)制成，一个纸靶子和一块布来阻挡气枪弹丸。由一个 [Arduino](http://arduino.cc/) 和一个 [ioBridge 模块](http://www.iobridge.com/)控制，它有一个网络界面，所以他可以从更高的范围切换程序。你可以看到他在休息后射击一些目标的视频示例。接下来，他应该把每个目标被击中的地方发到推特上。
