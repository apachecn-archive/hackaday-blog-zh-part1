# 晶片大师 3000

> 原文:[https://hackaday.com/2011/02/24/der-wafflemeister-3000/](https://hackaday.com/2011/02/24/der-wafflemeister-3000/)

![der_wafflemeister3000](../Images/b9d6f7d68d7c4e57e82beee6f43c45d6.png "der_wafflemeister3000")

当人们想到机器人时，一些不同的事情浮现在脑海中。我们喜欢机器人，因为它们负责单调乏味的工作。机器人也非常适合在危险环境中完成任务。然而，当[查默斯机器人学会]想到机器人时，他们会想，“早餐！”

CRS 建造了一台名为 Wafflemeister3000 的自动华夫饼干烘烤机[。它可以一次生产多达 5 个华夫饼，在 3 分钟多一点的时间内将它们烹饪成漂亮的金黄色。请想一想——这相当于每小时 90 块华夫饼！](http://www.tofsen.se/articles/27/wafflemeister3000-by-chalmers-robotics-society)

这个项目并不是全新的，第二次迭代已经在 2007 年完成了。然而，由于第三个版本的特点是产量增加了 400%，我们认为这值得一提。
