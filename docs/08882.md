# WRT300N 路由器的 RAM 升级

> 原文:[https://hack aday . com/2011/10/20/ram-upgrade-for-wrt 300n-router/](https://hackaday.com/2011/10/20/ram-upgrade-for-wrt300n-router/)

![](../Images/f0cfa5c499722d8461fe49a65021f67f.png "wrt300n-ram-upgrade")

[Heli]有一个 WRT300N 无线路由器坐在那里收集灰尘。他决定通过尝试升级内存来获得更多的娱乐价值。他设法将路由器的内存增加了一倍，并发布了一个遍历图来帮助你做同样的事情。

更换表面贴装 RAM 芯片不是世界上最容易的事情，你一定想知道是什么促使这一点。看起来他想在路由器上运行 LuCI 包，但是用 16 Mb 的硬件启动太慢(甚至无法启动)。他首先从一台旧的奔腾 III 计算机上采购了一些引脚兼容的替换芯片。当他的烙铁还热的时候，他还连接了一个 JTAG 接头，通过左边可见的红线连接。当他第一次启动该设备时，他很高兴它能够启动，但它仍然只检测到 16 Mb。

事实证明，你需要运行自己的内核来利用升级带来的好处。OpenWRT 的源代码很容易找到，并且有大量的编译指南。如果你尝试这样做，一定要仔细阅读[Heli 的]帖子，因为他有一些重要的配置信息，可以帮助你避免阻塞路由器。
