# 简单的感应加热器

> 原文:[https://hackaday.com/2011/10/27/a-simple-induction-heater/](https://hackaday.com/2011/10/27/a-simple-induction-heater/)

[![](../Images/e9b5f14ebc9338d6046d41d78839c9e8.png "induction-heater-wiring")T2】](http://hackaday.com/2011/10/27/a-simple-induction-heater/induction-heater-wiring/)

对于那些不熟悉的人来说，感应加热器是一种能够利用变化的磁场快速加热物体的设备。[RMC Cybernetics]决定[建造一个](http://www.rmcybernetics.com/projects/DIY_Devices/diy-induction-heater.htm)并且很好地为互联网的学习和娱乐写了这个项目。他们的网站上提供了完整的解释以及示意图和建造说明。

这种加热器的工作原理与大多数变压器相同。当导体附近的磁场发生变化时，就会感应出电流并产生热量。有趣的是，虽然变压器被设计成最小化这种热量，但感应加热器的目标是最大化线圈内放置的任何物体的热量。

[RMC]已经提供了一个视频，说明如何建立加热器，以及它在休息后的行动！跳到 to 1:42，观看加热过程。或者观看整个过程，看看它是如何建成的。
