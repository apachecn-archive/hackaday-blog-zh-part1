# 神圣的烤面包机套件

> 原文：<https://hackaday.com/2009/06/23/the-holy-toaster-kit/>

![456124655_30491edce5_m](img/9d430c835c64eceb24d07520fb605175.png "456124655_30491edce5_m")

神圣的烤面包机是一个开源工具包，可以把耶稣放在你的面包上，包括成本明细和工程数据文件。一个激光加工的不锈钢耶稣阻挡了辐射，防止烤面包变褐。安装非常简单。用一把钳子弯曲几个拉环作为支撑，然后将图案插入烤面包机。可能需要调整褐变以获得足够的对比度。最好的结果似乎来自蜂蜜麸乡村面包。他们的 Flickr 页面包含了[更多的照片和原型](http://www.flickr.com/photos/sixmilliondollardan/sets/72157618798710665/)被切割的视频。让每个人都知道你用我们在 2008 年报道过的[推特烤面包机](http://hackaday.com/2008/12/16/twittering-toaster/)做的神圣的烤面包。下面是教学视频。
