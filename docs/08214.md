# 对 Kinect 进行编程，使其与 Android 兼容

> 原文:[https://hack aday . com/2011/08/08/programming-the-Kinect-to-work-with-Android/](https://hackaday.com/2011/08/08/programming-the-kinect-to-work-with-android/)

[![](../Images/516103b4fe50a598f944a92fa5d9636b.png "android_kinect")T2】](http://hackaday.com/2011/08/08/programming-the-kinect-to-work-with-android/android_kinect/)

Kinect 被黑客攻击的目的很多，但是这个 [Android 实现](http://raymondlo84.blogspot.com/2011/07/howto-using-microsoft-kinect-on-tegra.html)教程是我们第一次听说它被用在手机平台上。虽然现在还不是成品，但是对于那些想尝试融合这两种技术的人来说，[Raymond 的]教程是一个很好的起点。

本教程的 Kinect 编程已经使用面向 Android 3.0 的 Tegra Ventana 开发工具包完成。使用了 Ubuntu-Linux 安装，但是根据作者所说，这种技术应该可以移植到任何 Linux 系统。

给出的过程非常简单，作者甚至在休息后的视频中提供了一个结果示例。有趣的是，当连接时，Kinect 被视为两个设备，“Xbox NUI 相机”和“Xbox NUI 音频”。我们期待这款设备的新技术问世，可能会使用这套工具。
