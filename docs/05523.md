# 柯契拉灯

> 原文:[https://hackaday.com/2010/06/19/coachella-lamp/](https://hackaday.com/2010/06/19/coachella-lamp/)

![](../Images/ca89fb21a3f290d2e52091f495a27dcf.png "nixie-lamp")

这件小艺术品可能正是给你的房间增添疯狂科学家的样子的好东西。它被称为科契拉灯，它使用了几个复古的显示设备。顶部的氩放电灯发出紫外线。你可以看到从四面突出来的一组十电子回旋加速器。还有四个数码管条形图矗立在基地的每个角落。最后一点是 led 提供的中心彩色发光。休息之后我们嵌入了一些设备的视频。

该灯由墙壁插座供电，并由 Arduino 控制。我们已经看到[电子管被用作计时装置](http://hackaday.com/2009/04/06/dekatron-kitchen-timer/)，并且希望看到一些时钟功能被添加到灯中。试图从不同的谢妮显示器上破译时间会把它和其他一些[难以阅读的钟表](http://hackaday.com/2010/06/07/color-clock-makes-telling-time-impossible/)放在一起。
