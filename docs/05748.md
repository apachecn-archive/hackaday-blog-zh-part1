# Android 控制 Mindstorms NXT

这是一台控制乐高思维风暴机器人的安卓设备。[乔希]和他的几个同事开发了软件，让 Android 2.1(上面的视频)和 Android 1.6 ( [视频](http://www.youtube.com/watch?v=m6q0eUw3nrg))都可以作为蓝牙控制设备。NXT cube 运行的是 [leJOS](http://lejos.sourceforge.net/) ，一个 Java 虚拟机，允许你用 Java 编写思维风暴。虽然他们的代码处于早期的 Alpha 阶段，但这表明它是可行的，并且一旦他们达到一个更稳定的点并将其发布给大众，它将是一个非常受欢迎的 NXT 开发工具。
