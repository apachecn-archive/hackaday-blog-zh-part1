# 骑自行车的好方法

> 原文：<https://hackaday.com/2009/10/04/a-bright-way-to-cycle/>

[https://www.youtube.com/embed/FQrNmnU8kJk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/FQrNmnU8kJk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

晚上骑车想被人看见？[Neon Dean]想出了这个可能的解决方案，并在 Nuit Blanche 进行了巡游。这是一辆自行车[，霓虹灯](http://hackaday.com/2008/11/10/neon-light-christmas-tree/)安装在每一个可能的表面。[迪安]对他的车进行了类似的处理，解释了它的工作原理。所有的电子管都由 12v 直流电池供电[他把它放在一个腰包里](http://smithvictor.com/products/detail.asp?prodid=227)。12V 的电压对电子管来说太低了，所以[一个升压变压器](http://en.wikipedia.org/wiki/Transformer)被用来提高这个数字。[迪安]还决定在每个轮子上安装一个霓虹灯管。为了给它们输送电力，他在每个轮子上安装了一个转子，两个导电轨道靠近每个转子的边缘。两条钢条充当电刷(类似于槽车上的[电刷)，将逐步增加的功率传递给电子管。一个有创意但可能不太聪明的想法是[迪安]的霓虹灯管头盔。](http://en.wikipedia.org/wiki/Slot_car#How_it_works)
