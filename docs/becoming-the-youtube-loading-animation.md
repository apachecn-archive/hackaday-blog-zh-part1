# 成为 YouTube 加载动画

> 原文：<https://hackaday.com/2012/03/12/becoming-the-youtube-loading-animation/>

![](img/97dcab611366610a1e1f237035544206.png "youtube")

无论你是习惯了 wifi 连接中断，还是你的室友用完了所有的流媒体带宽，我们都见过旋转的八位磁盘，这就是 YouTube 加载动画。[技术官僚]认为这将是一个伟大的想法，实际上成为 YouTube，并着手设计一个加载动画的物理表现。

[技术官僚]用八个乒乓球作为建筑的主要结构。在钻出每个乒乓球并粘上白色 led 后，剩下的唯一事情就是将所有东西连接到 Arduino。代码在每个 LED 之间循环，并提供 YouTube 已经烙进我们记忆中的“光迹”动画。

为了完成这一构建，[技术官僚]将他的身体负重动画附在一件印有 YouTube 标志的黑色 t 恤上，以使一切更容易理解。虽然它可能不像死亡沙滩球那样丰富多彩，也不像 Windows 的“旋转沙漏”那样有概念，但我们真的很喜欢这个版本。休息之后请看视频。
