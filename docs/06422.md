# 乐高机器预测未来的日食

> 原文:[https://hack aday . com/2010/12/09/Lego-machine-predicts-future-eclipse/](https://hackaday.com/2010/12/09/lego-machine-predicts-future-eclipses/)

![](../Images/57dbb7ac73e1535d49828c00f442ea20.png "Lego-Antikythera")

隐藏在这台机器白色面板后面的是齿轮架，它们构成了已知最古老的机械计算机之一的复制品。这是由乐高积木制成的安蒂基西拉机械装置的工作模型。在视频中，你绝对不能错过休息后，机器被分解成各种组件。每个机械单元利用齿轮比组合来执行多级数学功能，以显示未来天体事件将发生的日期和时间。

原始设备上的[背景信息读起来像是《七宝奇谋](http://www.nature.com/news/2010/101124/full/468496a.html)[的续集](http://www.imdb.com/title/tt0089218/)的剧本。据信可以追溯到公元前 100-150 年，~~石~~青铜机械装置是在二十世纪初的一次沉船中发现的。x 射线分析的使用有助于打开功能，并证实其运作的理论。

让这一点如此有趣的部分原因是历史联系。但是视频的制作质量(平心而论，这似乎是一个广告)真正让人们意识到这个过程是多么复杂。现在是时候让我们多看几遍视频了，勾画出齿轮传动装置，看看它是否如他们所说的那样工作。
