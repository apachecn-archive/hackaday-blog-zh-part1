# 用 IPhone 碾过你自己

> 原文：<https://hackaday.com/2009/10/24/use-iphone-to-run-yourself-over/>

![run-yourself-down-with-iphone](img/22a92934a1aceb5bf72dcab18cf07264.png "run-yourself-down-with-iphone")

柏林精神团队已经开发了一个 [iPhone 应用程序来远程控制一辆小型货车](http://robotics.mi.fu-berlin.de/pmwiki/pmwiki.php)。他们不必对车辆做太多工作，因为该平台是为 2007 年 Darpa 城市挑战赛开发的。iPhone 通过 WiFi 与驱动电路相连，并提供了一个油门按钮、一个刹车按钮和一个转向按钮，以使加速度计能够转向。前置摄像头视频实时传输到 iPhone。

在上面的图片中，你可以看到货车摄像机画面中央的操作员。看起来货车的最高速度是有限的，但是记住我们自己驾驶遥控车辆的无能，我们希望这不会导致 T2 奖。休息之后我们嵌入了一段视频。每个人都喜欢看一些马里奥赛车重演。你可以在 2:28 左右看到一些视频。享受吧。
