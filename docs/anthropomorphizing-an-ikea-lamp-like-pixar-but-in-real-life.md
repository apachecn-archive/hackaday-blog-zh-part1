# 拟人化宜家灯具(类似皮克斯，但在现实生活中)

> 原文：<https://hackaday.com/2011/09/08/anthropomorphizing-an-ikea-lamp-like-pixar-but-in-real-life/>

ArduinoArts 正在制作一款便宜的宜家灯具作为参赛作品。Seeed Studio 的[玩具黑客大赛](http://www.seeedstudio.com/blog/2011/07/27/write-a-step-by-step-instruction-win-200-coupon/)号召参赛者使用 Grove 玩具套件施展魔法，这是一个用于 Arduino 的可扩展传感器连接系统。套件中的大部分物品用于增加灯具的互动性。休息后看看视频，看看两个伺服提供的运动。灯可以像摇头一样前后移动其灯罩，并且整个臂组件可以相对于底座旋转。当你重新定位灯头时，传感器会检测到，如果这个装置不欣赏它的新姿势，它会对你大喊大叫。它还能对噪音和运动做出反应，在两种情况下都会打开取代原来灯泡的 LED，并在检测到运动时问:“你是莎拉·寇娜吗？”这些基本的修改确实创造了一些有趣的电子动画行为。
