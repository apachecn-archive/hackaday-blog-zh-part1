# 76 个燃烧的长号引领着大游行

> 原文：<https://hackaday.com/2010/09/09/76-flaming-trombones-led-the-big-parade/>

![](img/8192c268ae2274acfd8a4890c09d72d9.png "flaming-trombone")

[乔纳森·克劳福德]已经准备好并愿意用他那燃烧的长号点燃激情。几年前，他的乐队老师在储藏室整理乐器。这只长号遭受了严重的[红腐病](http://www.brassarts.com/about_red_rot.htm)，再也不能正常工作了，所以【乔纳森】能够得到它并开始工作。

他开始打磨仪器，并用高温喷漆对其进行喷漆。用于制冰机的柔性铜管被用于将丙烷出口重新定位在仪器的钟形件内。由玩家左手拇指控制的烧烤点火器点燃火焰。

乔纳森使用的火炬只能让少量的气态丙烷从喷嘴中出来。他最后钻了一个孔，并用一小段乙烯基管来桥接喷嘴和辅助铜管之间的间隙。在最大爆炸时，液态丙烷会泄漏，所以要小心。
