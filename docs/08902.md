# Altoid Tin 蚀刻教程

> 原文:[https://hack aday . com/2011/10/23/altoid-tin-etching-tutorial/](https://hackaday.com/2011/10/23/altoid-tin-etching-tutorial/)

![](../Images/832a81b7d4f085de2ba383d1a4325f0c.png "banner")

杰出的蒸汽朋克[杰克·冯·斯拉特]为蒸汽朋克圣经写了一篇关于蚀刻糖果罐的小文章，但是书中有限的篇幅不允许完整的阐述。为了弥补他不完整的教程，他发布了[这篇穿越](http://steampunkworkshop.com/etching-tins-salt-water-and-electricity-compliment-steampunk-bible-article)来赞美圣经的文章。

这个过程非常类似于我们在家用墨粉转移法蚀刻 PCB 上看到的许多 [教程](http://hackaday.com/2009/11/09/diy-circuit-etching-video-tutorial/)。把颜料从罐子里拿出来，做一个面具，印在周日的传单上，然后把熨斗拿到罐子里，这些都是家庭作坊的老把戏了。

与 PCB 制造不同的是，[冯·斯莱特先生]不关心氯化铁或其他有害的化学物质——他用电解来做一切事情。在一桶水中加入几汤匙食盐后，[杰克]拿出一个 DC 电源，把正极引线接到盖子上，负极引线接到底座上。在盖子的角上缠一点绝缘胶带可以防止金属变得太薄。

通过擦拭硫酸铜溶液，可以在成品锡上涂上一层漂亮的铜涂层——硫酸铜是“杀根”产品中的一种常见成分。当然，这不是必要的步骤；你可以很容易地享受和优雅的铝锡裸金属。
