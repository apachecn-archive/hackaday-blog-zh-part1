# 儿童 RFID 点唱机

> 原文:[https://hackaday.com/2012/03/14/rfid-jukebox-for-the-kids/](https://hackaday.com/2012/03/14/rfid-jukebox-for-the-kids/)

![](../Images/7d904295ab3e5af04e17939560e5701e.png "rfid-jukebox")

[张秀坤]为他的女儿[安娜]制作了一个有趣的音乐玩具。这是一个自动点唱机，[让她播放她最喜欢的音乐，使用 RFID 标签](http://scalotta.tumblr.com/post/18998879764/contact-free-jukebox)在它们之间进行选择。

这个项目很简单，但是很强大。附件是一个木制工艺盒，你可以花几块钱买到。里面有一个带波形屏蔽的 Arduino，可以处理音频播放。RFID 阅读器从他获得的一组卡片标签中获取输入。内部锂电池为该设备供电，USB 端口用于充电。

当然，这些勇气是有代价的。但是没有可以打碎的液晶显示器，我们认为如果安装正确的话，这些板可以承受滥用。另外，这里有很大的未来潜力。当我们看到这些卡片时，我们会想到那些会发出动物声音的玩具(“牛在说什么……哞”)。这可以用于非常小的孩子。然后当它们变得有点旧的时候被重新利用到这个自动点唱机中。如果你把内脏放进一个新的盒子里，它看起来会是一个全新的玩具，对吗？
