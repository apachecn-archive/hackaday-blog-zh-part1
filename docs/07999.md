# 复古收音机:建立在另一个 WiFi 无线电黑客之上

> 原文:[https://hackaday . com/2011/07/06/retro-radio-building-on-other-wifi-radio-hack/](https://hackaday.com/2011/07/06/retro-radio-building-on-another-wifi-radio-hack/)

![](../Images/083d9c7c5c62f4742ec63101a49f0a55.png "arduino-based-wifi-radio")

[Jay Collett]喜欢听网络电台，并决定建立自己的独立硬件，让他不用坐在电脑前就能听。但在这个项目上，他不是从第一步开始的。[Jay]建立在[Mighty Ohm]发表的史诗指令集之上，当时他第一次建立了[自己的 WiFi 无线电](http://hackaday.com/2009/02/03/wifi-streaming-radio-update/)。

这两种收音机使用相同的方法上网；被黑的路由器。我们有点喜欢这一点，因为这意味着你应该能够使用无线电作为有线以太网设备的桥梁。但是我们跑题了。[Jay]使用 Arduino 及其 IDE 与 LCD 屏幕进行交互，使其线/列方向不同于原始方向，并实施不同的控制方案，从而背离了这些说明。如果你不想使用自己的电子产品，这让这个项目变得更容易实现。

休息之后看一下视频。路由器需要相当长的时间来启动，但是一旦启动，似乎控制既直观又响应迅速。
