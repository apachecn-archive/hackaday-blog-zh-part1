# 没有风险的智能手机黑客-此外，需要坏掉的手机

> 原文:[https://hack aday . com/2011/09/06/smart phone-hacking-without-risk-plus-breaked-phones-needs/](https://hackaday.com/2011/09/06/smartphone-hacking-without-risk-plus-broken-phones-needed/)

![](../Images/153b78c13230b50d97ec2f4c0c5c3dc3.png "unbrickable-mod")

[Adam Outler]和[Rebellos]一直在狂热地工作，以推进移动设备黑客的世界。他们在两条战线上进行攻击，使普通黑客更容易不受惩罚地篡改手机的固件和操作系统，并通过寻找方法将普通手机变成低级黑客的开发硬件。

[蜂鸟拦截器引导程序](http://forum.xda-developers.com/showthread.php?t=1242466) (HIBL)绕过了运行 Cortex-A8 系列处理器的智能手机上的信任链。这打开了许多大门，尤其是能够运行任何能够移植到硬件上的操作系统。我们确信 Android 版本将首先出现，因为它们是开源的，但有传言称，在一些严重的组装黑客攻击后，iOS 或 Windows Phone 将会运行。

但是这两位开发者正试图用他们最近的黑客技术吸引更多的人。[Adam]已经发出了对你损坏的硬件的请求。他需要你报废的智能手机主板来逆向工程电路。从处理器上的 OM5 引脚到 OM1 电阻焊接一根线将使手机不易碎([我们在 7 月份听说过](http://hackaday.com/2011/07/12/one-click-unbrick-for-samsung-phones/))，并消除了在 JTAG 接口焊接的需要。手里拿着破解的硬件[，他跳出处理器，追踪这个连接](http://forum.xda-developers.com/showthread.php?t=1206216)以及 UART 管脚。

焊接不是一个简单的过程，但这是一个显著的进步，打破了更多阻止人们侵入他们梦寐以求的硬件的障碍。广告之后的视频展示了如果你的手机在修改固件或操作系统的时候出了问题，恢复它是多么的容易。
