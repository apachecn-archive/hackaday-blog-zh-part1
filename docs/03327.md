# 自主全地形车

> 原文：<https://hackaday.com/2009/03/23/autonomous-atv/>

[https://www.youtube.com/embed/Ja3ND8Pk_0M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Ja3ND8Pk_0M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

北卡罗来纳大学夏洛特分校的一个团队一直在研发一种自动驾驶汽车来拖动一辆装有传感设备的手推车。从一辆本田 ATV 开始，添加了不同的系统，让[瑞萨处理器](http://www.renesas.com/fmwk.jsp?cnt=m16c62p_root.jsp&fp=/products/mpumcu/m16c_family/m16c60_series/m16c62p_group/ "Renesas 62P processor")控制 ATV。一架模型飞机的接收器被连接到瑞萨飞机上，为项目的第一阶段提供远程控制。基本上他们已经把 ATV 变成了一辆巨大的遥控汽车。

后来的版本将包括激光雷达，摄像机和多个 GPS 单元，这样 ATV 可以以高精度自动穿越大多数地形。到那时，路径规划将成为项目的一大部分。
