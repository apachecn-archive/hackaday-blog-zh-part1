# 2011 年 3 月 3 日每周综述

![](../Images/f295aafa8c4c1f5ca6171070a45844ab.png "ru3.3.12")

如果你第一次错过了它们，这里是我们过去一周最受欢迎的帖子:

排在第一位的是[少将·格蕾丝·赫柏]的一段视频，她讲述了如何用一根铜线来想象一纳秒的时间。

获得第二名是我们在 HAD 感到非常兴奋的一件事。[树莓派终于上市了！对于那些没有关注它的进展的人来说，这是一台小型(大约信用卡大小)的 Linux 计算机，更好的型号只需 35 美元！我们可以想出各种我们想用它们来做的事情，但是，唉，甚至我们还不能把我们的脏手放在它们上面，因为它们已经完全卖完了。](http://hackaday.com/2012/02/29/raspberry-pi-launched/)

接下来，我们有一个《我的世界》境内的虚拟天气气象站。这个项目使用一个小 LCD 和一个 Arduino 来告诉你是否需要穿上你的虚拟雨衣。

如果你一直在寻找一种在家阳极氧化钛的方法，而不是把你的零件送到维修站，这里有一篇文章给你。

最后，我们以一个我们非常喜欢的帖子结束。这是关于三架遥控“飞机”在布鲁克林大桥附近飞行，从远处看很像人。
