# 照相软件学会从人群中认出你

> 原文:[https://hackaday . com/2011/04/21/camera-software-learn-to-pick-you-out-of-a-crowd/](https://hackaday.com/2011/04/21/camera-software-learns-to-pick-you-out-of-a-crowd/)

![tld_object_tracking](../Images/a770ee9361b9ef3fd35f49daf26ad539.png "tld_object_tracking")

虽然 Kinect 非常擅长跟踪身体的总体运动，并识别一个人的骨骼的哪一部分在摄像头前移动，但这款设备肯定有其不足之处。例如，面部识别是非常有限的，我们猜测它不能很容易地跟踪整个房间中一个人的眼睛。

不，像这样的跟踪，你需要更强大的东西。在[王南钧·米科拉伊奇克和吉里·马塔斯]的指导下，博士生[兹德内克·卡拉尔]一直致力于开发一款名为 TLD 的软件，这款软件拥有一些相当惊人的功能。该软件使用几乎任何与电脑相连的摄像机同时 **T** 跟踪一个物体， **L** 获得其外观， **D** 检测出出现在视频流中的物体。正如你在下面的视频中所看到的，这个软件非常有效，以至于它被称为“掠夺者”。

一旦他在相机的视野内选择了一个物体，软件就会监控这个物体，越来越多地了解它在不同条件下的样子。该软件的学习能力允许它挑选出个人的面部特征，跟踪视频中的移动物体，并可以在一群人当中识别出一个人的脸。

虽然该软件目前一次只能跟踪一个物体，但我们可以想象，随着一些额外的开发和计算能力，这项技术将变得更加惊人。
