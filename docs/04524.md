# 灯泡表盘时钟套件发布

> 原文:[https://hack aday . com/2009/12/10/bulb dial-clock-kit-released/](https://hackaday.com/2009/12/10/bulbdial-clock-kit-released/)

![](../Images/bed6b50722bcea83bcec27edfa46e4d5.png "bulbdial-clock-kit")

邪恶的疯狂科学家实验室刚刚发布了一个球形时钟套件。自从他们的[第一个原型](http://hackaday.com/2009/04/08/the-bulbdial-clock-comes-to-life/)以来，这已经走了很长一段路，其特点是三个 PCB 携带 72 个 charlieplexed LEDs。为了精确起见，他们为 [ChronoDot precision RTC](http://hackaday.com/2009/10/27/parts-chronodot-rtc-module-ds3231/) 配备了可选的标题。

有了好看的表面和激光切割的丙烯酸表壳，这可能很快就会装饰我们的外套。是时候给圣诞老人写信了…
