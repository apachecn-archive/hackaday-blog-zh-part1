# SMD LED 矩阵

> 原文：<https://hackaday.com/2009/10/02/smd-led-matrix/>

![3573544803_03000843fc](img/da3b62bd3300d925b3432f6c1169adaf.png "3573544803_03000843fc")

每个人都喜欢一个好的 LED 矩阵黑客，我们已经看到了我们的[公平份额](http://hackaday.com/?s=matrix)，但从来没有一个 16 乘 24 手工焊接的 SMD。[bear24rw]和他的团队需要一个反映他的一个班级的兴趣的项目，作为一个电子工程专业的学生，一个 [LED matrix](http://bear24rw.blogspot.com/2009/10/led-matrix.html) 正是入场券。基于[这个指令性的](http://www.instructables.com/id/LED-matrix-using-shift-registers/)，他能够拼凑出一个驱动板；然而代码是不可接受的，他想出了一个双缓冲数组解决方案。你可以在这里获得 eagle 文件和源代码[，休息后可以看一段视频。](https://ucfilespace.uc.edu/~thrunml/led_board.zip)
