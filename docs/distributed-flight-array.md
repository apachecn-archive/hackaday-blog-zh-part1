# 分布式飞行阵列

> 原文：<https://hackaday.com/2010/06/09/distributed-flight-array/>

[https://www.youtube.com/embed/7iHZKMDvdNI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/7iHZKMDvdNI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

虽然还处于开发的初级阶段，但[分布式飞行阵列](http://www.idsc.ethz.ch/Research_DAndrea/DFA/)看起来已经非常有趣了。每个单位都可以利用道具的向下力在地面上滑行，但是当 4 个或更多的单位联合起来时，它们可以起飞和飞行。文件显示，它们应该足够聪明，能够以随机配置飞行，尽管在视频中我们只看到标准的 4 道具布局。这是生产[平衡立方体](http://hackaday.com/2010/05/19/balancing-cube-looks-more-like-a-star/)的同一批人正在研究的。
