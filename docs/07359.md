# 用 Kinect 控制武器

> 原文:[https://hack aday . com/2011/04/11/用 kinect 控制武器/](https://hackaday.com/2011/04/11/controlling-weapons-with-kinect/)

![kinect_rocket_launcher](../Images/2d144144de9670b4c7008a57ce663d0b.png "kinect_rocket_launcher")

有人想出如何将他们的 Kinect 武器化只是时间问题。一天，黑客乔纳斯·瓦格纳(Jonas Wagner)摆弄着他的 Kinect，他认为简单地通过打手势发射导弹会很酷。由于手头没有任何真正的导弹，他决定控制一个 USB 供电的泡沫导弹发射器。

他在他的火箭发射器顶部安装了一个网络摄像头来记录他的受害者的视频，用一点 Python 和 libfreenect 图书馆，他正在走向世界隔间统治。Kinect 等待他戏剧性地将手从皮套中抽出，监控他的动作以达到追踪的目的。一旦发射器准备好了，Kinect 就会在发射火箭前观察[乔纳斯]是否把手从画面中抽出来。

我们怀疑你会看到这个东西很快控制 DARPA 的武器，但它仍然很酷。发射器似乎移动有点慢，但我们猜测，随着伺服升级，事情可能会有点快。
