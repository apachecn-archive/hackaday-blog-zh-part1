# MIDI 空气鼓让你在任何地方演奏

> 原文:[https://hack aday . com/2011/07/29/midi-air-drums-let-you-play-anywhere/](https://hackaday.com/2011/07/29/midi-air-drums-let-you-play-anywhere/)

[Maayan Migdal]来信分享了他制作的一套非常酷的架子鼓,它有一个特别之处——完全没有鼓。使用简单的 MIDI 设备和 Arduino，他的“空气鼓”看起来非常甜美。

黑客利用一对花园耙子，作为他的鼓棒。耙子被削减和修改，以允许增加加速度计和一些 USB 电缆。左棒包含一个用于记录踩镲击打的加速度计，而右棒配备了一对模块，用于触发响弦和撞击符号击打。他修改了一双凉鞋，使其在打鼓时更合脚，然后在每只鞋上安装了一个传感器。左边的凉鞋包含一个加速度计来记录低音鼓的击打，而右边的鞋子使用一个光传感器来模拟踩镲踏板的使用。

我们认为结果很棒，但也可以看看下面的视频，看看我们的意思。如果《吉他英雄》在《中断》中没有~~死在水里~~，我们认为这种设置将会是游戏附带的脆弱的架子鼓的一个很好的替代品。
