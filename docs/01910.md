# 六足数控机器人

> 原文:[https://hackaday.com/2008/06/11/hexapod-cnc-bot/](https://hackaday.com/2008/06/11/hexapod-cnc-bot/)

![](../Images/392e0993eac705db7ff25517d460d9aa.png)
当所有的八条腿的野兽[在](http://www.youtube.com/watch?v=iTDr7ut5ERA)周围蹒跚而行，[举办香肠节](http://www.youtube.com/watch?v=2whmvlpFrKE)时，很高兴看到六条腿的机器人实际上是有生产力的。[马特·丹顿]的[六足机器人数控刳刨机](http://www.hexapodrobot.com/forum/viewtopic.php?f=14&t=12)是一台令人印象深刻的机器。[b . f . hexadop](http://www.micromagicsystems.com/#/bfhexapodv5/)是使用 Hitech 的 HSR-5995TG 建造的，这比类似尺寸的装置具有更高的扭矩。每只脚都安装了球形接头，以方便[适应地形](http://www.youtube.com/watch?v=3DkL6BQiVKQ)。[Matt]最近才开始玩 CNC。首先，他做了一个笔式绘图仪的概念验证。现在这个机器人可以在聚苯乙烯上磨出 3D 表面。它仍然是一个小课程，可能会一直有点不精确，因为它没有栓住。他还计划将其从标准的 1/8 英寸钻头转换为 3 毫米的路由器钻头。我们很想看到这个机器人在复杂的浅浮雕上工作。没有固定的工作范围确实为这样的机器和 [Hektor](http://www.hektor.ch/) 提供了可能性。视频和最终产品嵌入下面。
