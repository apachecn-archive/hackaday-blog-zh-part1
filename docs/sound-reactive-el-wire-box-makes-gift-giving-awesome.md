# 声音反应电接线盒让送礼变得棒极了

> 原文：<https://hackaday.com/2012/01/15/sound-reactive-el-wire-box-makes-gift-giving-awesome/>

![sound-reactive-el-wire-display](img/249f1a6aca4c6561921231700f594d43.png "sound-reactive-el-wire-display")

[Jonathan Thomson]对 EL wire 显示器进行了反复思考，认为他见过的大多数显示器都是无聊的静态固定装置或装置，它们只是以固定的速率闪烁 EL wire。他认为 EL wire 的潜力远不止于此，并着手建造更令人兴奋的东西。他用一件我们相信你很熟悉的图形均衡器 t 恤，组装了一个光滑的声音感应 EL 线显示器。

他首先从前面提到的 t 恤上取下 EL 面板和逆变器，将显示屏分成两部分。他将面板放在一边，专注于将逆变器的带状电缆连接到他为该项目挑选的一组 el 电线束上。一旦他把所有东西都装好，他就把一个设计放在一个纸板盒上，他打算用它来包装圣诞礼物。随着假期的结束，[乔纳森]打破了他原来的展示，建造了另一个展示，提供一些有趣的生日祝福。

虽然 EL inverter 最初是为了显示板载麦克风检测到的声音，但[Jonathan]在他的扬声器上增加了一个 3.5 英寸的立体声插孔，这样他就可以使用 MP3 播放器直接将音频输入显示器。

继续阅读，看看 EL 显示器的实际应用，如果你想让今年的送礼更有情趣，一定要看看他的文章。
