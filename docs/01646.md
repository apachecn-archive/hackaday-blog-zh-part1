# 多士炉回流控制器

【内森】[送来了他的](http://forum.sparkfun.com/viewtopic.php?p=42893)烤面包机烤箱的回流控制器。他用三端双向可控硅开关来驱动加热元件，用热电偶来监控温度，用 PIC 控制器来控制演出。我给他在控制面板上的风格加分。希望他完成后有时间写一份完整的报告。你可以在这里查看项目照片。
