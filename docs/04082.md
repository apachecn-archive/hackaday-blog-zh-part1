# 新的 Wii 驱动硬件仿真

> 原文：<https://hackaday.com/2009/09/22/new-wii-drive-hardware-emulation/>

![wii_drive_emulation](img/eeaca85dbf55a9a20be0d107a0e0b3f3.png "wii_drive_emulation")

Wii 光驱模拟器(沃德)可以从 SD 卡或 USB 存储设备中加载 Wii 和 GameCube ISO 文件。这种黑客使用带状电缆将光驱连接到 Wii，不需要焊接。沃德基于 ARM9 处理器，运行 Linux，具有背光 LCD 屏幕和 4 向中央点击操纵杆。存储器可以热插拔，然后使用棒和 LCD 显示器选择 ISO。选定的 ISO 文件会出现在游戏频道中，就像一张原始光盘刚刚插入驱动器中一样。

开发者声称 Wii 固件升级将无法锁定沃德。还有第二个带状电缆连接器用作通道，如果您愿意，可以选择保持光驱连接。现在，寻找一种能够容纳所有这些新硬件，并且看起来仍然像原来一样漂亮的替代机箱的竞赛已经开始了。我们不会惊讶地看到一个家酿频道程序，允许 ISO 选择，而不必走到控制台。
