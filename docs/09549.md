# 把集线器塞进键盘

> 原文:[https://hack aday . com/2012/01/22/把一个集线器塞进那个键盘/](https://hackaday.com/2012/01/22/shove-a-hub-into-that-keyboard/)

![](../Images/5b337526fd7898bbeda7b6481d252af7.png "Untitled")

作为技术大师，我们的桌子上经常堆满了杂物。我们每个角落都有电缆，我们的计算机端口都满了。我们最不希望的是，在添加像 USB 集线器这样的东西时，会有更多的东西碍事，或者被成堆的技术文档埋没。这就是[迈克尔]的用武之地，[把一个小 USB 集线器塞进他的罗技键盘](http://mtkaalund.blogspot.com/2011/11/adding-usb-hub-to-keyboard.html)。

键盘已经是 USB 了，集线器有 3 个前向端口，后面还有第四个单端口。把两个设备分开，他用键盘上已经有的 USB 线代替了集线器的输入线。然后，他卸下后端口，直接将键盘连接到集线器上。

从那里，它只是一个问题，找出他想要的枢纽，并切断塑料。他用了一把刀，对一些轻微的割伤有着美好的回忆，这让我们建议要(更加)小心。在刀刃上施一点火就能走很长一段路。

一旦键盘重新组装起来，他的键盘背面就有了一个方便的 3 端口集线器，看起来像是工厂生产的，可以避免混乱。
