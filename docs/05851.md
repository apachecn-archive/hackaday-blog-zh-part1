# 仪表板内摩托车显示器

> 原文:[https://hackaday.com/2010/08/24/in-dash-motorcycle-display/](https://hackaday.com/2010/08/24/in-dash-motorcycle-display/)

![](../Images/c9b0aa7d1ad7b742b1c355bbca1b2dd8.png "motorcycle-LCD-add-on")

[Muth] [在他的摩托车](http://code.google.com/p/mini-buell-ecm-spy/wiki/TheStory)仪表盘上加了一个辅助显示器。他开始使用 PIC 16F877A 进行原型制作，并使用它通过 ECM 诊断连接来访问信息。一旦他开始工作，他就发现了这个微型显示器，它完美地安装在速度计和转速表之间。休息后有一个简短的演示，您可以看到过去 30 分钟的自适应燃油值和发动机温度的历史记录，以及一个辅助信息屏幕。

这是我们收集的[车辆显示器](http://hackaday.com/2010/05/05/vehicle-information-display-hacks/)、[踏板车控制器](http://hackaday.com/2010/06/28/touchscreen-scooter-computer/)、[档位指示器](http://hackaday.com/2010/06/04/motorcycle-current-gear-indicator/)和[摩托车电脑](http://hackaday.com/2010/05/01/motorcycle%c2%a0computer/)的又一个好的补充。
