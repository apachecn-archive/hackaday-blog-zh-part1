# 旋转显示器使用录像机磁头和发光二极管

> 原文：<https://hackaday.com/2010/06/10/rotary-display-uses-vcr-head-and-leds/>

![](img/59a103452ae18fb398fb6e56c6759371.png "VCR-head-rotary-display")

[丹尼尔·戴格尔]正在开发一种利用视觉暂留来绘制数据的旋转显示器。他使用的硬件包括一个录像机的旋转头、一些发光二极管和一个显示 360 度数据的定时电路。他的时序输入使用波形，因此可以在任何产生 PWM 信号的应用中使用。

休息后看看他的视频，展示了一个单线图和一个六线图。
