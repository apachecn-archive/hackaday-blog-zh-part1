# DIY 吉他模糊踏板

> 原文:[https://hackaday.com/2011/07/09/diy-guitar-fuzz-pedal/](https://hackaday.com/2011/07/09/diy-guitar-fuzz-pedal/)

![diy_fuzz_pedal](../Images/2e9cd13f81e78b80a597662dcfa1839e.png "diy_fuzz_pedal")

Instructables 用户和社区经理 Randy Sarafan 最近编写了一个教程[，讲述如何建立一个古老的音乐标准](http://www.instructables.com/id/Fuzz-Pedal)“模糊踏板”。他说，摇滚的秘密是模糊，所以如果你能处理烙铁和吉他，这个项目是给你的。

当你仔细观察时，踏板的组件实际上非常简单。失真是由一对晶体管造成的，在他的例子中是 RadioShack 的普通 npn。我们已经介绍过之前的[其他失真踏板构建，它们使用锗晶体管来获得“更好的”声音——你应该可以毫不费力地将这些便宜的替换成升级的模型。](http://hackaday.com/2010/12/27/building-a-germanium-fuzz-face-guitar-pedal/)

少数几个组件被整齐地焊接到一块 perf 板上，并放入一个坚固的金属外壳中，看起来它可以承受最恶劣的虐待。他的文章中有原理图和 BOM，所以让你远离摇滚周末的只是一点时间和一个烙铁——你还在等什么？
