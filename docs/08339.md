# 我看到了未来，它有成群的蚂蚁

> 原文:[https://hack aday . com/2011/08/16/I-have-seed-the-future-and-it-has-swarnoanoids/](https://hackaday.com/2011/08/16/i-have-seen-the-future-and-it-has-swarmanoids/)

![swarmanoid_robots_rescue_book](../Images/99913380d0e5d63d5e4f66e627893fb8.png "swarmanoid_robots_rescue_book")

想象一下，你想要一本书，它放在几个房间之外的书架上，但是你不想离开你的椅子。除了当场开发远程教育，你除了自己起身去取书之外别无他法——也就是说，除非你有一大群群机器人听命于你。

这个机器人群体是比利时布鲁塞尔自由大学的 Marco Dorigo 博士]的得意项目，至少可以说令人印象深刻。随着《碟中谍》式视频的播放，你会看到几个不同的机器人协同工作，飞行，攀爬，开车从书架上取书。这些机器人没有关于它们周围环境的信息，一旦目标被定位，它们就不得不相互学习和“交谈”以达到它们的目标。

看着这些机器人一起工作真的很令人惊讶，但不要相信我们的话。看看下面正在行动的群兽。
