# Bar2d2 实现自动化

> 原文：<https://hackaday.com/2009/02/24/bar2d2-gets-automated/>

[https://www.youtube.com/embed/ITcC3QazHyg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ITcC3QazHyg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当我们向你展示 [Bar2d2](http://hackaday.com/2008/09/29/the-pinnacle-of-modern-robotics-bar2d2/) 这个酒吧招待派对机器人时，还有一个巨大的障碍需要解决。这只是一个饮料分配器，虽然很棒，但实际上不能混合饮料。该问题已在本次更新中得到解决。现在，一台电脑处理所有的饮料订单，然后通过蓝牙将其发送到 bar2d2。酒就在你面前混合在 bar2d2 霓虹灯发出的令人舒适的蓝光中。你可以在这里查看[更多建筑图片](http://www.flickr.com/photos/28127470@N03/)。
