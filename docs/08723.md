# 机械臂跟随你自己的肢体运动

> 原文:[https://hack aday . com/2011/10/02/robotic-arm-follow-the-movements-of-your-own-limbs/](https://hackaday.com/2011/10/02/robotic-arm-follows-the-movements-of-your-own-limbs/)

![](../Images/dada0e316470e0dda56ed5dfa93a0a77.png "robotic-arm")

【亚历杭德罗】和他的朋友们最近完成了第一个刮擦式机器人手臂的原型。他们有一些很好的电子工作台设备用于这样的项目，但实际的制作工作要去厨房。

正如你在休息后的视频中看到的，他们在这个建筑中使用 PVC 作为原材料。平板是这样生产的:将一根 PVC 管从中间切开，在烤箱中加热直到变软，然后用一大壶水作为临时压机的砝码，将它压在两块地砖之间。组成关节的伺服电机的安装孔是用手钻钻的，该组件被固定在一张旧 CD 上作为基底。

组装完成后，他们将它连接到控制电路上，构建出一套你可以戴在手臂上的传感器。现在你的肘部，手腕和食指控制着伺服系统。该功能的演示在视频开始大约两分钟时开始。

我们已经看到了其他不使用机床制造机器人手臂的例子。这个由 ShapeLock 塑料制成的手臂是最有趣的例子之一。
