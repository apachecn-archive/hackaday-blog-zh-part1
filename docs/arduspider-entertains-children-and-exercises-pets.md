# ArduSpider 娱乐儿童和锻炼宠物

> 原文：<https://hackaday.com/2011/01/14/arduspider-entertains-children-and-exercises-pets/>

机器人蜘蛛有很多好的方面。它的设计精致而稳健，固件中编码的行为非常壮观。

车身是由碳纤维板之间的一块轻木制成的。腿是碳棒，使用两个伺服电机用于左右腿的移动，第三个伺服电机可以像飞机的滚动轴一样移动中间的腿。在 Arduino 的系统控制下，前面有一个红外传感器用于避障。更多硬件信息[请查看他的构建日志](https://docs.google.com/document/d/19LsFuKmBgKtkqmkHkKizFh4S8bjGPOoUeJypeRv5PNY/edit?hl=es&pli=1#)。

休息之后不要错过视频。你会看到这个小机器人可以手动控制，或者允许自由漫游。就像我们之前说的，这种行为太棒了。何塞不仅编写了有趣的特性，比如蜘蛛累了会坐下来休息一会儿，而且当它醒来时会跳到空中。对人类和猫科动物来说，这些运动看起来都很有趣；要是你的家猫能这么幸运就好了。
