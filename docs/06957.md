# 巨型示波器音乐显示器

> 原文:[https://hack aday . com/2011/02/28/giant-oscope-music-display/](https://hackaday.com/2011/02/28/giant-oscilloscope-music-display/)

![oscilloscope_wall](../Images/756291c5fedcbd174a4879ac5568b33e.png "oscilloscope_wall")

[Andrew]给我们发了一段他自制的示波器墙的视频，他用它来可视化他创作的不同音轨。这面墙是由他在工作室里用电线连接起来的九个旧的广播监视器组成的。起初，显示器被分成两组，每组四个，中央显示器单独运行，但进一步观察后，他似乎可以独立控制每个显示器的显示。

我们非常确定他没有使用实际的示波器来产生视觉效果，而是使用可视化软件来模拟 X-Y 显示模式下 O-scope 的输出。虽然这些屏幕是独立运行的，但如果能看到所有九个屏幕协同工作来显示一个大型可视化效果，那就太好了。[Andrew]提到高清版本正在开发中，因此我们可能会在该版本中看到更多功能。
