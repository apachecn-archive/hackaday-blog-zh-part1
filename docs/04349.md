# XBMC 挽着胳膊跑

> 原文:[https://hackaday.com/2009/10/30/xbmc-running-on-arm/](https://hackaday.com/2009/10/30/xbmc-running-on-arm/)

![beagleboard](../Images/fa6e26a2869dc63528b03a6da8f0c67b.png "beagleboard")

XBMC 团队发布了一个预告，展示了这个流行的开源媒体软件的 ARM 端口的当前状态。我们在休息后嵌入了它，你可以看到包启动和播放高清视频。从图中我们可以看到，系统解码信号很好，但图像渲染需要一些调整才能完成。

使用的硬件是一个 Beagleboard ，它运行 600 MHz ARM 处理器，具有 OpenGL 2D/3D 加速，通过 DVI 端口输出高清，售价约为 150 美元。3 英寸乘 3 英寸的主板可以使用 USB WiFi 加密狗连接到网络。虽然通过[黑掉电视固件](http://hackaday.com/2009/10/18/samsung-tv-firmware-hacking/)来集成 XBMC 还有很长的路要走，但我们会考虑将其中一个放到我们的高清电视后面，并摆脱娱乐中心后面笨重的个人电脑。

不能等待这个版本达到一个稳定的版本，不介意使用更大的硬件吗？[查看本指南](http://lifehacker.com/5391308/build-a-silent-standalone-xbmc-media-center-on-the-cheap)，在[200 美元的 Acer Aspire Revo](http://www.newegg.com/Product/Product.aspx?Item=N82E16883103228) 上设置 XBMC。
