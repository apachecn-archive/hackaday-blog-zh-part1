# 颂扬手机背后的设计原则

> 原文：<https://hackaday.com/2011/06/14/celebrating-the-design-principles-behind-cellphones/>

![](img/4cde22d8f3abfa3f40efdb2f582bc1ef.png "cellphone-design-contraints")

想要改善项目的最终外观吗？通过观察消费电子产品的选择，你可以学到很多东西。[Bill Hammack]探索可能是最精致的电子设备，手机。具体来说，[他讨论了每个手机制造商面临的七个设计限制](http://www.engineerguy.com/videos/video-cell-02.htm)。它们是:紧凑性与可用性、消费者偏好、能源可用性、经济资源和可用基础设施、材料知识、社会需求和文化约束。

任何一个在公共场所完成他们拼凑的项目的人都明白文化约束。尤其是当[在公共场所忘记你的背包](http://www.reddit.com/r/reddit.com/comments/hwnb3/the_other_day_i_left_my_backpack_in_a_field_this/)这些天可以让拆弹小组充分警惕。但是除了那些轶事，比尔偶尔看看手机真的展示了我们喜欢的智能设计，这要归功于进化过程，这已经成为 21 世纪的手表。休息之后看看他在视频里有什么要说的。
