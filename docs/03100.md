# 卷尺生成器

> 原文：<https://hackaday.com/2009/01/15/tape-measure-generator/>

![generator](img/f624becf5eb25ab44d6fccc3dc777d73.png "generator")

[韦斯特博士]想尽自己的绵薄之力来拯救地球。他用步进电机和卷尺制造了一个小型发电机。卷尺的卷尺部分换成了一些细绳。弹簧加载机构使弦自动重新缠绕，这很好。他展示了它连接在一条腿上，绳子系在另一条腿上，在他走路时发电。这很麻烦，但似乎相当有效。休息过后，你可以在视频中看到他醒来。每次拉动后，用飞轮或摩擦驱动来保持马达旋转，会有好处吗？他怎样才能操纵它只靠一条腿的运动来避免绊倒的危险呢？
