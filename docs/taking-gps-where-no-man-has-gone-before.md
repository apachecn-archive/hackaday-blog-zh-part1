# 带着 GPS 去没人去过的地方

> 原文：<https://hackaday.com/2011/11/14/taking-gps-where-no-man-has-gone-before/>

![](img/57ae10d851960e740045805f88e1dc9a.png "summit")

[Willem]有个朋友想带着 GPS 数据记录器去吉尔吉斯斯坦的一座无人攀登的山上。为探险建造的全球定位系统记录器到达了 T2 的顶峰，但是它并没有一路向上。自从伐木工回到伦敦后，[Willem]能够做一个完整的拆卸和故障分析。

数据记录器是围绕一个带有 GPS 单元和 MicroSD 读卡器的 [Jeenode](http://jeelabs.com/products/jeenode) 建造的。做了几块分隔板，其中两个坏男孩被安置在防水防尘的围栏里。由四节 AA 电池供电，数据记录器能够应对被扔下楼梯的严格测试以及伦敦的恶劣温度。然而，吉尔吉斯斯坦的荒野发生了变化。

从登山探险中获得的数据并不是最大的——一些电线在被扔进一辆俄罗斯卡车的后部并被推来推去后松了。AA 电池只能为数据记录器供电三天，而伦敦的电池寿命为 12 天。下一次旅行需要一些改进——一些隔热和不使用实芯电线——但[Willem]并没有发现他为朋友的下一次探险准备的漏洞。
