# 贝斯英雄结合了吉他英雄和舞蹈革命

> 原文:[https://hackaday . com/2011/07/27/bass-hero-combines-guitar-hero-with-dance-dance-revolution/](https://hackaday.com/2011/07/27/bass-hero-combines-guitar-hero-with-dance-dance-revolution/)

![](../Images/72a2429a1a9e3dfa475f709974db74f2.png "bass-hero")

站起来玩跳跳舞革命类型的游戏是如此多的工作。令人欣慰的是，[Jebadiah0001]通过改变吉他控制器来玩跳舞游戏，将剧烈运动的成分从游戏中移除。

他称之为 Bass Hero，因为 DDR 游戏只使用四个输入，将吉他控制器减少到四个弦乐按钮，就像电贝司一样。他的实现使用 GameCube 控制器连接到控制台。他把它拆开，以便够到钮扣接头。吉他上的每个琴弦按钮的一端连接到 GC 控制器上的按钮，另一端是公共连接。但他没有将这些信号直接拉至地面，而是通过扫弦致动器发送信号。这样，玩家可以准备好正确的按钮，然后在正确的时间弹奏以完成循环。

这无疑使较难的 DDR 级别变得容易了许多。休息后在视频里自己看吧。
