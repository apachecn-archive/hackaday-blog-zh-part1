# Android 进入 Linux 内核 3.3

> 原文:[https://hack aday . com/2012/03/19/Android-rolled-into-Linux-kernel-3-3/](https://hackaday.com/2012/03/19/android-rolled-into-linux-kernel-3-3/)

![](../Images/5732a007fc02d06f60dde1a4e0051b33.png "android-in-the-linux-kernel")

Linux 内核的最新版本刚刚在周日发布，我们认为有一点额外的好处值得考虑。似乎 Android 的许多变化都融入了 3.3 版本的 Linux 内核。

这听起来可能不多，但是这是开源力量的一个很好的例子。由于基于 Linux 内核的设备特定的变化必须在相同的许可下发布，硬件制造商被迫发布它们的源代码。但是通常情况下，只有当你有足够的专业知识去钻研他们的代码并找到你想要的部分时，这才会对你有所帮助。但是随着这些变化融入到主内核中，你应该能够相对容易地在 Android 设备上运行你自己的发行版；硬件支持已经有了。当然，交叉编译操作系统仍然需要很多专业知识。

这可能有可能开放旧的 Android 硬件作为开发平台。想想 Chumby 硬件在机器人项目中的应用。现在，如果你的老式蓝牙和 WiFi 手机有一个股票内核，在你自己的项目中使用起来也很容易和开放，会怎么样？
