# IPhone OBD-II 应用程序

> 原文:[https://hack aday . com/2008/09/27/use-rev-on-your-iphone-to-read-your-OBD-ii-port/](https://hackaday.com/2008/09/27/use-rev-on-your-iphone-to-read-your-obd-ii-port/)

由 [DevToaster](http://devtoaster.com/index.html) 开发的 [Rev](http://devtoaster.com/products/rev/index.html) 是为 [iPhone](http://www.mahalo.com/Iphone) 和 [iPod Touch](http://www.mahalo.com/IPod_Touch) 开发的一个应用程序，可以实时监控来自 [OBD-II 端口](http://www.obdii.com/)的车辆 ECU 数据。Rev 与 WiFi OBD-II 加密狗接口。

如果您的“检查发动机”灯亮起或闪烁，REV 能够检查发动机代码，列出车辆中存储的所有发动机代码，并重置存储的代码或检查发动机灯。

Rev 能够实时监控；车速、转速、油耗、发动机冷却液温度、燃油压力、计算的发动机负荷、节气门位置、进气歧管压力、进气温度、正时提前、空气质量流量、燃油油位、大气压力、蒸发排放系统蒸汽压力和燃油微调。
