# 克拉布挑战赛冠军揭晓

> 原文：<https://hackaday.com/2008/09/18/crabfu-challenge-winners-announced/>

![](img/1ca0be37e8e023a97243acea73257479.png "crabfu_hack")

Trossen 机器人博客[宣布了“克拉布挑战”](http://blog.trossenrobotics.com/index.php/2008/09/17/the-crabfu-challenge-winners-announced/http://blog.trossenrobotics.com/index.php/2008/09/17/the-crabfu-challenge-winners-announced/)的获胜者。由[Crabfu]发起的挑战是制造一个充满个性的机器人。它不需要有一个目的或者是自主的，它只需要充满个性。

第一名的获胜者，如上图所示，在他的视频中“眨眼”了一会儿，赢得了我们的好感。奇怪的是如此简单的东西却能增添如此多的生命。
