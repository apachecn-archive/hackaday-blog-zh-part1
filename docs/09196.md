# 使用 Arduino IDE 进行图形编程

> 原文:[https://hack aday . com/2011/11/30/graphical-programming-with-the-arduino-ide/](https://hackaday.com/2011/11/30/graphical-programming-with-the-arduino-ide/)

![](../Images/b10e00c938a394b5a45dccec1165c837.png "ardublock")

Arduino 是第一个优秀的嵌入式开发套件，提供了一个很好的电子入门，并有可能让儿童进入编程。[David]认为向非程序员扔 C 不是学习编程的最好方法，所以他开发了 Arduino 的图形化编程语言 [ArduBlock](http://blog.ardublock.com/) 。

在我们这个时代，我们已经看到了许多图形化的、基于块的编程语言，最著名的是 [Scratch](http://scratch.mit.edu/) 。[大卫]发现了一个名为 [OpenBlocks](http://education.mit.edu/openblocks) 的项目，它是 [Android 应用发明者](http://www.appinventorbeta.com/about/)的基础。

[David]分叉了 OpenBlocks 项目，开始开发他的新图形编程语言。ArduBlocks 使用 Arduino IDE，所以在 C 语言中用 Arduino 实现的一切都应该可以用 ArduBlocks 实现。在模块的设计中投入了大量的思想——第一次迭代太“忙”了，但是[David]清理了它，使项目变得可以理解。

对于一个完全的初学者来说，我们想象不出比这更好的了。ArduBlocks 对孩子们来说很棒，我们迫不及待地想看到 ArduBlocks 在触摸屏上的正确实现。
