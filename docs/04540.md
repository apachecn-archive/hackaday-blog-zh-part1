# 新宠物，413 项目

> 原文：<https://hackaday.com/2009/12/14/new-pet-project-413/>

![I wonder if anyone ever mouses over images anymore. Oh silly me, xkcd reference.](img/3c8fb66f77bf169fb970d9b073b3fabd.png "CIMG0835")

[Jacob]提交了他团队的最后一个项目，项目 413。虽然听起来像是直接出自动作片，但别担心，这不是下一个终结者。而是一个 [eBox](http://www.embeddedpc.net/eBox2300/tabid/110/Default.aspx) (代替 EEE pc)、omni wheels、马达、电池和点睛之笔——一个仓鼠雪貂球的组合。如果这对于某个网络漫画来说听起来很熟悉，那么你是对的，因为 XKCD 是他们的主要灵感。遗憾的是，网络摄像头和“有灵魂”功能并没有出现在他们的版本中，但通过 Wii 遥控器控制总是一个优势。休息之后看一段视频。
