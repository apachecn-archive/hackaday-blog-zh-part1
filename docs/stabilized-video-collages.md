# 稳定的视频拼贴画

> 原文：<https://hackaday.com/2008/06/12/stabilized-video-collages/>

 <http://www.flickr.com/apps/video/stewart.swf?v=49235>

   
这是一些漂亮的作品。片段特点[多个视频流稳定](http://www.flickr.com/photos/ibftp/2425386864/in/set-72157604726171955/)然后组装成一个整体。首先，[ibftp]使用了 [Motion 3](http://www.apple.com/finalcutstudio/motion/) (苹果 Final Cut Studio 2 的一部分)中的“防抖”功能来消除剪辑中的相机抖动。然后，他能够将“融合”设置为“乘法”的视频混合在一起。如果您可以使用这些工具，自己动手应该不会太难。我们确信 SIGGRAPH[中的某个人已经在尝试做同样的事情了。如果你想看到图像稳定真的有所不同，看看下面嵌入的 Zapruder 电影的稳定版本。](http://www.mahalo.com/Siggraph)
