# 野外的 Printrbot 文件

> 原文:[https://hack aday . com/2012/02/05/printrbot-files-in-the-wild/](https://hackaday.com/2012/02/05/printrbot-files-in-the-wild/)

![](../Images/62db1c2ba74228f284772cea4112c0d3.png "printr")

经过几个月的热切等待，【布鲁克·德拉姆】终于[发布了他的创新型 3D 打印机](http://www.thingiverse.com/thing:16990) [Printrbot](http://printrbot.com/gallery/) 的文件。如果你在 Kickstarter 上没有订购其中的一个，你现在可以打印你自己的一套零件。

去年 11 月[Brook]将他的 print rbt[送给了世界](http://hackaday.com/2011/11/18/the-cheapest-and-easiest-3d-printer-weve-seen-so-far/)，承诺价格非常便宜，非常容易制造，并且具有相对较高的打印质量。printer bot 的简单性令人惊讶，这可能导致 printer bot 在最初的 [Kickstarter](http://www.kickstarter.com/projects/printrbot/printrbot-your-first-3d-printer?ref=card) 上获得了价值 83 万美元的资金。虽然 3D 打印部件的文件现在已经不在了，但是除了一张 [Flickr 幻灯片](http://www.flickr.com/photos/printrbot/sets/72157628821699187/with/6678242223/)之外，仍然没有任何关于如何构建它的说明。

[Brook]承诺更早发布 Printrbot 的文件，但我们猜测他一直忙于打印和组装他在 Kickstarter 中声称的 1200 个 Printrbot。当我们谈到廉价 3D 打印机的话题时，【Richard Sum】，SUMPOD 背后的英国绅士发来了他的一台 600 美元的打印机[铣削 MDF](http://3d-printer-kit.com/?p=717) 和连续挤压[七个小时](http://3d-printer-kit.com/?p=694)的链接。伙计们，我们正处在星际迷航式的复制人的尖端。
