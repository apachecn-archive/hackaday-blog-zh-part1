# 车床改造

> 原文：<https://hackaday.com/2010/02/18/lathe-modification/>

[https://www.youtube.com/embed/maB28gXoZV4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/maB28gXoZV4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

NerdKits 的人得到了一台崭新的史密斯 1220xl 车床/铣床。然而，他们发现调整 Z 轴并不像他们期望的那样精确。他们决定[通过安装一个数字卡尺和一个 LCD 界面](http://www.nerdkits.com/videos/digital_calipers_dro/)来显示调整来解决这个问题。文章写得很详细，结果似乎很好。
