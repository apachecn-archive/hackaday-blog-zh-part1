# 1000W 感应加热器

> 原文：<https://hackaday.com/2010/06/04/1000w-induction-heater/>

[蒂姆·威廉姆斯]喜欢用他做的这个感应加热器加热东西。在峰值时，它可以使用 1000 瓦，正如你在视频中看到的，这足以加热，燃烧和融化大量不同的物体。机箱设计使用中心分隔器来隔离开关噪声和磁场，整个单元封装在铝中，因为它不会因杂散磁场而发热。他在卖计划和工具包，以防你需要，但我们不知道我们会用它来做什么。
