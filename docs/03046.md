# 叽叽喳喳洗衣机

如果你家里连一个推特设备都没有，那你就落伍了。最新的一款产品是一台会发微博的洗衣机。[Ryan]告诉我们，他总是不小心忘记从洗衣机中取出衣服，导致衣服发臭发霉。现在，他的洗衣机叽叽喳喳地宣布它完成了。它在房子里还有一个标志，显示它的状态，以帮助他记住。我们已经看过[未出生的婴儿](http://hackaday.com/2008/12/12/baby-twitters-via-kicks/)和[烤面包机推特](http://hackaday.com/2008/12/16/twittering-toaster/)，还剩下什么？还记得那些本该让你知道某些食物没了的冰箱吗？为什么我们还没有看到一个会叽叽喳喳的冰箱？
