# 通用 Commodore 64 墨盒加速演示生产

> 原文：<https://hackaday.com/2012/01/11/universal-commodore-64-cartridge-speeds-up-demo-production/>

![](img/843e690c39f789e3a0ba7fa45c7329ed.png "c64")

作为他尊敬的 64 代准将的终身爱好者，[Frank]在编写 C64 演示时一直在寻找一种加快开发时间的方法。他的解决方案是一个通用 C64 墨盒，可以通过 USB 接口连接到电脑。

该板由一个 CLPD 和一个微控制器供电，该微控制器加载了来自[Frank]以前的 [C64 USB 控制器适配器](http://hackaday.com/2011/07/31/commodore-64-usb-controller-adapter-for-your-pc/)的代码。一个 16 兆位的闪存芯片能够存储 31 个经典游戏，如*陷阱，挖挖，*和*矿脉赛跑者。*

在他的 Google+公告上，[弗兰克]说这是一个非常早期的原型。他计划缩小电路板的尺寸，使之适合标准的 C64 盒式磁盘，而且微型计算机和 CLPD 的固件还没有完成。也就是说，[弗兰克]确实有一个董事会做他想让它做的事情:极快的 C64 开发。

看看[Frank]的演示，他在几秒钟内编译并重新上传了一个简单的演示到他珍爱的电脑上。这比使用 [1541 Ultimate](http://www.1541ultimate.net/content/) 或其他 SD 读卡器要快得多。
