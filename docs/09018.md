# 十六进制揭示了你的数据总线持有的秘密

> 原文：<https://hackaday.com/2011/11/07/hex-out-reveals-the-secrets-your-data-bus-holds/>

![](img/6b221b99bf7a532f08c711ca1199c270.png "hex-out")

[Quinn Dunki]正在寻求增强她在电子工作台上可用的工具，并将[HEX 建成模拟逻辑嗅探器](http://quinndunki.com/blondihacks/?p=610)。该设备读取 8 或 16 位输入，在 7 段显示器上显示这些连接的当前状态。这要求你能自如地阅读十六进制代码，但如果你不，这几乎就像学习闪存卡；不久你就能不假思索地阅读它们了。

她在博客上分三部分讲述了设计和建造过程。上面的链接是第一部分，她分享了承载显示硬件的顶层的开发过程。其他两部分将会在接下来的几周内呈现给你。

你会注意到她在这部分项目上的设计仍然需要大量的点对点焊接，尽管她蚀刻了自己的电路板。我们没有仔细观察，但如果可以的话，这似乎是值得的。
