# USB 外壳的搅拌板

> 原文:[https://hack aday . com/2011/03/25/搅拌盘来自 usb 外壳/](https://hackaday.com/2011/03/25/stirring-plate-from-usb-enclosure/)

![](../Images/37287b02984b1de77403b38454c46a61.png "usb-enclosure-stirring-plate")

用硬盘外壳做了一个自己的搅拌盘。这是和我们几周前看到的一模一样的[黑客。一块磁铁被粘在电脑风扇的中心，这使得烧杯内的磁性胶囊旋转。这产生了一个漩涡，均匀地混合液体。](http://hackaday.com/2011/03/16/beginner-project-super-cheap-magnetic-mixer/)

使用硬盘驱动器外壳是一个绝妙的主意。它被设计成放在显眼的地方，这样你就能获得非常漂亮的外观。它的大小也正好适合风扇本身。安装在外壳顶盖上的电位计允许变速，DC 筒形千斤顶用作电源。现在我们想起来，这可能是我们见过的外置硬盘驱动器外壳的最佳用途(甚至掩盖了它的原始用途)。休息之后，请查看该项目的视频和图像库。

我们把这归类为啤酒黑客，因为搅拌板通常是由爱好者建造的，用于培养家庭酿造中使用的酵母发酵剂。
