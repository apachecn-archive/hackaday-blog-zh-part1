# 让一台 92 年的自制扫描仪重新运转起来

> 原文:[https://hackaday . com/2012/03/19/get-a-home-built-scanner-from-92-up-and-running-again/](https://hackaday.com/2012/03/19/getting-a-home-built-scanner-from-92-up-and-running-again/)

![](../Images/d40e2f9602ccef513ebffcee3e9eb464.png "scanner")

1992 年，[Arpi]没有太多时间玩忍者神龟，任天堂，以及其他他的极客伙伴们成长过程中浪费的美好时光。他正忙着为他的准将 64 号建造扫描仪。虽然这个令人印象深刻的建筑可能会被时间埋没，但他还是把他的项目从阁楼里拿出来，参加了一个“尝试再次使用它”的聚会。虽然这个派对不是正式的比赛，但我们会说那晚[Arpi]带着最极客的信誉走回家。

虽然没有建造细节，但从[画廊](http://gabucino.be/files/c64scan.html)可以收集到一堆关于这台机器是如何建造的信息。我们很确定这个建筑的大部分曾经是一台打字机，看起来好像在某个地方有一个[挡风玻璃雨刷马达](http://thot.banki.hu/arpi/scanner/arpie/IMG_9059.JPG)。像这个[完全不相关但相似的建造](http://www.sentex.net/~mwandel/tech/c64scanner.html)，【Arpi】的扫描仪使用光敏电阻和几个发光二极管将图像数据传输到[定制软件](http://thot.banki.hu/~gabucino/IMG_0130.JPG)。如果你想知道，是的，[古老的 5 1/4 软盘](http://thot.banki.hu/~gabucino/IMG_0131.JPG)仍然可读——这是这些磁盘上巨大扇区的少数优点之一。

休息之后，看看这台扫描仪的运行视频，如果你的阁楼里有一台几十年的老黑客(还记得你制作的声学调制解调器吗？)，通过[发送到提示行](http://hackaday.com/contact-hack-a-day/)。
