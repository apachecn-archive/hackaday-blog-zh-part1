# 黑客的边缘安全有助于归还被盗电脑

> 原文:[https://hack aday . com/2010/12/25/a-hackers-marginal-security-helps-return-失窃电脑/](https://hackaday.com/2010/12/25/a-hackers-marginal-security-helps-return-stolen-computer/)

围拢过来，听听黑客如何智取罪犯的故事。[Zoz]被抢劫了，他们拿走了他的台式电脑。走了，对吧？没有。由于他的电脑配置和罪犯的愚蠢的特殊组合，他把它拿了回来。他在他的 Defcon 18 演讲中分享了这个故事，视频在休息后嵌入。

(Zoz 的)第一点运气来了，因为他已经设置机器使用动态 DNS 服务，通过脚本更新。因为罪犯没有清除硬盘，所以他能在网上找到机器。从那里，他发现他可以 SSH 到它，甚至使用 VNC 窃听新主人。这个，加上他安装的键盘记录器，让他得到了他需要的所有信息；这个家伙的名字，出生日期，网站的登录和密码信息，最重要的是他的街道地址。他把这些有趣的数据交给了警方，他们设法恢复了系统。
