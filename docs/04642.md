# 最简单最无用的机器

> 原文：<https://hackaday.com/2009/12/30/simplest-most-useless-machine/>

[https://www.youtube.com/embed/Z86V_ICUCD4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Z86V_ICUCD4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Brett]发布了他最没用的机器的版本。他的项目给了我们一个机会来屠杀标题中的语言，因为它使用了我们在[这些机器](http://hackaday.com/2009/09/19/show-us-your-most-useless-machine/)中见过的最少的零件。逻辑由我们的朋友 555 定时器控制。再加上一个伺服电机、两个开关、三个电阻、两个电容、一个二极管和一个电池组，你就大功告成了。最难找到的地方是伺服，但检查在业余爱好/钢筋混凝土商店。如果你不必下零件订单，这可能是你的下一个冲动项目。
