# 视觉障碍者的触觉 GPS 运动鞋

> 原文:[https://hack aday . com/2011/04/13/haptic-GPS-为视障人士设计的运动鞋/](https://hackaday.com/2011/04/13/haptic-gps-sneakers-for-the-visually-impaired/)

![gps_haptic_shoe](../Images/f6283f0c96b84e556ef6fe5d47cc6a34.png "gps_haptic_shoe")

当你缺乏观察世界的能力时，这个世界会变得很难驾驭。全球有许多视力受损的人，一些数字声称多达 4000 万人受到影响。虽然手杖和导盲犬可以提供很大的帮助，但印度惠普实验室多模态交互小组的[Anirudh]和印度浦那(COEP)工程学院的一些学生一直在努力为盲人构建触觉导航系统。

[Anirudh Sharma 和 Dushyant Mehta]在 COEP 举办的麻省理工学院媒体实验室研讨会上首次展示了他们的触觉反馈鞋设计。在目前的形式下，谷歌地图和 GPS 数据来自 Android 设备，通过蓝牙传输到 Arduino。Arduino 然后激活安装在鞋子衬垫上的四个 led 中的一个，这些 led 用于指示个人应该向哪个方向行进，以便安全到达目的地。虽然目前的迭代使用 led，但在最终版本中，它们将被替换为小型振动电机。

我们一直是辅助技术黑客的粉丝，我们认为这个很棒。这个概念运作良好，[就像我们在](http://hackaday.com/2011/03/20/gps-enabled-bag-allows-for-carefree-city-roaming/)之前看到的那样，所以现在的问题就是让这个项目变得更好，并交到那些需要它的人的~~手中~~。
