# 雅各布的梯子在软盘盒里就像在家里一样

> 原文:[https://hackaday . com/2011/03/30/Jacobs-ladder-makes-in-home-a-floppy-disk-box/](https://hackaday.com/2011/03/30/jacobs-ladder-makes-itself-at-home-in-a-floppy-disk-box/)

![](../Images/8785f8c3321ee1c80534dd5f099ae66a.png "OLYMPUS DIGITAL CAMERA")

[Plasanator]将他的[雅各布之梯放在一个熟悉的围栏里](http://www.instructables.com/id/Jacobs-ladder-with-Auto-Coil)，增加了一点安全性。制造一个这样的东西并不需要很多元件，但是为了让[获得高电压，你需要某种线圈](http://hackaday.com/2011/03/22/high-voltage-how-to-cook-your-goose-in-62-easy-steps/)。他使用一辆旧车的电气系统中的一个，然后在它周围建立一个 15mf 220V 的大电容器，一个通常用于家庭布线的调光开关，端子板，以及一些用于火花穿过的蒸棒或衣架。

休息后的视频显示了这一点，我们同意[Plasanator]的说法，这是对你的万圣节装饰的一个很好的补充。当然你想让手指远离危险的部分，这就是外壳和钥匙锁发挥作用的地方。我们不确定他用什么做了这个直立的圆筒，但是这个底座是从过去炸出来的。还记得那些曾经骄傲地坐在每张桌子上，向你展示你所拥有的信息有多么重要的人吗？

不想这样玩高压电？你可以[用 EL 线](http://hackaday.com/2010/12/12/jacobs-ladder-using-el-wire/)造一个假的。
