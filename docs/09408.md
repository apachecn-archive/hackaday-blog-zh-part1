# 在家制作纯咖啡因

> 原文:[https://hack aday . com/2012/01/01/在家制作纯咖啡因/](https://hackaday.com/2012/01/01/making-pure-caffeine-at-home/)

如果你曾经想过从咖啡豆中提取咖啡因，那么本就是你要找的人。上次我们遇到他的时候，他已经用一些化学物质、管道配件和一些二氧化碳生产了一些[气凝胶单片](http://hackaday.com/2011/11/16/making-aerogel-at-home/)。我们猜测他需要他的超临界干燥室的另一种用途，所以在查看了几个[专利](http://www.google.com/patents/US5288511?printsec=abstract&dq=5288511&ei=b1b_TsycDeTd0QH6o42GAg&source=gbs_overview_r&cad=0#v=onepage&q=5288511&f=false)后，他决定在他的车库里制造纯咖啡因。

提取从干燥室内的生咖啡豆开始。液态 CO2 被泵入并加热到超临界点。几个小时后，咖啡因将从咖啡豆中提取出来，并从容器中排出。目前，这一过程会产生一种含有咖啡因的黑色粘性物质。[Ben]用二氯甲烷对此做了进一步的改进，他将在未来的视频中记录这个过程。

当然不言而喻，这个版本有点危险。我们相信[本]有能力知道他在做什么，但我们不会为互联网上的每个小丑背书。另外，两勺咖啡因[会杀死你](http://abcnews.go.com/Health/Sleep/british-man-dies-caffeine-overdose/story?id=12033005#.Tv9n9NQS3pU)，所以也不要乱用。
