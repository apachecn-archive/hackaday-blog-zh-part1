# 硬盘驱动器作为旋转输入

> 原文：<https://hackaday.com/2008/11/12/hddj-hard-drive-as-rotary-input/>

![dj_hdd](img/54a3ef5b2017179066e5845a97bcebb1.png "dj_hdd")
