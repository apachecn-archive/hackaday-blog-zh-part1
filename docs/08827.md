# 由回收零件制成的激光水平仪三脚架

> 原文:[https://hack aday . com/2011/10/14/laser-level-tripod-made-from-recycled-parts/](https://hackaday.com/2011/10/14/laser-level-tripod-made-from-recycled-parts/)

![laser_level_tripod](../Images/3de8011773618cdd7f9b1e0b0bb1ba67.png "laser_level_tripod")

[msuzuki777]自称是“懒惰的老极客”，空闲时间太多了。他最近拿起了一个激光十字架，并打算利用这段时间为家里的各种项目制作一个激光三脚架。

他拿出一个旧的相机三脚架，改装了一个不用的 CD 首饰盒，这样它就可以拧到传统的相机支架上。他在平台上增加了三个螺栓，在上面他安装了另一个 CD 盒，让他可以调整激光平台和三脚架。

他为激光器组装了一个简单的电源，然后把它安装在一对 CD 上，一张夹在另一张上面。然后，CD 平台被放在一个旧 CD 播放器的内脏上，允许他将激光笔旋转到任何方向，而不必重新调平。

激光十字三脚架看起来确实有点复杂，但[msuzuki777]说它工作很好，让他可以轻松地挂照片之类的东西。他还提到，他想用 Arduino 来自动化调平过程，这是我们希望看到的。
