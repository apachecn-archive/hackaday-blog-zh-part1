# 这个玩具对讲系统比一对锡罐和一些绳子要好的多

> 原文:[https://hackaday . com/2011/12/06/this-toy-intercom-system-is-way-better-than-pair-of-tin-cans-and-some-string/](https://hackaday.com/2011/12/06/this-toy-intercom-system-is-way-better-than-a-pair-of-tin-cans-and-some-string/)

![toy-intercom-system](../Images/62893001d45001a9d9ec8a284007abe7.png "toy-intercom-system")

在他的博客上，[Kenneth Finnegan]最近展示了一个他小时候常玩的有趣玩具的复制品，一个电话对讲系统。设置非常简单，只需要一对模拟电话、一块电池和一个电阻。

这些电话通过标准电话线相互连接，但[Kenneth]使用 9v 电池向回路中引入一个小的偏置电流，使两端的扬声器都能听到彼此的声音。他还在电路中添加了一个小 LED，这样当两部手机都摘机时，就会有一个可视的指示。

目前设置非常简单，尽管[Kenneth]确实有一些想法来增强他的内部通话系统。除了其他功能之外，他还希望当本地电话被接听时，远程电话能够响起来。

电话技术并不新鲜，但是只要几美元(或者更少),你的孩子就可以享受几个小时的娱乐，就像肯尼斯当年那样。
