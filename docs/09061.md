# 你会用这个充满激光的网球射瞎你的眼睛

> 原文:[https://hackaday . com/2011/11/11/你将用这个充满激光的网球射掉你的眼睛/](https://hackaday.com/2011/11/11/youll-shoot-your-eye-out-with-this-laser-filled-tennis-ball/)

![laser-ball](../Images/13d7040f0b16de6c75347872ba76aab9.png "laser-ball")

我们爱激光，你们爱激光……谁不爱？

[Matt Leone]最近[将他对激光的热情发挥到了顶点](http://www.instructables.com/id/Laser-Ball/?ALLSTEPS)并建造了一个叫做激光球的小东西。受够了网上泛滥的 LED 立方体，他说激光球是镇上的新警长——我们倾向于同意。

他在一个标准的网球上钻了几个洞，并在上面安装了 14 个红色激光二极管。在他把激光器安装到球上之前，他用一个小的衍射光栅修改了每一个激光器，以使显示更加生动。激光连接到一个微小的微控制器上，微控制器和一个小的可充电脂肪电池一起被塞在球里。

虽然激光球已经很棒了，但[Matt]认为它还没有完成。利用 Adafruit 公司的红外遥控包，他增加了从远处触发激光球光模式的能力。你知道，只是为了好玩。

一定要看看下面的视频，看看激光球在行动！
