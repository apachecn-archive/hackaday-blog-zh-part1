# 10×10 LED 矩阵

> 原文:[https://hackaday.com/2010/02/26/10x10-led-matrix/](https://hackaday.com/2010/02/26/10x10-led-matrix/)

![](../Images/3a55188bf12b073ff8515a35c43423f6.png "10x10-led-display")

[KopfKopfKopfAffe]刚刚完成了一项为期 3 年的爱的劳动，创造了这个[10 x 10 LED 矩阵](http://hackedfrompieces.wordpress.com/2010/02/23/projektbunt/)。这超过了今天早些时候的 [Shiftbrite table](http://hackaday.com/2010/02/25/shiftbrite-coffee-table/) ，它在显示屏上实际显示了 100 个 led。这些 led 的成本比 Shiftbrites 低得多，但由于他们没有自己的板载控制器，这个项目需要更多的后端工作。总共 25 个 ATmega8 微控制器驱动该显示器，依次通过 RS-232 连接到计算机进行控制。

我们喜欢[旅鼠](http://en.wikipedia.org/wiki/Lemmings_%28video_game%29)动画。这让我们意识到，这将是一个伟大的候选人发挥[超级像素兄弟](http://hackaday.com/2010/02/19/update-most-interesting-game-in-64-pixels/)。休息之后，请观看视频，了解硬件快照以及 matrix 的运行视频。你可能还想细读一下[翻译的工作日志](http://translate.google.com/translate?hl=en&sl=de&tl=en&u=http%3A%2F%2Frgb-led-matrix.blog.de%2F)。
