# 可控刚毛机器人

> 原文：<https://hackaday.com/2009/06/16/controllable-bristlebot/>

[sprite_tm]的项目我们已经在过去报道过[，他将流行的 bristlebot 发挥到了极致，创造了一个](http://hacker.org/?s=sprite_tm)[可控版本](http://spritesmods.com/?art=bristlebot&f=had)。一个[鬃毛机器人](http://www.evilmadscientist.com/article.php/bristlebot)由一个小型振动马达组成，该马达在牙刷头部装有电池。这些微型机器人随机嗡嗡作响，他试图驯服它们。他使用了一个双刚毛机器人平台，并添加了一个来自激光鼠标和 ATtiny13 的光学传感器。光学传感器用于确定机器人的相对运动，从而可以相应地调整电机。他还有一个机器人使用传感器在地板上找到标记并保持在界限内的视频。虽然它不太精确，但它的行为就像传统的循线机器人。
