# 阿达果庆祝阿达·洛芙莱斯日

> 原文:[https://hack aday . com/2011/10/07/ada fruit-beheritis-ada-lovelace-day/](https://hackaday.com/2011/10/07/adafruit-celebrates-ada-lovelace-day/)

![ada_lovelace_day](../Images/f80f7232fcbecc63e42df270e5b49f31.png "ada_lovelace_day")

如果你还不知道，今天是阿达·洛芙莱斯日，所以 Ladyada 和 Adafruit Industries 在这一天庆祝女性在科技、艺术、工程和电子领域取得的成就。具体来说，重点是帮助塑造和促进该行业的女性制造商/黑客。

Adafruit Industries 每小时都会报道一位有影响力的女性，向她的工作和对科技/制造商/艺术/科学领域的贡献致敬。这一天你会看到很多熟悉的面孔，包括杰出的电子人[Jeri Ellsworth]，Make 自己的[Becky Stern]，以及[Sherry Huss]和[Louise Glasgow]，Maker Faire 的两位女主角。

简历当然值得一看——除了一些更熟悉的面孔，这是一个了解一些你以前可能没有听说过的制造商的好机会。

作为一个额外的奖励，Adafruit 使用他们网站上的优惠券代码为所有从商店购买的商品提供 10%的折扣，所以一定要查看一下！
