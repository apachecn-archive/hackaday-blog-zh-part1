# 汤姆 GS 和汤姆 MP

[https://www.youtube.com/embed/a0e6oujZOwY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/a0e6oujZOwY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[丹尼斯]向我们透露了他的项目。这看起来像两个项目使用非常相似的硬件。第一个是大部头；一个看起来可以挂在钥匙链上的微型游戏系统。这显然是这个系统的第二次修改，基于他的一些评论。他做得很好，看起来很结实，很实用。它甚至包括一个 zigbee 无线模块，用于与他人通信。他表示，图形应该大致与 gameboy 的进展相同。第二款是 Tome MP，这是一款便携式媒体播放器，大小和第一代 iPod nano 差不多。它配备了蓝牙，尽管他将在下一次迭代中删除它，以支持 zigbee。我们真的很想看到更多关于这些项目背后的设计信息。
