# PS3 枪/控制器混合

> 原文:[https://hackaday.com/2011/03/29/ps3-guncontroller-hybrid/](https://hackaday.com/2011/03/29/ps3-guncontroller-hybrid/)

![](../Images/d90fc581c208bfcbcbdaeb8555fc93c1.png "ps3-gun-controller-hybrid")

[Luis]对他的游戏控制器非常挑剔。他想改装一个六轴控制器来适应 Nerf 枪体，但是没有足够的空间来容纳所有的组件。逛了一会儿商店后，他发现了一个由 Namco 制造的有线枪支控制器，这是为游戏时间危机开发的。他拿起一个，开始工作[用从无线控制器中取出的一套替换内脏](http://www.youtube.com/watch?v=QdL8_KkylBI)。

像这样的 mod 的大部分工作是扩展每个组件的范围。打开枪控制器的外壳后，[Luis]开始准备和焊接六轴控制器 PCB 上的所有二十个触点，然后完成每个重新定位的组件所需的连接。这确实让我们想知道，模拟棒发出的信号中是否会引入一些噪声元素？他提到其中一个有“故障”，但那可能是因为他用的是易贝的二手控制器。

我们从中获得了一些好的建议。由于塑料外壳的设计可以安全地容纳每个原始 PCB，[Luis]将它们重新用作替换组件的安装面。对 protoboard 进行一点创造性的使用，然后在油漆店呆一段时间，你就完成了。休息之后，请观看整个过程的视频，其中也分享了最终的结果。
