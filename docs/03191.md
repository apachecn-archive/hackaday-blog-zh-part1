# 红外控制调光器

> 原文:[https://hackaday.com/2009/02/07/ir-controlled-light-dimmer/](https://hackaday.com/2009/02/07/ir-controlled-light-dimmer/)

![dimmer](../Images/5f387e5c5ae3636eb2a0067120bbe7a0.png "dimmer")

这是一个非常有用的设备。这是一个由红外线控制的调光器。它与主电源保持一致，并控制有多少光线通过。你可以用电视遥控器调节亮度。终于，我们可以退休了我们的[拍板](http://www.youtube.com/watch?v=-XUOhjW2AXM)！这些都是可以买到的，但是自己制作更有趣。你可能需要在他们的网站上挖掘一些原理图和编程。那里还有很多其他项目，所以玩得开心。休息之后你可以看到一段视频。
