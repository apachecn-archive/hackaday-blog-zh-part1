# 乐高 3D 打印

> 原文:[https://hackaday.com/2010/10/18/3d-printing-with-lego/](https://hackaday.com/2010/10/18/3d-printing-with-lego/)

![](../Images/3239381395121a720273991862ce000f.png "lego-3d-printing")

用 2400 块乐高积木和大量的耐心，【威尔·戈尔曼】[建造了一台乐高 3D 打印机](http://battlebricks.com/makerlegobot/)。它类似于 RepRap 或 Makerbot，但它不是挤出塑料，而是使用预挤出的积木(又名乐高积木)。在装置上方延伸很远的灰色墙壁是一个装着原材料的进料仓。Java 应用程序获取 MLCad 文件，并将其翻译成打印机的构建指令。这些指令然后通过 USB 发送到设备。休息之后再看。

现在只需要把这个和乐高分拣机结合起来，就能源源不断地得到砖块。
