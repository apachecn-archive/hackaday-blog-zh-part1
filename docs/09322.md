# 不用离开沙发就能控制你的圣诞灯

> 原文：<https://hackaday.com/2011/12/18/controlling-your-christmas-lights-without-ever-getting-off-the-couch/>

![remote-xmas-tree-light-switch](img/a409e01382cbc58c10b3f2fd93e1c507.png "remote-xmas-tree-light-switch")

14 岁的康纳·史密斯(Connor Smith)在这个假期一直很忙，他在想办法改善家里的照明状况。

几周前，他组装了这个三通道灯控制器来切换他父母的外部灯，结合了一个 Arduino 来控制。Arduino 用于在指定的时间间隔打开和关闭频道，以便在房子的外部创建一个简单的灯光表演。他不满足于只有几串闪亮的灯，他把他的控制器拿回里面做了一些额外的修改。

他已经厌倦了每天爬在圣诞树后面插上插下，决定让自己过得轻松些。他从一台旧 VCR 上拆下 IR 接收器，并使用非远程库将其与灯光控制器中的 Arduino 接口。在花了一点时间对电视遥控器上两个不常用按钮的值进行解码后，他给自己做了一个圣诞树灯开关，他可以在房间的另一端激活它。
