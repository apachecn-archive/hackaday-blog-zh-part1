# 快看，我的衣服是电动的。不，说真的

> 原文:[https://hacka day . com/2012/02/28/check-it-out-my-would-is-electric-no-serious/](https://hackaday.com/2012/02/28/check-it-out-my-clothes-are-electric-no-seriously/)

![](../Images/0d86d3cebde73c700c05c6c93953b722.png "nano-material-felt-power-generator")

有一天，你也许可以用你的胯部或腋窝给手机充电。见鬼，也许甚至不会有电池，只有一个从能量毡获取能量的电容器，能量毡是一种将身体热量转化为电能的织物。

现在我们提到阴曹地府，因为它很有趣，但也因为它最有意义。研究人员开发了一种含有碳纳米管的织物，这种织物可以根据温差发电。我们认为身体上热量损失高的区域将是该织物最有效的位置，因为目前生产该织物的成本非常高(希望大规模生产可以将成本降低几个数量级)。因此，我们认为电池充电简报是一个明确的可能性。

我们在这里看到的是一个纳米级的珀尔帖发电机。这和基于蜡烛的发电机的概念是一样的，除了能量毡的效率提高了，让你浪费的身体热量代替了火焰。

有一份关于这个主题的白皮书，但是你不能不放弃一些(乔治·华盛顿)。
