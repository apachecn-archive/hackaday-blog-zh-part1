# 手机耐久性测试

> 原文:[https://hackaday.com/2011/01/01/cell-phone-endurance-tests/](https://hackaday.com/2011/01/01/cell-phone-endurance-tests/)

一部手机能让你用一辈子的日子已经一去不复返了，进入了为性感和精致而设计的玻璃外壳手机时代。这些新手机带来了新的测试方法。像诺基亚这样的公司仍然致力于制造最好的手机，并通过严格的测试使其经久耐用。[文章](http://gizmodo.com/5688232/how-phones-get-tested-and-abused-by-manufacturers)中展示的例子是模拟一部手机从衬衫口袋掉到地板上。诺基亚声称对他们产品线中的每个新型号进行了 200 项耐久性测试，包括温度、极端使用([使用这个按钮](http://hackaday.com/2010/12/21/this-hack-really-pushes-our-buttons/)进行自己的测试)、物理跌落和湿度暴露。让人不禁想知道其他公司用什么来做耐力测试。休息之后是诺基亚 N8 跌落测试的视频，如果你知道其他有趣的测试方法，不要忘记留下评论。
