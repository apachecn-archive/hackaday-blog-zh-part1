# 佳能 40D 被黑录电影

> 原文：<https://hackaday.com/2008/09/23/canon-40d-hacked-to-record-movies/>

[https://www.youtube.com/embed/OlKywUVFJW4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/OlKywUVFJW4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

佳能和 T2【尼康】最近都发布了 DSLR 相机，现在包括了大多数消费级数码相机已经拥有一段时间的功能:录制电影的能力。让电影录制在 DSLR 上特别有吸引力的是有多种镜头可供选择，以获得您想要的效果。如果你是[佳能 40D](http://www.mahalo.com/Canon_40d) 的拥有者，你可能想要关注【数据幽灵】在 CHDK 论坛上的[进展，因为他目前正致力于将这一功能引入 40D。](http://chdk.setepontos.com/index.php/topic,2259.0.html)

虽然[DataGhost]有一个概念的工作证明，但他指出，在录制视频、自动对焦和写入存储卡时，仍然存在一些与相机断电有关的问题。除此之外，[DataGhost]已经取得了相当大的进展，并正在考虑通过模式转盘添加自定义用户设置，以真正提供一些创造性的控制。我们对此感到兴奋，并迫不及待地向公众发布。
