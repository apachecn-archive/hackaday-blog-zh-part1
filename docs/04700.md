# 乐高机器人会放多米诺骨牌而不是鸡蛋

> 原文：<https://hackaday.com/2010/01/11/lego-robot-lays-dominoes-not-eggs/>

[Mattias]用他的[乐高机器人再次带来了令人敬畏的多米诺骨牌](http://woodgears.ca/domino/)。你会从[木制键盘盒](http://hackaday.com/2009/12/08/creating-wooden-enclosures/)和[机械二进制加法器](http://hackaday.com/2009/10/13/binary-adder-will-give-you-slivers/)中想起他的作品。这一次，他仍然通过制作自己的多米诺骨牌来锻炼木工技能，但这是机器人让这变得有趣。在休息后的必看视频中，该设备直直地摆放着，间隔完美的多米诺骨牌，只求被宠坏的小孩打翻。这个机器人只不过是一小撮由磁带驱动的乐高零件。零件可能很少，但在设计中有大量的独创性。
