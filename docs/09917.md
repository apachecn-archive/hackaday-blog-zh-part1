# 黑客日链接:2012 年 3 月 16 日

三叶草帽

![](../Images/9e11c41400f384bace9bf0693750f318.png "links-st-patties-day-hat")

[乔希]通过给这顶圆顶礼帽添加绿色发光二极管的轮廓，迅速制作出一顶三叶草主题的帽子。只要记住，不要喝酒和焊接。大家圣帕特里克节快乐！

电池供电的圣诞灯可能更有用

**![](../Images/79376da92f823e4a9aa65da350dd7341.png "links-battery-pack-reuse")**

 **[Karl]拿了一串几乎没用的电池供电的圣诞灯，并找到了它们的好用途。他用升压转换器取代了主板，并在市电不方便时使用两节 AA 电池作为 5V 电源。

**3D 打印器具修复**

![](../Images/b2b55ff9a259c315aa90b55158c7c395.png "links-3d-printed-appliance-parts")

这里看到的塑料碎片使洗碗机在运转时保持关闭。修理工找不到零件，但最好的解决方案是[打印一个完美的替换零件](http://hacklab.akan.fi/archives/868)。

**生成电阻存储标签**

**![](../Images/31322d2b7ac06d6547f0215b4b487464.png "links-color-coded-resistor-storage")**

 **[Darrell]拿起一个多余的试管架作为电阻存储。这是一个很棒的系统，但是他的使用 LaTeX 生成彩色编码标签的 Ruby 脚本也是一个很好的尝试。

DeLorean 四轴飞行器建造日志

![](../Images/aca722a440974f60e2d800a1b14aa7d5.png "BTTF")

不，这不是一个骗局。[Alp_X]找到了链接到标题为[我们最后的链接帖子](http://hackaday.com/2012/03/10/hackday-links-march-10-2012/)的[一个德洛尔四轴飞行器](http://forum.rcdesign.ru/f123/thread250976.html)的建造日志。这是俄语，但是[机器翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fforum.rcdesign.ru%2Ff123%2Fthread250976.html)应该会有所帮助。****
