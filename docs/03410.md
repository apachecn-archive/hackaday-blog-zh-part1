# 家庭 3D 打印

> 原文：<https://hackaday.com/2009/04/19/3d-printing-at-home/>

![3dprint](img/13508f9bc50f94677b45a8467a50fb13.png "3dprint")

我们最近已经看到了很多 3d 打印，有 [RepRap](http://hackaday.com/2008/10/19/lasercut-reprap-kit/) 和 [Cupcake](http://hackaday.com/2009/03/16/cupcake-cnc-kit/) ，两者都是[熔融沉积成型](http://en.wikipedia.org/wiki/Fused_deposition_modeling)类型。我们不常看到喷墨法。这是一个在某人家里建造的的很好的例子。他使用喷墨系统将像胶水一样的物质沉积到石膏层中，而不是铺设熔化的塑料层。这个项目比其他两个项目的分辨率高得多，正如你在下面制作钢筋混凝土发动机箱的视频中看到的那样。他目前正在重建它，以使其更好、更大。
