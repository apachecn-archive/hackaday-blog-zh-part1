# 将曲线铣削成镜子和玻璃

> 原文:[https://hack aday . com/2011/08/26/将曲线磨成镜子和玻璃/](https://hackaday.com/2011/08/26/milling-curves-into-mirrors-and-glass/)

![](../Images/c5409ad93ac86649b564ad83116954ab.png "milling-curves-into-glass")

[Ben Krasnow]正在进行一个项目，该项目使用了一种极其昂贵的专业镜子。他需要在上面切割出曲线，小心不要弄碎或打碎材料。他找到了用数控铣床做这件事的可靠方法，并分享了他的方法。

他正在研究的材料是一面冷镜子；它反射可见光，同时允许红外光通过。他必须定制，破损是不可接受的。[Ben]解释说，磨玻璃时最大的风险是使用的夹紧方法。他制作了自己的夹具，并使用垫片，而不是夹子，沿着 X 和 Y 轴固定材料。它被压在 Z 轴上，使用一根丙烯酸棒从一边跨到另一边，底部有橡胶脚。

一个钻石毛刺刀具做这项工作，转速为 3000 转/分。[Ben]建议尽可能以最慢的速度移动切割头，以便让切割机有时间完成工作。当然，材料需要通过泵送切削液来保持冷却。正如他在休息后的视频中展示的那样，你得到的是一块边缘干净光滑的玻璃。

万一我们激起了你的好奇心，这面镜子将在核磁共振扫描中使用。病人以 45 度角看着镜子里的监视器。与此同时，红外摄像机从镜子的另一侧记录病人，以监控他们在看哪里。
