# 天行者:一辆非常非常高的自行车

> 原文：<https://hackaday.com/2011/01/12/skywalker-a-really-really-tall-bike/>

[布拉德·格雷厄姆]喜欢制造自行车。他组装了一辆名为[天行者的高大自行车，然后分享了制造细节](http://atomiczombie.com/ct-skywalker.html#skywalker)(链接死亡，[试试互联网档案版本](https://web.archive.org/web/20110624223327/https://www.atomiczombie.com/ct-skywalker.html))。它拥有你在固定齿轮自行车中所期望的一切；一个座位，踏板，方向盘和两个轮子。你必须爬一点才能坐上马鞍，但是内置的梯子可以兼作乘客空间。[Brad]说这东西一次可以支持四个人。休息后查看视频，可以看到梯子和大高度转向架使自行车易于停止和启动，无需依靠任何东西。哇哦。

我们既兴奋又害怕。至少[上下串联有全尺寸的车轮](http://hackaday.com/2010/06/23/another-take-on-a-bicycle-built-for-two/)。但是这个长方形作品上的那些小边缘呢？呀！
