# 自制固体推进剂火箭发动机

> 原文：<https://hackaday.com/2010/09/28/homemade-solid-propellant-rocket-motors/>

KoD 和纳维克正在用糖和硝酸钾制造固体推进剂发动机。他们把这两种原料与水和粘合剂一起煮。他们发现玉米糖浆特别有助于增进感情，烹饪这种奇怪的饮料与其说是科学，不如说是艺术。不管怎样，休息后的视频证明了这项爱好的危险性。用浴室磅秤测试发动机推力对磅秤来说结果很糟糕。

材料的独创性令人满意。对于套管，他们使用 PVC 管，并通过使用相当于水管工环氧油灰的东西形成一个圆锥来集中推力。成品电机的封盖剂是磨碎的猫砂。

这是一个有趣的阅读，但现在我们将坚持使用水火箭。
