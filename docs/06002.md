# 激光麦克风让窃听变得非常简单

> 原文:[https://hack aday . com/2010/09/25/laser-mic-makes-窃听-非常简单/](https://hackaday.com/2010/09/25/laser-mic-makes-eavesdropping-remarkably-simple/)

这里有一个非常简单的方法来为自己制作一个基于激光的监听设备。它由两个模块组成，一个发射器和一个接收器。发射器是一组激光器，一个是可见的红色用于瞄准，另一个是红外的用于测量一个表面的振动。将发射器对准你想监听的房间的窗户，激光就能反射回接收器。接收器模块有一个光电晶体管来拾取红外激光，还有一个 LM386 音频放大器来产生发送到一对耳机的音频信号。需要很好地对齐，这很容易使用一对三脚架。休息后请欣赏演示。
