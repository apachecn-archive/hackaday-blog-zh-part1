# RGB 频闪吉他调音

> 原文:[https://hack aday . com/2010/11/03/RGB-频闪-吉他-调音/](https://hackaday.com/2010/11/03/rgb-stroboscopic-guitar-tuning/)

![](../Images/a2ef3817c631277767adedc70bcee35d.png "defon-badge-strobe-tuner")

这是【迈克尔·奥斯曼的】 [RGB LED 频闪吉他调音器](http://ossmann.blogspot.com/2010/11/introducing-def-con-super-rocker-18.html)。如果他的名字耳熟能详，那是因为[我们提到过](http://hackaday.com/2010/10/09/pulling-data-from-the-im-me-spectrum-analyzer/)，他将在 ToorCon 与【Travis Goodspeed】进行一次谈话。但他也去了 DefCon，并在他的酒店房间里度过了周末，试图赢得徽章黑客大赛。

尽管身处逆境，他还是让调谐器工作了。它内置在一把玩具吉他中，他可以在公路旅行中随身携带。通过在两个品之间添加一排 RGB 发光二极管，他可以使用同调弦的振动频率来闪烁三种不同的颜色。如果弦不在调上，三种颜色会跳舞，但与 LED 频率匹配会产生稳定的颜色。然后，他使用那个黄色的大按钮前进到下一根弦。休息之后看他的示范。

这基本上是一个内置的[拨子调谐器](http://hackaday.com/2009/11/23/electronic-guitar-pick-tunes-the-strings-for-you/)，它使用一个 LED 包而不是两个。
