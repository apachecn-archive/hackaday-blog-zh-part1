# 可拆卸闹钟——浪费电线，但对孩子来说很有趣

> 原文:[https://hack aday . com/2011/09/07/defusable-alarm-clock-wastes-wire-but-fun-for-the-kids/](https://hackaday.com/2011/09/07/defusable-alarm-clock-wastes-wire-but-fun-for-the-kids/)

![](../Images/caffe9d6a18bb0e4c83df0f4e009e856.png "defusable-Clock")

没有什么比床头柜上的定时炸弹更能让你感受到早上起床的压力了。它可能看起来像是来自 ACME 的邮件，但所有这些都是一些覆盖着牛皮纸的木钉和 Arduino 兼容板。7 段显示器可以作为一个时钟，或倒计时到你的末日。你可以设置一个闹钟，要求你剪断电线来关闭它。每次设置警报时，电线都是随机选择；一个人引爆炸弹，一个人安全拆除，其他人什么也不做。休息之后自己看吧。

电线很容易更换，因为它们是通过接线板连接的。这似乎仍然是对电线的极大浪费。我们喜欢[Think Geek 炸弹时钟概念](http://www.youtube.com/watch?v=oCn31UuRF5E)，它的工作方式基本相同，但使用的电线有一个公/母 RCA 插头对，可以断开和重新连接，不会造成浪费。

这一个显然将作为一个工具包，在这一点上的原理图和代码将被释放。但是你自己从零开始造一个应该不会太难，如果你有孩子的话，这是一个明显的赢家。
