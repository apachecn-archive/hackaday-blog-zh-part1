# 发微博说“伊沃克人”

> 原文:[https://hackaday.com/2011/12/27/a-little-tweeting-ewok/](https://hackaday.com/2011/12/27/a-little-tweeting-ewok/)

[![](../Images/ab301df50f2745ec33578052b800e32e.png "ewok_naked_clothed")T2】](http://hackaday.com/2011/12/27/a-little-tweeting-ewok/ewok_naked_clothed/)

也就是推特接收。这个名为“Ewen the Cheerlight”的伊沃克人模型，能够左右转动头部，还能显示表情。然而，这个黑客最有趣的特点是，每当有人在@cheerlights 上发布“颜色”时，小伊沃克人实际上就会醒来，并让它的主人[Joel]知道他对你选择的“颜色”的看法。[Joel]坚持说他在[HAD]上是这样的，尽管在推文开始滚动后它是否会很快被关闭还有待观察…

至于设备如何工作，用一个简单的业余爱好伺服电机转动头部，表情显示在一个 LED 矩阵上。这个模型本身是由一个聚苯乙烯球和一个旧台灯制成的。这座建筑看起来真的很棒。

所有这些都是由一个[简易 USB 接口板](http://www.pozible.com/index.php/archive/index/4416/description/0/0)控制的，这个接口板列在[【Pozible】](http://www.pozible.com/)，澳大利亚版的【Kickstarter】。休息之后一定要看看“Ewen”在行动的视频！
