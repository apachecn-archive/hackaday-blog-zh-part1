# 机器人在不平衡的轮子上行走

> 原文：<https://hackaday.com/2011/03/18/robot-gets-around-on-lopsided-wheels/>

![](img/eeb05df740c87f1bc9b723f16265c19a.png "robot-uses-lopsided-wheels")

这个[概念机器人利用角动量绕](http://www.jcopro.net/2011/03/07/the-angular-momentum-roller/)滚动。你可以看到在机器人的两端有两个圆盘，圆盘的一边被切开，使它们失去平衡。为了移动，两个 DC 马达旋转不接触地面的外层圆盘。这种旋转作用在车身上施加一个反方向的力，使车身移动。

这不是一个完美的系统，使用这个系统有一个主要的缺陷。当力相等时，加速将停止，最终会停止。你不能停止转动马达，因为那会起到刹车的作用。但是，这仍然是一个我们从未见过的概念，我们喜欢这里正在进行的实验。休息后看看测试片段，如果这开始让你头顶上的灯泡亮了，不要犹豫让我们知道。
