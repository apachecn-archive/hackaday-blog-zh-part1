# LED 落地灯真正将房间联系在一起

> 原文:[https://hack aday . com/2011/03/28/led-floor-lamp-really-ties-the-room-together/](https://hackaday.com/2011/03/28/led-floor-lamp-really-ties-the-room-together/)

![led_floor_lamp](../Images/2db9d76cf4e8df859f1a20cadf6ab132.png "led_floor_lamp")

Instructables 用户[lincomatic]正在装修房子，想找一些能真正把房间搭配起来的东西。他决定不加一块漂亮的地毯，他想要的是一个灯具。他没有选择房间角落的简单灯具，而是建造了一个 8×8 RGB LED 灯具，他称之为 Lampduino 。

他用一些剩余的纸板包装构建了 8×8 的网格，然后开始为 LED 线布线。在一个相当非正统的举动中，他决定用金属丝缠绕 LED 引线，而不是焊接。一旦发光二极管就位，他就把一切都连接到他的 Colorduino 上，这是一种 Arduino 衍生产品，专门用于驱动大量的发光二极管。在纸板的两面都放上一张薄薄的绘图纸，然后把它放在一个架子上展示。

他的蠢朋克风格的灯可以编程显示从颜色图案到视频游戏精灵的任何东西，如果需要，它还可以根据音乐定时。我们认为它看起来很棒，如果他厌倦了这个架子，可以挂在墙上。虽然绕线技术听起来大大加快了这个项目的开发，但我们很想知道它在使用几个月后的表现。
