# 声音敏感的机器人用莫尔斯电码说话

> 原文:[https://hack aday . com/2011/03/10/sound-sensitive-Android-figure-speaks-in-morse-code/](https://hackaday.com/2011/03/10/sound-sensitive-android-figure-speaks-in-morse-code/)

![mechanized_android_figure](../Images/d58d3ce55effab15617c4a288204203e.png "mechanized_android_figure")

Instructables 用户[tanbata]最近得到了一个谷歌 Anroid 小雕像，认为虽然它看起来很棒，但它没有真正的用途。他决定改变这一点，将这块一度无用的塑料块改造成一个可以移动并对声音做出反应的微型机器人。

他撬开了雕像的头部，并忙着在机器人的身体里安装一个伺服系统，以使头部能够移动。增加了一个 ATiny 来控制这个数字，还有一个麦克风使它能够对声音做出反应。压电被插入以传递莫尔斯电码信息，一些发光二极管被安装在人物的体腔和眼睛里只是为了刺激。

当机器人通电并感应到足够大的声音时，眼睛会发光，头部会左右旋转，机器人会用莫尔斯电码发出随机信息，如你在下面的视频中所见。

这不是最先进的项目，但经过一些调整，它可以成为一个伟大的 USB 供电的电子邮件或即时消息通知系统。更好的是，对于对电子产品感兴趣的孩子来说，这是一个很好的项目，因为他们可以制作一个可以保留的很酷的机器人玩具。
