# 后续:巨魔物理解决

> 原文:[https://hack aday . com/2012/01/27/follow up-troll-physics-solved/](https://hackaday.com/2012/01/27/followup-troll-physics-solved/)

![](../Images/cc6ea3002e5835f2df013991b584e521.png "LED")

一个月前，我们从 YouTube 用户[Fredzislaw100]那里看到了一个关于巨魔物理学的精彩演示。在他的视频中，我们看到了一个由三个开关和三个发光二极管串联而成的电路，但并不像肩膀一样。这篇文章的许多[评论](http://hackaday.com/2011/12/19/ask-hackaday-troll-physics-edition/#comments)引出了合理的解释，比如修改电池或通过后期效果的纯相机魔法。谢天谢地，[艾伦]介入并向我们展示了[是如何做到的](http://hackaday.com/2012/01/06/followup-reproducing-impossible-circuits/)。该解决方案使用两个交流电源，其中两个开关中有二极管，第三对开关中有 led 和电感。与原始视频相比，艾伦的体型相当大，所以我们想知道如何使这个电路隐形。

[Fredzislaw100]刚刚发布了一个关于他如何做的视频。像[Alan]的构建一样，它使用两个交流电源、二极管和电感。与每一个关于电路隐藏在哪里的猜测相反，*大部分构建都在电池连接器*内部。[Fredzislaw]做了一些惊人的工作，将 74LV132 四路 NAND 施密特触发器隐藏在电池连接器内。二极管很容易隐藏在 LEDs 1 和 3 上，涂上一些红色指甲油，但我们对标题图片中 LED 内置的电感器感到惊讶。

所以你走吧。凭借大量的电子技术和极其稳定的手感(以及显微镜)，你也可以构建自己的 troll 电路。休息之后请看视频。
