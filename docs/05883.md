# 浮动地球仪，被砍来旋转

> 原文:[https://hack aday . com/2010/08/31/floating-globe-hacked-to-rotate/](https://hackaday.com/2010/08/31/floating-globe-hacked-to-rotate/)

![they need to hire this guy](../Images/aec5a24653f3ea2f3b6710426b9a6adb.png "levitating-rotating-globe_motor")

[Alexy Sha]已经做了这个神奇的黑客，他[修改了一个磁性浮动球，使其机动化，并在倾斜的轴上旋转](http://hackedgadgets.com/2010/08/30/levitating-rotating-globe/)。最初的地球仪只是通过安装在里面的磁铁悬浮起来。虽然你可以用手旋转它，但它不是电动的，实际上是完全垂直漂浮而不是倾斜的。

[Alexy]想进一步发展这个想法，让它在旋转轴上自动旋转。他建造了一个旋转组件，基本上是一个马达，偏离中心悬挂，连接在地球的中心。他必须通过隐藏在底座中的线圈为它供电，这样它才能保持足够轻的重量漂浮起来。他做了一件了不起的工作，最终产品看起来像是它应该被销售的真实方式。

休息之后，请看一段视频。当他测试时，我们实际上喜欢旋转的环，就像最终的旋转球体一样。
