# 破老式唱机重生为便携式 MP3 柜

> 原文:[https://hackaday . com/2011/09/07/broken-vintage-record-player-reborn-as-a-portable-MP3-cabinet/](https://hackaday.com/2011/09/07/broken-vintage-record-player-reborn-as-a-portable-mp3-cabinet/)

![old_record_player_pc_jukebox](../Images/f59669c160ac6fc806a0e94bf98d2a0b.png "old_record_player_pc_jukebox")

[Julian]捡起一个坐在某人垃圾堆里的旧唱机，把它带回家看看是否能恢复正常工作。当他把它带回家时，他发现它根本就不工作，所以他和他的妻子决定让它现代化一点。

为了同时让自己和他的音乐收藏重聚，并在这个过程中激怒音响发烧友/古董收藏家，他把收音机的内部掏空，开始把它改造成 MP3 点唱机。一旦内部被拆除，他的妻子整修橱柜，并给前格栅一个新的布套。

一台旧电脑安装在柜子里，旁边还有一套相对便宜(但比纸盆好)的扬声器。一对定制切割的 plexi 面板用于覆盖计算机，同时为显示器和苹果无线键盘+触控板[Julian]用于管理点唱机提供空间。

被拒绝的电唱机看起来相当不错，虽然它可能会引起几个不同的纯粹主义者团体的愤怒，但我们认为它很酷。
