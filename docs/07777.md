# 自动浓缩咖啡装载机为那些深夜黑客会议

> 原文:[https://hacka day . com/2011/06/07/automatic-espresso-loader-for the-late-night-hack-sessions/](https://hackaday.com/2011/06/07/automatic-espresso-loader-for-those-late-night-hack-sessions/)

![](../Images/ea9e4508c1c5e87800159ceeac93677c.png "IMAG0131")

在[罗伊]的研究生电子学课上，他决定做一个黑客世界内外的许多人肯定会喜欢的东西，[一个自动浓缩咖啡装载机](http://oppressedhacker.blogspot.com/ "automatic espresso loader")。有三种口味可供选择。为了实现这一点，Parralax 推进器板用于控制三个伺服系统，这些伺服系统调节分配的咖啡量，由方便的 LCD HMI(人机界面)选择。

咖啡分发后，斜道会向旁边摆动。然后使用改进的喷墨墨盒载体(来自失效的打印机)压实咖啡豆。休息之后，请观看这台机器的运行视频。

[https://www.youtube.com/embed/Poe8Jqmq5BY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Poe8Jqmq5BY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这台机器不仅用似乎是透明的聚碳酸酯外壳建造得很好，它还使用了其他标准的黑客工具，包括:伺服系统、回收的打印机部件、视差处理器和液晶显示器。到目前为止，这台机器只进行了盐测试，而“阀门”只是连接到伺服系统的简单平板。这台机器在某种程度上仍然是一个原型，但它仍然是一个非常整洁的黑客。
