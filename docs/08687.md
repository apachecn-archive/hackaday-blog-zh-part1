# 在所有情况下，您应该订购哪些电阻值？

> 原文:[https://hack aday . com/2011/09/28/what-resist-values-should-you-order-for-all-conditions/](https://hackaday.com/2011/09/28/which-resistor-values-should-you-order-for-all-circumstances/)

![](../Images/33ff2e60a7ddb6b63f823752d98da05e.png "plotting-resistor-values")

硬盘崩溃，以及其他一些我们不太清楚的事情，导致[Devbisme]下了一个部件订单。因为他想充分利用他的运输成本，所以他决定用他最终会用到的零件来填写订单。他一直从事表面贴装设计，希望从使用 0805 封装的电阻转向 0603 封装。在手头一无所有的情况下，他设计了一种用最少数量的不同电阻计算几乎所有标准值的方法[。](http://devbisme.webfactional.com/blogs/devbisme/2011/09/23/thinking-too-much)

这听起来有点拗口，但他实际上做的是找出什么样的电阻组合可以最好地并联，以实现不同的标准电阻值。这样，如果他没有一个具体的值，他可以将一个 0603 表面贴装电阻焊接在另一个电阻上来实现。他最终编写了一个 Python 程序来最好地计算这组值。他设计了一套装置，让他只用 19 个实际电阻值，就能合成 168 个标准电阻值中的 159 个，误差在+/- 4%以内。他的方法需要一到三个电阻来达到每个值。将三个 0603 封装焊接在一起可能不是最有趣的，但它有助于轻松管理零件库存。
