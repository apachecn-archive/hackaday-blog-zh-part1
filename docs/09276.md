# 用压电元件检测地震波

> 原文：<https://hackaday.com/2011/12/11/detecting-seismic-waves-with-a-piezo-element/>

虽然我们通常看到压电元件被用来输出音频，但[Veedo]认为它们可以用在更有用的地方。他买了太多压电薄膜片，决定用它们来制作一个临时地震传感器。

压电标签的一端附有重物，尽管在测试时，他发现它们或多或少只能检测频率在千赫范围内的振动。由于地震倾向于产生 30-80 赫兹范围内的振动，他不得不调整他的装置来检测正确的频率。为了做到这一点，他附加了一个由螺钉和垫圈组成的重物，检查示波器上的输出信号，直到主要感测频率在 40 赫兹的范围内。

该传感器连接到一个试验板上，然后通过一个电荷放大器产生一个小的交流信号，该信号在 2.5Vdc 上浮动。该波的下半部分被一个二极管切断，然后被馈入 Arduino Mega。地震数据随后被上传到他的 Pachube 账户进行存储，尽管他可以通过编程到 Arduino 的网络服务器在本地查看这些数据。

我们不确定在地震发生时，这种设置会给你多少提前通知，但无论如何，这似乎都是一个有趣的项目。
