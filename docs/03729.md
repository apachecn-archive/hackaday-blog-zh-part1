# 助手机器人阿奇

> 原文：<https://hackaday.com/2009/07/22/archie-the-helper-bot/>

[https://www.youtube.com/embed/TaGnnMavbAg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/TaGnnMavbAg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们注意到了 BotJunkie 上这篇关于助手机器人 Archie 的文章。阿奇应该在家里帮忙做饭、打扫卫生和做其他日常工作。[埃文]提出了一个很好的观点。为什么人们坚持给他们的机器人装上令人毛骨悚然的头。他们没有让它们变得更可爱，只是让人毛骨悚然。虽然这是一个非常敏锐的观察，但我们想再补充一些。观看上面的视频，研究休息后的图像。阿奇似乎不是一个功能机器人。他从来不会在视频里靠自己的力量移动。他们陪他“散步”的场景是喜剧黄金。他的头一直向后仰，或者可能掉了下来。没有驱动手的助手机器人有什么用？视频是德语的，所以也许我们漏掉了什么。也许阿奇是一个模型或笑话，我们只是需要翻译。
