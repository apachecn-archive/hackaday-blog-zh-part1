# 斯坦福的 Stickybot3

> 原文：<https://hackaday.com/2010/04/15/stanfords-stickybot3/>

[https://www.youtube.com/embed/0y2MVuSyQlg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/0y2MVuSyQlg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

Stickybot 经历了一次彻底的升级。你可能还记得网上流传的[壁虎玻璃行走机器人](http://bdml.stanford.edu/twiki/bin/view/Rise/StickyBot)。虽然这很酷，但这项技术已经走得更远了。它不仅被设计成看起来像壁虎，实际上它的脚也以同样的方式附着在表面上。他们正在使用一种基于[范德华力](http://en.wikipedia.org/wiki/Gecko#Gecko_toes:_setae_and_van_der_Waals_forces)的粘合系统。虽然 stickybot 3 还不会走路，但它的脚已经让人印象深刻了。看看脚与玻璃接触的面积有多小。看他能多容易地移除和重新粘贴它？简直太神奇了。
