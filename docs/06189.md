# 虚幻扬声器构建

> 原文:[https://hackaday.com/2010/10/27/unreal-speaker-build/](https://hackaday.com/2010/10/27/unreal-speaker-build/)

![](../Images/66d29a6b18e0854b64b0895accf48534.png "unreal-speaker-build")

这些扬声器是手工制作的，几乎是独一无二的。[Lluís Pujolàs]没有提出最初的设计，但他确实在制作它们方面做了令人惊叹的工作，包括[一份 11 页的构建日志](http://www.mundohi-fi.com/portal/index.php?module=Pagesetter&func=viewpub&tid=8&pid=8) ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://www.mundohi-fi.com/portal/index.php%3Fmodule%3DPagesetter%26func%3Dviewpub%26tid%3D8%26pid%3D8))。它们被称为奥德赛 2 号，以最初的设计命名。底部的贝壳状空腔最初是作为木制骨架建造的，然后被覆盖以形成最终的形状。但是中高范围的外壳是在车床上由木材胶合而成的。做这种木工活需要一家正规的机械商店。底座由混凝土浇注而成，浸有铅珠，有助于隔振。每辆车有 330 磅重，可以理解的是，在把它们停在最终位置之前，他在轮子上测试了它们，如上图所示。
