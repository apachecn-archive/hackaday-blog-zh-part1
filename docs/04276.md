# 牛逼机器人爱芬达

> 原文：<https://hackaday.com/2009/10/19/awesome-robots-love-fanta/>

[https://www.youtube.com/embed/SOESSCXGhFo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/SOESSCXGhFo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

你在上面看到的不是芬达的广告，尽管我们认为这可能是一个好广告。它也没有以加速的速度播放。这是 ABB 机器人达到的精确度和速度的实时展示。当我们点击播放时，我们感到惊讶，甚至震惊。我们不知道是谁想出了这个主意，但我们想要那些机器人，我们想要一些芬达。我们有点好奇什么行业需要饮料跟踪机器人。

上一次我们看到如此令人惊叹的机器人绝技，它们是在半空中弹跳球和抓东西。
