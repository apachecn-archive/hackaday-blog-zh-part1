# 用旧货店废品建造的蓝牙机器人

> 原文:[https://hack aday . com/2011/04/01/bluetooth-bot-constructed-from-thrift-store-rejects/](https://hackaday.com/2011/04/01/bluetooth-bot-constructed-from-thrift-store-rejects/)

![bluetooth_bot](../Images/377c13aeb138a22cbfe007d523689829.png "bluetooth_bot")

[John]写信来分享他的最新创作，[一个 Arduino 控制的蓝牙机器人](http://www.instructables.com/id/Arduino-controlled-Bluetooth-bot)。你可能会记得他以前的一个黑客，[劳恩博特 400](http://hackaday.com/2010/03/25/lawnbot400-saves-wheelbarrow-trips/) 。这一次，他决定缩小规模，专注于小型遥控玩具。

他的蓝牙机器人是用他在旧货店花了大约 1.5 美元买到的廉价遥控坦克制造的。除了框架、电池盒和驱动踏板的马达，他拆除了机器人的所有部件。他添加了一个 Arduino，并与 SparkFun 的 Ardumoto 盾配对。电机屏蔽成本约为 25 美元，但他有可用的原理图，所以你可以滚动自己的，如果你想这样做。

汽车上增加了蓝牙伴侣，允许与任何其他蓝牙设备进行串行通信。一旦所有东西都连接好，他就把机器人和他的电脑配对，然后开始通过简单地按键盘上的键来驱动它。

它看起来像一个有趣的小玩具，而且似乎很容易建造。看看下面他的机器人的视频，如果你有兴趣建造自己的机器人，一定要看看他的代码/原理图。
