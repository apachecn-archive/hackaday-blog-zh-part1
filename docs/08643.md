# 复古游戏墙艺术；可玩！

> 原文:[https://hack aday . com/2011/09/22/retro-gaming-wall-art-its-playable/](https://hackaday.com/2011/09/22/retro-gaming-wall-art-its-playable/)

![](../Images/09846ac6e6d96ccfb44ac54f38b339a8.png "retro-gaming-wall-art")

这里有一面挂在接待区的墙，让您的顾客在等待时玩复古游戏。为了给你一些比例感，右边的按钮(标有开始/跳转，但我们称它们为 A 和 B)是比传统街机按钮大的~~街机按钮~~。屏幕本身是三星宽屏电脑显示器——我们打赌它是[的 16 寸型号](http://www.samsung.com/ph/consumer/monitor-peripherals-printer/monitor/lcd-monitor/LS16CMYSFUXP/index.idx?pagetype=prd_detail)，但我们只是猜测。它被一个角撑架固定在木框里。

这是我们在 6 月份看到的一次黑客攻击的产品，其中[一个 Arduino 被用来控制数字化复古 LCD 游戏](http://hackaday.com/2011/06/26/digitized-retro-lcd-games-played-using-an-arduino-as-a-controller/)。使用相同的硬件，用 Arduino 监控按钮，并使用 Python 脚本将它们翻译成计算机上的按键。这意味着这不是一个独立的，但需要一台电脑来运行游戏和液晶显示器。尽管如此，我们仍然喜欢它的外观，并希望有一天它会迁移到 FPGA 控制(他们没有问题[驱动液晶显示屏](http://hackaday.com/2011/09/09/putting-laptop-lcds-to-use-with-an-fpga/))和可选择的游戏。
