# BT 手机不仅仅是翻新

> 原文：<https://hackaday.com/2010/01/04/bt-phone-is-much-more-than-retrofit/>

![](img/e87121c4bd0c116b1e3ca4e8f407dfad.png "bt-ericphone")

[圣地亚哥]把他的 [Ericfon 变成了蓝牙手机](http://btericofon.blogspot.com/)。这与我们上个月看的[手机改造](http://hackaday.com/2009/12/23/bluetooth-handset-hacks/)完全不同。这是因为他没有简单地打开一个 BT 耳机，然后塞进他的手机。他开发了自己的全功能硬件。

这是一个[开源](http://btericofon.blogspot.com/2009/12/software.html)项目，有[可用的硬件细节](http://btericofon.blogspot.com/2009/12/original-idea.html)，他打算把它变成一个工具包。[Santiago]设计了一个 PIC 微控制器来连接一个 [WT32 蓝牙模块](http://www.bluegiga.com/WT32_Bluetooth_Audio_Module)。PIC 允许拨号音，用原来的转盘拨号，并在电话铃响时产生原来的声音。他现在拥有的是一个没有固定电话的家庭电话。正如休息后看到的视频，Ericfon 的工作原理和新的时候一样，只是连接是通过蓝牙而不是通过铜电话线进行的。
