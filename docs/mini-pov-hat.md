# 迷你 POV 帽子

> 原文：<https://hackaday.com/2008/10/22/mini-pov-hat/>

[https://www.youtube.com/embed/lfYU4grHD7g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/lfYU4grHD7g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

用这款[迷你 POV bowler](http://www.instructables.com/id/Mini_POV_hat/) 让你自己在万圣节脱颖而出。我们将与 [nadsat](http://en.wikipedia.org/wiki/Nadsat) 一起退出，以避免踢到院子里，你有我们最真诚的 appy polly loggy 首先开始。[cheeto4493]修改了一个迷你 POV，使其位于圆顶礼帽之上，并向容纳灯的边缘伸出一个延伸部分。安装在帽子内部的马达带动整个装置旋转。需要一些平衡物来防止整个东西晃动得太厉害。

值得注意的是，在人，信息滚动缓慢。摄像头的刷新率导致它在视频中似乎在跳来跳去。
