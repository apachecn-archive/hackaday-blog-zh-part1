# 从火箭的角度看模型火箭学

> 原文：<https://hackaday.com/2011/08/03/model-rocketry-from-the-rockets-point-of-view/>

![model_rocketry_video](img/fc8109d6360ab41d8bce72aa905d91a4.png "model_rocketry_video")

当有人来信说，“嘿，[我在火箭](http://www.vladcazan.com/blog/exploring-model-rocketry)上绑了一台相机，拍下了它发射的视频！”我们很难不被骗进去。尽管我们尽了最大努力，我们还是忍不住看了看[Vlad]录制的他的火箭模型“探索”的视频。

受我们 7 月 4 日的帖子的启发，他给自己买了一个 808 钥匙扣相机，并决定尝试一些高空飞行视频。为了减轻重量，他用几块透明胶带把相机绑在他的 46 英寸 Estes 火箭上，然后出发去他的发射台。

他每次发射都使用埃斯蒂斯·C6-5 发动机，他估计由于增加的重量，火箭将上升到 300 英尺的高度，而不是通常的 500 英尺。虽然不是特别有用，但这个视频看起来仍然非常有趣，也许它会激励其他人在更大、更强大的火箭上安装相机。

我们只能希望。
