# DIY 有机发光二极管

> 原文:[https://hackaday.com/2010/11/29/diy-oleds/](https://hackaday.com/2010/11/29/diy-oleds/)

![](../Images/4ad5f494dd20818917eba6bb0da9c471.png "oled")

Jeri ells worth 收集了几个视频，讲述了她如何制作自己的有机发光二极管。在休息之后的第一个视频中，它讨论了常规、刚性半导体 led 和有机 led 之间的差异。视频接着展示了如何用连续的材料层制作有机发光二极管。玻璃上的氧化铟锡(ITO)形成透明阳极。然后涂上 [PEDOT:PSS](http://en.wikipedia.org/wiki/PEDOT:PSS) ，一种用作空穴传输层的导电聚合物混合物。然后加入红钻钌络合物以产生发射层。阴极层是低功函数金属，最初是镓铟共晶合金，后来显示其他金属也能工作。第二个视频展示了如何将荧光棒榨汁，并用这种液体制作 OLEDs。蓝色荧光棒中的染料 [9，10-二苯基蒽](http://en.wikipedia.org/wiki/Diphenylanthracene)，是一种有机半导体，当电流通过时会发光。发光棒方法似乎有一些问题，因为 ITO 涂覆的玻璃板会被发光棒化学物质降解。看看使用多孔铝或来自[Jeri]的[柔性电致发光显示器](http://hackaday.com/2010/11/16/jeri-makes-flexible-el-displays/)的类似技术是否可以用作电极将会很有趣。
