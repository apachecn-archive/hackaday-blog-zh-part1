# 用通用结构组件连接玩具积木

> 原文：<https://hackaday.com/2012/03/20/connecting-toy-blocks-with-a-universal-construction-set/>

我们都曾经是孩子，当然我们中的一些人还记得试图让一种类型的积木和另一种类型的积木一起工作的痛苦。自由艺术和技术实验室的人对你内心的孩子有一个答案:[适配器，可以将任何类型的积木](http://fffff.at/free-universal-construction-kit/)连接到任何其他类型的积木。

该项目被称为免费通用建筑工具包。这“哎呀，我希望我想到的想法”是一套 79 个播放设置适配器，允许任何孩子混合他们的 Duplo，慧鱼，齿轮！齿轮！Gears、K'Nex、Krinkles、Lego、Lincoln Logs、Tinkertoy、ZomeTool 和 Zoob 以任何你能想到的方式搭建。这些适配器中的大多数[都在 Thingiverse](http://www.thingiverse.com/uck) 上，随时可以用 3D 打印机打印出来或者送到 Shapeways。

F.A.T .实验室工作的一个有趣方面是法律和知识产权方面；由于这是几个建筑套件逆向工程的产品，这些玩具的制造商完全有可能不希望适配器出现在野外。不过，这支队伍确实打好了基础。在十个玩具系统中，有八个不再受专利保护，这让 MEGA Bloks 背后的公司非常懊恼。剩下的两个系统——Zoob 和 ZomeTool——的适配器将分别在 2016 年和 2022 年专利到期后才会发布。
