# 为 Dockstar 构建更多 OpenWRT 图像

> 原文:[https://hack aday . com/2010/10/04/more-openwrt-image-building-for-the-dock star/](https://hackaday.com/2010/10/04/more-openwrt-image-building-for-the-dockstar/)

![](../Images/3e86588316d2b144c7a66628ff85504f.png "freeagent-dockstar")

[Der_picknicker]写信告诉我们关于为 dockstar ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://matze.efreun.de/%3Fp%3D103))构建 OpenWRT 图像的指南。他们最终得到的是一个漂亮的小型网络附加存储设备，在 OpenWRT 的保护伞下运行 SAMBA 和 subversion。早在 7 月份，我们就看到了[为这款设备闪烁和构建 OpenWRT](http://hackaday.com/2010/07/17/openwrt-on-a-seagate-freeagent-dockstar/) 图像。OpenWRT 的开发分支还没有达到一个稳定的版本，但是在过去的几个月里已经做了很多。

机器翻译有点粗糙，但编译过程很容易理解。如果你不介意辛苦的编译(这显然需要 1-2 个小时)，他们也提供了他们的图片供下载。应该可以通过 SSH 进行闪存，但是为了安全起见，您可能希望为设备添加一个串行端口。
