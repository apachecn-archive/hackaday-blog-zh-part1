# 沃兹的手表让航空旅客紧张

> 原文：<https://hackaday.com/2009/11/03/wozs-watch-makes-air-travelers-nervous/>

史蒂夫·沃兹尼亚克(Steve Wozniak ),世界上最后的自由黑客和最伟大的剑客，过着你做梦都想不到的黑客生活。在 [medGadget 工作的人碰到了他，并了解了他的手表](http://www.medgadget.com/archives/2009/10/the_woz_showing_off_a_prized_gadget.html)。在他们的采访中(插播在休息之后)[史蒂夫]展示了数码管手表，我们只能假设他创造了自己。

这款手表由两根小管、一块印刷电路板和电池组成。其中必须有一个加速度计，因为除非手表处于某个可配置的角度，否则显示屏会关闭。一旦保持在正确的位置，显示屏闪烁小时，然后分钟。

时间由表壳内的两个按钮设定。[史蒂夫]继续解释当他在飞行中拆卸手表并开始摆弄里面的按钮时，他的乘客有多害怕。为了不被电子胡闹所阻止，他还分享了自己偷偷把陶瓷刀带上船的快乐，这样他就可以正确地切牛排了。

抛开生活方式不谈，[我们见过的小型谢妮时钟套装](http://hackaday.com/2009/09/03/single-tube-nixie-clock/)甚至都无法与之媲美。我们假设这对于电池来说相当困难，并且想知道如果你被困在一个长时间的会议中并且不停地看时间，手表是否会变热。我们可以克服这两个局限，仅仅是因为酷这个因素。

**更新:**读者【Stephen】留下一条评论，直指[手表的来历](http://www.cathodecorner.com/nixiewatch/)。它由 PIC 微处理器控制，拥有每天 50 次浏览的四个月电池寿命，还有一些可用的[代码](http://www.cathodecorner.com/nixiewatch/firmware/firmware.html)。它是待售的，但是目前缺货。
