# Nook 颜色变蜂窝

> 原文:[https://hackaday.com/2011/02/01/nook-color-gets-honeycomb/](https://hackaday.com/2011/02/01/nook-color-gets-honeycomb/)

![](../Images/9aae0fd3e828cab9ff88f045738f9a52.png "11x02011045nookcolor")

[深蓝]已经发布了让 Android honeycomb 在你的 nook color 上工作所需的所有文件。我们有机会[玩了一会儿 nook color](http://hackaday.com/2010/12/12/nook-color-rooted-hands-on/) ，但我们的只是在 Android 版本上。从那时起，他们似乎已经在这个简单的电子阅读器的功能上走了很长的路。虽然他已经构建出了大部分功能，但仍然缺少一些基本功能，比如声音。正如你在休息后的视频中看到的，滚动有点不连贯，但应用程序本身看起来相当快。我们迫不及待地想看看这一点点改进后的效果。
