# POE 家庭自动化控制面板

> 原文:[https://hack aday . com/2011/10/14/Poe-home-automation-control-panel/](https://hackaday.com/2011/10/14/poe-home-automation-control-panel/)

![poe_home_automation_control_panel](../Images/deead911aa0baaa6603bdea10b38690f.png "poe_home_automation_control_panel")

[Scott]写来分享他目前正在做的一个项目，一个依靠以太网供电的家庭自动化系统。虽然他还没有完全完成，但他已经取得了很大的进步，他目前所做的工作无疑激起了我们的兴趣。

[斯科特]的部分设计依赖于我们不久前展示给你的一些[逆向工程的黑莓屏幕](http://hackaday.com/2011/04/21/blackberry-lcd-reverse-engineering/)。他为自己的公寓建造了一个小型控制面板，其中包括一个前述的黑莓屏幕，以及 10 个触觉开关和一个 PIR 传感器。该面板可以安装在墙上或作为一个独立的单元，允许他控制家中的各种灯光和电器。

[Scott]在他的控制面板的通信协议和用户界面上花了很多时间。就像现在这样，这个面板利用了他设计的类似 VNC 的协议，允许他与桌面计算机上的 Java 应用程序进行交互。

到目前为止，一切看起来都很棒，[Scott]已经为不久的将来计划了一些改进。当它完成的时候，我们迫不及待地检查它。
