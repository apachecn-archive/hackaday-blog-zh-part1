# 攀登建筑物用的真空手套

> 原文：<https://hackaday.com/2009/07/25/vacuum-gloves-for-climbing-buildings/>

![spiderman](img/7da0327bc9ecbbc9a9e1e89037a2d54b.png "spiderman")

吸力强大得不可思议，可以用[几种](http://hackaday.com/2009/02/18/vacuum-former-toy/) [不同的](http://hackaday.com/2009/06/18/arduino-powered-cd-changing-robot/) [方式](http://hackaday.com/2008/11/26/vacuum-micromouse/)施展。[Jem Stansfield]为 BBC 的一部电视剧制作了一套真空手套，以展示吸力到底有多强大。他爬上了一座 100 英尺高的建筑物的侧面，但不得不依靠靠近顶部的安全线。他跳伞后大胆升天的视频。
