# 遥控轮式割草机

> 原文：<https://hackaday.com/2010/05/03/remote-control-reel-mower/>

![](img/f3405b8723a0cc92716b66501eba6564.png "radio-controlled-reel-mower")

受到炎热天气和陡峭草皮的挑战，[格雷森·西格勒] [改装了他的轮式割草机，使用电动马达](http://www.lumenlab.com/d/blog/romow-rc-reel-mower)。最终产品将是无线电控制的，但他现在缺乏必要的零件来使它无线化。为了不被吓倒，他使用了一个有线控制器进行原型制作和测试，一旦零件到达，就可以很容易地更换。可悲的是，他家院子的坡度导致小轮子打滑，所以他试图在胎面上插入螺钉来增加牵引力。这个小型割草机没有劳恩博特 400 的[重量和占地面积大。也就是说，太阳能充电是未来的目标之一，所以这个被他命名为 RoMOW 的建筑在绿色规模上胜出。我们在广告之后加入了原型视频，展示了叶片旋转的过程。](http://hackaday.com/2009/11/14/rc-lawnmower/)
