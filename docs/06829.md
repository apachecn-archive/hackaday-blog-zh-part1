# 超级布里特 LED 运动鞋

> 原文:[https://hackaday.com/2011/02/15/super-brite-led-snakers/](https://hackaday.com/2011/02/15/super-brite-led-snakers/)

![](../Images/f89decf0e3e969f6ba28a0c67b6d0c3b.png "led shoes")

[我是珍]需要给一些鞋子增添情趣，还有什么比用微控制器控制的发光二极管更好的方法呢？为了制造 LED 灯条，创造性地使用了 Velcro。带子的一半固定在鞋子上，而另一半在粘性面上有一张绝缘胶带。然后在金属条上打小孔，然后焊接发光二极管。

电子方面，鞋子使用一对 6 伏电池，没有现成的支架，一个“AA”电池支架被切成一定大小，然后粘在一起。图案由一个真正裸露的 Arduino 板控制(即使你不是 Arduino 的粉丝，这也是一个很酷的 28 针芯片的小 AVR 分线板)，并使用鞋子内侧边缘的磁铁和簧片开关循环不同的图案。

即使这不是你的风格，也可以看看一些好的技巧，休息后加入我们，观看一个简短的视频。
