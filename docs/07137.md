# Bill Hammack 解释了 LED 背光液晶显示器的工作原理

> 原文：<https://hackaday.com/2011/03/17/bill-hammack-explains-how-led-backlit-lcd-monitors-work/>

![](img/6882a1d3f68fd81aa7730838153596b5.png "how-lcd-monitors-work")

我们对液晶显示器的工作原理有了一个基本的了解，你可能也是。但问题是，[比尔·哈马克]不仅仅是解释基础知识。因为他是工程师，所以他解释了 LED 背光液晶显示屏背后的工程原理。但是他用一种每个人都能理解的方式去做。

休息过后，我们嵌入了他的五分钟视频。在影片中，你会看到他把显示器拆到背板上，然后一点一点地组装起来。我们喜欢他关于漫射板如何一起工作以均匀和分布光线的讨论。这些是由几层组成的，尽管我们知道他们从[开始就在那里用回收的液晶显示屏](http://hackaday.com/2010/09/28/launchpad-not-limited-to-value-line-chips/)工作，但我们从来不知道他们到底在做什么。他还介绍了每个液晶单元如何与偏振片一起阻止或允许光线通过。他将通过展示屏幕上每个子像素中的薄膜晶体管如何工作来实现显示的多路复用，把它带回家，[，就像我们 10 月份对南瓜](http://hackaday.com/2010/10/26/70-led-matrix-in-a-jack-o-lantern/)所做的那样。
