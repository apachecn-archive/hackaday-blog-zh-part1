# 来自 AVR 程序员的开发板

> 原文:[https://hack aday . com/2011/08/26/dev-board-from-an-AVR-programmer/](https://hackaday.com/2011/08/26/dev-board-from-an-avr-programmer/)

![](../Images/09e884c28f0738faa745b7488e681997.png "usbasp")

[jethomson]送来了他一直在做的一个构建，它把一个廉价的 AVR 程序员变成了开发板。构建基于基于 ATmega8 的非常便宜的 USBasp 编程器。中国以不到一个汉密尔顿的价格提供了数百个这样的板[，我们想知道这些板多久才会出现在其他项目中。](http://www.ebay.com/sch/i.html?_from=R40&_trksid=p5197.m570.l1313&_nkw=usbasp&_sacat=See-All-Categories)

在摆弄了 AVR 保险丝设置之后，[jethomson]设法将一些代码烧到了微控制器上。之后，上传软件给程序员所需要的只是一点焊料、一个按钮和一个外部程序员。[jethomson]还设法通过对 boards.txt 文件进行小的添加，使这个小 AVR 程序员与 Arduino 环境兼容。

[jethomson]包括了一些软件作为例子——他移植了[闹鬼的 USB 线](http://imakeprojects.com/Projects/haunted-usb-cable/)并改编了[的 USB 名片](http://www.instructables.com/id/USB-PCB-Business-Card/)打印出来，“只工作不玩耍，聪明的孩子也变傻。”干得好，我们迫不及待地想在未来的项目中看到它。
