# 极其有组织的原型制作

> 原文:[https://hack aday . com/2010/07/25/极有条理-原型制作/](https://hackaday.com/2010/07/25/extremely-organized-prototyping/)

![](../Images/f6738de4b4de6a5d4d14f98a659e700a.png "briefcase-prototyping")

我们有几个非常高科技的鞋盒，用来存放我们的原型配件。您会发现一系列 LCD 模块、分线板上的芯片、焊接到自制板上的开关，以便轻松进行试验，等等。这是假设你能在那些乱七八糟的组件中找到任何东西。

[Shahriar]采取了不同的方法。他把所有的原型制作工具都装在一个公文包里。这一大堆高端主板包括 PIC 原型、各种液晶屏和 SparkFun 库存的很大一部分主板。它比[的 Arduino to-go 平台](http://hackaday.com/2010/04/06/on-the-go-prototyping/)先进得多，休息后你可以看到系统的完整演示。
