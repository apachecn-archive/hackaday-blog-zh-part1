# 无轮驾驶游戏方向盘控制器

> 原文:[https://hack aday . com/2011/07/06/driving-game-steering-wheel-controller-without-the-wheel/](https://hackaday.com/2011/07/06/driving-game-steering-wheel-controller-without-the-wheel/)

![](../Images/7ae666517c527b0441900f5310e5683f.png "kinect-driving-controller")

出于某种原因，这个项目让我们想到了 Tron 中的光周期。你知道，骑车人抓住组成控制装置的魔杖后，自行车就形成了？当然，你不会凭空看到一辆汽车，但这个[驾驶控制器让你抓住任何东西来控制赛车游戏](http://www.youtube.com/watch?v=Bq7mdl0yAy0)。

你可以看到它使用 Kinect 来映射玩家的身体，并将你的动作转换为动作控制。文件夹下方嵌入的演示视频显示了校准步骤，随后是可用的控制选项。向前推方向盘打开氮气，向前或向后倾斜加速和刹车，几个手臂信号可以让你浏览游戏菜单。

这是通过将手势映射到击键来实现的。[Rajarshi Roy]告诉我们在他们的库中有一个非常原始的[代码包，但是计划在这个周末清理它。他们还将致力于 Wiki、文档和教授软件新手势的教程。](https://github.com/spaghetti-coder/NIGameController)

我们只是不知道我们更喜欢什么，看到 kinect 扩展为像这样的游戏控制器，或者在机器人技术中使用它，像四轴飞行器。
