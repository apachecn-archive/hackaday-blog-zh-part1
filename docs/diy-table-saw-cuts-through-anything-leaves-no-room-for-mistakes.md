# DIY 台锯可以切割任何东西，不会出错

> 原文：<https://hackaday.com/2011/07/07/diy-table-saw-cuts-through-anything-leaves-no-room-for-mistakes/>

瑞士苏黎世联邦理工学院巴斯特里实验室的学生已经在相当长的一段时间里坚持使用动力不足和不可靠的锯子。锯在切割印刷电路板时经常会卡住，通常使用起来很费力。当小组成员[Mario Mauerer]在他的地下室偶然发现一个大功率无刷电机时，[他决定是时候升级实验室的切割工具了](http://www.bastli.ethz.ch/index.php?page=bastli_circularsaw)。

他和同学[Lukas Schrittwieser]一起搭建了一个测试平台，看看电机的功率到底有多大，对结果感到满意后，两人开始制作自己的台锯。外壳很快就被包好了，留下两人去寻找电源。他们没有购买，而是建造了一个 700 瓦的庞然大物，用 PSU 为他们的电锯供电。

正如你在下面的视频中看到的，锯子可以轻松地锯开大多数东西，但学生们给锯子添加了一个“助推按钮”，以防他们需要全速运转。

虽然我们不能完全忽视他们演示中缺乏手指和眼睛保护，但它看起来确实是一个很好的小工具。
