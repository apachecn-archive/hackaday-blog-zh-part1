# 光束机器人漫无目的地翻滚

> 原文：<https://hackaday.com/2009/11/17/beam-robot-tumbles-aimlessly/>

几年前的翻滚机器人是对简单运动的极好研究。除了轮子和腿，他用四个鳍状肢在房间里滚动机器人。使用两个马达，每个负责两个鳍状肢。相同但独立的电路驱动电机，74HC240 gate IC 监控连续旋转。当脚蹼被卡住时，电路会逆转马达的旋转方向，这样这个简单的机器人就可以摆脱困境。

电路没有我们在之前见过的一些[光束构建先进。这并没有减少他设计的聪明之处，我们认为](http://hackaday.com/2008/07/10/spotlight-on-tinyminds/)[光束机器人](http://en.wikipedia.org/wiki/BEAM_robotics)是让你的大脑摆脱计算机代码，只使用硬件的好方法。休息过后，你可以观看一些翻滚动作的视频。我们还包括了他网站上的另一个机器人的视频，它使用同心环进行另一种独特的运动。
