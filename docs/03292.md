# Arduino 捕鼠器

> 原文：<https://hackaday.com/2009/03/11/arduino-mouse-trap/>

![mouser](img/4f55030032a031075561c63d8d307151.png "mouser")

[scott]需要捕捉一只鼠标，并想学习如何对 arduino 编程，所以他制作了一个由 Arduino 控制的捕鼠器。它是由他放在房子周围的东西做成的，比如一些[特百惠容器](http://www.mahalo.com/Tupperware "Tupperware - Mahalo")和木棒。这个程序非常简单，当绊线被触碰时，伺服系统会将木棒拉出，关闭容器。休息之后你可以看到它在工作。绊网似乎是一个大的故障点。他表示，这只是一根电线，稍微高于锡箔带。这似乎只是一个很小的区域，鼠标需要触摸才能触发它。他还能设计出什么更好的开关，既便宜又快捷？
