[离经叛道的奥拉姆]，开锁人和[饮料冷却比赛](http://hackaday.com/2008/05/28/beverage-cooling-contraption-contest/ "Beverage cooling contraption contest  - Hack a Day")主持人，最近在维也纳，大概是为了 [DeepSec](https://deepsec.net/) 。在那里，他在[金属实验室](http://metalab.at/wiki/English "English - Metalab")黑客空间停下来，查看了他们的 [RepRap](http://reprap.org/ "WebHome < Main < Reprap") 快速原型制造机。你可以在上面看到他访问的视频。他让他们为他的傻瓜相机的闪光灯定制了一个合适的盖子。这就是我们喜欢快速成型的原因。我们在这里讨论的许多项目解决了一个特定的问题，但是永远不会被认为具有足够的商业可行性来投入生产。随着快速原型制作技术的日益普及，黑客们可以开始生产更复杂的物品来满足他们的特殊需求，其成品更接近于商业产品。
