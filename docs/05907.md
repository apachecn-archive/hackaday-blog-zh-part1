# 固体冰啤酒罐

> 原文:[https://hackaday.com/2010/09/05/solid-ice-beer-caddy/](https://hackaday.com/2010/09/05/solid-ice-beer-caddy/)

![brrrrr](../Images/8c40b4af5ba83957ffa2ab4f6efc1680.png "ice_beer")

[Rob]在冷藏我们的酒方面做出了惊人的贡献。他不满足于仅仅冷藏他的冰袋，甚至把它们放在一个装满冰的冷藏箱里，他还用固体冰做了一个啤酒罐。虽然它最初看起来很简单，但你可以从他的文章中看到，实际上它花了相当多的时间来正确冷冻，然后在移除瓶子占位符的过程中幸存下来。最终，他发现装满沙子的玻璃瓶效果最好，你必须把瓶子一层层冷冻起来。尽管他称自己是一个没有耐心的人，但我们见过一些人坚持让他们的[冷盘更快地变得性冷淡。休息之后你可以看到一段冰盒的视频。](http://hackaday.com/2008/05/28/beverage-cooling-contraption-contest/)
