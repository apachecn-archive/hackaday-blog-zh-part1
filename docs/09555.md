# 浓缩咖啡升级为您提供更多关于咖啡因的数据

> 原文:[https://hack aday . com/2012/01/23/espresso-upgrade-give-you-more-data with-you-you-咖啡因/](https://hackaday.com/2012/01/23/espresso-upgrade-gives-you-more-data-with-your-caffeine/)

![](../Images/886b7b7d1056dc8d29453cd5534c93ec.png "espresso-display-upgrade")

不管你是单杯还是双杯，一个伟大的咖啡师都想知道咖啡机的细节。[鸢]正在愉快地制作早茶，这时他意识到他机器上的针式温度计不再工作了。他没有花很多钱直接更换，而是为这台咖啡机制作了自己的显示器和控制器 ( [译为](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.tobias-schlegel.de%2F%3Fpage_id%3D767))。

他在这次黑客攻击中进了几个球。显然，他需要更换温度计，但他也想要一个彩色显示屏和一些计时选项。他得到了一个漂亮的小有机发光二极管展示品，可以放在空出来的地方，而且只花了几块钱。他有自己的迷你轧机，在制作电路板以容纳驱动附加设备的 ATmega16 时，这很方便，但他也用它来制作屏幕更换的支架。

现在他的机器修好了，看起来更现代了，而且有了更多的功能，休息后会在视频中展示。如果你正在寻找给你的咖啡仪式添加一些定制电路，你也可以从这个类似的咖啡机黑客中得到一些灵感。T3
