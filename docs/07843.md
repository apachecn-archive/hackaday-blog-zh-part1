# 一个简单的 DIY 太阳能跟踪器

> 原文:[https://hackaday.com/2011/06/17/a-simple-diy-solar-tracker/](https://hackaday.com/2011/06/17/a-simple-diy-solar-tracker/)

[![solar panel](../Images/654aac61f17f2a95c50b8b524208b4ad.png "solar panel")T2】](http://hackaday.com/2011/06/17/a-simple-diy-solar-tracker/solar-panel/)

太阳是一个巨大的能源，然而，有效地收集这种能量可能很难做到。有一件事可以提高太阳能利用的效果，那就是实际跟踪太阳的运动。[范曼 1981]用一些非常巧妙的技术连接了他自己的自制太阳能跟踪器。

在这次黑客攻击中，他使用了两个 Harbor freight [45 瓦太阳能套件](http://www.harborfreight.com/45-watt-solar-panel-kit-90599.html "Harbor Freight")，底部的一些支柱，以及一个连接到“老派”天线马达的碟形网络卫星天线支架。虽然有人可能认为这个设备使用某种传感器来跟踪太阳，但实际上只是用遥控器将设备连接起来，以定时间隔旋转。这个区间是借助 livingonsolar.com 的[上的一些图表算出来的。](http://www.livingonsolar.com/ "living on solar.com")

要查看它的使用情况，请在休息后观看视频。他很好地解释了一切是如何组合在一起的，但如果你只是想看它移动，快进到 5:26 左右，真的很令人印象深刻。
