# 一日一黑就是招人！

> 原文:[https://hackaday.com/2008/04/10/hack-a-day-is-hiring/](https://hackaday.com/2008/04/10/hack-a-day-is-hiring/)

这是真的，尽管[威尔]和我喜欢做一个斗志昂扬、态度‘永不言败’的二人店，并且喜欢在成功编译后挥拳猛击，但我们需要更多的人(甚至是 MOAR)。我们正在寻找更多的贡献者来帮助每日发布，挖掘额外的模糊内容，特别是我们已经开始重新研究的方法。

这是一个有偿的自由职业职位，需要专业性、一致性和可靠性。我们想听听对软件/硬件黑客和日益增长的每日黑客有热情的人的意见。如需申请，请将以下材料寄至 jobs@hackaday.com

*   关于你自己的简短简历
*   3 个以每日黑客风格撰写的每日帖子示例
*   3 个你可以亲自执行的方法。关于我们过去所做工作的例子，请看这里的、这里的、这里的和这里的。
*   两句关于如何通过功能或内容来改进网站的话
*   还有什么其他原因让你成为“每日黑客”的合适人选吗

请不要发送任何附件，尤其是你的甜蜜乘坐的照片。厌恶大写字母不是必须的，但绝对是鼓励的。
