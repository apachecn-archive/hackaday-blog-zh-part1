# 全自动高尔夫球枪

> 原文：<https://hackaday.com/2010/04/07/fully-automated-golf-ball-gun/>

![](img/b20c72b363f351cee3bb5212cf0fc51f.png "golfball-gattling-gun")

如果你进入射击线，这个高尔夫球发射器会留下痕迹，或者更糟。它与[香肠枪](http://hackaday.com/2010/02/20/sausage-gun-19mm/)基于相同的前提，但现在一切都是自动化的，没有使用肉制品。

漏斗储存一排高尔夫球。到了装填时间，一个球落入燃烧室，启动燃料借助风扇喷入燃烧区，两个燃烧室密封后，推进剂被点燃。
