# Kinect 开源驱动演示和黑客攻击

> 原文：<https://hackaday.com/2010/11/10/kinect-open-source-driver-demo-and-hacking/>

Kinect 第一个开源驱动的竞争正在升温。【Marcan42】已经[发布了一个做视频和深度的驱动](http://git.marcansoft.com/?p=libfreenect.git)。他不用 Xbox 就能做到这一点，休息之后你就能看到他的表现了。【LadyAda】也一直在努力工作，[记录并转储数据](http://www.adafruit.com/blog/2010/11/09/kinect-hacking-video-and-data-dump/)，甚至利用 Kinect 中的电机编写了[“hello world”。
我们不确定[Marcan42]是如何记录他的数据的，但我们可以看到[Ladyada]正在使用高速 Beagle USB 480 来记录 Xbox360 和 Kinect 之间的双向数据。这就是我们想要的那种玩具。对于那些不知道这些大惊小怪是为了什么的人来说，](http://www.adafruit.com/blog/2010/11/09/our-first-kinect-hack-hello-world-with-the-motors/)[有一个竞赛](http://hackaday.com/2010/11/04/kilobuck-open-kinect-project-prize/)看谁能首先得到一个开源驱动程序。每当微软说它不好的时候，这个奖就会增加。
