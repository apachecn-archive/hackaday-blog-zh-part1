# 看啊，卵母细胞！

> 原文:[https://hack aday . com/2011/06/27/looke-the-oomphalapotronium/](https://hackaday.com/2011/06/27/behold-the-oomphalapompatronium/)

![Oomphalapompatronium](../Images/9c44660276ea2ff207fdbe6ee5fa53a2.png "Oomphalapompatronium")

在超过 25 年的时间里，[莱恩·所罗门]一直在表演一个人的综艺节目，该节目以看起来疯狂的、在空气中操作的手工乐器为特色。他的一些更著名的乐器包括他建造的 callioforte，以及他的雄伟的 Bellowphone，当然，两者都是由某种形式的风箱驱动的。

他最近的音乐创作是[Len]喜欢称之为“ [Oomphalopompatronium](http://www.bellowphone.com/photo-and-press.html) ”(试着说快五倍)。风箱驱动的威利·旺卡式风琴看起来是用他能得到的任何东西建造的。我们发现了一些塑料和玻璃瓶，大量的聚氯乙烯管，和一些锡罐，然后我们停止寻找，只是听。

乍一看，你可能会认为 Oomphalopompatronium 会为音乐制造一些不和谐的借口，但一旦[Len]走到键盘前，我们就会感到惊喜。声音是一个小规模的 Oompah 乐队，就像它的名字所暗示的那样。

我们认为这是一个奇妙的创造——我们只是对不能亲自去看看感到失望。
