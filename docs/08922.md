# 巨大的柔性 LED 矩阵几乎可以佩戴在任何地方

> 原文:[https://hack aday . com/2011/10/26/huge-flexible-led-matrix-can-beat-by-anywhere/](https://hackaday.com/2011/10/26/huge-flexible-led-matrix-can-be-worn-almost-anywhere/)

![lightbright_led_mesh](../Images/7ff7134cdc42c8dc5266ec57f56ce727.png "lightbright_led_mesh")

[Erik]一直忙于他的最新项目，一个灵活的 RGB LED 矩阵，他称之为[“Project Light Bright”](http://lightbright.net/)。 [BuildLounge](http://www.buildlounge.com/) 的人们告诉我们，这是他们“光竞赛”迄今收到的第一份参赛作品，他们将向最佳参赛作品赠送[一台免费激光切割机](http://www.buildlounge.com/2011/10/07/buildlounge-and-full-spectrum-lasers-are-giving-away-a-laser-cutter/)。

[Erik]手工焊接 10 个 16×16 RGB LED 面板，以构建这个显示器，结果令人惊叹。整个事情由一个支持 WiFi 的 Beagleboard 控制，它驱动所有的面板和一些面板。Beagleboard 具有嵌入式 web 和 DNS 服务器，这使它可以充当无线 AP，使他能够使用任何支持 WiFi 的设备来控制显示器。Light Bright 显示各种预定义的艺术作品，但[Erik]也可以通过他的手机随时改变显示。整个装置由一个合理大小的脂肪电池组供电，他把它藏在口袋里，这使得显示器可以连续运行大约 20 个小时。
