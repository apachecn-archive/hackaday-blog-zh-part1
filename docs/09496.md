# 更换过热的平板电脑电源

> 原文:[https://hack aday . com/2012/01/13/更换过热的平板电脑电源/](https://hackaday.com/2012/01/13/replacing-an-overheating-tablet-power-supply/)

![](../Images/f6ee40c3ef4a35d46a9ef6af5673bf44.png "asus-power-transformer-replacement")

[Dave]有一台有点问题的华硕平板电脑。当插入一个特殊的墙壁变压器时，该设备通过对接连接器的 USB 电缆充电。问题是墙壁单元往往会过热，并由内部的热敏电阻关闭，以避免永久损坏。网上的说法是把它放进一个拉链袋里，放在冰箱里冷藏一会儿。尽管这很有效，但这不是他所寻求的永久解决方案。相反，他扔进了零件箱，[没有买任何东西就自己制造了电源替代品](http://www.youtube.com/watch?v=UaG__WOWEHI)。

该设备只是在 USB 电缆的电源引脚(引脚 1)上寻找 12V。[Dave]翻遍了他堆积如山的未使用的交流适配器，找到了一个符合库存设备的电压和电流规格的适配器。他还抓起一个布满灰尘的旧主板，从背面拔下 USB 端口。一点原型板是一个很好的基础，可以将交流适配器线连接到端口，然后用一个大的收缩管覆盖。结果如上所示，并在休息后的剪辑中演示。
