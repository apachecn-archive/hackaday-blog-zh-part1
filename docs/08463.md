# 昆虫变成了管弦乐队；没有伤害，但非常不方便

> 原文:[https://hack aday . com/2011/08/31/昆虫变成乐队-没有受到伤害-但是非常不方便/](https://hackaday.com/2011/08/31/insects-turned-into-orchestra-not-harmed-but-terribly-inconvenienced/)

![](../Images/a2394de09afabce461b2217d39d5229c.png "cricuts")

[Jochem]一直对自然界的混沌着迷，无论是一大群蚂蚁还是培养皿中的进化。他的[昆虫乐队](http://jochemvangrieken.wordpress.com/2011/08/28/the-insect-orchestra/)将自然世界的混乱变成完全人工的东西。在这种情况下，MIDI。

为了建造，一个光传感器被放置在一个试管的底部。然后将一只蟋蟀(或蚱蜢，或蝗虫)放入试管中。然后用装有发光二极管的盖子将试管封闭。Arduino 读取光传感器，然后[通过 MIDI](http://shiftmore.blogspot.com/2010/01/quick-and-dirty-arduino-midi-over-usb.html) 传输数据。MIDI 命令是在 Abelton Live 中拾取的，它将所有内容转换为音频。

[Jochem]装配了 Abelton 让昆虫在四种不同的模式下表演——乐器、合成、样本和鼓。相对于不起眼的墨西哥跳豆来说，这绝对是一个进步。
