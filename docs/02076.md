# AirPiano:触摸免费红外钢琴

> 原文：<https://hackaday.com/2008/06/27/airpiano-touch-free-ir-piano/>

 <https://www.youtube.com/embed/Gh4Ovsh2DZY?version=3&rel=0&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <br/>我们很兴奋地得知<a href="http://createdigitalmusic.com/2008/06/27/airpiano-touch-free-sensing-gestural-music-controller/" target="_blank">【欧默·约沙】的 AirPiano </a>，但作为挑剔的书呆子，我们认为它更像是特雷门琴而不是钢琴。该设备让用户在钢琴上挥挥手就能弹奏 24 个不同的音符。“键”排列成三层矩阵，每层八个音符；如果用户将手放在钢琴上，相应的音符也会持续。</p> <p>它的结构相对简单:一个 Arduino、几个红外传感器和对应于音符的 led 组成了这个设备。它没有任何内置的音频硬件，所以它最基本的用途是作为 MIDI 控制器。尽管如此，它本质上是一个未指定的输入设备，所以理论上它可以用于其他目的。对我们来说，看着这架飞机飞行很有趣。</p> <ul><li class="readlink"><a href="http://createdigitalmusic.com/2008/06/27/airpiano-touch-free-sensing-gestural-music-controller/">永久链接</a></li></ul> </body> </html>
