# 手机控制的灯光显示简单有趣

> 原文:[https://hack aday . com/2011/12/28/phone-controlled-light-display-is-simple-and-fun/](https://hackaday.com/2011/12/28/phone-controlled-light-display-is-simple-and-fun/)

![android-phone-light-toy](../Images/54669a89ed6a1c59a15732d84c048c12.png "android-phone-light-toy")

[Ytai Ben-Tsvi]写道分享了他和朋友[Al 克林]一起做的一个小小的假日项目，[一个动态灯光显示器](http://www.instructables.com/id/Android-Controlled-LED-Strip-IOIO-Powered/?ALLSTEPS)，它从他的 Android 智能手机中获取线索。

这个展示适合放在[Ytai]家庭活动室的花瓶里，虽然建造它并不便宜，但它看起来确实不错。花瓶里布满了羽毛状的装饰小块，有助于隐藏可寻址的 RGB LED 灯条。灯光由一个 IOIO 板控制，这对情侣也把它藏在花瓶里。

IOIO 板还配备了 USB 蓝牙加密狗，允许它与任何运行相对较新版本的 Android 操作系统的手机进行通信。当连接上时，手机通过内置的摄像头对周围环境进行采样，命令花瓶将手机看到的颜色混合到闪烁的显示屏中。

正如你在下面的视频中所看到的，它在处理颜色鲜艳的实心物体时效果很好。虽然目前只是一个有趣的玩具，但[Ytai]和[Al]对如何扩展其可用性有很多想法。
