# 离散逻辑驱动游戏开发

> 原文:[https://hack aday . com/2011/09/25/discrete-logic-driving-game-development/](https://hackaday.com/2011/09/25/discrete-logic-driving-game-development/)

![](../Images/88e3f559eb7637e0cecdca5dfbd51108.png "discrete-logic-driving-game")

[Caleb]正在努力开发一款基于 7400 系列逻辑芯片的驾驶游戏。这将是他在[公开赛 7400 逻辑竞赛](http://hackaday.com/2011/09/02/get-out-the-graph-paper-get-started-on-the-new-discrete-logic-contest/)中的参赛作品，它真正概述了为什么这场竞赛特别棘手。

游戏背后的概念很简单。你是一辆汽车的司机(上面显示的正方形底部的红点),当你开车时需要在路上的弯道中导航。这与我们在六月份看到的[在收据纸](http://hackaday.com/2011/06/20/receipt-racer-wastes-a-lot-of-paper/)上玩的游戏相同。[Caleb 的]使用 LED 矩阵作为显示器，我们有信心，如果我们找到我们最喜欢的微控制器，我们可以在一个下午内将它安装并运行在 8×8 双色显示器上。但是，没有可编程芯片的支持，它真的会激发你内心的聪明工程师。

上面看到的电路是一个 [Logisim](http://hackaday.com/2011/02/09/logisim-open-source-digital-logic-simulator/) 概念验证，然后【Caleb】在试验板上进行[测试。他认为他已经想通了一切，直到他意识到他的数据触发器偶尔不会像他预测的那样上电。别担心，他找到了](http://cpprojects.blogspot.com/2011/09/little-bit-of-bread-boarding.html)[解决问题的方法](http://cpprojects.blogspot.com/2011/09/test-test-and-retest.html)。但是我们迫不及待地想看到他在完成这个项目的过程中还会遇到哪些障碍。
