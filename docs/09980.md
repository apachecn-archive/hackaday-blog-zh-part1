# 用铁磁流体建造人工心脏

> 原文:[https://hack aday . com/2012/03/27/用铁磁流体建造一个人造心脏/](https://hackaday.com/2012/03/27/building-an-artificial-heart-with-ferrofluids/)

![](../Images/92eba344915434f47275c35b24d54663.png "heart")

这是我们认为永远不会在黑客日看到的东西。克里斯·苏布洛克正在开发一种人工心脏，他称之为“钢铁之心”。这是一个由电磁体和铁磁流体驱动的人造心脏。

[克里斯]“人造心脏”背后的想法简单而巧妙。一个弹性薄膜在一个框架上被拉伸，磁性液体(或者铁磁流体，如果你喜欢的话)被倒在薄膜上。电磁体被激活，薄膜伸展开来，模拟心脏的跳动。将这些放在一起，你就有了一个紧凑的，生物惰性的泵，非常适合替换老化的心脏。

[Chris]“使用铁磁流体和电磁体作为人工心脏的计划让我们停下来想想他在这里做了什么。以前，人工心脏使用气动或马达将血液泵送到全身。气动泵需要从体内伸出塑料管，这不是一个令人满意的长期解决方案。电机驱动的泵会使红细胞破裂，导致[溶血](http://en.wikipedia.org/wiki/Hemolysis)。使用铁磁流体和弹性薄膜可以两全其美——无损血细胞和透皮感应充电。

克里斯不仅设计了一个怪异的人造心脏，他还提出了铁磁流体的一个有用的应用。我们几乎已经准备好将悬浮在液体中的磁性粒子作为[酷科学玩具](http://www.youtube.com/watch?feature=player_embedded&v=OE2pB1pyZN0)或艺术灵感一笔勾销。休息之后，你可以看看[Chris]的 indiegogo 视频，里面有铁磁流体泵的演示。
