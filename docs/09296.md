# 小便灯为夜间小便提供指引

> 原文:[https://hacka day . com/2011/12/14/pee-light-gies-guidance-for-night-tinkes/](https://hackaday.com/2011/12/14/pee-light-gives-guidance-for-nocturnal-tinkles/)

![](../Images/a157b7571a151bac027030639b5625a4.png "pee-light")

这盏灯是一个相当暗的 LED 模块，其目的是在你晚上使用洗手间时为你提供一点照明。如果你依靠它而不是使用浴室里的顶灯，你将能够找到回到床上的路，而你的夜视不会受到干扰。

[Fred]建立这个项目是为了学习更多关于使用 MSP430 微控制器的知识。上面看到的原型板有一对母引脚接头，用于接收一个 [MSP430-PIR 板](http://olimex.com/dev/msp-pir.html)，该板使用低功耗微控制器来监控 PIR 运动传感器。芯片可以被重新编程，[Fred]就是这样做的，使用[ez 430-f 2013 开发棒](http://hackaday.com/2011/01/30/hands-on-with-ez430-f2013/)的 USB 加密狗一侧。现在，当传感器检测到运动时，芯片首先检查原型板上的光敏电阻，以查看浴室中是否黑暗。如果是这样，它会打开 LED，并设置一个计时器来关闭它。

该系统采用 9V 电池供电，对于 12V 规格的 LED 模块来说，这有点不足。但是[弗雷德]说它产生的光强度正好合适。
