# 惊人的机箱黑客

> 原文:[https://hackaday.com/2010/01/11/amazing-chassis-hacks/](https://hackaday.com/2010/01/11/amazing-chassis-hacks/)

![](../Images/4ae2b898e548abc42300b8fcd458be60.png "RC-chassis-hacks")

[Crabfu]在一个遥控传动系统上完成了一些伟大的底盘工作。他最近的作品[将这个小小的旅行者变成了一辆月球车](http://crabfuartworks.blogspot.com/2010/01/micro-rc-moon-buggy.html)，上面配备了乘客和通讯阵列。为此，他从一个玩具中获得零件，但用手绘细节提升了真实感，让我们惊叹不已。他之前的项目[从一个卡车模型套件](http://crabfuartworks.blogspot.com/2009/12/micro-crawler-with-model-kit-body.html)中获得了尸体。还是那句话，是油漆工作让我们羡慕他的技术。

这两个项目都隐藏了一个 [Losi 1/24 比例的微型岩石爬行器](http://www.losi.com/Products/Default.aspx?ProdID=LOSB0236BD)，它提供了一些令人难以置信的移动。休息后观看两个版本的视频。
