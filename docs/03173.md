# 移动 RFID 扫描

[Chris Paget]将于本周在华盛顿特区举行的 2009 年 ShmooCon 上发表演讲。他向收银员展示了他的 RFID 演讲的[预览。上面的视频演示了在旧金山巡游时读取和记录随机标签和](http://www.theregister.co.uk/2009/02/02/low_cost_rfid_cloner/ "Passport RFIDs cloned wholesale by $250 eBay auction spree • The Register")[护照卡](http://travel.state.gov/passport/ppt_card/ppt_card_3926.html "U.S. Passport Card")的唯一 id。他使用一个符号 XR400 RFID 阅读器和一个安装在他车内的摩托罗拉 AN400 贴片天线。这是工业设备，通常用于跟踪包裹或牲畜的移动。这一代比 2005 年 Flexilis 创造的距离阅读记录[还要新。](http://blog.makezine.com/archive/2005/07/_defcon_rfid_wo.html "@ DEFCON RFID World record attempt...")

护照卡上唯一的身份证号码不会泄露主人的私人信息，但对他们来说仍然是唯一的。它可以用来跟踪主人，当与其他细节结合时，如他们的 RFID 信用卡，可以建立一个人的档案。这就是为什么美国公民自由联盟反对目前形式的护照卡。美国确实为卡提供了一个屏蔽套…当然，它是和卡一起邮寄给你的，卡放在屏蔽套外面。

每当 RFID 卡被读取时，存在产生随机 ID 的技术。最近更新了对 RANDOM_UID 的支持。
