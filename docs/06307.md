# Jeri 制造柔性 EL 显示器

> 原文:[https://hack aday . com/2010/11/16/jeri-makes-flexible-El-displays/](https://hackaday.com/2010/11/16/jeri-makes-flexible-el-displays/)

一次失败的化学实验让 Jeri Ellsworth 发现了一种用于电致发光显示器的柔性基板。我们熟悉玻璃面板背面的 EL 显示器，就像你在音频接收器中会发现的那样，但在用铝箔制成网格后[Jeri]研究了使用多孔金属作为磷光体的宿主。她从清洁铝箔开始，用乙烯贴纸来防止铝的腐蚀。然后它进入硼酸池，箔片作为阳极通电。当箔片被蚀刻时，她用激光穿透箔片来测试进程。在这之后，磷光体被施加到箔的背面，覆盖在电介质中，并以导电墨水结束，该导电墨水将携带激发磷光体所必需的交流电。这是与[她的 EL PCB 实验](http://hackaday.com/2010/08/25/el-wire-make-it-connect-it-power-it/)相反的分层材料。休息之后，请参见[Jeri]在视频中自己解释这一点。

你可以在上面看到，这产生了一个非常明确的显示区域。这让我们想起了那个[变色涂料显示器](http://hackaday.com/2009/09/15/color-changing-paint-display/)。我们认为使用这种方法构建一些 7 段显示器是值得一试的。
