# 超级任天堂时钟

> 原文:[https://hackaday.com/2008/11/11/super-nintendo-clock/](https://hackaday.com/2008/11/11/super-nintendo-clock/)

![super_nintendo_clock](../Images/7931490a4ae0506e941d67facf0ba428.png "super_nintendo_clock")

[Aaron]给我们送来了这个他制作的可爱的超级任天堂时钟。当他发现自己带着一个坏掉的 SNES 时，一个朋友向他挑战，要他[做他以前的 NES](http://mavrinac.com/index.cgi?page=nesclock)钟。这一次，显示器将位于盒中，并且是可移动的。设置时间是在 SNES 控制器上完成的，正如你在休息后的视频中看到的那样，它似乎工作得很好。
