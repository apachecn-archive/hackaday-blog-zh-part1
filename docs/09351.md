# 花板 LED 立方体

> 原文:[https://hackaday.com/2011/12/21/flowerboard-led-cube/](https://hackaday.com/2011/12/21/flowerboard-led-cube/)

[![](../Images/ef7ff12f7624ae2a7f800361270ec14e.png "cube")T2】](http://hackaday.com/wp-content/uploads/2011/12/cube.jpg)

这是一个用电子断路器[花原型板](http://www.elecfreaks.com/store/flower-22-protoboardmega-shield-p-378.html)制作的整洁的 [4x4x4 LED 立方体](http://www.elecfreaks.com/2452.html)。

几天前，我们[发布了](http://hackaday.com/2011/12/16/hackaday-links-december-16-2011/)一款专为 SMD 工作制作的整洁的新原型板。ElecFreaks 团队决定采用一种花形垫，而不是通常的“带圆圈的孔”原型板布局。这使得在构建任何组件时处理 SMD 组件变得特别容易。为了展示他们的新原型板，ElecFreaks 建造了一个令人敬畏的 4^3 LED 立方体。看看那些[焊料痕迹](http://www.elecfreaks.com/wp-content/uploads/2011/12/ledcube-09.jpeg)就知道了。

LED 立方体本身就是[没什么](http://hackaday.com/2011/09/21/attiny-hacks-attiny-controlled-4x4x4-led-cube-has-a-unique-design/) [我们](http://hackaday.com/2011/09/18/attiny-hacks-2313-driving-a-4x4x4-led-cube/) [在](http://hackaday.com/2011/08/14/largest-led-cube-weve-ever-seen-is-still-only-half-complete/)之前没见过 [，但是这个东西的构造很神奇。整个建筑在 Arduino Mega Flower shield 上，这意味着根本没有电线。从电阻到晶体管，一切都是 SMD 元件。现在唯一的问题是弯曲和焊接所有这些 LED 引线。](http://hackaday.com/2011/08/04/using-an-led-cube-as-an-audio-visualizer/)

这个花原型板开始看起来越来越有趣了；休息之后，请查看实际情况。
