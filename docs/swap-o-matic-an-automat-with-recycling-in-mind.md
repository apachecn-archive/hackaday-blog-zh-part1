# Swap-O-Matic:一种考虑回收的自动装置

> 原文：<https://hackaday.com/2011/09/16/swap-o-matic-an-automat-with-recycling-in-mind/>

![](img/52e57ec86b021c346f4f9405b3d77162.png "swapomatic")

Swap-O-Matic 自动售货机是为回收而不是消费而制造的。你可以把你不再需要的东西换成其他东西，而不是把钱放进机器里。这是一个伟大的概念，有着伟大的复古设计，可能是受了自动机器人[时代的影响。](http://www.youtube.com/watch?v=ZrbyqUOObdU)

[莉娜·费内基多]和[里克·卡西迪]在[莉娜]获得艺术硕士学位时建造了这个 Swap-O-Matic。版本[在 2005 年上线](http://www.wired.com/culture/lifestyle/news/2005/05/67665)，但该项目从那时起就一直在更新，并在纽约州布鲁克林的[发射台](http://www.facebook.com/pages/LaunchPad/289165126884)有了一个新家。第一个版本的[在机器旁边使用了一个独立的电脑，给门上的锁分配密码。看起来新版本已经改进了，集成了触摸屏和电脑激活的锁，给[Lina]和[Rick]留下了非常干净的造型。](http://vimeo.com/15691335)

如果你有一个有相似兴趣的相对同质的人群，这是一个很好的想法，所以我们希望在一些黑客空间看到一些这样的人出现。休息之后，请观看 Swap-O-Matic 宣传片。
