# 银行家能减轻你的挫败感

> 原文:[https://hack aday . com/2012/01/23/whac-a-banker-to-relief-your-sufference/](https://hackaday.com/2012/01/23/whac-a-banker-to-relieve-your-frustration/)

![](../Images/87eb3aefc14ca675454f86c146133278.png "banker")

[蒂姆·亨金]，杰出的建筑家和*机器的秘密生活*的主持人，对当前的经济气候有点沮丧，决定用一个游戏[来发泄他的挫折感。](http://www.timhunkin.com/a153_whackabanker-words.htm)

[Tim]版本的经典 Whac-A-Mole 游戏使用微型气缸上下驱动五个银行家小雕像。这些雕像是用聚亚安酯浇铸的，据报道保持得很好。柜子真的很有意思。不玩游戏时，会显示一个[非常无聊的前面板](http://www.timhunkin.com/page_pictures/a153_in-arcade.jpg)。在插入一些硬币后，面板落入机器中，显示出[有趣和令人兴奋的记分牌](http://www.timhunkin.com/page_pictures/a153_pacman-with-whacka-banker.jpg)。

如果你曾经去过英国的萨福克，你可以去看看南沃尔德码头的 Whac-A-Banker 和许多其他的[Tim Hunkin]作品。谢谢[约翰]送来这封信。是的，我们确实在探索频道上看到了*美国机器的秘密生活*，但是它已经被关于卡车运输的节目取代了。这里是《T4》整个系列的档案，供你观赏。吻别你的夜晚。
