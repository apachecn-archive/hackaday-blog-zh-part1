# USB Reddit 向上投票/向下投票按钮

> 原文:[https://hack aday . com/2011/07/30/USB-Reddit-upvotedownovote-button/](https://hackaday.com/2011/07/30/usb-reddit-upvotedownvote-button/)

![reddit_upvote_downvote_button](../Images/32835ff36a629b2146753403c23fc650.png "reddit_upvote_downvote_button")

[Chris]最近自称是 Reddit 的粉丝，他想开发一些东西来简化帖子投票的过程。受不久前 [Make](http://blog.makezine.com/archive/2011/04/the-awesome-button.html) 上的[牛逼按钮 hack](http://hackaday.com/2011/04/13/hackaday-links-april-13-2011/) 的启发，他认为一个物理的向上/向下投票按钮将是满足他所有 Reddit 需求的理想外设。

他非常喜欢使用 Reddit 增强套件，该套件允许您只需一次按键就可以提交投票。他将这个浏览器扩展与一个小型开发板结合起来，很快就做出了投票按钮的原型。一旦他对用来模拟所需按键的 Arduino 草图进行了微调，他就开始忙于为他的创作制作一个外壳。

他拿出他的 3D 打印机，在大约 30 分钟内就有了一个定制的工作项目箱。他为“向上投票”和“向下投票”按钮打印了箭头，将所有的东西合在一起，然后旋转了一下他的快速投票箱。一切都很完美，他对自己的创作非常满意。

我们想象我们的[驻地 Reddit 上瘾首席](http://hackaday.com/2011/04/27/motion-controlled-reddit-vote-sign/)现在相当嫉妒…
