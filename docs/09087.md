# 头戴式特斯拉线圈——会有什么问题呢？

> 原文:[https://hackaday . com/2011/11/18/a-头戴式特斯拉线圈-什么-可能-出错/](https://hackaday.com/2011/11/18/a-head-mounted-tesla-coil-what-could-possibly-go-wrong/)

我们现在可以听广告了…

“真正的 geniuuuuuuus 人！这是给你的无所畏惧先生，[在你的帽子上唱特斯拉线圈戴着家伙……"](http://tcengineering.wordpress.com/2011/11/13/hat-coil/)

称他为天才或称他为疯子——我们只知道[泰勒的]特斯拉线圈帽棒极了！即使现在是 11 月中旬，我们也不能错过这套万圣节服装。

[泰勒]的发明本质上可以归结为一个标准的双共振固态特斯拉线圈，并将其缩小到一个合理的大小，以便安装在他的头骨上。迷你特斯拉的特点是有一个非常紧凑的增压核心，至少在一段时间内工作得相当好。他说，这种助推器从未真正正常工作过，需要重新设计，这是他有空时会处理的事情。

不管有没有问题，我们认为下面的视频说明了一切。这顶帽子当然是一个引人注目的东西，我们认为它很棒——即使在头上戴一个特斯拉线圈不一定是最安全的事情。
