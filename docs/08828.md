# 参赛轮盘赌

> 原文:[https://hackaday.com/2011/10/14/contest-entry-roulette/](https://hackaday.com/2011/10/14/contest-entry-roulette/)

![](../Images/84be91feaa7511bca3fbd4bb48e4d9a0.png "scrap-project-contest-entries")

超过 150 个由废弃零件制成的项目已经发布，供你观赏。它们构成了《c 't》杂志“快速制造废料”项目的条目。我们已经参观了[一台卫生纸打印机](http://hackaday.com/2011/09/27/toilet-paper-printer-made-from-scrap-parts/)，但是还有很多其他有趣的东西可以看。

每次在顶部加载链接时，页面都会选择一组不同的条目来显示。你可以点击所有页面，或重新加载玩一个小项目轮盘赌。上面的图像显示了三个引起我们注意的。右上方是一个骑着老式电脑鼠标的 lighbulb-man 反牛仔风格。奇怪的是，任何人只要能接触到电镀设备，就能把像这样简单的东西制作成令人敬畏的桌面雕塑。

顺时针方向移动，我们有一个小的 [USB 驱动器与一个旧的真空管](http://www.heise.de/mach-flott/projekte/Steampunk-Speicherstick-1351348.html)配对。机器翻译有点粗糙，但它看起来像拇指驱动器的 LED 给管一点红色的光芒。我们只是觉得随身携带真空管很有趣。

最终的横幅图像显示的是[摄像机的陀螺仪](http://www.heise.de/mach-flott/projekte/Schwebestativ-fuer-Videokameras-1338944.html)。这是一个很棒的设置，休息后你可以看到它的运行。一对损坏的硬盘驱动器为摄像机提供运动稳定性。整个组件的顶部有一个带有万向节的手柄。当驱动器旋转时，平台保持不动，即使操作员疯狂地转动设备。
