# 大多数无用的机器减少了碳足迹

[克莱顿·博耶]将电力从无用的机器中取出，[制造出一台像时钟一样运转的机器](http://www.lisaboyer.com/Claytonsite/mysterybox.htm)。关于这一点，我们一直看到这些[无用的机器使用电动马达](http://hackaday.com/2009/12/30/simplest-most-useless-machine/)。[Clayton]的巧妙设计使用了发条和一系列木制齿轮来带来乐趣，使其成为您制作的[二进制加法器](http://hackaday.com/2009/10/13/binary-adder-will-give-you-slivers/)的绝佳伴侣。上面的视频显示了内部工作和设计计划是出售。我们很想[打印出零件](http://hackaday.com/2010/06/05/3d-printed-makerbot/)或者也许只是从木头[上激光切割它们，就像这个蜘蛛机器人](http://hackaday.com/2010/01/21/spiderbot-prompts-laser-envy/)的腿一样。
