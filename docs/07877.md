# 每次都是完美的啤酒

> 原文:[https://hack aday . com/2011/06/18/the-perfect-beer-every-time/](https://hackaday.com/2011/06/18/the-perfect-beer-every-time/)

![](../Images/d5d3f0d0c55a8188b56c0de65fae33e3.png "pour-master")

倒酒大师 Pro 是一个啤酒倒酒机器人，由一群啤酒/机器人爱好者设计和制造，作为他们参加红牛创作大赛的参赛作品。倒酒大师保持它的简单(与我们见过的其他[酒吧机器人](http://www.youtube.com/watch?v=7KRxHKaR2jo "barbot")相反)，它使用一个修改过的啤酒桶和水龙头，以及一些传感器，它用这些传感器来保持它的状态并倒出完美的啤酒。kegerator 上的标准塔被使用 Vex 机器人设计系统构建的齿轮齿条驱动塔所取代，这使得倾倒大师可以使用限位开关和一套超声波测距仪将其高度设置为任何玻璃的尺寸。

对于一次完美的倒酒，啤酒不能溢出玻璃杯的边缘，需要一个合适的 3/4 英寸的头，为了做到这一点，倒酒师使用超声波测距仪来检测头的厚度和高度。整个事情由运行有限状态机的 Arduino 控制，该状态机通过 LCD 显示器向用户提供状态反馈。休息之后，看看他们的参赛视频，现在你需要的是一个机器人来制作啤酒，为什么不让另一个机器人来让 T2 喝啤酒呢？
