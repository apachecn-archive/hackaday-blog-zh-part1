# 四代运动模拟器

> 原文：<https://hackaday.com/2009/12/21/four-generations-of-motion-simulators/>

![](img/26460f67fd3fc931bfca4a018c474496.png "motion-sims")

我们喜欢好的飞行模拟器，但经常发现缺乏可用的控制方案。[Roland]不仅建立了自己的控制，而且[创造了完整的驾驶舱，将物理运动](http://www.simprojects.nl)添加到混音中。他去年完成了他的第三代驾驶舱。上图和休息后的视频都是如此。这个设计使用一个安全带系统来移动装饰好的驾驶舱。

现在他已经开始为第四代开发原型。这一次，他用三个 Sarrus 连杆取代了皮带系统。我们昨天在一台[挤压机原型](http://hackaday.com/2009/12/20/sarrus-linkage-3d-printer-2/)中看到了这些连杆，如果它们能够处理负载，应该可以很好地用于这种应用。原型的视频在休息后嵌入，但要注意，下流的抽插动作不适合心脏虚弱的人。T3
