# 曼哈顿风格的业余无线电接收器

> 原文:[https://hack aday . com/2011/09/04/a-ham-radio-receiver-Manhattan-style/](https://hackaday.com/2011/09/04/a-ham-radio-receiver-manhattan-style/)

如果你从未听说过[“曼哈顿风格”赛道建设](http://aa7ee.wordpress.com/2011/07/24/the-wbr-a-simple-high-performance-regen-receiver-for-40m-by-n1byt/)，你并不孤单。这种工艺在业余无线电圈很受欢迎，看起来比直接死虫式电路构建更好，但不像蚀刻自己的 PCB 那么复杂——可以认为这是一种不错的折中解决方案。

这种结构通常用于在覆铜外壳内构建电路，这在业余无线电爱好者中是一种常见的做法。曼哈顿式电路是用粘在金属垫上的元件组装而成的。有人可能会认为上图中的大焊盘会限制通孔元件，但事实绝对不是这样。还有多种常见引脚配置的 SMD 焊盘，允许您使用几乎任何类型的元件。

虽然它可能不适合你工作的每个项目，但曼哈顿风格的电路和覆铜盒肯定会为某些项目增添一抹亮色，比如你在上面看到的惠斯通电桥再生接收器。
