# 电锯驱动的自行车

> 原文：<https://hackaday.com/2009/07/11/chainsaw-powered-bicycle/>

![chainsawbike](img/8f3a83a3ec5afe7332de1a64c74ca8f0.png "chainsawbike")

在之前，我们已经介绍过除草机自行车，现在你甚至可以购买 66cc 转换套件。[geoff390]在 YouTube 上发布了一些关于另一种电机选择的视频。在下面嵌入的第一个视频中，他[在检查他的自行车](http://www.youtube.com/watch?v=zeofT8lUmAY)的基本部件。链锯马达安装在后轮胎的侧面，从链锯轴伸出的金属驱动轮与它接触。电机组件安装在铰链上，驱动轮和后轮胎之间的摩擦力用螺丝扣调节。他发布了一个[更详细的第二个视频](http://www.youtube.com/watch?v=xeRsg3-Jy4c)，其中他回顾了自行车的一些细节和他遇到的一些问题。 [https://www.youtube.com/embed/zeofT8lUmAY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zeofT8lUmAY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
