# 噪声发生器移植到小型 AVR 上运行，也兼容 Arduino

> 原文:[https://hack aday . com/2011/08/08/08/noise-generator-ported-to-run-on-small-AVR-also-arduino-compatible/](https://hackaday.com/2011/08/08/noise-generator-ported-to-run-on-small-avr-also-arduino-compatible/)

![](../Images/9fb63fcf43289f2ea6d90af587157f9f.png "luna-mod-for-AVR-and-Arduino")

由于手头没有 PICAXE，Rob Miles 感到有点受冷落，他决定将 Luna Mod 代码移植到 AVR 芯片上。他选择围绕 ATtiny45 构建他的，但也提到这是 Arduino 兼容的。

这个外壳布局与最初的 Make 版本略有不同，但我们更喜欢这个外观。它可能不会[满足你对时髦外观外壳](http://hackaday.com/2011/05/16/making-the-case-for-cool-project-enclosures/)的需求，但改造后的澳洲坚果盒看起来似乎就是为此而造的。休息后看一下视频，看看最终产品，听听它喷出新组成的杂音。[Rob]正在以 dropbox 文件的形式分享草图，但我们也在广告之后加入了我们自己的托管链接，以防它停止工作。
