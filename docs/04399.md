# 单极电机

> 原文：<https://hackaday.com/2009/11/10/homopolar-motor/>

![homopolar-motor](img/9d188b2b7ceda3e0d25cfcbafb87b318.png "homopolar-motor")

今天在办公室过得很慢？这里有一个让你的同事微笑的窍门。《危险的乐趣》有一个用电池、铜线和磁铁制造单极电机的指南。与固定磁铁相比，同极电机不依赖电枢中的电磁铁改变其极性来推动旋转运动。相反，它们利用电流对磁场的定向来提供排斥力或旋转力。在这种实现方式中，电流通过一圈铜线从电池的一极流向另一极。电池一极上的稀土磁体提供磁场。

休息之后，我们嵌入了这个简单例子的视频，以及几个更复杂的同极例子，如五速版本。运转中的发动机无疑给我们带来了笑容，并将它牢牢地放在无用机器家族中。
