# 自动鸡舍门

> 原文：<https://hackaday.com/2010/05/03/automated-chicken-coop-door/>

[https://www.youtube.com/embed/qUKIlZIZDqc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/qUKIlZIZDqc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这是一个非常酷的自动门的实现，是为鸡舍建造的。电子设备相当标准，一个 Arduino 和一个二手无绳钻。最终产品不仅是无线的，而且是基于环境光的自动控制的。我们还发现锁定机制相当优雅。他还提供了一个链接，链接到[另一个自动化鸡舍门](http://www.fabiens.org/ckblog/?page_id=106)，里面有一个不错的文字介绍，有图表之类的。下一步，他将不得不[自动喂食](http://hackaday.com/2005/08/25/automated-chicken-feeder/)。
