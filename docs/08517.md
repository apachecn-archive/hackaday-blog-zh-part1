# 初级概念:MOSFETs

> 原文:[https://hackaday.com/2011/09/07/beginner-concepts-mosfets/](https://hackaday.com/2011/09/07/beginner-concepts-mosfets/)

![](../Images/363bc4c6e59c5cf83e79c4e49bec558d.png "mosfet-beginners-guide")

Moser 的[MOSFET 介绍指南](http://reibot.org/2011/09/06/a-beginners-guide-to-the-mosfet/)为不熟悉器件的人提供了快速介绍。它们的作用类似于 2N2222 等双极性结型晶体管，可以切换大负载。但从根本上说，它们是不同的。金属氧化物场效应晶体管有三个引脚，分别用于漏极、源极和栅极，而不是你可能习惯的集电极、发射极和基极。栅极是器件的控制引脚，与双极性结型晶体管相比，它具有优势，因为它与沟道绝缘。这意味着与普通晶体管的基极相比，流入栅极的电流要少得多，从而节省了功率并为逻辑电路提供了保护。

如果这让你头晕，不要担心。(莫泽的)评论简短扼要，但并未淡化。你可以得到一个基本的概述，如果你想了解更多，他链接到数据表，并有基本的术语，很容易与谷歌搜索澄清。他提供的最强大的工具之一是上面看到的简单 MOSFET 和驱动器电路图。这使得非常快速地切换非常大的负载成为可能；MOSFET 的真正功率。
