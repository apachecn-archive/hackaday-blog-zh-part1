# 自动 DVD 分配器

> 原文:[https://hackaday.com/2009/03/05/automated-dvd-dispenser/](https://hackaday.com/2009/03/05/automated-dvd-dispenser/)

![dvd](../Images/bd393c805dd0ac1745641fd9d47eacd7.png "dvd")

[Andrew]为他的高中毕业设计制作了这个[DVD 分发器](http://einfacheleben.com/dvd.html)。它的大脑使用 ATmega8515，LCD 使用定制的编码驱动程序。正如你在视频中看到的，休息之后，你可以通过各种标识符如流派或标题选择一张 [DVD](http://www.mahalo.com/DVD_Hacks "DVD Hacks - Mahalo") 。然后，它会将 DVD 推出机架，这样您就可以抓取它。现在，所有的 DVD 都必须放在预定的位置，但这是一个不错的开始。感谢您发送此信息[Andrew]。
