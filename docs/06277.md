# 修复未来

> 原文：<https://hackaday.com/2010/11/10/fixing-the-future/>

![](img/a982afe31c17bbd42fec3f53d4463a94.png "Picture 4")

[ [iFixit](http://www.ifixit.com/) ](在之前我们已经多次发布了关于[的消息)发起了](http://hackaday.com/?s=ifixit)[一个充满激情的宣言](http://www.ifixit.com/Manifesto)来推广修理的技能和知识，作为对技术消费主义和浪费的解决方案。他们用加纳电子垃圾堆放场的有力镜头来表明，我们必须集体改变我们使用和联系高科技的方式——休息后看看。

该宣言反对隐瞒维修知识(如手册、错误代码和示意图)的做法——将责任推给制造商——但也明确表示，我们每个人都有责任告知自己，并重视功能而不是新奇。

考虑到我们正面临的多方面的资源危机，任何促使我们的行为朝着可持续和体贴的生活方式发展的努力都应该被考虑。作为黑客，我们将修复、重复使用和重新思考技术作为我们技能的一部分，但我们也因对技术挑战的热情而享有特权。真正的战斗是将我们所拥有的知识和技能传播给大众。这就是[ [ifixit](http://www.ifixit.com/) ]的核心信息发挥作用的地方:为每台电子设备创建一个开放的、可编辑的在线维修手册。如果你有东西要教，为什么不去帮助扩展他们的数据库呢？
