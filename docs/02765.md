# 黑掉迷你库柏

> 原文:[https://hackaday.com/2008/11/10/hacking-the-mini-cooper/](https://hackaday.com/2008/11/10/hacking-the-mini-cooper/)

![cooper_power](../Images/b45ce2a37a705e7c539a618981664c5b.png "cooper_power")

[war6763]给我们发了这个[黑客给迷你库伯](http://www.jointhetalk.net/?p=6)里的放大器供电。显然，除了异常英俊，他们也异常兴奋。放大器通常连接到点火装置或音响上，在汽车启动时打开。由于一些奇怪的布线，这只是不可能在迷你库珀。有些人用点烟器给他们的放大器供电，但是[war6763]想要一个能让他的点烟器自由地做其他事情的东西。他建立了一个电路，监控内置在 USB 输入线的汽车，并相应地打开和关闭他的放大器。整个单位成本约 10.00 美元。休息之后你可以看视频演示。
