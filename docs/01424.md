# CCCamp 2007:四轴飞行器

> 原文：<https://hackaday.com/2007/08/10/cccamp-2007-quad-copters/>

遥控四轴飞行器在夏令营中非常受欢迎。我们的朋友丹·卡明斯基拍摄了两个不同模型在飞行中的视频。第一个是 10K 欧元的商业版本，但第二个是一个 500 欧元的自制版本。商业单位有一个头戴式显示器，所以你可以从机载相机飞行。休息之后有第二个视频展示了可操作性。
