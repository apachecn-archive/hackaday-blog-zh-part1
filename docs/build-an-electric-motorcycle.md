# 制造一辆电动摩托车

> 原文：<https://hackaday.com/2012/02/16/build-an-electric-motorcycle/>

在“当我们有几个空闲的周末时”我们要建造的东西的清单上，电动摩托车就在最前面。20 英里的范围内，他们可能不像汽车或卡车那样多才多艺，但我们无法想象一种车辆更适合在镇上快速短途旅行。[本·纳尔逊]刚刚完成了他的电动摩托车，并在将一辆 81 年的川崎 KZ440 转换成电池动力上贴了一张[大说明书](http://www.instructables.com/id/Build-Your-Own-ELECTRIC-MOTORCYCLE/?ALLSTEPS)。

在检查了很少被提及的[执照、注册和保险](http://www.instructables.com/id/Build-Your-Own-ELECTRIC-MOTORCYCLE/?ALLSTEPS#step2)之后，【Ben】从他的自行车上取下引擎，安装了一个电动马达，开始了他的组装。使用的[电池](http://www.instructables.com/id/Build-Your-Own-ELECTRIC-MOTORCYCLE/step6/Batteries/)不是昂贵的脂肪电池，而是便宜的铅酸电池。铅酸电池的计算里程为 26 英里，非常适合往返于下一个城镇。

在所有东西都连接好，链条绕在插座上，一个令人敬畏的黄色油漆工作完成后，[本]终于试驾了他的自行车。休息后看看视频，了解一下[本]的自行车能跑多快。
