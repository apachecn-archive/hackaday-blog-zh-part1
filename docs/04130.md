# 噪音盒合成器放下一些节拍

> 原文:[https://hack aday . com/2009/09/29/noise-box-synth-lays-down-some-beats/](https://hackaday.com/2009/09/29/noise-box-synth-lays-down-some-beats/)

![arduino_noise_box_synth](../Images/1db702710d8ce367e0c4e68652dd3e4f.png "arduino_noise_box_synth")

[Tim]把他的噪音盒合成器发给了我们。[框是一个十六步合成器](http://mechomaniac.com/ArduinoNoiseBoxSynth)，可以生成正弦波、方波、三角波和锯齿波以及一组声音效果(休息后的视频)。硬件简单；一个 Arduino，四个电位计，四个按钮，一个开关，一个扬声器和一些 led。这是给一个三岁孩子的礼物，但我们自己打开也会很开心。我们没有找到原理图，但是所有的连接和硬件都可以从[源代码](http://mechomaniac.com/node/67)中推断出来。

Arduino 有时在这里名声不好。这个项目，[Tim 的]第一个使用 Arduino 的项目，证明了平台的可访问性使得直接跳入深水区成为可能。休息后看视频。[https://www.youtube.com/embed/lIgjR_NYKVg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/lIgjR_NYKVg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T3】
