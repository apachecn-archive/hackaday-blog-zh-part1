# 黑客泰迪·鲁斯平

> 原文：<https://hackaday.com/2011/02/18/hacking-teddy-ruxpin/>

[DJ Sures]得到了一个破损的泰迪 Ruxpin 娃娃，并决定给它一个新的机器人生命。休息后，在观看视频之前，您可能想让孩子们离开房间。故事从小泰迪被割喉开始，为了得到头部的部分。在用伺服系统改造了眼睛、嘴巴、脖子和手臂后，泰迪得到了蓝牙升级，可以让你无线控制他的动作。[DJ Sures]甚至计划让这个小家伙站起来走路，这远远超出了最初的能力。
