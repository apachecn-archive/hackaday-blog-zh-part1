# 后续:星球大战树得到升级

> 原文:[https://hack aday . com/2011/12/09/follow-up-of-star-wars-tree-gets-an-upgrade/](https://hackaday.com/2011/12/09/follow-up-star-wars-tree-gets-an-upgrade/)

![star-wars-tree-sound](../Images/f9003acb5cb0755fbf9a94e84c652b25.png "star-wars-tree-sound")

我们问了，扎克听了。

本周早些时候，[我们展示了他建造的](http://hackaday.com/2011/12/05/the-force-is-strong-with-this-christmas-tree-light-show/)电路，这使得他的小小星球大战圣诞树在视觉上复制了该系列的主题曲。你们中的一些人，包括我自己，认为如果这棵树还能播放音乐来配合灯光表演，那会酷得多，所以[扎克]开始添加这一功能。

由于担心音乐会随着灯光不断播放而变得令人讨厌，他调整了他的电路设计，加入了一个可以轻松开关的压电蜂鸣器。在将其连接到驱动灯光表演的 MSP430 后，他调整了程序，以同时向灯串和蜂鸣器输出信号。

虽然灯光秀准确地再现了歌曲，但他最初忽略了平坦和尖锐的音符，因为它们对眼睛来说是难以区分的。然而，在音频形式中，缺失的音符会非常明显，所以他重新转录了乐谱，产生了你在下面看到的视频。

如果你碰巧跟随[Zach]的领导，在你自己的房子里放了一个这样的东西，一定要去他的网站看看，获取最新的代码，包括音频。
