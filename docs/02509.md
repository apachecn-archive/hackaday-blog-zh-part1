# LED 咖啡桌

> 原文：<https://hackaday.com/2008/09/13/led-coffee-table/>

![](img/07133f1406acb2f320db8407767263c4.png "sparkfun led coffee table")

早在五月份，Spark Fun 在 Maker Faire 上的核心产品就是这个 [LED 咖啡桌](http://www.sparkfun.com/commerce/tutorial_info.php?tutorials_id=91)。他们最近刚刚发布了它是如何建造的。表面由 64 个 8×8 RGB LED 矩阵板制成，总共 4096 个 LED。八行连接到一个定制的路由器板，这样一条 SPI 线就可以控制整个显示器。主微控制器是 Olimex LPC2106 开发板。它运行一个四人合作的乒乓球游戏，其中多个球随着时间的推移被添加。每个玩家都得到一个经典的[雅达利](http://www.mahalo.com/Atari "Atari - Mahalo")控制桨。休息之后，你可以看到一个运行屏保的桌面视频。
