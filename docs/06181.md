# 机器人手爪用咖啡拿起任何东西

> 原文：<https://hackaday.com/2010/10/26/robot-gripper-uses-coffee-to-pick-up-anything/>

拿起一个生鸡蛋不是我们认为机器人手爪擅长的事情。但是[这个模型使用了球状尖端](http://www.hizook.com/blog/2010/10/25/jamming-robot-gripper-gets-official-article-published-pnas)而不是爪子，这使得压碎物体不再是一个问题。

那个尖端有点像气球。它充满了咖啡渣，但空气也可以被泵入和吸出。当要抓住一个物体时，一点空气被泵入，灯泡被压在目标上。一旦就位，所有的空气被吸出，将咖啡渣锁在物体周围。休息之后，看看有多少东西可以用这种技术抓住。

现在真正的问题是，它能给我一杯啤酒吗？
