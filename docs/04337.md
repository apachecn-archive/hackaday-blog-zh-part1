</span> <p>[Ruyck]给我们发来了他的迷你 Keepon 机器人的视频。对于那些还没有入门的人来说，<a href="http://en.wikipedia.org/wiki/Keepon" target="_blank"> Keepon </a>是一个非常情绪化，而且非常昂贵的跳舞机器人。他看起来似乎很简单，但正如你在[Ruyck]的版本中看到的，它相当复杂。[Ruyck]使用了一个迷你遥控总距直升机组件进行运动，这使得控制它相当直观。起初，我们对[Ruyck]的最终实现没有太多印象，你可以在休息后与 Keepon 的对比视频一起看到。然后我们意识到，他要做的就是找到一种方法将泡沫体的底部连接到底座上，以实现 keepon 的更多挤压和拉伸运动。一点创造性的程序设计，这个小家伙就能自主并与音乐同步。<br/> <span id="more-17887"/></p> <span class="embed-youtube" style="text-align:center; display: block;"><iframe class="youtube-player" width="800" height="480" src="https://www.youtube.com/embed/hRuB2PNAXI4?version=3&amp;rel=1&amp;showsearch=0&amp;showinfo=1&amp;iv_load_policy=1&amp;fs=1&amp;hl=en-US&amp;autohide=2&amp;wmode=transparent" allowfullscreen="true" style="border:0;" sandbox="allow-scripts allow-same-origin allow-popups allow-presentation"/></span> <span class="embed-youtube" style="text-align:center; display: block;"><iframe class="youtube-player" width="800" height="480" src="https://www.youtube.com/embed/3g-yrjh58ms?version=3&amp;rel=1&amp;showsearch=0&amp;showinfo=1&amp;iv_load_policy=1&amp;fs=1&amp;hl=en-US&amp;autohide=2&amp;wmode=transparent" allowfullscreen="true" style="border:0;" sandbox="allow-scripts allow-same-origin allow-popups allow-presentation"/></span> </body> </html>
