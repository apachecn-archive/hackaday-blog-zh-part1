# 自制的热电放大器激光器

> 原文：<https://hackaday.com/2010/09/12/home-made-t-e-a-lasers/>

[Nyle]对在家里制造激光器很感兴趣，但觉得外来零件清单太令人生畏。直到他发现了 T.E.A .激光器。用少量的铝和一些高电压就可以制造出热电子激光器。它们发出紫外光，你可以在他的例子中看到，他把它们射过一个装满水的罐子，里面混有荧光笔墨水。

他发布了几个不同大小的变体，以及它们在行动中的大量图像。休息之后你可以看到一段视频。我们还必须指出视频中美妙的音乐。这让我们想起了 50 年代遗留下来的学校录像。
