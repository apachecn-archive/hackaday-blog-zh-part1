# Dish-o-Tron 6000 回来了，比以往任何时候都好

> 原文:[https://hackaday . com/2011/08/24/the-dish-o-tron-6000-is-back-and-better-from ever/](https://hackaday.com/2011/08/24/the-dish-o-tron-6000-is-back-and-better-than-ever/)

![dish_o_tron_6000_extreme](../Images/06873df01679c98d955840a826883d26.png "dish_o_tron_6000_extreme")

在工作台上设计电路时，有时事情会比现实生活中做得好得多。[奎因·邓基]在过去的几个月里吸取了这一教训[，她在与她最近的创作之一](http://quinndunki.com/blondihacks/?p=388)Dish-o-Tron 6000 斗争。[我们在 4 月份](http://hackaday.com/2011/04/12/dishwasher-notifier-for-the-absent-minded/)推出了 Dish-o-Tron，当时事情似乎对【奎因】很有利。然而，随着时间的推移，她发现这种设备并不可靠。除了每隔几周吃掉一块电池，它还能自发地在“脏”和“干净”之间转换状态。是时候把 Dish-o-Tron 带回工作台进行一些调试了。

从“脏”到“干净”的随机状态翻转相对容易解决，需要在 set 引脚和地之间连接一个小电容，以消除引起故障的电噪声。她将自发的“干净”到“脏”翻转固定在一个卡住的倾斜开关上，她用一个基于水银的模型替换了它，使事情更加可靠。她通过给墙上的 12v 电源接线解决了电池问题，这可能不会更节能，但它确实让她不用一直更换电池。

看到项目如何随着时间的推移而发展，以及不可避免的错误是如何从最初的设计中解决的，这总是很好的。
