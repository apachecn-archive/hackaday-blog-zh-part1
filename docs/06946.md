# 手表中的谷歌双因素认证

> 原文:[https://hack aday . com/2011/02/27/Google-手表中的双因素认证/](https://hackaday.com/2011/02/27/google-two-factor-authentication-in-a-wristwatch/)

![chronos_two_factor_authentication](../Images/6aa3b796cf0ae3d5d835a155ed00d955.png "chronos_two_factor_authentication")

德州仪器(Texas Instruments)的 Chronos 手表是一个方便的小硬件，如果放在合适的人手里。如果你不熟悉这个平台，它被推销为“运动手表中的可穿戴无线开发系统”。简单地说，它是一个可穿戴的无线 MCU，配有 96 段 LCD，集成了压力传感器和 3 轴加速度计。它能够运行定制固件，这使得它可以做任何你想做的事情。

[Huan Trong]希望利用谷歌新的双因素认证，并且[认为他的 Chronos 将是一个很好的表链](http://tnhh.net/pancake/chronos-otp.xml)，因为他可能会在大多数时间里戴着这款手表。他组装了一些定制固件，使手表能够充当身份认证钥匙，为用户提供有效的谷歌密码。

他警告说，该软件充其量是阿尔法代码，说明它甚至不允许手表保持时间。我们非常期待在不久的将来看到更多的代码，继续努力吧！
