# Kinect For Windows 资源

> 原文:[https://hack aday . com/2012/03/22/Kinect-for-windows-resources/](https://hackaday.com/2012/03/22/kinect-for-windows-resources/)

![](../Images/96218bc70b1f659074600b64d6668674.png "kinect")

尽管已经推出了近两个月，但世界还没有看到一个像样的 Windows 版 Kinect 指南。虽然 Xbox 和 Windows 版本的 Kinect 使用基本相同的硬件，但仍有细微但重要的差异。多亏了[Matthew Leone]和他令人敬畏的[开发者资源摘要](http://main.alcestech.com/2012/03/kinect-for-windows-developer-resources.html)，让你的 Kinect 项目启动和运行现在容易多了。

从[微软 Kinect for Windows 网站](http://www.microsoft.com/en-us/kinectforwindows/develop/overview.aspx)获得 SDK 后，你可能想查看一下[微软编程指南](http://msdn.microsoft.com/en-us/library/hh855348.aspx)。Windows Kinect 只能与 Visual Studio 一起使用，但这种不灵活性带来了一些附加功能。两个版本的 Kinect 都有一个麦克风阵列，可以确定声源的方向。[开源驱动](http://openkinect.org/wiki/Protocol_Documentation#NUI_Audio)对音频输入的支持很少，但是官方的微软版本[已经为音频捕获、源本地化和语音识别准备好了所有的 API](http://msdn.microsoft.com/en-us/library/hh855349.aspx)。

Windows 版 Kinect 售价 250 美元，是一笔相当大的投资。一台二手 Xbox Kinect 的售价约为 80 美元，因此我们非常确定黑客群体将会远离 Windows 版本。尽管如此，如果你曾经为 Kinect 开发过一些东西，你可能想要 XBox 版本中没有的友好的 API 和功能。
