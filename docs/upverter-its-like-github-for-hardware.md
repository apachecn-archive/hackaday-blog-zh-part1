# Upverter:这就像硬件的 Github

> 原文：<https://hackaday.com/2011/09/14/upverter-its-like-github-for-hardware/>

![](img/052352b1dc828761ec4c87250af861bc.png "upverter")

如果说我们已经注意到硬件黑客和电子项目的一件事，那就是建立一个项目的所有资源都分散在互联网的论坛、博客和个人网页上。输入 [Upverter](http://upverter.com/) 。创办 Upverter 的人也有同样的观察，在得到一些 [Y Combinator 的资助](http://blog.upverter.com/our-y-combinator-experience)后，他们推出了他们希望成为的“电子产品 Github”和“硬件设计的谷歌文档”

Upverter 拥有我们所期望的特性——分叉、版本控制，以及与 Github 集成项目代码。已经有几个很酷的项目，像一个 [PIC 程序员](http://upverter.com/mkiss/852/PIC-Chip-Programmer/)，一个 [TV-B-Gone](http://upverter.com/mkiss/313/TV-B-Gone/) ，还有一个[康威的《生命游戏》的微型版本](http://upverter.com/mkiss/551/The-Game-Of-Life/)。

我们还没有准备好挥舞一个 Upverter 粉丝的旗帜。原理图编辑器中有相当多的可用组件，但根据我们的经验，组件库可以通过删除重复项和增加部件数量来进行一些改进。我们还想在我们的愿望清单上为示意图添加一个缩放控件。Upverter 也没有 PCB 编辑器，但是从[VC rag](http://venturebeat.com/2011/09/13/upverter/)的这篇文章来看，这个团队正在开发一个。虽然这真的只是吹毛求疵——Upverter 在不到 24 小时前推出——但我们很高兴看到 Upverter 上托管的一些项目在 tip line 上滚动。
