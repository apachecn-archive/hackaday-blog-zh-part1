# 示波器上的文本

> 原文：<https://hackaday.com/2009/07/31/text-on-an-oscilloscope/>

[https://www.youtube.com/embed/zLpfq4tHZxE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zLpfq4tHZxE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这是一个有趣的小玩具。一块可以在示波器上显示文本的小板子。该组件，或者说只是组件，是一个 PIC16F628A。除了电源，就这些了。它可以一次显示 10 个字符，正如你在上面看到的，还可以滚动显示。我们真的不知道它的实际用途，但它会成为一个很好的恶作剧工具。如果你想要一些更复杂的示波器效果，请查看两个的[网球。](http://hackaday.com/2008/07/16/tennis-for-two-resurrected/)
