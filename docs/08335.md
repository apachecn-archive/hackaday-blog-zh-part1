# 自动坦克会跟踪你，在你身上留下伤痕

> 原文:[https://hack aday . com/2011/08/16/autonomous-tank-will-track-you-down-cover-you-in-welts/](https://hackaday.com/2011/08/16/autonomous-tank-will-track-you-down-cover-you-in-welts/)

![autonomous_airsoft_tank](../Images/00d4a460f20e2f4267501a7e353ee102.png "autonomous_airsoft_tank")

[Dan]写来分享他最近完成的一个项目，[一个自动气枪坦克](http://trandi.wordpress.com/2011/05/14/tiger-1-bb-airsoft-rc-tank-%E2%80%93-v3/)。玩具坦克利用各种各样的技术来完成任务，并会不择手段地追捕你(前提是你戴着红外信标)。

Arduino 板用于控制坦克的马达，而 Lego NXT 模块处理大多数其他操作。水箱使用超声波传感器四处走动，确保它不会卡在任何晃动的家具上或挂在角落里。虽然自动驾驶很好，但[丹]通过让气枪炮塔完全自动驾驶增加了一点赌注。

他在坦克上安装了一个 Wiimote 红外传感器，经过反复试验，成功地将其与 NXT 模块连接起来。现在一切都准备好了，他可以把他的红外信标放在房间的任何地方，坦克将会四处扫描周围直到发现目标。一旦坦克锁定，一连串的气枪子弹就会干掉阻挡它的任何东西。

我们认为[丹]在这方面做得非常出色，但你可以在休息后观看嵌入的视频。
