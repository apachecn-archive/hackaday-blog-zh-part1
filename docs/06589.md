# 改进六足机器人的设计

> 原文：<https://hackaday.com/2011/01/15/improving-a-hexapod-design/>

[JC]根据他在互联网上找到的一个项目为自己建造了一个六足机器人。它工作得相当好，但机械性能差，容易破损。他开始改进设计，并设计出了上面看到的装置。它使用三个伺服电机来控制六条腿，在休息后的快速剪辑中可以看到它走得很好。它不像[我们昨天看到的小杂技六腿机器人](http://hackaday.com/2011/01/14/arduspider-entertains-children-and-exercises-pets/)那么灵活，但是动作非常令人愉快，它能够向前、向后移动和转弯。[JC 的]帖子总共有四页，所以不要忘记找到他关于[构造](http://jcopro.net/2010/12/08/pegleg-the-six-legged-hexapod-robot-part-2-construction/)、[联动](http://jcopro.net/2010/12/10/pegleg-the-six-legged-hexapod-robot-part-3-kinematics-or-how-it-walks/)和[伺服控制](http://jcopro.net/2010/12/12/pegleg-the-six-legged-hexapod-robot-part-4-programming-and-servo-control/)页面的链接，以找到概念图、cad 设计和他对该过程的想法。
