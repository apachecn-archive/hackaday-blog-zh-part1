# USB 控制的 spka 夜灯

> 原文:[https://hack aday . com/2012/03/14/USB-controlled-spoka-night-light/](https://hackaday.com/2012/03/14/usb-controlled-spoka-night-light/)

![usb-controlled-spoka-nightlight](../Images/b59091d3fd801f1106a68b4bbc95d545.png "usb-controlled-spoka-nightlight")

[wejp]挑选了一款宜家 spka 夜灯，但他对其功能并不完全满意。按下幽灵的头顶会使它在几种颜色之间循环，第二次按下它会锁定显示当前颜色，直到它再次被点击。这个 spka hack 使用了另一种版本的夜灯，受到[的启发，他把它拆了下来，看看能做些什么。](http://hackaday.com/2012/01/19/controlling-a-cute-ikea-night-light-with-android-on-the-cheap/)

在剥去外壳后，他发现内部结构与发光的同类有很大不同，尽管它们完全符合[wejp]的想法。他移除了可充电电池组和控制板，控制板位于与 led 分开的 PCB 上。他用一个 ATtiny25 代替了普通的微型照相机，他用它来让自己对灯光显示有更多的控制。

他不能把所有他想要的功能都塞进这个小东西里，但是他计划用他的电脑给灯供电，所以他在后面安装了一个小的 USB 接口。当连接到他的 PC 时，spka 可以比单独运行时更精确地控制。

不幸的是，没有 spka light 运行的视频，但在他的网站上有很多图片。
