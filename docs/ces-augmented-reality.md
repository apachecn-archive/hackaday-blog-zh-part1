# CES:增强现实

我得到了一副增强现实眼镜，它和一个猴子球风格的游戏一起展示。就增强现实而言，这并不是什么太新的东西，但是游戏中使用的眼镜在镜片外侧的中心装有立体相机，这使得你不必盯着屏幕就可以获得真正的 3D 增强视觉。这些眼镜仍处于演示阶段，但如果它们能够进入零售市场，我认为它们将会是 AR 从有趣的玩具到有用工具的桥梁。休息后的另一个回顾视频。
