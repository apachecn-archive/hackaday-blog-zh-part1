# 万圣节黑客:死神开派对

> 原文:[https://hackaday . com/2011/10/25/Halloween-hacks-death-throws-a-party/](https://hackaday.com/2011/10/25/halloween-hacks-death-throws-a-party/)

![](../Images/8a4123db22326d47a0737a46a4e9af08.png "Halloween")

![](../Images/03fe1f6ec3b0b6d7941071934d1d38da.png "party")

如果你叫他死神，阿兹铁路，马吉尔·海因，或者简单地叫他“死神”，他肯定会开一个不错的派对。[Victor]在他的工作岗位上有一个名为[Hein]的骨骼解剖模型，为了庆祝他的生日，他的同事决定为[Hein]举办一个派对。

[Victor]的第一个基于骨骼的构建是在去年夏天。夏天的阳光在[海因]的眼眶里闪烁，这是给骷髅穿上夏装的绝佳时机。[海因]穿着一件时髦的夏威夷衬衫，被放在走廊的一个橱窗里。当然，如果没有惊喜的成分，骷髅是没有用的，所以[海因]拿着一次性相机，每当有人走过时就闪一下。

[海因]的生日庆典需要一些更令人震惊的东西。在[海因]的下颌骨上加了一个[的鸣笛声](http://www.biztrademarket.com/User/159325/bb/pa08005a-50k_party_horn_blower_aw7.jpg)。有了 PIR 传感器和 TI 发射台，每当有人走过时，[海因]就会被命令吹哨子。

[Victor]的造型可能不具备震撼力——甚至不像万圣节那么恐怖——但他有一些好主意，可以成为一个很棒的庭院展示。休息后查看视频，了解[Hein]与[Victor]的同事打招呼的情况。
