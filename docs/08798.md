# 宣布我们下一个主题:万圣节黑客

> 原文：<https://hackaday.com/2011/10/11/announcing-our-next-theme-halloween-hacks/>

![](img/d5f6eff35ab5c90964eee2222f13e735.png "Halloween")

似乎每年，当谈到万圣节黑客时，Hack A Day 都有点落后于时代。像[马里奥服装](http://hackaday.com/2010/11/02/halloween-prop-mario-bros-with-full-sound-effects/)和[豪斯医生演唱*惊悚*](http://hackaday.com/2010/11/01/halloween-house-has-been-known-to-burst-into-song/) 这样的版本很棒，但是我们在万圣节之前发布它们更有意义。

为了介绍我们的万圣节主题，我们要展示[重量级重量级]的万圣节项目。这是一个小型灯光自动化构建，它将一串灯光同步到~~最佳~~ ~~万圣节~~ ~~圣诞节~~第二最佳【蒂姆伯顿】电影*圣诞节前的噩梦*的主题。[重量级豪]使用 [Vixen](http://www.vixenlights.com/) 对灯光进行排序，并使用 Arduino 解释来自 Vixen 的串行命令。这是一个很好的建筑，坐在门廊上，旁边放着一大碗糖果，看起来会很棒。

如果你有想要炫耀的万圣节装扮，比如鬼屋之旅、令人敬畏的服装，甚至是非常棒的南瓜灯，请通过[提示热线](http://hackaday.com/contact-hack-a-day/)发送。我们计划每天至少发布一个万圣节帖子，所以请继续发送这些帖子。
