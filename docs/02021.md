# 借助移动云

> 原文：<https://hackaday.com/2008/06/22/mit-mobile-cloud/>

麻省理工学院移动体验实验室刚刚开发了这个雄心勃勃的[互动装置，叫做云](http://www.thecloud.ws/overview.html)。位于意大利费伦泽的“云”是一种雕塑，由超过 15000 个发光二极管和几英里长的光纤组成。纤维的尖端会发光，但它们也会随着人类的互动而改变颜色，包括触摸它或站在它附近。云使用接近和触摸传感器的组合来实现这一点。它还有两个摄像头和一个麦克风，这使它能够从各种来源获得输入，并输出更丰富、更有机的响应。
