# 如何不造一台机器人割草机

> 原文：<https://hackaday.com/2011/08/25/how-not-to-build-a-robotic-lawnmower/>

[shadeydave]想要建造他自己的 Lawnbot，但是他不知道从哪里开始。他在网上购买了一些 DIY 计划，这些计划看起来似乎可以完成任务，但后来他大大偏离了方向，在这个过程中花了很多钱。

在他的指导性文章中，他详细描述了他犯下的每一个错误，解释了为什么他的选择是错误的，以及每个错误给他带来的损失。听起来好像几乎所有可能出错的事情都出错了，从花钱买不必要的微控制器到选择错误的轮子。我们最喜欢的部分是他提到他不知道如何在他的发射器与快速旋转的死亡机器失去联系的情况下为 Lawnbot 创建一个“死亡开关”。

[shadeydave]很清楚他的构建有多糟糕，主要是为了给其他可能决定承担类似项目的人提供警示。他说劳恩波特在大部分情况下都能工作，但是他会用他新发现的智慧，从他的错误中吸取教训，修改这个机器人。

事实上，我们喜欢看到这类文章，因为它们对试图组织类似项目的人非常有益。因此，如果你有一些重大失误，不要羞于把它们挖出来，让我们知道。正如托马斯·爱迪生所说，“我没有失败。我刚刚发现了一万种行不通的方法。”
