# 模块化合成器安装在机架上，可重新配置

> 原文:[https://hack aday . com/2011/05/31/模块化合成器是机架安装的并且可重新配置的/](https://hackaday.com/2011/05/31/modular-synthesizer-is-rack-mounted-and-reconfigurable/)

![](../Images/7aeabb1ebbf9d87a62719ae33195e2d3.png "modular-patchable-synthesizer")

[德克]让我们了解一下这个奇妙的[音乐合成实验装置](http://www.musiksynthesizer.de/projekte.htm) ( [翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http%3A%2F%2Fwww.musiksynthesizer.de%2Fprojekte.htm))。关掉你的电脑扬声器(以避免当每一页加载时自动播放音乐),在这个知识库中挖掘丰富的信息。实际上，几十个模块已经建成，并安装在机架系统上。使用以香蕉插头为终端的跳线，每个模块都可以与其他模块连接成数量惊人的不同设置。

[模块外壳](http://translate.googleusercontent.com/translate_c?hl=en&ie=UTF8&prev=_t&rurl=translate.google.com&sl=auto&tl=en&twu=1&u=http://www.musiksynthesizer.de/article/cases/index.htm&usg=ALkJrhghdv37tsHHlQj_MI6q2SDQIz7RhQ)本身适合安装在标准的 19″机架中。前挡板由 CAD 设计，外壳的其余部分主要由铝制成。由于每个模块往往非常小，因此几个模块被组合成一个机架框架以节省空间。您可以在上面的图片中看到，每个机架片有多达八个模块。

你会喜欢阅读这里使用的许多不同的声音芯片。但事情不一定要就此结束。如果这激起了你对自己机架安装系统的兴趣，那么你很幸运。[下载区](http://translate.googleusercontent.com/translate_c?hl=en&ie=UTF8&prev=_t&rurl=translate.google.com&sl=auto&tl=en&twu=1&u=http://www.musiksynthesizer.de/download.htm&usg=ALkJrhgns1pzpVskOumHUKhjYH0LSMoBqg)有原理图、电路板插图和大多数模块的构建信息。
