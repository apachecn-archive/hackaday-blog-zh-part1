# 这个 GLaDOS 土豆是个谎言

> 原文:[https://hack aday . com/2011/08/21/this-glados-potato-is-a-lie/](https://hackaday.com/2011/08/21/this-glados-potato-is-a-lie/)

当你可以将数字世界变成现实时，为什么要满足于虚拟现实呢？当*不*玩传送门 2 的时候【乔希】想让[一个 GLaDOS 土豆陪伴他度过人生](http://www.robotdialogs.com/2011/08/potato-glados-from-portal-2.html)。他开始工作，看看他能想出什么样的道具复制品。从上面的图片和休息后的演示视频来看，他把它钉在了马铃薯上。

不用担心腐烂。土豆和一些部分由[油渣](http://www.sculpey.com/)成型，然后在烤箱中烘烤。因为假土豆中间是空的，所以很容易隐藏让它说话的部分。一个旧的 MP3 播放器加载了游戏中的引用，并通过一个 LM386 音频放大器电路和一个隐藏在电子眼下的扬声器播放它们。眼睛由五个黄色发光二极管照亮，这些发光二极管也连接到放大器上，使它们随着音频信号的强度而闪烁和衰减。

油漆工作和钉子和电线真的让建筑看起来刚刚好。现在[乔希]需要举办一个以极客为主题的万圣节派对，这样他就可以好好炫耀一下了。
