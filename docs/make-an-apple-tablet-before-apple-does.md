# 抢在苹果之前生产苹果平板电脑

> 原文：<https://hackaday.com/2009/12/18/make-an-apple-tablet-before-apple-does/>

![](img/a9ce5e60d7e85d7e7a831ef86f02c05e.png "apple-tablet")

[Andrew]想要一台平板电脑，但厌倦了等待苹果公司推出平板电脑。为了拯救易贝和克雷格列表，他拿起一些零件，并建立了自己的平板电脑。你可以在休息后看看它的视频导览，或者细读他工作日志的部分[一](http://underdesign.wordpress.com/2009/12/09/ebook-1/)、[二](http://underdesign.wordpress.com/2009/12/11/ebook-2/)、[三](http://underdesign.wordpress.com/2009/12/15/ebook-3/)、[四](http://underdesign.wordpress.com/2009/12/18/ebook-4/)。

该单元由 500 MHz iBook 组装而成。[Andrew]从 ebay 上买了一个触摸屏，并把一个 USB 操纵杆变成了卡通，以便在无键盘设计中添加一些按钮。最终产品相当不错。很久以来，我们一直想要一台平板电脑挂在墙上，这可能是让我们实现梦想的灵感。

你觉得这不能满足你对苹果平板电脑的需求吗？在[我们的 mac 平板电脑综述](http://hackaday.com/2008/07/14/mac-tablets-made-by-fans/)中还可以看到更多。
