# 易于移动的 RFID 门锁非常适合宿舍

> 原文：<https://hackaday.com/2011/09/19/easily-movable-rfid-door-lock-is-great-for-dorm-rooms/>

[Eric]上大学后最先组装的东西之一是一个 RFID 门锁系统。他发现自己经常匆匆忙忙地进出宿舍，而使用一把简单的无线钥匙是一种简化生活的好方法。

这些年来，他已经完善了他的设计，虽然他最初的原型是功能性的，但边缘有点粗糙。在他网站上发布的视频中，他详细解释了他的系统是如何构建的，并展示了他随着时间的推移所做的修改。构建该系统时需要考虑的一个关键因素是，安装必须是非永久性的。由于学校通常不赞成对你的房间进行物理改造，他找到了一种非侵入性的方式来安装他的系统，如拉链和泡沫板。

他的 RFID 门锁看起来工作得很好，我们特别喜欢包含簧片开关，以确保系统知道门是否被打开。如果你有大约半个小时的空闲时间，并且对构建自己的 RFID 入口系统感兴趣，请务必查看下面的[Eric 的]视频了解所有细节。
