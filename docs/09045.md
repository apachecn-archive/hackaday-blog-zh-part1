# 扩音器模式让你在[肯尼·基]美妙的音调中驱散愤怒的人群

> 原文：<https://hackaday.com/2011/11/10/bullhorn-mod-lets-you-disperse-angry-crowds-to-the-dulcet-tones-of-kenny-g/>

![megaphone-mod-aux-audio-in](img/2ea61b84d62930aa6a326c12a3ceb8ef.png "megaphone-mod-aux-audio-in")

[Randy]有一个便宜的扩音器，像这个价格范围内的大多数型号一样，它没有音频输入插孔。他希望能够通过扩音器从外部麦克风传输音乐和音频，在一个简短的教程中，他展示了如何修改他的扩音器来做到这一点。

大多数扩音器都将电子设备放在内置麦克风附近，就在电池盒外面。从[Randy 的]模型的盖子上取下螺丝，露出了一块电路板和几根连接到板上麦克风的电线。他的扩音器碰巧播放了一些来自次级电路板的录音音乐，他在音频插孔和主电路板之间插入了一个内置的切断开关。他还在插孔和麦克风电路之间添加了一个电阻，以适当衰减线路电平信号。一旦他重新组装好所有东西，然后使用便携式音频播放器测试他的输入，慢慢地将音量提高到所需的水平。

并不是每天你都需要通过扩音器播放音乐的能力，但是当你需要的时候,[Randy 的]修改是一个很好的补充。
