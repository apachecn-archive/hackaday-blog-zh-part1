# 简单流体发动机

> 原文：<https://hackaday.com/2011/03/09/simple-fluidyne-engine/>

[Mirslav] [自己造了这个 fuidyne 发动机](http://www.arthropodsystems.com/Fluidyne/Fluidyne1.html)。这是一个单活塞模型，但你不会在这里找到任何精密铣削气缸。这是因为 [fluidyne 发动机](http://en.wikipedia.org/wiki/Fluidyne_engine)使用水柱作为活塞。在上面显示的装置中，你可以看到一根金属管作为回路的冷端。绝缘层下还有另一根热管，使电路完整。当那根管子被加热时，它导致回路内的空气膨胀，迫使塑料管开口侧(左边)的液体上升。一旦空气逃逸到回路的另一侧，开口管中的水活塞再次回落。这导致连续的振荡，该振荡可用于使用一对止回阀驱动泵。
