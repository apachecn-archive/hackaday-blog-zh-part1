# 电钻驱动的推车

> 原文:[https://hackaday.com/2010/01/28/drill-powered-go-cart/](https://hackaday.com/2010/01/28/drill-powered-go-cart/)

![](../Images/a605e080a5e9937dea286b179826fe2f.png "drill-powered-go-kart")

这里我们有一辆[玩具车，改装后使用钻头作为动力系统](http://www.youtube.com/user/HansTillema)。[汉斯]扔了一个 12v 的 4.5 安培的电池给马达提供动力，它的最高时速约为 9 英里。这在概念上类似于上周的[垃圾推车](http://hackaday.com/2010/01/22/electric-go-cart-from-trash/)，但这次有视频。他制作了几个这样的东西，休息后会有两个的片段。我们最喜欢的部分来自第一个视频，当儿子在听起来很刻薄的机器上在附近兜风时，爸爸在摄像机后面开心地笑着。
