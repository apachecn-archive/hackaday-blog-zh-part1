# 我的书桌是我的电脑机箱

> 原文：<https://hackaday.com/2011/05/26/my-desk-is-my-computer-case/>

如果你认为你的水冷钻机很可爱，看看荷兰个人电脑爱好者【彼得·布兰德】 ( [谷歌翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=nl&tl=en&u=http%3A%2F%2Fgathering.tweakers.net%2Fforum%2Flist_messages%2F1420280))的[这个创造。](http://gathering.tweakers.net/forum/list_messages/1420280)

随着他的电脑尽他所能的调整，他决定稍微装饰一下他的办公室。在这个过程中，他最终只是稍微调整了一下他的电脑。在看到另一位电脑爱好者组装的电脑后，他开始制作一张桌子，在上面展示他的电脑。他花了一些时间用谷歌 Sketchup 绘制计划，并在一位友好邻居的帮助下，开始建造他的办公桌/电脑箱。

这张桌子由 3 毫米厚的铝制成，在一块厚厚的玻璃下面放着他电脑的大部分部件。电脑中唯一没有放在桌子里的部分是他用于水冷系统的 9 风扇散热器。这一部分存在于他的爬行空间，他通过一对穿过瓷砖地板的大水管连接到他的电脑。如果你有兴趣，你可以在这里看到[所有 800+的建造图片。](http://tweakers.net/gallery/45974/fotoalbum/?MapID=17839)
