# 魔法吊坠盒

> 原文:[https://hackaday.com/2011/07/06/magic-locket/](https://hackaday.com/2011/07/06/magic-locket/)

[![](../Images/4c0fc4fefb28d1a1f0f69bad61d79ea7.png "magic_pendant")T2】](http://hackaday.com/2011/07/06/magic-locket/magic_pendant/)

RTFM 的[Andrey]只用三个部分和一些简单的代码为自己制作了一个[发光 LED 挂件](http://www.rtfms.com/rtfms-7-microcontroller-meets-jewelery.htm "main link")。黑客并不特别复杂，但[Andrey]通过 RS232 串口和 RGB LED 控制提供了一些关于鹤嘴锄编程的体面指令，以产生漂亮的发光效果。该挂件包含一个 RGB LED，一个镐-08 微控制器和几个纽扣电池。为了把所有的东西都塞进盒子里，[Andrey]不得不用锉刀把 LED 灯和鹤嘴锄-08 打磨到最小尺寸。

项目页面上提供了挂件的所有基本代码，[Andrey]描述了他如何设法对所有三个 LED 引脚进行 PWM 以实现色彩效果。休息之后的视频可能会让那些以前没有尝试过 Picaxe 编程的人感兴趣，或者让那些想尝试一些新的嵌入式设备而又不想花太多钱的初学者感兴趣。
