# IPhone 控制的电视升降机

> 原文：<https://hackaday.com/2009/05/02/iphone-controlled-tv-lift/>

![pcb](img/1f80fa8ce2d328882005d5461fa0440b.png "pcb")

[William]提交了这个项目，他用他的[iPhone 控制他的电视升降](http://www.alternet.us.com/?p=364)。他用 pic18f122 微处理器制作了一个定制的 PCB 来与电梯通信。他说，它通过一个串行端口连接到电梯控制器。休息之后你可以看到一段视频。我们真的觉得影子变成箭头很酷。那是一次幸运的事故。
