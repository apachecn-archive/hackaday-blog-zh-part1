# 为…连体双胞胎造的自行车？

> 原文：<https://hackaday.com/2012/01/07/a-bicycle-built-for-siamese-twins/>

![](img/45d2e4ad7782fc1141f0bd1955cc0202.png "side-by-side-bicycle")

这个项目真正结束了关于谁必须坐在双人自行车后座的争论。在观看[卡洛斯]和他的女儿骑着这辆肩并肩的自行车的首航时，我们要求你不要笑。视频可以在休息后找到。

这当然比上下串联更有意义，而且制造过程非常容易控制。这需要改变座位，把手和踏板，但自行车的大部分(框架，齿轮，叉，车轮)是不变的。

曲柄已经被一个定制的焊接凸轮机构取代，提醒我们桨船上的踏板是如何工作的。两名骑手必须同时踩踏板并加速。为了给每个人一个坐的地方，座位杆被转换成一个 T 形杆，以容纳框架左右两侧的鞍座。最后，把手是最复杂的。额外的框架被焊接到前管的两侧，以提供安装两对把手的地方。其中一个固定在适当的位置，另一个可以转动，使用杠杆机构来控制前叉。

开始的时候看起来有点尴尬，但是一旦两个骑手都起来了，看起来就很稳定了。
