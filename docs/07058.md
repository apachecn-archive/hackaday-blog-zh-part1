# 项目完成后，黑客空间相机装备会自动提醒互联网

> 原文:[https://hack aday . com/2011/03/10/hacker space-camera-rig-auto-alerts-the-internet-when-projects-are-finished/](https://hackaday.com/2011/03/10/hackerspace-camera-rig-auto-alerts-the-internets-when-projects-are-finished/)

![alerttheinternets_cam](../Images/f674da115cd02ce8c1004b99cc1dd6a7.png "alerttheinternets_cam")

似乎每个黑客空间都有自己的方式向成员传达状态信息。维也纳[MetaLab]的黑客们发明了一种[相当新颖的方式](http://metalab.at/wiki/Alert_The_Internets) ( [谷歌翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=de&tl=en&u=http%3A%2F%2Fmetalab.at%2Fwiki%2FAlert_The_Internets))让世界知道他们已经完成了一个项目。虽然一些黑客空间只是通知他们的成员他们已经在[开始营业](http://hackaday.com/2011/03/08/twittering-wall-switch-lets-dutch-hackers-know-when-its-time-to-play/)，但是这个黑客更进一步。

当项目被视为完成时，摄像机从 dock 中移除，可以录制任意数量的视频。当摄像机被放回 dock 时，一个录制的介绍视频会被添加到录像中，然后所有内容都会自动上传到 YouTube。不需要额外的时间，也不需要进行视频编辑——他们的作品一旦完成拍摄，就会立即上传到互联网上。

这是一个伟大的想法，也是每个黑客空间都应该有的东西。如果能看到这些东西安装在公共场所，以便在事件发生时立即报告，那就更好了。

如果你愿意，一定要看看[MetaLab]的[YouTube 频道](http://www.youtube.com/user/AlertTheInternets)。
