# 为申请大学玩电子游戏

> 原文：<https://hackaday.com/2012/02/09/playing-video-games-for-a-college-application/>

![](img/789bfc81e9eb0c5a9b9641259cdc0d27.png "game")

作为一名高三学生，几个月来[欧文]一直在等待他申请的大学的消息。他的一些申请想要一份年中报告，看看他是否患有老年痴呆症。[欧文]意识到这些大学允许高中成绩单之外的额外材料，所以他[制作了一个小型视频游戏](http://www.hackniac.com/blog/?p=927)来展示他的电子和编程技能。

[欧文]称他的体型为 Demomite，是一件令人惊叹的作品。整个系统是基于 ATtiny2313，程序内存只有区区 2kB。除了 Sparkfun 的一个图形 [LCD 和一个重新设计的 NES 控制器之外，没有太多其他东西可以构建。作为一个对极简主义和简约主义的研究，[欧文]从我们这里得到了很大的祝贺。](http://www.sparkfun.com/products/10168)

整个游戏适合在 ATtiny 上的 2kB 闪存中，主要是因为在汇编中编码整个东西[。你可以看看[欧文]的延时建设视频，软件演示，以及休息后他发给大学的视频。](http://www.hackniac.com/blog/?p=974)
