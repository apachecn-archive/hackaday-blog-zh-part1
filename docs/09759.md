# 通过让 Arduino 按下您的按钮来赢得视频游戏

> 原文:[https://hack aday . com/2012/02/24/winning-video-games-by-let-arduino-push-your-buttons/](https://hackaday.com/2012/02/24/winning-video-games-by-letting-arduino-push-your-buttons/)

![](../Images/e75e0dd031b22556ac4ad095102fa2d0.png "arduino-pushes-your-buttons")

[Lars]向您展示如何在 BIT 的前四个级别中获得满分。使用 Arduino 通过[按下](http://arduino.cc/playground/Main/Keyboardgame)计时并发送按钮，使跑步者跳闸。这是一个非常简单的游戏，使用几个按钮跳过或滑过障碍。角色的恒定速度使得在没有任何游戏输入的情况下为这些动作计时变得非常容易。这意味着一些网页游戏机器人使用的[像素采样](http://hackaday.com/2012/02/07/web-game-bot-coded-with-python/)在这里并不真正必要。算出时间，硬编码到草图里就行了。休息之后你会看到，它运行得非常完美

这个黑客的真正价值是他写的从 Arduino 硬件发送按键的指南。这一点也不难，但有几个步骤，这将让你马上开始运行。从这里你会去哪里？这很容易变成一个键盘恶作剧，拼错你所有的单词。
