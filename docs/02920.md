# 特斯拉线圈吉他放大器

> 原文：<https://hackaday.com/2008/12/07/tesla-coil-guitar-amp/>

[https://www.youtube.com/embed/JnmZZNWzZPc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/JnmZZNWzZPc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

你可能已经看过[音乐特斯拉线圈](http://hackaday.com/2008/06/12/singing-tesla-coils/)。人们将它们连接到电脑和键盘上，使它们成为危险而可怕的扬声器。这个视频通过一个弹出了某人[弹吉他的画面。这是粗糙的，不太擅长](http://gizmodo.com/5103456/electric-guitar-played-through-a-tesla-coil-for-added-metal)[复调](http://en.wikipedia.org/wiki/Polyphony)，但仍然很酷。我们希望看到几个线圈，由不同的过滤器分开，以便能够听到更多的吉他和弦。要是有人在制作《闪光戈丹》时为《女王》做了这些就好了。
