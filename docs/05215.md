# 卵孵化室

> 原文:[https://hackaday.com/2010/04/21/egg-incubation-chamber/](https://hackaday.com/2010/04/21/egg-incubation-chamber/)

![](../Images/79811e48e906f6e1d5aea09306c0be6d.png "egg-incubator")

[Mazomen]把昂贵的现成选项留给了其他人，建造了自己的[蛋孵化室](http://translate.google.com/translate?js=y&prev=_t&hl=en&ie=UTF-8&layout=1&eotf=1&u=http%3A%2F%2Fwww.elektroda.pl%2Frtvforum%2Fviewtopic.php%3Ft%3D1622770&sl=pl&tl=en)。它把鸡蛋放在泡沫聚苯乙烯内衬的盒子里保持温暖和快乐。温度调节由 ATtiny26 微控制器和 DS18B20 温度传感器共同处理。当温度下降时，鸡蛋上方的室内两个 60 瓦的灯泡打开，空气通过一个小风扇循环。如果你已经转向[自动化蔬菜种植](http://hackaday.com/2009/09/15/grow-box-controls-heater-fans-and-water/)，这个项目也会让你的养鸡变得更容易。
