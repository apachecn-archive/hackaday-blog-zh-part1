# 基于浏览器的电路模拟器拥有大量的功能

> 原文:[https://hack aday . com/2012/03/02/browser-based-circuit-simulator-has-a-mountain-of-features/](https://hackaday.com/2012/03/02/browser-based-circuit-simulator-boasts-a-mountain-of-features/)

![](../Images/2e3e713c80933e85d40ce5fa7357e351.png "circuitlab-browser-based-simulator")

CircuitLab 是一个可以在浏览器中运行的电子模拟器。上面您可以看到一个示例电路，它有助于展示应用的强大功能。你可以建立你的原理图(也许你想试试【Jeri 的】 [psu 关闭定时器？](http://hackaday.com/2012/03/01/auto-power-circuit-for-an-arcade-machine/))在编辑器模式下，然后切换到模拟器，从组件获取数据。在这种模式下，光标变成一个探针，单击电路的不同部分将返回该元件的计算输入和输出电压。但是等等，还有更多。除电压模拟器外，它还具有时间和频率模拟功能。这可以让你看到通过模拟滤波器输入的波形，或者上面 555 定时器电路中的定时数据。

这个神奇的工具来自哪里？NerdKits 的幕后人员[Humberto Evans]和[Mike Robbins]开发了这个网站。我们在这里见过很多他们的黑客，比如[铣削螺线管，让他们弹奏木琴](http://hackaday.com/2010/12/02/make-your-own-solenoids-then-play-the-xylophone/)。休息之后，看看他们制作的 CircuitLab 快速入门视频。T3[https://www.youtube.com/embed/f52GV1IpwVk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/f52GV1IpwVk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T5】
