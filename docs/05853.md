# EL Wire:制作、连接、供电

> 原文:[https://hack aday . com/2010/08/25/El-wire-make-it-connect-it-power-it/](https://hackaday.com/2010/08/25/el-wire-make-it-connect-it-power-it/)

![](../Images/2904c9723f3099af7f7f0cabedf7309e.png "pcb-el-panel")

[Jeri's]带来了一系列视频，概述了[一步一步的电致发光电线制造](http://www.youtube.com/watch?v=cV2HS7_Mg6o)，用 PCB 制造 [EL 面板](http://www.youtube.com/watch?v=h4O3BGjxd5I)，以及[为 EL 硬件组装电源](http://www.youtube.com/watch?v=AcXHx5ikjM4)。这些概念实际上很容易理解，这是我们从那些在家里自己制造集成电路的人那里得不到的。

这里的概念是通过磷光体的交流电将激发它们并产生光。你需要两个被电介质隔开的导体来完成这项工作。对于电线，[Jeri]使用一股漆包线和一股裸线。搪瓷使它们绝缘，防止短路。

但这还不是全部，她还使用电路板作为 EL 面板进行测试。通过重新利用接地层作为导体之一，并使用阻焊膜作为电介质，她能够在磷光产品上绘画，从而产生发光面板。

最后，你必须让电路通电，这就是她的电源视频进入画面的地方。休息之后我们已经植入了所有三个。这可能比闪烁的 led 灯更酷，而且开始使用也相当便宜。电路是宽容的，只要你不被交流电电击到。
