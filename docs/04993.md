# LED 和风扇控制器

> 原文:[https://hackaday.com/2010/03/03/led-and-fan-controller/](https://hackaday.com/2010/03/03/led-and-fan-controller/)

![](../Images/2c1989b2f994d78aeedc259d4d523bbf.png "led-and-fan-controller")

需要一个前置风扇来保持硬盘冷却，[CalcProgrammer1]发现他对风扇的单一 LED 颜色不满意。他把它们换成了一组四个 RGB 发光二极管，并为该单元组装了自己的控制板。它以 ATmega168 为基础，并在主板上的 COM2 接口中插入补丁，提供串行接口。[CalcProgrammer1]编写了一个 GUI 来控制风扇速度和单个 LED 颜色设置。休息之后，你可以看看令人着迷的、坐在座位边上的滑块控件如何工作的演示。如果[硬盘 LED 时钟](http://hackaday.com/2009/09/07/hard-drive-rgb-clock/)可以改装成使用风扇，这样前面板上就有一个彩色的模拟表盘，那不是很棒吗？ [https://www.youtube.com/embed/4J4vlAMgQDQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/4J4vlAMgQDQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
