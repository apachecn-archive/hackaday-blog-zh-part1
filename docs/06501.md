# 唱歌:用等离子！

> 原文:[https://hackaday.com/2010/12/28/singing-with-plasma/](https://hackaday.com/2010/12/28/singing-with-plasma/)

![](../Images/b22c749213ab2ef2bc064fa999bbafbe.png "Plasma Speaker")

虽然有很多方法可以产生可听见的声音，但等离子扬声器肯定是最酷的一种。通常非常复杂和昂贵，我们收到了初学者指南的提示，制作这些令人印象深刻的新奇事物之一。方便地包括一套示意图(一个非常简单的示意图，另一个带有一些艺术插图)。还有一组警告，包括提醒注意 MOSFETs 产生的热量，以及确保输入信号不要太强。成品听起来很有趣，所以休息后一定要看看示例视频。
