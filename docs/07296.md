# Kinect Hack 让愚人节恶作剧成为现实

> 原文:[https://hack aday . com/2011/04/04/Kinect-hack-makes-April-fools-prank-a-reality/](https://hackaday.com/2011/04/04/kinect-hack-makes-april-fools-prank-a-reality/)

![gmail_motion](../Images/b21096f13288233273fbac09ebe2c78e.png "gmail_motion")

除非你过去一周左右一直躲在山洞里，否则你一定听说过谷歌今年的愚人节玩笑。Gmail Motion 据称是 Gmail 的一个动作驱动界面，包括完成日常电子邮件任务的滑稽姿势和手势。不幸的是，这是一个精心设计的玩笑，没有基于手势的 Gmail 界面即将推出…至少谷歌没有。

南加州大学创新技术研究所的团队已经加快步伐，让谷歌的恶作剧成为现实。你可能还记得这些人，他们在基于 Kinect 的魔兽世界界面中使用身体动作来模仿游戏中的键盘动作。利用他们灵活的动作和关节骨架工具包(FAAST)，他们为 Gmail 开发了一个 Kinect 界面，他们称之为软件库优化强制挥动(SLOOW)。

他们的骨骼追踪软件允许他们使用谷歌想象出来的所有人工手势来控制你的收件箱，不管这些手势可能多么不切实际。我们喜欢一个好的愚人节玩笑，但是当它们通过一些巧妙的思考变成现实时，我们真的很享受。
