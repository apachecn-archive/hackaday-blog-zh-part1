# LED 环境灯条

> 原文:[https://hackaday.com/2008/05/30/led-ambient-light-strips/](https://hackaday.com/2008/05/30/led-ambient-light-strips/)

![](../Images/3dd59a42f8b7645025b0c26d479512ed.png)
【阴影】在他的环境中发出 LED 条形项目。他从易贝拿了一吨 RGB(红/绿/蓝)发光二极管，制造了几条发光二极管带。为了启动和运行，他使用了一个 [LED-wiz 控制器](http://www.retroblast.com/Hardware/RetroBlast-Review-GGG-LED-Wiz-Review.php)。有了现成的控制器，这是一个非常简单的项目，环境照明效果看起来很棒。休息后或在[项目页面](http://www.instructables.com/id/Building-your-own-color-LED-bars/)查看视频。

2006 年，我们发布了关于[rafkep]类似的[环境照明项目](http://www.hackaday.com/2006/07/10/adaptive-ambient-light-for-movies/)。
