# 你的计算器厄运获得颜色升级

> 原文:[https://hack aday . com/2012/01/19/doom-for-your-calculator-gets-a-color-upgrade/](https://hackaday.com/2012/01/19/doom-for-your-calculator-gets-a-color-upgrade/)

![](../Images/e01482526df2823871621c88fbcde753.png "ndoomfinal")

你不会还在玩黑白游戏吧？你生活在什么年代？令人欣慰的是，TI-nspire 计算器的厄运端口已经[升级到支持颜色](http://tiplanet.org/forum/ndl3ss.php)。前提是你有运行它的硬件。

休息后的视频(和上面的图片)显示了一个 TI-nspire CX 运行流行的第一人称射击游戏。自从一年前我们看到搭载了不同的 TI-83 hack 的测试版以来，它已经经历了几次升级。控制方案已经调整，并增加了一个菜单系统。这和你在 DOS 版本的游戏中看到的屏幕菜单不一样，但是它完成了同样的事情。这个端口与解锁硬件的 Ndless 程序打包在一起，因此您可以执行自己的黑客攻击。

不幸的是，游戏仍然没有声音，但这是一个不同时期的项目。我们知道这一定是可能的，因为我们已经见过用于播放存储在拇指驱动器上的音乐的 TI-84+。T3
