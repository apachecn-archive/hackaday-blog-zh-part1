# 数控光绘

> 原文：<https://hackaday.com/2012/02/11/cnc-light-painting/>

[![](img/1ef9688e1ef887a9cc5d6539609bec79.png "had_light_pt")](http://hackaday.com/2012/02/11/cnc-light-painting/had_light_pt/)

光绘是一种用光源绘制形状的技术，同时相机对其进行长时间曝光拍摄。手工做好这件事需要很多技巧，所以我很自然地决定用 CNC 路由器制作我的[“光艺术”。](http://www.jcopro.net/2012/02/10/cnc-light-painting-vast-improvement-using-a-remote-trigger/ "CNC light painting")

使用这种技术，LED 灯就像正常情况下的雕刻钻头一样。不同的是， [Y 轴与 Z 轴](http://www.jcopro.net/2012/02/01/swap-the-y-and-z-axes-on-a-cnc-router-with-mach3/ "swap X and Y axes in Mach3")互换，可以在你看到的上图显示的平面内轻松移动。这允许旧的 Y 轴以同样的方式打开和关闭灯，雕刻钻头被提升停止雕刻，被降低开始[(在此解释)](http://www.jcopro.net/2012/02/06/cnc-light-painting-some-refinements/ "cut off light in CNC light engraving")。虽然不是一点，但这是一个开关。
