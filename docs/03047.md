# 自动钢丝切割机和剥线钳

> 原文：<https://hackaday.com/2009/01/03/automated-wire-cutter-and-stripper/>

![witecutter](img/87c46a694d3c7b57f08ad47093f5ea3e.png "witecutter")

套件制造商[oom lout](http://oomlout.com/)——我们特别介绍了[他们的伺服机器人](http://hackaday.com/2008/11/22/simple-servo-bot-plans/ "Simple servo bot plans  - Hack a Day")——需要生产大量预切割的电线。在削减和剥离了超出公平份额的部分后，他们决定应用一些重型工程来使事情变得更容易。他们建造了一台[机器来替他们做这项工作](http://www.flickr.com/photos/snazzyguy/sets/72157612098886506/ "WIRC-Wire Cutter and Stripper - a set on Flickr")。它有三个主要组件:一个伺服驱动的送线器来测量长度，一个双伺服剥线器，使用 exacto 刀片，最后是一个剪线器和一个钻孔电机。机器由 Arduino 控制。他们已经[向 Thingiverse 发布了所有的计划和代码](http://www.thingiverse.com/thing:268/ "Automatic Wire Cutter and Stripper by oomlout - Thingiverse"),以防其他任何人想为他们自己的工具箱建造类似的机器。下面嵌入了机器的视频。
