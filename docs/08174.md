# 相机闪光灯:真的假的？

> 原文:[https://hack aday . com/2011/07/27/camera-flash-marquee-real-of-fake/](https://hackaday.com/2011/07/27/camera-flash-marquee-real-of-fake/)

![](../Images/4481651e5eaf28923184993ad4c3b47d.png "camera-flash-marquee")

到了大家最喜欢的评论线程游戏:真的假的？本周的版本来自于[Fabian]发给我们的关于 Androp 乐队的音乐视频《明亮的汽笛》。该视频首先展示了正在分类并连接到试验板的电缆束。我们得到了一个大型 LED 矩阵(大概是用于测试目的)的简短镜头，然后是许多带外部闪光灯的 DSLR 相机的镜头。这些安装在机架上，以产生上图中看到的字幕。在接下来的视频中，乐队在它面前表演。

我们嵌入了原始视频，以及休息后的“制作”视频。还有一个网站可以让你在字幕上写下自己的信息。这一点可以很容易地在 flash 中完成，所以没有 que，你会注意到没有实时馈送。虽然我们认为这个理论是真实的，但我们有点怀疑这个表演是真实的还是视频编辑的魔法。在幕后剪辑中，你可以看到用橡皮筋将试验板连接到每个相机闪光灯上，所以我们猜测至少有一些硬件被设置好了。但是我们想知道动画效果是不是像《T2》中的茶光动画那样在剪辑中完成的。请留下评论，让我们知道您的想法。
