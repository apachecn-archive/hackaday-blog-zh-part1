# 直接用于服装印花

> 原文：<https://hackaday.com/2008/09/20/direct-to-garment-printing/>

[https://www.youtube.com/embed/f3iOuKzhUlQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/f3iOuKzhUlQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

读者[deren lik]向我们指出了直接服装印刷的世界。你可以购买使用喷墨技术直接打印到 t 恤上的商用机器。不幸的是，这些机器的价格高达~$10K，所以黑客们决定填补这些空白。DIYDTG 主持如何建立自己的 DTG 打印机的计划。他们的标准说明基于 Epson C88 打印机。定制载体被构建，然后打印机组件被栓接在顶部。商业 DTG 打印机也是基于爱普生部件的 T2，你可以很容易地购买服装墨水，即使你没有为你的打印机支付额外费用。
