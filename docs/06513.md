# Arduino EMF 传感器

> 原文:[https://hackaday.com/2010/12/31/arduino-emf-sensor/](https://hackaday.com/2010/12/31/arduino-emf-sensor/)

![](../Images/b1aa0ed358dc3e6dc9e2c42f2a803c7c.png "arduino-emf")

作为一名生物医学设备技术员[Adam Outler]设备需要处于最佳工作状态。急诊室的工作人员抱怨心电图上有错误的噪音，他的工作就是解决这个问题。他怀疑是电动势干扰，所以作为快速的第一步，他决定用 Arduino 组装一个电动势检测器。它使用一排发光二极管作为指示条来反映红色天线接收到的电动势。在休息后的视频中，亚当检查了一个房间，寻找可能的干扰源，认为应急灯的充电电路是最有可能的罪魁祸首。由于心电图对电磁场的敏感度是 Arduino 的许多倍，这被证明是一种快速简单的方法来确保他没有找错对象。
