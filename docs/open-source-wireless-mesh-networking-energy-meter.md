# 开源无线网状网络电能表

> 原文：<https://hackaday.com/2011/03/28/open-source-wireless-mesh-networking-energy-meter/>

![](img/700a43b319fc568981313848bed0711f.png "homeaut")

[Jay Kickliter]写信告诉我们他的开源能源/功率表。他们与好友[Frank Lynam]一起设计了一款小型设备，该设备可嵌入现有的电源盒中，并使用 8 核 propeller (P8X32A)微控制器，通过电流互感器来执行真有效值电压和电流测量。[弗兰克]和[杰伊]不要就此打住。该仪表还具有 xbee pro 900 MHZ，为整个考验提供无线(甚至网状网络)能力。

[Jay 和 Frank]估计每台样机的总单位成本约为 80 美元。随着数量的增加，价格下降了大约一半。随着更多的单位，网状网络的魔力，我们可以看到更便宜的 xbee 的成本下降一些。如果你感兴趣的话，可以查看[谷歌代码](http://code.google.com/p/open-energy-meter/)页面了解详情，或者查看[示意图](http://www.chasingtrons.com/storage/post-images/energy-meter/EnergyMeter-Schematic.pdf) (pdf)。

到目前为止，该项目还处于测试阶段，只有一个单一的模块向运行 OS X 可可应用程序的个人电脑发送数据。[Jay]即将被商船队和[Frank]海军占领，所以他们认为我们可以尝试一段时间。

我们已经看到了用于监控功耗的其他硬件，但是将这个电路塞进每个电源盒是一个好主意。

[Frank]在跳跃后的视频中解释了整个项目。
