# led 多点触控液晶屏

> 原文:[https://hackaday.com/2009/10/10/multi-touch-lcd-from-leds/](https://hackaday.com/2009/10/10/multi-touch-lcd-from-leds/)

[https://www.youtube.com/embed/BiJsBaBAUL4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/BiJsBaBAUL4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Thomas]在近一年前承诺，他将向社区提供他如何制作他的 [LED 多点触控面板](http://hackaday.com/2007/12/16/diy-led-multi-touch-panel/)的代码和示意图，但我们必须先等待他的下一个版本。嗯[它终于来了](http://www.pcb-dev.com/index.php?option=com_content&task=view&id=34&Itemid=87&lang=english)，比我们想象的更大更好。他的新版本是由 1536 个 led 组成的 48×32 矩阵，但他更进一步，将它放在一个剥离的 LCD 后面，制成多点触摸显示屏。它看起来没有[原来的](http://cs.nyu.edu/~jhan/ledtouch/index.html)那么快或准确，但可能比我们以前见过的更便携。是的，这一次他包括了他的 [LED 矩阵是如何制造的](http://www.pcb-dev.com/index2.php?option=com_docman&task=doc_view&gid=24&Itemid=38)。
