# 刻录 Arduino 引导程序的初学者指南

> 原文:[https://hack aday . com/2011/08/26/a-初学者-刻录指南-arduino-bootloaders/](https://hackaday.com/2011/08/26/a-beginners-guide-to-burning-arduino-bootloaders/)

说到学习微控制器，每个人都必须从某个地方开始。[Lady Ada]最近发布了一个关于将 Arduino 引导程序烧录到独立芯片上的教程,供那些刚刚开始使用微处理器的人使用。

本教程举了一个常见的例子，有人正在使用 Arduino 芯片进行一个项目，但他们不想牺牲自己的开发板，因为只有微处理器和一些其他组件就足够了。需要一份简短的供应清单，包括一个 Arduino、一个 proto shield 套件、一个 ZIF 插座和一个空白的 ATmega 芯片。

这一过程可能非常简单，并且可能为在座的许多人所熟悉。ZIF 插座被焊接到电路板上，并添加了少量电线用于给新芯片供电和编程。像其他护盾一样，原型板安装在 Arduino 的顶部，使用[Lady Ada]提供的草图，引导加载程序的刻录过程是轻而易举的。

对于新手来说，这是一个有用的技巧，对于经验丰富的 Arduino 用户来说，这也是一个必须掌握的技能。
