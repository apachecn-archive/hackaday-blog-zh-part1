# 自动蒸馏水机得到一次大修

> 原文:[https://hack aday . com/2011/07/27/automated-water-distiller-gets-an-overhaul/](https://hackaday.com/2011/07/27/automated-water-distiller-gets-an-overhaul/)

![automated_water_distiller_stillduino_v2](../Images/65430d2221359bbd00a1b09b35424377.png "automated_water_distiller_stillduino_v2")

我们喜欢黑客社区的一点是，大多数人都有修补和返工他们“已完成”项目的动力。几周前，我们写了一篇关于[Kyle]一起黑出来的水蒸馏装置的文章，这使得他能够自动化他的蒸馏过程。他把他的项目带回车间，做了一些调整，完成后给了我们提示。

他得到了一台新的蒸馏设备，并决定转让他的自动化设备。他清理了他的第一个蒸馏器的大部分组件[，包括玩具钟楼表盘(我们碰巧觉得很有趣)。同样的继电器和 Arduino 被用在第二个版本的蒸馏器中，但是他重新编写了所有的代码来使用他的新控制界面。](http://hackaday.com/2011/07/11/controlling-a-counter-top-water-distiller-with-salvaged-parts/)

新型号的运动液晶面板，允许用户通过一个按钮旋转编码器与机器互动。现在，他可以轻松地浏览一系列菜单，设置蒸馏量和开始时间，让蒸馏器去做艰苦的工作。每次启动时，蒸馏器还会进行快速安全检查，以确保在加热元件启动前一切正常。

[凯尔]说他将继续调整蒸馏器，尽管我们认为它看起来已经很棒了。
