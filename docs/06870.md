# 蓝牙 Tomy Omnibot 黑客

> 原文:[https://hack aday . com/2011/02/19/bluetooth-tomy-omni bot-hack/](https://hackaday.com/2011/02/19/bluetooth-tomy-omnibot-hack/)

![tomy_omnibot_hack](../Images/b7d4c173a290c7860d0ed41a3a20fd8a.png "tomy_omnibot_hack")

[DJ Sures]给我们发来了他的最新作品，这肯定会让那些喜欢 80 年代怀旧的人高兴。他在易贝找到了一个 Tomy Omnibot，并立即把它拿了起来。当他在等待它到来的时候，他计划着一旦它出现在他的门口，他会给它添加什么。机器人一交付，他就把它清理干净，涂上颜色，然后开始工作。

最初的机器人没有移动手臂或头部的能力，所以他立即添加了伺服系统来实现这些动作。一个无线摄像头被插在 Omnibot 的双眼之间，一个无线麦克风被安装在他的胸部。他还增加了一些在他以前的项目中发现的功能，如语音识别、视觉识别、语音合成和远程控制机器人的能力。他计划在不久的将来增加一个超声波 ping 传感器和红外地板传感器。到目前为止，这是一个很好的构建，听起来他还有很多要做。
