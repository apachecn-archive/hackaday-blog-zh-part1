# 从电视屏幕上大规模蚀刻一个草图

> 原文：<https://hackaday.com/2009/05/13/massive-etch-a-sketch-from-tv-screen/>

 <https://www.youtube.com/embed/Mhu3zojL5Y4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>[Jeri]为《胖子和赛车手秀》制作了一个绝对大规模的蚀刻草图。她从一台高清背投电视上拆下了 DLP 芯片，并决定改变 52 英寸屏幕的用途。移动机构使用带尼龙线的屏蔽门滑轮。两组线以十字交叉的方式馈送，使得平行线以相同的方向移动。这些线在 x 和 y 方向移动帐篷杆，从而控制高尔夫球座触针的移动。它由两个 9 美元的 Harbor Freight 18V 钻机的高扭矩电机驱动。他们尝试了几种不同的粉末，但最终使用了原始蚀刻草图中的铝粉，因为它可以粘在任何东西上。一旦他们得到一个足够大的 h 桥，它最终会被 IRC 机器人控制。</p> <p>[途径<a href="http://www.adafruit.com/blog/2009/05/13/hacked-up-net-enabled-high-definition-etch-a-sketch/" title="Hacked up net enabled high definition Etch-A-Sketch « adafruit industries blog" target="_blank"> adafruit </a></p> </body> </html>
