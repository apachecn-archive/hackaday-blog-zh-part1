# 体热感应电脑安全系统

> 原文:[https://hack aday . com/2011/04/08/body-heat-sensing-PC-security-system/](https://hackaday.com/2011/04/08/body-heat-sensing-pc-security-system/)

![lockifnothot](../Images/4eb36f559ca722d814d22afdfbef1e63.png "lockifnothot")

[Didier Stevens]来信告诉我们他最近开发的一款个人电脑安全软件。他的应用程序 [LockIfNotHot](http://blog.didierstevens.com/2011/04/06/lockifnothot/) ，与你的电脑和红外温度传感器一起工作，以便在你离开时锁定你的电脑。

这个系统背后的理论非常简单。基本上，红外温度传感器监测你何时在办公桌前，通过你身体散发的热量来感知你的存在。然而，一旦你离开，它就会锁定电脑，因为周围环境的温度会立即下降。这很简单，但正如你在下面的视频中看到的，它工作得相当好。

该软件具有可配置的设定点和超时值，这使得它足够灵活，以适应您的具体情况。他碰巧使用现成的红外传感器，但我们认为任何 USB 温度模块都可以做到这一点。如果你碰巧处理敏感信息，但经常忘记锁定你的工作站，这是你的计划！
