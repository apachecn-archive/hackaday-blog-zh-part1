# 用眼睛控制火星车

> 原文：<https://hackaday.com/2008/12/08/controlling-a-rover-with-your-eyes/>

[https://www.youtube.com/embed/oDcbPUQ8kvY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/oDcbPUQ8kvY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

仅仅通过观察你想要的位置来控制一个机器人是相当令人敬畏的。一个网络摄像头对准你的脸，分析你的动作和瞳孔方向来发送机器人信号。看着一个位置，机器人就去，改变你的表情，发送其他命令给机器人。这在易用性上轻松超越了[激光制导辅助机器人](http://hackaday.com/2008/12/02/point-laser-get-beer/)。
