# Wacom 灯光涂鸦

> 原文:[https://hackaday.com/2009/09/20/wacom-light-graffiti/](https://hackaday.com/2009/09/20/wacom-light-graffiti/)

![wacom_light_graffiti](../Images/72459960c2725bc879b81dc50e426c6d.png "wacom_light_graffiti")
