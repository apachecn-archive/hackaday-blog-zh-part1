# 过时技术带

> 原文：<https://hackaday.com/2008/06/05/obsolete-technology-band/>

  <http://www.vimeo.com/moogaloop.swf?clip_id=1109226&amp;server=www.vimeo.com&amp;show_title=1&amp;show_byline=1&amp;show_portrait=0&amp;color=&amp;fullscreen=1>

 
【电台司令】举办了一场比赛，让歌迷们将他们专辑*中的单曲*裸体**进行混音。主唱汤姆·约克在 NPR 上提到，这场比赛基本上是一个笑话，因为裸体音乐是以 6/8 定时和 63bpm 录制的，比传统的混合音乐慢得多。上面来自[詹姆斯·休斯顿]的视频是最有创意的参赛作品之一。利用旧的电脑硬件，他以一种非常独特的方式重新创作了这首歌。他用辛克莱·ZX 谱做吉他音轨，用点阵打印机做鼓，用扫描仪做低音，用硬盘阵列做人声。

想用过时的技术制作自己的乐队？点按几个指针，让你开始。

<http://www.youtube.com/v/1BU4f2Xj_hs&amp;hl=en>

 
扫描仪曾经是他们那个时代炙手可热的外设；现在许多人沦落到收集灰尘。复活这些文物并把它们作为乐器好好利用[。](http://www.ganjatron.net/misc/scanjet/)

在电台司令的视频中，我们看到一系列硬盘被用作扬声器。Afrotechmod 制作了上面这个以星球大战为主题的视频。他[将音频输入直接连接到用于控制磁头和磁盘运动的线圈](http://afrotechmods.com/cheap/hdspeakers/hdspeakers.htm)。因为所有的硬盘都是不同的，所以可能需要一点点摸索才能得到想要的效果。

<http://www.youtube.com/v/2KsHuPSCRIM&amp;hl=en>

 
点阵打印机以其标志性的声音而闻名。在上面的视频中，[Sue Harding]向我们讲述了她与它们打交道的经历，以及她是如何摆弄颜色和字符密度来创造新声音的。
<http://www.youtube.com/v/dsg4P-e10MQ&amp;hl=en>

 
最后，我们还有辛克莱·ZX 谱。虽然还不完全清楚使用的是什么程序，[平台上有相当多的音频跟踪器可用](http://en.wikipedia.org/wiki/List_of_audio_trackers#ZX_Spectrum)。上面的视频是数字现实制作的 ZX 频谱演示。
