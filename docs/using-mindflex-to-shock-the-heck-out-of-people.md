# 使用 Mindflex 让人们震惊

> 原文：<https://hackaday.com/2010/03/04/using-mindflex-to-shock-the-heck-out-of-people/>

我们喜欢看到雄辩的黑客，但这不是其中之一。[Aaron]和他的虐待狂同伴们正在使用一款带有电击的 Mindflex 游戏。如果你的大脑是空闲的，你会没事的，但是脑袋里太多的活动会让你的手臂受到严重的冲击。休息之后看一下视频(哔哔声，但可能是 NSFW)。

我们以前见过 [Mindflex](http://hackaday.com/2009/11/07/mindflex-teardown/) ，他们使用它的界面的方式和我们见过的力量训练器一样，通过[接入 led](http://hackaday.com/2009/10/20/brain-control-for-the-arduino/)。[电击由 Qkit](http://store.qkits.com/moreinfo.cfm/FK901) 提供，所以希望它背后有足够的工程技术来保证“参赛者”的安全。
