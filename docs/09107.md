# 用雨滴点亮发光二极管

> 原文：<https://hackaday.com/2011/11/18/lighting-leds-with-raindrops/>

[Steven]身边放着一张音乐礼品卡，他想他还不如重新使用里面的压电扬声器。在没有特定项目的情况下，他将一个 LED 焊接到压电元件上，并点击它，这使得 LED 如预期的那样发光。他开始想知道点亮 LED 需要多大的力量，以及雨滴能否做到。

他首先在淋浴中测试了他的理论，正如你在下面的视频中看到的，它实际上是有效的，尽管光线像你想象的那样昏暗和零星。他最终发现，为了获得最佳照明效果，压电元件在被单个水滴撞击时工作得最好，水滴从最低 4 英尺的高度落下，中间有停顿。为了实现这些规格的水流，他建造了一个雨水漏斗，这样他就可以控制水滴的频率和强度。

从我们所看到的来看，它似乎工作得很好。我们似乎想不出水力 LED 的任何实际应用，但这仍然是一组有趣的实验。
