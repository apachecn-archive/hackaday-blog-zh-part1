# 火人节 2010 上的火球

> 原文：<https://hackaday.com/2010/10/06/pyrosphere-at-burning-man-2010/>

![](img/b48a367773aa8f604b098273c245af54.png "pyrosphere-at-burningman")

什么东西有 37 英尺高，有 91 个火焰喷射器，由 Arduino 控制？为什么是今年火人节的互动艺术作品[火圈](http://nexusorg.org/2010/burning-man/the-pyrosphere-video-and-photos/)。它随着音乐点亮了夜晚，让我们想要搭一把草坪躺椅来观看表演。你可以在休息后的视频中看到自己，但你真的应该穿上 [LED 皮衣](http://hackaday.com/2010/09/30/dazzling-coat-sure-to-be-in-demand-with-pimps-everywhere/)亲自去看。
