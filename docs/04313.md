# 万圣节道具:索隆馅饼

> 原文：<https://hackaday.com/2009/10/25/halloween-props-pie-of-sauron/>

![pie-of-sauron](img/e558fea9c708e39908d71a907061ab52.png "pie-of-sauron")

[理查德]将创意雕刻与真空管电子产品[相结合，打造出独特的南瓜产品](http://www.decadecounter.com/vta/articleview.php?item=880)。他使用模板和切割的雕刻方法，利用一种廉价的雕刻工具获得了很好的效果。他从手电筒中回收了一个 LED 模块来提供内部照明，但这是我们最喜欢的中心功能。[理查德的]使用了辉光转移计数管，或[德卡龙](http://en.wikipedia.org/wiki/Dekatron)，它为该项目提供了类似圆形[赛昂眼](http://en.wikipedia.org/wiki/Cylon_(Battlestar_Galactica))的东西。休息之后有一段视频。

你可能没有看起来很邪恶的迪卡侬，但我们猜测你有一个微控制器和一些发光二极管，可以作为一个晚上的替身。我们很乐意做一个读者南瓜灯综述，所以如果你建立了什么，给我们发一张图片！
