# 围绕 TTL CPU 构建计算机

> 原文:[https://hack aday . com/2011/07/18/building-a-computer-around-a-TTL-CPU/](https://hackaday.com/2011/07/18/building-a-computer-around-a-ttl-cpu/)

![](../Images/166bad5d4108989541471660f21ab386.png "magic-1-homebrew-computer")

[比尔的]在他的自制电脑上工作了将近十年。他没有像我们看到的许多项目那样从 Z80 处理器开始，而是从 74 系列 TTL 芯片和大量的电线连接来构建 CPU 本身。

休息后的视频展示了该功能。我们喜欢前面板，它挤满了信息，但设法保持有序，并提供许多方便的功能。我们最喜欢的是通过旋转转盘来暂停执行和滚动寄存器的能力。时钟信号具有可变速度，该速度由内部 DIP 开关组件选择，可以在暂停期间改变。它运行 MINIX，有一个程序库，但最令人惊讶的可能是它提供网页服务的能力。

最近，我们对深入研究程序语言抽象感兴趣，以了解芯片内部发生了什么。这让我们对那些从零开始构建处理器的人有了新的尊重。这样想，如果你真的需要在 gates 之外构建每条指令，你将能够理解这些指令在最基本的层面上是如何工作的。
