# 愚弄我一次…或者尽可能多的愚弄我

> 原文：<https://hackaday.com/2012/03/26/fool-me-once-or-as-often-as-possible/>

![](img/8a78dc16b27bef3a24fd1044d7775913.png "april-fools-day")

曾经有一段时间，发布一个虚假的故事对所有参与者来说都很有趣。但是在这个不断钓鱼的时代，我们精明的读者群几乎不可能完成它。我们不会让你因为一个很糟糕的建议而失望，我们会打电话给你，听听你有什么恶作剧。把这当成另一个节日主题，然后[告诉我们你在愚人节有什么计划](http://hackaday.com/contact-hack-a-day/)。

像往常一样，我们正在寻找你自己关于这个话题的帖子。我们总是想在该表扬的地方给予表扬，所以把你的恶作剧贴在你的博客或网络上其他喜欢的角落，并把链接发给我们。没地方放？你可以随时在我们的项目日志论坛中[发起一个帖子，或者查看](http://forums.hackaday.com/viewforum.php?f=3&sid=468fbe8766001a61b24e3ec9be2e7802)[的其他帖子来支持你的主张](http://hackaday.com/2010/12/07/project-hosting-a-common-question/)。

如果你需要一些帮助，我们有几个例子可以帮助你思考。你可以走假视频演示的路线，比如这个 Gmail 手势骗局，或者最近的 T2 人类鸟翼项目。但是这需要大量的生产时间和巧妙的种子创意。也许一些真正简单的东西会对室友们有很大帮助。我们在考虑用透明指甲油覆盖的[肥皂](http://todayilearned.co.uk/2012/03/25/covered-by-nail-polish-the-soap-bar-wont-lather/)来防止起泡，或者也许你想[重新配置你的路由器来颠倒页面](http://www.ex-parrot.com/~pete/upside-down-ternet.html)。我们迫不及待地想看看你想出了什么！
