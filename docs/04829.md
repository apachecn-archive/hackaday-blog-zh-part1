# 我们对 IPad 的看法

本周早些时候发布的 [iPad](http://www.mahalo.com/ipad) 、[已经成为一场大型媒体盛宴。有些人兴高采烈，有些人失望。我们被一遍又一遍地问及对这款设备的看法。休息之后加入我们来寻找答案。](http://www.mahalo.com/apple-event-summary)

我奶奶会喜欢这个的。我真的认为她是他们的目标。它易于使用，可移植，并有一个自定义的界面，消除了所有的 windows 风格的界面 cruft。我个人不需要。我有一部 iPhone(是的，我知道你们大多数人讨厌它们)。除了更大的屏幕，iPad 没有给我任何我的 iPhone 没有的东西。

不过，我很兴奋。我对一款带有电器风格界面的中级[触摸屏](http://www.engadget.com/2010/01/28/cypress-demos-14-inch-truetouch-capacitive-multitouch-screen/)平板电脑的前景感到兴奋。我迫不及待地想看到可以用自定义界面运行 Linux 的仿冒品。山寨机将实际上有 USB 端口，和一个不摇晃的背部。这些山寨机很可能有一个前置摄像头。我想要一个。

就可攻击性而言，这实际上可能是相当可攻击的。处理器实际上并不像有些人认为的那样是专有的。它基于 ARM，与我们见过的其他设备没有太大区别。看起来好像[他们计划了一个摄像机](http://www.engadget.com/2010/01/27/iphone-sdk-calls-out-nonexistent-ipad-cam-confirms-split-views/)。主板上甚至可能会有一个点。我完全期待看到它“越狱”，但我想看到的是被修改过的内脏。我想看到家庭黑客添加外部存储(卡槽)，加载不同的操作系统，增加存储。话又说回来，我对 iPhones 和 iPod touches 也有同样的想法，但我真的一点也没见过硬件黑客。

这是我第一次听说 iPad。我厌倦了所有苹果平板电脑的谣言，所以我不再关注了。我只运行 Linux，除非这个设备能与我的 Linux 系统同步，否则我不感兴趣。我通常也不会对任何花费超过 250 美元的东西进行硬件破解，但是我不会马上打开这个东西的外壳。

iPad 有机会成为一款伟大的产品，但可能不适合我，也不适合我们的大多数读者。它把自己推销给那些想坐在沙发上听音乐、浏览脸书或看 Youtube 的普通互联网用户，而不是铁杆图片编辑、作家或程序员。我很高兴看到像苹果这样的人真正推动了市场，迫不及待地想看到回应，尤其是来自像微星和华硕这样的[人。就我个人而言，我在等待有人制作一款真正伟大的可转换平板上网本，支持 Linux。](http://www.engadget.com/2010/01/08/msi-shows-off-10-inch-android-tablet-running-new-tegra-chipset/)

**【Jakob】**—我讨厌苹果产品，我的意思是几乎带着一种激情。但我确实拥有一部 iPhone——我承认，这是一部漂亮的手机。小巧、功能强大、方便——所有 iPad 没有的东西。这就像苹果公司不能决定再做一个 iPhone 版本或一个完整的平板电脑，而是决定给我们一个半神经兮兮的 iPad。据[我所知，没有多任务](http://gizmodo.com/5458382/8-things-that-suck-about-the-ipad)和其他几个功能，如果可以包含在 iPhone 中，为什么他们要从 iPad 中删除？最后一点:作为一个苹果产品，它比它应该的价格多 500 美元。

(它甚至没有额外吸收的翅膀)

嗯，我不认为黑客入侵 iPad 会有太大的影响，我不会买一个，因为我看不到拥有它的理由。如果其他人也有同样的想法，没有多少人会去买 iPad，因此也没有多少人会去黑 iPad。
