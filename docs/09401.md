# 所以你想学 FPGAs

> 原文：<https://hackaday.com/2011/12/30/so-you-wanna-learn-fpgas/>

FPGAs 是蜜蜂的膝盖。除了通过告诉芯片做什么来给芯片编程，FPGA 允许你告诉芯片做什么*。*像所有事情一样，需要一套新的技能来充分利用 FPGAs 的能力。[Mike Field]决定回馈整个互联网社区，开设了一个 FPGA 设计的[速成班](http://hamsterworks.co.nz/mediawiki/index.php/FPGA_course)。

现在，[Mike]有几个模块，包括二进制数学、总线、计数等科目，当然还有设置 FPGA 硬件。推荐的硬件是 [Papilio One](http://www.papilio.cc/index.php?n=Papilio.Hardware) ，虽然 [Digilent Nexys2](http://digilent.org/Products/Detail.cfm?Prod=NEXYS2) 是【迈克】迄今为止一直使用的。

我们已经看到了大量使用 FPGAs 的令人惊叹的东西，如[仿真 Mac 加](http://hackaday.com/2011/09/16/recreating-the-mac-plus-with-an-fpga/)、[破解 HDCP](http://hackaday.com/2011/11/30/hdcp-falls-to-fpga-based-man-in-the-middle-attack/) ，以及[时间陶笛](http://hackaday.com/2011/12/13/building-the-ocarina-of-time-with-an-fpga/)。[Mike]的教程看起来是一些 FPGA 工作的一个很好的起点。[Mike]也在寻找关于他的教程的反馈，所以如果你知道他应该讲些什么，一定要给他写信。
