# 带有 PIC Micro 的立体 3d

> 原文：<https://hackaday.com/2009/11/04/stereoscopic-3d-with-a-pic-micro/>

![tomy2 (Custom)](img/c0133da52575441c035837a8ba43dbbd.png "tomy2 (Custom)")

[Pyrofer]提交了他的[立体游戏项目](http://www.pyrofersprojects.com/blog/?page_id=92)，我们兴奋得头晕目眩。他拆开了一台旧的 TomyTronic 3D 手持浏览器，并装上了新的内脏。他正在使用 PIC micro 将立体图像推送到双液晶显示器上。他从头开始编写所有代码，包括 3d 库、wii nunchuck 驱动程序和 LCD 驱动程序。这东西甚至有蓝牙，这样他就可以玩多人游戏，如果他有第二个的话。整个装置通过锂聚合物电池保持活力，因此除了 wii nunchuck 之外，您不必担心任何电线。这东西太棒了，我们很想玩一玩。休息之后可以看一段视频。
