# 遥控你的破车(危险地)

> 原文：<https://hackaday.com/2009/11/10/remotely-control-your-crappy-car-dangerously/>

这里有一个让我们回想起多年前发布《毁灭》最初几集时那种眩晕的感觉。滑铁卢实验室的疯子们已经通过笔记本电脑、iPhone 和…改装了一辆破旧的奥兹莫比尔汽车进行遥控。等等… [动力轮](http://en.wikipedia.org/wiki/Powerwheels)。

刹车和油门踏板是用一个扳手来驱动的，这个扳手连接着一个用螺栓固定在汽车地板上的马达。方向盘已经被齿轮取代，并通过摩托车链条连接到马达上。就像我们上个月看到的货车一样，一个 iPhone 应用程序被编写成无线控制末日之车。这导致一些汽车冲浪和[幽灵骑着鞭子](http://en.wikipedia.org/wiki/Ghost_riding_the_whip)在休息后的视频。

令我们高兴的是，他们还实现了迄今为止最非正统的汽车界面，动力轮。增加了一条链条来测量玩具方向盘的方向，并使用光学编码器来测量微型电动车的速度。看起来它并没有很好地转化为一辆全尺寸的汽车，但它最大化了他们的风格点。
