# 转盘光序列发生器

> 原文:[https://hackaday.com/2010/01/21/turntable-light-sequencer/](https://hackaday.com/2010/01/21/turntable-light-sequencer/)

![](../Images/4c256253d79660c34b4d9f1a7c5b0d46.png "turntable-light-sequencer")

[本杰明]制造了一个音序器，它使用一个转盘和光传感器来放下一个时髦的节拍。如果你喜欢令人毛骨悚然的视频，在休息之后，我们会为你报道。撇开艺术电影不谈，他有一个有趣的项目。四个光传感器安装在转弯记录的下方，led 在上方盘旋。他对老唱片的憎恨是显而易见的，因为必须在光盘上钻孔才能让光线穿过。序列中的四个音符可以改变声音和颜色，以及控制电机速度和方向。该项目还有四个手动输入，为重复的节拍序列添加一些变化。它比[的便士序列器](http://hackaday.com/2009/10/15/turntable-sequencer-scratches-with-coins/)稍不实用，但仍然很有趣。
