# 乐高自动变速器

> 原文：<https://hackaday.com/2010/10/13/lego-automatic-transmissions/>

![](img/76252b1aa8b8b3e53d90392f7afa9966.png "lego-automatic-transmission")

[AviatorBJP]正在使用乐高零件制造一些令人印象深刻的自动变速器。你最好去看看他的 YouTube 频道，因为他有很多与这个话题相关的视频。我们已经嵌入了第一代和第二代车辆的测试镜头，以及休息后的最新飞轮设计。但是我们言过其实了，让我们看看这个系统是如何工作的。

每个传动装置都以一个机构为中心，该机构包括连接在中心轴上的铰接臂。两臂用橡皮筋固定在一起，但随着轮轴旋转得越来越快，它们会克服橡皮筋的弹力，开始向外转动。这将向一个方向拉动轴，将其齿轮向上移动到变速箱中的下一个位置。为了测试这个系统,[AviatorBJP]使用了跑步机。一根绳子系在车的前部以保持其位置，跑步机被打开以模拟发动机动力。

这个设计非常出色，他没有独享。如果你已经厌倦了自己制作的[手动乐高变速器](http://hackaday.com/2010/04/08/lego-gearbox-seven-speed-plus-reverse/)，你可以按照他的多视频制作流程为自己制作一个。
