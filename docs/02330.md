# 品牌:电视

> 原文：<https://hackaday.com/2008/08/04/make-television/>

<http://www.vimeo.com/moogaloop.swf?clip_id=1465573&amp;server=www.vimeo.com&amp;show_title=0&amp;show_byline=0&amp;show_portrait=0&amp;color=23C50B&amp;fullscreen=1>



以[创客节](http://www.hackaday.com/tag/makerfaire)而闻名的《Make》杂志宣布了一个名为[Make:TV](http://www.makezine.com/tv/)的新项目。该节目将从 2009 年初开始在美国各地的公共电视台播出。好消息是，你可以提交你的项目的 2 分钟长的视频，以纳入该节目的[创客频道](http://www.makerchannel.org/)部分。更大的消息是，如果你的视频被选中，他们将向你发送一张价值 50 美元的 Maker Shed 礼券和一本免费的 Make 杂志。
