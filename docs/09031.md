# 厨房黑客:玛格丽塔机

> 原文:[https://hack aday . com/2011/11/08/kitchen-hacks-the-margarita-machine/](https://hackaday.com/2011/11/08/kitchen-hacks-the-margarita-machine/)

![](../Images/c811999f788cd02962149af402a0cf32.png "the-margarita-machine")

如果有一天(保罗·德根科尔布)真的心血来潮决定做这件事(就像他说的那样)，我们认为他应该辞去工作，进入一个充满奇思妙想的状态，度过余生。玛格丽塔机在一个快速且相对安静的过程中制作了五加仑的冰镇烈酒,整个后院都是排队的客人，不仅仅是为了喝酒，也是为了看看你这次到底想出了什么。

很容易看出这个容器是一个冰屋冷却器，但你从哪里得到足够强大的电机和叶片组件来将冰块变成雪泥呢？只需点击家庭中心，挑选出你所选择的垃圾处理器型号。随着球阀服务龙头关闭，处理器吸下液体和冰，通过一些 PVC 管将粉碎的混合物射回到冷却器的顶部。这种循环有助于混合东西，但有时[保罗]用一个玻璃杯作为活塞来搅动流氓冰块。
