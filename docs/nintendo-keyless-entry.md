# 任天堂无钥匙进入

> 原文：<https://hackaday.com/2008/12/08/nintendo-keyless-entry/>

[https://www.youtube.com/embed/bUm7daf36Mk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/bUm7daf36Mk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

任天堂的无钥匙进入系统会警惕地保护你的门免受入侵。输入正确的密码你就能进入，输入错误的密码它会拒绝你并给你拍照。[action_owl]在这个锁上做了一个了不起的工作，主要使用回收的部件，如旧的光盘和一个 [Arduino](http://www.mahalo.com/Arduino "Arduino - Mahalo") 。不管有没有计算机，它都能工作。如果你选择不使用电脑，你就不会看到输错密码的人的照片。

我们喜欢这个项目，但我们觉得需要指出的是，这个锁是相当无用的。每个人都知道进入的密码是[上，上，下，下，左，右，左，右，B，A](http://en.wikipedia.org/wiki/Konami_Code) 。
