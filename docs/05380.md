# 火花塞音乐

> 原文:[https://hackaday.com/2010/05/23/spark-plug-music/](https://hackaday.com/2010/05/23/spark-plug-music/)

[https://www.youtube.com/embed/92qql941DM4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/92qql941DM4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[这是](http://www.youtube.com/watch?v=92qql941DM4)(上面的视频)也许是有史以来最抽象的声音播放方式。是的，我们听过[硬盘音乐](http://hackaday.com/2008/06/07/more-hardware-music/)和[过时的技术乐队](http://hackaday.com/2008/06/05/obsolete-technology-band/)，但是【DJ Sures】带给我们有史以来第一个，火花塞乐器。

就像 Velcro 和特氟隆一样，音乐火花塞被认为是一个意外。[DJ Sures]用他的火炬塔点火控制器测试了能源使用和火花功率后，发现可以产生不同的频率。在 [死亡金属](http://www.youtube.com/watch?v=98ew0VtHmik) 小星星一闪一闪被听到之前只是重新编程的问题。现在他只需要稍微提炼一下，搭建一个[全立体声音箱](http://hackaday.com/2009/08/14/solid-state-tesla-coil/http://hackaday.com/2009/08/14/solid-state-tesla-coil/)。
