# 机械白板

> 原文：<https://hackaday.com/2008/05/08/mechanical-white-board/>

[Romado12187]刚刚在 Instructables 上发布了他的[机械干擦板](http://www.instructables.com/id/Mechanical-Dry-Erase-Board/)项目的演示。上周末在 Maker Faire 的微软展台上展出了它。不像[sprite_tm]的版本[我们之前提到的](http://www.hackaday.com/2008/04/28/remote-control-whiteboard/)，这个版本更像一个传统的针式绘图仪。建筑完全在[Romado12187]的宿舍里完成，大部分框架使用胶合板和 PVC 管。机械部件从 [SDP/SI](https://sdp-si.com/eStore/) 购买，控制器硬件为[phi gets](http://www.phidgets.com/)。他包括一个操纵杆，但它也有一个 C#命令行程序进行控制。[Romado12187]的 write up 也有很多改进，帮助你的第一次尝试比他的更好；建议购买预安装板，并精确计算电机功率。捕捉一个视频的绘图仪在行动后，跳转。
