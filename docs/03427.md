# 自治坦克恐吓校园

> 原文：<https://hackaday.com/2009/04/23/autonomous-tank-terrorizing-campus/>

![tank](img/33a57b58ba72443e17957267c82ccf1b.png "tank")

作为 ECE4007 的高级设计项目，[Nate]、[An]、[Chris]、[Wink]搭建了一个[自主玩具坦克](http://www.autotank.nateklein.net/)。它使用松下红外运动传感器来寻找目标，然后一旦面对目标，它就通过网络摄像头切换到视觉运动跟踪。如果它能足够接近，它会停下来，开始转动炮塔，以获得更高的精度。最后它发射一颗弹丸。它的大脑是一个运行 windows CE 的 [ICOP 技术 eBox-2300](http://www.embeddedpc.net/Default.aspx?tabid=110) 。网站上提供了所有的编程，以及各种传感器和硬件的分类。正如你在休息后的视频中看到的，它做得很好。如果再给一些时间，我们确信他们可以加快目标收购的进程。也许我们应该为[佐治亚](http://hackaday.com/2005/07/07/mechatronic-guitar/) [科技](http://hackaday.com/2007/12/20/talcum-powder-print-bot/) [决赛](http://hackaday.com/2008/12/10/gigapixel-panorama/) [项目](http://hackaday.com/2007/12/12/autonomous-foosball-plus/)增加一个类别。
