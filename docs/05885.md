# 用于彩弹罐的可穿戴控制器

> 原文:[https://hack aday . com/2010/08/31/wearable-controller-for-your-paint ball-tank/](https://hackaday.com/2010/08/31/wearable-controller-for-your-paintball-tank/)

![](../Images/dbe7386c9808d66caf6dc94c82690e15.png "head-controlled-tank")

如果你太虚弱而承受不了彩弹的全部冲击力，那就让这个坦克充当你的替代品。位于平台顶部的摄像机将视频反馈给操作员的头戴式显示器。用户不用操纵杆或其他传统控制器，而是通过环顾四周来瞄准，他或她的头部运动由相机和坦克的枪管来模仿。它看起来比听起来更酷，所以休息后和我们一起去看看吧。如果你要对抗这个东西，我们建议你瞄准镜头。
