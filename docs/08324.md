# 我们见过的最大的 LED 立方体仍然只完成了一半

> 原文:[https://hacka day . com/2011/08/14/largest-led-cube-we ve-ever-seen-is-was-only-half-complete/](https://hackaday.com/2011/08/14/largest-led-cube-weve-ever-seen-is-still-only-half-complete/)

![](../Images/a1f4c24221abe93fd7a29c884bbf17a5.png "cube")

[Brendan Vercoelen]是一名在新西兰学习工程的大学生。他说，与其他学生项目相比，他最近的[巨型 LED 立方体建筑](http://no8hacks.com/blog/2011/7/11/3d-led-cube)“不是很严肃”，但这并不意味着它不令人印象深刻。最初的建造计划是一个 16x16x16 的三色 LED 立方体。在意识到焊接的工作量之后，[Brendan]将他的设计缩小到 16x16x8 的立方体*oid*，但是另一半可以在项目完成时附加。

从费用明细来看，[Brendan]只花了大约 550 美元，远低于我们的预期。最贵的商品是 4000 多个红绿橙三色发光二极管。我们讨论过的最大的 LED 立方体( [1](http://hackaday.com/2011/01/02/512-led-cube/) 、 [2](http://hackaday.com/2011/03/18/512-led-cube-again/) 、 [3](http://hackaday.com/2011/03/20/third-times-a-charm-512-led-cube-kicks-it-up-a-notch-with-rgb-leds/) )已经达到 8x8x8 的最大值，或者说总共 512 个 LEDS。尽管[Brendan]的构建只完成了一半，但它的体积仍然比我们见过的最大的 LED 立方体大四倍。

战书已经发出。伙计们，这是我们要打败的。休息之后，请观看立方体的视频。
