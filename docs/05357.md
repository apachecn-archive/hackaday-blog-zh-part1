# 使用 FOSS 的命令行视频处理

[丹尼尔·帕卢斯卡]正在通过从命令行编辑视频来摆脱点击操作。使用免费的开源软件包 FFmpeg、Imagemagick 和 Sox，他从多个视频中制作新的剪辑，效果包括叠加、切片和将每个视频分配到不同的象限。最后一个选项对于同时显示同一事物的不同角度的照片很有用，但是我们确信你可以找到一种方法来使用它们。他使用 shell 脚本来自动化一些过程，但是如果这是您第一次尝试这些工具，这些命令还是很容易理解的。毕竟，伟大的视频制作对成为[网络红人](http://hackaday.com/2009/09/19/how-to-make-your-project-an-internet-sensation/)大有帮助。
