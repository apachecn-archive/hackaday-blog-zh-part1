# Segway 和输入滤波

> 原文:[https://hackaday.com/2009/02/19/segway-and-input-filtering/](https://hackaday.com/2009/02/19/segway-and-input-filtering/)

![seg](../Images/1742554ba9f55696ab0c92a9b46d5b3e.png "seg")

[Kerry]正在为一个高级项目制造一辆赛格威型汽车。虽然看起来还没有完成，但是已经完成了一些很好的工作。与任何平衡机器人一样，它严重依赖陀螺仪的输入，至少在这种情况下，是加速度计。这些数据可能会非常混乱。传感器的敏感性意味着人们经常在项目中以振动或漂移而告终。[Kerry]已经做了一些工作来寻找一种平滑的方法，这种方法可以产生快速、准确但平滑的反应。休息之后你可以看到一些方法对比的视频。
