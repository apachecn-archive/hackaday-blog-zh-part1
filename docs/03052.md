# Game Boy 口袋背光

> 原文：<https://hackaday.com/2009/01/04/game-boy-pocket-backlight/>

![gameboy](img/3d4436220f7da2bf0b4bd32e3d7b4992.png "gameboy")

[palmertech]和[Bibin]最近都完成了 [Game Boy](http://www.mahalo.com/Game_Boy "Game Boy - Mahalo") Pocket 的背光项目。移植过程中最困难的部分是小心地移除 LCD 上的反光衬垫。彻底清洁后，添加了扩散器和背光面板。[palmertech]使用了从 DS 中回收的[背光，而[Bibin]](http://benheck.com/forums/viewtopic.php?p=326152 "View topic - Backlit Gameboy Pocket COMPLETED!")[使用 led 建造了自己的](http://benheck.com/forums/viewtopic.php?t=27567&postdays=0&postorder=asc&start=0 "View topic - THE OTHER Backlight!")。你可以在下面嵌入的视频中看到他的背光。还有一个[拆解视频](http://www.youtube.com/watch?v=kU5qCXmFbHg "YouTube - Taking apart the backlit GBP")也有。
