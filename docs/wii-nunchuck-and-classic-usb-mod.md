# Wii 双截棍和经典 USB 模式

> 原文：<https://hackaday.com/2009/03/18/wii-nunchuck-and-classic-usb-mod/>

[https://www.youtube.com/embed/WLWAWtpcelM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/WLWAWtpcelM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[kero905]送来他正在做的这个项目。通过 USB 连接 Wii 控制器是一个[开源硬件转换器。它用一个](http://slotcover.wordpress.com/2009/03/12/wii-extension-controller-to-usb-project/) [Arduino](http://www.mahalo.com/Arduino "Arduino - Mahalo") 作为它的大脑。它仍在开发中，唯一的工作控制器是双截棍和经典。他注意到有足够多的额外引脚可以很容易地适应街机控制器。代码可以在网站上找到，还有一个粗略的零件清单。
