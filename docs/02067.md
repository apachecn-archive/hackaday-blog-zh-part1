# 联邦通信委员会审查 MSI 风

> 原文:[https://hackaday.com/2008/06/26/msi-wind-under-fcc-review/](https://hackaday.com/2008/06/26/msi-wind-under-fcc-review/)

我们这些一直在急切等待涉足 MSI Wind 的人可以看到隧道尽头的一些曙光，因为它是目前正在由 FCC 审查国内销售的。Wind 是微星对华硕改变游戏规则的超便携电脑 Eee PC 的回应，规格略好。它采用英特尔凌动 1.6GHz CPU，1GB 内存。它还内置了蓝牙和网络摄像头。在 jkkmobile 或[查看一些风的内部部件的照片，从 FCC 的网站](https://fjallfoss.fcc.gov/oetcf/eas/reports/ViewExhibitReport.cfm?mode=Exhibits&RequestTimeout=500&calledFromFrame=N&application_id=993537&fcc_id=%27I4L-MS6837D1%27)获取它们。
