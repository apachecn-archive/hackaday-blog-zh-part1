# 树莓派可能不是蒸汽制品

> 原文：<https://hackaday.com/2011/08/26/raspberry-pi-might-not-be-vaporware/>

当我们[第一次报道](http://hackaday.com/2011/05/06/miniature-pc-packs-a-big-punch/)Raspberry Pi 时，我们暗暗想知道这个项目是否会变成一个软件。随着树莓派总部[阿尔法主板](http://www.raspberrypi.org/?p=78)的发布，看起来这款小巧、成熟的电脑可能真的会生产出来。

在去年五月发布的树莓派(Raspberry Pi)的[公告中，这款主板大约有一个 USB 拇指驱动器那么大，售价 25 美元。看起来在最终版本中，主板会稍微大一点——大约一张名片的大小——并且 alpha 主板的生产成本有点高。也就是说，这些主板似乎比我们以前看到的更有能力。Raspberry Pi 现在拥有 10/100 以太网和模拟音频和视频，同时保留了我们在原版中看到的 HDMI、USB 和 SD 卡支持。](http://www.bbc.co.uk/blogs/thereporters/rorycellanjones/2011/05/a_15_computer_to_inspire_young.html)

Raspberry Pi 团队已经在这些主板上运行 Debian，所以现在我们想知道 gumstix 主板何时会降价。当然，这个项目旨在用于教育，但我们迫不及待地想看看这个硬件能做些什么。如果你已经知道你会怎么做，请在评论中告诉我们。
