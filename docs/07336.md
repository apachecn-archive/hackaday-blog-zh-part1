# PCB 零件回收变得容易

> 原文：<https://hackaday.com/2011/04/08/pcb-parts-salvaging-made-easy/>

花点时间想想这些年来你扔掉了多少旧的或损坏的电脑主板。当然，我们会尽可能地重新利用一切，但是如果没有合适的方法，从复杂的设备中回收电子元件会非常耗时。

理想情况下，如果我们试图从主板上移除组件，一个旧炉子或一个未使用的烤箱会很好。如果你手头没有这两样东西，你总是可以用烙铁和一些辫子来攻击电路板——但是谁有时间这么做呢？

PCBHeaven 的 Giorgos Lazaridis 制作了一个快速视频，展示了他最喜欢的在主板失效后回收组件的技术。使用一个 2000 瓦的热风枪，几个手动工具和几个金属锅，他在大约 30 分钟的时间内将整个电路板剥光。他谈到了他在回收时发现的一些最好的部件，并指出了一些很难找到的物品，这些物品可以通过拆下主板轻松获得。

当然，他的过程可能不像在烤箱中翻转 PCB 那么简单，但他的方法便宜，便携，占用空间很小。

继续阅读观看[Giorgos']视频演示，不要忘记看看他过去做过的其他一些很酷的东西，比如他的[温控焊接站](http://hackaday.com/2010/05/19/solder-station-hack-adds-temperature-control/)，这个[丙烯酸弯曲机](http://hackaday.com/2010/12/02/heater-for-bending-acrylic/)，或者这个[台式函数发生器](http://hackaday.com/2011/03/27/function-generator-built-and-mounted-inside-electronics-bench/)。
