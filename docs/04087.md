# arduino-人类合成器

> 原文:[https://hackaday.com/2009/09/23/arduino-human-synthesizer/](https://hackaday.com/2009/09/23/arduino-human-synthesizer/)

(感谢[亚伦]的提示)作为[[加尔文·哈里斯](http://www.calvinharris.co.uk/)]的宣传片，索尼音乐公司的一些创意人员将[组装成一个基于 Arduino 的合成器](http://www.arduino.cc/playground/Main/ArduinoSynth)，由 15 个比基尼女郎组成。通过分析哪些电路是闭合的，[Arduino Mega 能够](http://arduino.cc/en/Main/ArduinoBoardMega)告诉音序器演奏哪个样本。唯一的创新是电路被涂在前面提到的女孩身上，这是一种被称为裸漆的导体漆。

这种颜料是由皇家艺术学院的学生开发的，目前还不能买到，但是他们愿意混合一批用于艺术装置或表演。技术统计(如阻力)尚未公布，但对于一种可洗的油漆来说，它似乎表现得相当好。
