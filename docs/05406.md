# 商用网络摄像头多点触控即将推出

> 原文：<https://hackaday.com/2010/05/27/commercial-webcam-multi-touch-coming-soon/>

[https://www.youtube.com/embed/sBgg33J695I?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/sBgg33J695I?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

看起来东芝已经[推出了基于网络摄像头的多点触控显示屏](http://www.youtube.com/watch?v=sBgg33J695I)。该视频展示了一个类似 iPod 的相册界面，用户站在显示器前，用双手操纵它。这与我们见过的一些[其他多点触控](http://hackaday.com/2009/02/15/rear-projection-tv/) [显示器的区别在于](http://hackaday.com/2009/12/22/subcycles-multitouch-music-controller/)没有触摸的必要(再见指纹！).用户的图像以一种让我们想起最初的 Playstation Eye 的方式叠加在屏幕上。显然这要精致得多，让我们想知道是更好的相机，更好的处理，还是两者兼而有之。
