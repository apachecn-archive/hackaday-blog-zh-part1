# 在雨中表演

> 原文:[https://hack aday . com/2010/02/07/在雨中表演/](https://hackaday.com/2010/02/07/putting-on-a-show-in-the-rain/)

![](../Images/cd7dfc73c8bf2eb52710d95e1e7c36a2.png "arduino-controlled-umbrella")

面对现实吧，在雨中走来走去糟透了。[Matth3w]试图通过在他的雨伞上添加一个 LED 矩阵来为一次不愉快的经历增添一点奇思妙想。该阵列包含 80 个可单独寻址的发光二极管。这是一个依赖于 MIC2981 源驱动器的多路复用阵列，8 行(或本例中的环)，10 列由 Arduino 处理。在休息后的视频中你可以看到效果非常好。现在他已经证明了这一点，你可能想蚀刻自己的 PCB，以摆脱 Arduino 板和原型屏蔽，使控制电路更容易防水。这将是你的[发光雨伞库存](http://hackaday.com/2009/11/12/raid-the-dollar-store-for-an-illuminated-umbrella/)的一个很好的补充。
