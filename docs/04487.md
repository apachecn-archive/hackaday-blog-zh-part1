# McLVDT:一种基于吸管的传感器

> 原文：<https://hackaday.com/2009/12/01/mclvdt-a-straw-based-sensor/>

![](img/f40d8276af31c47c9e79083eebc690ae.png "McLVDT")

我们上周在 YouTube 视频中看到了[Kevin 的]自制线性可变差分变压器，并想了解更多。我们很幸运，他同意分享所有细节以及这些传感器的一系列信息。一个 [LVDT](http://en.wikipedia.org/wiki/LVDT) 被用来测量直线路径上的距离。与[线性光学编码器](http://hackaday.com/2009/11/12/linear-optical-encoder/)不同，这种方法通过测量两个电线圈之间的电感来判断距离。

[Kevin]用一些电磁线缠绕两根不同直径的吸管来制作他的传感器。信号发生器连接到初级线圈，并且测量在次级线圈中感应的结果信号，以揭示物理位置的变化。休息后请观看视频，看看结果。

不难找到一根麦当劳吸管(因此得名“Mc'LVDT”)、一根更小的内部吸管和几英尺长的电磁线。当黑暗的冬天开始降临到你身上时，这将是一个有趣的尝试。
