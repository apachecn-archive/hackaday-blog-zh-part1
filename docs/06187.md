# 万圣节道具:穿着死神长袍的会说话的骷髅

> 原文：<https://hackaday.com/2010/10/27/halloween-props-talking-skeleton-in-reaper-robes/>

![](img/66f166943ae29435f9b0e6a17c4bf557.png "talking-reaper-skeleton")

这里有一个在万圣节守卫前门的好方法。山姆·塞德建造了一个运动控制的会说话的骨架。电子设备相当简单，由 Arduino，WAV shield，PIR 运动传感器，下巴的伺服电机和眼睛的几个红色 led 组成。但是[山姆]在骨架本身的设计上做了一些非常巧妙的事情。正如我们在呕吐的海盗 T2 身上看到的，他用聚氯乙烯建造了身体，这样他就可以把它拆开，便于存放。在收割者长袍下，你会发现一组连接到 WAV 防护罩的电动电脑扬声器。伺服电机安装在头骨中，使用一个小的线臂移动下颚。由于整个事情有点灵活(感谢 PVC)，电机的扭矩导致骨架移动，增加了生活的气息。休息之后不要错过制作精良的视频预演。
