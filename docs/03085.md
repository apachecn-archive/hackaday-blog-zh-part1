# 照明设备

> [https://hacked ay . com/2009/01/12/illuminate/](https://hackaday.com/2009/01/12/illuminato/)

![illuminato](img/823daaac7c80a0da6c756fcafbff27ad.png "illuminato")

[Matt]给了我们这个看起来很酷的项目，叫做 [Illuminato](http://antipastohw.blogspot.com/2009/01/introducing-illuminato-100-gnu-gpld.html) 。在就如何改进 [Arduino](http://www.mahalo.com/Arduino "Arduino - Mahalo") 进行了头脑风暴，并收到了许多关于他的想法的输入之后，[Matt]已经组装了这个具有几项不错改进的板。它有更多的 I/O，更多的代码空间，更快的串行通信速度，内置 PWM，并与现有的 Arduino shields 配合使用。它不仅做了这些改进，而且做得很有型。完成是相当不错的，布局是令人愉快的，它甚至有两排背面的 SMD 发光二极管，增加了一些极客的信誉。休息之后你可以看到几个视频。
