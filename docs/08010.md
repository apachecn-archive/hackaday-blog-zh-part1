# 使用蓝牙打开你的车库，远程启动你的汽车

> 原文:[https://hackaday . com/2011/07/07/use-bluetooth-to-open-your-garage-and-start-your-car-remote/](https://hackaday.com/2011/07/07/use-bluetooth-to-open-your-garage-and-start-your-car-remotely/)

![bluetooth_remote_garage_opener_and_car_starter](../Images/42446587f1df8479b68086104f79baf0.png "bluetooth_remote_garage_opener_and_car_starter")

Instructables 用户[tcollinsworth]是他的 Android 智能手机的忠实粉丝。他几乎到哪里都带着它，所以他认为将尽可能多的家用电子产品与手机集成在一起会很酷。他的车库开门器看起来很容易成为目标，当他在车库时，他决定也黑掉他汽车的远程启动器。

他组装了一个小电路，允许他通过一个名为 Daisy On/Off 的应用程序触发任何设备，这是专为他选择的蓝牙板制作的。一组插脚连接到车库门开启器的终端，另一组连接到他的遥控钥匙。一旦他把所有东西都连接起来，他就用 MakerBot 把他的组件打包到一个项目箱中。完成后，他在 Daisy 应用程序中组装了一个简单的界面，只需按一下按钮就可以启动他的汽车或打开车库。

应该提到的是，[tcollinsworth]为 Daisy 工作，所以他选择组件和 Android 应用程序背后的原因是显而易见的。也就是说，Daisy 蓝牙板的原理图可以在线获得，并且可以使用 BlueTerm 来控制该设备，因此如果您愿意，可以随时推出自己的实施方案。

我们唯一困扰的想法是，该系统可能应该包括一个反馈电路，该电路将信息中继到手机，指示车门确实打开了，汽车已经启动。在没有打开车库的情况下启动你的车绝对是一件坏事。
