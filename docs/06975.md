# LED 多米诺骨牌带来无穷乐趣

> 原文:[https://hack aday . com/2011/03/01/led 多米诺骨牌的无尽乐趣/](https://hackaday.com/2011/03/01/endless-fun-with-led-dominoes/)

![le_dominoux](../Images/7c56b4368d7acc0b893f5885dcf4a80e.png "le_dominoux")

每次推倒多米诺骨牌有大约 30 秒的时间是非常有趣的，当你不忙着为下一轮做准备的时候。[Randy]认为是时候进行电子改造了，以获得持续的，即时的满足感。用一些简单的电子元件，他创造了 Le Dominoux。

这些基于 LED 的电子多米诺骨牌实际上非常简单。每个基本的 Le Dominoux 都构建在一个小正方形的原型板上，由一个光电池或光电晶体管、一个 555 定时器和一个 LED 组成，全部由一个硬币电池供电。当多米诺骨牌背面的光敏元件暴露在足够亮的光线下时，配置为单次触发的 555 定时器被触发。多米诺骨牌前端的 LED 然后被点亮一次。这一过程相当于电子版的多米诺骨牌倒塌。

他设计了几种 Le Dominoux 的变体，作为闪光触发器，用于输出声音，以及用于急转弯。这些变体允许多米诺骨牌以多种不同的方式配置，创造自我维持的灯光表演。如果有人正在寻找一个有趣的项目，向孩子们介绍电子产品，这肯定是它。留下来看看 Le Dominoux 的视频——我们打赌你会欲罢不能。

这当然是兰迪参加的 555 设计大赛的参赛作品，比赛将于美国东部时间今晚午夜结束。
