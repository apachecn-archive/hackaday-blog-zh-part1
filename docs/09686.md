# 将哑秤连接到智能手机

> 原文:[https://hack aday . com/2012/02/13/connecting-a dumb-scale-to-your-smart phone/](https://hackaday.com/2012/02/13/connecting-a-dumb-scale-to-your-smartphone/)

![](../Images/db0774cb30d7c60a3ef69ef041117590.png "bluetooth-enabled-scale")

[Casainho]想用他的 Android 手机上的一个应用程序来跟踪他的体重。他只是需要一种方法将重量读数自动显示在设备上。最后，他在浴室磅秤上安装了蓝牙，并入侵了应用程序，从中获取数据。

他破解的秤是一个数字模型，如果你知道自己在做什么，就可以读取重量数据。[Casainho]已经完成了[一个重量记录秤的黑客攻击](http://hackaday.com/2009/08/14/logging-weight-changes/)，将数据存储在 SD 卡上。所以这是那个项目的再创造，但是用一个蓝牙模块作为输出，而不是存储卡。

现在你可以购买支持 WiFi 的秤，但这远没有像这样的黑客那样有趣。另外，其中一个将花费你大约 200 美元，这个版本的硬件只需 75 美元。它包括一个 LPC2103 开发板，6 美元的蓝牙模块，字符显示器，电池和其他用品。黑客的软件部分得到了很大的帮助，因为(卡萨尼奥)使用的安卓应用程序都是开源的。
