# 快速廉价的远程插座

> 原文：<https://hackaday.com/2009/01/19/quick-cheap-remote-outlets/>

![ir_plugs1](img/95bc01ea616777f5ee9594b3f36c0600.png "ir_plugs1")

【jwad650】想要一个[遥控插线板](http://www.instructables.com/id/Remote_Control_Power_Strip/)。这些都相当昂贵，一个[单插座](http://www.amazon.com/SVAT-WRC101-Wireless-Outdoor-Control/dp/B000HAVVKG/ref=sr_1_1?ie=UTF8&qid=1232393209&sr=8-1)大约 15 美元。他花了大约 50 美元造了一个 6 插座的版本。他正在用一个 [SIS-7c](http://www.simerec.com/chips.html) 解码来自通用遥控器的信号。每个插头由一个 3 安培继电器单独控制。做这个的时候要小心，那些乱七八糟的电线里有很多危险的电击。我们建议您也将其限制在一个外壳中。[jwad650]计划添加外壳、LED 指示灯和保险丝。休息之后你可以看到一段视频。如果你想了解更多关于使用继电器的信息，请查阅[使用继电器](http://hackaday.com/2008/12/05/working-with-relays/)的文章。
