# 材料选择:玻璃毡笔

> 原文:[https://hack aday . com/2011/09/04/material-of-choice-felt-pen-on-glass/](https://hackaday.com/2011/09/04/material-of-choice-felt-pen-on-glass/)

如果你花大价钱买那些落地窗，为什么不把它们也做成你艺术的画布呢？Der Kritzler 是一款电动绘图仪，可以将这一切变成现实。这是一个激光切割的笔筒，悬挂在一对皮带轮上。这些皮带有配重，这使得步进电机更容易平稳地移动笔夹具。Arduino 上运行的控制 Der Kritzler 的固件有一些非常精确的设置要求。由于 Arduino 没有反馈来检测笔的位置，所以当电源打开时，两个步进电机必须相距 1500 毫米，在支架和每个步进电机之间有 1060 毫米的齿带。

使用处理草图将输入图像转换为设备代码。到目前为止[Alex]已经尝试了几种不同的效果，从矢量图形开始，或者使用一些开源工具将位图转换为矢量图形。不要错过他在休息后嵌入的视频演示。
