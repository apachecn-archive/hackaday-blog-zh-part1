# 激光脉冲

> 原文:[https://hackaday.com/2009/02/04/laser-pup/](https://hackaday.com/2009/02/04/laser-pup/)

![laser_pup](../Images/3f263ead705502c1e66361c5b96ebd59.png "laser_pup")

[Joe]给我们发来了这个叫做[激光脉冲](http://laserpup.com/)的项目。在看过 ioBridge 的许多其他项目后，如[倒啤酒](http://hackaday.com/2009/01/01/pouring-a-beer-via-iphone/)和[狗请客配药](http://hackaday.com/2008/12/05/iphone-controlled-dog-treat-dispenser/)，他想制作自己的项目。他的项目允许他通过安装在天花板上的激光笔和他的狗玩耍。我们知道，你希望有更多类似于[激光猫](http://www.hulu.com/watch/1437/saturday-night-live-snl-digital-short-laser-cats-2)的东西，但这仍然很酷。他专门为 iPhone 建立了一个网络界面来控制激光、房间灯光，并显示小狗玩耍的实时画面。休息之后你可以看到一段视频。
