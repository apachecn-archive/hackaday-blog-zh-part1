# 罗斯三岁了

> 原文：<https://hackaday.com/2010/11/09/ros-turns-three/>

![Turtles!](img/345ffc43fb0fd2760500cedac4455e30.png "Turtles!")

自从三年前首次亮相以来， [ROS](http://www.botjunkie.com/2010/11/09/ros-turns-3/) 已经在机器人领域获得了一些[的人气](http://www.ros.org/wiki/Robots)。你可能记得不久前那些令人印象深刻的[四轴飞行器](http://hackaday.com/2010/05/28/quadcopter-acrobatics-like-nothing-weve-seen/)的幕后。ROS 通过为常用组件提供大量代码来帮助抽象机器人的低级功能(例如 [wiimote](http://www.ros.org/browse/details.php?name=wiimote) )。作为一个“操作系统”,它有很多你期望的好特性，比如包管理器。它是开源的，他们的许多项目都有很好的文档记录，这使得它很容易上手。

休息之后，请观看视频，了解它的运行情况
