# 家用晒黑灯成为多氯联苯暴露床的器官捐赠者

> 原文:[https://hack aday . com/2011/08/23/home-tanning-lamps-become-organ-donors-for-a-PCB-exposure-bed/](https://hackaday.com/2011/08/23/home-tanning-lamps-become-organ-donors-for-a-pcb-exposure-bed/)

![](../Images/9ff3c63a3f21d35b643b4cb8ad843c50.png "pcb-exposure-bed")

一些项目从建筑商能够找到的零件中受益匪浅。拿这个[光刻胶覆铜板紫外曝光床](http://raumzeitlabor.de/blog/diy-platinenbelichter) ( [译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http%3A%2F%2Fraumzeitlabor.de%2Fblog%2Fdiy-platinenbelichter))。它看起来像一个商业产品，但实际上是由[TabascoEye]和他的黑客伙伴们制造的。

零件的主要来源是一台平板扫描仪(充当机箱)和两个使用 UVA 荧光灯泡的自晒黑灯。完全靠运气，灯泡和它们的反射器的大小正好适合扫描仪的顶部和底部空腔。控制硬件以 ATtiny2313 微控制器为中心，它从可点击的旋转编码器获取输入，并在字符 LCD 上显示曝光信息。成品应该放在我们看过的其他[专业外观的曝光箱](http://hackaday.com/2011/08/17/diy-pcb-exposure-box-looks-professionally-made/)旁边。
