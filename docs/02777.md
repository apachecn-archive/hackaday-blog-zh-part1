# Wii 轨 ball Mod

> 原文:[https://hackaday.com/2008/11/12/wiimote-trackball-mod/](https://hackaday.com/2008/11/12/wiimote-trackball-mod/)

![wiimote](../Images/c92decb4c366d33a5462c10fc9584460.png "wiimote")

紧随他们最近的快速射击模式之后，AcidMods 的创造性思维又为 Wii 控制器提供了另一个有用的黑客技术。这一次，他们用一个[可点击轨迹球](http://www.acidmods.com/forum/index.php?topic=25141.0)取代了外围设备的“A”按钮，允许玩家快速浏览菜单，而无需重新调整对 Wiimote 的控制。凭借类似于他们在[早期项目](http://hackaday.com/2008/11/07/acidmods-custom-fps-controller/)中展示的天赋，该团队甚至花时间给轨迹球内部添加了 led。休息之后，看看国防部的视频。
