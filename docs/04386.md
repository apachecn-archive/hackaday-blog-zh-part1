# 城市保卫者:位置感知游戏

帮派、领地、城市和发光的蓝色小球有什么共同点？不，不是那个你似乎不记得的醉酒之夜，而是一个新的位置感知游戏，叫做[城市保卫者](http://iad.projects.zhdk.ch/physicalcomputing/seminare/embodied-interaction-hs-2009/projektgruppen/nino-dondi-philipp/)。

游戏背后的[概念](http://iad.projects.zhdk.ch/physicalcomputing/seminare/embodied-interaction-hs-2009/projektgruppen/nino-dondi-philipp/concept/)很简单。一名球员拿着一个球，它知道自己的当前位置，如果需要，它可以通过 led 和扬声器通知你周围环境的变化。然后他或她绕着城市跑，直到球告诉他们一个无人认领或敌人的领土。将球对着一座建筑弹起，那块领土就被占领了。

这个球本身是一个[原型](http://iad.projects.zhdk.ch/physicalcomputing/seminare/embodied-interaction-hs-2009/projektgruppen/nino-dondi-philipp/prototype-manufacture/),由 Arduino、加速度计、振动马达、led、XBee、电池和电线组合而成，全部包装在一个工业橡胶健身球内。不幸的是，在测试后，Urban Defender 团队发现 GPS 和一些其他组件需要放在球的外面和球员身上。

最后，一个保证使用 Arduino 的项目。有人想玩游戏吗？
