# 城市保卫者:位置感知游戏

> 原文:[https://hack aday . com/2009/11/07/urban-defender-location-aware-game/](https://hackaday.com/2009/11/07/urban-defender-location-aware-game/)

 <https://www.youtube.com/embed/zrAFYava258?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>帮派、领地、城市和发光的蓝色小球有什么共同点？不，不是那个你似乎不记得的醉酒之夜，而是一个新的位置感知游戏，叫做<a href="http://iad.projects.zhdk.ch/physicalcomputing/seminare/embodied-interaction-hs-2009/projektgruppen/nino-dondi-philipp/" target="_blank">城市保卫者</a>。</p> <p>游戏背后的<a href="http://iad.projects.zhdk.ch/physicalcomputing/seminare/embodied-interaction-hs-2009/projektgruppen/nino-dondi-philipp/concept/" target="_blank">概念</a>很简单。一名球员拿着一个球，它知道自己的当前位置，如果需要，它可以通过 led 和扬声器通知你周围环境的变化。然后他或她绕着城市跑，直到球告诉他们一个无人认领或敌人的领土。将球对着一座建筑弹起，那块领土就被占领了。</p> <p>这个球本身是一个<a href="http://iad.projects.zhdk.ch/physicalcomputing/seminare/embodied-interaction-hs-2009/projektgruppen/nino-dondi-philipp/prototype-manufacture/" target="_blank">原型</a>,由 Arduino、加速度计、振动马达、led、XBee、电池和电线组合而成，全部包装在一个工业橡胶健身球内。不幸的是，在测试后，Urban Defender 团队发现 GPS 和一些其他组件需要放在球的外面和球员身上。</p> <p>最后，一个保证使用 Arduino 的项目。有人想玩游戏吗？</p> </body> </html>
