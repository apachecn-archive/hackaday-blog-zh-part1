# 仪表板上安装的 Nook Color 是一个完美的头部组件替代品

> 原文:[https://hack aday . com/2011/12/05/dash-mounted-nook-color-is-a-perfect-head-unit-replacement/](https://hackaday.com/2011/12/05/dash-mounted-nook-color-is-a-perfect-head-unit-replacement/)

![jeep-nook-mount](../Images/625748fff0d1d36497071288cff07d1a.png "jeep-nook-mount")

XDA 论坛成员[craigbru]想增强他吉普车上的音频设置，并认为他的[Nook Color 将是一个合适的替代品。](http://forum.xda-developers.com/showthread.php?t=1372396)自从他越狱了电子阅读器，主机的升级让他可以做任何你能想象的事情，一切都可以在舒适的驾驶座上完成。

看到他将继续驾驶吉普车越野，他想设计一种既方便又坚固的安装解决方案，我们认为他做得很好。

dock 是用一个方形的 iVault 立体声底座和他手头的一个塑料平板底座建造的。因为方便是首要目标，他设计了一个快速释放充电解决方案，以及一个使用一组触摸触点的简单音频接口。这使得他可以将平板电脑从吉普车上连接和拆卸下来，而不必每次都用一堆电缆来忙乱。他还在设置中添加了音频均衡器和蓝牙 OBDII 接口，允许他轻松调整自己的曲调和监控引擎。

我们认为这是一个很棒的设置，如果我们说我们不想要一个也是在撒谎。
