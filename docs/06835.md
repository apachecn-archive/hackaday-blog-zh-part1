# 远程激光夜视

> 原文：<https://hackaday.com/2011/02/16/long-range-laser-night-vision/>

[Oneironaut]又回来了，在这个[长距离夜视建造](http://lucidscience.com/pro-laser%20night%20vision-1.aspx)中又制造出一个伟大的黑客。这不是我们第一次看到他制作夜视装置，你可能还记得[他用旧摄像机的取景器组装的单片眼镜](http://hackaday.com/2010/10/02/diy-night-vision-monocle/)。这一次他会用激光代替发光二极管让你看远处的物体。他从一个旧的 CD 刻录机中取出一个红外激光二极管，然后用一个透镜将光点展开，以便照亮更大的区域。一个标准的步枪瞄准镜被用作光学系统，还有一个可以探测红外线的安全摄像头。一如既往，他在图像和文字上做得非常出色。休息之后你会看到他的概述视频。
