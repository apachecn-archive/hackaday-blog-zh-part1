# 万圣节道具:跳舞的蜘蛛

> 原文:[https://hack aday . com/2010/10/18/万圣节-道具-跳舞-蜘蛛/](https://hackaday.com/2010/10/18/halloween-props-dancing-spider/)

![](../Images/9f0646ef394f252e7b63a7d30cb665c7.png "halloween-spider")

[艾萨克]从一台旧打印机上取下马达，用它来让一只蜘蛛在万圣节的墙上跑上跑下。PIC 12F683 使用 MOSFET 来驱动电机。程序循环有一点戏剧性的闪光，用一点颠簸的动作抬起蜘蛛，给它一些生命，然后等待一段时间，然后悄悄地放下蜘蛛(希望是在一个不知情的聚会上)。驱动板是为两个电机设置的，便于在将来的项目中重复使用。这是非常有效的，我们唯一可以建议的是增加几个红色的发光二极管作为发光的眼睛。
