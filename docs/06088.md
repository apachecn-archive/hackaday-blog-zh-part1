# Reddit 黑客为选票和利润

> 原文:[https://hack aday . com/2010/10/08/Reddit-hacking-for-votes-and-profit/](https://hackaday.com/2010/10/08/reddit-hacking-for-votes-and-profit/)

![](../Images/2350e079cc1733e71f71527744fcec68.png "spamming-reddit")

看来有人想出了如何利用 Reddit 系统。这可能以前有人做过，但据我们所知，没有人真正详细地分享过这些方法。[Esrun]写了一些脚本，允许他[注册多个账户，并使用它们来支持故事](http://www.esrun.co.uk/blog/cheating-reddit-auto-votes/)。

黑客是这样做的。脚本注册一组帐户。每个都使用不同的 IP，唯一需要干预的部分是输入验证码。这不需要很长时间。你可以看到上面的脚本界面以及休息后的演示视频。

一旦获得了帐户，就提交一个故事，新帐户对其进行投票。然而，它们并不都是向上投票，因为同时拥有向上和向下投票会将文章放入 Reddit 的有争议部分(这是可取的)，并且不会引起版主的太多怀疑。他做了几个测试，他分享了这些测试，看起来只要文章是有趣的，就可以相当成功。
