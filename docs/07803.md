# 实验音乐 IPad Dock

> 原文:[https://hack aday . com/2011/06/09/experimental-music-ipad-dock/](https://hackaday.com/2011/06/09/experimental-music-ipad-dock/)

![](../Images/53d905acd6183e788f8f882631885c15.png "experimental-music-ipad-dock")

如果你不介意倾家荡产，你可以为你的 iPod 购买不错的音频突破设备。这在一定程度上是因为需求并没有高到令人难以置信的程度，因此商业突破硬件并没有从批量折扣中受益。但也是因为苹果对第三方配件收取授权费(通常被称为“苹果税”)。[里德·加扎拉]决定避开整个局面，制造自己的配件，他称之为[iPad 音频桌](http://www.flickr.com/photos/moonbear3325/sets/72157626688587705/with/5768941165/)。

一切都始于一块分线板。 [PodBreakout Mini](http://www.sparkfun.com/products/10645) 为 iPad 提供了一个易于焊接的接口，并确保反复插拔连接不会破坏焊接连接。从那时起，[里德的]围栏整理技能接管。铝板的形状和曲线给人一种适合昂贵平板设备的感觉。沿着背面，您可以看到用于线路输入、线路输出、视频、麦克风/吉他和耳机的插孔，使 dock 变得非常有用。做一个并不难…但是要做一个看起来这么棒的可能很难。休息之后自己看吧。
