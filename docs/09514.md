# EMGRobotics 机器人控制板

[![](../Images/a9e1b89084c769d9192b7deda7ad0107.png "Spider_Hack_V5_picture_with_IR_motors_RC_servos")T2】](http://hackaday.com/2012/01/17/the-emgrobotics-robot-control-board/spider_hack_v5_picture_with_ir_motors_rc_servos/)

有很多微控制器可以用来制造机器人，但很少有微控制器具备制造机器人所需的精确功能。认识一下[【EMG robotics MSP 430g 2553】](http://buildsmartrobots.ning.com/profiles/blogs/emgrobotics-low-cost-msp430g2553-robot-controller-with-dual-dc-mo "robot control board")机器人控制器板。

不带 CPU 的价格为 15 美元，或者已经插入插座的[MSP430G2553]价格为 17 美元，这款控制板可能会让一些 Arduino 爱好者为他们的下一个项目做准备。除了非常有吸引力的价格(你必须去[主页购买](http://buildsmartrobots.ning.com/))，这款主板还配有一个内置的红外范围传感器，可以驱动多达四个业余爱好伺服电机。如果这对你来说还不够，两个 3 伏 DC 电机可以直接焊接到电路板上的连接，并在任何方向上独立控制。换句话说，你不必浪费时间去尝试构建你自己的 H 桥电路，它已经为你解决了！

文章显示它控制了一只六角蜘蛛。[EMGRobotics]实际上已经在这个平台上做了类似的事情(并且有很好的记录),所以一定要看看关于[黑掉 Hexbug](http://hackaday.com/2012/01/05/giving-the-hexbug-spider-freedom-to-explore-on-its-own/) iteslf 的帖子！
