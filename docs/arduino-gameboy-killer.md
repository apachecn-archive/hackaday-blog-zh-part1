# Arduino 游戏机黑仔

> 原文：<https://hackaday.com/2008/10/28/arduino-gameboy-killer/>

[https://www.youtube.com/embed/OgKSLxRPtu8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/OgKSLxRPtu8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

大约 250 美元，你也可以拥有这个基于 [Arduino 平台](http://www.mahalo.com/Arduino_Hacks "Arduino Hacks - Mahalo")的[小玩意](http://gizmodo.com/5069690/open-source-gamepack-lets-you-bake-a-game-boy-from-scratch)。它是开源的，有一些令人印象深刻的规格。配备震动包、锂离子电池、[有机发光二极管](http://www.mahalo.com/OLED "OLED - Mahalo")显示屏和扩展空间。你可以买预先组装好的零件，也可以成套购买。如果你得到了这个工具包，你会看到控制板已经被设计好了，这样你就可以用左手或右手来焊接你的按钮和操纵杆。唯一的问题？还没有游戏。
