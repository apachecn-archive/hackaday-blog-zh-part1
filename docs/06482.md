# Teensy 可以教你吉他英雄

> 原文:[https://hack aday . com/2010/12/23/teensy-can-school-you-at-guitar-hero/](https://hackaday.com/2010/12/23/teensy-can-school-you-at-guitar-hero/)

![](../Images/790d5aac70f80c8b3750580bbe008d23.png "guitar-hero-autonomous-controller")

[约翰尼·李中]组装了一个完美的吉他英雄系统。他正在使用 PlayStation 2 版本，正如你在上面看到的，他结合了一个控制器连接器和一个小型微控制器板，使用其本机 SPI 协议与控制台进行通信。这款定制吉他控制器通过 USB 从一台计算机接收信号，该计算机从控制台监控视频并计算完美游戏所需的控制器信号。[Johnny]编写了一个 OpenCV 程序，它可以监控视频，从虚拟指板中移除透视，并分析从屏幕上下来的音符的颜色和速度。

正如你所看到的，休息之后，它的工作像一个魔咒。从编程的角度来看，这很有趣，但是如果你想玩黑客游戏，也许你应该[打造你自己的班卓琴英雄](http://hackaday.com/2009/03/26/banjo-hero/)。
