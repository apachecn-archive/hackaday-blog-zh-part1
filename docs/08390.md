# 杰克的视频已经修复。

> 原文:[https://hackaday.com/2011/08/22/jacks-video-has-been-fixed/](https://hackaday.com/2011/08/22/jacks-video-has-been-fixed/)

![](../Images/036f1d6e173efce598ac0542a7ad0e30.png "High Voltage")

今天早些时候，[Jack] [发布了他的最新视频](http://hackaday.com/2011/08/22/video-shocking-jack-into-submission-with-high-voltage/)。然后，他迅速跑到秘密的黑客日总部，开始下一个工作。不幸的是，Youtube 出现了一个奇怪的问题，导致视频无法观看。我们现在已经修复了它，并希望让每个人都知道。

我们也收到了很多关于视频的问题，主要是关于我们将来会报道什么内容。显然，到目前为止，我们只做了一些非常简单和愚蠢的事情。我们想从简单开始，以便掌握一切。[Jack]告诉我们，接下来的视频将会更加严肃，涉及 pcb 设计的一些不同方面。我们迫不及待地想看看他会想出什么。
