# [Sprite_tm]的三分量调频发射机

> 原文：<https://hackaday.com/2012/01/26/sprite_tms-three-component-fm-transmitter/>

![](img/edcaeec7a892689cbadcb6d45f819a41.png "radio")

当 Regency TR-1 晶体管收音机在 20 世纪 50 年代上市时，它被誉为现代微电子学的奇迹。TR-1 只有四个晶体管和少量其他元件，是现代工程的奇迹。[Sprite_tm]可能会击败那些老前辈。他用有史以来最少的零件制造了一台调频发射机。

像大多数[Sprite_tm]的构建一样，这是一个难以想象的聪明作品。[Sprite]将 ATtiny45 的内部 RC 振荡器超频到 24 MHz。在意识到 PLL 以振荡器频率的四倍运行时，正好在 FM 波段的中间，他开始设计一个微型 FM 发射器。

[Sprite_tm]想起了他在[单调](http://www.oldskool.org/pc/MONOTONE)上的作品，为 hit ATtiny 做了一首短歌。用于构建的固件从他的歌曲中提取音符，并稍微改变 96 MHz 的 PLL 频率，从而充当微型 FM 发射器。

有用吗？好吧，如果你想把它比作一个麦克风先生，范围是非常有限的。话虽如此，但它确实有效。这是一个由微控制器和电池组成的调频发射机，令人印象深刻。休息之后看看[Sprite_tm]的演示。
