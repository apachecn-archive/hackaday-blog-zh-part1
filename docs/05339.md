# Biopunk 手表:时间、温度、游戏

![](../Images/78df8294011ec17eb59b7ba99ed9052b.png "arduino-biopunk-watch")

[Matthew Garten]基于 Arduino 打造了这款[手表。表盘是一个小的彩色显示屏，允许您选择以数字、二进制或模拟格式显示时间。为了与 Hackaday 上最近的趋势保持一致，他有一个基于手套的附件，在手指上有温度传感器；对于消防员或手指神经受损的人(我们在考虑](http://www.optimizedforce.com/2010/05/arduino-watch-biopunk-edition.html)[、黑暗人](http://en.wikipedia.org/wiki/Darkman))。对于任何情况下的娱乐，他包括一个轨迹球和玩突破或 16 位颜色绘画的能力。细节很少，但显然他很快会分享更多。现在，在休息后观看视频，并想办法将它缩小到一个漂亮的包装中，就像 Pong Watch 喜欢的[一样。](http://hackaday.com/2009/09/28/pong-on-your-wrist/)
