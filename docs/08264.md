# 单芯片 USB AVR 开发

> 原文:[https://hackaday.com/2011/08/07/1-chip-usb-avr-development/](https://hackaday.com/2011/08/07/1-chip-usb-avr-development/)

![](../Images/dcf37502deb76ab841f311dfb9fdb992.png "placaavr")

如果有一个内置 USB 设备支持的 AVR 微控制器，这样你就不需要单独的编程器或串行链路了，这不是很好吗？事实上有很多，这个很棒的教程([谷歌翻译](http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=auto&tl=en&u=http://webdelcire.com/wordpress/archives/269&act=url))是[使用 ATMega 16/32U4 微控制器](http://webdelcire.com/wordpress/archives/269)的快速简单速成班。

这些 8 位 AVR(16k 和 32k)具有你在 Mega AVR 中期望的通常特性列表。26 个 GPIO 引脚和一个非常容易焊接的 44 引脚表面贴装封装，微控制器还具有一个 USB 2.0 全速/低速设备模块，并允许通过标准引导加载程序进行编程。

一旦你有一个非常标准的板组装你需要 Atmel 的 [FLIP](http://www.atmel.com/dyn/products/tools_card.asp?tool_id=3886) RS232，USB，或 CAN 设备编程软件(Windows 或 Linux)和你最喜欢的 AVR IDE 设置，你就可以走了！
