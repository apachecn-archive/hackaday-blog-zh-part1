# 电子管替身

> 原文:[https://hackaday.com/2010/07/06/dacatron-stand-in/](https://hackaday.com/2010/07/06/dacatron-stand-in/)

![](../Images/571229b1307b951396769d271ed82fc2.png "fake-decatron")

认为 Decatron 的旋转辉光很酷，但不想处理电压问题？[Osgeld]通过开发一个假的电子管避开了这个问题。纳尔逊上将(摩根船长的廉价表弟)提供了一个飞机大小的酒瓶形式的外壳。发光二极管是常见的东西(不确定是阳极还是阴极)，所以它们最终可以通过末端出来的一堆电线单独寻址。这将极大地简化我们一直想要构建的[厨房定时器](http://hackaday.com/2009/04/06/dekatron-kitchen-timer/)。休息过后，看到闪烁的灯光一圈又一圈。
