# 用旧的立体声接收器控制 LED 照明

> 原文:[https://hack aday . com/2011/08/10/control-led-lighting-with-a-old-stereo-receiver/](https://hackaday.com/2011/08/10/control-led-lighting-with-an-old-stereo-receiver/)

![stereo_controlled_home_lighting](../Images/494cf37ea91c56fb5eef5f7294f6b759.png "stereo_controlled_home_lighting")

[Marklar]正在进行的一个项目需要一个红外接收器，而他当地的电子产品商店刚刚缺货。他在垃圾堆里翻找，发现了一个旧的立体声接收器，所以他决定在扔掉它之前把红外模块拿出来。一旦他把它拆开，他认为他可以利用他在里面找到的各种各样的电子元件，然后[开始着手一个新项目](http://arduino.cc/forum/index.php/topic,67382.0.html)。

控制面板上有他最感兴趣的部件。使用 Arduino，他能够轻松地与旋转编码器以及按钮进行交互，这给他提供了一种廉价而简单的方法来控制他的家庭照明系统。通过一点编程，他能够将照明预设映射到各种按钮，并使用旋转编码器控制 led 的亮度和颜色。作为一个额外的奖励，他保持了红外接收器完好无损，也可以无线控制他的设置。
