# 复古游戏机爱好者的便携式游戏

> 原文:[https://hackaday . com/2011/02/28/便携游戏-复古游戏机爱好者/](https://hackaday.com/2011/02/28/portable-gaming-for-retro-console-lovers/)

![atari_2600_portable](../Images/322a770acb3af7b126425693ddaec05f.png "atari_2600_portable")

这些年来，便携式视频游戏控制台的转换已经有了相当大的份额，但很少有人比这款更能拨动我们热爱复古的心弦。Modretro 论坛成员[Mario]用 Flashback 2 Atari 控制台克隆体构建了一个看起来很棒的便携式 Atari 2600。

他将闪回板拆开，放入一个小塑料盒中，然后添加了一个 3.5 英寸的 LCD 屏幕，以及其他便携式游戏系统捐赠的一些控制器。添加了一对可充电电池，以及一个小型放大器和声音扬声器。

虽然闪回内置了 40 个游戏，但他真的想添加一个盒式磁带端口，所以他在箱子里留下了一点空间，他就这样做了。当一切完成后，他喷上几层复古的橙色油漆，然后就收工了。真的，唯一缺少的是一些漂亮的假木皮，也许还有一些粗毛地毯。
