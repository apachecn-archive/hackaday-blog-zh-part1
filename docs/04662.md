# RFID 仿真器

> 原文:[https://hackaday.com/2010/01/03/rfid-emulator/](https://hackaday.com/2010/01/03/rfid-emulator/)

![](../Images/fd482f380c309132059855e9afc3a5f6.png "rfid-emulator")

【亚历山大】[搭建了一个 RFID 仿真器](http://www.alexanderguthmann.de/en/emulator.html)。它使用一个线圈(此处未画出)和一个 ATmega8 来表示任何符合 EM4001 的标签。这种迭代需要连接到计算机，以将标签 ID 信息发送到微控制器。在休息后的视频中，看起来他正在使用一个 DIY RFID 阅读器来测试这个。如果将这两者结合起来，不再需要电脑，他手上就会有一个 [RFID 电子欺骗器](http://hackaday.com/2006/02/13/proximity-card-spoofer-proxmarkii/)。
