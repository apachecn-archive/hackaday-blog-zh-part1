# USB 伺服喷射器

> 原文:[https://hackaday.com/2008/12/26/usb-servo-squirter/](https://hackaday.com/2008/12/26/usb-servo-squirter/)

![squirt](../Images/3515e8e065e1525d59a1f94501cc19e7.png "squirt")

这是一个学习如何控制伺服的好项目。他们用 [USB NerdKit](http://www.nerdkits.com/kits/) ，一个泵和一个伺服系统制作了一个 [USB 控制的水枪](http://www.nerdkits.com/videos/servosquirter/)。这是一个学习[脉宽调制](http://en.wikipedia.org/wiki/Pulse_width_modulation)和控制伺服系统的好教程。教程非常全面，有很棒的图片和视频，你可以在休息后看到。
