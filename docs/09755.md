# 旋转平台使最无用的机器概念变得有用

> 原文:[https://hack aday . com/2012/02/24/rotating-platform-makes-most-used-machine-concept-used/](https://hackaday.com/2012/02/24/rotating-platform-makes-most-useless-machine-concept-useful/)

![](../Images/ed2d0cf6204ee558fefa187a71d7f0fa.png "rotating-platform-with-auto-reverse")

【瓦伦丁】用一个简单的概念[建造了这个自动反转旋转平台](http://www.doityourselfgadgets.com/2012/02/simple-rotating-plate.html)。这个概念非常简单，电机的导线连接到一个双刀双掷开关上，该开关允许极性反转。朝一个方向拨动开关，它就会顺时针旋转。向另一个方向翻转，它就会逆时针旋转。

在这种情况下，他利用了最没用的机器的力量。这种常见的黑客使用类似的开关，但通过让移动部件充当致动器来完成任何事情。这个很有用，利用单臂或双臂扳动开关，让平台向后旋转。在休息后的视频中，你可以看到它被用来创建一个扫描安全摄像头。但是[瓦伦丁]也展示了它作为畅销商品的转盘的作用。我们认为对于这两个目的来说，齿轮传动有点太快了，但是放慢它又是另一天的事情。T3
