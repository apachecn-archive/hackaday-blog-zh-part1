# 带上你自己的胸牌

> 原文:[https://hackaday.com/2012/02/07/bring-your-own-name-badge/](https://hackaday.com/2012/02/07/bring-your-own-name-badge/)

![](../Images/fa237809ed0c7e59af5d8bdf1dd6763f.png "bring-your-own-name-badge")

以前是口袋保护套里面的东西直接反映了你的极客水平。但是，在我们这个快节奏的数字时代，这是行不通的。不过，我们认为[乔纳森]正朝着正确的方向前进，他创造了一个滚动的 LED 姓名徽章，带着它去参加会议。有了合适的外壳，这可以达到与[沃兹的手表](http://hackaday.com/2009/11/03/wozs-watch-makes-air-travelers-nervous/)相同的极客水平。这个小设备里有很多东西，但近距离的可读性看起来不像是其中的一个功能，所以在你接近他进行对话之前，请确保你看了一眼标签。

如您所见，该项目的 PCB 与景观 ID 卡的外形相同。它拥有一个 8×5 LED 矩阵，与运行它的 MSP430 芯片的寄存器配合良好。他承认，硬件可能不会持续很长时间，因为芯片直接复用显示器，没有电阻或 LED 驱动器进行电流保护。但是这个设计有潜力。它使用可充电电池(我们喜欢这种电池),他还在电路板上添加了一个二维码，以便于交流联系信息。休息之后，我们嵌入了他对该项目的描述。 <https://www.youtube.com/embed/k1NqT2BIg6E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>
