# 在 RGB LED 矩阵上显示视频和 gif

> 原文:[https://hack aday . com/2011/05/19/displaying-video-and-gif-on-RGB-led-matrix/](https://hackaday.com/2011/05/19/displaying-video-and-gifs-on-rgb-led-matrices/)

![led_matrix](../Images/0f8a81ce5420f3aa2892171c0b218575.png "led_matrix")

[Mathieu]在中国度假，在那里的众多电子市场中挑选了一些有趣的玩具。他发现的最有趣的东西[是一对 RGB LED 矩阵](http://www.limpkin.fr/index.php?post/2011/05/12/Oops...-I-dit-it-again%21)。它们有两种不同的风格，一种用于室内显示，一种用于室外显示，分辨率分别为 64×32 和 32×16。

如果你之前读过他的博客，你就会知道他是 LED 矩阵的忠实粉丝，所以他一回家就买了一大堆 LED 矩阵并开始实验，这很自然。使用他在之前的黑客中展示的同一 Atmel FPSLIC LED 矩阵控制板[，他能够立即启动并运行 LED 矩阵。他调整了他的网络摄像头项目，以利用新的面板，他还添加了一个全新的功能。通过 MatLab，他现在可以在面板上显示任何类型的动画 gif，正如你在下面的视频中看到的。面板看起来很棒，如果我们周围有一些这样的面板，毫无疑问我们可能会无限重复播放](http://hackaday.com/2011/04/10/webcam-images-processed-and-played-back-on-led-display/)[这个视频](http://www.youtube.com/watch?v=fUDiAmaQ0Vw)。

他说，尽管它们的来源有些问题，但这些面板质量一流，如果其他人感兴趣，他愿意组织某种形式的团购。
