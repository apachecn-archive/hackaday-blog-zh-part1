# 迷你拱廊橱柜

> 原文：<https://hackaday.com/2009/04/15/mini-arcade-cabinets/>

![mini-arcade](img/c69fc009ee89c7dd5a52de71e86203aa.png "mini-arcade")

[Pocket _ 鲁乔]在这一点上做得非常出色。他正在用旧游戏机制作微型拱廊橱柜。这篇帖子主要是在谈论世嘉天才版，但他也为 [PC 引擎](http://en.wikipedia.org/wiki/Turbografx_16)(又名 turbografix 16)做过一个。他带我们从芯片中提取 RGB 视频，以及从一个廉价的街机控制器中收集按钮。对于屏幕，他用的是 PSone 便携式液晶显示器，几乎没有修改过。真正突出的是最终的布局。他建造了大约一英尺高的小型拱廊式橱柜来存放它们。这些太棒了，我们想要一个。不，我们想要一整个迷你拱廊。休息之后可以看一段视频。
