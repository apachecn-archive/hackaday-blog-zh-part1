# POV 名片保证会引起你的注意

> 原文:[https://hack aday . com/2011/03/21/POV-business-card-is-guaranteed to-get-you-notified/](https://hackaday.com/2011/03/21/pov-business-card-is-guaranteed-to-get-you-noticed/)

有人说，分发名片是一种过时的做法，因为智能手机无处不在，可以用来交易或记录联系信息。然而，Instructables 用户[海绵]并不同意，他正在推动一场“名片复兴”,用他的 POV 名片。

这些卡片是在他的地下室里手工制作的，上面有几个贴片发光二极管，显示他的名字，来回挥动时会显示他的电话号码。他的 verison 几乎与标准名片一样大，使用 PIC 来管理显示，并使用倾斜传感器来监控卡片的运动。他的演练相当全面，包括构建卡所需的每个步骤的教程。他讨论了建造你自己的蚀刻槽，转换用于 PCB 转移目的的层压机，建造回流焊炉控制器，以及黑客攻击水族箱泵用作真空动力的拾取和放置。

最终的结果是一张漂亮的名片，确保你不会忘记见过他。请继续阅读，观看卡片运行的视频。
