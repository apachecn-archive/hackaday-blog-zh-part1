# Arduino 热摄像机

> 原文：<https://hackaday.com/2011/03/09/arduino-thermo-cam/>

[Maxbot]想要一个红外摄像头，他可以用它来扫描一个区域，并在他的电脑上显示结果，结果是这个 [便宜-热感摄像头](http://arduino.cc/forum/index.php?PHPSESSID=4440bd3c2c274523bc8dee7e6effefd6&topic=54655.msg391245#msg391245) 。这个小装置是一个廉价的热传感器，安装在一对用于 XY 扫描的伺服系统上，定制 java 软件和一个 Arudino 来配合这两者。

sanner 的分辨率为 42×32 像素，虽然有限，但仍然有用。为了有助于颜色的平滑，成品的每隔一行相对于其相邻行进行插值。该设备需要大约两分钟来测量 1344 个点中的每一个点，但它在速度上的不足远远弥补了成本，大约 100 美元的建造成本和易于建造。

虽然目前还不知道你需要什么部件以及如何连接一切的确切细节，但在工作中有一页会让你“很快”知道所有好的细节，尽管如果你受到启发要复制这些漂亮设备中的一个，似乎最困难的部分是接入传感器的电子设备和定制软件。

便宜，粗糙，甚至自带胶带。休息之后我们为你准备了一小段视频。
