# Nexus One 获得 Linux

> 原文:[https://hackaday.com/2010/07/07/nexus-one-gets-linux/](https://hackaday.com/2010/07/07/nexus-one-gets-linux/)

[https://www.youtube.com/embed/3wQ9XogfjRU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/3wQ9XogfjRU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

考虑到 Nexus One 已经如此容易被黑客攻击，我们只能想象由于在设备上运行 Ubuntu 而带来的一系列全新的有趣的事情。[Max Lee]下定决心不仅要在 Nexus One 上安装 Ubuntu，还要安装 Debian，他写了一个完美的安装指南来帮助那些想要尝试的人。

他有一点欺骗，让 Ubuntu 在后台运行，而 X11 界面只是简单的 VNCed，但他仍然做了一件了不起的工作，用大量的图片和细节帮助你在 Nexus One 上实现 Ubuntu。
