</span> <p>上面是一个视频，详细介绍了<a href="http://www.youtube.com/watch?v=6M3MeADb1dE" target="_blank">填充双面表面贴装 PCB </a>的一种方法。几周前，我们报道过<a href="http://hackaday.com/2009/10/13/how-to-populate-a-surface-mount-pcb/">使用模板</a>为 PCB 涂焊膏。在评论中，有一个关于使用模板的优点的争论，以及一个关于如何填充双面电路板的<a href="http://hackaday.com/2009/10/13/how-to-populate-a-surface-mount-pcb/#comment-101103">问题</a>。这是一个很好的问题，因为将电路板回流两次会导致底部的元件脱落。</p> <p>[Wim L's] <a href="http://hackaday.com/2009/10/13/how-to-populate-a-surface-mount-pcb/#comment-101384"> comment </a>提到有几种方法可用于双边总体。在视频中，您将看到没有使用模板，而是通过踏板驱动的注射器进行粘贴。首先将糊状物涂在电路板的下面，然后加入一点环氧树脂来固定元件。然后将每个零件正常定位，并在回流焊炉中烘烤。这个过程既回流焊料，又固化环氧树脂。当电路板第二次回流时，当顶部焊料达到其熔点时，环氧树脂将底部元件固定在适当的位置。</p> <p>这种涂锡膏的方法比使用模板要慢。但是如果操作正确，每个组件都可以获得所需的焊料量。</p> </body> </html>
