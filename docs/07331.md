# 基于 Cycle II FPGA 板构建的序列器

> 原文:[https://hack aday . com/2011/04/08/sequencer-built-on-a-cycle-ii-FPGA-board/](https://hackaday.com/2011/04/08/sequencer-built-on-a-cycle-ii-fpga-board/)

![](../Images/4807c88ad1d2a2c87b14ba3fc9e30843.png "cyclone-II-sequencer")

[马特的]完成他的计算机科学学位。作为课堂作业的一部分，他编写了自己的序列器，运行在 Cyclone-II FPGA 开发板上。我们在折页下面嵌入了一段视频，向您展示它能做什么。板上提供的按钮和 led 实际上允许他创建一个不错的用户界面。每个滑动开关上方都有一个表面贴装 LED，提供循环中哪些节拍打开和关闭的反馈。上面蓝色部分还有一排瞬时按钮。[Matt]使用这些来调整设置，例如为每个滑动开关存储的音高。他甚至用 VGA 输出来做灯光秀。

我们以前见过这个 Altera 板，[用来驱动一个落沙游戏](http://hackaday.com/2010/12/15/falling-sand-game-on-and-fpga/)。这个硬件将花费你大约 200 美元，但是考虑到你可以用它做的所有有趣的事情，这并不坏。
