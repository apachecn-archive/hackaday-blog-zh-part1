# 自动 CD 开膛手建立从乐高和其他部分

> 原文:[https://hack aday . com/2011/10/31/automated-CD-ripper-build-from-Lego-and-other-parts/](https://hackaday.com/2011/10/31/automated-cd-ripper-build-from-lego-and-other-parts/)

![](../Images/fd0ac9c043b898a4eff4ed2a8f8c1b5a.png "automated-cd-ripper")

[Paul Rea]决定通过将数据翻录到硬盘上来处理掉他的 CD 和 DVD 库。他收藏了相当多的光盘，而且不喜欢一张一张地翻看。因此，他开始着手建造自己的自动 CD 开膛手/复制器。

他马上就有了几个建造规格。他希望它是独立于平台的、可靠的，并且构建成本低廉。我们认为他真的击中了目标，但他确实提到他已经有了第二台复印机的想法。这个版本的手臂和传动装置大量使用乐高零件。基座上有一个步进电机，可将机械臂摆动至一个圆弧，该圆弧可到达输入堆、光驱 try 和输出箱。机械臂本身有一个由两部分组成的木制夹持器，位于 CD 上方，并使用一个限位开关来感应垂直方向何时处于夹持光盘的正确位置。我们喜欢阅读他的日志，因为他讨论了他遇到的各种建筑挑战以及如何克服每一个挑战。

我们以前见过一些类似的其他版本。我们最喜欢的之一是很久以前的。
