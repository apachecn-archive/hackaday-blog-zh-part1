# 使用专用 Android 手机监控车库门

> 原文：<https://hackaday.com/2011/10/11/garage-door-monitoring-and-control-using-a-dedicated-android-phone/>

![](img/47c8c31a699ce49c8e43499eae943753.png "android-phone-garage-monitor")

[Sean]碰巧有一部额外的 Android 手机，想看看他可以用它来实现什么类型的家庭自动化。一个简单的硬件改造和一些来自安卓市场的应用程序让他能够远程监控他的车库门。

硬件改造是一个我们已经看过的黑客行为。BTmate 使用一个带附加晶体管的蓝牙耳机来缩短车库门开启器上的连接。唯一的问题是你需要在蓝牙工作的范围内。[Sean]通过使用两部 Android 手机增加了一层抽象。一个永久安装在车库里，处理蓝牙连接，而另一个使用 VNC 隧道在任何有互联网连接的地方。

但是为什么就此打住呢？他知道这一个功能是多余的，并增加了第二个功能，这是手机的完美选择。由于它有自己的摄像头，他使用 tinyCam 应用程序创建了一个网络摄像头服务器。这甚至允许他打开和关闭 LED，以便在昏暗的光线条件下获得更好的视野。休息之后看[Sean]的演示。

过度杀戮？也许吧，但是如果你有一部坏了 LCD 的手机，这可能就是赋予它新用途的东西。
