# 无线电控制的甲虫飞行镜头

> 原文:[https://hack aday . com/2009/09/25/无线电遥控甲虫飞行镜头/](https://hackaday.com/2009/09/25/radio-controlled-beetle-flight-footage/)

今年早些时候，当加利福尼亚大学的研究人员通过电子植入控制一只甲虫时，我们感到惊讶。最初报道时可用的视频显示，当通过芯片施加电刺激时，甲虫被拴在原地。新的[自由飞行](http://scienceblogs.com/neurophilosophy/2009/09/flight_of_the_remote_controlled_cyborg_beetle.php?utm_source=selectfeed&utm_medium=rss)视频现已发布。虽然这个动作是相当零星的，但很明显，开始飞行、停止飞行和向左或向右转弯的简单命令正在产生预期的效果。休息后看看这个机器人的动作。难道 [DARPA](http://www.darpa.mil/) 离释放大批昆虫战士攻击毫无防备的大众又近了一步？
