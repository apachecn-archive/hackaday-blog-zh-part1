# 中情局的神奇机器人

> 原文：<https://hackaday.com/2011/02/10/the-cias-amazing-bots/>

当你有几乎无限的预算时，你可以完成一些令人惊奇的事情。这一点最近变得非常明显，因为中央情报局一直在展示它的一些老技术。你在上面看到的那只蜻蜓接近真人大小，实际上会飞。他们雇佣了一个手表制造商来制造一个微型内燃机来运行它。光是这一点就已经相当惊人了，但是这个东西实际上是在 70 年代飞行的。在[进一步检查翅膀](http://www.flickr.com/photos/ciagov/5416228423/in/photostream/#/photos/ciagov/5416228423/in/photostream/lightbox/)后，我们实际上不知道这个吸盘应该如何飞行。尽管我们持怀疑态度，但你可以在休息后看到一小段视频。你还可以观看机器鲶鱼“查理”的视频。
