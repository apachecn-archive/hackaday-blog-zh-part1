# 大型磁铁在万圣节会产生火花，谁知道呢？

> 原文：<https://hackaday.com/2010/10/31/large-magnets-spark-on-halloween-who-knew/>

![](img/f88870989ed110c2ff0074329038e739.png "I remember one halloween I went as a mad scientist, and this brings the question: What are you guys going as for Halloween? (I'm going as StarFox...chicks dig fluffy tails).")

这个[过大的磁铁](http://www.arduino.cc/cgi-bin/yabb2/YaBB.pl?num=1288424668)当然完成了疯狂科学家的样子(为了更疯狂的样子，拿一罐红色食用色素的水，放在一个大花椰菜里，即时大脑在罐子里)。

磁铁的底部是用临时热刀切割的涂有油漆的泡沫；为了让磁铁产生火花，激光蚀刻出带有分形图案的丙烯酸树脂，并在丙烯酸树脂末端嵌入发光二极管。Arduino 处理闪烁的 led，并为火花的嗡嗡声产生 60Hz 的 PWM 脉冲。最终的结果是令人满意的疯狂，当你练习邪恶的不祥的笑声时，在跳跃后捕捉磁铁的视频。
