# 乐高路由器案件击败工厂完成

> 原文：<https://hackaday.com/2010/01/13/lego-router-case-bests-factory-finish/>

![](img/51c4f05ba745722db51b6ed025b8f215.png "lego-router-case-brings-the-awesome")

盯着【卢克的】[乐高路由器案](http://tfvlrue.wordpress.com/2010/01/08/lego-router-wrt54gl/)；盯着它！路由器没什么特别的，一个很好玩的 WRT54GL 黑。我们已经看到了[被用作机器人](http://hackaday.com/2008/10/07/inexpensive-powerful-router-based-robot/)，变成了[的战争驱动盒](http://hackaday.com/2006/08/05/self-contained-war-driving-box/)，显然这就是 [dd-wrt 定制固件](http://hackaday.com/2006/04/07/dd-wrt-running-on-wrt54g-version-5/)的开始。

[Luke]在 MLCAD 中设计了这个箱子，并找到了一个零件的卖家，价格刚刚超过 50 美元。我们认为它比股票案例[更好看，如果你把它用于不同的项目](http://hackaday.com/2009/12/03/make-a-mac-on-the-cheap/)，这是取代它的一种方式。休息之后我们嵌入了[卢克的]集会视频。如果你喜欢这个案例，看看[他的乐高 PC 案例](http://tfvlrue.wordpress.com/2008/06/25/lego-computer/)。
