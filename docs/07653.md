# 当你可以拥有一个象棋机器人时，为什么要建造一个数控铣床呢？

> 原文:[https://hack aday . com/2011/05/23/why-build-a-CNC-mill-when-you-can-have-a-chess-robot-instead/](https://hackaday.com/2011/05/23/why-build-a-cnc-mill-when-you-can-have-a-chess-robot-instead/)

[Patrick McCabe]喜欢与电脑下棋的挑战，但他不满足于屏幕游戏的单调体验。没问题，他只是建造了自己的[龙门式象棋机器人](http://myrobotlab.org/node/95)，可以和它对弈。不要混淆，他仍然不必触摸棋子，而是使用上图左侧看到的专用控制板。安装在龙门架上的机械臂负责两个玩家的移动。

这是一个非常普通的 CNC 构建，使用四个步进电机沿着精密杆滑动移动位。一台 Arduino Mega 驱动该系统，一台 PC 使用一个名为[我的机器人实验室](http://myrobotlab.org/faqs)的程序来完成繁重的工作。

我们当然喜欢[Patrick]花点时间让橱柜和可见部分看起来很漂亮。国际象棋是一种文明的游戏，未完成的部分是不合适的。我们在他的文章中没有看到，但我们真的希望他实现的一个功能是让机器人在游戏开始时自动重置棋盘的能力。
