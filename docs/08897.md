# 装有灯和警报器的玩具车是孩子们的最爱

> 原文:[https://hack aday . com/2011/10/23/装有灯光和警报器的玩具汽车是儿童的乐园/](https://hackaday.com/2011/10/23/toy-car-fitted-with-lights-and-sirens-is-a-childrens-delight/)

![police_car_lights_and_sirens](../Images/a717f352cbc96048b17eae023ebccc5b.png "police_car_lights_and_sirens")

[Nathan]是 DangerousPrototypes 论坛的成员，他正在寻找一个可以用来参加他们正在举行的 7400 逻辑竞赛的项目。他的孩子乘坐了一辆小型警车，但是顶部的灯条没有灯，当他的孩子在房子周围追捕坏人时，汽车没有发出任何声音。[内森]得到了他所需要的所有灵感，并去了他的工作室，以解决玩具创造者的明显疏忽。

他设计了一个电路，松散地围绕着他不久前在邪恶的疯狂科学家实验室看到的赛昂风格的灯，该实验室使用振荡器和 4107 十进制计数器来控制灯。他的设计使用 74HC04 hex 逆变器作为振荡器，而十进制计数器用于调制警报器的频率并控制旋转的 LED 信标。

如果你问我们，最后的结果是伟大的。根据[内森]在这个项目上花费的时间，房子里的一位“未透露姓名的成年女性”对这些额外的东西没有什么印象，但他的孩子们绝对兴奋不已。
