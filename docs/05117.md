# 谢妮转速表

> 原文：<https://hackaday.com/2010/03/30/nixie-tachometer/>

 <https://www.youtube.com/embed/nHiYCZkAq5o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent>

</span> <p>谢妮转速表对我们来说是新的。我们已经看到了各种各样的显示器，但没有一个转速表。在对他的超级棒的<a href="http://en.wikipedia.org/wiki/Holden_Gemini" target="_blank">霍顿双子星</a>的工厂点火定时系统有了广泛的不满之后，【布雷特】安装了一个 MegaJolt 电子点火系统。最重要的是，他在仪表板上安装了一个谢妮转速表，增加了一点额外的特色。它不仅有数字读出，你可以看到一个谢妮“条形图”在顶部。如果你想直接去看电影，就跳到 2:30 左右。你可以从论坛帖子下载 Arduino 代码。</p> </body> </html>
