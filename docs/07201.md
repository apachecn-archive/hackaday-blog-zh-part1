# DIY 扫描电镜

> 原文：<https://hackaday.com/2011/03/23/diy-scanning-electron-microscope/>

![](img/2f35aaa1eebaf94e7e936bec2f02c9a7.png "sem2 (Custom)")

[Ben Krasnow]最近完成了一台自制的扫描电子显微镜,并在他的博客上发布了一段视频。

构建本身使用了许多现成的组件，非常有创造性。我们特别喜欢多长的螺纹黄铜棒不仅用于支撑，而且还用于保持列对齐和微调各种光束聚焦组件之间的间距。一个大的玻璃“钟罩”覆盖了整个装置，当用机械真空泵从内部抽出空气时，它被密封在底板上。

为了产生图像，一个类似于传统 CRT 电视显像管的电子枪以 5kV 的电压加速电子从显微镜顶部向下穿过一个长的铜柱。在这一过程中，光束被电子透镜聚焦和控制，就像传统的光学透镜控制光线一样。在主柱的底部附近，有垂直放置在 X 和 Y 方向上的静电偏转板，允许光束在样品表面上精确扫描。当这种高能电子束扫描样品时，散射的表面电子被附近的探测器拾取，该探测器由荧光屏和光电倍增管组成——该系统据称比直接测量少量电子具有更高的灵敏度。

虽然最初几次扫描的分辨率仅为 50 微米左右，但这一早期成功清楚地表明，该设备的功能符合预期，并将为未来的改进提供一个良好的起点，最终目标是分辨率降至 1 微米范围。

尽管本保证在这种能量水平下产生的 x 射线甚至不会穿透玻璃室，但你可以肯定的是，如果我们参观他的车库，我们肯定会穿上一些锡纸保护，就像这些家伙一样。
