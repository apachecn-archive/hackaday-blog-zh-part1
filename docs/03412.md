# RepRap 上的印刷电路

> 原文：<https://hackaday.com/2009/04/19/printing-circuits-on-the-reprap/>

![reprappcb](img/d049d2522031a0bc3bf45c919b9e435d.png "reprappcb")

[里斯·琼斯]一直在与 RepRap 团队合作开发一种印刷电路板的方法。机器首先在塑料基底上打印出金属沉积的通道。他们改造了他们的[夹送轮进料器](http://hackaday.com/2009/01/17/reprap-pinch-wheel-extruder/ "RepRap pinch wheel extruder  - Hack a Day")来处理实芯焊料(焊剂会造成混乱)。挤压焊料的比热不足以熔化塑料。他们制作了一个测试视频(嵌入下面):一个光学终点挡板。该团队还一直在试验[将进料机构从挤出机](http://blog.reprap.org/2009/04/bowden-extruder-concept.html "RepRap: Blog: Bowden extruder concept")中分离出来。
