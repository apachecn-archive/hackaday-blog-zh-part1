# USB 加密狗生成并输入您的密码，因此您不必

> 原文:[https://hack aday . com/2012/03/05/USB-dongle-generated-and-enter-your-passwords-so-you-don ' t-have-to/](https://hackaday.com/2012/03/05/usb-dongle-generates-and-enters-your-passwords-so-you-dont-have-to/)

![usb-password-dongle](../Images/297ddc3e68025f9ad943d3be0bc9bb12.png "usb-password-dongle")

像许多其他企业一样，[joon as Pihlajamaa]的雇主要求他每隔几个月更换一次密码。他没有一次又一次地想出一个复杂而又容易记忆的密码，而是制造了一个小型的 USB 设备来替他完成这项工作。

他拆除了一个旧的 USB 记忆棒，在一小块 perfboard 上安装了一个 ATtiny85 及其所需的组件。利用他从以前的 USB HID 修补中收集到的知识，他对 ATtiny 进行了编程，使其充当 USB 键盘，每当他插入它时，它就会为他输入密码。

USB 加密狗不仅可以为他输入密码，还可以在他需要的时候通过简单的击键生成一个新密码。显然，只需要有人拿到他的 u 盘就能危及安全，但它确实比键盘下的便利贴管用。
