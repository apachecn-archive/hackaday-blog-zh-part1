# 遥控末日之爪

> 原文：<https://hackaday.com/2009/10/27/remote-controlled-claw-of-doom/>

[https://www.youtube.com/embed/P-9_dMJKh9E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/P-9_dMJKh9E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

机器人爪子太棒了，就这样。远程控制的机器爪，这是一个全新的酷的水平——即使目的只是捡起积木。设置很简单，一个[视差推进器](http://www.parallax.com/tabid/407/Default.aspx)控制整个系统，首先通过轮询网络服务器，查看用户调用的变量的变化。这些变化然后被传递给控制爪子的继电器。为了避免在黑暗中摸索，他甚至包括一个网络摄像头。我们讨厌看到他用如此昂贵的设备来控制一个玩具，但也许有一天他会搬到[更大更好的东西](http://hackaday.com/2009/09/08/hand-of-man-mechanical-claw/)。
