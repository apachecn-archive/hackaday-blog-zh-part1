# 二维码钥匙链帮助您丢失的钥匙找到回家的路

> 原文:[https://hacka day . com/2011/10/06/QR-code-key-fob-helps-your-lost-keys-find-the-way-home/](https://hackaday.com/2011/10/06/qr-code-key-fob-helps-your-lost-keys-find-their-way-home/)

![qrcode_key_finder](../Images/3ab7860b4f225e47c238cee210457a1b.png "qrcode_key_finder")

你不讨厌这种感觉吗？当你意识到你不知道你可能把钥匙放在哪里时的那种感觉？如果你不幸在公共场所丢失了它们，很有可能它们已经不在了。泵站一名成员[celtwolf]认为，如果你的钥匙可以帮助别人在找到他们时立即与你联系，那就太好了，所以他发明了一种钥匙链来实现这一功能。

SMS 可以使用一种类似于我们都熟悉的“mailto”协议的 URI 方案，因此[celtwolf]生成了一个 URI，可以向他的手机发送一条短信“我找到了你的钥匙！”。他用 URI 生成了一个二维码，然后用激光切割机将其蚀刻在一块丙烯酸板上。他用黑色的聚合物粘土填充凹陷部分，烘烤，然后涂上一层指甲油以增加耐用性。

现在，如果有人找到他的钥匙，并用智能手机拍下二维码，他会立即收到一条短信，让他知道钥匙安然无恙。多好的主意啊！
