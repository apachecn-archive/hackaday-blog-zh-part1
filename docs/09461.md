# 投影仪项目没有结果，但它是一个有趣的旅程

> 原文:[https://hackaday . com/2012/01/09/projector-project-bears-no-fruit-but-it-a-fun-ride/](https://hackaday.com/2012/01/09/projector-project-bears-no-fruit-but-it-was-a-fun-ride/)

![](../Images/25399b059d75b1169ad9f03d89b1690e.png "projector-project-fail")

不管你的意图有多好，或者你的黑客能力有多强，有时候你就是不能在每个项目上都跨越终点线。这是一个我们不愿看到未完成的项目，但很明显[大量的工作已经投入到回收这些智能白板投影仪](http://milwaukeemakerspace.org/2012/01/smartboard-projector-project-abandoned/)中，是时候减少损失了。

硬件是一台 Smartboard Unifi 35”计算机，投影仪安装在伸缩杆上。它是为使用触敏白板而制造的，密尔沃基创客空间的人没有这种白板。投影仪可以工作，但它只会显示一条信息，指示用户将计算机连接到白板。既然他们已经有了几个这样的投影仪，挽救一下功能就好了。

第一次尝试是替换投影仪的视频信号。蚀刻了几块测试板来试验 DVI 输入。这包括几次逻辑嗅探运行，以查看计算机正在推动什么来显示警告消息。遗憾的是，该小组无法让设备做出反应。但这为你提供了一个玩周一早间黑客游戏的绝佳机会。看看他们在上面的链接中发布的数据，并在评论中告诉我们你会怎么做。
