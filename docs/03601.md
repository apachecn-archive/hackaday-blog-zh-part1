# 感应电炉

> 原文:[https://hackaday.com/2009/06/22/induction-furnace/](https://hackaday.com/2009/06/22/induction-furnace/)

![induction furnace](../Images/67dab67f44c45994c6b41663f2b084d1.png "induction furnace")

【蒂姆·威廉姆斯】自己做了[感应炉](http://webpages.charter.net/dawill/tmoranwms/Elec_IndHeat1.html)。铜管线圈形成初级绕组，因为要加热的材料成为短路的次级绕组。负载材料受到在射频下工作的高功率磁场的影响。快速变化的磁场在材料中感应出电流，产生大量热量。强大的动力需要一个冷却系统来匹配。在下面的视频中，可以看到感应炉正在熔化普通食盐。
