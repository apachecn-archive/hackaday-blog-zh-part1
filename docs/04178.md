# 如何打开主锁

> 原文:[https://hackaday.com/2009/10/06/how-to-crack-a-master-lock/](https://hackaday.com/2009/10/06/how-to-crack-a-master-lock/)

很久很久以前，我们介绍了一种在大约 30 分钟或更短时间内破解主锁的方法。这是相同方法的复兴，但现在[检索组合的指令是由](http://www.markedwardcampos.com/files/gimgs/13_mcamposfinal.png)[马克·爱德华·坎波斯](http://www.markedwardcampos.com/)创建的信息图形格式。

如果你第一次没有尝试过，这里是它是如何工作的:一个物理弱点，数学和蛮力的组合被使用。首先，在转动拨号盘的同时拉起锁闩，就可以得到密码的最终数字。由于锁的构造方式，正确的数字可以通过这个技巧推断出来。其次，已经为你计算了所有可能的第一和第二数字组合的表格。第三，你的工作是强行找出正确的可能性表，它只包含大约一百种组合。

我们并不真的对盗窃重罪感兴趣，希望你也不是。但是，我们有一个令人讨厌的习惯，需要使用在抽屉里放了几年的密码锁，却不知道正确的密码是什么。
