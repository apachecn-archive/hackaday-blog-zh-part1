# 电子收费系统

> 原文:[https://hackaday.com/2011/02/17/electronic-tolling-system/](https://hackaday.com/2011/02/17/electronic-tolling-system/)

![](../Images/2336a361bd1412d45b30ba22acd3d589.png "electronic-toll-project")

对我们来说，任何项目中最难的部分都是提出开创性的想法。偶尔你只需要通过再造一个现有的产品来作弊。这就是 EngineersGarage 在这个[收费广场项目](http://www.engineersgarage.com/microcontroller/8051projects/simple-toll-plaza-rfid-interface-at89c51)中所做的。如果你看看这篇文章顶部的各个选项卡，你会发现他们使用了 8051 微控制器来集成字符 LCD、RFID 阅读器和键盘。从那里开始，需要大量的编码来添加读取多个标签、查找储值和创建补充系统的功能。当然，它在这种形式下并没有太大的用处，但它会让你和你周围那些闪亮的部分有所关联，它可能会引导你走向更有意义的东西。像往常一样，休息后会有一段演示视频。

如果这不能让你满意，也许这另一个 [RFID 访问系统](http://hackaday.com/2011/02/07/simple-rfid-access-system-clone/)更适合你。
