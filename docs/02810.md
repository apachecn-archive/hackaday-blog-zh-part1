# 闹钟自动百叶窗

> 原文:[https://hack aday . com/2008/11/18/闹钟-自动百叶窗/](https://hackaday.com/2008/11/18/alarm-clock-automated-blinds/)

![alarm](../Images/0cb6f8d7731ac9354b81f80a42c352aa.png "alarm")

[Anupam Pathak]知道被一个传统的闹钟叫醒是多么的不和谐。他决定黑掉一个闹钟，这样它就能打开他房间的窗帘，让自然光进来。他找到了当警报被触发时变高的那根针，并用它来发出警报信号。微控制器激活直接连接到百叶窗的伺服系统。他在钟的侧面有开关来手动控制百叶窗和切断声音警报的电源。休息后嵌入的视频。T3
