# DIY 电子动画企鹅摇摇和凹槽

> 原文:[https://hack aday . com/2011/02/25/DIY-animatronic-penguin-shakes-and-grooves/](https://hackaday.com/2011/02/25/diy-animatronic-penguin-shakes-and-grooves/)

![animatronic_penguin](../Images/400f785cf14bb557a6e589ea7b0a0ab8.png "animatronic_penguin")

Instructables 的用户[djsfantasi]想用木偶做一个动画节日展示，为当地的艺术项目筹集资金。然而，在做了一些阅读和研究后，他认为为显示器制造电子动画角色并不是一个遥不可及的想法。

他的第一个想法是造一只企鹅，这样他就可以主要关注躯干的运动，而不必同时活动手臂和腿。他的目标是让他的企鹅通过摆动和摇动以及拍打翅膀来“跳舞”。使用胶合板，四个伺服，连同一些杂项连杆和电缆，他去工作。

企鹅使用 SSC-32 伺服控制器操作，其核心是 ATmega168 MCU。这使得他可以独立地控制所有的伺服系统，也可以协调一致地进行组合运动。企鹅的嘴也有功能，使用一个电路将它的动作同步到一个音频文件。

虽然机器人目前通过串行电缆连接到他的计算机上，[djsfantasi]提到他目前正在开发一个 iPhone 应用程序，该应用程序将能够无线控制机器人。他现在需要做的就是制造一个电子动画汤姆伺服，然后在这个上面盖上一张乌鸦皮，他就万事俱备了！
