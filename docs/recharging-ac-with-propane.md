# 用丙烷给交流电充电

> 原文：<https://hackaday.com/2010/06/15/recharging-ac-with-propane/>

[https://www.youtube.com/embed/v2vN_mxW4Bc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/v2vN_mxW4Bc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

随着夏季的升温，空调系统在许多气候条件下是必不可少的。[格雷森的]系统在冬天遭受了一些损坏，导致它排出制冷剂，避免了爆炸的情况。在他可以在里面冷静下来之前，他需要给它充电，他选择在他的冷却系统中使用丙烷。根据我们的朋友谷歌的说法，这不是他最初的想法，但以前已经做过很多次了。[格雷森]指出，虽然丙烷是可燃的，但它在火灾中并不一定比二氟甲烷或 R22 更危险，R22 是一种逃离他的系统进入高层大气的讨厌的小气体。

上面的视频包括对系统充电和所需工具的简要说明。在鼓起勇气尝试之前，我们需要仔细考虑一段时间。现在，我们将坚持使用[格雷森]更普通的方法，比如[让一些伺服电机唱歌](http://hackaday.com/2009/07/16/cnc-tetris-theme/)或[缓解我们的庭院工作困境](http://hackaday.com/2010/05/10/toby-reel-mower-bot-evolved/)。
