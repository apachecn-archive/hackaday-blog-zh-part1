# 电动雨披让你在旅途中保持温暖

> 原文：<https://hackaday.com/2012/03/22/electric-poncho-keeps-you-warm-on-the-go/>

![electric-poncho](img/9b97da13a6f417fe1c46286b3e8b3ac2.png "electric-poncho")

由于我们已经经历了温暖的冬天和春天，考虑保暖的方法可能看起来很奇怪，但是没有比现在更好的时间来为寒冷的天气做准备了。

在他的博客中,[Berto]正在思考当天气再次变冷时如何保暖，并决定手边准备一件保暖的雨披。他找到了一个非常简单的方法来制作他的加热雨披，只需要一点电线，一些线和布，以及一把剪刀。

[Berto]拿起一个 12V 的电动盲板，然后开始在它的中间切开一个人头大小的裂缝，避开加热线圈。他在洞周围缝了一块布，以确保它不会随着时间的推移而撕裂，然后他将毯子连接到藏在背包里的 12V 电池上。

结果就是一件温度可变的雨披，你可以用它在各种情况下保暖。来自 Hacked Gadgets 的[Alan Parekh]说他认为雨披在全天的雪地摩托旅行中会很棒，我们认为他是对的！
