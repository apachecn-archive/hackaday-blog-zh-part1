# 释放 Evalbot 操作系统

> 原文:[https://hackaday.com/2011/08/25/evalbot-os-set-free/](https://hackaday.com/2011/08/25/evalbot-os-set-free/)

![](../Images/97b1adcf717aca33b76757ed5536d5b2.png "sa_announcement_01")

[Theo]向我们透露了每个 TI Evalbot 所有者可能感兴趣的一些事情，uC/OS-III 内核的手册和源代码现在可以下载。UC/OS-III 是 evalbot 附带的，它是该芯片和许多其他芯片的实时操作系统。对于大多数爱好水平的人来说，它的问题是仅仅手册就要 100 美元，除非你已经了解这个系统，否则它听起来没有什么吸引力。

但现在，US/OS-III 的作者 micrium 已经发布了源代码，可以在非商业应用中免费使用，并且支持它的每个芯片的手册可能会激起人们对这个整洁的小 RTOS 的更多兴趣。尽管它需要用户登录。
