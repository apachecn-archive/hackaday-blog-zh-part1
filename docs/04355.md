# 简易拨号电话

> 原文：<https://hackaday.com/2009/11/01/easy-dial-telephone/>

![easy-dial-phone](img/2d969ef844d1c6793a5136e584f420b9.png "easy-dial-phone")

[Taufeeq 的]祖母需要能够给她的家人打电话，但是由于视力和记忆力的问题，这是一个困难的任务。他决定帮助她制造一部按下按钮就能自动拨号的电话。[Taufeeq]制作了一个挂在墙上的盒子，里面有一个接收器挂钩和两个自动拨号按钮。基于我们在一月份报道的 [LED 按钮](http://hackaday.com/2009/01/29/led-push-buttons/),这些按钮被点亮并且很松散。一个单独的盒子里装有一个微控制器和一个用于拨号的双音多频集成电路。这些都是从一个标准的电话连接到一个印刷电路板。

结果看起来很棒，并且通过简化的控制使使用手机更加容易。休息之后，我们将演示视频包括在内。
