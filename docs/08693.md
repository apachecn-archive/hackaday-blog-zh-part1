**一次性咖啡机**

![](../Images/d4ae2b8efdca58b29426705bca1e18e6.png "links-disposable-coffee-maker")

[Sepehr]没有咖啡机，当地的咖啡店都没有 joe。他用一次性杯子和刀子做了一个滴滤咖啡机。

**柔性盲文显示器**

![](../Images/e54ed5ad4f91e0b3b8f9c0bfc047955a.png "links-braille-display")

正在开发薄膜技术来帮助视力受损的人。这种柔性有机发光二极管显示器有[嵌入的肌肉细胞，这些肌肉细胞创造了一种盲文显示器](http://www.youtube.com/watch?v=B6PpNhe_PX8)。[谢谢艾伦]

**可打印的 iPhone 三脚架**

![](../Images/632933b8e3232cedf6deeb2498716623.png "printable-iphone-tripod-mount")

想让这些 iPhone 视频更稳定一点，图片不那么模糊吗？试试这个[克里斯]发明的可打印三脚架。

**Arduino 微距照片**

![](../Images/e32469e243fe49c8e635f54389ca328c.png "arduino-macro-images")

说到照片，[丹尼尔]来信分享了一些他拍摄的 Arduino 的微距照片。那些热爱 Arduino 的读者肯定会对它们感兴趣。

**给老鼠铺地毯**

![](../Images/bd7493e581545c6fc7d1428f56a0f030.png "links-carpet-mouse")

用布料覆盖鼠标的一部分，为鼠标增添独特的质感。上面的[鼠标外壳的下半部分被一层类似地毯的材料](http://forum.pcekspert.com/showthread.php?p=1953449)[翻译](http://translate.google.com/translate?hl=en&sl=auto&tl=en&u=http%3A%2F%2Fforum.pcekspert.com%2Fshowthread.php%3Fp%3D1953449)。[谢谢点击]
