有旧光盘和 Arduino 吗？为什么不把它做成[旋转视点显示](http://www.bricogeek.com/index/cat/7/nid/913/)。它是用一个软驱和一个 cdrom 的部件，以及一个 Arduino 和一个小型无线收发器建造的。Arduino 使用红外传感器来检测旋转，并向 led 发送数据。向设备发送新消息时使用无线。这个链接是西班牙语的，所以这里有[谷歌翻译](http://translate.google.com/translate?u=http%3A%2F%2Fwww.bricogeek.com%2Findex%2Fcat%2F7%2Fnid%2F913%2F&hl=en&ie=UTF-8&sl=es&tl=en)。
