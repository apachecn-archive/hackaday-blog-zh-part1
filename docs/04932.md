# 打造自己的轮毂电机

> 原文:[https://hackaday.com/2010/02/18/build-your-own-hub-motor/](https://hackaday.com/2010/02/18/build-your-own-hub-motor/)

![](../Images/b445f149089244e1a29788cff06f03a8.png "razor-hub-motor")

[轮毂电机](http://www.instructables.com/id/Make-Your-Own-Miniature-Electric-Hub-Motor/)将动力放入车轮内部。[Teamtestbot]深入探讨了制造这些电机的方法和原因，从零件到绕组，再到功率比背后的数学原理。工作示例将一个电动马达放在一辆 Razor scooter 的后轮内。[过去的项目使用皮带](http://hackaday.com/2009/10/18/1480w-scooter-motor-guarantees-head-trauma/)将电机的工作转移到踏板车的轮子上。通过整合马达和轮子，你最终会得到一个看起来更干净的产品。休息后，请查看电机测试和踏板车试驾。

想了解更多关于制造自己的电动马达的技巧，请看一下我们在 11 月份报道过的 [Fly Electric 页面](http://hackaday.com/2009/11/17/rc-airplane-motors-from-computer-trash/)。
