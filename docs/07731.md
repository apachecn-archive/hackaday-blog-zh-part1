# 枫木板上的 VGA 输出

> 原文:[https://hackaday.com/2011/06/02/vga-out-on-a-maple-board/](https://hackaday.com/2011/06/02/vga-out-on-a-maple-board/)

![](../Images/dcf00094e6f8f0bb684ad7f99c7b343f.png "Leaf")

Leaf Labs 的团队刚刚发布了一个新的库来展示他们的 Maple 开发板的 VGA 功能。虽然它只是一个 16×18 像素的图像，但它显示了过去在 Maple 上实现视频的许多发展。

枫叶是一个很棒的兼容 Ardunio 的小主板，有一个[非常熟悉的 IDE](http://leaflabs.com/docs/ide.html) 。我们已经在之前介绍过枫树[。Maple 使用 72MHz 的 ARM，而不是有些有限的 AVR，这使得需要一些马力或严格定时的应用程序变得容易得多。](http://hackaday.com/2009/08/22/maple-beats-up-arduino-takes-its-shields/)

我们已经看到一些项目使用增加的功率，如[吉他效果屏蔽](http://hackaday.com/2010/07/25/guitar-effect-shield-for-maple/)。枫树有可能被制成一个游戏控制台，让 [Uzebox](http://belogic.com/uzebox/index.asp) 相形见绌，但我们想知道黑客读者会用这个开发板做什么。

观看跳转后的视频，看看枫树的 VGA 能力在仅仅[几个月](http://www.youtube.com/watch?v=rT1eKLEvIvo)后已经走了多远，或者查看 Leaf Lab 的[枫树库](https://github.com/leaflabs/libmaple)。
