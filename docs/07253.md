# 螺旋桨灯

> 原文:[https://hackaday.com/2011/03/30/propeller-lamp/](https://hackaday.com/2011/03/30/propeller-lamp/)

![](../Images/499fee889290adfe79f3f44efdeb027a.png "lamp")

为了替换客厅里耗电的卤素灯，[杰森·多利]设计了一款遥控可调光的 led 灯([图片](http://www.facebook.com/album.php?aid=344728&id=671403972&l=4320ebfe4f))。灯的主体是一个非常有趣的想法，用半透明的 HDPE 外壳作为灯罩，用 PVC 柱作为结构，展示了几个废物筐。

柱子被包裹在一个 16 英尺长的螺旋 led 灯带中，并用电线连接，这样它们就可以被分组控制。光输出(估计)大约与 100-150 瓦的白炽灯相同，而仅消耗 24 瓦。

该灯通过通用遥控器控制，采用由螺旋桨驱动的 TLC5940，所有这些都位于 CNC 加工的 PCB 上。在一盏灯下有这么多马力，你可以预期它不会只是简单地变暗和熄灭，所以在休息后加入我们的视频，看看如何打开一盏灯。
