# 用自制的 x 光透视一切

> 原文:[https://hack aday . com/2012/01/01/用自制 x 光透视一切/](https://hackaday.com/2012/01/01/see-through-everything-with-a-home-made-x-ray/)

![](../Images/ada81c60661f36e28cc91c553967b7d3.png "xray")

[掷弹兵]建造了他自己的 x 光机。他对高压并不陌生——我们见过他的[雅各布之梯](http://hackaday.com/2011/08/17/who-knew-jacobs-ladder-builds-had-so-many-options/)和[马克思发电机](http://hackaday.com/2011/08/23/lightning-simulator-cant-send-you-back-to-the-future/)。他肯定能应付高压和危险的设备。有了这台便携式 x 光机，[掷弹兵]已经开始让盖革计数器超载。我们很高兴他知道自己在做什么。

[掷弹兵]便携式 x 光机的关键部件是[柯立芝管](http://en.wikipedia.org/wiki/X-ray_tube#Coolidge_tube)，这是一种简单的真空管，在 75 千伏电源的帮助下产生 x 射线。完成建造的[看起来棒极了。两个仪表显示进入 x 射线管的毫安和千伏，三个数码管显示曝光时间。](http://i.imgur.com/wzdTd.jpg)

尽管[掷弹兵]没有 x 光胶片，但他可以通过闪烁屏幕透视东西，当暴露于电离辐射时，闪烁屏幕会发出荧光。有两张 x 射线照片——一张显示了笔的内部和硬盘的内部(如标题图所示)。

x 射线的输出用盖革计数器测量。[掷弹兵]能够在 50 码处每秒左右击中一次，在 1 英尺处发出非常大的白噪声。休息之后，看看[掷弹兵]的 Buildlounge 激光切割机大赛提交的视频。
