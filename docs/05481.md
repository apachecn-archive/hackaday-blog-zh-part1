# 加压聚氯乙烯水枪

> 原文：<https://hackaday.com/2010/06/11/pressurized-pvc-water-gun/>

在看到这款 [PVC 水枪](http://www.atterberry.net/funstuff/pvc-water-gun/)紧随[乐高狙击步枪](http://hackaday.com/2010/06/11/seven-six-two-millimeter-full-plastic-jacket/)之后，Hackaday 的【Caleb Kraft】将今天命名为“儿童登陆日”。如果你不介意让孩子们使用你的空气压缩机上的快速连接，这是一个很棒的夏季项目。它实际上只是一个由聚氯乙烯制成的“T ”,有两个阀门用于空气和水的管理，第三条腿上有一个汽水瓶作为储水池。在休息后的短片中，你可以看到，与 DIY 超级酒鬼的[相比，每次充电并没有获得很多拍摄时间，但这种构建也没有那么复杂。](http://hackaday.com/2010/03/20/diy-super-soaker/)
