# 自动对焦辅助灯

> 原文:[https://hackaday.com/2009/01/26/autofocus-assist-light/](https://hackaday.com/2009/01/26/autofocus-assist-light/)

![afassist](../Images/9247e73bca4e28cd5364903a67a0eddb.png "afassist")

[Aki]的尼康 D2H 没有配备自动对焦辅助灯。他的其他相机都有，他喜欢这个功能，所以他决定在他的 D2H 中安装一个。他连接到自动对焦系统，这样当快门按下一半时，他的 LED 就会有电压。虽然电路需要改进，但他发现快门释放期间灯一直亮着，影响了他的测光。休息之后你可以看到黑客的行动。
