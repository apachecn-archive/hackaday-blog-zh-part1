# 健壮的漫游者看起来很有趣

> 原文：<https://hackaday.com/2011/07/22/beefy-rover-looks-like-tons-of-fun/>

![mantis_six_wheeled_remote_control_robot](img/85cee9fc1625794597e905ed92a1d2f8.png "mantis_six_wheeled_remote_control_robot")

[Eric]和技术发烧友的工作人员，刚刚对一个被他们称为“螳螂”的非常强大的机器人进行了最后的修饰。

乍一看，这个六轮链条驱动的机器人令人印象深刻。机器人以 Netduino 为中心，控制四个 60 安培的速度控制器。速度控制器的任务是驱动四个结实的 CIM 电机，这些电机推动机器人以相当不错的速度前进。

为了这个项目，团队设计了他们自己的控制器应用程序，名为 [IPGamepad](http://code.google.com/p/ipgamepad/) ，它允许机器人由任何 Android 设备控制。正如你在下面的视频中看到的，界面非常简单，使得控制这个机器人非常容易。

虽然螳螂无论如何都不是廉价产品，但它看起来确实很有驾驶乐趣。工作人员将带着这个机器人和另外两个机器人去底特律的 Maker Faire，所以如果你在这个地区，一定要过来打个招呼。
