# 增强现实技术

> 原文:[https://hack aday . com/2011/12/28/augmented-reality-ex-nihilo/](https://hackaday.com/2011/12/28/augmented-reality-ex-nihilo/)

[![](../Images/7cf8437add0e47b3645bdf5178c3c12a.png "kinect")T2】](http://hackaday.com/wp-content/uploads/2011/12/kinect.jpg)

[David]提交了一个很好的项目，用 ARtoolkit 演示增强现实,并讨论虚无意义的深层哲学基础。好消息是他能够用记号笔在一张纸上创建一个音量控制按钮。坏消息是哲学治疗有点薄弱；[大卫]建立了一些很酷的东西，所以我们现在可以让它滑动。

这个建筑的灵感来自于用 Kinect 和一张纸制作的[即兴声卡](http://www.youtube.com/watch?v=7yuqMQWYVnM&feature=autoplay&list=PL49E51448D885EECC&lf=plpp_video&index=8&playnext=2)。声卡背后的想法很简单——在纸上画一些按钮，用它们来播放简短的声音剪辑。【大卫】带着这个想法为[奥卡姆剃刀](http://webs.uvigo.es/occams-razor/)做了一个关于增强现实的小教程。

硬件非常简单——只需要一个网络摄像头、一张纸和一支马克笔。在[David]在纸上画了一个大正方形后，代码会将其识别为音量控件。逆时针旋转纸张会增加音量，顺时针旋转会降低音量。这是进入增强现实基础的一个简洁的构建。

休息之后，请观看[David]的构建视频演示。
