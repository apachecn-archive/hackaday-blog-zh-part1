# PR2 发号施令

> 原文:[https://hackaday.com/2010/06/17/the-pr2-calls-the-shots/](https://hackaday.com/2010/06/17/the-pr2-calls-the-shots/)

![](../Images/48298b00e95735c2d7bc35301150c031.png "pr2-calls-the-pool-shots")

你能在台球比赛中击败这个机器人吗？这让我们产生了一种“让伍基人赢”的态度，但我们仍然喜欢看这个视频。这是[PR2](http://hackaday.com/2009/07/07/willow-garages-pr2-robot/)打台球感谢在柳树车库的人们。它使用激光传感器来检测台球桌的腿，使用摄像机来发现台面上的钻石和球，而不是使用头顶摄像机的。他们减少了使用开源台球物理库 [FastFiz](http://billiards2.stanford.edu/FastFiz/main.html) 所需的编码工作。最后一步是建立一个界面，让机器人可以使用线索。休息之后来看看吧(没有双关语的意思)。
