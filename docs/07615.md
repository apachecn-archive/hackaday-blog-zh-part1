# 廉价可靠的便携式人脸识别系统

> 原文:[https://hack aday . com/2011/05/16/廉价可靠的便携式人脸识别系统/](https://hackaday.com/2011/05/16/cheap-and-reliable-portable-face-recognition-system/)

![faceaccess_portable_facial_recognition](../Images/f03639f7d4d8239571064ccebba033c0.png "faceaccess_portable_facial_recognition")

对于他们的高年级 ECE 4760 项目，工程系学生[Brian Harding 和 Cat Jubinski]组装了一个令人印象深刻的便携式人脸识别系统，名为 face access。该系统依靠特征脸方法来帮助区分不同的用户，这是两人使用 MatLab 完成的一个过程。

他们说，在培训期间，该系统只需要与计算机连接一次。在此期间，在 MatLab 中扫描和处理人脸以创建特征脸集，然后上传到扫描仪。

一旦被编程，扫描仪独立于计算机运行，由它自己的 ATmega644 微控制器供电。用户只需按下系统上的一个按钮就可以注册他们的面部，并将他们的身份作为特征脸的组合存储在板载闪存芯片中。一旦个人被注册，可以按下第二个按钮来访问人脸识别系统保护的任何资源。

学生们说，他们的系统在 88%的时间里是准确的，没有假阳性——考虑到系统的便携性和成本，这是非常令人印象深刻的。
