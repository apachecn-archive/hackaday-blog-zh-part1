# 一键安卓生根

> 原文:[https://hackaday.com/2009/08/17/1-click-android-rooting/](https://hackaday.com/2009/08/17/1-click-android-rooting/)

![rooting](../Images/b8d1f2d6f1893adcd42fc269cd78b0be.png "rooting")

在过去的几天里，扎根 T-Mobile G1 和 myTouch 3G 变得容易多了。[Zinx]发布了 FlashRec，可以让你在你的 Android 手机上刷新一个新的恢复镜像。它利用了 Linux 内核漏洞 [CVE-2009-2692](http://web.nvd.nist.gov/view/vuln/detail?vulnId=CVE-2009-2692 "National Vulnerability Database (NVD) National Vulnerability Database (CVE-2009-2692)") 。该应用程序可以让你备份当前的图像，然后闪现 Cyanogen 的恢复图像 1.4。一旦完成，你就可以使用任何你想要的定制 Android 版本了。Android 和 Me[在他们的网站](http://androidandme.com/2009/08/news/how-to-root-a-t-mobile-g1-and-mytouch-3g-android-phone/ "How to root a T-Mobile G1 and myTouch 3G Android phone")上记录了整个过程，并指出[存在数量多得离谱的定制 rom](http://db.androidspin.com/android_build_information.asp "AndroidSPIN: Android ROM Information v11.0")。下面嵌入的是来自【unknownkwita】的视频，展示了生根过程。
