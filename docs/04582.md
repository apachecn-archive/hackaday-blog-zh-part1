# 你没有看到双:RGB 圣诞树

> 原文:[https://hack aday . com/2009/12/25/you-not-seeing-double-RGB-Christmas-trees/](https://hackaday.com/2009/12/25/youre-not-seeing-double-rgb-christmas-trees/)

[https://www.youtube.com/embed/GAt6A98lXQw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/GAt6A98lXQw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[mrpackethead]，[创造了这棵树的怪物](http://doityourselfchristmas.com/forums/showthread.php?t=9639)。正如视频中所示，它能够显示动画、图案和潜在的视频。这个 6 米高的作品布满了 [2000 防水 RGB](http://response-box.com/rgb/2009/09/and-now-for-something-different/) LED 模块。圣诞树的软件是用[苹果自己的 Quartz Composer](http://developer.apple.com/graphicsimaging/quartz/quartzcomposer.html) 和[编写的，集成到 Madrix](http://www.madrix.com/) 中，这是一款旨在控制 led 的软件。600W 系统 [100%无 Arduino](http://www.enttec.com/index.php?main_menu=Products&prod=70304&show=description)，在新西兰运行的成本不到相当于每小时 0.04 美元。

[Geoist]选择了 Arduino 的方式[装配他自己的](http://thegeoist.blogspot.com/2009/12/arduino-powered-christmas-tree.html)更小的 RGB 圣诞树。[Geoist]在他当地的百货商店发现了一个有点俗气的光纤模型，他渴望利用它的变色能力。打开后，人们发现它只不过是由一个灯泡[和一个彩色滤光器转盘](http://www.nbweixing.com/english/prod2-04.htm)控制的。[Geoist]为了支持一个有 3 个 RGB 灯连接到 Arduino 的试验板，拆除了设置。它的草图可以在他的网站上找到。
